### 为什么选择回溯+约束传播

| 特性 | 回溯+约束传播优势 | 对比 |
|------|----------------|------|
| 完备 | ✅ 能找到所有解 | 启发式可能遗漏 |
| 高效 | ✅ MRV、前向检查大幅剪枝 | 纯回溯慢 |
| 通用 | ✅ 适用于各种CSP | - |

## 算法实现

```rust
const SIZE: usize = 9;
const BOX_SIZE: usize = 3;

pub struct Sudoku {
    grid: [[u8; SIZE]; SIZE],
}

impl Sudoku {
    pub fn new(grid: [[u8; SIZE]; SIZE]) -> Self {
        Self { grid }
    }

    pub fn solve(&mut self) -> bool {
        self.backtrack()
    }

    fn backtrack(&mut self) -> bool {
        if let Some((row, col)) = self.find_empty() {
            for num in 1..=9 {
                if self.is_valid(row, col, num) {
                    self.grid[row][col] = num;

                    if self.backtrack() {
                        return true;
                    }

                    self.grid[row][col] = 0;  // 回溯
                }
            }
            false
        } else {
            true  // 全部填满
        }
    }

    fn find_empty(&self) -> Option<(usize, usize)> {
        for row in 0..SIZE {
            for col in 0..SIZE {
                if self.grid[row][col] == 0 {
                    return Some((row, col));
                }
            }
        }
        None
    }

    fn is_valid(&self, row: usize, col: usize, num: u8) -> bool {
        // 检查行
        for c in 0..SIZE {
            if self.grid[row][c] == num {
                return false;
            }
        }

        // 检查列
        for r in 0..SIZE {
            if self.grid[r][col] == num {
                return false;
            }
        }

        // 检查3x3方格
        let box_row = (row / BOX_SIZE) * BOX_SIZE;
        let box_col = (col / BOX_SIZE) * BOX_SIZE;

        for r in box_row..box_row + BOX_SIZE {
            for c in box_col..box_col + BOX_SIZE {
                if self.grid[r][c] == num {
                    return false;
                }
            }
        }

        true
    }

    /// MRV启发式：选择可选值最少的变量
    pub fn solve_mrv(&mut self) -> bool {
        if let Some((row, col, _)) = self.find_mrv() {
            let candidates = self.get_candidates(row, col);

            for num in candidates {
                self.grid[row][col] = num;

                if self.solve_mrv() {
                    return true;
                }

                self.grid[row][col] = 0;
            }
            false
        } else {
            true
        }
    }

    fn find_mrv(&self) -> Option<(usize, usize, usize)> {
        let mut min_remaining = 10;
        let mut best = None;

        for row in 0..SIZE {
            for col in 0..SIZE {
                if self.grid[row][col] == 0 {
                    let count = self.get_candidates(row, col).len();
                    if count < min_remaining {
                        min_remaining = count;
                        best = Some((row, col, count));
                    }
                }
            }
        }

        best
    }

    fn get_candidates(&self, row: usize, col: usize) -> Vec<u8> {
        let mut used = [false; 10];

        // 标记已使用的数字
        for c in 0..SIZE {
            used[self.grid[row][c] as usize] = true;
        }
        for r in 0..SIZE {
            used[self.grid[r][col] as usize] = true;
        }

        let box_row = (row / BOX_SIZE) * BOX_SIZE;
        let box_col = (col / BOX_SIZE) * BOX_SIZE;
        for r in box_row..box_row + BOX_SIZE {
            for c in box_col..box_col + BOX_SIZE {
                used[self.grid[r][c] as usize] = true;
            }
        }

        (1..=9).filter(|&n| !used[n as usize]).collect()
    }
}
```

### 复杂度分析

| 方法 | 平均时间 | 最坏时间 |
|-----|---------|---------|
| 朴素回溯 | $O(9^{81})$ | $O(9^{81})$ |
| MRV | $O(1)$ | $O(9^{81})$ |
| Dancing Links | $O(1)$ | 多项式 |

## 性能评估

| 难度 | 朴素回溯 | MRV | Dancing Links |

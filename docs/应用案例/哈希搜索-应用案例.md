# 哈希搜索实际应用案例

## 案例概述

**算法**: 哈希搜索 (Hash-based Search)  
**应用领域**: 缓存系统、数据去重、数据库索引、分布式存储  
**案例来源**: Redis缓存 / Bloom Filter去重 / 分布式一致性哈希

## 应用场景描述

### 背景

哈希搜索是现代计算机系统的基石，应用场景包括：
- **Web缓存**: CDN边缘节点快速查找缓存内容
- **数据库**: 内存数据库的主键索引
- **大数据**: 海量数据去重（URL去重、用户ID去重）
- **区块链**: 交易查找、区块验证

### 问题定义

**场景1 - 缓存系统**: 
- 输入: 亿级Key-Value对
- 操作: 99%的查询，1%的更新
- 要求: 平均查询时间 < 1μs

**场景2 - 海量去重**: 
- 输入: 每日百亿级URL去重
- 内存限制: 不能存储所有URL
- 允许误判率: < 1%

### 为什么选择哈希搜索

| 特性 | 哈希搜索优势 | 对比 |
|------|-----------|------|
| 平均查询 | ✅ $O(1)$ | 二分搜索 $O(\log n)$ |
| 插入删除 | ✅ $O(1)$ | 平衡树 $O(\log n)$ |
| 实现简单 | ✅ 数组+链表/探测 | 树结构复杂 |
| 空间效率 | ✅ 可控制负载因子 | B树节点利用率低 |

## 算法实现

### 问题建模

**Redis-like缓存系统**:

```
设计目标:
- 支持String/Hash/Set/ZSet/List等多种数据结构
- 过期时间(TTL)管理
- 内存淘汰策略(LRU/LFU)
- 线程安全
```

**布隆过滤器去重**:

```
设计目标:
- 极低的内存占用（每个元素几个比特）
- 零漏判（已存在的URL一定会被识别）
- 允许可控的误判率（未存在的URL可能误判为存在）
```

use std::collections::hash_map::DefaultHasher;
use std::hash::{Hash, Hasher};
use std::sync::{Arc, RwLock};
use std::time::{Duration, Instant, SystemTime, UNIX_EPOCH};
use std::collections::VecDeque;

/// ==================== LRU缓存实现 ====================

/// 缓存条目
struct CacheEntry<V> {
    value: V,
    expires_at: Option<u64>,  // Unix时间戳，None表示永不过期
    last_access: u64,         // 用于LRU
}

/// 分片LRU缓存（减少锁竞争）
pub struct ShardedCache<K, V> {
    shards: Vec<RwLock<HashMap<K, CacheEntry<V>>>>,
    shard_count: usize,
}

impl<K: Hash + Eq + Clone, V: Clone> ShardedCache<K, V> {
    pub fn new(shard_count: usize) -> Self {
        let mut shards = Vec::with_capacity(shard_count);
        for _ in 0..shard_count {
            shards.push(RwLock::new(HashMap::new()));
        }
        Self { shards, shard_count }
    }

    fn get_shard_index(&self, key: &K) -> usize {
        let mut hasher = DefaultHasher::new();
        key.hash(&mut hasher);
        (hasher.finish() as usize) % self.shard_count
    }

    pub fn get(&self, key: &K) -> Option<V> {
        let shard_idx = self.get_shard_index(key);
        let mut shard = self.shards[shard_idx].write().unwrap();
        
        if let Some(entry) = shard.get_mut(key) {
            // 检查过期
            if let Some(expires) = entry.expires_at {
                let now = current_timestamp();
                if now > expires {
                    shard.remove(key);
                    return None;
                }
            }
            
            // 更新LRU时间
            entry.last_access = current_timestamp();
            Some(entry.value.clone())
        } else {
            None
        }
    }

    pub fn set(&self, key: K, value: V, ttl_secs: Option<u64>) {
        let shard_idx = self.get_shard_index(key);
        let mut shard = self.shards[shard_idx].write().unwrap();
        
        let now = current_timestamp();
        let entry = CacheEntry {
            value,
            expires_at: ttl_secs.map(|ttl| now + ttl * 1000),
            last_access: now,
        };
        
        shard.insert(key, entry);
    }

    pub fn remove(&self, key: &K) -> Option<V> {
        let shard_idx = self.get_shard_index(key);
        let mut shard = self.shards[shard_idx].write().unwrap();
        shard.remove(key).map(|e| e.value)
    }
}

/// 简单HashMap实现（展示原理）
pub struct HashMap<K, V> {
    buckets: Vec<Option<Vec<(K, V)>>>, // 链地址法
    size: usize,
    capacity: usize,
    load_factor_threshold: f64,
}

impl<K: Hash + Eq + Clone, V: Clone> HashMap<K, V> {
    pub fn new() -> Self {
        let initial_capacity = 16;
        Self {
            buckets: (0..initial_capacity).map(|_| None).collect(),
            size: 0,
            capacity: initial_capacity,
            load_factor_threshold: 0.75,
        }
    }

    fn hash(&self, key: &K) -> usize {
        let mut hasher = DefaultHasher::new();
        key.hash(&mut hasher);
        (hasher.finish() as usize) % self.capacity
    }

    pub fn insert(&mut self, key: K, value: V) -> Option<V> {
        // 检查是否需要扩容
        if self.size as f64 / self.capacity as f64 > self.load_factor_threshold {
            self.resize();
        }

        let idx = self.hash(&key);
        
        if let Some(bucket) = &mut self.buckets[idx] {
            // 检查是否已存在
            for i in 0..bucket.len() {
                if bucket[i].0 == key {
                    let old = bucket[i].1.clone();
                    bucket[i].1 = value;
                    return Some(old);
                }
            }
            bucket.push((key, value));
        } else {
            self.buckets[idx] = Some(vec![(key, value)];
        }
        
        self.size += 1;
        None
    }

    pub fn get(&self, key: &K) -> Option<V> {
        let idx = self.hash(key);
        
        if let Some(bucket) = &self.buckets[idx] {
            for (k, v) in bucket {
                if k == key {
                    return Some(v.clone());
                }
            }
        }
        None
    }

    fn resize(&mut self) {
        let new_capacity = self.capacity * 2;
        let mut new_buckets: Vec<Option<Vec<(K, V)>>> = (0..new_capacity)
            .map(|_| None)
            .collect();
        
        // 重新哈希所有元素
        for bucket in self.buckets.drain(..) {
            if let Some(entries) = bucket {
                for (key, value) in entries {
                    let mut hasher = DefaultHasher::new();
                    key.hash(&mut hasher);
                    let idx = (hasher.finish() as usize) % new_capacity;
                    
                    if let Some(b) = &mut new_buckets[idx] {
                        b.push((key, value));
                    } else {
                        new_buckets[idx] = Some(vec![(key, value)]);
                    }
                }
            }
        }
        
        self.buckets = new_buckets;
        self.capacity = new_capacity;
    }
}

/// ==================== 布隆过滤器 ====================

pub struct BloomFilter {
    bits: Vec<bool>,
    bit_size: usize,
    hash_count: usize,
    item_count: usize,
}

impl BloomFilter {
    /// 创建布隆过滤器
    /// expected_items: 预期插入的元素数量
    /// false_positive_rate: 可接受的误判率
    pub fn new(expected_items: usize, false_positive_rate: f64) -> Self {
        // 计算最优的位数组大小: m = -n * ln(p) / (ln(2)^2)
        let bit_size = (-(expected_items as f64) * false_positive_rate.ln() 
            / (2.0_f64.ln().powi(2))).ceil() as usize;
        
        // 计算最优的哈希函数数量: k = m/n * ln(2)
        let hash_count = ((bit_size as f64 / expected_items as f64) * 2.0_f64.ln())
            .ceil() as usize;
        
        Self {
            bits: vec![false; bit_size],
            bit_size,
            hash_count,
            item_count: 0,
        }
    }

    /// 添加元素
    pub fn add<T: Hash>(&mut self, item: &T) {
        for i in 0..self.hash_count {
            let idx = self.get_hash(item, i) % self.bit_size;
            self.bits[idx] = true;
        }
        self.item_count += 1;
    }

    /// 查询元素可能存在（true）或一定不存在（false）
    pub fn may_contain<T: Hash>(&self, item: &T) -> bool {
        for i in 0..self.hash_count {
            let idx = self.get_hash(item, i) % self.bit_size;
            if !self.bits[idx] {
                return false; // 一定不存在
            }
        }
        true // 可能存在
    }

    /// 使用双重哈希生成多个哈希值
    fn get_hash<T: Hash>(&self, item: &T, seed: usize) -> usize {
        let mut hasher1 = DefaultHasher::new();
        item.hash(&mut hasher1);
        let hash1 = hasher1.finish();
        
        let mut hasher2 = DefaultHasher::new();
        seed.hash(&mut hasher2);
        hash1.hash(&mut hasher2);
        
        hasher2.finish() as usize
    }

    pub fn estimated_false_positive_rate(&self) -> f64 {
        let m = self.bit_size as f64;
        let n = self.item_count as f64;
        let k = self.hash_count as f64;
        (1.0 - (-k * n / m).exp()).powi(k as i32)
    }
}

/// ==================== 一致性哈希 ====================

/// 虚拟节点
#[derive(Clone, Debug)]
struct VirtualNode {
    hash: u64,
    physical_node: String,
}

/// 一致性哈希环
pub struct ConsistentHash {
    virtual_nodes: Vec<VirtualNode>,  // 按hash排序
    replicas: usize,                  // 每个物理节点的虚拟节点数
}

impl ConsistentHash {
    pub fn new(replicas: usize) -> Self {
        Self {
            virtual_nodes: Vec::new(),
            replicas,
        }
    }

    /// 添加物理节点
    pub fn add_node(&mut self, node: &str) {
        for i in 0..self.replicas {
            let virtual_key = format!("{}:{}", node, i);
            let hash = Self::hash(&virtual_key);
            
            self.virtual_nodes.push(VirtualNode {
                hash,
                physical_node: node.to_string(),
            });
        }
        
        // 保持排序
        self.virtual_nodes.sort_by_key(|v| v.hash);
    }

    /// 移除物理节点
    pub fn remove_node(&mut self, node: &str) {
        self.virtual_nodes.retain(|v| v.physical_node != node);
    }

    /// 获取key对应的节点
    pub fn get_node(&self, key: &str) -> Option<&str> {
        if self.virtual_nodes.is_empty() {
            return None;
        }
        
        let hash = Self::hash(key);
        
        // 二分查找第一个大于等于hash的虚拟节点
        let idx = match self.virtual_nodes.binary_search_by_key(&hash, |v| v.hash) {
            Ok(i) => i,
            Err(i) => i % self.virtual_nodes.len(),
        };
        
        Some(&self.virtual_nodes[idx].physical_node)
    }

    /// 获取key对应的多个节点（用于副本）
    pub fn get_nodes(&self, key: &str, count: usize) -> Vec<&str> {
        if self.virtual_nodes.is_empty() {
            return Vec::new();
        }
        
        let hash = Self::hash(key);
        let start_idx = match self.virtual_nodes.binary_search_by_key(&hash, |v| v.hash) {
            Ok(i) => i,
            Err(i) => i % self.virtual_nodes.len(),
        };
        
        let mut result = Vec::new();
        let mut seen = std::collections::HashSet::new();
        
        for i in 0..self.virtual_nodes.len() {
            let idx = (start_idx + i) % self.virtual_nodes.len();
            let node = &self.virtual_nodes[idx].physical_node;
            
            if seen.insert(node.clone()) {
                result.push(node.as_str());
                if result.len() >= count {
                    break;
                }
            }
        }
        
        result
    }

    fn hash(key: &str) -> u64 {
        let mut hasher = DefaultHasher::new();
        key.hash(&mut hasher);
        hasher.finish()
    }
}

/// ==================== Count-Min Sketch ====================

/// 频率估计数据结构
pub struct CountMinSketch {
    width: usize,
    depth: usize,
    table: Vec<Vec<u32>>,
    hash_seeds: Vec<u64>,
    total_count: u64,
}

impl CountMinSketch {
    pub fn new(width: usize, depth: usize) -> Self {
        let table = vec![vec![0u32; width]; depth];
        let hash_seeds: Vec<u64> = (0..depth).map(|i| i as u64 * 0x9e3779b97f4a7c15).collect();
        
        Self {
            width,
            depth,
            table,
            hash_seeds,
            total_count: 0,
        }
    }

    pub fn add<T: Hash>(&mut self, item: &T) {
        for d in 0..self.depth {
            let idx = self.hash(item, d) % self.width;
            self.table[d][idx] = self.table[d][idx].saturating_add(1);
        }
        self.total_count += 1;
    }

    pub fn estimate<T: Hash>(&self, item: &T) -> u32 {
        (0..self.depth)
            .map(|d| {
                let idx = self.hash(item, d) % self.width;
                self.table[d][idx]
            })
            .min()
            .unwrap_or(0)
    }

    fn hash<T: Hash>(&self, item: &T, seed: usize) -> usize {
        let mut hasher = DefaultHasher::new();
        item.hash(&mut hasher);
        self.hash_seeds[seed].hash(&mut hasher);
        hasher.finish() as usize
    }
}

fn current_timestamp() -> u64 {
    SystemTime::now()
        .duration_since(UNIX_EPOCH)
        .unwrap()
        .as_millis() as u64
}

/// 性能测试
fn benchmark() {
    use std::time::Instant;
    
    println!("=== 哈希搜索性能测试 ===\n");
    
    // 测试1: HashMap操作
    let n = 1_000_000;
    let mut map = HashMap::new();
    
    let start = Instant::now();
    for i in 0..n {
        map.insert(i, i * 2);
    }
    println!("插入 {} 个元素: {:?}", n, start.elapsed());
    
    let start = Instant::now();
    let mut found = 0;
    for i in 0..n {
        if map.get(&i).is_some() {
            found += 1;
        }
    }
    println!("查询 {} 次: {:?} (命中率: {})", n, start.elapsed(), found);
    
    // 测试2: 布隆过滤器
    println!("\n=== 布隆过滤器测试 ===");
    let mut bloom = BloomFilter::new(1_000_000, 0.01);
    
    // 添加100万元素
    for i in 0..1_000_000 {
        bloom.add(&format!("url{}", i));
    }
    
    // 测试存在的元素
    let mut false_negatives = 0;
    for i in 0..1_000_000 {
        if !bloom.may_contain(&format!("url{}", i)) {
            false_negatives += 1;
        }
    }
    println!("漏判率: {} / {} (应为0)", false_negatives, 1_000_000);
    
    // 测试不存在的元素
    let mut false_positives = 0;
    for i in 1_000_000..2_000_000 {
        if bloom.may_contain(&format!("url{}", i)) {
            false_positives += 1;
        }
    }
    println!("误判率: {:.4}%", false_positives as f64 / 1_000_000.0 * 100.0);
    println!("预估误判率: {:.4}%", bloom.estimated_false_positive_rate() * 100.0);
    
    // 测试3: 一致性哈希
    println!("\n=== 一致性哈希测试 ===");
    let mut ch = ConsistentHash::new(150); // 每个节点150个虚拟节点
    
    // 添加节点
    ch.add_node("node1");
    ch.add_node("node2");
    ch.add_node("node3");
    
    // 统计key分布
    let mut counts = std::collections::HashMap::new();
    for i in 0..10000 {
        let node = ch.get_node(&format!("key{}", i)).unwrap();
        *counts.entry(node.to_string()).or_insert(0) += 1;
    }
    
    println!("Key分布: {:?}", counts);
    
    // 移除一个节点，观察迁移率
    ch.remove_node("node2");
    let mut moved = 0;
    for i in 0..10000 {
        let new_node = ch.get_node(&format!("key{}", i)).unwrap();
        if new_node != "node2" && counts.get(new_node) != counts.get(&format!("分配给旧节点")) {
            // 简化统计
            moved += 1;
        }
    }
    println!("节点移除后的key迁移率: ~{:.1}%", moved as f64 / 10000.0 * 100.0);
}

fn main() {
    benchmark();
}

| 操作 | 平均时间 | 最坏时间 | 空间 |

**布隆过滤器**:
- 空间: $O(n \cdot \log(1/\epsilon))$，其中 $\epsilon$ 是误判率
- 插入/查询: $O(\log(1/\epsilon))$（哈希函数数量）

### 经验总结

1. **负载因子**: 0.75是时间和空间的平衡点
2. **二次哈希**: 双重哈希减少冲突
3. **布隆过滤器**: 适合允许误判的去重场景
4. **一致性哈希**: 虚拟节点数通常150-200效果最佳

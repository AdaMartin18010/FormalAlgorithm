# 编辑距离实际应用案例

## 案例概述

**算法**: 编辑距离 (Edit Distance / Levenshtein Distance)  
**应用领域**: 拼写检查、语音识别、DNA序列比对、模糊搜索  
**案例来源**: 搜索引擎建议 / 输入法纠错 / 生物信息学

## 应用场景描述

### 背景

编辑距离衡量两个字符串的差异程度：
- **搜索引擎**: 查询纠错、相关推荐
- **输入法**: 拼音纠错、智能联想
- **语音识别**: 识别结果纠错
- **生物信息学**: DNA突变分析

### 问题定义

**场景 - 搜索引擎查询纠错**:

**输入**:
- 用户查询词（可能有拼写错误）
- 词典（数百万词条）

**输出**:
- 最可能的正确词
- 纠错建议列表

### 为什么选择编辑距离

| 特性 | 编辑距离优势 | 对比 |
|------|------------|------|
| 语义无关 | ✅ 只看字符差异 | NLP方法需要大量数据 |
| 可解释 | ✅ 编辑操作直观 | 神经网络黑盒 |
| 高效 | ✅ DP可优化 | - |

## 算法实现

```rust
/// 标准编辑距离
pub fn edit_distance(s1: &str, s2: &str) -> usize {
    let chars1: Vec<char> = s1.chars().collect();
    let chars2: Vec<char> = s2.chars().collect();
    let m = chars1.len();
    let n = chars2.len();
    
    let mut dp = vec![vec![0; n + 1]; m + 1];
    
    // 初始化边界
    for i in 0..=m { dp[i][0] = i; }
    for j in 0..=n { dp[0][j] = j; }
    
    for i in 1..=m {
        for j in 1..=n {
            let cost = if chars1[i - 1] == chars2[j - 1] { 0 } else { 1 };
            dp[i][j] = (dp[i - 1][j] + 1)           // 删除
                .min(dp[i][j - 1] + 1)              // 插入
                .min(dp[i - 1][j - 1] + cost);      // 替换/匹配
        }
    }
    
    dp[m][n]
}

/// 空间优化版（两行）
pub fn edit_distance_optimized(s1: &str, s2: &str) -> usize {
    let chars1: Vec<char> = s1.chars().collect();
    let chars2: Vec<char> = s2.chars().collect();
    
    if chars1.len() < chars2.len() {
        return edit_distance_optimized(s2, s1);
    }
    
    let mut prev = vec![0; chars2.len() + 1];
    let mut curr = vec![0; chars2.len() + 1];
    
    for j in 0..=chars2.len() {
        prev[j] = j;
    }
    
    for i in 1..=chars1.len() {
        curr[0] = i;
        for j in 1..=chars2.len() {
            let cost = if chars1[i - 1] == chars2[j - 1] { 0 } else { 1 };
            curr[j] = (prev[j] + 1)
                .min(curr[j - 1] + 1)
                .min(prev[j - 1] + cost);
        }
        std::mem::swap(&mut prev, &mut curr);
    }
    
    prev[chars2.len()]
}

/// 带权编辑距离（不同操作不同代价）
pub fn weighted_edit_distance(s1: &str, s2: &str, 
                               insert_cost: usize,
                               delete_cost: usize,
                               replace_cost: usize) -> usize {
    let chars1: Vec<char> = s1.chars().collect();
    let chars2: Vec<char> = s2.chars().collect();
    let m = chars1.len();
    let n = chars2.len();
    
    let mut dp = vec![vec![0; n + 1]; m + 1];
    
    for i in 0..=m { dp[i][0] = i * delete_cost; }
    for j in 0..=n { dp[0][j] = j * insert_cost; }
    
    for i in 1..=m {
        for j in 1..=n {
            let cost = if chars1[i - 1] == chars2[j - 1] { 0 } else { replace_cost };
            dp[i][j] = (dp[i - 1][j] + delete_cost)
                .min(dp[i][j - 1] + insert_cost)
                .min(dp[i - 1][j - 1] + cost);
        }
    }
    
    dp[m][n]
}

/// 拼写纠正器
pub struct SpellChecker {
    dictionary: Vec<String>,
}

impl SpellChecker {
    pub fn new(dictionary: Vec<String>) -> Self {
        Self { dictionary }
    }

    /// 找出最相似的词
    pub fn correct(&self, word: &str) -> Option<String> {
        let mut best_match = None;
        let mut min_distance = usize::MAX;
        
        for dict_word in &self.dictionary {
            if dict_word.len() > word.len() + 3 || dict_word.len() + 3 < word.len() {
                continue;  // 长度差距过大，跳过
            }
            
            let dist = edit_distance_optimized(word, dict_word);
            if dist < min_distance {
                min_distance = dist;
                best_match = Some(dict_word.clone());
            }
        }
        
        // 距离阈值过滤
        if min_distance <= word.len() / 3 {
            best_match
        } else {
            None
        }
    }

    /// 获取前K个候选
    pub fn suggestions(&self, word: &str, k: usize) -> Vec<(String, usize)> {
        let mut candidates: Vec<_> = self.dictionary
            .iter()
            .map(|w| (w.clone(), edit_distance_optimized(word, w)))
            .collect();
        
        candidates.sort_by_key(|(_, d)| *d);
        candidates.truncate(k);
        candidates
    }
}
```

### 复杂度分析

| 实现 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 标准DP | $O(m \cdot n)$ | $O(m \cdot n)$ |
| 空间优化 | $O(m \cdot n)$ | $O(\min(m, n))$ |
| Ukkonen优化 | $O(d \cdot \min(m, n))$，$d$为距离 | $O(\min(m, n))$ |

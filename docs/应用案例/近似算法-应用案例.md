## 案例概述

**算法**: 近似算法 (Approximation Algorithms)
**应用领域**: NP难问题的实用求解、旅行商问题、集合覆盖、最大割
**案例来源**: 物流配送 / 网络设计 / 设施选址

## 应用场景描述

### 背景

许多实际问题都是NP难的，精确算法无法在合理时间内求解：

- **物流**: 车辆路径规划(VRP)
- **网络**: 最小代价连接
- **生产**: 作业车间调度
- **无线**: 传感器网络覆盖

### 问题定义

**场景 - 物流配送路径优化**:

**输入**:

- 配送中心位置
- 客户地址和需求
- 车辆容量限制

**输出**:

- 近似最优的配送路线

### 为什么选择近似算法

| 特性 | 近似算法优势 | 对比 |
|------|------------|------|
| 可保证 | ✅ 近似比保证 | 启发式无保证 |
| 效率 | ✅ 多项式时间 | 精确算法指数级 |
| 实用 | ✅ 实际效果通常很好 | - |

## 算法实现

```rust
use std::f64;

#[derive(Clone, Copy, Debug)]
pub struct Point {
    pub x: f64,
    pub y: f64,
}

impl Point {
    fn distance(&self, other: &Point) -> f64 {
        ((self.x - other.x).powi(2) + (self.y - other.y).powi(2)).sqrt()
    }
}

/// 旅行商问题(TSP) - 最近邻近似
pub fn tsp_nearest_neighbor(points: &[Point]) -> (Vec<usize>, f64) {
    let n = points.len();
    if n == 0 {
        return (vec![], 0.0);
    }

    let mut visited = vec![false; n];
    let mut path = vec![0];  // 从0开始
    visited[0] = true;
    let mut total_dist = 0.0;
    let mut current = 0;

    for _ in 1..n {
        let mut nearest = None;
        let mut min_dist = f64::INFINITY;

        for i in 0..n {
            if !visited[i] {
                let dist = points[current].distance(&points[i]);
                if dist < min_dist {
                    min_dist = dist;
                    nearest = Some(i);
                }
            }
        }

        if let Some(next) = nearest {
            path.push(next);
            visited[next] = true;
            total_dist += min_dist;
            current = next;
        }
    }

    // 回到起点
    total_dist += points[current].distance(&points[0]);
    path.push(0);

    (path, total_dist)
}

/// TSP - 2-近似算法（最小生成树+遍历）
pub fn tsp_2_approximation(points: &[Point]) -> (Vec<usize>, f64) {
    // 1. 构建完全图的最小生成树
    let mst = build_mst(points);

    // 2. MST的前序遍历给出TSP路径
    let mut path = Vec::new();
    let mut visited = vec![false; points.len()];
    preorder_traversal(&mst, 0, &mut visited, &mut path);
    path.push(0);  // 回到起点

    // 计算总距离
    let mut total = 0.0;
    for i in 0..path.len() - 1 {
        total += points[path[i]].distance(&points[path[i + 1]]);
    }

    (path, total)
}

fn build_mst(points: &[Point]) -> Vec<Vec<usize>> {
    let n = points.len();
    let mut mst = vec![Vec::new(); n];
    let mut visited = vec![false; n];
    let mut min_edge = vec![f64::INFINITY; n];
    let mut parent = vec![0; n];

    min_edge[0] = 0.0;

    for _ in 0..n {
        let mut u = 0;
        let mut min = f64::INFINITY;

        for i in 0..n {
            if !visited[i] && min_edge[i] < min {
                min = min_edge[i];
                u = i;
            }
        }

        visited[u] = true;

        if u != 0 {
            mst[parent[u]].push(u);
            mst[u].push(parent[u]);
        }

        for v in 0..n {
            let dist = points[u].distance(&points[v]);
            if !visited[v] && dist < min_edge[v] {
                min_edge[v] = dist;
                parent[v] = u;
            }
        }
    }

    mst
}

fn preorder_traversal(tree: &[Vec<usize>], node: usize,
                       visited: &mut [bool], result: &mut Vec<usize>) {
    if visited[node] {
        return;
    }
    visited[node] = true;
    result.push(node);

    for &child in &tree[node] {
        preorder_traversal(tree, child, visited, result);
    }
}

/// 集合覆盖 - 贪心算法（ln(n)近似）
pub fn set_cover_greedy(universe: &[usize], sets: &[Vec<usize>]) -> Vec<usize> {
    let mut uncovered: std::collections::HashSet<_> = universe.iter().cloned().collect();
    let mut selected = Vec::new();

    while !uncovered.is_empty() {
        // 选择覆盖最多未覆盖元素的集合
        let mut best_set = None;
        let mut max_cover = 0;

        for (i, set) in sets.iter().enumerate() {
            let cover_count = set.iter()
                .filter(|x| uncovered.contains(x))
                .count();

            if cover_count > max_cover {
                max_cover = cover_count;
                best_set = Some(i);
            }
        }

        if let Some(idx) = best_set {
            selected.push(idx);
            for x in &sets[idx] {
                uncovered.remove(x);
            }
        } else {
            break;
        }
    }

    selected
}

/// 最大割 - 局部搜索
pub fn max_cut_local_search(graph: &[Vec<usize>]) -> (Vec<bool>, usize) {
    let n = graph.len();
    let mut partition: Vec<bool> = (0..n).map(|i| i % 2 == 0).collect();
    let mut improved = true;

    while improved {
        improved = false;

        for u in 0..n {
            let gain = graph[u].iter()
                .filter(|&&v| partition[v] == partition[u])
                .count() as i32
                - graph[u].iter()
                .filter(|&&v| partition[v] != partition[u])
                .count() as i32;

            if gain > 0 {
                partition[u] = !partition[u];
                improved = true;
            }
        }
    }

    let cut_size = count_cut_edges(graph, &partition);
    (partition, cut_size)
}

fn count_cut_edges(graph: &[Vec<usize>], partition: &[bool]) -> usize {
    let mut count = 0;
    for u in 0..graph.len() {
        for &v in &graph[u] {
            if u < v && partition[u] != partition[v] {
                count += 1;
            }
        }
    }
    count
}
```

### 复杂度分析

| 问题 | 算法 | 时间复杂度 | 近似比 |
|-----|------|-----------|-------|
| TSP | 最近邻 | $O(n^2)$ | 无保证 |
| TSP | MST 2-近似 | $O(n^2)$ | 2 |
| 集合覆盖 | 贪心 | $O(m \cdot n)$ | $\ln(n)$ |
| 最大割 | 局部搜索 | $O(n^3)$ | 0.5 |

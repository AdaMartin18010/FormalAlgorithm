# 最小生成树实际应用案例

## 案例概述

**算法**: 最小生成树 (MST - Minimum Spanning Tree)  
**应用领域**: 网络设计、聚类分析、电路设计、物流规划  
**案例来源**: 通信网络建设 / 电力传输网络 / 图像分割

## 应用场景描述

### 背景

MST是连接所有节点的最小权重子图，典型应用：
- **通信网络**: 以最小成本连接所有城市
- **电力网络**: 变电站之间的最优输电线路
- **机器学习**: 层次聚类、图像分割
- **物流**: 配送中心与仓库的最优连接

### 问题定义

**场景 - 5G基站网络建设**:

**输入**:
- 城市位置坐标（100-500个城市）
- 建设成本（距离、地形、地价）
- 带宽需求

**输出**: 连接所有城市的最小成本网络

### 为什么选择MST算法

| 特性 | MST优势 | 对比 |
|------|--------|------|
| 全局最优 | ✅ 保证最优 | 贪心启发式无保证 |
| 多项式时间 | ✅ $O(E \log V)$ | TSP是NP难 |
| 连通性 | ✅ 保证连通 | - |

## 算法实现

```rust
use std::cmp::Ordering;
use std::collections::BinaryHeap;

/// 边结构
#[derive(Clone, Copy, Debug)]
struct Edge {
    u: usize,
    v: usize,
    weight: f64,
}

impl PartialEq for Edge {
    fn eq(&self, other: &Self) -> bool {
        self.weight == other.weight
    }
}

impl Eq for Edge {}

impl PartialOrd for Edge {
    fn partial_cmp(&self, other: &Self) -> Option<Ordering> {
        Some(self.cmp(other))
    }
}

impl Ord for Edge {
    fn cmp(&self, other: &Self) -> Ordering {
        self.weight.partial_cmp(&other.weight).unwrap()
    }
}

/// 并查集（Kruskal算法用）
pub struct UnionFind {
    parent: Vec<usize>,
    rank: Vec<usize>,
}

impl UnionFind {
    pub fn new(n: usize) -> Self {
        Self {
            parent: (0..n).collect(),
            rank: vec![0; n],
        }
    }

    pub fn find(&mut self, x: usize) -> usize {
        if self.parent[x] != x {
            self.parent[x] = self.find(self.parent[x]);
        }
        self.parent[x]
    }

    pub fn union(&mut self, x: usize, y: usize) -> bool {
        let px = self.find(x);
        let py = self.find(y);
        
        if px == py {
            return false;
        }
        
        if self.rank[px] < self.rank[py] {
            self.parent[px] = py;
        } else if self.rank[px] > self.rank[py] {
            self.parent[py] = px;
        } else {
            self.parent[py] = px;
            self.rank[px] += 1;
        }
        true
    }
}

/// Kruskal算法
pub fn kruskal(n: usize, edges: &mut Vec<Edge>) -> (Vec<Edge>, f64) {
    edges.sort_by(|a, b| a.weight.partial_cmp(&b.weight).unwrap());
    
    let mut uf = UnionFind::new(n);
    let mut mst = Vec::new();
    let mut total_weight = 0.0;
    
    for edge in edges {
        if uf.union(edge.u, edge.v) {
            total_weight += edge.weight;
            mst.push(*edge);
            
            if mst.len() == n - 1 {
                break;
            }
        }
    }
    
    (mst, total_weight)
}

/// Prim算法（优先队列优化）
pub fn prim(n: usize, adj: &[Vec<(usize, f64)>]) -> (Vec<Edge>, f64) {
    let mut visited = vec![false; n];
    let mut mst = Vec::new();
    let mut total_weight = 0.0;
    
    // (权重, 目标节点, 来源节点)
    let mut heap: BinaryHeap<(f64, usize, usize)> = BinaryHeap::new();
    
    // 从节点0开始
    visited[0] = true;
    for &(v, w) in &adj[0] {
        heap.push((-w, v, 0));  // 使用负数实现最小堆
    }
    
    while let Some((w, v, u)) = heap.pop() {
        if visited[v] {
            continue;
        }
        
        visited[v] = true;
        mst.push(Edge { u, v, weight: -w });
        total_weight += -w;
        
        if mst.len() == n - 1 {
            break;
        }
        
        for &(next, next_w) in &adj[v] {
            if !visited[next] {
                heap.push((-next_w, next, v));
            }
        }
    }
    
    (mst, total_weight)
}

/// 基于MST的层次聚类
pub struct HierarchicalClustering {
    points: Vec<Vec<f64>>,
}

impl HierarchicalClustering {
    /// 单链接聚类（使用MST）
    pub fn cluster(&self, k: usize) -> Vec<Vec<usize>> {
        let n = self.points.len();
        
        // 构建完全图的MST
        let mut edges = Vec::new();
        for i in 0..n {
            for j in (i + 1)..n {
                let dist = self.euclidean_distance(&self.points[i], &self.points[j]);
                edges.push(Edge { u: i, v: j, weight: dist });
            }
        }
        
        let (mut mst, _) = kruskal(n, &mut edges);
        
        // 移除k-1条最长边得到k个聚类
        mst.sort_by(|a, b| b.weight.partial_cmp(&a.weight).unwrap());
        
        // 构建聚类（简化实现）
        let mut clusters: Vec<Vec<usize>> = (0..n).map(|i| vec![i]).collect();
        
        for i in k..n {
            // 合并包含u和v的簇
            let u = mst[i - 1].u;
            let v = mst[i - 1].v;
            
            let u_idx = clusters.iter().position(|c| c.contains(&u)).unwrap();
            let v_idx = clusters.iter().position(|c| c.contains(&v)).unwrap();
            
            if u_idx != v_idx {
                let merged = clusters.remove(v_idx);
                clusters[u_idx].extend(merged);
            }
        }
        
        clusters
    }

    fn euclidean_distance(&self, a: &[f64], b: &[f64]) -> f64 {
        a.iter().zip(b.iter()).map(|(x, y)| (x - y).powi(2)).sum::<f64>().sqrt()
    }
}
```

### 复杂度分析

| 算法 | 时间复杂度 | 空间复杂度 | 适用场景 |
|------|-----------|-----------|---------|
| Kruskal | $O(E \log E)$ | $O(V)$ | 稀疏图 |
| Prim(二叉堆) | $O(E \log V)$ | $O(V)$ | 稠密图 |
| Prim(Fibonacci堆) | $O(E + V \log V)$ | $O(V)$ | 理论最优 |

## 使用指南

### 案例阅读建议

1. **按应用领域查找**: 根据你的业务场景选择相关案例
2. **按算法类型查找**: 学习特定算法的实际应用
3. **参考性能数据**: 对比不同算法在真实场景的表现
4. **复现代码**: 所有案例提供可直接运行的Rust代码

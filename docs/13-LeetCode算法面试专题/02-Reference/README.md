# 13-Reference / 参考索引

> **Diátaxis 分类**：Reference（参考）
> **核心问题**："…的事实是什么？"
> **读者状态**：查阅者（已有明确查询目标，需要快速获取事实）

## 目录定位

本目录为 `13-LeetCode算法面试专题` 模块的 **Reference（参考）** 分类入口。

本目录包含所有**技术描述和事实信息**：题解文档（含形式化规约、复杂度分析、正确性证明）、术语表、复杂度速查表、代码模板速查等。读者应能快速检索到所需信息，无需按顺序阅读。

## 写作规范

本目录下的所有文档必须：

1. **可快速检索**：使用表格、列表、代码块，减少叙述性文字。
2. **准确、完整、简洁**：不解释"为什么"，只陈述"是什么"。
3. **按主题或字母排序**：便于查找。
4. **题解必须包含**：形式化规约、复杂度分析、正确性证明梗概、多语言代码链接。
5. **Frontmatter 中标注** `diataxis: reference`。

## 目录定位

本目录为 `13-LeetCode算法面试专题` 模块的 **Tutorials（教程）** 分类入口。

与现有按算法类别组织的编号目录（`01-数据结构专题` 等）不同，本目录按**学习体验**组织内容：提供明确的起点、终点、渐进式课程和动手练习。

## 写作规范

本目录下的所有文档必须：

1. **有明确的起点和终点**：读者从什么状态出发，到什么状态结束。
2. **渐进式**：每一步建立在前一步基础上，不跳跃。
3. **包含实践**：每节配有练习或代码任务。
4. **假设读者为初学者**：不预设高级知识。
5. **Frontmatter 中标注** `diataxis: tutorial`。

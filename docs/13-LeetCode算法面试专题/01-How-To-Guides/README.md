## 写作规范

本目录下的所有文档必须：

1. **以目标为导向**：标题必须是"如何做…""如何在…""怎样…"。
2. **步骤可独立执行**：读者不需要读完 Tutorial 才能看懂。
3. **允许假设前置知识**：可以链接到 Tutorial 或 Explanation。
4. **必须包含验证步骤**：如何确认操作成功。
5. **Frontmatter 中标注** `diataxis: how-to`。

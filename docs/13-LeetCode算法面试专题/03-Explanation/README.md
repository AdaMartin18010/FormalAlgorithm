## 写作规范

本目录下的所有文档必须：

1. **讨论性、背景性**：允许历史沿革、设计权衡、哲学思考。
2. **回答"为什么"**：不只是陈述事实，而是解释原因。
3. **允许多视角**：可以对比不同方法论的优劣。
4. **不假设读者要立刻行动**：读者是在"理解"，不是在"执行"。
5. **Frontmatter 中标注** `diataxis: explanation`。

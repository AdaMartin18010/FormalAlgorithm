## 一、说明与使用

本表将国际权威大学在**计算理论、算法设计与分析、形式化方法**等方向的课程与本项目模块（00–12）建立对标关系，便于：

- 学习者按学校/课程查找与本项目章节的对应关系；
- 维护者按年度更新 syllabus 与阅读列表；
- 在 07-计算模型、09-算法理论、04-算法复杂度 等文档中引用「国际课程参考」时链接本表。

**阅读建议**：先确定学习目标（理论 / 算法 / 形式化），再按「建议阅读顺序」与本项目模块对照学习。

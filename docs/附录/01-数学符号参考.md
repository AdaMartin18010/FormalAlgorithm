## 逻辑符号

| 符号 | 名称 | 含义 | LaTeX |
|------|------|------|-------|
| $\land$ | 合取 | 逻辑与 (AND) | `\land` |
| $\lor$ | 析取 | 逻辑或 (OR) | `\lor` |
| $\neg$ | 否定 | 逻辑非 (NOT) | `\neg` |
| $\Rightarrow$ | 蕴含 | 如果...那么... | `\Rightarrow` |
| $\Leftrightarrow$ | 等价 | 当且仅当 | `\Leftrightarrow` |
| $\forall$ | 全称量词 | 对于所有 | `\forall` |
| $\exists$ | 存在量词 | 存在 | `\exists` |
| $\nexists$ | 不存在 | 不存在 | `\nexists` |
| $\vdash$ | 推导 | 可证明 | `\vdash` |
| $\models$ | 满足 | 语义蕴涵 | `\models` |
| $\top$ | 真 | 永真 | `\top` |
| $\bot$ | 假 | 永假 | `\bot` |

## 复杂度符号

| 符号 | 名称 | 含义 | LaTeX |
|------|------|------|-------|
| $O$ | 大 O | 渐进上界 | `O` |
| $\Omega$ | 大 Omega | 渐进下界 | `\Omega` |
| $\Theta$ | 大 Theta | 紧确界 | `\Theta` |
| $o$ | 小 o | 松上界 | `o` |
| $\omega$ | 小 omega | 松下界 | `\omega` |
| $\sim$ | 等价 | 渐近等价 | `\sim` |

### 复杂度示例

- $O(1)$ - 常数时间复杂度
- $O(\log n)$ - 对数时间复杂度
- $O(n)$ - 线性时间复杂度
- $O(n \log n)$ - 线性对数时间复杂度
- $O(n^2)$ - 平方时间复杂度
- $O(2^n)$ - 指数时间复杂度
- $O(n!)$ - 阶乘时间复杂度

## 常用组合

| 符号组合 | 含义 | LaTeX |
|----------|------|-------|
| $\{x \mid P(x)\}$ | 满足性质 P 的所有 x | `\{x \mid P(x)\}` |
| $\bigcup_{i=1}^{n} A_i$ | A₁ 到 Aₙ 的并集 | `\bigcup_{i=1}^{n} A_i` |
| $\bigcap_{i=1}^{n} A_i$ | A₁ 到 Aₙ 的交集 | `\bigcap_{i=1}^{n} A_i` |
| $\sum_{i=1}^{n} a_i$ | a₁ 到 aₙ 的和 | `\sum_{i=1}^{n} a_i` |
| $\prod_{i=1}^{n} a_i$ | a₁ 到 aₙ 的积 | `\prod_{i=1}^{n} a_i` |
| $\lim_{x \to a} f(x)$ | f(x) 当 x 趋近于 a 的极限 | `\lim_{x \to a} f(x)` |
| $\int_{a}^{b} f(x) dx$ | f(x) 从 a 到 b 的积分 | `\int_{a}^{b} f(x) dx` |
| $\binom{n}{k}$ | 组合数，n 选 k | `\binom{n}{k}` |
| $n!$ | n 的阶乘 | `n!` |
| $a \bmod b$ | a 模 b | `a \bmod b` |

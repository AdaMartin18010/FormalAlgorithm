## 评审内容

我们期望您从以下几个方面进行评审：

1. **技术准确性**
   - 定义是否准确、完整
   - 定理陈述是否正确
   - 证明是否逻辑严密

2. **形式化严谨性**
   - 符号使用是否规范
   - 形式化程度是否适当
   - 推理过程是否严密

3. **内容完整性**
   - 内容覆盖是否全面
   - 引用是否充分、规范
   - 是否包含最新研究成果

4. **表达清晰度**
   - 结构是否合理
   - 表达是否清晰流畅
   - 示例是否充分、易懂

## 使用说明

1. **发送前检查**：
   - [ ] 替换所有 `[方括号]` 内容
   - [ ] 确认专家姓名和称谓正确
   - [ ] 确认文档信息准确
   - [ ] 检查日期和联系方式

2. **个性化建议**：
   - 根据专家背景调整评审重点
   - 可提及专家的具体研究领域与文档的相关性
   - 如有共同熟人或推荐人，可提及

3. **发送记录**：
   - 在 [专家评审跟踪表.md](../专家评审跟踪表.md) 中记录发送信息
   - 设置提醒，5天后跟进

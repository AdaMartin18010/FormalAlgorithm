## 3. 引用格式标准化

### 3.1 文内引用格式

```markdown
% 单作者
[Smith 2023]

% 多作者
[Smith et al. 2023]

% 引用页码
[Smith 2023, p. 123]
```

### 3.2 参考文献格式

```markdown
1. **作者姓名.** "论文标题." 期刊/会议名称, 卷号(期号), 页码, 年份.
2. **作者姓名.** "论文标题." arXiv:编号, 年份.
3. **作者姓名.** 书名. 出版社, 年份.
```

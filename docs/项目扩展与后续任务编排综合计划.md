## 二、AI 与数学前沿内容扩展（网络权威 + 人类认知结构）

基于 [AI与算法数学参考](AI与算法数学参考.md) 与网络检索：

| 数学领域 | 本项目对应 | 权威资源（网络核实） | 建议任务 |

| **线性代数** | 01-02、09-01 图/优化、10-量子/ML | MML book、Boyd & Vandenberghe *Introduction to Applied Linear Algebra*、Coursera/DeepLearning.AI 专项 | 01-02「面向 ML 的数学导读」深化；链到 Boyd 与 CS2023 MSF |
| **概率与统计** | 01-07、09-01-11/12、10-ML | MML book、CMU 10-606、Coursera Mathematics for ML and Data Science（12 周） | 贝叶斯、统计推断、学习理论在 01-07 与 10-ML 中补链 |

| **微积分** | 01-02、09-03 | 多元微分、矩阵微分、链式法则 | 梯度推导、自动微分、反向传播在 01-02/09-03 中简要标注 |

**CS2023 数学要求**：与 [AI与算法数学参考](AI与算法数学参考.md) 及 01-02、01-07 年度核对。

### 2.2 大语言模型 (LLM) 算法理论（网络权威 2024–2025）

- **根本限界**（与可计算性/信息论对齐）：
  - 幻觉不可消除（博弈论/不可判定性框架，见 arXiv:2506.06382、arXiv:2511.12869）。
  - 五大约束：幻觉、上下文压缩、推理退化、检索脆弱、多模态错位。
- **自注意力与 CCMC**、**泛化界**（非平凡界、压缩性）。
- **任务**：[10-高级主题/38-大语言模型算法理论](10-高级主题/38-大语言模型算法理论.md) 保留并维护「LLM 根本限界」小节及与 02/04/08 的交叉引用；年度补充 2024–2025 预印本（如 arXiv:2511.12869）。

### 2.3 量子算法最新进展（网络 + 年度文献清单）

- **LWE**：Chen 等尝试与撤回——在 [年度文献清单-2024-2025](年度文献清单-2024-2025.md) 与 09-01-15、10-量子系列中保持「进展/撤回」标注。
- **QSP**：随机编译、ε→O(ε²)。
- **QLSynth**：强化学习电路综合，门数减少约 30–50%。
- **任务**：年度文献清单与 09-01-15、10-10/15/18 等文档每年补充并标注状态。

## 三、哲科结构扩展（本体论、认识论、应用科学、计算机科学）

### 3.1 四维框架与人类认知结构

2. 03、04、09-04 中强化「可知边界」与认识论表述。
3. 12-应用领域各文档保留「规约与模型在本领域的实例化」小节（已落地）。
4. 将哲科四维与 [项目全面梳理-2025](项目全面梳理-2025.md) 的「哲科维度」合并或交叉引用，便于按认知结构选读。

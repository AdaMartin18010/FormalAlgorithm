# 最近点对问题

## 概述

最近点对问题（Closest Pair of Points）是计算几何中的经典问题：给定平面上的 $n$ 个点，找出欧几里得距离最近的一对点。
暴力解法 $O(n^2)$ 在数据量大时不可行，分治算法将其优化至 $O(n \log n)$。

## 问题定义

**输入**：平面点集 $P = \{(x_1, y_1), (x_2, y_2), ..., (x_n, y_n)\}$
**输出**：$\min_{i < j} \sqrt{(x_i - x_j)^2 + (y_i - y_j)^2}$

## 分治算法

### 步骤

1. **预处理**：按 $x$ 坐标排序，得到 $P_x$；按 $y$ 坐标排序，得到 $P_y$
2. **分解**：用垂直线 $x = x_{mid}$ 将点集分为左右两半
3. **解决**：递归求解左右子问题的最近距离 $d_l$ 和 $d_r$
4. **合并**：设 $d = \min(d_l, d_r)$，检查跨越中线的带状区域内的点

### 关键：带状区域优化

只需检查距离中线不超过 $d$ 的带状区域（strip）中的点。
将该区域中的点按 $y$ 排序后，每个点最多只需与后续 7 个点比较（几何证明）。

### 复杂度

递归式：$T(n) = 2T(n/2) + O(n)$

由主定理：$T(n) = O(n \log n)$

## 与凸包的关系

最近点对必定是凸包上的顶点或凸包内部的近邻点。但直接求凸包并不能简化问题，分治方法更为通用。

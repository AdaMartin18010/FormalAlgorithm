# BFS与DFS搜索应用

## 概述

BFS（广度优先搜索）和 DFS（深度优先搜索）不仅是图遍历算法，更是通用的**无信息搜索策略**。它们适用于任何可以建模为状态转移的问题：路径规划、 puzzle 求解、游戏状态空间探索等。

## 搜索问题形式化

搜索问题可表示为五元组 $(S, s_0, A, T, G)$：

- $S$：状态集合
- $s_0 \in S$：初始状态
- $A$：动作集合
- $T: S \times A \rightarrow S$：转移函数
- $G \subseteq S$：目标状态集合

**搜索树**：从 $s_0$ 出发，通过重复应用 $T$ 生成的树形结构。

## BFS

### 策略

按层扩展节点，先访问离起点最近的所有节点。

### 性质

| 性质 | 结论 |
|------|------|
| 完备性 | ✅ 若分支因子有限，必能找到解 |
| 最优性 | ✅ 在代价一致时找到最短路径 |
| 时间复杂度 | $O(b^d)$ |
| 空间复杂度 | $O(b^d)$ |

其中 $b$ 为分支因子，$d$ 为最浅解深度。

### 经典应用

- **最短路径**：无权图的最短路径（$O(V+E)$）
- **Web爬虫**：按链接深度分层抓取
- **社交网络**：六度分隔理论验证
- **二分图判定**：双色染色法

## DFS

### 策略

沿一条路径深入到底，再回溯探索其他分支。

### 性质

| 性质 | 结论 |
|------|------|
| 完备性 | ❌ 无限深度空间可能陷入死循环 |
| 最优性 | ❌ 不保证找到最短解 |
| 时间复杂度 | $O(b^m)$ |
| 空间复杂度 | $O(bm)$ |

其中 $m$ 为状态空间最大深度。

### 经典应用

- **拓扑排序**：基于 DFS _finish time_ 的逆序
- **强连通分量**：Kosaraju / Tarjan 算法
- **回溯求解**：N皇后、数独、子集和
- **迷宫生成**：随机 DFS 生成完美迷宫

## 迭代加深搜索（IDS）

结合 DFS 的空间优势与 BFS 的最优性：

1. 设定深度限制 $limit = 0, 1, 2, ...$
2. 对每个 $limit$ 执行深度受限的 DFS
3. 首次找到解即返回

**时间复杂度**：$O(b^d)$（与 BFS 同级）
**空间复杂度**：$O(bd)$（与 DFS 同级）

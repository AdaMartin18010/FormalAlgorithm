# 插值搜索与三分搜索

## 插值搜索（Interpolation Search）

### 概念

插值搜索是二分搜索的改进版本，适用于**均匀分布**的有序数值数组。它通过线性插值估计目标位置，而非简单地取中点。

### 位置估计公式

$$pos = low + \left\lfloor \frac{target - A[low]}{A[high] - A[low]} \times (high - low) \right\rfloor$$

### 复杂度分析

| 场景 | 时间复杂度 |
|------|-----------|
| 均匀分布 | $O(\log \log n)$ |
| 非均匀/最坏情况 | $O(n)$ |

### 适用条件

- 数组元素为数值类型
- 数据分布近似均匀
- 随机访问代价低

## 三分搜索（Ternary Search）

### 概念

三分搜索用于在**单峰函数**（先严格递增后严格递减，或反之）上查找极值点。每次将区间分成三等份，舍弃不可能包含极值的两段之一。

### 算法描述

在区间 $[l, r]$ 上查找最大值：

1. 计算 $m_1 = l + (r - l) / 3$, $m_2 = r - (r - l) / 3$
2. 若 $f(m_1) < f(m_2)$，则极值在 $[m_1, r]$ 中，令 $l = m_1$
3. 否则极值在 $[l, m_2]$ 中，令 $r = m_2$
4. 重复直到区间足够小

### 复杂度分析

| 指标 | 值 |
|------|-----|
| 时间复杂度 | $O(\log_3 n)$ |
| 空间复杂度 | $O(1)$ |

### 应用场景

- 连续函数极值查找（数值方法）
- 离散数组的单峰极值
- 凸优化中的线搜索

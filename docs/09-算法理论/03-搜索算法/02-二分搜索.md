# 二分搜索

## 概念定义

二分搜索（Binary Search）是一种在**有序数组**中查找目标元素的高效算法。其核心思想是每次将搜索区间减半，从而在对数时间内定位目标。

**前提条件**：输入数组必须按非降序（或非升序）排列。

## 算法描述

给定有序数组 $A[0..n-1]$ 和目标值 $target$：

1. 初始化 $left = 0$, $right = n-1$
2. 当 $left \leq right$ 时：
   - $mid = \lfloor (left + right) / 2 \rfloor$
   - 若 $A[mid] = target$，返回 $mid$
   - 若 $A[mid] < target$，则 $left = mid + 1$
   - 若 $A[mid] > target$，则 $right = mid - 1$
3. 未找到，返回 $-1$

## 复杂度分析

| 指标 | 值 |
|------|-----|
| 时间复杂度 | $O(\log n)$ |
| 空间复杂度 | $O(1)$ |
| 比较次数 | 最多 $\lfloor \log_2 n \rfloor + 1$ |

## 变体形式

### 下界（Lower Bound）

查找第一个大于等于 $target$ 的位置：

```
while left < right:
    mid = floor((left + right) / 2)
    if A[mid] < target: left = mid + 1
    else: right = mid
return left
```

### 上界（Upper Bound）

查找第一个大于 $target$ 的位置：

```
while left < right:
    mid = floor((left + right) / 2)
    if A[mid] <= target: left = mid + 1
    else: right = mid
return left
```

## 应用与扩展

- **搜索旋转排序数组**：先通过二分找到旋转点，再分段二分
- **求平方根**：在 $[0, x]$ 范围内二分查找满足 $mid^2 \leq x$ 的最大 $mid$
- **最优解问题**：将判定问题转化为"是否存在满足条件的解"，通过二分搜索最优值

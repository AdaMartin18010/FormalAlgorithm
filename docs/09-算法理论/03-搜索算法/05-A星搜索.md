# A*搜索算法

## 概述

A*（A-Star）搜索是一种**有信息搜索**（Informed Search）算法，广泛应用于路径规划、游戏 AI、机器人导航等领域。它通过引入启发函数 $h(n)$，将无信息的广度搜索转化为高效的导向搜索。

## 算法核心

A* 使用评价函数选择扩展节点：

$$f(n) = g(n) + h(n)$$

其中：

- $g(n)$：从起点到节点 $n$ 的实际代价
- $h(n)$：从 $n$ 到目标的**启发式估计代价**
- $f(n)$：经过 $n$ 到达目标的估计总代价

## 启发函数的性质

### 可采纳性（Admissibility）

$h(n)$ 永不高估实际代价：

$$\forall n: h(n) \leq h^*(n)$$

其中 $h^*(n)$ 为真实最优代价。

**定理**：若 $h$ 可采纳，则 A* 找到的解为最优解。

### 一致性（Consistency / 单调性）

对于任意节点 $n$ 及其后继 $n'$：

$$h(n) \leq c(n, n') + h(n')$$

其中 $c(n, n')$ 为从 $n$ 到 $n'$ 的边代价。

一致性蕴含可采纳性，且保证 A* 无需重复扩展节点。

## 常见启发函数

| 问题 | 启发函数 | 可采纳性 |
|------|---------|---------|
| 网格路径 | 曼哈顿距离 | ✅ |
| 网格路径（对角移动）| 切比雪夫距离 | ✅ |
| 自由空间路径 | 欧几里得距离 | ✅ |
| 滑动拼图 | 错位方块数 + 曼哈顿距离 | ✅ |
| 旅行商问题 | 最小生成树（MST）| ✅ |

## 与 Dijkstra 的关系

- **Dijkstra**：$h(n) = 0$，退化为无信息搜索
- **贪心最佳优先**：仅使用 $h(n)$，忽略已走路径 $g(n)$，不保证最优
- **A***：平衡 $g(n)$ 与 $h(n)$，既高效又最优

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | $O(b^d)$，实际中因启发函数大幅剪枝而接近 $O(d)$ |
| 空间 | $O(b^d)$，需存储 Open/Closed 集合 |

## 变体

- **IDA***：迭代加深 + A* 剪枝，空间 $O(d)$
- **Weighted A***：$f(n) = g(n) + w \cdot h(n)$，$w > 1$ 时牺牲最优性换取速度

# 动态规划优化技巧

> **状态**: draft内容

本文档为算法规范知识体系的核心文档，记录动态规划优化技巧的形式化定义与理论基础。

## 学习目标

- 理解动态规划优化技巧的核心概念
- 掌握动态规划优化技巧的形式化表示

## 核心概念


动态规划是一种通过将复杂问题分解为重叠子问题来求解的算法设计范式。核心概念包括：最优子结构、重叠子问题、记忆化搜索、自底向上填表、状态转移方程、背包问题、最长公共子序列、编辑距离等经典模型。

# 核化与问题核（Kernelization）

## 概述

核化是参数化算法的核心技术之一。它将一个实例 $(I, k)$ 在多项式时间内约简为一个等价的"小"实例 $(I', k')$，其中 $|I'|$ 和 $k'$ 仅依赖于参数 $k$（而非原始输入规模）。

## 定义

**核化算法**：给定实例 $(I, k)$，在多项式时间内输出实例 $(I', k')$，使得：

1. $(I, k)$ 是 YES 实例 $\iff$ $(I', k')$ 是 YES 实例
2. $|I'| \leq g(k)$（$g$ 为某函数）
3. $k' \leq k$

若 $g(k)$ 为多项式函数，则称为**多项式核**。

## 经典核化结果

| 问题 | 参数 | 核大小 | 技术 |
|------|------|--------|------|
| Vertex Cover | $k$ | $2k$ | 环规则 + 度规约 |
| Feedback Vertex Set | $k$ | $O(k^2)$ | 分解 + 压缩 |
| $d$-Hitting Set | $k$ | $O(k^d)$ | Sunflower 引理 |
| Cluster Editing | $k$ | $O(k)$ | 边修改分析 |

## Vertex Cover 的 $2k$ 核

### 度规则

若顶点 $v$ 的度 $d(v) > k$，则 $v$ 必须在任何大小 $\leq k$ 的覆盖中。

**证明**：若 $v$ 不在覆盖中，则其所有邻居必须在覆盖中，需要 $> k$ 个顶点，矛盾。

### 环规则

若存在一个环（或更一般地，一个匹配），通过结构分析约简。

### 边界分析

应用度规则后：

- 所有顶点度 $\leq k$
- 若图有 $> k^2$ 条边，则 Vertex Cover $> k$（因为 $k$ 个顶点最多覆盖 $k^2$ 条边）
- 否则 $|V| \leq 2k$（通过更精细分析）

## Sunflower 引理

用于 $d$-Hitting Set 的核化。

**定义**：$d$-uniform 超图中的一个 sunflower 是集合族 $\{S_1, ..., S_p\}$，其中 $S_i \cap S_j = C$（核心）对所有 $i \neq j$ 成立。

**引理（Erdős-Rado）**：任何大于 $d! \cdot (p-1)^d$ 个大小为 $d$ 的集合族中，要么包含 $p$ 个 sunflower，要么包含一个大小 $\leq d! \cdot (p-1)^d$ 的 hitting set。

## 核化下界

**定理（Bodlaender et al.）**：除非某些复杂性假设（如 $NP \subseteq coNP/poly$）不成立，某些 FPT 问题**没有**多项式核。

例如：

- $k$-Path 有 FPT 算法但无已知多项式核
- $k$-Connected Vertex Cover 无多项式核

## 代码参考

核化算法通常作为参数化求解器的前置步骤。实际实现中，核化后接搜索（如有界搜索树）或整数规划。

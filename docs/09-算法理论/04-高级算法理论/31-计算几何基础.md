## 1. 凸包算法

**凸包（Convex Hull）**：给定点集 $P \subset \mathbb{R}^d$，其凸包 $\text{CH}(P)$ 是包含 $P$ 的最小凸集。在二维中，凸包是一个凸多边形。

### 1.1 Graham Scan

**Graham Scan（Graham, 1972）** 时间复杂度为 $O(n \log n)$。

**算法步骤**：

1. 选取 $y$ 坐标最小的点 $p_0$（若相同则取 $x$ 最小）
2. 将其余点按相对于 $p_0$ 的极角排序
3. 维护一个栈，依次处理每个点，若新点使栈顶三点形成**非左转**（即右转或共线），则弹出栈顶
4. 方向判断使用叉积

**方向测试（Orientation）**：对于三点 $p, q, r$，计算

$$\text{orient}(p, q, r) = (q_x - p_x)(r_y - p_y) - (q_y - p_y)(r_x - p_x)$$

- $> 0$：左转（逆时针）
- $< 0$：右转（顺时针）
- $= 0$：共线

### 1.2 Andrew's Monotone Chain

**Monotone Chain（Andrew, 1979）** 同样为 $O(n \log n)$，但避免了三角函数计算，常数更小。

**算法步骤**：

1. 将所有点按 $x$ 坐标排序（$x$ 相同则按 $y$）
2. 分别构造**下凸包**和**上凸包**：
   - 下凸包：从左到右扫描，维护左转链
   - 上凸包：从右到左扫描，维护左转链
3. 合并下凸包和上凸包（注意去除重复的端点）

```rust
fn convex_hull(points: &mut [(f64, f64)]) -> Vec<(f64, f64)> {
    if points.len() <= 1 { return points.to_vec(); }
    points.sort_by(|a, b| a.0.partial_cmp(&b.0).unwrap().then(a.1.partial_cmp(&b.1).unwrap()));

    let mut lower = Vec::new();
    for &p in points.iter() {
        while lower.len() >= 2 && cross(lower[lower.len()-2], lower[lower.len()-1], p) <= 0.0 {
            lower.pop();
        }
        lower.push(p);
    }

    let mut upper = Vec::new();
    for &p in points.iter().rev() {
        while upper.len() >= 2 && cross(upper[upper.len()-2], upper[upper.len()-1], p) <= 0.0 {
            upper.pop();
        }
        upper.push(p);
    }

    lower.pop();
    upper.pop();
    lower.extend(upper);
    lower
}

fn cross(o: (f64, f64), a: (f64, f64), b: (f64, f64)) -> f64 {
    (a.0 - o.0) * (b.1 - o.1) - (a.1 - o.1) * (b.0 - o.0)
}
```

### 1.3 Jarvis March（Gift Wrapping）

**Jarvis March** 时间复杂度为 $O(nh)$，其中 $h$ 为凸包上的点数。

**算法步骤**：

1. 找到最左下角的点 $p_0$
2. 从 $p_0$ 出发，寻找使所有其他点位于右侧（或共线）的点 $p_{i+1}$
3. 重复直到回到 $p_0$

当 $h$ 较小（$h = O(\log n)$ 或常数）时，Jarvis March 优于 $O(n \log n)$ 算法。

## 2. 线段相交

### 2.1 方向测试

判断线段 $p_1p_2$ 和 $p_3p_4$ 相交的标准条件：

1. **一般相交**：$\text{orient}(p_1, p_2, p_3)$ 和 $\text{orient}(p_1, p_2, p_4)$ 异号，且 $\text{orient}(p_3, p_4, p_1)$ 和 $\text{orient}(p_3, p_4, p_2)$ 异号
2. **边界情况**：某点恰位于另一线段上（需用边界框检查）

### 2.2 Bentley-Ottmann 扫描线算法

**Bentley-Ottmann（1979）** 算法可在 $O((n + k) \log n)$ 时间内找出 $n$ 条线段间的所有 $k$ 个交点。

**核心思想**：

- 用一条垂直扫描线从左向右移动
- 维护**事件队列（Event Queue）**：线段端点和已发现的交点，按 $x$ 坐标排序
- 维护**扫描线状态（Sweep Line Status）**：当前与扫描线相交的线段，按交点的 $y$ 坐标排序（通常用平衡二叉树）

**事件处理**：

- **左端点**：将线段插入状态结构，检查其上/下邻居是否相交
- **右端点**：从状态结构删除线段，检查被删除线段的原上/下邻居是否相交
- **交点**：交换两条相交线段在状态结构中的顺序，检查新的相邻关系

## 3. 最近点对

**问题**：给定平面上的 $n$ 个点，找出欧几里得距离最近的一对点。

### 3.1 分治算法

**Shamos & Hoey（1975）** 提出的分治算法时间复杂度为 $O(n \log n)$。

**算法步骤**：

1. **划分**：将所有点按 $x$ 坐标排序，找到中位线 $x = x_m$，将点集分为左右两半 $P_L$ 和 $P_R$
2. **递归**：分别在 $P_L$ 和 $P_R$ 中求解，得到最小距离 $\delta = \min(\delta_L, \delta_R)$
3. **合并**：考虑**带状区域（strip）** $|x - x_m| < \delta$ 内的点，按 $y$ 坐标排序。对每个点，只需与其后最多 7 个点比较距离

**关键引理**：在带状区域内，任意 $\delta \times 2\delta$ 的矩形中最多包含 6 个点。因此每个点至多需与常数个点比较。

```rust
fn closest_pair(points: &[(f64, f64)]) -> f64 {
    let mut px: Vec<_> = points.iter().enumerate().collect();
    px.sort_by(|a, b| a.1.0.partial_cmp(&b.1.0).unwrap());
    let mut py = px.clone();
    py.sort_by(|a, b| a.1.1.partial_cmp(&b.1.1).unwrap());
    closest_pair_rec(&px, &py)
}

fn closest_pair_rec(px: &[...], py: &[...]) -> f64 {
    if px.len() <= 3 { return brute_force(px); }
    let mid = px.len() / 2;
    let mid_point = px[mid].1;

    let (pl_y, pr_y) = split_by_y(py, mid_point);
    let dl = closest_pair_rec(&px[..mid], &pl_y);
    let dr = closest_pair_rec(&px[mid..], &pr_y);
    let d = dl.min(dr);

    let strip: Vec<_> = py.iter().cloned()
        .filter(|(_, p)| (p.0 - mid_point.0).abs() < d)
        .collect();

    let mut min_strip = d;
    for i in 0..strip.len() {
        for j in i+1..strip.len().min(i+8) {
            if strip[j].1.1 - strip[i].1.1 >= d { break; }
            min_strip = min_strip.min(dist(strip[i].1, strip[j].1));
        }
    }
    min_strip
}
```

## 4. 点定位与三角剖分（简要）

### 4.1 点定位

**点定位（Point Location）**：给定平面剖分（如平面图或三角剖分）和查询点 $q$，确定 $q$ 所在的区域。

- **Kirkpatrick 层次结构**：对于 $n$ 个顶点的三角剖分，可在 $O(n)$ 预处理时间和 $O(\log n)$ 查询时间内完成点定位
- **梯形图（Trapezoidal Map）**：随机增量构造，期望 $O(n \log n)$ 构建时间，$O(\log n)$ 查询时间

### 4.2 三角剖分

**Delaunay 三角剖分**：对于给定点集 $P$，Delaunay 三角剖分 maximizes 最小角，且具有空外接圆性质：任意三角形的外接圆内不包含其他点。

- **构造算法**：
  - 分治法：$O(n \log n)$
  - 随机增量法：期望 $O(n \log n)$
  - Fortune 算法（基于扫描线）：$O(n \log n)$

Delaunay 三角剖分是 Voronoi 图的对偶图。

## 5. Rust 实现框架

```rust
#[derive(Clone, Copy, Debug, PartialEq)]
struct Point {
    x: f64,
    y: f64,
}

impl Point {
    fn dist_sq(&self, other: &Point) -> f64 {
        (self.x - other.x).powi(2) + (self.y - other.y).powi(2)
    }
}

fn orientation(a: &Point, b: &Point, c: &Point) -> f64 {
    (b.x - a.x) * (c.y - a.y) - (b.y - a.y) * (c.x - a.x)
}

fn segments_intersect(p1: &Point, p2: &Point, p3: &Point, p4: &Point) -> bool {
    let d1 = orientation(p1, p2, p3);
    let d2 = orientation(p1, p2, p4);
    let d3 = orientation(p3, p4, p1);
    let d4 = orientation(p3, p4, p2);

    if ((d1 > 0.0 && d2 < 0.0) || (d1 < 0.0 && d2 > 0.0)) &&
       ((d3 > 0.0 && d4 < 0.0) || (d3 < 0.0 && d4 > 0.0)) {
        return true;
    }

    // Collinear cases omitted for brevity
    false
}
```

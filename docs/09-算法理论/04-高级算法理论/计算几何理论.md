## 1. 引言

计算几何是研究几何对象算法问题的学科，在计算机图形学、机器人学、GIS、CAD等领域有广泛应用 [de Berg 2008]。

**典型问题**：
- 凸包：给定点集，求包含所有点的最小凸多边形
- 最近点对：给定点集，求距离最近的两个点
- 线段交：判断线段是否相交，求所有交点
- 范围搜索：给定查询区域，找出其中的点

## 2. 形式化定义

**定义 2.1 (凸集)**：集合 $S \subseteq \mathbb{R}^d$ 是凸集，如果对于任意 $p, q \in S$，线段 $\overline{pq} \subseteq S$。

**定义 2.2 (凸包)**：点集 $P$ 的凸包 $\text{CH}(P)$ 是包含 $P$ 的最小凸集，即所有包含 $P$ 的凸集的交。

$$\text{CH}(P) = \bigcap\{S \supseteq P : S \text{ 是凸集}\}$$

**定义 2.3 (方向测试)**：对于三点 $p, q, r \in \mathbb{R}^2$，定义有向面积：

$$\text{orient}(p, q, r) = \begin{vmatrix} p_x & p_y & 1 \\ q_x & q_y & 1 \\ r_x & r_y & 1 \end{vmatrix}$$

- $\text{orient} > 0$：$r$ 在 $\overrightarrow{pq}$ 左侧（逆时针）
- $\text{orient} < 0$：$r$ 在 $\overrightarrow{pq}$ 右侧（顺时针）
- $\text{orient} = 0$：三点共线

## 3. 凸包算法

### 3.1 Graham扫描算法

**算法思想** [Graham 1972]：
1. 找到最低最左的点作为极点
2. 按极角排序其他点
3. 使用栈扫描，维护凸包

```rust
/// Graham扫描算法求凸包
/// 时间复杂度: O(n log n)
pub fn graham_scan(points: &[(f64, f64)]) -> Vec<(f64, f64)> {
    if points.len() <= 3 {
        return points.to_vec();
    }
    
    // 1. 找到最低最左的点
    let start = points.iter()
        .enumerate()
        .min_by(|(_, a), (_, b)| {
            a.1.partial_cmp(&b.1).unwrap()
                .then(a.0.partial_cmp(&b.0).unwrap())
        })
        .map(|(i, _)| i)
        .unwrap();
    
    let pivot = points[start];
    
    // 2. 按极角排序
    let mut sorted: Vec<_> = points.iter()
        .enumerate()
        .filter(|(i, _)| *i != start)
        .map(|(_, p)| *p)
        .collect();
    
    sorted.sort_by(|a, b| {
        let angle_a = (a.1 - pivot.1).atan2(a.0 - pivot.0);
        let angle_b = (b.1 - pivot.1).atan2(b.0 - pivot.0);
        angle_a.partial_cmp(&angle_b).unwrap()
    });
    
    // 3. 扫描构建凸包
    let mut hull = vec![pivot, sorted[0]];
    
    for i in 1..sorted.len() {
        while hull.len() >= 2 {
            let n = hull.len();
            if orient(hull[n-2], hull[n-1], sorted[i]) > 0.0 {
                break;
            }
            hull.pop();
        }
        hull.push(sorted[i]);
    }
    
    hull
}

/// 计算方向（有向面积）
fn orient(p: (f64, f64), q: (f64, f64), r: (f64, f64)) -> f64 {
    (q.0 - p.0) * (r.1 - p.1) - (q.1 - p.1) * (r.0 - p.0)
}
```

**定理 3.1 (Graham扫描正确性)**：Graham扫描算法正确计算点集的凸包。

*证明*：
1. 极角排序确保按凸包边界顺序访问点
2. 当新点导致右转时（非左转），移除栈顶点保持凸性
3. 最终栈中点按逆时针顺序构成凸包

**定理 3.2 (Graham扫描复杂度)**：Graham扫描算法时间复杂度为 $\Theta(n \log n)$。

*证明*：
- 排序：$O(n \log n)$
- 扫描：每个点最多入栈出栈一次，$O(n)$
- 总计：$O(n \log n)$

### 3.2 Jarvis步进算法（礼物包装）

**算法思想** [Jarvis 1973]：
从极左点开始，每次找使右转角度最大的点，直到回到起点。

```rust
/// Jarvis步进算法求凸包
/// 时间复杂度: O(nh)，h为凸包顶点数
pub fn jarvis_march(points: &[(f64, f64)]) -> Vec<(f64, f64)> {
    if points.len() <= 3 {
        return points.to_vec();
    }
    
    // 找到最左点
    let start = points.iter()
        .enumerate()
        .min_by(|(_, a), (_, b)| a.0.partial_cmp(&b.0).unwrap())
        .map(|(i, _)| i)
        .unwrap();
    
    let mut hull = vec![];
    let mut current = start;
    
    loop {
        hull.push(points[current]);
        let mut next = (current + 1) % points.len();
        
        // 找使orient最大的点
        for i in 0..points.len() {
            let o = orient(points[current], points[next], points[i]);
            if o > 0.0 || (o == 0.0 && dist_sq(points[current], points[i]) > 
                         dist_sq(points[current], points[next])) {
                next = i;
            }
        }
        
        current = next;
        if current == start {
            break;
        }
    }
    
    hull
}

fn dist_sq(a: (f64, f64), b: (f64, f64)) -> f64 {
    (a.0 - b.0).powi(2) + (a.1 - b.1).powi(2)
}
```

**定理 3.3 (Jarvis步进复杂度)**：Jarvis步进算法时间复杂度为 $O(nh)$，其中 $h$ 为凸包顶点数。

*证明*：外层循环 $h$ 次（凸包顶点数），内层每次扫描 $n$ 个点找下一个顶点。

### 3.3 分治凸包算法

**算法思想**：
1. 按x坐标分点集为左右两半
2. 递归计算左右凸包
3. 合并两个凸包

```rust
/// 分治凸包算法
/// 时间复杂度: O(n log n)
pub fn divide_conquer_hull(points: &[(f64, f64)]) -> Vec<(f64, f64)> {
    if points.len() <= 3 {
        return points.to_vec();
    }
    
    let mut sorted = points.to_vec();
    sorted.sort_by(|a, b| a.0.partial_cmp(&b.0).unwrap());
    
    convex_hull_dc(&sorted)
}

fn convex_hull_dc(points: &[(f64, f64)]) -> Vec<(f64, f64)> {
    if points.len() <= 3 {
        return points.to_vec();
    }
    
    let mid = points.len() / 2;
    let left = convex_hull_dc(&points[..mid]);
    let right = convex_hull_dc(&points[mid..]);
    
    merge_hulls(&left, &right)
}

fn merge_hulls(left: &[(f64, f64)], right: &[(f64, f64)]) -> Vec<(f64, f64)> {
    // 找到左右凸包的上下切线并合并
    // 实现省略...
    left.iter().chain(right.iter()).copied().collect()
}
```

## 4. 最近点对问题

**问题定义**：给定点集 $P$，找到 $p, q \in P$ 使 $\|p - q\|$ 最小。

### 4.1 分治算法

**定理 4.1 (最近点对分治算法)**：存在 $O(n \log n)$ 时间复杂度的最近点对算法。

**算法** [Bentley 1976]：

```rust
/// 最近点对分治算法
/// 时间复杂度: O(n log n)
pub fn closest_pair(points: &[(f64, f64)]) -> f64 {
    if points.len() <= 3 {
        return brute_force_closest(points);
    }
    
    let mut sorted = points.to_vec();
    sorted.sort_by(|a, b| a.0.partial_cmp(&b.0).unwrap());
    
    closest_pair_dc(&sorted)
}

fn closest_pair_dc(points: &[(f64, f64)]) -> f64 {
    let n = points.len();
    if n <= 3 {
        return brute_force_closest(points);
    }
    
    let mid = n / 2;
    let mid_x = points[mid].0;
    
    let d_left = closest_pair_dc(&points[..mid]);
    let d_right = closest_pair_dc(&points[mid..]);
    let d = d_left.min(d_right);
    
    // 检查跨越中线的点对
    let strip: Vec<_> = points.iter()
        .filter(|p| (p.0 - mid_x).abs() < d)
        .copied()
        .collect();
    
    d.min(closest_in_strip(&strip, d))
}

fn closest_in_strip(strip: &[(f64, f64)], d: f64) -> f64 {
    let mut min_d = d;
    let mut sorted_strip = strip.to_vec();
    sorted_strip.sort_by(|a, b| a.1.partial_cmp(&b.1).unwrap());
    
    for i in 0..sorted_strip.len() {
        for j in (i + 1)..sorted_strip.len().min(i + 7) {
            let dist = distance(sorted_strip[i], sorted_strip[j]);
            min_d = min_d.min(dist);
        }
    }
    
    min_d
}

fn distance(a: (f64, f64), b: (f64, f64)) -> f64 {
    ((a.0 - b.0).powi(2) + (a.1 - b.1).powi(2)).sqrt()
}
```

**证明**：关键点在于中间带最多只需要检查6个点，因此合并时间为 $O(n)$。

## 5. 复杂度分析

| 算法 | 时间复杂度 | 空间复杂度 | 备注 |
|------|-----------|-----------|------|
| Graham扫描 | $O(n \log n)$ | $O(n)$ | 通用方法 |
| Jarvis步进 | $O(nh)$ | $O(1)$ | $h$小时最优 |
| 分治凸包 | $O(n \log n)$ | $O(n)$ | 可并行化 |
| 最近点对 | $O(n \log n)$ | $O(n)$ | 最优下界 |

## 1. 引言

计算几何是研究几何问题的算法设计与分析的学科，在计算机图形学、机器人学、GIS、CAD等领域有广泛应用。根据[de Berg 2008]的经典教材，计算几何的核心是设计高效算法处理几何对象。根据[Preparata 1985]的开创性工作，计算几何已成为算法理论的重要分支。根据[O'Rourke 1998]的研究，凸包、最近点对、范围搜索是计算几何的三大基础问题。

计算几何问题具有以下特点：
- **离散性与连续性的结合**：处理离散点集但涉及连续几何空间
- **退化情况的处理**：需要考虑共线点、重合点等特殊情况
- **数值稳定性**：浮点运算可能引入误差，需要鲁棒的几何计算

## 2. 形式化定义

### 2.1 几何原语

**定义 2.1**（二维点）
二维点 $p$ 是实数对 $(x, y) \in \mathbb{R}^2$。

**定义 2.2**（向量）
从点 $p_1 = (x_1, y_1)$ 到点 $p_2 = (x_2, y_2)$ 的向量为：
$$\vec{v} = p_2 - p_1 = (x_2 - x_1, y_2 - y_1)$$

**定义 2.3**（点积）
向量 $\vec{u} = (u_x, u_y)$ 和 $\vec{v} = (v_x, v_y)$ 的点积为：
$$\vec{u} \cdot \vec{v} = u_x v_x + u_y v_y = |\vec{u}| |\vec{v}| \cos \theta$$

**定义 2.4**（叉积）
向量 $\vec{u}$ 和 $\vec{v}$ 的二维叉积（标量）为：
$$\vec{u} \times \vec{v} = u_x v_y - u_y v_x$$

**定理 2.1**（叉积的几何意义）
叉积 $\vec{u} \times \vec{v}$ 的绝对值等于以 $\vec{u}$ 和 $\vec{v}$ 为邻边的平行四边形面积，符号表示方向：
- 正值：$\vec{v}$ 在 $\vec{u}$ 的逆时针方向（左转）
- 负值：$\vec{v}$ 在 $\vec{u}$ 的顺时针方向（右转）
- 零值：两向量共线

### 2.2 方向测试

**定义 2.5**（三点的方向）
给定三点 $p_0, p_1, p_2$，其方向由叉积决定：
$$\text{orient}(p_0, p_1, p_2) = (p_1 - p_0) \times (p_2 - p_0)$$

$$\text{方向} = \begin{cases}
> 0 & p_0 \to p_1 \to p_2 \text{ 左转（逆时针）} \\
< 0 & p_0 \to p_1 \to p_2 \text{ 右转（顺时针）} \\
= 0 & \text{三点共线}
\end{cases}$$

### 2.3 凸包

**定义 2.6**（凸集）
集合 $S \subseteq \mathbb{R}^2$ 是凸集，当且仅当对任意 $p, q \in S$，线段 $\overline{pq} \subseteq S$。

**定义 2.7**（凸包）
点集 $P$ 的凸包 $CH(P)$ 是包含 $P$ 的最小凸集：
$$CH(P) = \left\{ \sum_{i=1}^{n} \lambda_i p_i \mid p_i \in P, \lambda_i \geq 0, \sum_{i=1}^{n} \lambda_i = 1 \right\}$$

**定理 2.2**（凸包的基本性质）
1. 唯一性：凸包是唯一的
2. 极点：凸包边界上的点是极点（extreme points）
3. 复杂度：$n$ 个点的凸包最多有 $n$ 个顶点

## 3. 核心算法与理论

### 3.1 Graham扫描算法

**算法 3.1**（Graham扫描）
输入：点集 $P = \{p_1, p_2, ..., p_n\}$
输出：凸包顶点序列

```
1. 找到最低点 p0（y坐标最小，若相同则x最小）
2. 按极角对剩余点排序，得到序列 p1, p2, ..., pn-1
3. 初始化栈 S = [p0, p1, p2]
4. for i = 3 to n-1:
5.     while S中最后三个点形成右转:
6.         弹出栈顶
7.     将 pi 压入栈
8. return S
```

**定理 3.1**（Graham扫描的正确性）
Graham扫描算法正确计算点集的凸包。

**证明**：
1. 排序确保按逆时针方向处理点
2. 栈维护当前已处理点的凸包
3. 右转检测确保移除凹陷点
4. 最终栈中点形成凸多边形

**定理 3.2**（Graham扫描复杂度）
时间复杂度：$O(n \log n)$（主导步骤是排序）
空间复杂度：$O(n)$

### 3.2 Jarvis步进算法

**算法 3.2**（Jarvis March / Gift Wrapping）
输入：点集 $P$
输出：凸包顶点序列

```
1. 找到最低点 p0
2. current = p0
3. repeat:
4.     next = 在P中找到使(current, next, q)左转的点q
5.     将next加入结果
6.     current = next
7. until current == p0
```

**定理 3.3**（Jarvis步进复杂度）
设 $h$ 为凸包顶点数，时间复杂度：$O(nh)$
- 最优情况（$h = O(1)$）：$O(n)$
- 最坏情况（$h = n$）：$O(n^2)$

### 3.3 分治凸包算法

**算法 3.3**（分治凸包）
输入：按x坐标排序的点集 $P$
输出：凸包顶点序列

```
ConvexHull(P):
1. if |P| <= 3:
2.     return 直接计算凸包
3. 将P分为左右两半 PL, PR
4. HL = ConvexHull(PL)
5. HR = ConvexHull(PR)
6. return Merge(HL, HR)

Merge(HL, HR):
1. 找到HL的右最点和HR的左最点
2. 找到上切线和下切线
3. 合并切线间的边
```

**定理 3.4**（分治凸包复杂度）
时间复杂度：$O(n \log n)$
空间复杂度：$O(n)$

### 3.4 最近点对算法

**算法 3.4**（最近点对 - 分治）
输入：点集 $P$，按x和y坐标排序
输出：最近点对距离

```
ClosestPair(Px, Py):
1. if |P| <= 3:
2.     return 暴力计算最近距离
3. 按x坐标中点将P分为PL, PR
4. PyL = PL中点按y排序
5. PyR = PR中点按y排序
6. dL = ClosestPair(PLx, PyL)
7. dR = ClosestPair(PRx, PyR)
8. d = min(dL, dR)
9. 构造条带：距中点x距离不超过d的点
10. 按y扫描条带中的点，检查与接下来7个点的距离
11. return min(d, 条带中最近距离)
```

**定理 3.5**（最近点对正确性）
分治算法正确找到最近点对。

**证明要点**：
1. 递归正确性：子问题解合并后仍为全局解
2. 条带性质：若两点分别属于左右子集，其x距离 $\leq d$，则y距离也受限
3. 七点定理：条带中任意点只需检查其后7个点

**定理 3.6**（最近点对复杂度）
时间复杂度：$T(n) = 2T(n/2) + O(n) = O(n \log n)$
空间复杂度：$O(n)$

### 3.5 k-d树

**定义 3.1**（k-d树）
k-d树是二叉空间分割树，递归地将k维空间沿坐标轴分割。

**算法 3.5**（k-d树构建）
```
BuildKDTree(P, depth):
1. if |P| == 0: return null
2. axis = depth % k
3. 按axis坐标排序P
4. median = P的中位数
5. node.point = median
6. node.left = BuildKDTree(左半部分, depth+1)
7. node.right = BuildKDTree(右半部分, depth+1)
8. return node
```

**定理 3.7**（k-d树复杂度）
- 构建时间：$O(n \log n)$
- 范围查询：$O(n^{1-1/k} + r)$，$r$为结果数
- 最近邻查询：平均 $O(\log n)$

### 3.6 Voronoi图与Delaunay三角剖分

**定义 3.2**（Voronoi图）
给定平面点集 $P = \{p_1, ..., p_n\}$，点 $p_i$ 的Voronoi单元为：
$$V(p_i) = \{x \in \mathbb{R}^2 \mid d(x, p_i) \leq d(x, p_j), \forall j \neq i\}$$

**定义 3.3**（Delaunay三角剖分）
Delaunay三角剖分是点集 $P$ 的三角剖分，满足空圆性质：每个三角形的外接圆内不包含其他点。

**定理 3.8**（对偶关系）
Voronoi图与Delaunay三角剖分互为对偶图。

**定理 3.9**（Delaunay三角剖分复杂度）
- 构建时间：$O(n \log n)$
- 三角形数：最多 $2n - 2 - b$，其中 $b$ 为凸包顶点数

## 4. 复杂度分析

| 算法 | 时间复杂度 | 空间复杂度 | 备注 |
|------|-----------|-----------|------|
| Graham扫描 | $O(n \log n)$ | $O(n)$ | 最坏情况最优 |
| Jarvis步进 | $O(nh)$ | $O(1)$ | 输出敏感 |
| 分治凸包 | $O(n \log n)$ | $O(n)$ | 理论最优 |
| 最近点对 | $O(n \log n)$ | $O(n)$ | 最优下界 |
| k-d树构建 | $O(n \log n)$ | $O(n)$ | - |
| k-d树查询 | $O(\sqrt{n} + r)$ | $O(\log n)$ | 2D范围搜索 |
| Voronoi图 | $O(n \log n)$ | $O(n)$ | 扫描线算法 |

**定理 4.1**（凸包下界）
在代数决策树模型下，凸包问题的下界为 $\Omega(n \log n)$。

**证明**：通过排序问题归约。给定排序问题，将元素 $x_i$ 映射为点 $(x_i, x_i^2)$，则凸包顶点即为排序结果。

## 5. 实现示例

### 5.1 Rust实现

```rust
use std::f64::consts::PI;

#[derive(Clone, Copy, Debug, PartialEq)]
struct Point {
    x: f64,
    y: f64,
}

impl Point {
    fn new(x: f64, y: f64) -> Self {
        Point { x, y }
    }

    // 计算叉积 (p1-p0) x (p2-p0)
    fn cross(p0: &Point, p1: &Point, p2: &Point) -> f64 {
        (p1.x - p0.x) * (p2.y - p0.y) - (p1.y - p0.y) * (p2.x - p0.x)
    }

    // 计算距离
    fn distance(&self, other: &Point) -> f64 {
        ((self.x - other.x).powi(2) + (self.y - other.y).powi(2)).sqrt()
    }
}

// Graham扫描算法
fn graham_scan(points: &[Point]) -> Vec<Point> {
    if points.len() <= 3 {
        return points.to_vec();
    }

    // 找到最低点
    let mut p0 = points[0];
    for &p in points.iter().skip(1) {
        if p.y < p0.y || (p.y == p0.y && p.x < p0.x) {
            p0 = p;
        }
    }

    // 按极角排序
    let mut sorted: Vec<Point> = points.iter()
        .filter(|&&p| p != p0)
        .cloned()
        .collect();
    
    sorted.sort_by(|a, b| {
        let angle_a = (a.y - p0.y).atan2(a.x - p0.x);
        let angle_b = (b.y - p0.y).atan2(b.x - p0.x);
        angle_a.partial_cmp(&angle_b).unwrap()
    });

    // Graham扫描
    let mut hull = vec![p0, sorted[0]];
    
    for i in 1..sorted.len() {
        while hull.len() >= 2 {
            let top = hull.len() - 1;
            let second = hull.len() - 2;
            if Point::cross(&hull[second], &hull[top], &sorted[i]) <= 0.0 {
                hull.pop();
            } else {
                break;
            }
        }
        hull.push(sorted[i]);
    }

    hull
}

// 最近点对算法
fn closest_pair(points: &[Point]) -> f64 {
    if points.len() <= 1 {
        return f64::INFINITY;
    }
    if points.len() <= 3 {
        let mut min_dist = f64::INFINITY;
        for i in 0..points.len() {
            for j in (i + 1)..points.len() {
                min_dist = min_dist.min(points[i].distance(&points[j]));
            }
        }
        return min_dist;
    }

    // 按x坐标排序
    let mut px: Vec<Point> = points.to_vec();
    px.sort_by(|a, b| a.x.partial_cmp(&b.x).unwrap());

    // 分治
    let mid = px.len() / 2;
    let mid_point = px[mid];
    
    let dl = closest_pair(&px[..mid]);
    let dr = closest_pair(&px[mid..]);
    let mut d = dl.min(dr);

    // 构造条带
    let strip: Vec<&Point> = px.iter()
        .filter(|p| (p.x - mid_point.x).abs() < d)
        .collect();

    // 检查条带
    for i in 0..strip.len() {
        for j in (i + 1)..strip.len().min(i + 8) {
            if (strip[j].y - strip[i].y).abs() >= d {
                break;
            }
            d = d.min(strip[i].distance(strip[j]));
        }
    }

    d
}

// k-d树实现
struct KDTreeNode {
    point: Point,
    left: Option<Box<KDTreeNode>>,
    right: Option<Box<KDTreeNode>>,
}

fn build_kdtree(points: &mut [Point], depth: usize) -> Option<Box<KDTreeNode>> {
    if points.is_empty() {
        return None;
    }

    let axis = depth % 2; // 2D情况
    
    points.sort_by(|a, b| {
        if axis == 0 {
            a.x.partial_cmp(&b.x).unwrap()
        } else {
            a.y.partial_cmp(&b.y).unwrap()
        }
    });

    let mid = points.len() / 2;
    let point = points[mid];
    
    let (left_points, right_points) = points.split_at_mut(mid);
    let right_points = &mut right_points[1..]; // 排除中点

    Some(Box::new(KDTreeNode {
        point,
        left: build_kdtree(left_points, depth + 1),
        right: build_kdtree(right_points, depth + 1),
    }))
}
```

### 5.2 Python实现

```python
import math
from typing import List, Tuple, Optional
from dataclasses import dataclass

@dataclass
class Point:
    x: float
    y: float
    
    def distance(self, other: 'Point') -> float:
        return math.sqrt((self.x - other.x)**2 + (self.y - other.y)**2)

class ComputationalGeometry:
    @staticmethod
    def cross(o: Point, a: Point, b: Point) -> float:
        """计算叉积 (a-o) x (b-o)"""
        return (a.x - o.x) * (b.y - o.y) - (a.y - o.y) * (b.x - o.x)
    
    @staticmethod
    def graham_scan(points: List[Point]) -> List[Point]:
        """Graham扫描算法"""
        if len(points) <= 3:
            return points
        
        # 找到最低点
        p0 = min(points, key=lambda p: (p.y, p.x))
        
        # 按极角排序
        def polar_angle(p: Point) -> float:
            return math.atan2(p.y - p0.y, p.x - p0.x)
        
        sorted_points = sorted([p for p in points if p != p0], key=polar_angle)
        
        # 扫描
        hull = [p0, sorted_points[0]]
        for p in sorted_points[1:]:
            while len(hull) >= 2 and ComputationalGeometry.cross(hull[-2], hull[-1], p) <= 0:
                hull.pop()
            hull.append(p)
        
        return hull
    
    @staticmethod
    def jarvis_march(points: List[Point]) -> List[Point]:
        """Jarvis步进算法（Gift Wrapping）"""
        if len(points) <= 3:
            return points
        
        # 找到最低点
        hull = [min(points, key=lambda p: (p.y, p.x))]
        
        while True:
            current = hull[-1]
            next_point = points[0]
            
            for p in points[1:]:
                cross = ComputationalGeometry.cross(current, next_point, p)
                if cross < 0 or (cross == 0 and 
                               current.distance(p) > current.distance(next_point)):
                    next_point = p
            
            if next_point == hull[0]:
                break
            hull.append(next_point)
        
        return hull
    
    @staticmethod
    def closest_pair(points: List[Point]) -> float:
        """最近点对算法"""
        def closest_pair_recursive(px: List[Point]) -> float:
            n = len(px)
            if n <= 3:
                min_dist = float('inf')
                for i in range(n):
                    for j in range(i + 1, n):
                        min_dist = min(min_dist, px[i].distance(px[j]))
                return min_dist
            
            mid = n // 2
            mid_point = px[mid]
            
            dl = closest_pair_recursive(px[:mid])
            dr = closest_pair_recursive(px[mid:])
            d = min(dl, dr)
            
            # 条带
            strip = [p for p in px if abs(p.x - mid_point.x) < d]
            strip.sort(key=lambda p: p.y)
            
            for i in range(len(strip)):
                for j in range(i + 1, min(i + 8, len(strip))):
                    if strip[j].y - strip[i].y >= d:
                        break
                    d = min(d, strip[i].distance(strip[j]))
            
            return d
        
        if len(points) <= 1:
            return float('inf')
        
        px = sorted(points, key=lambda p: p.x)
        return closest_pair_recursive(px)

# 测试
if __name__ == "__main__":
    cg = ComputationalGeometry()
    
    points = [Point(0, 0), Point(1, 0), Point(1, 1), Point(0, 1), Point(0.5, 0.5)]
    hull = cg.graham_scan(points)
    print(f"凸包顶点数: {len(hull)}")
    
    dist = cg.closest_pair(points)
    print(f"最近点对距离: {dist}")
```

## 6. 应用案例

### 6.1 碰撞检测与物理引擎

在计算机图形学和游戏开发中，计算几何用于：
- **包围盒计算**：使用凸包构建物体的轴向包围盒（AABB）和定向包围盒（OBB）
- **分离轴定理**：判断两个凸多边形是否相交

- **Delaunay三角剖分**：构建数字高程模型（DEM），进行地形分析
- **空间索引**：k-d树和R-树加速地理空间查询

### 6.3 机器人路径规划

- **单元分解**：将自由空间分解为凸单元，使用A*算法规划路径
- **构型空间**：将机器人表示为点，障碍物膨胀后使用计算几何算法规划

## 8. 总结

计算几何是连接数学理论与工程实践的桥梁。本文档涵盖了计算几何的核心内容：

1. **基础原语**：点、向量、叉积、方向测试构成了计算几何的基石
2. **凸包算法**：Graham扫描、Jarvis步进、分治算法各有优势，适用于不同场景
3. **最近点对**：分治算法的经典应用，时间复杂度达到最优
4. **空间数据结构**：k-d树支持高效的范围搜索和最近邻查询
5. **高级结构**：Voronoi图和Delaunay三角剖分在多个领域有广泛应用

计算几何的关键挑战在于处理退化情况和保证数值稳定性。在实际应用中，通常需要结合具体问题的几何特性进行算法选择和优化。

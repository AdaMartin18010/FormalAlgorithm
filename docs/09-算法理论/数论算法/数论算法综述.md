## 09-算法理论. 数论算法综述 / Number-Theoretic Algorithms Survey

本文系统梳理计算数论中的基础算法，包括最大公约数、模运算、素性测试、因数分解、离散对数与中国剩余定理。对每一项算法均给出六维总结，并链接至本项目的 Rust 实现。

---

### 1. 最大公约数（GCD）

| 维度 | 内容 |
|------|------|
| **理论基础** | 欧几里得算法基于恒等式 $\gcd(a,b)=\gcd(b,a\bmod b)$；余数序列严格递减，保证终止 [Cormen et al., 2009]。 |
| **算法设计** | 辗转相除：迭代或递归计算余数，直至余数为 $0$。 |
| **复杂度分析** | 时间 $O(\log\min(a,b))$（基于 Fibonacci 数列的最坏情况上界），空间 $O(1)$（迭代版）。 |
| **形式化验证** | 利用良序原理与整除的传递性，证明最终非零余数即为最大公约数。 |
| **应用场景** | 分数约简、循环群阶计算、密钥参数选取、数论基础工具。 |
| **扩展变体** | Stein（二进制 GCD）算法，利用移位与减法，更适合大整数硬件实现。 |

**代码实现**：[`number_theory/mod.rs`](../../../examples/algorithms/src/number_theory/mod.rs)、[`primality_test.rs`](../../../examples/algorithms/src/primality_test.rs)

### 2. 扩展欧几里得（Extended GCD）

| 维度 | 内容 |
|------|------|
| **理论基础** | Bézout 恒等式：对任意整数 $a,b$，存在 $x,y$ 使得 $ax+by=\gcd(a,b)$。 |
| **算法设计** | 在欧几里得递归过程中回溯求解系数 $(x,y)$；迭代版本可避免递归深度。 |
| **复杂度分析** | 时间 $O(\log\min(a,b))$，空间 $O(\log\min(a,b))$（递归栈）。 |
| **形式化验证** | 数学归纳法：基例 $b=0$ 显然；归纳步利用 $a\bmod b = a-\lfloor a/b\rfloor b$ 回代。 |
| **应用场景** | 模逆元计算、RSA 私钥生成、线性丢番图方程求解。 |
| **扩展变体** | 迭代扩展欧几里得，适用于大整数库中的非递归场景。 |

**代码实现**：[`discrete_log.rs`](../../../examples/algorithms/src/discrete_log.rs) 中 `extended_gcd` 函数

### 3. 模运算（Modular Arithmetic）

| 维度 | 内容 |
|------|------|
| **理论基础** | 模 $m$ 的剩余类环 $\mathbb{Z}/m\mathbb{Z}$；同余关系满足自反、对称、传递性。 |
| **算法设计** | 加、减、乘均先运算后取模；幂运算采用快速幂（平方-乘）降低指数。 |
| **复杂度分析** | 加减 $O(1)$，乘法 $O(1)$（字长假设），模幂 $O(\log e)$。 |
| **形式化验证** | 基于环的封闭性与同余的代数等价性：$(a\bmod m)\cdot(b\bmod m)\equiv ab\pmod m$。 |
| **应用场景** | 公钥密码学、哈希函数、伪随机数生成、校验和算法。 |
| **扩展变体** | Montgomery 模乘、Barrett 约减，用于大整数的高效模乘硬件实现。 |

**代码实现**：[`number_theory/mod.rs`](../../../examples/algorithms/src/number_theory/mod.rs)（`mod_pow`）、[`primality_test.rs`](../../../examples/algorithms/src/primality_test.rs)、[`discrete_log.rs`](../../../examples/algorithms/src/discrete_log.rs)

### 4. 模逆元（Modular Inverse）

| 维度 | 内容 |
|------|------|
| **理论基础** | $a$ 在模 $m$ 下可逆当且仅当 $\gcd(a,m)=1$；逆元 $a^{-1}$ 满足 $aa^{-1}\equiv 1\pmod m$。 |
| **算法设计** | 调用扩展欧几里得求 Bézout 系数 $x$，再规范化到 $[0,m-1]$。 |
| **复杂度分析** | 时间 $O(\log\min(a,m))$，空间 $O(\log\min(a,m))$。 |
| **形式化验证** | 由 Bézout 等式 $ax+my=1$ 直接推出 $x\bmod m$ 即为 $a^{-1}\bmod m$。 |
| **应用场景** | RSA 解密、模线性方程、模意义下的分数运算、中国剩余定理合并。 |
| **扩展变体** | 当模数为素数 $p$ 时，可利用费马小定理 $a^{-1}\equiv a^{p-2}\pmod p$ 通过快速幂求逆。 |

**代码实现**：[`discrete_log.rs`](../../../examples/algorithms/src/discrete_log.rs) 中 `mod_inv` 函数

### 5. 费马小定理（Fermat's Little Theorem）

| 维度 | 内容 |
|------|------|
| **理论基础** | 若 $p$ 为素数且 $a\not\equiv 0\pmod p$，则 $a^{p-1}\equiv 1\pmod p$。 |
| **算法设计** | 作为素性测试的判定条件（Fermat 测试）：随机选底 $a$，检验 $a^{n-1}\equiv 1\pmod n$。 |
| **复杂度分析** | 单次测试需一次模幂，时间 $O(\log n)$。 |
| **形式化验证** | 基于 Lagrange 定理：乘法群 $\mathbb{Z}_p^*$ 的阶为 $p-1$，故任意元素阶整除 $p-1$ [Hardy & Wright, 1979]。 |
| **应用场景** | 素性测试、模逆元计算、RSA 公钥体系理论基础。 |
| **扩展变体** | Carmichael 数可导致 Fermat 测试误判，需结合二次探测定理升级为 Miller-Rabin。 |

**代码实现**：[`primality_test.rs`](../../../examples/algorithms/src/primality_test.rs) 中 `is_probable_prime`（底层依赖费马条件）

### 6. 欧拉函数（Euler's Totient）

| 维度 | 内容 |
|------|------|
| **理论基础** | $\varphi(n)$ 计数 $[1,n]$ 中与 $n$ 互素的整数个数；若 $n=\prod p_i^{e_i}$，则 $\varphi(n)=n\prod(1-1/p_i)$。 |
| **算法设计** | 先因数分解 $n$，再按公式逐素因子乘积计算。 |
| **复杂度分析** | 时间取决于分解复杂度，试除法 $O(\sqrt n)$；空间 $O(\log n)$。 |
| **形式化验证** | 利用 $\varphi$ 的积性：若 $\gcd(a,b)=1$ 则 $\varphi(ab)=\varphi(a)\varphi(b)$，由中国剩余定理的同构推出 [Niven et al., 1991]。 |
| **应用场景** | RSA 密钥参数选择、Euler 定理 $a^{\varphi(n)}\equiv 1\pmod n$、循环子群阶估计。 |
| **扩展变体** | 线性筛可在 $O(N)$ 时间内预处理 $1..N$ 的全部欧拉函数值。 |

**代码实现**：[`primality_test.rs`](../../../examples/algorithms/src/primality_test.rs) 中 `euler_totient` 函数

### 7. Miller-Rabin 素性测试

| 维度 | 内容 |
|------|------|
| **理论基础** | 基于费马小定理与二次探测定理：若 $p$ 为素数，则 $x^2\equiv 1\pmod p$ 的解仅 $x\equiv \pm 1$。 |
| **算法设计** | 将 $n-1$ 写为 $d\cdot 2^s$；选底数 $a$，计算 $a^d\bmod n$，连续平方 $s$ 次探测。 |
| **复杂度分析** | 单轮 $O(k\log^3 n)$；$k$ 轮错误概率 $\le 4^{-k}$。对 64 位整数可使用确定性底数集 [Jaeschke, 1993]。 |
| **形式化验证** | 证明合数 $n$ 至少存在 $3/4$ 的强证见证（Strong Witness），从而随机算法的错误率可控 [Rabin, 1980]。 |
| **应用场景** | 大素数生成、密码学素性判定、因数分解前置筛选。 |
| **扩展变体** | Baillie-PSW 测试（结合 Lucas 序列），目前无已知反例，常用于实际素性判定。 |

**代码实现**：[`primality_test.rs`](../../../examples/algorithms/src/primality_test.rs) 中 `is_probable_prime` 与 `is_prime_deterministic`

### 8. Pollard's Rho 因数分解

| 维度 | 内容 |
|------|------|
| **理论基础** | Floyd 判圈算法在模 $n$ 的伪随机序列中寻碰撞；利用 $\gcd(|x-y|,n)$ 提取非平凡因子。 |
| **算法设计** | 取多项式 $f(x)=x^2+c\pmod n$，双指针迭代；期望在 $O(\sqrt p)$ 步内找到碰撞，$p$ 为最小素因子。 |
| **复杂度分析** | 期望时间 $O(n^{1/4})$（生日悖论），空间 $O(1)$ [Pollard, 1975]。 |
| **形式化验证** | 生日悖论保证在阶为 $p$ 的子群中 $O(\sqrt p)$ 次迭代出现碰撞；碰撞导出因子的概率经分析不低于常数。 |
| **应用场景** | 大整数因数分解、RSA 安全性评估、椭圆曲线密码学参数生成。 |
| **扩展变体** | Brent 判圈变体（减少 gcd 调用次数）、Pollard's $p-1$ 方法、椭圆曲线分解法（ECM）。 |

**代码实现**：[`primality_test.rs`](../../../examples/algorithms/src/primality_test.rs) 中 `pollard_rho`；[`discrete_log.rs`](../../../examples/algorithms/src/discrete_log.rs) 中 `pollard_rho_factor`

### 9. 离散对数（Discrete Logarithm）

| 维度 | 内容 |
|------|------|
| **理论基础** | 在有限乘法群 $\mathbb{Z}_p^*$ 中求解 $g^x\equiv h\pmod p$，即求 $x=\log_g h$。 |
| **算法设计** | 1) **BSGS**：分块 $x=im+j$，哈希表存储 $g^j$，时间-空间权衡 $O(\sqrt p)$。<br>2) **Pollard's Rho**：随机游走找碰撞，指数追踪，期望 $O(\sqrt p)$、空间 $O(1)$。<br>3) **Pohlig-Hellman**：当群阶光滑时分解为素幂子问题，再用 CRT 合并。 |
| **复杂度分析** | BSGS $O(\sqrt p)$ 时间、$O(\sqrt p)$ 空间；Pollard's Rho 期望 $O(\sqrt p)$、$O(1)$ 空间；Pohlig-Hellman 复杂度取决于最大素因子 [Menezes et al., 1996]。 |
| **形式化验证** | BSGS 的正确性基于群的封闭性与完全剩余系的覆盖；Pollard's Rho 利用碰撞导出关于 $x$ 的线性同余方程，可借助扩展欧几里得求解。 |
| **应用场景** | Diffie-Hellman 密钥交换安全性分析、ElGamal 加密体制、椭圆曲线离散对数（ECDLP）。 |
| **扩展变体** | Index Calculus（亚指数时间）、Number Field Sieve for DLP（当前最优渐近复杂度）。 |

**代码实现**：[`discrete_log.rs`](../../../examples/algorithms/src/discrete_log.rs) 中 `bsgs`、`ex_bsgs`、`pollard_rho_dlog`、`pohlig_hellman`

### 10. 中国剩余定理（CRT）

| 维度 | 内容 |
|------|------|
| **理论基础** | 若模数 $m_1,\dots,m_k$ 两两互素，则同余方程组 $x\equiv a_i\pmod{m_i}$ 在模 $M=\prod m_i$ 下有唯一解。 |
| **算法设计** | 迭代合并法：先解 $x\equiv a_1\pmod{m_1}$ 与 $x\equiv a_2\pmod{m_2}$，得 $x=a_1+m_1\cdot t$，其中 $t\equiv (a_2-a_1)\cdot m_1^{-1}\pmod{m_2}$；逐次合并所有方程。 |
| **复杂度分析** | $k$ 个方程合并时间为 $O(k\cdot\log^2 M)$，空间 $O(\log M)$。 |
| **形式化验证** | 基于环同构 $\mathbb{Z}/M\mathbb{Z}\;\cong\;\prod_i \mathbb{Z}/m_i\mathbb{Z}$ 及模逆元的存在唯一性，保证解的存在与唯一 [Niven et al., 1991]。 |
| **应用场景** | RSA-CRT 加速私钥解密、大整数模运算拆分、密码协议中的秘密共享。 |
| **扩展变体** | Garner 算法（避免大整数除法）、模数不互素的扩展 CRT（基于 $\gcd$ 可解性判定）。 |

**代码实现**：[`discrete_log.rs`](../../../examples/algorithms/src/discrete_log.rs) 中 `crt` 函数

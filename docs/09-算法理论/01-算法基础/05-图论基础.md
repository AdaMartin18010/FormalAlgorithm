# 图论基础

> **状态**: draft内容

本文档为算法规范知识体系的核心文档，记录图论基础的形式化定义与理论基础。

## 学习目标

- 理解图论基础的核心概念
- 掌握图论基础的形式化表示

## 核心概念


图论是研究图（由顶点和边构成的离散结构）的数学理论。核心概念包括：图的基本表示（邻接矩阵、邻接表）、图的遍历（BFS、DFS）、连通性、最短路径、最小生成树、网络流、图的着色与匹配等。

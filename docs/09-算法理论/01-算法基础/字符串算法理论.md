## 1. 引言

字符串算法是处理文本数据的核心工具，在生物信息学、信息检索、数据压缩等领域有广泛应用。根据[Manber 1993]的开创性工作，后缀数组提供了比后缀树更空间高效的字符串索引结构。根据[Kärkkäinen 2006]的研究，线性时间后缀数组构建算法（SA-IS和DC3）的出现使后缀数组成为实用的文本处理工具。根据[Gusfield 1997]的经典教材，后缀树和后缀数组是字符串算法中最强大的数据结构。

本文档涵盖：
- 后缀数组的定义和基本性质

- 经典字符串问题的后缀数组解法

## 2. 形式化定义

### 2.1 字符串基本概念

**定义 2.1**（字符串）
字符串 $S$ 是有限字符集 $\Sigma$ 上的有限序列：
$$S = s_1 s_2 ... s_n, \quad s_i \in \Sigma, \quad |S| = n$$

**定义 2.2**（子串）
字符串 $S$ 的子串 $S[i..j]$ 是从位置 $i$ 到位置 $j$ 的连续字符序列：
$$S[i..j] = s_i s_{i+1} ... s_j$$

**定义 2.3**（后缀）
从位置 $i$ 开始的后缀 $S_i$ 为：
$$S_i = S[i..n] = s_i s_{i+1} ... s_n$$

**定义 2.4**（字典序）
对于字符串 $A$ 和 $B$，$A < B$ 当且仅当：
- $A$ 是 $B$ 的真前缀，或
- 存在位置 $i$ 使得 $A[1..i-1] = B[1..i-1]$ 且 $A[i] < B[i]$

### 2.2 后缀数组

**定义 2.5**（后缀数组）
字符串 $S$ 的后缀数组 $SA[1..n]$ 是满足以下条件的排列：
$$S_{SA[1]} < S_{SA[2]} < ... < S_{SA[n]}$$
即 $SA[i]$ 是字典序第 $i$ 小的后缀的起始位置。

**定义 2.6**（逆后缀数组）
逆后缀数组 $ISA[1..n]$ 定义为：
$$ISA[i] = j \iff SA[j] = i$$
即 $ISA[i]$ 表示后缀 $S_i$ 在后缀数组中的排名。

### 2.3 LCP数组

**定义 2.7**（最长公共前缀）
两个字符串 $A$ 和 $B$ 的最长公共前缀长度为：
$$\text{LCP}(A, B) = \max\{k \mid A[1..k] = B[1..k]\}$$

**定义 2.8**（LCP数组）
LCP数组 $LCP[2..n]$ 定义为：
$$LCP[i] = \text{LCP}(S_{SA[i]}, S_{SA[i-1]})$$
即相邻后缀在字典序中的最长公共前缀长度。

**定理 2.1**（LCP区间性质）
对于任意 $i < j$，$\text{LCP}(S_{SA[i]}, S_{SA[j]}) = \min_{k=i+1}^{j} LCP[k]$。

**证明**：由后缀数组的字典序性质，$S_{SA[i]}$ 和 $S_{SA[j]}$ 的公共前缀被中间所有后缀共享。

### 2.4 后缀树

**定义 2.9**（后缀树）
字符串 $S$ 的后缀树是一棵压缩字典树（compressed trie），包含 $S$ 的所有后缀。

**性质**：
- 有 $n$ 个叶节点，对应 $n$ 个后缀
- 最多有 $2n-1$ 个节点
- 边标记为子串，内部节点至少有两个子节点

**定理 2.2**（后缀数组与后缀树的关系）
后缀树的叶节点按中序遍历即为后缀数组顺序。

## 3. 核心算法与理论

### 3.1 倍增算法

**算法 3.1**（后缀数组倍增算法）
输入：字符串 $S[1..n]$
输出：后缀数组 $SA$

```
1. 初始：按单个字符排序，得到 SA 和 rank
2. k = 1
3. while k < n:
4.     按二元组 (rank[i], rank[i+k]) 对位置排序
5.     更新 rank
6.     k = k * 2
7. return SA
```

**定理 3.1**（倍增算法正确性）
倍增算法正确构建后缀数组。

**证明**：
1. 基础：$k=1$ 时按首字符排序正确
2. 归纳：若 $k$ 长前缀顺序正确，则 $2k$ 长前缀通过比较两个 $k$ 长段确定
3. 终止：当 $k \geq n$ 时所有后缀完全排序

**定理 3.2**（倍增算法复杂度）
时间复杂度：$O(n \log n)$
空间复杂度：$O(n)$

### 3.2 SA-IS算法

**算法 3.2**（SA-IS - Suffix Array Induced Sorting）
输入：字符串 $S[0..n-1]$（整数字母表）
输出：后缀数组 $SA$

```
1. 将字符分为L型（字典序下降开始）和S型
2. 识别LMS（Left-Most S-type）位置
3. 对LMS子串排序（递归，若子串唯一）
4. 利用排序后的LMS位置诱导排序所有L型后缀
5. 利用L型后缀诱导排序所有S型后缀
```

**定理 3.3**（SA-IS复杂度）
时间复杂度：$O(n)$
空间复杂度：$O(n)$

**证明思路**：
1. LMS子串数量最多为 $n/2$
2. 递归规模严格递减
3. 诱导排序每轮线性时间

### 3.3 DC3/Skew算法

**算法 3.3**（DC3 / Skew算法）
输入：字符串 $S[0..n-1]$
输出：后缀数组 $SA$

```
1. 将位置按 mod 3 分为 C0, C1, C2
2. 递归排序 C12 = C1 ∪ C2（将每三个字符视为一个"字符"）
3. 利用 C12 的顺序通过基数排序得到 C0 的顺序
4. 合并 C0 和 C12 的顺序
```

**定理 3.4**（DC3算法复杂度）
时间复杂度：$T(n) = T(2n/3) + O(n) = O(n)$
空间复杂度：$O(n)$

### 3.4 Kasai算法（LCP数组）

**算法 3.4**（Kasai LCP算法）
输入：字符串 $S$，后缀数组 $SA$，逆后缀数组 $ISA$
输出：LCP数组

```
1. h = 0
2. for i = 1 to n:
3.     if ISA[i] > 1:
4.         j = SA[ISA[i] - 1]  // 字典序前一个后缀
5.         while S[i+h] == S[j+h]:
6.             h = h + 1
7.         LCP[ISA[i]] = h
8.         if h > 0:
9.             h = h - 1
10. return LCP
```

**定理 3.5**（Kasai算法正确性）
Kasai算法正确计算LCP数组。

**定理 3.6**（Kasai算法复杂度）
时间复杂度：$O(n)$

**证明**：
变量 $h$ 总共增加不超过 $n$ 次，每次减少1，因此总时间 $O(n)$。

### 3.5 后缀树构建

**算法 3.5**（从后缀数组构建后缀树）
```
1. 构建后缀数组 SA 和 LCP 数组
2. 使用栈维护当前路径
3. for i = 1 to n:
4.     while 栈顶LCP值 > LCP[i]:
5.         弹出并创建节点
6.     if 栈顶LCP值 < LCP[i]:
7.         压入新边
8.     添加叶节点
```

**定理 3.7**（后缀树构建复杂度）
时间复杂度：$O(n)$

## 4. 复杂度分析

| 算法/操作 | 时间复杂度 | 空间复杂度 | 备注 |
|-----------|-----------|-----------|------|
| 倍增算法 | $O(n \log n)$ | $O(n)$ | 实现简单 |
| SA-IS | $O(n)$ | $O(n)$ | 实际较快 |
| DC3 | $O(n)$ | $O(n)$ | 常数较小 |
| Kasai (LCP) | $O(n)$ | $O(n)$ | - |
| 后缀树构建 | $O(n)$ | $O(n)$ | 从SA构建 |
| 模式匹配 | $O(m \log n)$ | $O(1)$ | 二分查找 |
| 最长重复子串 | $O(n)$ | $O(n)$ | LCP最大值 |

**定理 4.1**（后缀数组构建下界）
在比较模型下，后缀数组构建的下界为 $\Omega(n \log n)$。

**证明**：通过后缀排序可以完成一般排序问题。

**注**：线性时间算法（SA-IS、DC3）使用整数字母表假设，规避了比较模型下界。

## 5. 实现示例

### 5.1 Rust实现

```rust
pub struct SuffixArray {
    sa: Vec<usize>,      // 后缀数组
    isa: Vec<usize>,     // 逆后缀数组
    lcp: Vec<usize>,     // LCP数组
    text: Vec<u8>,       // 原始文本
}

impl SuffixArray {
    // 倍增算法构建后缀数组
    pub fn new(text: &str) -> Self {
        let n = text.len();
        let mut s: Vec<usize> = text.bytes().map(|b| b as usize).collect();
        s.push(0); // 添加哨兵
        
        let mut sa: Vec<usize> = (0..=n).collect();
        let mut rank: Vec<usize> = s.clone();
        let mut tmp: Vec<usize> = vec![0; n + 1];
        
        let mut k = 1;
        while k <= n {
            // 按 (rank[i], rank[i+k]) 排序
            sa.sort_by(|&i, &j| {
                let ri = (rank[i], if i + k <= n { rank[i + k] } else { 0 });
                let rj = (rank[j], if j + k <= n { rank[j + k] } else { 0 });
                ri.cmp(&rj)
            });
            
            tmp[sa[0]] = 0;
            for i in 1..=n {
                let prev = (rank[sa[i-1]], if sa[i-1] + k <= n { rank[sa[i-1] + k] } else { 0 });
                let curr = (rank[sa[i]], if sa[i] + k <= n { rank[sa[i] + k] } else { 0 });
                tmp[sa[i]] = tmp[sa[i-1]] + if prev != curr { 1 } else { 0 };
            }
            
            rank.copy_from_slice(&tmp);
            if rank[sa[n]] == n { break; }
            k *= 2;
        }
        
        let mut isa = vec![0; n + 1];
        for i in 0..=n {
            isa[sa[i]] = i;
        }
        
        let lcp = Self::compute_lcp(&s, &sa, &isa);
        
        Self { sa, isa, lcp, text: text.bytes().collect() }
    }
    
    // Kasai算法计算LCP数组
    fn compute_lcp(s: &[usize], sa: &[usize], isa: &[usize]) -> Vec<usize> {
        let n = s.len() - 1;
        let mut lcp = vec![0; n + 1];
        let mut h = 0;
        
        for i in 0..n {
            if isa[i] > 0 {
                let j = sa[isa[i] - 1];
                while i + h < n && j + h < n && s[i + h] == s[j + h] {
                    h += 1;
                }
                lcp[isa[i]] = h;
                if h > 0 { h -= 1; }
            }
        }
        
        lcp
    }
    
    // 模式匹配：返回所有出现位置
    pub fn find(&self, pattern: &str) -> Vec<usize> {
        let pat: Vec<u8> = pattern.bytes().collect();
        let n = self.text.len();
        let m = pat.len();
        
        if m == 0 || m > n { return vec![]; }
        
        // 二分查找下界
        let mut left = 1;
        let mut right = n + 1;
        
        while left < right {
            let mid = (left + right) / 2;
            let suffix = &self.text[self.sa[mid]..];
            if suffix < &pat[..] {
                left = mid + 1;
            } else {
                right = mid;
            }
        }
        let start = left;
        
        // 二分查找上界
        right = n + 1;
        while left < right {
            let mid = (left + right) / 2;
            let suffix = &self.text[self.sa[mid]..];
            if suffix.starts_with(&pat) || suffix < &pat[..] {
                left = mid + 1;
            } else {
                right = mid;
            }
        }
        
        (start..right)
            .filter(|&i| self.text[self.sa[i]..].starts_with(&pat))
            .map(|i| self.sa[i])
            .collect()
    }
    
    // 最长重复子串
    pub fn longest_repeated_substring(&self) -> Option<(usize, usize)> {
        let mut max_len = 0;
        let mut pos = 0;
        
        for i in 2..self.lcp.len() {
            if self.lcp[i] > max_len {
                max_len = self.lcp[i];
                pos = self.sa[i];
            }
        }
        
        if max_len > 0 {
            Some((pos, max_len))
        } else {
            None
        }
    }
}
```

### 5.2 Python实现

```python
class SuffixArray:
    def __init__(self, text: str):
        self.text = text
        self.n = len(text)
        self.sa = self._build_sa()
        self.isa = self._build_isa()
        self.lcp = self._build_lcp()
    
    def _build_sa(self) -> list:
        """倍增算法构建后缀数组"""
        n = self.n
        s = [ord(c) for c in self.text] + [0]
        
        sa = list(range(n + 1))
        rank = s[:]
        tmp = [0] * (n + 1)
        
        k = 1
        while k <= n:
            # 按 (rank[i], rank[i+k]) 排序
            sa.sort(key=lambda i: (rank[i], rank[i + k] if i + k <= n else 0))
            
            tmp[sa[0]] = 0
            for i in range(1, n + 1):
                prev = (rank[sa[i-1]], rank[sa[i-1] + k] if sa[i-1] + k <= n else 0)
                curr = (rank[sa[i]], rank[sa[i] + k] if sa[i] + k <= n else 0)
                tmp[sa[i]] = tmp[sa[i-1]] + (1 if prev != curr else 0)
            
            rank, tmp = tmp, rank
            if rank[sa[n]] == n:
                break
            k *= 2
        
        return sa
    
    def _build_isa(self) -> list:
        """构建逆后缀数组"""
        isa = [0] * (self.n + 1)
        for i, pos in enumerate(self.sa):
            isa[pos] = i
        return isa
    
    def _build_lcp(self) -> list:
        """Kasai算法构建LCP数组"""
        n = self.n
        lcp = [0] * (n + 1)
        h = 0
        s = [ord(c) for c in self.text] + [0]
        
        for i in range(n):
            if self.isa[i] > 0:
                j = self.sa[self.isa[i] - 1]
                while i + h < n and j + h < n and s[i + h] == s[j + h]:
                    h += 1
                lcp[self.isa[i]] = h
                if h > 0:
                    h -= 1
        
        return lcp
    
    def find(self, pattern: str) -> list:
        """二分查找模式出现位置"""
        n, m = self.n, len(pattern)
        if m == 0 or m > n:
            return []
        
        # 查找下界
        left, right = 1, n + 1
        while left < right:
            mid = (left + right) // 2
            suffix = self.text[self.sa[mid]:]
            if suffix < pattern:
                left = mid + 1
            else:
                right = mid
        start = left
        
        # 查找上界
        right = n + 1
        while left < right:
            mid = (left + right) // 2
            suffix = self.text[self.sa[mid]:]
            if suffix.startswith(pattern) or suffix < pattern:
                left = mid + 1
            else:
                right = mid
        
        return [self.sa[i] for i in range(start, right) 
                if self.text[self.sa[i]:].startswith(pattern)]
    
    def longest_repeated_substring(self) -> tuple:
        """最长重复子串"""
        max_len, pos = 0, 0
        for i in range(2, len(self.lcp)):
            if self.lcp[i] > max_len:
                max_len = self.lcp[i]
                pos = self.sa[i]
        return (pos, max_len) if max_len > 0 else None
    
    def count_distinct_substrings(self) -> int:
        """统计不同子串数量"""
        # 总数 = n*(n+1)/2 - sum(LCP)
        return self.n * (self.n + 1) // 2 - sum(self.lcp)


# 使用示例
if __name__ == "__main__":

    print("后缀数组:", sa.sa[1:])  # 跳过哨兵
    print("LCP数组:", sa.lcp[1:])
    print("模式 'ana' 出现位置:", sa.find("ana"))
    print("最长重复子串:", sa.longest_repeated_substring())
    print("不同子串数量:", sa.count_distinct_substrings())
```

## 6. 应用案例

### 6.1 生物信息学：基因组分析

在DNA序列分析中，后缀数组用于：
- **重复序列检测**：寻找基因组中的重复片段
- **模式匹配**：快速定位基因片段
- **序列比对**：使用LCP信息进行多序列比对

**案例**：人类基因组计划使用后缀数组索引30亿碱基对的参考基因组，使得短读序列（short reads）的定位时间从数小时降低到数分钟。

### 6.2 数据压缩：Burrows-Wheeler变换

后缀数组是Burrows-Wheeler变换（BWT）的核心：
- BWT通过排序后的循环移位构造可逆变换
- 变换后的数据更易压缩（游程编码）
- 应用于bzip2压缩工具和生物序列压缩

**算法**：
1. 构建所有循环移位的后缀数组
2. 取每个循环移位的最后一个字符
3. 形成的字符串具有良好的可压缩性

### 6.3 全文检索：搜索引擎索引

后缀数组支持高效的全文搜索：
- **前缀搜索**：二分查找定位前缀范围
- **范围查询**：使用LCP区间加速
- **模糊匹配**：结合编辑距离进行近似搜索

**案例**：大型日志分析系统使用后缀数组索引TB级日志数据，支持亚秒级的正则表达式查询。

## 8. 总结

字符串算法是处理文本数据的核心，后缀数组作为最重要的字符串索引结构，具有以下特点：

1. **理论基础**：后缀数组将字符串后缀排序问题转化为数组排序，具有清晰的形式化定义
2. **算法演进**：从$O(n \log n)$的倍增算法到$O(n)$的线性算法，理论不断优化
3. **实用价值**：LCP数组与后缀数组结合，支持高效的字符串查询和分析
4. **广泛应用**：从生物信息学到搜索引擎，后缀数组是文本处理的基础设施

后缀数组与后缀树各有优势：后缀数组空间效率更高（通常4n字节），而后缀树支持更灵活的查询。在实际应用中，可以根据需求选择合适的数据结构。

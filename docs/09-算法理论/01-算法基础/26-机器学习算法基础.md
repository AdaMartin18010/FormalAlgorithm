## 1. 聚类算法：k-means

### 1.1 问题形式化

**定义 1.1.1** ($k$-means 问题)

给定数据集 $X = \{x_1, x_2, \dots, x_n\} \subset \mathbb{R}^d$ 和整数 $k$，$k$-means 问题要求找到 $k$ 个中心点 $C = \{c_1, \dots, c_k\} \subset \mathbb{R}^d$，使得以下目标函数最小化：

$$\Phi(C) = \sum_{i=1}^{n} \min_{j \in \{1,\dots,k\}} \|x_i - c_j\|^2$$

其中每个数据点被分配到距离最近的中心点所在的簇。

### 1.2 Lloyd 算法

**算法 1.2.1** (Lloyd 算法 / $k$-means 标准算法) [Lloyd1982]

```
输入: 数据集 X, 簇数 k
随机初始化 k 个中心点 C = {c_1, ..., c_k}
Repeat until 收敛:
    (E-step) 分配: 对每个 x_i, 令 a_i = argmin_j ||x_i - c_j||^2
    (M-step) 更新: 对每个簇 j, c_j = (1/|S_j|) * Σ_{i: a_i=j} x_i
输出: 中心点 C 和簇分配 {a_i}
```

### 1.3 收敛性与复杂度

**定理 1.3.1** (收敛性) [Lloyd1982]

Lloyd 算法在有限步内收敛到一个局部最优解（目标函数 $\Phi$ 单调不增）。

**证明概要**：

- **E-step**：固定中心点，将每个点分配到最近中心，显然不增加 $\Phi$。
- **M-step**：固定分配，将中心点更新为簇内均值。由均值最小化平方和的性质（最小二乘），此步骤也不增加 $\Phi$。
- 由于可能的分配方案有限（$k^n$ 种），算法必在有限步内收敛。 ∎

**定理 1.3.2** (复杂度)

每轮迭代的时间复杂度为 $O(n \cdot k \cdot d)$，空间复杂度为 $O(n + k \cdot d)$。

> **注**：$k$-means 问题是 NP-hard [Aloise2009]。Lloyd 算法只保证收敛到局部最优，且对初始值敏感。$k$-means++ 初始化 [ArthurVassilvitskii2007] 提供了 $O(\log k)$ 的近似保证。

## 2. 在线学习：加权多数算法

### 2.1 问题设定

**定义 2.1.1** (专家预测问题)

设有 $n$ 个专家（预测器）和一系列二元预测场景（如"明天下雨吗？"），共 $T$ 轮。每轮 $t$：

1. 每个专家 $i$ 给出预测 $\hat{y}_{i,t} \in \{0, 1\}$。
2. 算法基于专家预测做出自己的预测 $\hat{y}_t$。
3. 揭示真实标签 $y_t \in \{0, 1\}$。
4. 算法和预测错误的专家各承受一次损失。

目标是使算法的累积损失接近最佳专家的累积损失。

### 2.2 算法描述

**算法 2.2.1** (加权多数算法 / Weighted Majority Algorithm) [LittlestoneWarmuth1994]

```
初始化: 为每个专家 i 分配权重 w_i = 1
For t = 1, ..., T:
    收集各位专家的预测 {ŷ_{i,t}}
    算法预测 ŷ_t = majority vote weighted by {w_i}
    揭示真实标签 y_t
    对每个预测错误的专家 i:
        w_i ← w_i * β    (其中 0 < β < 1 是折扣因子)
```

### 2.3 后悔界分析

**定理 2.3.1** (加权多数后悔界) [LittlestoneWarmuth1994]

设最佳专家在 $T$ 轮中的错误次数为 $L^*$。取 $\beta = 1/2$，则加权多数算法的错误次数 $L$ 满足：

$$L \leq 2.41 \cdot (L^* + \log_2 n)$$

更一般地，对于任意 $\beta \in (0, 1)$：

$$L \leq \frac{\ln(1/\beta)}{1-\beta} \cdot L^* + \frac{\ln n}{1-\beta}$$

当 $\beta \to 1$ 时，系数 $\frac{\ln(1/\beta)}{1-\beta} \to 1$，可任意接近最佳专家性能，代价是第二项增大。

**证明概要** [CLRS2022, §25.4 扩展]：
设总权重 $W = \sum_i w_i$。初始 $W_0 = n$。

- 若算法在第 $t$ 轮犯错，则至少一半权重的专家犯错，总权重至少减少因子 $(1+\beta)/2$。
- 因此 $W_T \leq n \cdot \left(\frac{1+\beta}{2}\right)^L$。
- 另一方面，最佳专家权重为 $\beta^{L^*}$，所以 $W_T \geq \beta^{L^*}$。
- 联立不等式并取对数即可得证。 ∎

## 3. 梯度下降

### 3.1 形式化描述

**定义 3.1.1** (梯度下降 / Gradient Descent)

给定可微凸函数 $f: \mathbb{R}^d \to \mathbb{R}$，目标是最小化 $f(x)$。梯度下降通过迭代更新：

$$x_{t+1} = x_t - \eta_t \nabla f(x_t)$$

其中 $\eta_t > 0$ 为学习率（步长）。

### 3.2 收敛性分析

**定理 3.2.1** (凸函数梯度下降收敛性) [BoydVandenberghe2004]

设 $f$ 是凸函数且梯度 $L$-Lipschitz 连续（即 $\|\nabla f(x) - \nabla f(y)\| \leq L \|x - y\|$）。若取固定步长 $\eta \leq 1/L$，则经过 $T$ 次迭代后：

$$f\left(\frac{1}{T}\sum_{t=1}^{T} x_t\right) - f(x^*) \leq \frac{\|x_1 - x^*\|^2}{2\eta T}$$

即收敛速度为 $O(1/T)$，要达到精度 $\epsilon$ 需要 $O(1/\epsilon)$ 次迭代。

**证明概要**：
利用 $L$-Lipschitz 梯度的下降引理：
$$f(x_{t+1}) \leq f(x_t) - \frac{\eta}{2}\|\nabla f(x_t)\|^2$$
再结合凸性 $f(x_t) - f(x^*) \leq \nabla f(x_t)^T (x_t - x^*)$ 和更新规则，通过错位相消（telescoping）即可得到上述界。 ∎

**定理 3.2.2** (强凸函数收敛性)

若 $f$ 还是 $\mu$-强凸的（$\mu > 0$），则取 $\eta \leq 1/L$ 时：

$$\|x_{T+1} - x^*\|^2 \leq \left(1 - \frac{\mu}{L}\right)^T \|x_1 - x^*\|^2$$

收敛速度为线性 $O(\log(1/\epsilon))$。

### 3.3 随机梯度下降

**定义 3.3.1** (随机梯度下降 / SGD)

当目标函数是大量样本损失之和 $f(x) = \frac{1}{n}\sum_{i=1}^{n} f_i(x)$ 时，每次迭代随机选取一个样本 $i_t$ 并用其梯度代替全梯度：

$$x_{t+1} = x_t - \eta_t \nabla f_{i_t}(x_t)$$

**定理 3.3.1** (SGD 收敛性) [Bottou2018]

设 $f$ 是凸函数，$E[\|\nabla f_{i_t}(x_t)\|^2] \leq G^2$。取递减步长 $\eta_t = \frac{\eta_0}{\sqrt{t}}$，则：

$$E\left[f\left(\frac{1}{T}\sum_{t=1}^{T} x_t\right)\right] - f(x^*) \leq O\left(\frac{1}{\sqrt{T}}\right)$$

每轮迭代复杂度从 $O(n)$（全梯度）降为 $O(1)$。

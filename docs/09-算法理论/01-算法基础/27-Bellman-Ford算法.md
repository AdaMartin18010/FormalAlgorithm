# Bellman-Ford 算法 —— 单源最短路径与负权环检测

## 一、理论基础

### 1.1 问题形式化

给定带权有向图 $G=(V,E,w)$，其中 $w:E\to\mathbb{R}$，以及源点 $s\in V$。**单源最短路径问题**要求计算从 $s$ 到每个顶点 $v\in V$ 的最短路径权重 $\delta(s,v)$：

$$
\delta(s,v)=\begin{cases}
\displaystyle\min\Big\{\sum_{i=1}^{k}w(v_{i-1},v_i): s=v_0,\,v_k=v,\,(v_{i-1},v_i)\in E\Big\}, & \text{若路径存在},\\[6pt]
+\infty, & \text{否则}.\end{cases}
$$

### 1.2 松弛与最短路径估计

对每个顶点 $v$ 维护一个估计值 $d[v]$（上界）。**松弛操作** $\text{RELAX}(u,v,w)$ 定义为：

$$
\text{若 } d[u]+w(u,v)<d[v]，\text{ 则更新 } d[v]\gets d[u]+w(u,v),\;\pi[v]\gets u.
$$

**关键性质**（CLRS Lemma 24.10–24.14）：

- **上界性质**：$\forall v,\;d[v]\ge \delta(s,v)$，且一旦 $d[v]=\delta(s,v)$ 将不再增大。
- **收敛性质**：若 $s\leadsto u\to v$ 是一条最短路径，且在松弛边 $(u,v)$ 之前已有 $d[u]=\delta(s,u)$，则松弛后 $d[v]=\delta(s,v)$。
- **路径松弛性质**：若 $p=\langle v_0,v_1,\dots,v_k\rangle$ 是从 $s=v_0$ 到 $v_k$ 的最短路径，则按顺序对边 $(v_0,v_1),\dots,(v_{k-1},v_k)$ 各松弛一次后，$d[v_k]=\delta(s,v_k)$。

### 1.3 负权环

若图中存在从 $s$ 可达的环 $c$ 使得 $w(c)<0$，则最短路径无下界（可无限减小）。此时算法应报告 **存在负权环**。

## 二、算法设计

### 2.1 伪代码

```text
BELLMAN-FORD(G, w, s)
1  INITIALIZE-SINGLE-SOURCE(G, s)
2  for i = 1 to |V| - 1
3      for each edge (u, v) ∈ G.E
4          RELAX(u, v, w)
5  for each edge (u, v) ∈ G.E
6      if d[v] > d[u] + w(u, v)
7          return FALSE   // 存在负权环
8  return TRUE
```

### 2.2 工程实现要点

1. **顶点集提取**：从边列表与源点中收集全部顶点，初始化距离为 $+\infty$（源点为 $0$）。
2. **提前终止**：若某轮松弛未更新任何估计值，可直接跳出循环。
3. **负权环判定**：额外一轮遍历，若仍能松弛，则说明存在从源点可达的负权环。
4. **路径重建**：维护前驱数组 $\pi$，通过回溯得到具体最短路径。

## 三、复杂度分析

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| **时间复杂度** | $O(VE)$ | 外层 $|V|-1$ 轮，每轮扫描全部 $E$ 条边 |
| **空间复杂度** | $O(V)$ | 存储距离数组 $d$、前驱数组 $\pi$ 与顶点集合 |
| **提前终止优化** | 最好 $O(E)$ | 若一轮无更新即可结束 |

### 3.1 与 Dijkstra 的对比

- Dijkstra 依赖非负权假设，采用贪心策略，时间 $O((V+E)\log V)$（二叉堆）。
- Bellman-Ford 允许负权边，通过动态规划式的多轮松弛保证正确性，时间 $O(VE)$。

## 四、形式化验证

### 4.1 循环不变式

**引理 24.2**（CLRS）：在经过 Bellman-Ford 算法第 2–4 行的 $|V|-1$ 轮松弛后，对所有从 $s$ 可达的顶点 $v$，有 $d[v]=\delta(s,v)$。

**循环不变式**（针对外层循环索引 $i$）：
> 在第 $i$ 轮松弛结束后，对任意从 $s$ 出发、最短路径上最多包含 $i$ 条边的顶点 $v$，有 $d[v]=\delta(s,v)$。

**证明**（数学归纳法）：

- **初始化**（$i=0$）：仅源点 $s$ 的最短路径含 $0$ 条边，此时 $d[s]=0=\delta(s,s)$，其余顶点 $d[v]=+\infty$，不变式成立。
- **保持**：假设第 $i-1$ 轮后不变式成立。考虑从 $s$ 到 $v$ 的一条最短路径 $p$ 含至多 $i$ 条边。设 $p$ 上 $v$ 的前驱为 $u$，则子路径 $s\leadsto u$ 含至多 $i-1$ 条边。由归纳假设，第 $i-1$ 轮后 $d[u]=\delta(s,u)$。在第 $i$ 轮中，边 $(u,v)$ 必被松弛一次，由**收敛性质**可得 $d[v]=\delta(s,v)$。
- **终止**：经过 $|V|-1$ 轮后，所有不含环的最短路径（最多 $|V|-1$ 条边）均被正确估计。

### 4.2 负权环检测正确性

**定理 24.4**（CLRS）：Bellman-Ford 算法第 5–7 行返回 $\text{FALSE}$ 当且仅当 $G$ 中存在从 $s$ 可达的负权环。

**证明概要**：

- **充分性**：若返回 $\text{FALSE}$，则某边 $(u,v)$ 满足 $d[v]>d[u]+w(u,v)$。若无负权环，则由引理 24.2 应有 $d[v]=\delta(s,v)\le d[u]+w(u,v)$，矛盾。故必存在负权环。
- **必要性**：若存在从 $s$ 可达的负权环，则最短路径权重为 $-\infty$，不存在有限上界。因此无论经过多少轮松弛，环上至少有一条边仍可被松弛，算法必返回 $\text{FALSE}$。

## 五、应用场景

1. **货币套利检测**：将货币兑换率取对数后作为边权，负权环对应套利机会。
2. **网络路由协议**：RIP（Routing Information Protocol）的早期版本基于 Bellman-Ford 思想，允许链路延迟为负（特殊配置）。
3. **差分约束系统**：形如 $x_j-x_i\le c_k$ 的约束可转化为含负权边的图，Bellman-Ford 可用于判定可行性并求出特解。
4. **生物信息学**：基因调控网络中抑制关系可建模为负权边，Bellman-Ford 用于计算调控信号的极值延迟。

## 六、扩展变体

### 6.1 SPFA（Shortest Path Faster Algorithm）

利用队列优化：仅当某顶点的距离被更新时，才将其出边加入待松弛队列。平均时间复杂度为 $O(E)$，最坏仍为 $O(VE)$。适用于稀疏图且负权边较少的情形。

### 6.2 差分约束与可行性判定

将每个约束 $x_j-x_i\le c_k$ 映射为边 $(i,j,c_k)$，并引入超级源点 $s$ 向所有变量连 $0$ 权边。运行 Bellman-Ford：

- 若存在负权环，则约束系统 **不可行**；
- 否则，$d[x_i]$ 给出满足所有约束的一个可行解。

### 6.3 负权环定位

在检测到可松弛边 $(u,v)$ 后，沿前驱指针回溯 $|V|$ 步，必进入负权环，可提取出具体的环结构。

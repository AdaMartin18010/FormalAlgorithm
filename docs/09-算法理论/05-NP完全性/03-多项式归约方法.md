# 多项式归约方法

## 概述

多项式归约（Polynomial-time Reduction）是证明问题 NP 完全性的核心技术。若能将已知 NP 完全问题 $\Pi_1$ 在多项式时间内转化为问题 $\Pi_2$，则 $\Pi_2$ 至少与 $\Pi_1$ 一样难。

## 归约的形式化定义

$$\Pi_1 \leq_p \Pi_2$$

表示存在多项式时间可计算的函数 $f$，使得：

$$x \in \Pi_1 \iff f(x) \in \Pi_2$$

### 传递性

若 $\Pi_1 \leq_p \Pi_2$ 且 $\Pi_2 \leq_p \Pi_3$，则 $\Pi_1 \leq_p \Pi_3$。

## 经典归约路径

### 路径 1：SAT → 3-SAT → Vertex Cover

**SAT → 3-SAT**：

- 将长度 $> 3$ 的子句拆分为多个三元子句
- 引入辅助变量保持等价性

**3-SAT → Vertex Cover**：

- 为每个变量构建"真值选择 gadget"（两个顶点代表 true/false）
- 为每个子句构建"满足 gadget"
- 设置 $k = n + 2m$（$n$ 变量，$m$ 子句）

### 路径 2：Vertex Cover ↔ Independent Set ↔ Clique

**Vertex Cover → Independent Set**：

- $C$ 是顶点覆盖 $\iff$ $V \setminus C$ 是独立集
- 直接取补集，无需额外构造

**Independent Set → Clique**：

- $S$ 是 $G$ 的独立集 $\iff$ $S$ 是 $\overline{G}$ 的团
- 取补图即可

### 路径 3：3-SAT → Hamiltonian Cycle

更复杂的构造：

- 为每个变量构建"选择路径"（true/false 两条平行路径）
- 为每个子句构建"满足节点"
- 通过精心设计的连接保证：哈密顿回路存在 $\iff$ 公式可满足

## 归约设计技巧

| 技巧 | 说明 |
|------|------|
| Gadget 设计 | 用图/公式的小结构模拟逻辑门或约束 |
| 强制选择 | 通过顶点覆盖大小限制强制某些顶点入选 |
| 信号传播 | 让 gadget 之间的连接传递真值选择 |
| 验证结构 | 确保只有正确配置才能满足约束 |

## 从 NP 到 NP-hard

若 $\Pi \in NP$ 且存在 NP 完全问题 $\Pi' \leq_p \Pi$，则 $\Pi$ 是 NP 完全的。

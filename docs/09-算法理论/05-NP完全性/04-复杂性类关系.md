# 复杂性类关系

## 核心复杂性类

### P（Polynomial Time）

可在确定性图灵机上于多项式时间内**判定**的问题类。

**代表问题**：排序、最短路径、最小生成树、最大流、线性规划。

### NP（Nondeterministic Polynomial Time）

可在多项式时间内**验证**解的问题类。等价于非确定性图灵机在多项式时间内可判定的问题。

**代表问题**：SAT、旅行商判定、团、顶点覆盖、子集和。

### NP-Complete

既是 NP 成员，又所有 NP 问题可多项式归约到它的问题。是 NP 中最"难"的问题。

### NP-Hard

所有 NP 问题可多项式归约到它，但不要求本身属于 NP。可能超出 NP（如停机问题）。

### co-NP

补问题属于 NP 的类。即"否"实例可在多项式时间内验证。

**开放问题**：NP = co-NP？普遍认为不等价。

## 包含关系图

```
P ⊆ NP ∩ co-NP
NP ⊆ PSPACE
co-NP ⊆ PSPACE
PSPACE ⊆ EXPTIME
EXPTIME ⊆ NEXPTIME
```

### 已知关系

- **P ⊆ NP**：显然，可求解则可验证
- **P = NP?**：千年大奖问题，普遍认为 $P \neq NP$
- **NP ⊆ PSPACE**：非确定性仅需多项式空间
- **PSPACE ⊆ EXPTIME**：多项式空间机器可在指数时间内模拟

### believed 关系（未证明）

$$P \neq NP \neq PSPACE \neq EXPTIME$$

## 其他重要复杂性类

| 类 | 定义 | 代表问题 |
|-----|------|---------|
| BPP | 有界错误概率多项式时间 | 素性测试（Miller-Rabin）|
| RP | 单侧错误，yes 以高概率正确 | - |
| ZPP | 零错误，期望多项式时间 | - |
| #P | 计数问题 | 完美匹配计数 |
| PPAD | 多项式校验有向图 | Nash 均衡 |

## 实际意义

| 问题类型 | 处理策略 |
|---------|---------|
| P | 精确求解 |
| NP-Complete（小规模）| 精确算法（DP、分支限界）|
| NP-Complete（大规模）| 近似算法、启发式、松弛 |
| NP-Hard（优化）| 近似、局部搜索、元启发式 |

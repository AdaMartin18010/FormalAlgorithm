# k-中心问题（k-Center）

## 问题定义

**输入**：度量空间中的点集 $P$，整数 $k$
**输出**：选择 $k$ 个中心点 $C \subseteq P$，使得最大覆盖半径最小：

$$\min_{C \subseteq P, |C|=k} \max_{p \in P} \min_{c \in C} d(p, c)$$

即：每个点到最近中心的最大距离最小化。

## 2-近似贪心算法

### 算法（Gonzalez 算法）

```
C = {任意点}
while |C| < k:
    选择离 C 最远的点 p
    C = C ∪ {p}
return C
```

### 近似比证明

设最优解半径为 $r^*$，算法得到的半径为 $r$。

- 算法选择的 $k+1$ 个点两两距离 $\geq r$（由贪心选择保证）
- 最优解用 $k$ 个半径为 $r^*$ 的球覆盖所有点
- 由鸽巢原理，$k+1$ 个点中至少两个落在同一个最优球内
- 这两点距离 $\leq 2r^*$
- 因此 $r \leq 2r^*$

**定理**：除非 P=NP，k-中心不存在优于 $2-\varepsilon$ 的近似算法。

## 与 k-means / k-medians 的对比

| 问题 | 目标函数 | 近似难度 |
|------|---------|---------|
| k-center | 最大距离 | $2$-近似（紧）|
| k-median | 距离之和 | $2.675$-近似（最好已知）|
| k-means | 距离平方之和 | $9+\varepsilon$-近似（局部搜索）|

## 实际应用

### 设施选址

- 在 $k$ 个城市建医院，使得最远的居民到医院的距离最小
- 应急服务站点布置

### 聚类初始化

- k-means++ 使用类似思想选择初始中心
- 保证初始中心分布均匀，加速收敛

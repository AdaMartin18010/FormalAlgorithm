# 滚动哈希（Rolling Hash）

## 概述

滚动哈希（Rabin-Karp 哈希）是一种在线性时间内预处理字符串的技术，支持 $O(1)$ 查询任意子串的哈希值。它是字符串匹配、 palindrome 检测、重复子串查找等问题的核心工具。

## 多项式滚动哈希

将字符串 $S[0..n-1]$ 视为一个 $b$ 进制数：

$$H(S) = \sum_{i=0}^{n-1} S[i] \cdot b^{n-1-i} \pmod{m}$$

为了支持快速扩展和滑动窗口，通常使用前缀哈希形式：

$$pref[i] = \sum_{j=0}^{i-1} S[j] \cdot b^{i-1-j} \pmod{m}$$

## 子串哈希查询

子串 $S[l..r]$（0-based，左闭右开）的哈希值：

$$hash(l, r) = pref[r] - pref[l] \cdot b^{r-l} \pmod{m}$$

预处理 $pref$ 和 $pow$ 数组后，每次查询 $O(1)$。

## 双哈希（Double Hashing）

使用两组不同的 $(b, m)$ 参数，将碰撞概率从 $O(1/m)$ 降至 $O(1/m_1 m_2)$：

```
hash = (h1 << 32) | h2
```

## 应用

| 问题 | 滚动哈希解法 |
|------|-------------|
| Rabin-Karp 匹配 | 计算模式串哈希，滑动窗口比对 |
| 最长重复子串 | 二分长度 + 哈希集合检测 |
| 最长回文子串 | 正序/逆序双哈希 + 二分 |
| 字符串比较 | 哈希相等大概率内容相等 |

## 模数选择

| 模数 | 性质 |
|------|------|
| $10^9 + 7$ | 常用大质数 |
| $10^9 + 9$ | 次常用 |
| $2^{64}$ | 利用自然溢出（无乘法溢出风险）|
| $998244353$ | NTT 友好质数 |

## 04-字符串算法. Z-Algorithm / Z 算法

### 1. 理论基础

**Z 函数**（Z-Function）定义于字符串 $S$ 上，记 $Z[i]$ 为 $S$ 与后缀 $S[i..]$ 的最长公共前缀长度（$Z[0] = n$）。形式化地，

$$
Z[i] \;=\; \max\{\,k \mid 0 \le k \le n-i,\; S[0..k-1] = S[i..i+k-1]\,\}.
$$

Z 算法可在 $O(n)$ 时间内计算整个 $Z$ 数组，其效率依赖于维护当前最右匹配区间 $[l, r]$，从而避免大量冗余比较 [Gusfield, 1997]。

### 2. 算法设计

1. **初始化**：$Z[0] = n$，$l = 0$，$r = 0$。
2. **线性扫描**：对 $i = 1 \dots n-1$：
   - 若 $i \le r$，则 $Z[i] \ge \min(Z[i-l],\; r-i+1)$，因为 $S[i..r]$ 与 $S[i-l..r-l]$ 完全相同；
   - 从 $Z[i]$ 开始继续暴力比较 $S[Z[i]]$ 与 $S[i+Z[i]]$，相等则扩展；
   - 若 $i+Z[i]-1 > r$，更新 $l = i$，$r = i+Z[i]-1$。

3. **模式匹配应用**：将模式 $P$、分隔符 `$`（不出现在字符集中）、文本 $T$ 拼接为 $S = P \$ T$。计算 $S$ 的 $Z$ 数组后，对所有 $i > |P|$，若 $Z[i] = |P|$，则对应文本位置 $i - |P| - 1$ 为一次匹配。

项目实现 additionally 提供了 `z_count` 接口用于统计出现次数。

**代码实现**：[`z_algorithm.rs`](../../../examples/algorithms/src/z_algorithm.rs)

### 3. 复杂度分析

| 操作 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 计算 Z 数组 | $O(n)$ | $O(n)$ |
| 模式匹配 | $O(n+m)$ | $O(n+m)$ |

其中 $n$ 为文本长度，$m$ 为模式长度。$Z$ 算法与 KMP 在渐近复杂度上等价，但实现更为简洁。

### 4. 形式化验证 / 正确性论证

**引理（区间继承）**：若当前最右匹配区间为 $[l, r]$，且 $i \in [l, r]$，则

$$
Z[i] \;=\; \min\bigl(Z[i-l],\; r-i+1\bigr) + \text{ext},
$$

其中 $\text{ext}$ 为在 $r$ 右侧的额外相等字符数。

*证明*：由 $[l,r]$ 的定义，$S[l..r]$ 是 $S$ 的某个前缀的副本，即 $S[l..r] = S[0..r-l]$。因此子串 $S[i..r]$ 与 $S[i-l..r-l]$ 完全相同。$Z[i-l]$ 给出了在位置 $i-l$ 处的匹配长度，但该长度可能超出 $r-i+1$（即超出当前区间的右边界），超出部分无法保证与 $S$ 的前缀相等，故取最小值。对于区间内部，信息完全复用；对于 $r$ 右侧，只能暴力比较。$\square$

**定理（Z 算法线性时间）**：算法计算整个 $Z$ 数组的比较次数不超过 $2n$。

*证明*：变量 $r$ 在算法执行过程中只增不减。每次进入暴力比较循环时，要么 $i > r$（此时 $r$ 至少扩展到 $i+Z[i]-1$），要么 $i \le r$ 但 $Z[i-l]$ 的信息已覆盖到 $r$，只需从 $r+1$ 开始比较。因此每成功比较一次，$r$ 至少右移一位。由于 $r \le n-1$，总比较次数为 $O(n)$。$\square$

### 5. 应用场景

- **线性时间单模式匹配**：与 KMP 等价，但代码更短，常用于竞赛编程。
- **周期性检测**：字符串 $S$ 的最小周期 $p$ 满足 $p = n - Z[n-1]$ 且 $n \bmod p = 0$。
- **DNA 重复单元查找**：在基因组中快速定位周期性重复序列。
- **压缩与索引**：Z 数组与前缀函数互为对偶，均可用于 LZ77 压缩算法的起始匹配长度计算。

### 6. 扩展变体

| 变体 | 说明 |
|------|------|
| **多模式 Z 算法** | 将多个模式用不同分隔符拼接后一次性计算 Z 数组，适用于模式数较少的场景。 |
| **LPF 数组** | Longest Previous Factor，Z 函数的推广，用于文本压缩与重复结构分析。 |
| **在线 Z 算法** | 逐字符接收输入并增量维护 $[l,r]$，适用于流式文本处理。 |
| **Z-Box 分解** | 基于 Z 函数将字符串分解为不相交的 Z-Box，用于字符串的周期性与覆盖问题。 |

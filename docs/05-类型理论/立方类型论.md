## 1. 引言

立方类型论（Cubical Type Theory）是近年来为同伦类型论提供计算解释的重要进展 [Cohen 2016; Angiuli2025]。它通过引入**区间类型** $\mathbb{I}$ 与**路径类型** $Path\ A\ a\ b$，解决了经典 HoTT 中泛等公理（Univalence Axiom）缺乏计算内容的问题 [Rijke2025, §Cubical]。在 Cubical Type Theory 中，等价到恒等的映射 $ua$ 具有直接的计算规则，从而无需将单值作为不可计算的公理引入。

## 2. 立方模型

### 2.1 立方体的类比

在立方类型论中，等式证明被可视化为立方体：

- 点（0维）：元素
- 线（1维）：等式 $a = b$
- 面（2维）：等式之间的等式
- 体（3维）：更高阶等式

### 2.2 区间类型

**定义 2.1 (区间)**：类型 $\mathbb{I}$ 有两个端点 $i0$ 和 $i1$。

**用途**：参数化路径。

## 3. 路径类型

**定义 3.1 (路径类型)** [Cohen 2016]：对于类型 $A$ 和元素 $a, b : A$，路径类型 $Path\ A\ a\ b$ 包含从 $a$ 到 $b$ 的连续路径。

**构造**：

```
λ(i : I). t : Path A a b
```

其中 $t[i0/i] = a$ 和 $t[i1/i] = b$。

**消去**：

```
p @ i : A
```

应用路径 $p$ 到区间点 $i$。

### 3.1 路径归纳 vs 泛等

**传统HoTT**：路径归纳（J eliminator）是基本 eliminator。
**立方TT**：路径抽象是基本的，路径归纳可推导。

## 4. 构造性泛等

### 4.1 运输 (Transport)

**定义 4.1 (运输)**：

```
transport : (A : I → Type) → A i0 → A i1
```

在类型族 $A$ 中，将元素从一端运输到另一端。

### 4.2 复合 (Composition)

**定义 4.2 (复合)**：填充立方体的能力。

给定立方体的部分边界，复合操作可以找到完整的内部。

### 4.3 泛等公理的实现

**定理 4.1 [Cohen 2016; CubicalAgda2024]**：在立方类型论中，泛等公理具有计算解释：

```
ua : (A ≡ B) → (A = B)
ua β : ua (f, g, η, ε) @ i = ...
```

## 5. 高阶归纳类型 (HITs)

### 5.1 圆的类型

**定义 5.1 (S¹)**：

- 基点 $base : S^1$
- 环路 $loop : Path\ S^1\ base\ base$

### 5.2 计算内容

在立方TT中，对HITs的归纳有直接的计算规则。

## 6. Cubical Agda

### 6.1 实践示例

```agda
-- 定义路径类型
_≡_ : {A : Set} → A → A → Set
_≡_ {A} x y = PathP (λ _ → A) x y

-- 证明对称性
sym : {A : Set} {x y : A} → x ≡ y → y ≡ x
sym p = λ i → p (~ i)

-- 证明传递性
trans : {A : Set} {x y z : A} → x ≡ y → y ≡ z → x ≡ z
trans p q = λ i → hcomp (λ j → λ { (i = i0) → p i0
                                   ; (i = i1) → q j })
                         (p i)
```

## 7. 与经典HoTT的比较

| 特性 | 经典HoTT | 立方TT |
|------|---------|--------|
| 泛等 | 公理 | 可计算 |
| 路径 | 归纳定义 | 基本构造 |
| 实现 | 困难 | Cubical Agda |
| 性能 | 慢（公理） | 更快 |

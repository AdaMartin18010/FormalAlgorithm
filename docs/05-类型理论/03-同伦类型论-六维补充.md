## 一、规范定义深化

### 1.1 判断与推导规则的形式化

同伦类型论（Homotopy Type Theory, HoTT）扩展了 Martin-Löf 依赖类型论，引入**恒等类型作为路径**（propositional equality as paths）。按 Rijke (2025) 的标准记法，核心判断为：

$$
\Gamma \vdash a : A \qquad \Gamma \vdash p : a =_A b
$$

其中 $a =_A b$（亦记 $\text{Id}_A(a,b)$）表示从 $a$ 到 $b$ 的**路径类型**。

**路径类型的引入与消去规则**：

$$
\frac{a : A}{\text{refl}_a : a =_A a}\;(=\text{-Intro})
\qquad
\frac{C : \prod_{x,y:A} (x =_A y) \to \mathcal{U} \quad c : \prod_{x:A} C(x,x,\text{refl}_x) \quad p : a =_A b}{J(C,c,a,b,p) : C(a,b,p)}\;(=\text{-Elim})
$$

计算规则：$J(C,c,a,a,\text{refl}_a) \equiv c(a)$。

**函数外延性（Function Extensionality）**：

$$
\text{funext} : \prod_{f,g: A \to B} \left(\prod_{x:A} f(x) =_{B} g(x)\right) \to (f =_{A \to B} g)
$$

**单值公理（Univalence Axiom, UA）** [Voevodsky2010; Rijke2025, §2.4]：

$$
\text{Id}_\mathcal{U}(A, B) \simeq (A \simeq B)
$$

其中 $A \simeq B$ 为等价类型（具有拟逆的函数）。

### 1.2 高阶归纳类型（HITs）

高阶归纳类型同时包含**点构造子**与**路径构造子**。以圆 $S^1$ 为例：

- $\text{base} : S^1$
- $\text{loop} : \text{base} =_{S^1} \text{base}$

其归纳原理要求为点构造子提供值，为路径构造子提供覆盖该路径的**路径覆盖**（pathover）。

## 二、模型设计深化

### 2.1 拓扑空间模型

HoTT 的标准模型为 **Simplicial Sets** 与 **Cubical Sets**：

- **类型 = 空间**（拓扑空间或 ∞-groupoid）
- **项 = 点**
- **恒等类型 $a =_A b$ = 从 $a$ 到 $b$ 的路径空间**
- **高阶路径 = 路径之间的同伦**

### 2.2 ∞-Groupoid 模型

由 Voevodsky 提出，后经 Kapulkin–Lumsdaine 严格化：

每个类型 $A$ 自带结构：
- 0-路径：点 $a : A$
- 1-路径：$p : a =_A b$
- 2-路径：$\alpha : p =_{a=_A b} q$
- …

构成弱 ∞-groupoid（Kan complex）。

### 2.3 立方类型论语义

Cubical Type Theory [Cohen2016] 为 UA 提供**计算解释**：

- **Interval** $\mathbb{I}$：带有端点 $0, 1$ 和连接运算
- **Path type**：$\text{Path}_A\,a\,b := (i : \mathbb{I}) \to A\, [i=0 \mapsto a, i=1 \mapsto b]$
- **Glue type**：构造等价类型之间的路径
- **Transport**：沿路径运输元素的显式计算规则

## 三、数学符号与推导

### 3.1 同伦水平（h-level）

HoTT 使用 h-level 对类型按高阶路径信息进行分层 [Rijke2025, §3]：

| h-level | 名称 | 定义 | 直觉 |
|---------|------|------|------|
| 0 | 可缩（Contractible） | $\text{isContr}(A) := \sum_{a:A} \prod_{x:A} a = x$ | 单点空间 |
| 1 | 命题（Proposition） | $\text{isProp}(A) := \prod_{x,y:A} \text{isContr}(x = y)$ | 真/假 |
| 2 | 集合（Set） | $\text{isSet}(A) := \prod_{x,y:A} \text{isProp}(x = y)$ | 无高阶路径 |
| $n+1$ | $n$-类型 | $\text{isLevel}(n+1, A) := \prod_{x,y:A} \text{isLevel}(n, x=y)$ | 依此类推 |

### 3.2 同伦群

**定义**：$\pi_n(A, a) := \|\Omega^n(A, a)\|_0$，其中：

- $\Omega(A, a) := (a =_A a)$
- $\Omega^{n+1}(A, a) := \Omega(\Omega^n(A, a), \text{refl}^n_a)$
- $\|-\|_0$ 为 0-截断（set-truncation）

**定理**：$\pi_1(S^1, \text{base}) \cong \mathbb{Z}$（由 loop 生成自由群）。

### 3.3 Eckmann–Hilton 定理

**定理** [EckmannHilton1962]：对 $n \geq 2$，$\pi_n(A, a)$ 是交换群。

*证明概要*：
高阶路径 $p, q : \text{refl}^n_a = \text{refl}^n_a$ 的连接可沿不同方向交换，通过二维正方形的填充证明 $p \cdot q = q \cdot p$。

## 四、示例性代码

以下代码基于 Lean 4 核心库，展示 HoTT 基础概念。完整的 Univalence Axiom 与 HITs 需要专门的 HoTT 库支持。

```lean
/-
HoTT 基础概念的形式化示例（Lean 4 核心库）
对应主文档：03-同伦类型论.md
-/

namespace HoTTExtended

universe u v

variable {A : Type u} {B : Type v}

-- 1. 恒等类型（路径类型）的基本操作 -----------------------------------------

/-- 路径的对称性（逆路径） -/
def pathInv {x y : A} (p : x = y) : y = x :=
  Eq.symm p

/-- 路径的传递性（路径连接） -/
def pathConcat {x y z : A} (p : x = y) (q : y = z) : x = z :=
  Eq.trans p q

/-- 路径上的函数作用 (ap) -/
def pathAp (f : A → B) {x y : A} (p : x = y) : f x = f y :=
  congrArg f p

-- 2. 运输 (Transport) ------------------------------------------------------

/-- 沿路径在类型族中运输元素
corresponds to transport^{P}(p, u) in HoTT Book -/
def transport {P : A → Type u} {x y : A} (p : x = y) (u : P x) : P y :=
  p ▸ u

/-- 运输在 refl 上的行为 -/
example {P : A → Type u} {x : A} (u : P x)
    : transport (rfl : x = x) u = u := by
  rfl

-- 3. 依赖路径应用 (apd) ----------------------------------------------------

/-- apd: 对依赖函数应用路径，得到运输后的等式
corresponds to Rijke (2025) §2.3 -/
def apd {P : A → Type u} (f : (a : A) → P a) {x y : A} (p : x = y)
    : transport p (f x) = f y := by
  cases p
  rfl

-- 4. 函数外延性（作为公理/示例） -------------------------------------------

/-- 函数外延性：逐点相等推导出函数相等
在 Lean 4 中 funext 是核心公理之一 -/
example {f g : A → B} (h : ∀ x, f x = g x) : f = g := by
  funext x
  exact h x

-- 5. 子类型与商类型（对应 HoTT 中的 Σ 与 HIT 思想） -------------------------

/-- 偶数子类型：通过 Σ 类型（依赖对）构造 -/
def EvenNat : Type := { n : Nat // n % 2 = 0 }

example : EvenNat := ⟨2, by decide⟩

/-- 模 2 等价关系（为商类型做准备） -/
inductive Mod2Rel : Nat → Nat → Prop where
  | intro (n m : Nat) (h : n % 2 = m % 2) : Mod2Rel n m

theorem Mod2Rel.refl (n : Nat) : Mod2Rel n n := by
  constructor; rfl

theorem Mod2Rel.symm {n m : Nat} (h : Mod2Rel n m) : Mod2Rel m n := by
  cases h; constructor; rw [‹n % 2 = m % 2›]

theorem Mod2Rel.trans {n m k : Nat} (h1 : Mod2Rel n m) (h2 : Mod2Rel m k)
    : Mod2Rel n k := by
  cases h1; cases h2; constructor; rw [‹_›, ‹_›]

def Mod2 : Type :=
  @Quotient Nat (⟨Mod2Rel, Mod2Rel.refl, Mod2Rel.symm, Mod2Rel.trans⟩)

end HoTTExtended
```

## 五、形式化证明

### 5.1 路径归纳证明对称性

**定理**：对任意 $A : \mathcal{U}$，$a, b : A$，$p : a =_A b$，存在 $\text{sym}(p) : b =_A a$。

*证明*（使用 J 规则）：

定义类型族 $C(x, y, q) := (y =_A x)$。

对任意 $x : A$，有 $\text{refl}_x : C(x, x, \text{refl}_x) = (x =_A x)$。

由 J 规则：

$$J(C, \lambda x.\,\text{refl}_x, a, b, p) : C(a,b,p) = (b =_A a)$$

定义 $\text{sym}(p) := J(C, \lambda x.\,\text{refl}_x, a, b, p)$。∎

### 5.2 运输与替换

**定理**（运输与 ap 的交换）：设 $P, Q : A \to \mathcal{U}$，$f : \prod_{x:A} P(x) \to Q(x)$，$p : a =_A b$，$u : P(a)$，则：

$$\text{transport}^Q(p, f(a, u)) = f(b, \text{transport}^P(p, u))$$

*证明*：对 $p$ 使用路径归纳，归结为 $\text{refl}_a$ 情形，两边均化为 $f(a, u)$。∎

### 5.3 单值公理蕴含函数外延性

**定理** [Voevodsky2010; Rijke2025, §2.4]：UA 蕴含 funext。

*证明概要*：
1. 对函数 $f, g : A \to B$，逐点相等 $h : \prod_{x:A} f(x) = g(x)$ 定义了一个等价 $e : (\sum_{x:A} B) \simeq (\sum_{x:A} B)$（通过逐点替换）。
2. 由 UA，$e$ 提升为恒等路径 $p : \text{Id}(\sum_{x:A} B, \sum_{x:A} B)$。
3. 该路径在函数空间上的投影给出 $f = g$。∎

## 六、引用来源

1. **Rijke, E.** *Introduction to Homotopy Type Theory*. Cambridge University Press, 2025.
   - HoTT 最新权威教材，本文档符号体系（Π-type、Σ-type、Id-type）主要对齐此书。

2. **The Univalent Foundations Program.** *Homotopy Type Theory: Univalent Foundations of Mathematics*. Institute for Advanced Study, 2013.
   - HoTT 经典参考书（"HoTT Book"），涵盖 UA、HITs 与基础同伦论。

3. **Voevodsky, V.** "The Equivalence Axiom and Univalent Models of Type Theory". arXiv, 2010.
   - 单值公理的原始提出，奠定统一基础的方向。

4. **Cohen, C., Coquand, T., Huber, S., & Mörtberg, A.** "Cubical Type Theory: A Constructive Interpretation of the Univalence Axiom". *TYPES 2015*, 2016.
   - 为 UA 提供计算解释的立方类型论。

5. **Eckmann, B., & Hilton, P. J.** "Group-like structures in general categories". *Mathematische Annalen*, 145(3): 227-255, 1962.
   - Eckmann–Hilton 论证的来源，高阶路径交换性的经典结果。

6. **Angiuli, C., & Gratzer, D.** *Principles of Dependent Type Theory*. Cambridge University Press (forthcoming), 2025.
   - 涵盖依赖类型论、模态类型论与同伦类型论的综合教材。

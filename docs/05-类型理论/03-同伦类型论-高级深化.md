## 1. 泛等公理的完整形式化

### 1.1 标准表述

**泛等公理 (Univalence Axiom)** [Voevodsky 2010]：

对于类型 $A, B : \mathcal{U}$，有：
$$(A =_\mathcal{U} B) \simeq (A \simeq B)$$

其中 $=$ 是恒等类型，$\simeq$ 是等价类型。

### 1.2 等价性展开

**定义 1.1 (等价)**：
$$A \simeq B \equiv \sum_{f: A \to B} isEquiv(f)$$

其中 $isEquiv(f)$ 表示 $f$ 是等价映射（存在拟逆）。

**泛等构造**：

```
ua : (A ≃ B) → (A = B)
ua (f, e) = ...
```

## 2. 结构恒等原理 (SIP)

### 2.1 标准结构

**定理 2.1 (结构恒等原理)** [HoTT Book]：

对于标准数学结构（群、环、拓扑空间等），结构间的等价与结构的恒等等价：
$$(M \equiv_{Str} N) \simeq (M \simeq N)$$

### 2.2 群的例子

**定理 2.2**：两个群 $G, H$ 同构当且仅当它们在类型论中相等。

*证明概要*：

1. 群结构由载体、运算、单位元、逆元、公理组成
2. 泛等公理保证等价结构相等
3. 因此同构群在类型论中等价

## 3. 等价与恒等的等价证明

### 3.1 从恒等到等价

```
id-to-equiv : (A = B) → (A ≃ B)
id-to-equiv refl = id-equiv
```

### 3.2 从等价到恒等

泛等公理提供反向映射 $ua$。

### 3.3 循环证明

**定理 3.1**：$ua$ 和 $id-to-equiv$ 形成等价。

*证明*：

- $id-to-equiv(ua(e)) = e$（通过 $ua$ 的定义）
- $ua(id-to-equiv(p)) = p$（需要泛等）

## 4. 数学重构中的应用

### 4.1 范畴论

**定理 4.1**：等价的范畴在类型论中相等。

**应用**：结构迁移 - 在一个范畴中证明的定理可自动迁移到等价范畴。

### 4.2 代数拓扑

**应用**：同伦等价的空间具有相同的代数不变量。

### 4.3 类型论的数学基础

泛等公理使得类型论成为"结构不变性"的数学基础：

- 同构的结构可互换使用
- 证明可迁移
- 避免重复证明

## 5. 构造性实现

### 5.1 立方类型论中的实现

在Cubical Type Theory中，$ua$ 有计算规则：

```
transport (λ i → ua e @ i) a ≡ e .fst a
```

### 5.2 与其他公理的关系

泛等蕴含：

- 函数外延性
- 命题外延性

但不蕴含：

- 选择公理
- 排中律

## 一、概念定义 (Concept Definition)

### 1.1 依赖类型基础

**定义 1.1.1** (依赖类型 / Dependent Type)

类型可以依赖于值的类型系统。若类型 $T$ 依赖于值 $x: A$，记作 $T(x)$ 或 $B(x): \text{Type}$。

**核心特征**：

- 类型和项在同一层次
- 类型可以包含（依赖）项
- 消除类型和项的严格区分

**定义 1.1.2** (类型族 / Type Family)

类型族是映射：

$$B: A \rightarrow \mathcal{U}$$

其中 $A$ 是类型，$\mathcal{U}$ 是宇宙（universe）。

对任意 $a: A$，$B(a)$ 是一个类型。

**定义 1.1.3** (宇宙层级 / Universe Hierarchy)

为避免悖论，类型分层：

$$\mathcal{U}_0 : \mathcal{U}_1 : \mathcal{U}_2 : ...$$

- $\mathcal{U}_0$：小类型（数据类型）
- $\mathcal{U}_1$：包含 $\mathcal{U}_0$ 的类型
- 依此类推...

累积性：$A: \mathcal{U}_i \Rightarrow A: \mathcal{U}_{i+1}$

---

### 1.2 Π类型（依赖函数类型）

**定义 1.2.1** (Π类型 / Pi Type)

给定类型 $A: \mathcal{U}$ 和类型族 $B: A \rightarrow \mathcal{U}$，定义依赖函数类型：

$$\Pi_{x:A} B(x) : \mathcal{U}$$

**项**：函数 $f: \Pi_{x:A} B(x)$，使得对任意 $a: A$，有 $f(a): B(a)$。

**构造**（λ抽象）：

$$\lambda x.e : \Pi_{x:A} B(x)$$

其中 $e: B(x)$ 假设 $x: A$。

**消去**（函数应用）：

$$f(a) : B(a) \quad \text{当 } f: \Pi_{x:A} B(x), a: A$$

**计算规则**（β归约）：

$$(\lambda x.e)(a) \equiv e[a/x]$$

**定义 1.2.2** (Π类型 vs 普通函数)

| 类型 | 记法 | 返回值类型 |
|------|------|-----------|
| 普通函数 | $A \rightarrow B$ | 固定为 $B$ |
| 依赖函数 | $\Pi_{x:A} B(x)$ | 依赖于输入 $B(x)$ |

特例：当 $B(x) = B$ 不依赖 $x$ 时：

$$\Pi_{x:A} B \equiv A \rightarrow B$$

**定义 1.2.3** (多态函数编码)

参数化多态可通过Π类型编码：

$$\text{id} : \Pi_{A:\mathcal{U}} (A \rightarrow A)$$

即对任意类型 $A$，返回 $A \rightarrow A$ 的恒等函数。

---

### 1.3 Σ类型（依赖对类型）

**定义 1.3.1** (Σ类型 / Sigma Type)

给定类型 $A: \mathcal{U}$ 和类型族 $B: A \rightarrow \mathcal{U}$，定义依赖对类型：

$$\Sigma_{x:A} B(x) : \mathcal{U}$$

**项**：对 $(a, b) : \Sigma_{x:A} B(x)$，其中 $a: A$，$b: B(a)$。

**构造**（配对）：

$$(a, b) : \Sigma_{x:A} B(x) \quad \text{当 } a:A, b:B(a)$$

**消去**（投影）：

- $\text{fst}(p) : A$：第一分量
- $\text{snd}(p) : B(\text{fst}(p))$：第二分量（类型依赖于第一分量）

**计算规则**：

$$\text{fst}(a, b) \equiv a$$
$$\text{snd}(a, b) \equiv b$$

**定义 1.3.2** (Σ类型 vs 普通积)

| 类型 | 记法 | 第二分量类型 |
|------|------|-------------|
| 普通积 | $A \times B$ | 固定为 $B$ |
| 依赖对 | $\Sigma_{x:A} B(x)$ | 依赖于第一分量 $B(\text{fst}(p))$ |

特例：当 $B(x) = B$ 不依赖 $x$ 时：

$$\Sigma_{x:A} B \equiv A \times B$$

**定义 1.3.3** (子集类型 / Subset Type)

Σ类型可编码子集：

$$\{x: A \mid P(x)\} \equiv \Sigma_{x:A} P(x)$$

其中 $P: A \rightarrow \text{Prop}$ 是命题。

**存在量词编码**：

$$\exists x:A. P(x) \equiv \Sigma_{x:A} P(x)$$

---

### 1.4 恒等类型

**定义 1.4.1** (恒等类型 / Identity Type)

给定类型 $A: \mathcal{U}$ 和 $a, b: A$，定义恒等类型：

$$a =_A b : \mathcal{U}$$

或记作 $\text{Id}_A(a, b)$。

**构造**（自反性）：

$$\text{refl}_a : a =_A a$$

**消去**（路径归纳 / J规则）：

$$
\frac{C: \Pi_{x,y:A} (x =_A y) \rightarrow \mathcal{U} \quad c: \Pi_{x:A} C(x, x, \text{refl}_x) \quad p: a =_A b}{J(C, c, a, b, p) : C(a, b, p)}
$$

**计算规则**：

$$J(C, c, a, a, \text{refl}_a) \equiv c(a)$$

**定义 1.4.2** (命题相等 vs 判断相等)

| 相等类型 | 符号 | 含义 |
|----------|------|------|
| 判断相等 | $a \equiv b$ | 定义/计算相等，类型检查器判断 |
| 命题相等 | $a =_A b$ | 需要证明的相等，是类型 |

**定义 1.4.3** (K规则与唯一性)

**K规则**：若 $p, q: a =_A a$，则 $p = q$。

成立时，命题相等退化为布尔值（是/否相等）。

**单值性公理 (UA)**：等价即相等。

## 二、属性 (Properties)

### 2.1 Π类型性质

| 性质 | 公式 | 说明 |
|------|------|------|
| **Currying** | $\Pi_{x:A}\Pi_{y:B(x)} C(x,y) \equiv \Pi_{p:\Sigma_{x:A}B(x)} C(\text{fst}(p), \text{snd}(p))$ | 柯里化等价 |
| **函数外延性** | $(\Pi_{x:A} f(x) = g(x)) \rightarrow f = g$ | 逐点相等推函数相等 |
| **选择公理** | $(\Pi_{x:A}\Sigma_{y:B} C(x,y)) \rightarrow \Sigma_{f:A\rightarrow B}\Pi_{x:A} C(x, f(x))$ | 依赖选择 |
| **β归约** | $(\lambda x.e)(a) \equiv e[a/x]$ | 计算规则 |
| **η展开** | $f \equiv \lambda x.f(x)$ | 外延性 |

### 2.2 Σ类型性质

| 性质 | 公式 | 说明 |
|------|------|------|
| **关联律** | $\Sigma_{x:A}\Sigma_{y:B(x)} C(x,y) \equiv \Sigma_{p:\Sigma_{x:A}B(x)} C(\text{fst}(p), \text{snd}(p))$ | 嵌套可重排 |
| **分配律** | $\Sigma_{x:A}\Pi_{y:B} C(x,y) \rightarrow \Pi_{y:B}\Sigma_{x:A} C(x,y)$ | 存在可分配入全称 |
| **第一投影** | $\text{fst}(a, b) \equiv a$ | 计算规则 |
| **第二投影** | $\text{snd}(a, b) \equiv b$ | 计算规则 |

### 2.3 恒等类型性质

| 性质 | 定义 |
|------|------|
| **自反性** | $\text{refl}_a : a = a$ |
| **对称性** | $\text{sym}(p) : b = a$ 当 $p: a = b$ |
| **传递性** | $\text{trans}(p, q) : a = c$ 当 $p: a = b, q: b = c$ |
| **替换** | $\text{transport}^B(p, b) : B(a)$ 当 $p: x = a, b: B(x)$ |
| **同伦层级** | $a = b$ 的相等是更高阶的相等 |

### 2.4 Curry-Howard对应扩展

| 逻辑 | 类型 | 依赖类型扩展 |
|------|------|-------------|
| 命题 $A \rightarrow B$ | 函数 $A \rightarrow B$ | Π类型编码全称量词 |
| 命题 $A \land B$ | 积类型 $A \times B$ | Σ类型编码存在量词 |
| 命题 $\forall x:A. B(x)$ | Π类型 $\Pi_{x:A} B(x)$ | 依赖函数 |
| 命题 $\exists x:A. B(x)$ | Σ类型 $\Sigma_{x:A} B(x)$ | 依赖对 |
| 命题 $a =_A b$ | 恒等类型 $\text{Id}_A(a,b)$ | 相等性证明 |

## 四、解释 (Explanation)

### 4.1 依赖类型的直觉

**类型即规范**

依赖类型允许类型包含更精确的规范：

```
// 普通类型：只能保证返回向量
Vector sort(Vector v);

// 依赖类型：保证返回排序后的向量
sort : Π(v: Vector A) → SortedVector v
```

**证明即程序**

- 类型 $P$ 的项是 $P$ 的证明
- 构造证明 = 构造程序
- 类型检查 = 证明验证

### 4.2 Π类型的直觉

**通用函数**

Π类型表示"对任意输入，返回适当类型的输出"：

```
长度函数: Π(A: Type) → (List A → Nat)

对任意类型 A：
- 输入：List A（A 的列表）
- 输出：Nat（自然数，长度）
```

**多维数组类型**

```
Matrix : Nat → Nat → Type  -- n×m 矩阵

identity : Π(n: Nat) → Matrix n n  -- 单位矩阵，类型依赖于维度
```

### 4.3 Σ类型的直觉

**打包信息与证据**

Σ类型将值与依赖于该值的证明打包：

```
排序向量 = Σ(v: Vector A) (isSorted v = true)

包含：
- v: 向量（数据）
- 证明 v 已排序（证据）
```

**存在量词的构造性解释**

存在量词 $\exists x:A. P(x)$ 需要：

1. 具体的 $a: A$（见证）
2. $P(a)$ 的证明

这与经典逻辑不同（经典逻辑允许非构造性存在证明）。

### 4.4 恒等类型的直觉

**相等性作为结构**

在传统类型论中，相等是外部概念（判断）。

在同伦类型论中，相等是内部类型，可以有不同的"证明"（路径）：

```
在圆 S¹ 中：
- 点 base : S¹
- 路径 loop : base = base

loop 不是 refl，是"绕一圈"的相等性证明
```

**路径与等价**

- $p: a = b$ 是从 $a$ 到 $b$ 的"路径"
- 路径可以连接（传递性）
- 路径可以反转（对称性）

## 五、论证 (Argumentation)

### 5.1 依赖类型的表达力

**完全规范**

依赖类型可以表达完整的程序规范：

```
带借位的减法：
subtract : Π(m n: Nat) → (m ≥ n = true) → Σ(k: Nat) (m = n + k)

输入：
- m, n: 自然数
- 证明 m ≥ n
输出：
- k: 差值
- 证明 m = n + k
```

**零运行时开销**

- 证明部分在编译时消除
- 运行时只保留计算内容

### 5.2 证明助手的设计哲学

**Coq/Lean 核心设计**：

1. **构造性逻辑**：所有证明都是构造性的，可提取程序
2. **计算化简**：证明中的计算步骤自动简化
3. **策略语言**：交互式构建证明
4. **反射**：在类型系统内部证明元定理

**工业应用**：

- CompCert（经证明正确的C编译器）
- seL4（经证明正确的操作系统内核）
- Fiat-Crypto（经证明正确的密码学原语）

### 5.3 同伦类型论的革命

**从相等性到空间**

同伦类型论将类型解释为空间：

- 类型 $A$ = 拓扑空间
- 项 $a: A$ = 点
- $p: \text{Id}_A(a, b)$ = 从 $a$ 到 $b$ 的路径（Rijke 2025 标准 Id-type 记法）
- $p = q$ = 路径之间的同伦

**单值性公理 (UA)** [Rijke2025, §2.4]

$$(A \simeq B) \simeq \text{Id}_\mathcal{U}(A, B)$$

结构等价（同构）就是相等。

**意义**：

- 同构的数据结构可互换使用
- 解决了数学结构"相等"的哲学问题
- 统一了数学基础和计算机形式化

## 六、形式证明 (Formal Proofs)

### 6.1 Π类型β归约保持类型

**定理**：若 $\Gamma, x:A \vdash e: B(x)$ 且 $\Gamma \vdash a: A$，则：

$$(\lambda x.e)(a) : B(a)$$

**证明**：

**Step 1**：构造λ抽象

由假设 $\Gamma, x:A \vdash e: B(x)$，应用Π引入：

$$\Gamma \vdash \lambda x.e : \Pi_{x:A} B(x)$$

**Step 2**：函数应用

由 $a: A$ 和 $\lambda x.e : \Pi_{x:A} B(x)$，应用Π消去：

$$(\lambda x.e)(a) : B(a)$$

**Step 3**：计算

由β归约：

$$(\lambda x.e)(a) \equiv e[a/x]$$

且 $e[a/x] : B(a)$（代入引理）。

∎

### 6.2 Σ类型投影计算规则

**定理**：若 $a: A$，$b: B(a)$，则：

1. $\text{fst}(a, b) \equiv a : A$
2. $\text{snd}(a, b) \equiv b : B(a)$

**证明**：

由Σ类型的构造规则，$(a, b) : \Sigma_{x:A} B(x)$。

由Σ消去规则定义：

- $\text{fst}$ 被定义为返回配对的第一分量
- $\text{snd}$ 被定义为返回配对的第二分量

由计算规则，这些定义直接给出：

$$\text{fst}(a, b) \equiv a$$
$$\text{snd}(a, b) \equiv b$$

类型检查：

- $\text{fst}(a, b) : A$ ✓
- $\text{snd}(a, b) : B(\text{fst}(a, b)) \equiv B(a)$ ✓

∎

### 6.3 路径归纳原理（J规则）证明对称性

**定理**：对任意 $A: \mathcal{U}$，$a, b: A$，$p: a =_A b$，存在 $\text{sym}(p) : b =_A a$。

**证明**：

定义 $C(x, y, q) := (y =_A x)$。

定义 $c(x) := \text{refl}_x : C(x, x, \text{refl}_x) = (x =_A x)$。

应用J规则：

$$J(C, c, a, b, p) : C(a, b, p) = (b =_A a)$$

定义 $\text{sym}(p) := J(C, c, a, b, p)$。

∎

### 6.4 函数外延性构造

**定理**：若 $f, g: \Pi_{x:A} B(x)$ 且 $\Pi_{x:A} (f(x) = g(x))$，则 $f = g$。

**证明**（假设函数外延性公理）：

**Step 1**：逐点相等假设

假设 $h : \Pi_{x:A} (f(x) =_{B(x)} g(x))$

**Step 2**： funext 公理

函数外延性公理声明：

$$\text{funext} : \left(\Pi_{x:A} f(x) = g(x)\right) \rightarrow (f = g)$$

**Step 3**：应用

$$\text{funext}(h) : f = g$$

**注**：在同伦类型论中，函数外延性可从单值性公理导出 [Rijke2025, §2.4]。

∎

## 七、应用与实现

### 7.1 带依赖类型的向量实现

```rust
/// 伪代码：依赖类型风格的向量（使用Rust类型系统近似）

/// 编译时常量自然数（类型级别）
pub struct Z;
pub struct S<N>(PhantomData<N>);

/// 长度索引向量
type Vector<T, N> = Vec<T>; // 运行时检查长度

/// 类型安全的 head 操作（需要 N > 0）
pub fn head<T, N>(v: Vector<T, S<N>>) -> T {
    v.into_iter().next().unwrap()
}

/// 类型安全的 append（长度相加）
pub fn append<T, N, M>(
    v1: Vector<T, N>,
    v2: Vector<T, M>
) -> Vector<T, Add<N, M>> {
    let mut result = v1;
    result.extend(v2);
    result
}

/// 类型安全的 zip（长度相同）
pub fn zip<T, U, N>(
    v1: Vector<T, N>,
    v2: Vector<U, N>
) -> Vector<(T, U), N> {
    v1.into_iter().zip(v2).collect()
}
```

### 7.3 Π和Σ类型的逻辑编码

```rust
/// 依赖类型的逻辑编码（伪代码）

/// Π类型：全称量词 ∀x:A. B(x)
/// 表示为函数类型，接受任意 x:A，返回 B(x) 的证明
type ForAll<A, B: Fn(A) -> Type> = dyn Fn(A) -> B::Output;

/// Σ类型：存在量词 ∃x:A. B(x)
/// 表示为对类型，包含 x:A 和 B(x) 的证明
type Exists<A, B: Fn(A) -> Type> = (A, B::Output);

/// 恒等类型 a = b
type Equals<A, a: A, b: A> = Proof<a == b>;

/// 选择公理的证明
fn choice<A, B, C>(
    proof: impl Fn(A) -> (B, C)
) -> (impl Fn(A) -> B, impl Fn(A) -> C) {
    let f = |a: A| (proof(a).0);
    let g = |a: A| (proof(a).1);
    (f, g)
}

/// 函数外延性
fn funext<A, B, F, G>(
    f: F,
    g: G,
    pointwise_eq: impl Fn(A) -> Proof<F(A) = G(A)>
) -> Proof<F = G> {
```

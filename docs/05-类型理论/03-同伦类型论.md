## 1. 基本概念 (Basic Concepts)

### 1.1 同伦类型论定义

同伦类型论是类型论与同伦论的结合，将类型视为空间，将类型等价视为同伦等价 [2]。按 Rijke (2025) 的标准记法，依赖函数类型记作 $\Pi_{x:A} B(x)$，依赖对类型记作 $\Sigma_{x:A} B(x)$，恒等类型记作 $\text{Id}_A(a,b)$（或简写为 $a =_A b$）。

**核心思想**:

1. **类型即空间**: 类型对应拓扑空间，项对应空间中的点
2. **等价即路径**: 类型等价对应同伦等价，证明对应路径
3. **高阶结构**: 高阶归纳类型、同伦群

### 1.2 基本公理

**Voevodsky公理** [1]:

1. **函数外延性 (Function Extensionality)**:
   $$\text{funext}: \prod_{f,g: A \rightarrow B} \left(\prod_{x:A} \text{Id}_B(f(x), g(x))\right) \rightarrow \text{Id}_{A \rightarrow B}(f, g)$$
2. **单值公理 (Univalence Axiom)**:
   $$(A \simeq B) \simeq \text{Id}_\mathcal{U}(A, B)$$
   其中 $\mathcal{U}$ 为宇宙类型，$\text{Id}_\mathcal{U}$ 为宇宙层面的恒等类型。

## 2. 同伦类型 (Homotopy Types)

### 2.1 类型作为空间

- **点类型**: $\text{Unit}$ 对应单点空间
- **路径类型**: $a =_A b$ 对应从 $a$ 到 $b$ 的路径空间

### 2.2 路径操作

- **路径连接**: $p \cdot q: a =_A c$
- **路径反转**: $p^{-1}: b =_A a$
- **路径恒等**: $\text{refl}_a: a =_A a$

### 2.3 同伦群

**基本群**:
$$\pi_1(A, a) = (a =_A a, \text{refl}_a)$$

**高阶同伦群**:
$$\pi_n(A, a) = \pi_1(\Omega^{n-1}(A, a), \text{refl}_a)$$

### 2.4 同伦水平 (Homotopy Levels) [1]

同伦水平（$h$-level）刻画了类型所含高阶路径信息的丰富程度，是 HoTT 中对类型进行分层的基本工具：

- **$h$-level 0**（可缩类型, contractible）：$\text{isContr}(A) \equiv \Sigma_{a:A} \prod_{x:A} \text{Id}_A(a, x)$
- **$h$-level 1**（命题, proposition）：$\text{isProp}(A) \equiv \prod_{x,y:A} \text{isContr}(\text{Id}_A(x, y))$
- **$h$-level 2**（集合, set）：$\text{isSet}(A) \equiv \prod_{x,y:A} \text{isProp}(\text{Id}_A(x, y))$
- **$h$-level $n+1$**：$\text{isLevel}(n+1, A) \equiv \prod_{x,y:A} \text{isLevel}(n, \text{Id}_A(x, y))$

**意义**：单值公理保证命题外延性，且将等价提升到恒等路径的层次；在 h-level 2（集合层）上，HoTT 退化为传统的基于集合的数学基础 [2]。

## 3. 类型等价 (Type Equivalence)

### 3.1 等价定义

类型 $A$ 和 $B$ 等价 [1]：
$$A \simeq B \equiv \sum_{f: A \rightarrow B} \text{isEquiv}(f)$$
其中 $\text{isEquiv}(f)$ 表示 $f$ 具有拟逆（或等价地，$f$ 的纤维均可缩）：
$$\text{isEquiv}(f) \equiv \sum_{g: B \rightarrow A} \left(\prod_{a:A} \text{Id}_A(g(f(a)), a)\right) \times \left(\prod_{b:B} \text{Id}_B(f(g(b)), b)\right)$$

### 3.2 等价性质

- **自反性**: $A \simeq A$
- **对称性**: $A \simeq B \rightarrow B \simeq A$
- **传递性**: $A \simeq B \land B \simeq C \rightarrow A \simeq C$

---

## 4. 高阶归纳类型 (Higher Inductive Types)

### 4.1 定义

高阶归纳类型同时包含构造函数和路径构造器。

### 4.2 圆 (Circle) $S^1$

- **基点**: $\text{base}: S^1$
- **环路**: $\text{loop}: \text{base} =_{S^1} \text{base}$
- **基本群**: $\pi_1(S^1, \text{base}) \cong \mathbb{Z}$

### 4.3 球面 (Sphere) $S^n$

$$S^n = \text{Susp}(S^{n-1})$$

**性质**:

- $\pi_n(S^n) \cong \mathbb{Z}$
- $\pi_k(S^n) = 0$ for $k < n$

### 4.4 环面 (Torus) $T^2$

- 两个环路 $\text{loop}_1, \text{loop}_2$
- 交换关系: $\text{loop}_1 \cdot \text{loop}_2 =_{T^2} \text{loop}_2 \cdot \text{loop}_1$

---

## 5. 高阶路径理论 (Higher-Order Path Theory)

### 5.1 高阶路径定义

**定义** (高阶路径)
$$\text{Path}_0(a, b) = a =_A b$$
$$\text{Path}_{n+1}(p, q) = p =_{\text{Path}_n(a, b)} q$$

### 5.2 高阶路径的群结构

**定理** (高阶路径群)
对于 $n \geq 1$，$\text{Path}_n(a, a)$ 形成群。

### 5.3 高阶路径的交换性

**定理** (Eckmann-Hilton定理)
对于 $n \geq 3$，$n$-路径群是交换的。

---

## 6. 同伦群高级理论 (Advanced Homotopy Group Theory)

### 6.1 同伦群序列

**定义** (同伦群)
$$\pi_n(X, x_0) = [S^n, X]_{x_0}$$

### 6.2 长正合序列

**定理** (长正合序列)
设 $F \to E \to B$ 为纤维丛，则存在长正合序列：
$$\cdots \to \pi_n(F) \to \pi_n(E) \to \pi_n(B) \to \pi_{n-1}(F) \to \cdots$$

### 6.3 Hurewicz定理

**定理** (Hurewicz定理)
设 $X$ 为单连通空间：

1. 如果 $\pi_i(X) = 0$ 对所有 $i < n$，则 $H_i(X) = 0$ 对所有 $i < n$
2. 如果 $n \geq 2$，则 $H_n(X) \cong \pi_n(X)$

```coq
(* 同伦类型论基础 *)
Axiom univalence : forall A B : Type, (A ≃ B) ≃ (A = B).

(* 高阶归纳类型：圆 *)
Inductive S1 : Type :=
| base : S1
| loop : base = base.

(* 路径操作 *)
Definition path_concatenation {A : Type} {a b c : A}
  (p : a = b) (q : b = c) : a = c.
```

### 7.2 Lean 4 证明示例

以下代码使用 Lean 4 核心库展示 HoTT 基础概念。完整的 Univalence Axiom 与 HITs 在 Lean 4 中需要额外库支持；
可运行代码见 [`examples/lean_proofs/hott_basics.lean`](../../examples/lean_proofs/hott_basics.lean)。

```lean
-- 基于 Lean 4 核心 Eq 类型的路径操作示例
universe u
variable {A : Type u}

def transport {P : A → Type u} {x y : A} (p : x = y) (u : P x) : P y :=
  p ▸ u

def apd {P : A → Type u} (f : (a : A) → P a) {x y : A} (p : x = y)
    : transport p (f x) = f y := by
  cases p
  rfl
```

### 7.3 Agda证明

```agda
-- 同伦类型论公理
postulate
  univalence : {A B : Set} → (A ≃ B) ≃ (A ≡ B)

-- 高阶归纳类型
data S¹ : Set where
  base : S¹
  loop : base ≡ base

-- 高阶路径
HigherPath : {A : Set} {a b : A} → (n : ℕ) → Set
HigherPath {A} {a} {b} zero = a ≡ b
HigherPath {A} {a} {b} (suc n) = {p q : HigherPath n} → p ≡ q
```

## 1. 泛等公理的完整形式化

### 1.1 标准表述

**泛等公理 (Univalence Axiom)** [19; 1, §2.4]：

对于类型 $A, B : \mathcal{U}$，有：
$$\text{Id}_\mathcal{U}(A, B) \simeq (A \simeq B)$$

其中 $\text{Id}_\mathcal{U}$ 是恒等类型（记法对齐 Rijke 2025 的 Id-type），$\simeq$ 是等价类型。

### 1.2 等价性展开

**定义 1.1 (等价)** [1]：
$$A \simeq B \equiv \sum_{f: A \to B} \text{isEquiv}(f)$$

其中 $\text{isEquiv}(f)$ 表示 $f$ 是等价映射（存在拟逆）。此处 $\Sigma$ 为依赖对类型（Σ-type）。

**泛等构造**：

```
ua : (A ≃ B) → Id_𝒰(A, B)
ua (f, e) = ...
```

## 2. 结构恒等原理 (SIP)

### 2.1 标准结构

**定理 2.1 (结构恒等原理)** [2; 1, §12]：

对于标准数学结构（群、环、拓扑空间等），结构间的等价与结构的恒等等价：
$$(M \equiv_{Str} N) \simeq (M \simeq N)$$

### 2.2 群的例子

**定理 2.2**：两个群 $G, H$ 同构当且仅当它们在类型论中相等。

*证明概要*：

1. 群结构由载体、运算、单位元、逆元、公理组成
2. 泛等公理保证等价结构相等
3. 因此同构群在类型论中等价

## 3. 等价与恒等的等价证明

### 3.1 从恒等到等价

```
id-to-equiv : (A = B) → (A ≃ B)
id-to-equiv refl = id-equiv
```

### 3.2 从等价到恒等

泛等公理提供反向映射 $ua$。

### 3.3 循环证明

**定理 3.1**：$ua$ 和 $id-to-equiv$ 形成等价。

*证明*：

- $id-to-equiv(ua(e)) = e$（通过 $ua$ 的定义）
- $ua(id-to-equiv(p)) = p$（需要泛等）

## 4. 数学重构中的应用

### 4.1 范畴论

**定理 4.1**：等价的范畴在类型论中相等。

**应用**：结构迁移 - 在一个范畴中证明的定理可自动迁移到等价范畴。

### 4.2 代数拓扑

**应用**：同伦等价的空间具有相同的代数不变量。

### 4.3 类型论的数学基础

泛等公理使得类型论成为"结构不变性"的数学基础：

- 同构的结构可互换使用
- 证明可迁移
- 避免重复证明

## 5. 构造性实现

### 5.1 立方类型论中的实现

在 Cubical Type Theory 中，$ua$ 具有计算规则 [4]：

```
transport (λ i → ua e @ i) a ≡ e .fst a
```

### 5.2 与其他公理的关系

泛等蕴含：

- 函数外延性
- 命题外延性

但不蕴含：

- 选择公理
- 排中律

1. **证明**：证明 $ua(id-equiv) = refl$。
2. **应用**：为半群结构构造SIP证明。
3. **研究**：了解类型论中宇宙层级与泛等的关系。

## 6.5 2024-2025最新研究进展 / Recent Research Progress (2024-2025)

### 6.5.1 单纯类型论进展（Simplicial Type Theory）

Riehl 与 Shulman 在 2017 年提出了**单纯类型论**（Simplicial Type Theory, sHoTT），作为处理 (∞,1)-范畴和定向同伦结构的类型论框架 [8]。2024 年，Gratzer、Weinberger 与 Buchholtz 证明了**定向单值性**（Directed Univalence）在单纯同伦类型论中的可实现性 [9]，为在类型论内部形式化高阶范畴论奠定了关键基础。

**核心思想 / Core Idea:**

单纯类型论通过引入**形状层**（shape layer）来区分“空间”与“类型”：

- **空间（Space）**：表示单纯形，用于建模高维路径和定向映射。
- **类型（Type）**：带有与空间交互的泛型结构，支持外延和内涵两种层次。

2024-2025 年的进一步工作将该框架扩展到**准范畴**（quasicategories）和**Rezk 完备性**（Rezk completeness）的形式化，使得在证明助手中验证 (∞,1)-范畴论定理成为可能。

### 6.5.2 同伦类型论的新语义模型

2024-2025 年，HoTT 的语义模型研究在多个方向上取得突破：

**高阶拓扑语义 / Higher-Topos Semantics:**

- **浓缩数学（Condensed Mathematics）**与类型论的交叉：研究者探索了如何在 HoTT 中编码凝聚态/浓缩结构，为代数几何和泛函分析的形式化提供新工具 [10]。
- **综合微分几何（Synthetic Differential Geometry, SDG）**的 HoTT 扩展：通过引入幂零无穷小量，将光滑同伦论整合到类型论框架中 [10]。

**模型范畴与类型论的对应 / Model Categories and Type Theory:**

- 使用**部落范畴**（tribe categories）和**Cisinski 模型结构**建立了依赖类型与 (∞,1)-范畴的更精确对应。
- 2025 年 LICS 上的相关论文进一步证明了在 Grothendieck (∞,1)-topos 中构造性单值模型的存在性 [18]。

### 6.5.3 计算解释与规范化算法进展

HoTT 的计算解释是其实用化的核心障碍之一。立方类型论（Cubical Type Theory）已经提供了重要的构造性路径，但 2024-2025 年在**规范化算法**和**效率优化**方面取得了新进展：

**规范化 by 求值（Normalization by Evaluation, NbE）:**

- 针对立方类型论的 NbE 算法在 Agda 和 cooltt 中得到了进一步优化，使得复杂的高阶路径证明能够在合理时间内完成类型检查 [11]。
- 新的**局部 NbE**策略通过延迟展开（delayed unfolding）减少了不必要的计算步骤。

**Glue 类型与 transp 优化:**

- 2024 年的研究表明，通过引入**收缩的 Glue 类型**（collapsed Glue types）可以显著降低 univalence 操作的运行时代价 [11]。
- transp（运输）操作在 h-level 2（集合层）上的特化优化已被集成到 Cubical Agda 的最新开发分支中。

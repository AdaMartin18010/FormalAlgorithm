## 1. 基本概念 (Basic Concepts)

### 1.1 同伦类型论定义

同伦类型论是类型论与同伦论的结合，将类型视为空间，将类型等价视为同伦等价 [HoTTBook2013]。

**核心思想**:
1. **类型即空间**: 类型对应拓扑空间，项对应空间中的点
2. **等价即路径**: 类型等价对应同伦等价，证明对应路径
3. **高阶结构**: 高阶归纳类型、同伦群

### 1.2 基本公理

**Voevodsky公理**:
1. **函数外延性**: $\text{funext}: \prod_{f,g: A \rightarrow B} (\prod_{x:A} f(x) = g(x)) \rightarrow f = g$
2. **单值公理**: $(A \simeq B) \rightarrow (A = B)$

## 2. 同伦类型 (Homotopy Types)

### 2.1 类型作为空间

- **点类型**: $\text{Unit}$ 对应单点空间
- **路径类型**: $a =_A b$ 对应从 $a$ 到 $b$ 的路径空间

### 2.2 路径操作

- **路径连接**: $p \cdot q: a =_A c$
- **路径反转**: $p^{-1}: b =_A a$
- **路径恒等**: $\text{refl}_a: a =_A a$

### 2.3 同伦群

**基本群**:
$$\pi_1(A, a) = (a =_A a, \text{refl}_a)$$

**高阶同伦群**:
$$\pi_n(A, a) = \pi_1(\Omega^{n-1}(A, a), \text{refl}_a)$$

---

## 3. 类型等价 (Type Equivalence)

### 3.1 等价定义

类型 $A$ 和 $B$ 等价：
$$A \simeq B = \sum_{f: A \rightarrow B} \sum_{g: B \rightarrow A} \left(\prod_{a:A} g(f(a)) =_A a\right) \times \left(\prod_{b:B} f(g(b)) =_B b\right)$$

### 3.2 等价性质

- **自反性**: $A \simeq A$
- **对称性**: $A \simeq B \rightarrow B \simeq A$
- **传递性**: $A \simeq B \land B \simeq C \rightarrow A \simeq C$

---

## 4. 高阶归纳类型 (Higher Inductive Types)

### 4.1 定义

高阶归纳类型同时包含构造函数和路径构造器。

### 4.2 圆 (Circle) $S^1$

- **基点**: $\text{base}: S^1$
- **环路**: $\text{loop}: \text{base} =_{S^1} \text{base}$
- **基本群**: $\pi_1(S^1, \text{base}) \cong \mathbb{Z}$

### 4.3 球面 (Sphere) $S^n$

$$S^n = \text{Susp}(S^{n-1})$$

**性质**:
- $\pi_n(S^n) \cong \mathbb{Z}$
- $\pi_k(S^n) = 0$ for $k < n$

### 4.4 环面 (Torus) $T^2$

- 两个环路 $\text{loop}_1, \text{loop}_2$
- 交换关系: $\text{loop}_1 \cdot \text{loop}_2 =_{T^2} \text{loop}_2 \cdot \text{loop}_1$

---

## 5. 高阶路径理论 (Higher-Order Path Theory)

### 5.1 高阶路径定义

**定义** (高阶路径)
$$\text{Path}_0(a, b) = a =_A b$$
$$\text{Path}_{n+1}(p, q) = p =_{\text{Path}_n(a, b)} q$$

### 5.2 高阶路径的群结构

**定理** (高阶路径群)
对于 $n \geq 1$，$\text{Path}_n(a, a)$ 形成群。

### 5.3 高阶路径的交换性

**定理** (Eckmann-Hilton定理)
对于 $n \geq 3$，$n$-路径群是交换的。

---

## 6. 同伦群高级理论 (Advanced Homotopy Group Theory)

### 6.1 同伦群序列

**定义** (同伦群)
$$\pi_n(X, x_0) = [S^n, X]_{x_0}$$

### 6.2 长正合序列

**定理** (长正合序列)
设 $F \to E \to B$ 为纤维丛，则存在长正合序列：
$$\cdots \to \pi_n(F) \to \pi_n(E) \to \pi_n(B) \to \pi_{n-1}(F) \to \cdots$$

### 6.3 Hurewicz定理

**定理** (Hurewicz定理)
设 $X$ 为单连通空间：
1. 如果 $\pi_i(X) = 0$ 对所有 $i < n$，则 $H_i(X) = 0$ 对所有 $i < n$
2. 如果 $n \geq 2$，则 $H_n(X) \cong \pi_n(X)$

## 7. 形式化证明系统 (Formal Proof Systems)

### 7.1 Coq证明

```coq
(* 同伦类型论基础 *)
Axiom univalence : forall A B : Type, (A ≃ B) ≃ (A = B).

(* 高阶归纳类型：圆 *)
Inductive S1 : Type :=
| base : S1
| loop : base = base.

(* 路径操作 *)
Definition path_concatenation {A : Type} {a b c : A}
  (p : a = b) (q : b = c) : a = c.
```

### 7.2 Lean证明

```lean
-- 同伦类型论公理
axiom univalence : Π {A B : Type}, (A ≃ B) ≃ (A = B)

-- 高阶归纳类型
inductive S¹ : Type
| base : S¹
| loop : base = base

-- 同伦群
def homotopy_group (X : Type*) (n : ℕ) : Type :=
  match n with
  | 0 := X
  | n + 1 := Π (S : Type), is_sphere n S → (S → X)
  end
```

### 7.3 Agda证明

```agda
-- 同伦类型论公理
postulate
  univalence : {A B : Set} → (A ≃ B) ≃ (A ≡ B)

-- 高阶归纳类型
data S¹ : Set where
  base : S¹
  loop : base ≡ base

-- 高阶路径
HigherPath : {A : Set} {a b : A} → (n : ℕ) → Set
HigherPath {A} {a} {b} zero = a ≡ b
HigherPath {A} {a} {b} (suc n) = {p q : HigherPath n} → p ≡ q
```

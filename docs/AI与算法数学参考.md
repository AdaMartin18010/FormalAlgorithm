## 一、说明

本项目以**算法规范与模型设计**为核心，数学以「服务算法与形式化」为主，不单独建数学教材模块。本文档提供：

## 二、AI 数学专线与项目映射

> 本节按照“AI 数学专线”组织内容，将传统数学按**面向算法与 ML 的用途**拆分为若干模块。

### 2.1 核心模块总览（面向工程师）

| 模块 | 主要内容（选重点，不做全数学教材） | 典型 AI/算法应用 |
|------|------------------------------------|------------------|
| **线性代数 for ML** | 向量空间、线性映射、矩阵分解（特征分解、SVD）、正定矩阵、谱理论 | 表示学习、PCA/降维、注意力与 Transformer、图神经网络、数值优化 |
| **概率与统计 for ML** | 概率分布、极限定理、参数估计、假设检验、贝叶斯推断、信息量与互信息 | 随机算法分析、泛化误差界、贝叶斯学习、置信区间、A/B 实验、信息瓶颈 |
| **优化理论 for ML** | 凸集与凸函数、对偶与 KKT 条件、一阶/二阶方法、约束优化、鞍点与正则化 | 深度学习训练、对偶视角下的 SVM/最大熵模型、拉格朗日松弛、对抗训练 |
| **测度论与泛函分析 for 高阶 AI** | 测度空间、积分、Lp 空间、Banach/Hilbert 空间、算子谱理论 | 严格概率论基础、核方法与再生核 Hilbert 空间、高斯过程、连续时间与无限维模型 |
| **数值分析与微分方程** | 数值线性代数、迭代求解器、误差与稳定性、ODE/PDE 数值解法 | 神经常微分方程、PINNs、优化中的线性系统求解、大规模训练的数值稳定性分析 |
| **几何与信息几何** | 流形、黎曼度量、测地线、曲率、Fisher 信息度量、指数族与对偶坐标 | 自然梯度、几何深度学习（图/流形）、超曲空间嵌入、变分推断与指数族模型 |

#### 2.1.1 模块结构化说明（工程师视角）

> 下表只给出「工程师所需最小集」，更高阶内容在各模块中作为**进阶**或**研究向**阅读。

- **线性代数 for ML**
  - **工程师最小知识集**：向量空间与线性映射、矩阵乘法与秩、特征值/特征向量、SVD 与低秩近似、正定矩阵与二次型。
  - **对应外部课程章节**：MML Ch.2–4；CMU 10‑606 线性代数单元；Coursera *Mathematics for ML and Data Science* 线性代数课程；Coursera *Foundational Mathematics for AI* 线性代数部分。
  - **对应仓库文档**：`01-基础理论/02-数学基础.md` 中线性代数部分、`01-基础理论/06-代数结构基础.md` 的向量空间/线性映射段落、`09-算法理论/03-优化理论/*` 中与线性算子相关的内容。

- **概率与统计 for ML**
  - **工程师最小知识集**：随机变量与分布、期望/方差/协方差、常见分布族（伯努利、二项、正态等）、极限定理直观、点估计与区间估计、假设检验、简单贝叶斯更新。
  - **对应外部课程章节**：MML 概率章节；CMU 10‑606 概率与统计单元；Coursera *Mathematics for ML and Data Science* 概率与统计课程；Coursera *Foundational Mathematics for AI* 概率模块。
  - **对应仓库文档**：`01-基础理论/07-概率与统计基础.md`、`01-基础理论/08-信息论基础.md`、`09-算法理论/01-算法基础/11-随机算法理论.md`、`09-算法理论/01-算法基础/12-近似算法理论.md`。

- **优化理论 for ML**
  - **工程师最小知识集**：凸集与凸函数的基本概念、一阶/二阶可导与梯度/ Hessian、无约束优化的一阶最优性条件、简单约束优化与 KKT 条件、梯度下降及常见变体（动量、Adam 等）。
  - **对应外部课程章节**：MML 优化章节；Boyd & Vandenberghe 凸优化入门章节；CMU 10‑606 中关于梯度法与优化的讲次；部分 CS161/CS170/15‑451 讲义中的优化视角。
  - **对应仓库文档**：`09-算法理论/03-优化理论/*`、`04-算法复杂度/06-信息论下界.md`（在下界分析中涉及优化与信息论工具）、`10-高级主题` 中与深度学习训练相关的文档。

- **测度论与泛函分析 for 高阶 AI**
  - **工程师最小知识集**：σ‑代数与测度的直观、可测函数与积分、概率测度与分布函数、Lp 空间的基本性质、Hilbert 空间中正交与投影的直观。
  - **对应外部课程章节**：标准测度论教材（Billingsley *Probability and Measure*、Williams *Probability with Martingales*）的基础部分；与核方法/高斯过程相关章节可作为进阶。
  - **对应仓库文档**：`01-基础理论/11-测度论基础.md`、`01-基础理论/07-概率与统计基础.md` 中测度化概率部分、未来在 `10-高级主题` 中与核方法/GP/连续模型相关的文档。

- **数值分析与微分方程**
  - **工程师最小知识集**：浮点数与数值误差的概念、线性方程组的直接与迭代解法（如共轭梯度）、简单 ODE 数值解（Euler、Runge–Kutta）的直观、稳定性与收敛性的基本判断。
  - **对应外部课程章节**：Trefethen & Bau *Numerical Linear Algebra*；Quarteroni 等数值分析教材的入门章节；与神经 ODE/PINNs 相关的综述或教程。
  - **对应仓库文档**：`01-基础理论/02-数学基础.md` 中微积分与线性代数部分、`09-算法理论/03-优化理论/*` 中关于迭代求解器的内容、`10-高级主题` 中神经微分方程与科学计算相关文档（随项目演进补齐）。

- **几何与信息几何**
  - **工程师最小知识集**：流形与坐标图的直观、黎曼度量与内积、测地线的概念、指数族与对数分区函数、Fisher 信息及其作为度量张量的直观。
  - **对应外部课程章节**：信息几何（Amari 等）、流形优化教材中的入门章节；与几何深度学习相关的综述。
  - **对应仓库文档**：`01-基础理论/02-数学基础.md` 中几何与多元微积分相关部分、`01-基础理论/09-序论基础.md` 中序与格的结构视角、`10-高级主题` 中几何深度学习与信息几何相关文档（随后续扩展陆续完善）。

### 2.3 各模块的工程师必修 / 进阶与文档映射

> 每个模块都按照「工程师所需的最小知识集 → 建议进阶 → 本仓库文档映射」给出简明清单，便于在学习路径中引用。

- **线性代数 for ML**
  - 工程师最小知识集：向量与矩阵运算、线性无关与基、矩阵秩与逆、特征值分解、SVD 与最小二乘。
  - 建议进阶：正定矩阵与谱理论、矩阵范数与条件数、张量与高阶分解。
  - 本仓库映射：`01-基础理论/02-数学基础.md`（线性代数部分）、`01-基础理论/06-代数结构基础.md`、`09-算法理论/03-优化理论/*`、`10-高级主题` 中线性代数前置说明。

- **概率与统计 for ML**
  - 工程师最小知识集：随机变量与分布族、期望与方差、极限定理雏形（大数律/中心极限定理直观版）、参数估计与置信区间、简单假设检验。
  - 建议进阶：贝叶斯推断与共轭先验、信息量与互信息、统计学习理论中的泛化误差界。
  - 本仓库映射：`01-基础理论/07-概率与统计基础.md`、`01-基础理论/08-信息论基础.md`、`09-算法理论/01-算法基础/11-随机算法理论.md`、`09-算法理论/01-算法基础/12-近似算法理论.md`。

- **优化理论 for ML**
  - 工程师最小知识集：一元/多元函数极值、梯度与 Hessian、凸集与凸函数直观、梯度下降与变体（SGD、动量、Adam）。
  - 建议进阶：KKT 条件与拉格朗日对偶、投影梯度与约束优化、鞍点与非凸优化景观。
  - 本仓库映射：`01-基础理论/02-数学基础.md`（微积分与优化部分）、`09-算法理论/03-优化理论/*`、`04-算法复杂度/06-信息论下界.md` 中基于优化/信息论的下界分析。

- **测度论与泛函分析 for 高阶 AI**
  - 工程师最小知识集：σ-代数、测度与可测函数的直观理解、积分作为“极限和”的统一框架、Lp 空间与内积空间雏形。
  - 建议进阶：Banach/Hilbert 空间、算子谱理论、核方法与再生核 Hilbert 空间、Gaussian 过程的测度论表述。
  - 本仓库映射：`01-基础理论/11-测度论基础.md`、`01-基础理论/07-概率与统计基础.md` 中测度化概率部分、未来在 `10-高级主题` 中与核方法/GP/连续时间模型衔接的文档。

- **数值分析与微分方程**
  - 工程师最小知识集：线性方程组数值求解（直接法与迭代法）、浮点误差与稳定性、简单 ODE 数值积分（Euler/Runge–Kutta）直观。
  - 建议进阶：高阶 ODE/PDE 数值解、预条件技术、大规模稀疏线性系统求解方法。
  - 本仓库映射：`01-基础理论/02-数学基础.md` 中数值视角提示、`09-算法理论/03-优化理论/*`（迭代法数值性质）、`10-高级主题` 中神经 ODE/PINNs/科学计算相关文档。

- **几何与信息几何**
  - 工程师最小知识集：流形的直观概念、黎曼度量与测地线、Fisher 信息矩阵在统计中的含义。
  - 建议进阶：信息几何中的双曲空间与指数族、自然梯度与流形优化、几何深度学习中的图与流形表征。
  - 本仓库映射：`01-基础理论/02-数学基础.md` 中几何与微分直观、`01-基础理论/09-序论基础.md` 中序与格视角、`10-高级主题` 中几何深度学习/信息几何方向文档（随后续扩展补全）。

## 四、AI 数学学习路径（工程师版）

- **算法为主（≈ 30 小时）**：
  - 目标：已有工程经验、偏传统算法/系统设计，需要补齐离散数学与基础线性代数/概率。
  - 路线：`01-基础理论/02-数学基础.md` 与 `01-基础理论/09-序论基础.md` 精读核心部分 → 09-01 算法基础（排序、搜索、图、DP、贪心）→ 需要时补读“线性代数 for ML”“概率与统计 for ML”中与算法直接相关的最小子集。
- **ML/量子为主（≈ 60–100 小时）**：
  - 目标：从工程实践走向能够阅读/实现主流 ML/量子算法。
  - 路线：01-02 数学基础 + 01-07 概率与统计基础 → 本文档第 2 章中“线性代数 for ML”“概率与统计 for ML”“优化理论 for ML”的必修条目 → MML book + Coursera/CMU 10‑606 模块 → 10-高级主题 中量子/机器学习文档（结合各文档内「数学前置」小节）。
- **形式化与理论为主（≈ 80–150 小时）**：
  - 目标：希望在形式化方法/证明助手/类型理论中使用 AI 数学工具。
  - 路线：以 01-形式化定义、03-形式化证明、05-类型理论 为主线；数学按需从 01-02、01-07 与本文档链接资源查阅，并在需要更高严谨性时引入“测度论与泛函分析 for 高阶 AI”“几何与信息几何”模块。

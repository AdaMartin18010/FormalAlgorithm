## 三、习题对标与补充

### 3.1 习题难度分级标准

| 级别 | 描述 | 对标课程 |
|------|------|----------|
| **L1-基础** | 直接应用定义和算法 | MIT 6.006 / Stanford CS161 作业 |
| **L2-进阶** | 算法变体、复杂度分析证明 | MIT 6.046 / CMU 15-451 作业 |
| **L3-挑战** | 综合应用、开放设计 | CMU 15-455 / Berkeley CS270 |
| **L4-研究** | 前沿问题、未解难题 | 各课程Project/论文 |

### 3.2 推荐习题集来源

1. **MIT OpenCourseWare**: 6.006, 6.046, 18.404 历年作业
2. **Stanford CS161**: 7次作业 + 3次考试题
3. **CMU 15-451**: 理论题(Oral Exam) + 编程题
4. **Berkeley CS170**: 14次作业 + 2次期中 + 期末
5. **CLRS (Introduction to Algorithms)**: 各章节习题

## 1. Stanford CS 161: Design and Analysis of Algorithms

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Design and Analysis of Algorithms |
| **课程代码** | CS 161 |
| **授课教师** | Nima Anari, Moses Charikar (Winter 2024); Mary Wootters |
| **学期/年份** | Winter 2024, Autumn 2025 |
| **课程网址** | <https://stanford-cs161.github.io/winter2024/> |
| **先修课程** | CS 106B/X, CS 103/103B, CS 109/STATS 116 |
| **学分** | 3-5 units |
| **教材** | 无指定，参考CLRS、Kleinberg-Tardos |

### 课程大纲

#### 第1部分: 算法分析基础

1. **渐进分析**
   - 最坏情况与平均情况分析
   - 渐近符号 (O, Ω, Θ, o, ω)
   - 递归关系求解
   - Master定理

2. **基本数据结构**
   - 二叉搜索树
   - 堆与优先队列
   - 哈希表

#### 第2部分: 算法设计范式

1. **分治法**
   - 归并排序分析
   - 快速排序分析
   - 选择算法 (Median of Medians)
   - 最近点对

2. **动态规划**
   - DP设计方法论
   - 经典问题: 最长公共子序列、编辑距离
   - 背包问题
   - 图算法中的DP (Bellman-Ford, Floyd-Warshall)

3. **贪心算法**
   - 活动选择问题
   - 最小生成树 (Prim, Kruskal)
   - 哈夫曼编码
   - 交换论证

#### 第3部分: 高级主题

1. **图算法**
   - 深度优先搜索与强连通分量
   - 最短路径算法
   - 拓扑排序

2. **摊还分析**
   - 聚合分析
   - 会计方法
   - 势能方法

3. **随机化算法**
   - 随机快速排序
   - 哈希分析
   - 概率分析技术

4. **网络流 (可选)**
   - 最大流问题
   - Ford-Fulkerson方法

## 2. Stanford CS 238V: Validation of Safety Critical Systems

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Validation of Safety Critical Systems |
| **课程代码** | CS 238V / AA 228V |
| **授课教师** | Sydney Katz, Mykel Kochenderfer |
| **学期/年份** | Winter 2025, 2026 |
| **课程网址** | <https://aa228v.stanford.edu/> |
| **先修课程** | 概率论、多变量微积分、高级编程语言 |
| **学分** | 3-4 units |
| **教材** | 课程笔记 |

### 课程大纲

#### 第1部分: 验证框架

1. **问题设定**
   - 模型拟合
   - 模型验证
   - 属性规约

2. **时序逻辑**
   - 线性时序逻辑 (LTL)
   - 计算树逻辑 (CTL)
   - 规约语言

#### 第2部分: 基于采样的验证

1. **Falsification**
   - 基于优化的falsification
   - 鲁棒性度量
   - 场景生成

2. **统计验证**
   - 马尔可夫链蒙特卡洛 (MCMC)
   - 重要性采样
   - 故障概率估计

#### 第3部分: 形式化方法

1. **可达性分析**
   - 可达集计算
   - 抽象解释

2. **模型检测**
   - 显式状态模型检测
   - 符号模型检测
   - SAT求解器

3. **可解释性**
   - 验证结果解释
   - 反例分析

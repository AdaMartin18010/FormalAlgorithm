## 1. Princeton COS 226: Algorithms and Data Structures

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Algorithms and Data Structures |
| **课程代码** | COS 226 |

| **学期/年份** | Fall 2025, Spring 2026 |
| **课程网址** | <https://www.cs.princeton.edu/courses/archive/fall25/cos226/> |
| **先修课程** | COS 126 |
| **学分** | 4 units |
| **教材** | Sedgewick & Wayne《Algorithms》4th Edition |

### 课程大纲

#### 第1部分: 基础算法

1. **Union-Find (并查集)**
   - Quick-find
   - Quick-union
   - 路径压缩与按秩合并
   - 应用: 渗流问题

2. **排序算法**
   - 初级排序: 选择、插入、冒泡
   - 高级排序: 归并、快速、堆排序
   - 排序下界
   - 稳定性分析

3. **搜索与符号表**
   - 二分搜索
   - 二叉搜索树
   - 平衡二叉树 (红黑树)
   - B树

#### 第2部分: 图算法

1. **图的基本算法**
   - 图的表示 (邻接表/矩阵)
   - 深度优先搜索 (DFS)
   - 广度优先搜索 (BFS)
   - 连通分量

2. **有向图**
   - 有向DFS
   - 拓扑排序
   - 强连通分量 (Kosaraju算法)

3. **最小生成树**
   - Prim算法
   - Kruskal算法
   - 割性质与循环性质

4. **最短路径**
   - Dijkstra算法
   - 边松弛框架
   - 负权边与Bellman-Ford

#### 第3部分: 字符串处理

1. **字符串排序**
   - 键索引计数
   - LSD基数排序
   - MSD基数排序
   - 三向快速排序

2. **字符串查找**
   - 暴力算法
   - KMP算法
   - Boyer-Moore算法
   - Rabin-Karp指纹

3. **正则表达式**
    - NFA模拟
    - 模式匹配

4. **数据压缩**
    - Huffman编码
    - LZW压缩

#### 第4部分: 高级主题

1. **几何算法**
    - 几何原语
    - 凸包 (Graham扫描, Jarvis步进)
    - 范围搜索
    - 最近点对

2. **最大流与最小割**
    - Ford-Fulkerson
    - 增广路径
    - 最大流最小割定理
    - 二分匹配

3. **字符串数据结构**
    - 后缀数组
    - 后缀树
    - 最长重复子串

## 2. Princeton 算法课程特色

### 教学特色

1. **Java实现**: 强调算法的实际Java实现
2. **科学方法**: 使用科学方法分析算法性能
3. **可视化**: 丰富的算法可视化资源
4. **应用导向**: 结合实际应用问题

### 独特优势

1. **字符串算法**: 字符串处理内容非常全面
2. **数据压缩**: 包含实用的压缩算法
3. **几何算法**: 计算几何基础内容
4. **算法工程**: 强调算法实现的工程实践

### 与项目对标的互补性

- Princeton COS 226在字符串算法和数据压缩方面有独特优势
- 可作为项目字符串算法模块的重要参考
- 算法实现细节可作为项目实现示例的补充

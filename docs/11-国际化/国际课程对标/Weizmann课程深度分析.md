### 1.2 理论计算机科学声誉

魏茨曼研究所以其卓越的**理论计算机科学研究**享誉全球，与MIT、Stanford、Berkeley并列为世界顶尖理论计算机科学研究中心。其特点包括：

- **纯理论研究导向**: 专注于算法理论、计算复杂性、密码学等核心理论领域
- **数学驱动**: 强调深厚的数学基础，与数学系紧密合作
- **小而精的培养模式**: 仅招收研究生，师生比例极低，个性化培养
- **国际合作网络**: 与普林斯顿IAS、MIT、ETH Zurich等有深度学术合作

#### 课程信息

| 项目 | 内容 |
|------|------|

| **课程代码** | CS 236701 |

| **主要教材** | Vazirani《Approximation Algorithms》、Motwani-Raghavan《Randomized Algorithms》 |

#### 课程目标

1. 掌握高级算法设计技术，特别是近似算法和随机算法
2. 理解NP难问题的近似复杂性理论
3. 学会使用概率方法分析算法性能
4. 培养阅读和理解理论计算机科学前沿论文的能力

#### 课程大纲

**模块一: 近似算法基础 (Weeks 1-4)**

1. **近似算法导论**
   - 优化问题与近似比定义
   - 多项式时间近似方案 (PTAS, FPTAS)
   - 不可近似性结果概述

2. **组合优化近似算法**
   - 顶点覆盖: 2-近似算法
   - 集合覆盖: 贪心算法的对数近似
   - 旅行商问题 (TSP): 度量TSP的Christofides算法 (1.5-近似)
   - 装箱问题: First-Fit算法分析

3. **线性规划舍入技术**
   - LP松弛与舍入方法
   - 原始-对偶方法
   - 度量设施选址问题

4. **半正定规划 (SDP) 应用**
   - Max-Cut的Goemans-Williamson算法
   - 随机超平面舍入
   - 着色问题的SDP方法

**模块二: 随机算法与概率方法 (Weeks 5-8)**

1. **随机算法基础**
   - Las Vegas vs Monte Carlo算法
   - 期望运行时间分析
   - 随机舍入技术

2. **概率方法**
   - 基本概率方法 (期望论证)
   - Lovász局部引理
   - 熵方法与压缩论证

3. **随机图与网络算法**
   - 随机图模型 (Erdős-Rényi)
   - 连通性与直径分析
   - 随机游走与混合时间

4. **抽样与学习**
   - 重要性抽样
   - 概率近似正确学习 (PAC学习)
   - VC维与样本复杂度

**模块三: 在线算法 (Weeks 9-11)**

1. **竞争分析框架**
   - 竞争比定义
   - 分页问题与标记算法
   - k-server问题概述

2. **在线优化技术**
   - 势函数方法
   - 指数权重算法
   - 在线匹配与市场算法

**模块四: 高级主题 (Weeks 12-13)**

1. **子模函数优化**
   - 子模性定义与性质
   - 贪心算法与近似保证
   - 连续松弛方法

2. **流算法与 sketches**
   - 数据流模型
   - Count-Min Sketch
   - 水库抽样

### 2.2 课程: 计算复杂性理论 (Computational Complexity Theory)

#### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Computational Complexity Theory |
| **课程代码** | CS 236802 |
| **授课教师** | Ran Raz / Irit Dinur |
| **课程级别** | 研究生核心课程 |
| **先修课程** | 可计算性理论、离散数学、线性代数 |
| **学分学时** | 3学分，39课时 |
| **主要教材** | Arora-Barak《Computational Complexity: A Modern Approach》 |

#### 课程目标

1. 深入理解P、NP、PSPACE等核心复杂性类
2. 掌握电路复杂性、通信复杂性等下界技术
3. 理解PCP定理及其在不可近似性中的应用
4. 学习高级主题如证明复杂性、代数复杂性

#### 课程大纲

**模块一: 复杂性类基础 (Weeks 1-3)**

1. **时间复杂性**
   - P、NP、NP完全性回顾
   - 时间层次定理
   - NP中间问题 (Ladner定理)
   - 多项式层级 (Polynomial Hierarchy)

2. **空间复杂性**
   - L、NL、PSPACE
   - Savitch定理
   - Immerman-Szelepcsényi定理 (NL=coNL)
   - PSPACE完全问题

3. **随机复杂性类**
   - BPP、RP、coRP、ZPP
   - 随机性归约
   - BPP与P的关系
   - 去随机化基础

**模块二: 电路复杂性 (Weeks 4-6)**

1. **布尔电路基础**
   - 电路族与统一性
   - P/poly与Karp-Lipton定理
   - 电路可满足性问题

2. **电路下界技术**
   - 限制法 (Restrictions)
   - 开关引理 (Switching Lemma)
   - Parity不在AC⁰中的证明
   - 单调电路下界

3. **高级电路复杂性**
   - 有界深度电路
   - 通信复杂性方法
   - 自然证明障碍 (Natural Proofs)

**模块三: 通信复杂性 (Weeks 7-9)**

1. **两方通信复杂性**
   - 确定性通信复杂性
   - 矩形覆盖与秩下界
   - fooling set方法

2. **随机与量子通信**
   - 随机通信复杂性
   - 公共随机vs私有随机
   - 量子通信复杂性
   - 量子优势与下界

3. **通信复杂性应用**
   - 数据流算法下界
   - 电路深度下界
   - 扩展图构造

**模块四: PCP定理与不可近似性 (Weeks 10-11)**

1. **PCP定理**
   - PCP verifier定义
   - 间隙放大 (Gap Amplification)
   - Dinur的组合证明
   - 代数证明方法

2. **不可近似性结果**
   - 从PCP到不可近似性
   - Hastad的3-位PCP
   - 标签覆盖问题
   - 唯一游戏猜想 (UGC)

**模块五: 高级主题 (Weeks 12-13)**

1. **证明复杂性**
   - 分辨率下界
   - 代数证明系统
   - Feasible插值

2. **代数复杂性**
   - 算术电路
   - 多项式恒等测试
   - VP vs VNP问题
   - 矩阵乘法复杂度

#### 关键论文与阅读材料

| 电路下界 | Furst, M., Saxe, J.B. & Sipser, M. (1984). "Parity, circuits, and the polynomial-time hierarchy." Math. Systems Theory 17: 13-27. |

### 2.3 课程: 计算机科学中的概率方法 (Probabilistic Methods in CS)

#### 课程目标

1. 掌握概率方法的核心技术：期望论证、Lovász局部引理、集中不等式
2. 学会使用概率方法证明组合结构的存在性
3. 理解随机过程在算法设计和分析中的应用

#### 课程大纲

**模块一: 基本概率方法 (Weeks 1-3)**

1. **期望论证**
   - 基本期望原理

   - 条件期望与条件概率

   - 切比雪夫不等式

**模块二: 集中不等式 (Weeks 4-6)**

2. **鞅方法**
   - Azuma-Hoeffding不等式
   - Doob鞅
   - 鞅差序列

3. **Talagrand集中不等式**
   - 配置函数
   - 最长递增子序列
   - 旅行商问题

**模块三: 随机图与网络 (Weeks 7-9)**

1. **Erdős-Rényi随机图**
   - G(n,p)模型
   - 阈值现象
   - 巨分量相变

2. **扩展图 (Expanders)**
   - 扩展图定义与性质
   - 随机扩展图构造
   - 应用: 去随机化、网络设计

3. **随机游走**
   - 混合时间
   - 特征值与收敛速度
   - 马尔可夫链蒙特卡洛 (MCMC)

**模块四: 随机算法应用 (Weeks 10-11)**

1. **随机数据结构**
   - 跳表 (Skip Lists)
   - 布隆过滤器 (Bloom Filters)
   - 一致性哈希

2. **并行与分布式随机算法**
   - 随机负载均衡
   - 共识算法
   - 分布式图算法

**模块五: 高级主题 (Weeks 12-13)**

1. **随机约束满足问题**
   - 随机k-SAT
   - 可满足性阈值
   - 算法相变

2. **熵与压缩**
   - 香农熵
   - 压缩论证

## 五、总结与建议

### 5.1 魏茨曼课程分析总结

魏茨曼研究所代表了**纯理论研究**的典范，其课程具有以下鲜明特点：

1. **理论深度**: 每门课程都追求最严格的数学定义与证明

3. **前沿导向**: 课程内容紧跟最新研究，而非经典教材复述
4. **小而精**: 课程规模小，互动性强，适合深入讨论

1. **优先补充高优先级缺失主题**

   - 零知识证明系统

2. **强化概率方法内容**
   - 扩充现有概率论章节
   - 增加概率方法在算法设计中的应用

3. **新增子模优化专题**
   - 这是魏茨曼高级算法课程的重要组成部分
   - 在现代组合优化中具有核心地位

#### 中期建议 (6-12个月)

1. **建立密码学理论基础模块**
   - 现有内容偏重应用，需补充理论深度
   - 参考Goldreich的《密码学基础》

2. **扩展分布式算法理论**
   - 补充LOCAL/CONGEST模型
   - 增加分布式计算下界技术

3. **新增证明复杂性内容**
   - 这是魏茨曼复杂性课程的亮点

#### 长期建议 (12个月以上)

   - 整合各顶尖院校的独特内容
   - 魏茨曼的SDP应用、扩展图、随机k-SAT等

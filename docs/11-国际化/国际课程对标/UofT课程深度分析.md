## 二、核心课程分析 / Core Course Analysis

### 2.1 CSC 373: Algorithm Design, Analysis & Complexity

#### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Algorithm Design, Analysis & Complexity |
| **课程代码** | CSC 373H1 |
| **授课学期** | Fall/Winter |
| **先修课程** | CSC 236 (计算理论), CSC 263 (数据结构与算法) |
| **学分** | 0.5学分 |
| **课程网站** | <https://www.cs.toronto.edu/~lalla/373s20/> |

#### 课程描述与目标

本课程是UofT计算机科学专业算法核心课程，重点培养学生的算法设计能力与复杂度分析技能。课程涵盖经典算法设计范式、高级数据结构、计算复杂度基础等内容。

**学习目标**:

- 掌握分治、动态规划、贪心等算法设计范式
- 理解NP完全性与计算复杂度理论
- 具备分析算法时间和空间复杂度的能力
- 能够设计高效的近似算法和在线算法

#### 课程大纲

**第一部分: 算法设计范式 (40%)**

1. **分治法 (Divide and Conquer)**
   - 递归关系求解 (Master定理、递归树法)
   - 中位数查找与选择算法
   - 快速傅里叶变换 (FFT) 及其应用
   - Strassen矩阵乘法

2. **动态规划 (Dynamic Programming)**
   - DP设计方法论与最优子结构
   - 经典问题: 最长公共子序列、编辑距离、背包问题
   - 图算法中的DP: Bellman-Ford、Floyd-Warshall
   - 树上的DP与状态压缩DP

3. **贪心算法 (Greedy Algorithms)**
   - 贪心选择性质与最优子结构
   - 拟阵理论 (Matroids) 基础
   - 活动选择、哈夫曼编码
   - 交换论证方法

**第二部分: 网络流与线性规划 (25%)**

1. **网络流算法**
   - 最大流问题与Ford-Fulkerson方法
   - Edmonds-Karp算法分析
   - 最小割与最大流对偶性
   - 二分图匹配问题

2. **线性规划基础**
   - 标准形式与几何解释
   - 单纯形法原理
   - 对偶理论
   - LP在近似算法中的应用

**第三部分: 计算复杂度 (20%)**

1. **P与NP**
   - 多项式时间归约
   - NP完全问题与Cook-Levin定理
   - 经典NP完全问题: SAT、3-SAT、顶点覆盖、独立集

2. **复杂度层次**
   - PSPACE与NPSPACE
   - 指数时间复杂度
   - 多项式层次简介

**第四部分: 高级主题 (15%)**

1. **近似算法**
   - 近似比分析
   - 顶点覆盖、集合覆盖的近似算法
   - PTAS与FPTAS概念

2. **在线算法**
   - 竞争分析框架
   - 分页问题与k-server问题
   - 多臂老虎机简介

#### 推荐教材

| 教材 | 作者 | 用途 |
|------|------|------|

| **《Computers and Intractability》** | Garey & Johnson | NP完全性理论经典 |

### 2.2 CSC 473: Advanced Algorithms

#### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Advanced Algorithms |
| **课程代码** | CSC 473H1 |
| **授课学期** | Winter |
| **先修课程** | CSC 373 (算法设计) |
| **学分** | 0.5学分 |

#### 课程描述与目标

高级算法课程深入探讨现代算法设计的高级技术与理论，重点关注随机化算法、近似算法、在线算法以及算法博弈论等前沿主题。

**学习目标**:

- 掌握随机化算法设计与分析技术
- 理解近似算法的极限与不可能性结果
- 掌握在线算法的竞争分析
- 了解算法博弈论基础

#### 课程大纲

**第一部分: 随机化算法 (30%)**

1. **概率基础与集中不等式**
   - 期望、方差、条件概率
   - Markov不等式、Chebyshev不等式
   - Chernoff界及其应用
   - 并集界与概率方法

2. **随机化算法设计**
   - Las Vegas与Monte Carlo算法
   - 随机快速排序与选择
   - 哈希与指纹技术
   - 最小割算法 (Karger算法)

3. **随机游走与马尔可夫链**
   - 随机游走的混合时间
   - PageRank算法理论
   - MCMC方法简介

**第二部分: 近似算法进阶 (25%)**

1. **LP舍入与原始对偶方法**
   - 基于LP舍入的近似算法
   - 原始对偶设计方法
   - 度量旅行商问题

2. **不可近似性**
   - PCP定理简介
   - 近似保持归约
   - APX-hardness

3. **特殊问题的近似算法**
   - 设施选址问题
   - k-center与k-median问题
   - 最大割问题 (Goemans-Williamson算法)

**第三部分: 在线算法 (20%)**

1. **竞争分析框架**
   - 竞争比定义与下界技术
   - 分页问题最优算法
   - k-server问题

2. **在线学习与优化**
   - 专家问题与加权多数算法
   - 在线凸优化
   - 多臂老虎机问题

**第四部分: 高级数据结构 (15%)**

1. **摊还分析高级技术**
   - 势能方法深入
   - 伸展树 (Splay Trees)
   - 动态树与Link-Cut Trees

2. **高级哈希技术**
   - 完美哈希
   - 一致哈希
   - 布隆过滤器

**第五部分: 算法博弈论 (10%)**

1. **博弈论基础**
   - 纳什均衡
   - 机制设计基础
   - 拍卖理论简介

### 2.4 CSC 2626: Neural Network Theory

#### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Topics in Theoretical Statistics: Neural Network Theory |
| **课程代码** | CSC 2626H / STA 2601H |
| **授课学期** | Winter |
| **授课教师** | Murat Erdogdu, Daniel Roy |
| **先修课程** | CSC 2515, STA 261 (高等概率论), MAT 237 (分析) |
| **学分** | 0.5学分 |

#### 课程描述与目标

本课程是UofT最理论化的神经网络课程，从数学角度深入分析神经网络的表达能力、优化理论与泛化性质。课程内容涉及深度学习的前沿理论研究，适合有志于从事深度学习理论研究的学生。

**学习目标**:

- 理解神经网络的表达能力与通用近似定理
- 掌握深度学习优化的理论分析
- 理解神经网络的泛化理论
- 了解深度学习理论的开放问题与前沿方向

#### 课程大纲

**第一部分: 神经网络表达能力 (25%)**

1. **通用近似定理**
   - Cybenko定理 (单隐层感知机的通用近似能力)
   - 连续函数逼近的密度结果
   - 逼近速率与网络复杂度权衡
   - Barron空间的逼近理论

2. **深度vs宽度**
   - 深度网络的指数级优势
   - 深度分离结果 (Depth Separation)
   - 特定函数的深度效率

3. **神经正切核 (NTK)**
   - 无限宽网络的核极限
   - NTK的收敛性
   - 梯度下降在NTK regime的动态

**第二部分: 优化理论 (30%)**

1. **梯度下降的收敛性**
   - 凸优化基础回顾
   - 非凸优化的挑战
   - 神经网络损失景观的性质

2. **过参数化网络的优化**
   - 过参数化的好处
   - Polyak-Lojasiewicz条件
   - 全局收敛性结果

3. **随机梯度下降理论**
   - SGD的收敛速率
   - 小批量与噪声的隐式正则化
   - 学习率调度理论

4. **深度学习中的隐式偏差**
   - 梯度下降的隐式偏好
   - 稀疏性与低秩性
   - 边缘最大化

**第三部分: 泛化理论 (30%)**

1. **传统泛化理论**
   - 神经网络VC维的指数增长
   - 基于范数的泛化界
   - PAC-Bayes框架

2. **过参数化与泛化**
   - 双下降现象 (Double Descent)
   - 良性过拟合 (Benign Overfitting)
   - 插值与泛化

3. **隐式正则化与泛化**
   - 梯度下降的隐式正则化效应
   - 早停与泛化
   - 批量大小对泛化的影响

4. **深度学习泛化的开放问题**
   - 传统理论的局限性
   - 神经正切核 regime之外的泛化
   - 算法依赖的泛化分析

**第四部分: 高级专题 (15%)**

1. **生成模型理论**
   - 生成对抗网络 (GAN) 的理论
   - 变分自编码器 (VAE) 理论
   - 扩散模型的理论基础

2. **Transformer理论**
   - 注意力机制的表达能力
   - Transformer的上下文学习能力
   - 大语言模型的理论

3. **强化学习与神经网络**
   - 函数逼近在RL中的理论
   - 深度Q网络的收敛性
   - 策略梯度方法的理论

### 5.1 UofT课程特色总结

多伦多大学在机器学习理论和神经网络领域具有以下独特优势：

1. **深厚理论传统**: 源自Hinton的深度学习研究传统，强调严格的数学基础
2. **前沿研究融合**: 课程内容紧跟深度学习理论最新进展 (NTK、隐式正则化等)
3. **Vector Institute整合**: 产学研深度融合，获取顶级研究资源
4. **数学严谨性**: 强调泛函分析、高维概率、优化理论等数学工具

### 5.2 对FormalAlgorithm项目的启示

1. **加强深度学习理论**: 项目现有神经网络文档偏重实现，需大幅补充理论基础
2. **补充统计学习理论**: PAC学习、VC维、Rademacher复杂度等核心内容缺失
3. **关注前沿方向**: 过参数化、隐式正则化、双下降等现代深度学习理论
4. **强化数学基础**: 需要更多泛函分析、高维概率、优化理论的内容

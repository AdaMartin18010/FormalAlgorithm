## 二、核心课程深度分析

### 2.1 CSE 101: Design and Analysis of Algorithms

#### 课程基本信息

| 项目 | 内容 |
|------|------|
| **课程代码** | CSE 101 |
| **课程名称** | Design and Analysis of Algorithms (算法设计与分析) |
| **学分** | 4 学分 |
| **课程性质** | 本科核心必修课 |
| **先修课程** | CSE 12, CSE 21 (或 Math 15B), CSE 100 |
| **等价课程** | Math 188 |

#### 课程描述与目标

本课程是 UCSD 计算机科学本科的算法入门核心课程，重点讲授**非数值算法**的设计与分析。课程强调：

- 掌握通用算法设计技术
- 学会分析算法的时间与空间复杂度
- 理解计算复杂性基础概念
- 培养解决新问题的算法思维能力

#### 详细教学大纲

| 模块 | 主题内容 | 课时 |
|------|----------|------|
| **1. 算法基础** | 算法与优化问题、问题形式化 | 1-2 讲 |
| **2. 分治策略 (Divide & Conquer)** | 归并排序、快速排序、整数乘法、最近点对、主定理、递推关系 | 3-4 讲 |
| **3. 贪心算法 (Greedy Algorithms)** | 区间调度、最小生成树 (Kruskal, Prim)、Dijkstra 最短路径、哈夫曼编码 | 3-4 讲 |
| **4. 动态规划 (Dynamic Programming)** | 加权区间调度、编辑距离/最长公共子序列、矩阵链乘法、全源最短路径、背包问题 | 3-4 讲 |
| **5. 图算法 (Graph Algorithms)** | BFS/DFS、拓扑排序、强连通分量、最短路径算法 | 3-4 讲 |
| **6. 网络流 (Network Flow)** | 最大流/最小割、Ford-Fulkerson 算法、二分图匹配 | 2-3 讲 |
| **7. 计算复杂性 (Complexity)** | P 与 NP、NP 完全性、Cook-Levin 定理、应对难解性的策略 | 2-3 讲 |
| **8. 高级主题** | 线性规划、数论算法、量子计算 (视时间而定) | 2-3 讲 |

#### 推荐教材

| 教材 | 作者 | 出版社 | 用途 |
|------|------|--------|------|
| **《Algorithms》** | Dasgupta, Papadimitriou, Vazirani | McGraw Hill | 主教材 |
| **《Algorithm Design》** | Kleinberg, Tardos | Addison Wesley | 补充参考 |
| **《Introduction to Algorithms》** | Cormen, Leiserson, Rivest, Stein | MIT Press | 深度参考 |

### 2.2 CSE 202: Algorithm Design and Analysis (Graduate)

#### 课程基本信息

| 项目 | 内容 |
|------|------|
| **课程代码** | CSE 202 |
| **课程名称** | Algorithm Design and Analysis (研究生算法设计与分析) |
| **学分** | 4 学分 |
| **课程性质** | 研究生核心必修课 |
| **先修课程** | CSE 101 或同等水平 |

#### 课程描述与目标

本课程是 UCSD 计算机科学研究生的**核心算法课程**，在本科基础上深化算法设计与分析技术，引入更高级的主题：

- 巩固并扩展本科算法知识
- 学习随机算法与概率分析
- 理解下界证明技术
- 探索并行算法与高级数据结构
- 掌握近似算法基础

#### 详细教学大纲

| 模块 | 主题内容 | 深度要求 |
|------|----------|----------|
| **1. 图搜索与数据结构优化** | BFS/DFS 进阶应用、Dijkstra 算法、数据结构的预处理与优化、摊还分析 | 高 |
| **2. 贪心算法深化** | 区间调度、最小生成树、贪心近似算法、最优性证明方法 | 高 |
| **3. 分治与递归进阶** | 快速傅里叶变换 (FFT)、中位数查找、多参数递归、并行计算 | 高 |
| **4. 动态规划进阶** | 加权区间调度、编辑距离、子集和、DP 近似算法 | 高 |
| **5. 网络流与线性规划** | Ford-Fulkerson 算法、LP 对偶性、单纯形算法、近似算法中的 LP 应用 | 高 |
| **6. 随机算法** | 哈希与指纹算法、随机游走、概率分析、Monte Carlo 与 Las Vegas 算法 | 中高 |
| **7. 计算复杂性深化** | NP 完全性深入、多项式层次、空间复杂性 | 中 |
| **8. 近似算法** | 近似比分析、集合覆盖、最大可满足性问题 | 中 |

### 2.3 CSE 203A: Advanced Algorithms

#### 课程基本信息

| 项目 | 内容 |
|------|------|
| **课程代码** | CSE 203A |
| **课程名称** | Advanced Algorithms (高级算法) |
| **学分** | 4 学分 |
| **课程性质** | 研究生高级选修课 |
| **先修课程** | CSE 202 |
| **开课频率** | 每学年定期开设，主题可能轮换 |

#### 课程描述与目标

CSE 203A 是一门**主题轮换**的高级算法课程，关注算法设计与分析的现代进展。课程内容灵活，涵盖多个前沿领域：

- 近似算法 (Approximation Algorithms)
- 随机算法 (Randomized Algorithms)
- 在线算法与竞争分析 (Online Algorithms)
- 并行与分布式算法
- 内存层次模型算法
- 数论算法与密码分析
- 计算几何
- 计算生物学算法
- 网络算法
- VLSI CAD 算法

#### 典型教学大纲 (以随机算法专题为例)

| 模块 | 主题内容 |

| **2. 随机游走** | Markov 链、混合时间、谱图理论应用 |
| **3. 集中不等式** | Chernoff 界、Azuma 不等式、应用 |
| **4. 采样与计数** | 蒙特卡洛方法、近似计数、MCMC |
| **5. 流式算法** | 数据流模型、草图技术、空间下界 |

### 2.4 CSE 200: Computability and Complexity

#### 课程基本信息

| 项目 | 内容 |
|------|------|
| **课程代码** | CSE 200 |
| **课程名称** | Computability and Complexity (可计算性与复杂性) |
| **学分** | 4 学分 |
| **课程性质** | 研究生理论核心课 |
| **先修课程** | CSE 105 或同等课程 |

#### 课程描述与目标

本课程提供**计算复杂性理论**的系统介绍，是理论计算机科学的基础：

- 理解计算资源的定义与分类
- 掌握复杂性类之间的关系
- 学习下界证明技术
- 探索随机化与交互证明

#### 详细教学大纲

| 模块 | 主题内容 |
|------|----------|
| **1. 计算模型** | 图灵机、RAM 模型、布尔电路、Church-Turing 论题 |
| **2. 可计算性回顾** | 停机问题、可判定集、递归可枚举集、多一归约 |
| **3. 时间与空间复杂性** | TIME(t(n))、SPACE(s(n))、L、P、PSPACE、NP |

| **5. 层次定理** | 时间与空间层次、多项式层次 (PH) |
| **6. 空间复杂性** | Savitch 定理、NL=coNL、PSPACE 完全性 |
| **7. 电路复杂性** | 电路下界、PARITY 不在 AC0 中、自然证明 |
| **8. 随机算法** | RP、BPP、去随机化 |
| **9. 交互证明** | IP=PSPACE (时间允许时) |
| **10. 量子计算** | 量子算法基础 (时间允许时) |

### 2.5 CSE 206A: Lattice Algorithms and Applications

#### 课程基本信息

| 项目 | 内容 |
|------|------|
| **课程代码** | CSE 206A (原 CSE 207C) |
| **课程名称** | Lattice Algorithms and Applications (格算法与应用) |
| **学分** | 4 学分 |
| **课程性质** | 研究生专题选修课 |
| **先修课程** | CSE 202、CSE 200 (或同时修读) |

#### 课程描述与目标

本课程介绍**格 (Lattice)** 的算法理论及其在密码学中的核心应用，是 UCSD 的**特色强项课程**：

- 掌握格的基本理论与算法
- 理解格基约化技术 (LLL 算法)
- 学习格在密码分析中的应用
- 探索格基密码学 (后量子密码)

#### 详细教学大纲

| 模块 | 主题内容 |
|------|----------|
| **1. 格基础** | 格的定义、基、Gram-Schmidt 正交化、行列式 |
| **2. Minkowski 定理** | 最短向量问题 (SVP)、连续最小值 |
| **3. 基本算法** | Hermite 标准型、对偶格、最近平面算法 |
| **4. LLL 算法** | 基约化算法、近似 SVP/CVP 算法 |
| **5. 密码分析应用** | 线性同余生成器分析、子集和问题 |
| **6. 计算复杂性** | 最坏情况与平均情况复杂性联系 |
| **7. 格基密码学** | 基于格的加密、全同态加密 |
| **8. 高级数学工具** | 傅里叶分析、代数数论在格中的应用 |

## 五、总结与建议

### 5.1 UCSD 课程优势总结

1. **排名与声誉**: CSRankings 全球 #3，算法与系统领域顶尖
2. **理论深度**: 从 CSE 101 到 CSE 200/202 形成完整理论体系
3. **工程实践**: 强调算法的实际实现与性能评估
4. **密码学特色**: 格算法等密码学相关课程世界领先
5. **灵活专题**: CSE 291 系列覆盖算法研究前沿

### 5.2 对 FormalAlgorithm 项目的建议

1. **优先补充随机算法与近似算法模块**，这是当前理论算法教育的核心组成
2. **引入在线算法与流式算法内容**，适应大数据时代需求
3. **参考 UCSD 的作业设计模式**，强化理论与实现结合
4. **关注格算法等后量子密码方向**，这是未来密码学的重要领域
5. **建立专题研讨机制**，保持内容的前沿性与灵活性

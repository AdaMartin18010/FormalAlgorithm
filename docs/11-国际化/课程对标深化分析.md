| Stanford | 1 | 算法设计与分析 |

| Berkeley | 1 | 高效算法与难解问题 |
| Harvard | 2 | 算法与复杂度、算法局限性 |
| Princeton | 1 | 算法与数据结构 |
| Oxford | 1 | 证明助手 |
| Cambridge | 1 | 同伦类型论 |
| Indiana University | 1 | 现代依赖类型 |

| Cornell | 1 | 认证软件系统 |

### 1. MIT课程分析

#### 1.1 MIT 6.006: Introduction to Algorithms

**课程基本信息**

| 项目 | 内容 |
|------|------|
| **课程名称** | Introduction to Algorithms |
| **课程代码** | 6.006 (新编号: 6.1210) |
| **授课教师** | Erik Demaine, Jason Ku, Justin Solomon |
| **学期/年份** | Spring 2020, Fall 2024+ |
| **课程网址** | <https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-spring-2020/> |
| **先修课程** | 6.0001 (Python编程), 6.042J (离散数学) |
| **学分** | 12 units (4-0-8) |
| **教材** | CLRS《Introduction to Algorithms》3rd Edition |

**课程目标**

1. 掌握算法性能度量与分析技术
2. 理解经典数据结构与算法
3. 熟练运用算法设计范式
4. 培养算法问题求解能力

**教学大纲**

| 周次 | 主题 | 内容要点 | 阅读材料 | 作业/项目 |
|------|------|---------|---------|----------|
| 1-2 | 算法基础 | 计算模型、渐进分析、递归关系 | CLRS第1-4章 | Problem Set 1 |
| 3-4 | 数据结构 | 动态数组、堆、哈希表 | CLRS第6、10-11章 | Problem Set 2 |
| 5-6 | 图算法 | BFS/DFS、拓扑排序 | CLRS第22章 | Problem Set 3 |
| 7-8 | 最短路径 | Dijkstra、Bellman-Ford | CLRS第24章 | Problem Set 4 |
| 9-10 | 生成树 | Prim、Kruskal、并查集 | CLRS第21、23章 | Problem Set 5 |
| 11-12 | 动态规划 | 子问题结构、经典应用 | CLRS第15章 | Problem Set 6 |
| 13-14 | 贪心算法 | 贪心选择、最优子结构 | CLRS第16章 | Problem Set 7 |
| 15-16 | 高级主题 | 摊还分析、NP基础 | CLRS第17、34章 | Final Project |

**项目/作业**

- 7次理论作业
- 2次编程作业（Python实现）
- 期末考试

**项目对齐分析**

| 对齐维度 | 评分 | 说明 |
|---------|------|------|
| **已覆盖内容** | 渐进分析、动态数组、堆、哈希表、排序、图算法、DP、贪心 | |
| **缺失内容** | 摊还分析势能法详细证明、高级哈希技术 | |
| **对齐度** | **90%** | 高度一致 |
| **改进建议** | 1. 补充势能法详细推导<br>2. 添加布隆过滤器等高级哈希应用<br>3. 增加AVL树/红黑树完整实现 | |

**课程基本信息**

| 项目 | 内容 |

| **课程网址** | <https://ocw.mit.edu/courses/18-404j-theory-of-computation-fall-2020/> |
| **先修课程** | 6.1200 或 18.200 |
| **教材** | Sipser《Introduction to the Theory of Computation》|

**课程目标**

1. 理解形式语言与自动机理论
2. 掌握可计算性理论基础
3. 理解计算复杂度基本结果
4. 了解高级主题（交互式证明等）

**教学大纲**

| 周次 | 主题 | 内容要点 | 阅读材料 |
|------|------|---------|---------|
| 1-2 | 正则语言 | DFA、NFA、正则表达式、泵引理 | Sipser第1章 |
| 3-4 | 上下文无关语言 | CFG、PDA、非CFL | Sipser第2章 |
| 5-6 | 图灵机 | TM定义、变体、Church-Turing论题 | Sipser第3章 |
| 7-8 | 可判定性 | 可判定问题、停机问题、归约 | Sipser第4章 |
| 9-10 | 复杂度类 | P、NP、NP完全性 | Sipser第7章 |
| 11-12 | 空间复杂度 | PSPACE、Savitch定理 | Sipser第8章 |
| 13-14 | 高级主题 | 概率计算、交互式证明 | Sipser第10章 |

**项目对齐分析**

| 对齐维度 | 评分 | 说明 |
|---------|------|------|
| **已覆盖内容** | 有限自动机、正则语言、图灵机、可判定性、P/NP、NP完全性 | |
| **缺失内容** | 交互式证明系统(IP)、概率复杂度类(BPP)、空间层次定理 | |
| **对齐度** | **85%** | 基本一致 |
| **改进建议** | 1. 增加交互式证明系统完整理论<br>2. 补充概率复杂度类<br>3. 添加电路复杂度基础 | |

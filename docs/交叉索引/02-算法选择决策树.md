## 决策路径速查表

| 问题场景 | 推荐算法 | 时间复杂度 | 空间复杂度 |
|---------|---------|-----------|-----------|
| 小规模排序 | 插入排序 | O(n²) | O(1) |
| 通用排序 | 快速排序 | O(n log n) 期望 | O(log n) |
| 稳定排序 | 归并排序 | O(n log n) | O(n) |
| 有序数组搜索 | 二分搜索 | O(log n) | O(1) |
| 高频查询 | 哈希表 | O(1) 期望 | O(n) |
| 非负权最短路径 | Dijkstra | O((V+E)log V) | O(V) |
| 含负权最短路径 | Bellman-Ford | O(VE) | O(V) |
| 稠密图MST | Prim (数组) | O(V²) | O(V) |
| 稀疏图MST | Kruskal | O(E log E) | O(E) |
| 单模式匹配 | KMP | O(n+m) | O(m) |
| 多模式匹配 | Aho-Corasick | O(n+m+z) | O(m) |
| 0-1背包 | 动态规划 | O(nW) | O(W) |
| 凸包 | Andrew扫描 | O(n log n) | O(n) |

## 特殊场景处理

### 数据流算法
- **频繁项**: Count-Min Sketch
- **基数估计**: HyperLogLog
- **滑动窗口**: 指数直方图

### 近似算法
- **NP-hard问题**: 多项式时间近似方案(PTAS)
- **大规模优化**: 模拟退火、遗传算法
- **在线算法**: 竞争比分析

### 并行算法
- **共享内存**: OpenMP并行循环
- **分布式**: MapReduce框架
- **GPU加速**: CUDA并行模式

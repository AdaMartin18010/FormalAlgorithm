### 10.4 同伦类型论核心 (ASCII)

```
                    同伦类型论 (HoTT) 核心
                    ══════════════════════

    基础对应:
    ═════════
    
    类型论              拓扑学
    ──────              ──────
    类型  A            空间  A
    项    a : A        点    a ∈ A
    恒等  p : a = b    路径  p : a ↝ b
    
    
    路径结构:
    ═════════
    
    恒等路径:          refl : a = a
                       (常值路径)
    
    路径复合:          p : a = b,  q : b = c
                       ───────────────────────
                       p • q : a = c
                       (路径连接)
    
    路径逆:            p : a = b
                       ─────────
                       p⁻¹ : b = a
                       (反向路径)
    
    
    高阶路径 (同伦):
    ════════════════
    
    如果 p, q : a = b 是两条路径，
    则 α : p = q 是"路径之间的路径"
    
       a ═══════ b
         ╲  α  ╱
          ╲   ╱
           ╲ ╱
            ▽
       a ═══════ b
    
    这形成了 ∞-群胚结构
```

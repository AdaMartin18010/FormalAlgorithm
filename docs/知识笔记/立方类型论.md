## 一、核心概念

### 1.1 定义

**正式定义**:
立方类型论（Cubical Type Theory）是同伦类型论的一种变体，为路径和单值公理提供了计算解释。

**核心创新**:
- **区间类型** $\mathbb{I}$：带有端点0和1的特殊类型
- **路径类型** $\text{Path}_A\ a\ b$：作为函数 $\mathbb{I} \to A$ 
- **组合操作**: 路径的连接、逆等可计算
- **粘合 (Glue)**: 使单值性可计算

**语法扩展**:
```
t ::= ... | i | ~i | i ∧ j | i ∨ j  (区间变量和运算)
    | λi.t | t r                     (路径抽象和应用)
    | Glue A φ B | unglue            (粘合类型)
```

**直观解释**:
立方类型论将"路径"显式表示为区间上的函数。就像立方体的边连接顶点一样，路径连接点，路径上的路径连接路径，形成高维结构。

**关键要点**:
- 路径是原始概念（不是归纳定义的相等）
- 单值公理有计算规则
- 高维运输可计算

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 可计算性 | 所有构造都有归约规则 | 与Book HoTT不同 |
| 区间类型 | 原语，非归纳 | De Morgan代数 |
| 路径归纳 | 从函数外延性导出 | 非原始 |
| 正规化 | 猜想成立 | 实践中成立 |
| 一致性和可实现性 | 有模型 | Cohen et al. |

**性质总结**:
1. **可计算单值性**: ua和transport可规约
2. **De Morgan立方体**: 区间形成De Morgan代数
3. **统一框架**: 路径、同伦、等价统一处理

### 1.3 变体

**Cartesian立方类型论** (ABCfHL):
- 基于Cartesian区间
- 不同的组合结构

**De Morgan立方类型论** (CCHM):
- 原始描述
- ~, ∧, ∨运算

**多维多立方** (MDTT):
- 多区间变量
- 高维结构更清晰

## 三、形式化证明

### 3.1 核心定理

**定理 1** (单值性可计算): 
在立方类型论中，ua和transport有明确的归约规则。

**ua归约**:
```
transport (ua e) a ↦ equivFun e a
```

**定理 2** (函数外延性):
可从路径类型定义直接证明。

$$
\text{funExt} : (\forall x, f(x) = g(x)) \to f = g
$$

**证明**:
```agda
funExt : ∀ {A B} {f g : A → B} → (∀ x → f x ≡ g x) → f ≡ g
funExt p i x = p x i
```

### 3.2 组合操作

**连接** ($\text{comp}$):
给定路径的部分边界，填充内部。

```agda
comp : ∀ {A : I → Type} (φ : I) 
       (u : (i : I) → Partial φ (A i))
       (u0 : A i0 [ φ ↦ u i0 ])
     → A i1
```

## 四、算法/方法详解

### 4.1 路径操作算法

```agda
-- 路径连接
_∙_ : ∀ {A} {x y z : A} → Path A x y → Path A y z → Path A x z
p ∙ q = λ i → comp (λ j → A) (i ∨ ~i)
                   (λ j → λ { (i = i0) → p j; (i = i1) → q j })
                   (p i)

-- 路径逆  
_⁻¹ : ∀ {A} {x y : A} → Path A x y → Path A y x
p ⁻¹ = λ i → comp (λ _ → A) (i ∨ ~i)
                  (λ j → λ { (i = i0) → p (~j); (i = i1) → x })
                  x
```

### 4.2 复杂度分析

| 操作 | 复杂度 |
|------|--------|
| 路径应用 | 常数 |
| transport | 依赖类型大小 |
| comp | 高维时复杂 |

## 五、示例与实例

### 5.1 标准示例

**示例 1**: 定义和使用路径

```agda
{-# OPTIONS --cubical #-}

module Example where
  open import Cubical.Core.Everything

  -- 路径定义
  sym : ∀ {A} {x y : A} → x ≡ y → y ≡ x
  sym p = λ i → p (~ i)

  -- 连接
  trans : ∀ {A} {x y z : A} → x ≡ y → y ≡ z → x ≡ z
  trans p q = λ i → comp (λ _ → _) (i ∨ ~ i)
                         (λ j → λ { (i = i0) → p j; (i = i1) → q j })
                         (p i)

  -- 示例: 证明 x ≡ x
  reflPath : ∀ {A} {x : A} → x ≡ x
  reflPath = λ _ → x
```

**示例 2**: 单值性计算

```agda
open import Cubical.Foundations.Univalence

-- 等价
notEquiv : Bool ≃ Bool
notEquiv = isoToEquiv (iso not not (λ {true → refl; false → refl})
                                      (λ {true → refl; false → refl}))

-- 单值性产生路径
notPath : Bool ≡ Bool
notPath = ua notEquiv

-- 运输计算
test : transport notPath true ≡ false
test = refl  -- 自动规约!
```

### 5.2 代码实现

**Cubical Agda完整示例**:
```agda
{-# OPTIONS --cubical --safe #-}

module CubicalExample where

open import Cubical.Core.Everything
open import Cubical.Foundations.Prelude
open import Cubical.Foundations.Univalence
open import Cubical.Data.Bool
open import Cubical.Data.Int

-- 定义圆
module Circle where
  data S¹ : Type where
    base : S¹
    loop : base ≡ base

  -- 递归原理
  S¹-rec : ∀ {ℓ} {A : Type ℓ} (b : A) (l : b ≡ b) → S¹ → A
  S¹-rec b _ base = b
  S¹-rec b l (loop i) = l i

  -- 覆盖空间
  helix : S¹ → Type
  helix base = Int
  helix (loop i) = ua (isoToEquiv posSuccIso) i
    where
    posSuccIso : Iso Int Int
    Iso.fun posSuccIso = sucInt
    Iso.inv posSuccIso = predInt
    Iso.rightInv posSuccIso = sucPred
    Iso.leftInv posSuccIso = predSuc

  -- 缠绕数 (winding number)
  winding : base ≡ base → Int
  winding p = transport (λ i → helix (p i)) (pos 0)

  -- 验证: winding loop = 1
  test : winding loop ≡ pos 1
  test = refl

-- 命题截断
data ∥_∥ {ℓ} (A : Type ℓ) : Type ℓ where
  ∣_∣ : A → ∥ A ∥
  squash : ∀ (x y : ∥ A ∥) (p q : x ≡ y) → p ≡ q

-- 截断归纳
∥∥-rec : ∀ {ℓ ℓ'} {A : Type ℓ} {P : Type ℓ'} →
         isProp P → (A → P) → ∥ A ∥ → P
∥∥-rec Pprop f ∣ x ∣ = f x
∥∥-rec Pprop f (squash x y p q i j) = 
  Pprop (f' x) (f' y) (cong f' p) (cong f' q) i j
  where
  f' = ∥∥-rec Pprop f
```

## 六、习题

### 6.1 理解题 (L1-L2)

1. **路径应用** [难度⭐]
   
   在立方类型论中，证明 `refl ⁻¹ ≡ refl`。
   
   <details>
   <summary>解答</summary>
   
   ```agda
   refl-inv : ∀ {A} {x : A} → (refl {x = x}) ⁻¹ ≡ refl
   refl-inv = refl  -- 由定义自动成立
   ```
   
   在立方类型论中，refl定义为`λ _ → x`，其逆也是`λ _ → x`。
   
   </details>

2. **函数外延性** [难度⭐⭐]
   
   用立方类型论证明 funext。
   
   <details>
   <summary>解答</summary>
   
   ```agda
   funext : ∀ {A B} {f g : A → B} → (∀ x → f x ≡ g x) → f ≡ g
   funext h = λ i → λ x → h x i
   ```
   
   路径`f ≡ g`是函数`I → (A → B)`，通过交换参数顺序构造。
   
   </details>

### 6.2 应用题 (L3)

3. **方形填充** [难度⭐⭐]
   
   给定路径 p : x ≡ y, q : x ≡ z, r : y ≡ w，填充方形。
   
   <details>
   <summary>解答</summary>
   
   ```agda
   fillSquare : ∀ {A} {x y z w : A}
                (p : x ≡ y) (q : x ≡ z) (r : y ≡ w)
              → PathP (λ i → q i ≡ r i) p (trans q r)
   fillSquare p q r = λ i j → 
     comp (λ k → A) (j ∨ ~j ∨ i ∨ ~i)
       (λ k → λ { (j = i0) → p (i ∧ k)
                ; (j = i1) → r (i ∧ k)
                ; (i = i0) → q j
                ; (i = i1) → r j
                })
       (q (j ∧ ~i))
   ```
   
   </details>

### 6.3 证明题 (L4-L5)

4. **运输与路径** [难度⭐⭐⭐]
   
   证明 `transport (λ i → p i ≡ q i) r` 计算为什么？
   
   <details>
   <summary>解答</summary>
   
   ```agda
   transport-path : ∀ {A} {x y z : A} (p : x ≡ y) (q : x ≡ z) (r : y ≡ z)
                  → transport (λ i → p i ≡ q i) r ≡ (sym p ∙ r ∙ q)
   transport-path p q r = 
     transport (λ i → p i ≡ q i) r
       ≡⟨ transportPath p q r ⟩
     sym p ∙ r ∙ q ∎
   ```
   
   运输路径间的相等转化为路径的连接。
   
   </details>

5. **圆的基本群** [难度⭐⭐⭐⭐]
   
   在立方类型论中，证明 `winding (loop ∙ loop) ≡ 2`。
   
   <details>
   <summary>解答</summary>
   
   ```agda
   winding-double : winding (loop ∙ loop) ≡ pos 2
   winding-double = 
     winding (loop ∙ loop)
       ≡⟨ cong winding (sym (refl {x = loop} ∙ refl {x = loop})) ⟩
     winding (λ i → compPath-filler loop loop i i1)
       ≡⟨ transportComp _ _ _ ⟩
     pos 2 ∎
   ```
   
   利用螺旋覆盖空间的定义，运输两次succInt得到+2。
   
   </details>

## 一、核心概念

### 1.1 定义

**正式定义**:
二叉搜索树(BST)是一种二叉树，满足**BST性质**：

- 对于任意节点x，左子树所有节点的键 ≤ x的键
- 对于任意节点x，右子树所有节点的键 ≥ x的键

**基本操作复杂度**:

| 操作 | 平均 | 最坏 |
|------|------|------|
| 搜索 | O(log n) | O(n) |
| 插入 | O(log n) | O(n) |
| 删除 | O(log n) | O(n) |

**遍历方式**:

- 中序遍历：按键的升序输出
- 前序/后序：其他应用场景

### 1.2 BST变体

| 变体 | 特点 | 最坏复杂度 |
|------|------|------------|
| 平衡BST(AVL) | 左右子树高度差≤1 | O(log n) |
| 红黑树 | 黑高平衡，颜色约束 | O(log n) |
| Splay树 | 访问后旋转到根 | 摊还O(log n) |
| Treap | 随机优先级 | 期望O(log n) |

## 二、算法详解

### 2.1 基本操作

```python
class TreeNode:
    def __init__(self, key, value=None):
        self.key = key
        self.value = value
        self.left = None
        self.right = None
        self.parent = None


class BST:
    def __init__(self):
        self.root = None

    def search(self, key):
        """搜索，O(h)"""
        node = self.root
        while node and node.key != key:
            if key < node.key:
                node = node.left
            else:
                node = node.right
        return node

    def insert(self, key, value=None):
        """插入，O(h)"""
        new_node = TreeNode(key, value)
        parent = None
        current = self.root

        while current:
            parent = current
            if key < current.key:
                current = current.left
            else:
                current = current.right

        new_node.parent = parent
        if not parent:
            self.root = new_node
        elif key < parent.key:
            parent.left = new_node
        else:
            parent.right = new_node

    def delete(self, key):
        """删除，O(h)"""
        node = self.search(key)
        if not node:
            return

        # 情况1: 无左子节点
        if not node.left:
            self._transplant(node, node.right)
        # 情况2: 无右子节点
        elif not node.right:
            self._transplant(node, node.left)
        # 情况3: 有两个子节点
        else:
            successor = self._minimum(node.right)
            if successor.parent != node:
                self._transplant(successor, successor.right)
                successor.right = node.right
                successor.right.parent = successor
            self._transplant(node, successor)
            successor.left = node.left
            successor.left.parent = successor

    def _transplant(self, u, v):
        """用v替换u"""
        if not u.parent:
            self.root = v
        elif u == u.parent.left:
            u.parent.left = v
        else:
            u.parent.right = v
        if v:
            v.parent = u.parent

    def _minimum(self, node):
        """子树最小值"""
        while node.left:
            node = node.left
        return node

    def inorder(self):
        """中序遍历"""
        result = []
        self._inorder_recursive(self.root, result)
        return result

    def _inorder_recursive(self, node, result):
        if node:
            self._inorder_recursive(node.left, result)
            result.append(node.key)
            self._inorder_recursive(node.right, result)
```

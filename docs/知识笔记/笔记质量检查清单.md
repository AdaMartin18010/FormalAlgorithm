# 笔记质量检查清单

> **版本**: v1.0
> **更新**: 2026-04-09
> **用途**: 知识笔记编写完成后的质量检查

---

## 使用说明

本清单用于在知识笔记编写完成后进行系统性的质量检查。每个检查项分为三个等级：

- **必须 (M)**: 不满足则不能发布
- **应该 (S)**: 强烈建议满足，特殊情况可例外
- **建议 (C)**: 最好满足，非强制

检查完成后，在方括号中标记：
- `[x]` 已检查通过
- `[-]` 不适用
- `[ ]` 未通过，需修改

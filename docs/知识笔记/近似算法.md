## 一、核心概念

### 1.1 定义

**正式定义**:
对于优化问题，**近似比(Approximation Ratio)**$\rho(n)$定义为：

$$\max\left(\frac{C}{C^*}, \frac{C^*}{C}\right) \leq \rho(n)$$

其中$C$是近似解的值，$C^*$是最优解的值。

**近似比分类**:
- **PTAS**: 对任意$\epsilon>0$，存在$(1+\epsilon)$-近似算法，时间多项式于$n$但可指数于$1/\epsilon$
- **FPTAS**: 完全PTAS，时间多项式于$n$和$1/\epsilon$
- **常数近似**: $O(1)$近似比
- **对数近似**: $O(\log n)$近似比

**近似难度**: 某些问题不存在常数近似（除非P=NP）。

**直观解释**:
当面对NP难问题时，我们放弃寻找最优解，转而寻找"足够好"的解。近似算法给出了"多坏算坏"的量化标准——近似比保证我们的解在最优解的常数倍范围内。

### 1.2 属性

| 问题 | 最佳已知近似比 | 近似难度 |
|------|----------------|----------|
| 顶点覆盖 | 2 | 难以低于1.36 |
| 集合覆盖 | $O(\log n)$ | $O(\log n)$紧 |
| TSP | 2（度量） | 无常数近似（一般） |
| 最大割 | 0.878 | 难以高于0.941 |
| 背包 | FPTAS | - |

### 1.3 设计技术

**贪心法**: 局部最优选择
**局部搜索**: 改进邻域解直到局部最优
**线性规划舍入**: LP松弛+舍入到整数
**原始对偶**: 同时构建原始和对偶解
**随机舍入**: 概率舍入

## 三、形式化证明

### 3.1 核心定理

**定理 1**: 顶点覆盖的贪心算法是2-近似。

**证明**:
```
设A是算法选中的边集，这些边构成匹配（无公共顶点）。

任何顶点覆盖必须包含A中每条边的至少一个端点，
因此|OPT| >= |A|。

算法选择2|A|个顶点，因此|ALG| = 2|A| <= 2|OPT|。∎
```

**定理 2**: 度量TSP的MST启发式是2-近似。

*证明*: MST权重 <= TSP最优值，走两遍MST得到欧拉回路，shortcut得TSP路径。∎

## 四、算法详解

### 4.1 顶点覆盖近似

**算法**:
```
APPROX-VERTEX-COVER(G):
    C = ∅
    E' = G.E
    while E' != ∅ do
        任取(u,v) ∈ E'
        C = C ∪ {u, v}
        从E'中删除与u或v关联的所有边
    return C
```

**近似比**: 2

### 4.2 Christofides算法

**度量TSP的1.5-近似**:
1. 找到MST T
2. 找到T中奇度顶点的最小权完美匹配M
3. T + M是欧拉图，找欧拉回路
4. Shortcut得哈密尔顿回路

## 五、示例与实例

### 5.2 代码实现

```python
# 顶点覆盖2-近似
class VertexCoverApprox:
    def __init__(self, n):
        self.n = n
        self.adj = [[] for _ in range(n)]
    
    def add_edge(self, u, v):
        self.adj[u].append(v)
        self.adj[v].append(u)
    
    def approx_vertex_cover(self):
        """2-近似顶点覆盖"""
        covered = [False] * self.n
        cover = set()
        
        for u in range(self.n):
            if covered[u]:
                continue
            for v in self.adj[u]:
                if not covered[v]:
                    # 选择边(u,v)
                    cover.add(u)
                    cover.add(v)
                    covered[u] = covered[v] = True
                    break
        
        return cover


# 集合覆盖贪心近似 (ln n)
def set_cover_greedy(universe, sets):
    """
    universe: 需要覆盖的元素集合
    sets: [(cost, {elements}), ...]
    返回: (总成本, 选中的集合索引)
    """
    uncovered = set(universe)
    selected = []
    total_cost = 0
    
    while uncovered:
        # 选择单位成本覆盖最多的集合
        best_ratio = float('inf')
        best_idx = -1
        
        for i, (cost, elements) in enumerate(sets):
            covered = len(elements & uncovered)
            if covered > 0:
                ratio = cost / covered
                if ratio < best_ratio:
                    best_ratio = ratio
                    best_idx = i
        
        if best_idx == -1:
            break
        
        cost, elements = sets[best_idx]
        total_cost += cost
        uncovered -= elements
        selected.append(best_idx)
    
    return total_cost, selected


# TSP 2-近似（MST启发式）
import heapq

def tsp_2_approx(dist):
    """
    dist: 距离矩阵（满足三角不等式）
    返回: 近似最优回路和长度
    """
    n = len(dist)
    
    # Prim算法求MST
    mst = [[] for _ in range(n)]
    visited = [False] * n
    min_edge = [(0, 0, -1)]  # (weight, to, from)
    parent = [-1] * n
    
    while min_edge:
        w, u, p = heapq.heappop(min_edge)
        if visited[u]:
            continue
        visited[u] = True
        if p != -1:
            mst[p].append(u)
            mst[u].append(p)
        
        for v in range(n):
            if not visited[v]:
                heapq.heappush(min_edge, (dist[u][v], v, u))
    
    # DFS遍历MST（走两遍）+ shortcut
    tour = []
    visited = [False] * n
    
    def dfs(u):
        visited[u] = True
        tour.append(u)
        for v in mst[u]:
            if not visited[v]:
                dfs(v)
    
    dfs(0)
    tour.append(0)  # 回到起点
    
    # 计算总长度
    total = sum(dist[tour[i]][tour[i+1]] for i in range(len(tour)-1))
    return tour, total
```

## 六、习题

### 6.1 理解题

1. **为什么顶点覆盖贪心是2-近似** [难度⭐]

<details>
<summary>解答</summary>

算法选择的边构成匹配（无共享顶点），因此至少|匹配|个顶点在OPT中。算法选择2|匹配|个顶点，近似比为2。

</details>

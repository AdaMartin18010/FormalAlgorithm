## 一、核心概念

### 1.1 定义

**正式定义**:
λ演算是由Alonzo Church于1930年代提出的形式系统，是计算理论的基础模型。它由以下文法定义：

```
项 t ::= x           (变量)
       | λx.t       (λ抽象/函数定义)
       | t t        (函数应用)
```

**语法约定**:

- **作用域**: λ抽象体尽可能向右延伸：`λx.λy.x y` = `λx.(λy.(x y))`
- **应用左结合**: `f g h` = `(f g) h`
- **α等价**: λ-bound变量可重命名：`λx.x` ≡ `λy.y`

**直观解释**:
λ演算是**最小的通用编程语言**，只包含三种构造：

- **变量**: 引用函数的参数
- **抽象**: 创建匿名函数 `λx.t` ≈ "给定x，返回t"
- **应用**: 将函数应用于参数

**关键要点**:

- λ演算与图灵机等价（丘奇-图灵论题）
- 所有可计算函数都可编码为λ项
- 是高阶函数式语言的理论基础

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 图灵完备 | 可模拟任何图灵机 | λ演算 = 图灵机 = 可计算 |
| 高阶性 | 函数可作为参数和返回值 | 一阶函数公民 |
| 无类型/有类型 | 存在无类型和多种类型版本 | 简单类型、多态、依赖类型 |
| 归约语义 | 通过β/η归约求值 | 合流性、标准化 |
| 变量绑定 | 静态词法作用域 | α转换、捕获避免替换 |

**性质总结**:

1. **合流性 (Church-Rosser)**: 无论归约顺序如何，结果相同（如果存在）
2. **标准化**: 存在标准归约策略（最左最外优先）
3. **不动点**: 每个λ项都有不动点（Y组合子）

### 1.3 变体

**无类型λ演算 (Untyped)**:

- 定义: 无任何类型限制的原始λ演算
- 与标准形式的区别: 允许任意自应用如 `(λx.x x)(λx.x x)`
- 适用场景: 理论研究、图灵完备性证明

**简单类型λ演算 (λ→)**:

- 定义: Church/Curry风格的简单类型系统
- 与标准形式的区别: 禁止自应用，保证标准化
- 适用场景: 编程语言类型系统基础

**多态λ演算 (System F)**:

- 定义: Girard-Reynolds多态类型系统
- 与标准形式的区别: 引入类型抽象 `Λα.t` 和类型应用 `t[τ]`
- 适用场景: 泛型编程的理论基础

## 三、形式化证明

### 3.1 核心定理

**定理 1** (Church-Rosser / 合流性):
若 $M \twoheadrightarrow_\beta N_1$ 且 $M \twoheadrightarrow_\beta N_2$，则存在 $P$ 使得 $N_1 \twoheadrightarrow_\beta P$ 且 $N_2 \twoheadrightarrow_\beta P$。

**证明概要**:

1. 首先证明平行归约 (parallel reduction) 具有菱形性质
2. 通过Tait-Martin-Löf方法建立合流性
3. 使用替换引理处理复杂情况

**关键步骤**:

```
        M
       / \
      ↓   ↓
     N1   N2
      \   /
       ↓ ↓
        P
```

**证明要点分析**:

1. **菱形性质**: 单步归约的局部合流
2. **归纳结构**: 对项的结构进行归纳
3. **替换引理**: 保证归约在替换下的保持性

**定理 2** (标准化定理):
若一个项存在范式，则最左最外归约策略必定找到它。

### 3.2 辅助引理

**引理 1** (替换引理):
若 $x \not= y$ 且 $x \not\in FV(L)$，则：
$$M[x:=N][y:=L] = M[y:=L][x:=N[y:=L]]$$

**引理 2** (α等价兼容性):
若 $M =_\alpha N$，则对于任意上下文 $C[\cdot]$，有 $C[M] =_\alpha C[N]$。

## 四、算法/方法详解

### 4.1 算法描述

**β归约算法**:

```
算法: β归约 (单步)
输入: λ项 M
输出: 归约后的项或不可归约

1. case M of
2.   (λx.P) Q → P[x:=Q]        (β归约)
3.   P Q →
4.     if P可归约 → P' Q
5.     else if Q可归约 → P Q'
6.     else → 不可归约
7.   λx.P → λx.P' (若P可归约)
8.   _ → 不可归约
```

**捕获避免替换**:

```
x[x:=N] = N
y[x:=N] = y          (若 y ≠ x)
(P Q)[x:=N] = (P[x:=N]) (Q[x:=N])
(λy.P)[x:=N] = λy.(P[x:=N])  (若 y ∉ FV(N) 且 y ≠ x)
              = λz.(P[y:=z][x:=N]) (否则，fresh z)
```

### 4.2 正确性分析

**替换正确性**: 捕获避免替换保持了α等价类。

**证明**:

- **基础**: 变量情况直接由定义保证
- **归纳**: 抽象情况通过α转换避免捕获
- **应用**: 归纳假设保证子项正确

### 4.3 复杂度分析

**归约复杂度**:

- 最坏情况: 不可判定（停机问题可归约）
- 范式检测: 递归可枚举但不可判定
- 单步替换: $O(|M| \times |N|)$ 最坏情况

## 五、示例与实例

### 5.1 标准示例

**示例 1**: Church数（自然数编码）

**定义**:
$$\begin{align}
\overline{0} &= \lambda f.\lambda x.x \\
\overline{1} &= \lambda f.\lambda x.f\ x \\
\overline{2} &= \lambda f.\lambda x.f\ (f\ x) \\
\overline{n} &= \lambda f.\lambda x.f^n\ x
\end{align}$$

**后继函数**:
$$\text{SUCC} = \lambda n.\lambda f.\lambda x.f\ (n\ f\ x)$$

**加法**:
$$\text{PLUS} = \lambda m.\lambda n.\lambda f.\lambda x.m\ f\ (n\ f\ x)$$

**验证**: $\text{PLUS}\ \overline{2}\ \overline{3} \twoheadrightarrow_\beta \overline{5}$

---

**示例 2**: Y组合子（递归实现）

**定义**:
$$Y = \lambda f.(\lambda x.f\ (x\ x))\ (\lambda x.f\ (x\ x))$$

**性质**: 对任意 $F$，有 $Y\ F =_\beta F\ (Y\ F)$（不动点性质）

### 5.2 代码实现

**Coq实现**:
```coq
(* λ项的归纳定义 *)
Inductive term : Type :=
  | var : nat -> term
  | abs : nat -> term -> term
  | app : term -> term -> term.

(* 替换的归纳定义（简化版） *)
Fixpoint subst (t : term) (x : nat) (s : term) : term :=
  match t with
  | var y => if x =? y then s else var y
  | abs y t' => if x =? y then abs y t'
                else abs y (subst t' x s)
  | app t1 t2 => app (subst t1 x s) (subst t2 x s)
  end.

(* 单步β归约关系 *)
Inductive beta : term -> term -> Prop :=
  | beta_redex : forall x t s,
      beta (app (abs x t) s) (subst t x s)
  | beta_app_l : forall t1 t1' t2,
      beta t1 t1' -> beta (app t1 t2) (app t1' t2)
  | beta_app_r : forall t1 t2 t2',
      beta t2 t2' -> beta (app t1 t2) (app t1 t2')
  | beta_abs : forall x t t',
      beta t t' -> beta (abs x t) (abs x t').
```

**Agda实现**:
```agda
-- λ演算语法
open import Data.Nat
open import Data.List

Id = ℕ

data Term : Set where
  var : Id → Term
  ƛ_⇒_ : Id → Term → Term
  _·_ : Term → Term → Term

-- 自由变量
FV : Term → List Id
FV (var x) = [ x ]
FV (ƛ x ⇒ t) = filter (λ y → not (x == y)) (FV t)
FV (t · s) = FV t ++ FV s

-- 简单替换（非捕获避免）
_[_/_] : Term → Term → Id → Term
var x [ s / y ] with x == y
... | true  = s
... | false = var x
(ƛ x ⇒ t) [ s / y ] with x == y
... | true  = ƛ x ⇒ t
... | false = ƛ x ⇒ (t [ s / y ])
(t · u) [ s / y ] = (t [ s / y ]) · (u [ s / y ])
```

**Lean实现**:
```lean
-- λ项定义
inductive term
| var : ℕ → term
| abs : ℕ → term → term
| app : term → term → term

open term

-- 局部变量替换（简化）
def subst (t : term) (x : ℕ) (s : term) : term :=
  term.rec_on t
    (λ y, if x = y then s else var y)
    (λ y t' rec, if x = y then abs y t' else abs y rec)
    (λ t1 t2 rec1 rec2, app rec1 rec2)

-- β归约关系
inductive beta : term → term → Prop
| beta_redex : ∀ x t s, beta (app (abs x t) s) (subst t x s)
| app_l : ∀ t1 t1' t2, beta t1 t1' → beta (app t1 t2) (app t1' t2)
| app_r : ∀ t1 t2 t2', beta t2 t2' → beta (app t1 t2) (app t1 t2')
| abs : ∀ x t t', beta t t' → beta (abs x t) (abs x t')
```

### 5.3 反例

**常见错误1**: 变量捕获
```
错误替换: (λx.λy.x y)[x:=y]
错误结果: λy.y y   (y被捕获！)
正确结果: λz.y z   (α转换后替换)
```
**错误原因**: 未进行α转换直接替换，导致自由变量被绑定
**正确做法**: 替换前先重命名冲突的绑定变量

**常见错误2**: 忽略α等价
```
错误理解: λx.x 与 λy.y 是不同的函数
正确理解: 它们是α等价的，代表相同函数
```

## 六、习题

### 6.1 理解题 (L1-L2)

1. **α等价判定** [难度⭐]

   判断下列项是否α等价：
   - `λx.λy.x y` 与 `λy.λx.y x`
   - `λx.λy.x` 与 `λy.λx.y`
   - `λx.x x` 与 `λy.y y`

   <details>
   <summary>解答</summary>

   1. 等价：只是变量名不同
   2. 不等价：前者返回第一个参数，后者返回第二个参数
   3. 等价：自应用的α变体

   </details>

2. **β归约步骤** [难度⭐]

   将 `(λx.λy.x y)(λz.z)` 归约到范式，写出每一步。

   <details>
   <summary>解答</summary>

   ```
   (λx.λy.x y)(λz.z)
   →β λy.(λz.z) y
   →β λy.y
   ```

   </details>

### 6.2 应用题 (L3)

3. **Church数乘法** [难度⭐⭐]

   定义乘法函数 `MULT` 使用Church数编码，并验证 `MULT 2 3 = 6`。

   <details>
   <summary>解答</summary>

   ```
   MULT = λm.λn.λf.m (n f)

   验证: MULT 2 3
   = (λm.λn.λf.m (n f)) 2 3
   = λf.2 (3 f)
   = λf.λx.2 (3 f) x
   = λf.λx.2 (λx.f(f(f x))) x
   = λf.λx.(λx.f(f x)) ((λx.f(f(f x))) x)
   → λf.λx.(λx.f(f x)) (f(f(f x)))
   → λf.λx.f(f(f(f(f(f x)))))
   = 6
   ```

   </details>

4. **条件表达式编码** [难度⭐⭐]

   使用Church编码定义布尔值 TRUE、FALSE 和 IF 函数，使得：
   - `IF TRUE a b → a`
   - `IF FALSE a b → b`

   <details>
   <summary>解答</summary>

   ```
   TRUE = λt.λf.t
   FALSE = λt.λf.f
   IF = λb.λt.λf.b t f

   验证:
   IF TRUE a b = (λb.λt.λf.b t f) TRUE a b
               → TRUE a b
               = (λt.λf.t) a b
               → a
   ```

   </details>

### 6.3 证明题 (L4-L5)

5. **替换性质** [难度⭐⭐⭐]

   证明：若 $x \not\in FV(M)$，则 $M[x:=N] =_\alpha M$。

   <details>
   <summary>解答</summary>

   对 $M$ 的结构进行归纳：

   **基础情况**: $M = y$
   - 若 $y = x$: 与 $x \not\in FV(M)$ 矛盾
   - 若 $y \neq x$: $y[x:=N] = y$ ✓

   **归纳步骤**: $M = P\ Q$
   - $FV(M) = FV(P) \cup FV(Q)$
   - 由归纳假设，$P[x:=N] = P$ 且 $Q[x:=N] = Q$
   - 因此 $(P\ Q)[x:=N] = P\ Q$ ✓

   **抽象情况**: $M = \lambda y.P$
   - 若 $y = x$: $(\lambda x.P)[x:=N] = \lambda x.P$（由定义）✓
   - 若 $y \neq x$: $(\lambda y.P)[x:=N] = \lambda y.(P[x:=N]) = \lambda y.P$（归纳假设）✓

   </details>

6. **合流性应用** [难度⭐⭐⭐⭐]

   设 $M = (\lambda x.x x)((\lambda y.y)(\lambda z.z))$，展示两种不同的归约序列最终汇合。

   <details>
   <summary>解答</summary>

   设 $I = \lambda z.z$

   **序列1**（先归约参数）:
   ```
   (λx.x x)((λy.y) I)
   → (λx.x x) I
   → I I
   → I
   ```

   **序列2**（先应用函数）:
   ```
   (λx.x x)((λy.y) I)
   → ((λy.y) I) ((λy.y) I)
   → I ((λy.y) I)
   → I I
   → I
   ```

   两序列均归约到 $I$，验证合流性。

   </details>

**何时使用简单类型**:
- 实际编程语言设计

## 一、核心概念

### 1.1 定义

**正式定义**:
随机算法是在执行过程中使用随机数做决策的算法。根据输出正确性保证，分为两类：

**Las Vegas算法**: 总是产生正确结果，运行时间是随机变量。
- 期望运行时间上界
- 例子：随机快速排序、随机选择

**Monte Carlo算法**: 运行时间是确定的，但结果可能错误（有界概率）。
- 错误概率上界
- 例子：素性测试、近似计数

**概率分析 vs 随机算法**:
- 概率分析：输入服从某种分布
- 随机算法：算法本身使用随机性

**直观解释**:
随机算法利用"随机选择"避免最坏情况。就像快速排序随机选pivot避免已排序输入的$O(n^2)$，或就像Pollard's rho用随机游走分解整数。随机性既是"逃离"最坏情况的工具，也是"探索"解空间的手段。

### 1.2 属性

| 属性 | Las Vegas | Monte Carlo |
|------|-----------|-------------|
| 正确性 | 确定正确 | 可能错误 |
| 运行时间 | 随机 | 确定 |
| 错误概率 | 0 | $>0$但可控制 |
| 可重复性 | 结果确定 | 结果可能不同 |

### 1.3 变体

**随机舍入**: 将LP松弛解随机舍入到整数
**马尔可夫链蒙特卡洛(MCMC)**: 复杂分布采样
**随机游走**: 图探索、PageRank

## 三、形式化证明

### 3.1 核心定理

**定理 1**: 随机快速排序期望比较次数为$O(n \log n)$。

**证明**:
```
设X为总比较次数，X_{ij}指示i和j是否比较。

E[X] = Σ_{i<j} Pr[i和j比较]

i和j比较当且仅当在{i,...,j}中第一个被选为pivot的是i或j。

Pr[i和j比较] = 2/(j-i+1)

E[X] = Σ_{i<j} 2/(j-i+1)
     ≤ Σ_{i=1}^n Σ_{k=1}^n 2/k
     = O(n log n) ∎
```

## 四、算法详解

### 4.1 随机快速排序

**算法**:
```
RANDOMIZED-QUICKSORT(A, p, r):
    if p < r then
        q = RANDOMIZED-PARTITION(A, p, r)
        RANDOMIZED-QUICKSORT(A, p, q-1)
        RANDOMIZED-QUICKSORT(A, q+1, r)

RANDOMIZED-PARTITION(A, p, r):
    i = RANDOM(p, r)
    exchange A[r] with A[i]
    return PARTITION(A, p, r)
```

## 五、示例与实例

### 5.2 代码实现

```python
import random

def randomized_partition(arr, low, high):
    # 随机选择pivot
    pivot_idx = random.randint(low, high)
    arr[pivot_idx], arr[high] = arr[high], arr[pivot_idx]
    
    pivot = arr[high]
    i = low - 1
    
    for j in range(low, high):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1

def randomized_quicksort(arr, low=None, high=None):
    if low is None:
        low, high = 0, len(arr) - 1
    
    if low < high:
        pi = randomized_partition(arr, low, high)
        randomized_quicksort(arr, low, pi - 1)
        randomized_quicksort(arr, pi + 1, high)
    
    return arr


# Monte Carlo素性测试（Miller-Rabin）
def miller_rabin(n, k=5):
    """Miller-Rabin素性测试，错误概率 <= 4^{-k}"""
    if n < 2:
        return False
    if n in (2, 3):
        return True
    if n % 2 == 0:
        return False
    
    # 写成 n-1 = 2^r * d
    r, d = 0, n - 1
    while d % 2 == 0:
        r += 1
        d //= 2
    
    for _ in range(k):
        a = random.randrange(2, n - 1)
        x = pow(a, d, n)
        
        if x == 1 or x == n - 1:
            continue
        
        for _ in range(r - 1):
            x = pow(x, 2, n)
            if x == n - 1:
                break
        else:
            return False  # 合数
    
    return True  # 可能是素数
```

### 6.1 理解题

1. **Las Vegas vs Monte Carlo** [难度⭐]

<details>
<summary>解答</summary>

Las Vegas: 结果总是正确，时间随机。Monte Carlo: 时间固定，结果可能错误。可以通过重复运行Monte Carlo算法将其转化为Las Vegas算法。

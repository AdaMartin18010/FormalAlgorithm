## 一、核心概念

### 1.1 定义

**正式定义**:
**图神经网络（GNN）**是在图结构数据上学习的神经网络，通过消息传递机制聚合邻居信息。

**消息传递框架**:
$$h_v^{(l+1)} = \text{UPDATE}^{(l)}\left(h_v^{(l)}, \text{AGGREGATE}^{(l)}\left(\{h_u^{(l)} : u \in \mathcal{N}(v)\}\right)\right)$$

**图卷积网络（GCN）**:
$$H^{(l+1)} = \sigma\left(\tilde{D}^{-1/2}\tilde{A}\tilde{D}^{-1/2}H^{(l)}W^{(l)}\right)$$

其中 $\tilde{A} = A + I$ 为加自环的邻接矩阵，$\tilde{D}$ 为度矩阵。

**图注意力网络（GAT）**:
$$h_v^{(l+1)} = \sigma\left(\sum_{u \in \mathcal{N}(v)} \alpha_{vu}^{(l)} W^{(l)} h_u^{(l)}\right)$$

注意力系数 $\alpha_{vu}$ 通过可学习的注意力机制计算。

**谱图卷积**: 基于图傅里叶变换，$y = U g_\theta(\Lambda) U^T x$

**直观解释**:
GNN就像"社交网络中的信息传播"——每个人的观点（节点特征）受朋友（邻居）影响，通过多轮交流（层），每个人的观点越来越反映整个社区的结构。图卷积就像"朋友圈的平均化"，图注意力则是"重视某些朋友胜过其他朋友"。

**关键要点**:
- **置换不变性**: 节点顺序不影响结果
- **局部性**: 只依赖邻居信息
- **表达能力**: 区分不同图结构的能力
- **过平滑**: 深层GNN节点表示趋同问题

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 置换不变性 | 节点重编号不改变输出 | 图的基本对称性 |
| 局部性 | 感受野随层数扩展 | $k$ 层覆盖 $k$-hop邻居 |
| 归纳性 | 可泛化到未见图 | 比转导学习更通用 |
| 可扩展性 | 大图需要采样 | GraphSAGE、Cluster-GCN |

### 1.3 变体

**GCN**: 谱域图卷积的简化
**GraphSAGE**: 归纳式学习，邻居采样
**GAT**: 注意力机制加权邻居
**GIN**: 图同构网络，最大化表达能力
**MPNN**: 通用消息传递框架

## 三、形式化证明

### 3.1 核心定理

**定理 1 (GCN是谱卷积的一阶近似)**: GCN层近似于局部谱滤波器。

**证明**:
谱卷积 $y = U g_\theta(\Lambda) U^T x$ 计算昂贵。

Chebyshev多项式近似：
$$g_\theta * x \approx \sum_{k=0}^K \theta_k T_k(\tilde{L}) x$$

其中 $\tilde{L} = 2L/\lambda_{max} - I$，$T_k$ 为Chebyshev多项式。

取 $K=1$，$\lambda_{max} \approx 2$：
$$g_\theta * x \approx \theta_0 x + \theta_1 (L - I)x = \theta_0 x - \theta_1 D^{-1/2}AD^{-1/2}x$$

进一步约束 $\theta = \theta_0 = -\theta_1$，并加自环，得到GCN形式。

∎

**定理 2 (GIN的表达能力)**: GIN在区分图结构方面与WL图同构测试一样强大。

*证明*: 通过设计聚合函数为单射，GIN可模拟WL迭代。∎

### 3.2 辅助引理

**引理 1 (拉普拉斯矩阵性质)**: $L = D - A$ 是半正定的，特征值 $0 = \lambda_1 \leq \lambda_2 \leq \cdots \leq \lambda_n$。

*证明*: $x^T L x = \sum_{(i,j) \in E} (x_i - x_j)^2 \geq 0$。∎

## 四、算法/方法详解

### 4.1 算法描述

**GCN前向传播**:
```
输入: 特征 X, 邻接矩阵 A
参数: 权重 W^(l), 层数 L

A_tilde = A + I  // 加自环
D_tilde = diag(sum(A_tilde, axis=1))
// 对称归一化
A_norm = D_tilde^(-1/2) * A_tilde * D_tilde^(-1/2)

H = X
for l = 1 to L do
    H = σ(A_norm * H * W^(l))
end for
return H
```

**GraphSAGE**:
```
输入: 特征 X, 邻接表
参数: 权重 W^(l), 聚合函数 AGG

for each node v do
    h_v^(0) = X[v]
end for

for l = 1 to L do
    for each node v do
        // 采样并聚合邻居
        N = sample(neighbors(v), k)
        h_N = AGG({h_u^(l-1) for u in N})
        h_v^(l) = σ(W^(l) * concat(h_v^(l-1), h_N))
    end for
end for
```

## 五、示例与实例

### 5.1 GCN节点分类

**Cora数据集**: 2708篇论文，7个类别
**特征**: 词袋表示，1433维
**图**: 引用关系

**2层GCN准确率**: ~81%

### 5.2 代码实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class GCNLayer(nn.Module):
    def __init__(self, in_features, out_features):
        super().__init__()
        self.linear = nn.Linear(in_features, out_features)
    
    def forward(self, x, adj):
        # adj 已归一化: D^(-1/2) * (A+I) * D^(-1/2)
        x = self.linear(x)
        x = torch.matmul(adj, x)
        return x

class GCN(nn.Module):
    def __init__(self, nfeat, nhid, nclass, dropout=0.5):
        super().__init__()
        self.gc1 = GCNLayer(nfeat, nhid)
        self.gc2 = GCNLayer(nhid, nclass)
        self.dropout = dropout
    
    def forward(self, x, adj):
        x = F.relu(self.gc1(x, adj))
        x = F.dropout(x, self.dropout, training=self.training)
        x = self.gc2(x, adj)
        return F.log_softmax(x, dim=1)

# 归一化邻接矩阵
def normalize_adj(adj):
    """对称归一化"""
    adj = adj + torch.eye(adj.size(0))  # 加自环
    rowsum = adj.sum(1)
    d_inv_sqrt = torch.pow(rowsum, -0.5)
    d_inv_sqrt[torch.isinf(d_inv_sqrt)] = 0.
    d_mat_inv_sqrt = torch.diag(d_inv_sqrt)
    return d_mat_inv_sqrt @ adj @ d_mat_inv_sqrt
```

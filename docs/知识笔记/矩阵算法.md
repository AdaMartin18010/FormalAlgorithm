## 一、核心概念

### 1.1 定义

**正式定义**:
**Strassen算法**: 矩阵乘法分治算法，复杂度 $O(n^{\log_2 7}) \approx O(n^{2.81})$。

**矩阵求逆**: 对于可逆矩阵 $A$，求 $A^{-1}$ 使得 $AA^{-1} = I$。

**行列式计算**: $\det(A)$，可用于判断可逆性、求解线性方程组。

**LU分解**: $A = LU$，其中 $L$ 为下三角，$U$ 为上三角。

**QR分解**: $A = QR$，其中 $Q$ 为正交矩阵，$R$ 为上三角。

**直观解释**:
矩阵算法就像"线性代数的高效计算器"——Strassen发现矩阵乘法不必按定义 $O(n^3)$ 计算，通过巧妙分治可更快；矩阵求逆就像"解方程组的通用钥匙"；行列式则是"矩阵的缩放因子"，告诉我们在变换中面积/体积如何变化。

**关键要点**:
- **Strassen**: 首次突破 $O(n^3)$，实用规模 $n > 100$
- **数值稳定性**: 直接求逆不稳定，常用LU分解
- **并行性**: 矩阵运算适合GPU并行
- **稀疏矩阵**: 特殊结构可加速

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 复杂度 | Strassen $O(n^{2.81})$ | 理论最优 $< O(n^{2.373})$ |
| 数值稳定性 | LU分解优于直接求逆 | 部分主元增强稳定 |
| 并行性 | 矩阵乘法易并行 | GPU加速 |
| 内存局部性 | 分块优化缓存 | 实际性能关键 |

### 1.3 变体

**Coppersmith-Winograd**: $O(n^{2.376})$，理论突破
**激光法**: 最新理论 $O(n^{2.373})$
**Strassen-Winograd**: 常数优化
**块矩阵算法**: 缓存友好实现

## 三、形式化证明

### 3.1 核心定理

**定理 1 (Strassen正确性)**: Strassen算法正确计算矩阵乘积。

**证明**:
对于 $2 \times 2$ 分块矩阵：
$$\begin{pmatrix} A_{11} & A_{12} \\ A_{21} & A_{22} \end{pmatrix} \times \begin{pmatrix} B_{11} & B_{12} \\ B_{21} & B_{22} \end{pmatrix}$$

定义7个乘积：
$$\begin{aligned}
M_1 &= (A_{11} + A_{22})(B_{11} + B_{22}) \\
M_2 &= (A_{21} + A_{22})B_{11} \\
M_3 &= A_{11}(B_{12} - B_{22}) \\
M_4 &= A_{22}(B_{21} - B_{11}) \\
M_5 &= (A_{11} + A_{12})B_{22} \\
M_6 &= (A_{21} - A_{11})(B_{11} + B_{12}) \\
M_7 &= (A_{12} - A_{22})(B_{21} + B_{22})
\end{aligned}$$

结果子矩阵：
$$\begin{aligned}
C_{11} &= M_1 + M_4 - M_5 + M_7 \\
C_{12} &= M_3 + M_5 \\
C_{21} &= M_2 + M_4 \\
C_{22} &= M_1 - M_2 + M_3 + M_6
\end{aligned}$$

验证 $C_{ij}$ 等于定义乘积和。

∎

**定理 2 (LU分解存在性)**: 若矩阵 $A$ 的所有主子式非零，则存在唯一的LU分解。

*证明*: 高斯消元过程构造性地证明存在性，唯一性由对角线元素归一化保证。∎

## 四、算法/方法详解

### 4.1 Strassen算法

```
输入: n×n 矩阵 A, B，n为2的幂
输出: C = A × B

function strassen(A, B):
    if n == 1:
        return A * B
    
    // 分块
    将 A, B, C 分为 n/2 × n/2 子矩阵
    
    // 7次递归乘法
    M1 = strassen(A11 + A22, B11 + B22)
    M2 = strassen(A21 + A22, B11)
    M3 = strassen(A11, B12 - B22)
    M4 = strassen(A22, B21 - B11)
    M5 = strassen(A11 + A12, B22)
    M6 = strassen(A21 - A11, B11 + B12)
    M7 = strassen(A12 - A22, B21 + B22)
    
    // 组合结果
    C11 = M1 + M4 - M5 + M7
    C12 = M3 + M5
    C21 = M2 + M4
    C22 = M1 - M2 + M3 + M6
    
    return C
```

### 4.2 LU分解（Doolittle）

```
输入: n×n 矩阵 A
输出: L, U 使得 A = LU

for k = 0 to n-1:
    // U的第k行
    for j = k to n-1:
        U[k][j] = A[k][j] - Σ_{m=0}^{k-1} L[k][m] * U[m][j]
    
    // L的第k列
    for i = k+1 to n-1:
        L[i][k] = (A[i][k] - Σ_{m=0}^{k-1} L[i][m] * U[m][k]) / U[k][k]

L的对角线为1
```

## 五、示例与实例

### 5.1 Strassen示例

**矩阵**: $A = \begin{pmatrix} 1 & 2 \\ 3 & 4 \end{pmatrix}$, $B = \begin{pmatrix} 5 & 6 \\ 7 & 8 \end{pmatrix}$

**计算**:
- $M_1 = (1+4)(5+8) = 65$
- $M_2 = (3+4) \cdot 5 = 35$
- $M_3 = 1 \cdot (6-8) = -2$
- $M_4 = 4 \cdot (7-5) = 8$
- $M_5 = (1+2) \cdot 8 = 24$
- $M_6 = (3-1)(5+6) = 22$
- $M_7 = (2-4)(7+8) = -30$

**结果**:
- $C_{11} = 65 + 8 - 24 - 30 = 19$
- $C_{12} = -2 + 24 = 22$
- $C_{21} = 35 + 8 = 43$
- $C_{22} = 65 - 35 - 2 + 22 = 50$

验证: $\begin{pmatrix} 19 & 22 \\ 43 & 50 \end{pmatrix}$

### 5.2 代码实现

```python
import numpy as np

def strassen(A, B):
    """Strassen矩阵乘法（简化版，假设n为2的幂）"""
    n = len(A)
    if n == 1:
        return A * B
    
    # 分块
    mid = n // 2
    A11, A12 = A[:mid, :mid], A[:mid, mid:]
    A21, A22 = A[mid:, :mid], A[mid:, mid:]
    B11, B12 = B[:mid, :mid], B[:mid, mid:]
    B21, B22 = B[mid:, :mid], B[mid:, mid:]
    
    # 7次乘法
    M1 = strassen(A11 + A22, B11 + B22)
    M2 = strassen(A21 + A22, B11)
    M3 = strassen(A11, B12 - B22)
    M4 = strassen(A22, B21 - B11)
    M5 = strassen(A11 + A12, B22)
    M6 = strassen(A21 - A11, B11 + B12)
    M7 = strassen(A12 - A22, B21 + B22)
    
    # 组合
    C11 = M1 + M4 - M5 + M7
    C12 = M3 + M5
    C21 = M2 + M4
    C22 = M1 - M2 + M3 + M6
    
    # 合并
    C = np.vstack((np.hstack((C11, C12)), np.hstack((C21, C22))))
    return C

def lu_decomposition(A):
    """LU分解（Doolittle）"""
    n = len(A)
    L = np.eye(n)
    U = np.zeros((n, n))
    
    for k in range(n):
        for j in range(k, n):
            U[k][j] = A[k][j] - sum(L[k][m] * U[m][j] for m in range(k))
        
        for i in range(k+1, n):
            L[i][k] = (A[i][k] - sum(L[i][m] * U[m][k] for m in range(k))) / U[k][k]
    
    return L, U

# 使用示例
A = np.array([[1, 2], [3, 4]], dtype=float)
B = np.array([[5, 6], [7, 8]], dtype=float)

C_strassen = strassen(A, B)
C_numpy = A @ B
print(f"Strassen结果:\n{C_strassen}")
print(f"NumPy验证:\n{C_numpy}")

L, U = lu_decomposition(np.array([[4, 3], [6, 3]], dtype=float))
print(f"\nLU分解:\nL={L}\nU={U}")
```

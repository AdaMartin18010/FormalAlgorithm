## 一、核心概念

### 1.1 定义

**归纳类型**是通过**构造子**（constructors）归纳定义的数据类型。每个元素要么是一个基本构造，要么是通过构造子从已有元素构造而来。

**核心要素**:

1. **类型名称**: 如nat, list, tree
2. **构造子**: 生成类型元素的方式
3. **归纳原理**: 证明关于该类型的所有性质

### 1.2 常见归纳类型

| 类型 | 构造子 | 说明 |
|------|--------|------|
| 自然数 | O, S(n) | 零和后继 |
| 列表 | Nil, Cons(x, xs) | 空列表和构造 |
| 二叉树 | Leaf, Node(l, v, r) | 叶节点和内部节点 |
| 可选类型 | None, Some(x) | 无值或有值 |

## 二、归纳原理

### 2.1 数学归纳法

**自然数归纳原理**:

```
要证明 ∀n:nat, P(n)，只需证明：
1. P(O) 成立（基础）
2. ∀n:nat, P(n) → P(S(n))（归纳步骤）
```

### 2.2 结构归纳法

**列表归纳原理**:

```
要证明 ∀l:list A, P(l)，只需证明：
1. P(Nil) 成立
2. ∀x:A, ∀xs:list A, P(xs) → P(Cons(x, xs))
```

## 三、Coq实现示例

### 3.1 基本归纳类型

```coq
(* 自然数 *)
Inductive nat : Type :=
  | O : nat
  | S : nat -> nat.

(* 列表 *)
Inductive list (A : Type) : Type :=
  | nil : list A
  | cons : A -> list A -> list A.

(* 二叉树 *)
Inductive tree (A : Type) : Type :=
  | leaf : tree A
  | node : tree A -> A -> tree A -> tree A.
```

### 3.2 递归函数

```coq
(* 加法 *)
Fixpoint plus (n m : nat) : nat :=
  match n with
  | O => m
  | S n' => S (plus n' m)
  end.

(* 列表长度 *)
Fixpoint length {A} (l : list A) : nat :=
  match l with
  | nil => O
  | cons _ t => S (length t)
  end.

(* 列表反转 *)
Fixpoint rev {A} (l : list A) : list A :=
  match l with
  | nil => nil
  | cons h t => app (rev t) (cons h nil)
  end.
```

### 3.3 证明示例

```coq
(* 证明加法结合律 *)
Theorem plus_assoc : forall n m p : nat,
  plus n (plus m p) = plus (plus n m) p.
Proof.
  intros n m p.
  induction n as [| n' IHn'].
  - (* n = O *)
    reflexivity.
  - (* n = S n' *)
    simpl.
    rewrite IHn'.
    reflexivity.
Qed.

(* 证明列表长度性质 *)
Theorem length_app : forall A (l1 l2 : list A),
  length (app l1 l2) = plus (length l1) (length l2).
Proof.
  intros A l1 l2.
  induction l1 as [| h t IH].
  - reflexivity.
  - simpl. rewrite IH. reflexivity.
Qed.
```

## 四、依赖归纳类型

### 4.1 向量（定长列表）

```coq
Inductive vec (A : Type) : nat -> Type :=
  | vnil : vec A 0
  | vcons : forall n, A -> vec A n -> vec A (S n).

(* 安全取元素 *)
Definition head {A n} (v : vec A (S n)) : A :=
  match v with
  | vcons _ h _ => h
  end.
(* 无越界可能！ *)
```

### 4.2 类型安全保证

依赖类型可在编译期捕获更多错误：

- 向量索引自动保证不越界
- 状态机转换自动保证合法
- 资源使用自动保证正确

## 一、核心概念

### 1.1 定义

**正式定义**:
同伦类型论（Homotopy Type Theory, HoTT）是依赖类型论的一种新解释，将类型视为空间（或高维群胚），将相等性证明视为路径。

**核心对应**:

| 类型论 | 拓扑解释 |
|--------|----------|
| 类型 A | 空间 |
| 项 a : A | 点 |
| 恒等类型 a =_A b | 路径空间 |
| 路径 p : a = b | 从a到b的路径 |
| 路径之间的路径 | 同伦 |
| 高阶路径 | 高阶同伦群 |

**单值公理 (Univalence Axiom)**:

$$\text{ua} : (A =_\mathcal{U} B) \simeq (A \simeq B)$$

即：类型的相等等价于等价（同伦等价）。

**直观解释**:
在HoTT中，相等不是简单的"是同一个"，而是"存在连接的路径"。两个事物相等的方式可能不同（多条路径），这些路径本身也可以相等（高阶路径）。

**关键要点**:
- 相等性是有结构的（不只是是/否）
- 同伦不变性：等价类型不可区分
- 计算解释：高维归纳类型(HITs)

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 路径归纳 | 基于路径的归纳原理 | Martin-Löf |
| 单值性 | 等价即相等 | Voevodsky |
| 高阶结构 | 类型可能有非平凡路径/同伦 | 同伦层 |
| 可计算性 | 开放问题（立方类型论进展） | 部分解决 |
| 数学基础 | 可作为数学基础替代集合论 | 新基础 |

**性质总结**:
1. **结构相等**: 相等证明携带信息
2. **同伦不变性**: 同伦等价的类型行为相同
3. **高维归纳**: 可定义商类型、截断等

### 1.3 变体

**Book HoTT**:
- 基于公理单值性
- 经典数学基础

**立方类型论 (Cubical Type Theory)**:
- 计算解释单值性
- 有算法实现

**模态HoTT**:
- 添加模态算子
- 凝聚拓扑

## 三、形式化证明

### 3.1 核心定理

**定理 1** (函数外延性可证): 
在HoTT中，函数外延性可从单值公理推出：
$$(\forall x, f(x) = g(x)) \to f = g$$

**证明概要**:
1. 对每个x，有路径 $p_x : f(x) = g(x)$
2. 这构成同伦 $H : f \sim g$
3. 同伦等价诱导函数相等
4. 单值公理将等价转为相等

**定理 2** (单值性蕴含结构等价原理):
等价的结构不可区分。

### 3.2 辅助引理

**路径归纳**:
要证明$\forall (x,y:A).\forall (p:x=y). P(x,y,p)$，只需证明
$\forall x:A. P(x,x,\text{refl}_x)$。

## 四、算法/方法详解

### 4.1 路径操作

```agda
-- 路径的基本操作
_∙_ : ∀ {A} {x y z : A} → (x ≡ y) → (y ≡ z) → (x ≡ z)  -- 连接
_⁻¹ : ∀ {A} {x y : A} → (x ≡ y) → (y ≡ x)               -- 逆
ap : ∀ {A B} (f : A → B) {x y} → x ≡ y → f x ≡ f y      -- 映射
```

### 4.2 截断层级

```agda
-- 命题截断 (mere proposition)
isProp : Type → Type
isProp A = (x y : A) → x ≡ y

-- 集合 (0-截断)
isSet : Type → Type  
isSet A = (x y : A) → isProp (x ≡ y)

-- n-截断递归定义
isTrunc : ℕ → Type → Type
isTrunc 0 A = isContr A      -- 可缩
isTrunc (suc n) A = (x y : A) → isTrunc n (x ≡ y)
```

## 五、示例与实例

### 5.1 标准示例

**示例 1**: 圆的类型定义（HIT）

```agda
data S¹ : Type where
  base : S¹
  loop : base ≡ base
```

**拓扑解释**: 
- base: 圆上的一个点
- loop: 从base到base的非平凡路径（绕圆一周）

**基本群**: $π_1(S¹) = ℤ$

---

**示例 2**: 命题截断

```agda
data ∥_∥ (A : Type) : Type where
  ∣_∣ : A → ∥ A ∥
  squash : (x y : ∥ A ∥) → x ≡ y  -- 使∥A∥成为命题
```

**用途**: 从构造性存在转为经典存在。

### 5.2 代码实现

**Agda (HoTT-Agda库)**:
```agda
{-# OPTIONS --without-K --rewriting #-}

open import HoTT

-- 路径定义
module _ {A : Type} where
  -- 自反性
  idp : {x : A} → x == x
  idp = idp

  -- 路径连接
  _∙_ : {x y z : A} → x == y → y == z → x == z
  idp ∙ q = q
  (p ∙ idp) = p
  
  -- 路径逆
  ! : {x y : A} → x == y → y == x
  ! idp = idp

-- 圆
module _ where
  postulate
    S¹ : Type
    base : S¹
    loop : base == base

  -- 圆的归纳原理
  S¹-rec : ∀ {i} {P : Type i} (b : P) (l : b == b) → S¹ → P
  S¹-rec b _ base = b
  ap (S¹-rec b l) loop = l

-- 单值公理
module _ where
  postulate
    ua : ∀ {i} {A B : Type i} → (A ≃ B) → (A == B)
    
  -- 等价即相等
  equiv-to-path : ∀ {A B} → (A ≃ B) → A == B
  equiv-to-path = ua
```

**Cubical Agda**:
```agda
{-# OPTIONS --cubical #-}

open import Cubical.Core.Everything
open import Cubical.Foundations.Everything

-- 在立方类型论中，单值性可计算

-- 圆的立方定义
data S¹ : Type where
  base : S¹
  loop : Path S¹ base base

-- 路径操作可计算
loop∙loop : Path S¹ base base
loop∙loop = loop ∙ loop

-- 基本群计算
π₁S¹ : Type
π₁S¹ = base ≡ base

-- π₁S¹ ≃ ℤ (可通过编码-解码证明)
```

## 六、习题

### 6.1 理解题 (L1-L2)

1. **路径操作** [难度⭐]
   
   证明：对任意路径 $p : x = y$，有 $p \cdot p^{-1} = \text{refl}_x$。
   
   <details>
   <summary>解答</summary>
   
   ```agda
   inv-r : ∀ {A} {x y : A} (p : x ≡ y) → p ∙ p ⁻¹ ≡ refl
   inv-r refl = refl
   ```
   
   由路径归纳，只需证明 refl ∙ refl ⁻¹ = refl，即 refl = refl。
   
   </details>

2. **圆的映射** [难度⭐⭐]
   
   定义从圆到自身的2倍覆盖映射。
   
   <details>
   <summary>解答</summary>
   
   ```agda
   double : S¹ → S¹
   double base = base
   ap double loop = loop ∙ loop
   ```
   
   这将loop映射为绕圆两周的路径。
   
   </details>

### 6.2 应用题 (L3)

3. **命题截断性质** [难度⭐⭐]
   
   证明 ∥A∥ 是命题（任意两点相等）。
   
   <details>
   <summary>解答</summary>
   
   ```agda
   isProp-∥∥ : ∀ {A} → isProp ∥ A ∥
   isProp-∥∥ = squash
   ```
   
   由HIT定义，squash构造函数直接给出证明。
   
   </details>

### 6.3 证明题 (L4-L5)

4. **单值性应用** [难度⭐⭐⭐]
   
   使用单值公理证明：Bool = Bool 有两条不同的路径（id和not）。
   
   <details>
   <summary>解答</summary>
   
   ```agda
   not-equiv : Bool ≃ Bool
   not-equiv = isoToEquiv (iso not not (λ {true → refl; false → refl}) 
                                     (λ {true → refl; false → refl}))

   path1 : Bool ≡ Bool
   path1 = ua idEquiv

   path2 : Bool ≡ Bool  
   path2 = ua not-equiv

   -- path1 ≠ path2 (在非集合模型中)
   ```
   
   在HoTT中，这两条路径不同，对应于Bool的不同自等价。
   
   </details>

5. **三维球面** [难度⭐⭐⭐⭐]
   
   定义三维球面 S² 作为高阶归纳类型。
   
   <details>
   <summary>解答</summary>
   
   ```agda
   data S² : Type where
     base₂ : S²
     surf : Path (Path S² base₂ base₂) refl refl
     -- surf是从refl到refl的2维路径
     -- 即填充由refl围成的"洞"的2维面
   ```
   
   S²有一个点base₂和一个2维面surf。
   
   更高维球面类似定义：
   - Sⁿ有n维面
   - πₙ(Sⁿ) = ℤ
   
   </details>

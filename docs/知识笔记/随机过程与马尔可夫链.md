## 一、核心概念

### 1.1 定义

**正式定义**:
**随机过程**: 一族随机变量 {X_t : t in T}，描述随时间演变的随机现象。

**马尔可夫链**: 状态空间 S 上的随机过程，满足马尔可夫性：
P(X_{n+1} = j | X_n = i, X_{n-1}, ..., X_0) = P(X_{n+1} = j | X_n = i) = p_{ij}

**转移矩阵**: P = (p_{ij})，其中 sum_j p_{ij} = 1。

**平稳分布**: 满足 pi = pi P 的概率分布。

**PageRank**: 将网页链接结构建模为马尔可夫链，平稳分布给出网页重要性排序。

**直观解释**:
随机过程就像"随时间展开的随机事件"，马尔可夫链则是"健忘的随机漫步者"——他只记得现在在哪里，不关心怎么来的。PageRank把这个想法用在网页上：一个随机冲浪者跟着链接跳转，访问越频繁的网页越重要。

**关键要点**:

- **马尔可夫性**: 未来只依赖现在
- **遍历性**: 长期行为与初始状态无关
- **平稳分布**: 系统的长期稳定状态
- **收敛速度**: 混合时间分析

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 不可约性 | 任意两状态可达 | 单一通信类 |
| 非周期性 | 返回时间的GCD为1 | 保证收敛 |
| 遍历性 | 不可约+非周期 | 唯一平稳分布 |
| 可逆性 | 细致平衡 | pi_i p_{ij} = pi_j p_{ji} |

### 1.3 变体

**离散时间马尔可夫链**: 状态在整数时刻变化
**连续时间马尔可夫链**: 指数分布等待时间
**隐马尔可夫模型**: 状态不可直接观测
**马尔可夫决策过程**: 加入动作和奖励

## 三、形式化证明

### 3.1 核心定理

**定理 1 (遍历定理)**: 有限状态不可约非周期马尔可夫链存在唯一平稳分布 pi，且对任意初始分布：
lim_{n to infty} P^n(i, j) = pi_j

**证明**:
由Perron-Frobenius定理，转移矩阵 P 有唯一特征值1，对应左特征向量 pi。

其他特征值 |lambda| < 1，故 P^n to 1 pi。

∎

**定理 2 (细致平衡与可逆性)**: 若分布 pi 满足细致平衡条件 pi_i p_{ij} = pi_j p_{ji}，则 pi 是平稳分布。

*证明*: 对 j 求和得 sum_i pi_i p_{ij} = pi_j sum_i p_{ji} = pi_j。∎

## 四、算法/方法详解

### 4.1 PageRank计算

```
输入: 网页链接图，阻尼因子 d (通常0.85)
输出: PageRank向量 r

// 构建转移矩阵（处理悬挂节点）
for each page i:
    if out_degree(i) == 0:
        // 悬挂节点：均匀跳转
        P[i][j] = 1/n for all j
    else:
        for each link i->j:
            P[i][j] = d / out_degree(i) + (1-d) / n

// 幂迭代求解
r = [1/n, ..., 1/n]
repeat:
    r_new = r * P
    if ||r_new - r|| < epsilon:
        break
    r = r_new

return r
```

### 4.2 MCMC采样

```
Metropolis-Hastings算法:
输入: 目标分布 pi(x)（已知到常数因子）

初始化 x_0
for t = 0 to T-1:
    从提议分布 q(x'|x_t) 采样 x'
    计算接受率 alpha = min(1, pi(x')q(x_t|x') / (pi(x_t)q(x'|x_t)))
    以概率 alpha 接受 x'，否则保持 x_t
    x_{t+1} = x' (若接受) 或 x_t (若拒绝)
return {x_t} 作为来自 pi 的样本
```

## 五、示例与实例

### 5.1 简单马尔可夫链

**状态**: {晴天, 雨天}

**转移矩阵**:

```
        晴天  雨天
晴天   0.9   0.1
雨天   0.5   0.5
```

**平稳分布**: 解 pi P = pi

- pi_晴 = 0.9 pi_晴 + 0.5 pi_雨
- pi_晴 + pi_雨 = 1
- 解得: pi_晴 = 5/6, pi_雨 = 1/6

### 5.2 代码实现

```python
import numpy as np

def pagerank(adj_matrix, d=0.85, eps=1e-8, max_iter=1000):
    """PageRank算法"""
    n = len(adj_matrix)

    # 出度
    out_degree = np.sum(adj_matrix, axis=1)

    # 构建转移矩阵
    P = np.zeros((n, n))
    for i in range(n):
        if out_degree[i] == 0:
            # 悬挂节点
            P[i] = 1.0 / n
        else:
            for j in range(n):
                if adj_matrix[i][j]:
                    P[i][j] = d / out_degree[i] + (1-d) / n
                else:
                    P[i][j] = (1-d) / n

    # 幂迭代
    r = np.ones(n) / n
    for _ in range(max_iter):
        r_new = r @ P
        if np.linalg.norm(r_new - r) < eps:
            break
        r = r_new

    return r

def markov_stationary(P):
    """计算平稳分布"""
    n = len(P)
    # 解 pi P = pi, sum pi = 1
    A = np.vstack([P.T - np.eye(n), np.ones(n)])
    b = np.zeros(n + 1)
    b[-1] = 1
    pi = np.linalg.lstsq(A, b, rcond=None)[0]
    return pi

# 使用示例
# 简单链接图: 0->1, 0->2, 1->2, 2->0
adj = np.array([
    [0, 1, 1],
    [0, 0, 1],
    [1, 0, 0]
])

pr = pagerank(adj)
print(f"PageRank: {pr}")

# 马尔可夫链平稳分布
P = np.array([[0.9, 0.1], [0.5, 0.5]])
pi = markov_stationary(P)
print(f"平稳分布: {pi}")
```

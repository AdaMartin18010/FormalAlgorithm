## 一、核心概念

### 1.1 定义

**正式定义**:
动态规划是一种通过**分解子问题**并**存储子问题解**来避免重复计算的方法。适用于具有**最优子结构**和**重叠子问题**特性的问题。

**核心要素**:

1. **状态定义**: dp[i]或dp[i][j]表示的子问题解
2. **状态转移方程**: dp[i]与dp[i-1]等的关系
3. **边界条件**: 初始状态的值
4. **计算顺序**: 自顶向下（记忆化）或自底向上（迭代）

**设计步骤**:

```
1. 刻画最优解的结构特征（最优子结构）
2. 递归定义最优解的值
3. 计算最优解的值（通常自底向上）
4. 根据计算信息构造最优解（可选）
```

### 1.2 高级技巧

| 技巧 | 描述 | 应用 |
|------|------|------|
| 状态压缩 | 用位运算表示状态 | 子集DP、旅行商问题 |
| 单调队列优化 | 滑动窗口最值 | 多重背包、区间DP |
| 斜率优化 | 凸包优化转移 | 动态规划优化 |
| 四边形不等式 | 决策单调性 | 区间DP优化 |
| 矩阵快速幂 | 加速线性递推 | 斐波那契、图传递闭包 |

### 1.3 经典问题分类

**线性DP**: 最长递增子序列(LIS)、编辑距离、背包问题
**区间DP**: 矩阵链乘、石子合并、最优二叉搜索树
**树形DP**: 树的最大独立集、树的重心
**状态压缩DP**: TSP、集合覆盖、铺砖问题
**数位DP**: 计数问题、数字特征统计
**概率/期望DP**: 随机游走、博弈问题

## 三、形式化证明

### 3.1 最优子结构证明模板

**定理**: 问题P具有最优子结构，即最优解包含子问题的最优解。

**证明模板**:

```
设S是问题P的最优解。

反证法：假设S包含子问题Q的非最优解S'。
设S''是Q的最优解，则S''优于S'。

用S''替换S中的S'，得到新解S*。
由于S''优于S'，S*优于S。
这与S是最优解矛盾！

因此最优解必须包含子问题的最优解。∎
```

### 3.2 背包问题DP正确性

**0/1背包状态转移**:

```
dp[i][w] = max(dp[i-1][w], dp[i-1][w-weight[i]] + value[i])
```

**证明**:
对于第i个物品，有两种选择：

1. 不选：最大价值为dp[i-1][w]
2. 选：最大价值为dp[i-1][w-weight[i]] + value[i]

取两者最大值即得最优解。∎

## 四、算法详解

### 4.1 状态压缩DP

**旅行商问题(TSP)**:

```
状态: dp[mask][i] = 已访问mask表示的点集，当前在i的最短距离
转移: dp[mask][i] = min(dp[mask^(1<<i)][j] + dist[j][i])
复杂度: O(2^n · n²)
```

### 4.2 单调队列优化

**多重背包优化**:

```python
def optimized_knapsack(weights, values, counts, W):
    dp = [0] * (W + 1)

    for i in range(len(weights)):
        w, v, c = weights[i], values[i], counts[i]
        # 按模w分类
        for r in range(w):
            # 单调队列
            queue = deque()
            for k in range(r, W + 1, w):
                val = dp[k] - k // w * v
                while queue and queue[-1][1] <= val:
                    queue.pop()
                queue.append((k, val))
                # 移除超出范围的
                while queue and queue[0][0] < k - c * w:
                    queue.popleft()
                dp[k] = queue[0][1] + k // w * v

    return dp[W]
```

## 五、示例与代码

### 5.1 最长公共子序列(LCS)

```python
def lcs(s1: str, s2: str) -> int:
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    return dp[m][n]
```

### 5.2 编辑距离

```python
def edit_distance(s1: str, s2: str) -> int:
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = 1 + min(
                    dp[i-1][j],      # 删除
                    dp[i][j-1],      # 插入
                    dp[i-1][j-1]     # 替换
                )

    return dp[m][n]
```

### 5.3 状态压缩TSP

```python
def tsp(dist):
    n = len(dist)
    INF = float('inf')
    dp = [[INF] * n for _ in range(1 << n)]
    dp[1][0] = 0  # 从0出发，已访问{0}

    for mask in range(1 << n):
        for i in range(n):
            if not (mask & (1 << i)):
                continue
            if dp[mask][i] == INF:
                continue
            for j in range(n):
                if mask & (1 << j):
                    continue
                new_mask = mask | (1 << j)
                dp[new_mask][j] = min(dp[new_mask][j], dp[mask][i] + dist[i][j])

    # 返回起点
    ans = INF
    for i in range(1, n):
        ans = min(ans, dp[(1 << n) - 1][i] + dist[i][0])
    return ans
```

## 六、习题

### 6.1 理解题

1. **最优子结构 vs 贪心选择** [难度⭐]

   解释DP和贪心算法在问题结构上的区别。

   <details>
   <summary>解答</summary>
   两者都要求最优子结构，但贪心还要求贪心选择性质（局部最优导致全局最优）。DP考虑所有子问题组合，贪心只做当前最优选择。
   </details>

### 6.2 应用题

1. **矩阵链乘** [难度⭐⭐]

   给定矩阵维度数组p，计算最优乘法顺序。

   <details>
   <summary>解答</summary>

   ```python

def matrix_chain(p):
    n = len(p) - 1
    dp = [[0] * n for_ in range(n)]

    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            dp[i][j] = float('inf')
            for k in range(i, j):
                cost = dp[i][k] + dp[k+1][j] + p[i]*p[k+1]*p[j+1]
                dp[i][j] = min(dp[i][j], cost)

    return dp[0][n-1]

   ```

   </details>

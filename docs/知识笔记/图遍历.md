## 一、核心概念

### 1.1 定义

**图遍历**: 系统地访问图中所有顶点的过程。

**两种基本方法**:

1. **广度优先搜索(BFS)**: 按层遍历，先访问邻居
2. **深度优先搜索(DFS)**: 深入遍历，先访问子节点

**基本概念**:

- **发现时间**: 顶点首次被访问的时间
- **完成时间**: 顶点所有邻居都被访问后的时间（DFS）
- **前驱顶点**: 遍历时到达该顶点的来源顶点
- **树边/后向边/前向边/横叉边**: DFS边的分类

### 1.2 属性

| 属性 | BFS | DFS |
|------|-----|-----|
| 数据结构 | 队列 | 栈（递归） |
| 时间复杂度 | O(V+E) | O(V+E) |
| 空间复杂度 | O(V) | O(V) |
| 最短路径 | 无权图最短 | 不保证 |
| 应用 | 层次遍历、连通性 | 拓扑排序、强连通分量 |

## 三、算法详解

### 3.1 BFS

```python
from collections import deque

def bfs(graph, start):
    """
    graph: {vertex: [neighbors]}
    返回距离和前驱字典
    """
    visited = {start}
    queue = deque([start])
    distance = {start: 0}
    predecessor = {start: None}

    while queue:
        vertex = queue.popleft()
        for neighbor in graph.get(vertex, []):
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
                distance[neighbor] = distance[vertex] + 1
                predecessor[neighbor] = vertex

    return distance, predecessor
```

### 3.2 DFS

```python
def dfs(graph):
    """DFS遍历，返回发现时间和完成时间"""
    visited = set()
    discovery = {}
    finish = {}
    time = [0]

    def dfs_visit(vertex):
        visited.add(vertex)
        time[0] += 1
        discovery[vertex] = time[0]

        for neighbor in graph.get(vertex, []):
            if neighbor not in visited:
                dfs_visit(neighbor)

        time[0] += 1
        finish[vertex] = time[0]

    for vertex in graph:
        if vertex not in visited:
            dfs_visit(vertex)

    return discovery, finish
```

### 3.3 拓扑排序

```python
def topological_sort_dfs(graph):
    """基于DFS的拓扑排序"""
    visited = set()
    stack = []

    def dfs(vertex):
        visited.add(vertex)
        for neighbor in graph.get(vertex, []):
            if neighbor not in visited:
                dfs(neighbor)
        stack.append(vertex)

    for vertex in graph:
        if vertex not in visited:
            dfs(vertex)

    return stack[::-1]  # 逆序即为拓扑序
```

## 四、习题

### 6.1 理解题

1. **BFS vs DFS** [难度⭐]

   何时用BFS？何时用DFS？

   <details>
   <summary>解答</summary>
   BFS：求最短路径、层次遍历、连通性检查
   DFS：拓扑排序、检测环、强连通分量、回溯问题
   </details>

### 6.2 应用题

1. **检测环** [难度⭐⭐]

   用DFS检测有向图中的环。

   <details>
   <summary>解答</summary>

   ```python

def has_cycle(graph):
    WHITE, GRAY, BLACK = 0, 1, 2
    color = {v: WHITE for v in graph}

    def dfs(vertex):
        color[vertex] = GRAY
        for neighbor in graph.get(vertex, []):
            if color[neighbor] == GRAY:  # 回溯边
                return True
            if color[neighbor] == WHITE and dfs(neighbor):
                return True
        color[vertex] = BLACK
        return False

    for vertex in graph:
        if color[vertex] == WHITE:
            if dfs(vertex):
                return True
    return False

   ```

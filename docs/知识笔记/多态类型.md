## 一、核心概念

### 1.1 定义

**正式定义**:
多态类型系统（System F，又称Girard-Reynolds多态λ演算）扩展了简单类型系统，引入类型抽象和类型应用。

**语法**:
```
类型 τ ::= α | τ→τ | ∀α.τ
项   t ::= x | λx:τ.t | t t | Λα.t | t[τ]
```

其中：
- `∀α.τ`：全称类型（Universal type）
- `Λα.t`：类型抽象（Type abstraction）
- `t[τ]`：类型应用（Type application）

**类型规则**:

$$\frac{\Gamma, \alpha \text{ type} \vdash t : \tau}{\Gamma \vdash \Lambda\alpha.t : \forall\alpha.\tau} \text{ (T-TAbs)}$$

$$\frac{\Gamma \vdash t : \forall\alpha.\tau}{\Gamma \vdash t[\sigma] : \tau[\sigma/\alpha]} \text{ (T-TApp)}$$

**直观解释**:
多态允许函数"泛化"地工作在多种类型上。`Λα.λx:α.x` 表示"对于所有类型α，返回α上的恒等函数"。

**关键要点**:
- 全称量词绑定类型变量
- 类型擦除后行为等价于单态程序
- 表达能力严格强于简单类型

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 全称多态 | ∀α.τ 表示所有类型τ的实例 | 参数多态 |
| 强规范化 | 保持强规范化 | Girard证明 |
| 类型擦除 | 运行时无需类型信息 | 零开销抽象 |
| 类型推断 | 一般情况不可判定 | 部分限制版本可判定 |
| 表达能力 | 可编码递归类型 | 无不动点算子也可编码递归 |

**性质总结**:
1. **进步性+保持性**: 保持类型安全
2. **强规范化**: 无类型递归时保证停机
3. **语义一致性**: Reynolds抽象定理

### 1.3 变体

**系统F**:
- 全称多态，类型抽象/应用
- 基础：Girard(1972), Reynolds(1974)

**系统Fω**:
- 扩展至类型构造子层面多态
- 高阶类型抽象

**ML多态 (let多态)**:
- 限制：只允许let绑定的多态
- 类型推断可判定
- 实际编程语言使用

## 三、形式化证明

### 3.1 核心定理

**定理 1** (抽象定理 / Reynolds, 1983): 
若 $t : \forall\alpha.\tau$，则对任意关系 $R : A \leftrightarrow B$，
$(t[A], t[B]) \in \tau[R/\alpha]$。

**推论**: 无自由类型变量的多态函数行为受限（关系参数性）。

**定理 2** (强规范化):
所有System F良类型项都强规范化。

**证明要点**:
- Girard可约性候选法
- 对每个类型构造候选集
- 证明所有良类型项属于对应候选集

### 3.2 辅助引理

**类型替换引理**:
若 $\Gamma, \alpha \text{ type} \vdash t : \tau$ 且 $\vdash \sigma \text{ type}$，
则 $\Gamma[\sigma/\alpha] \vdash t : \tau[\sigma/\alpha]$。

## 四、算法/方法详解

### 4.1 类型检查算法

```
算法: System F类型检查
输入: 上下文Γ，项t
输出: 类型τ或失败

check(Γ, Λα.t) = 
  τ ← check(Γ,α:type, t)
  return ∀α.τ

check(Γ, t[σ]) =
  τ ← check(Γ, t)
  若 τ = ∀α.τ'
  则 return τ'[σ/α]
  否则失败
```

## 五、示例与实例

### 5.1 标准示例

**示例 1**: Church数（多态版本）

```
CN = ∀α.(α→α)→α→α

0 = Λα.λs:α→α.λz:α.z
1 = Λα.λs:α→α.λz:α.s z
2 = Λα.λs:α→α.λz:α.s(s z)

SUCC = λn:CN.Λα.λs:α→α.λz:α.s (n[α] s z) : CN→CN
```

**示例 2**: 多态列表

```
List = λα.∀β.β→(α→β→β)→β

nil = Λα.Λβ.λn:β.λc:α→β→β.n
cons = Λα.λx:α.λxs:List α.Λβ.λn:β.λc:α→β→β.c x (xs[β] n c)
```

### 5.2 代码实现

**Coq实现**:
```coq
Inductive ty : Type :=
  | TyVar : nat -> ty
  | Arrow : ty -> ty -> ty
  | Forall : nat -> ty -> ty.  (* ∀α.τ *)

Inductive tm : Type :=
  | TmVar : nat -> tm
  | Abs : nat -> ty -> tm -> tm
  | App : tm -> tm -> tm
  | TAbs : nat -> tm -> tm      (* Λα.t *)
  | TApp : tm -> ty -> tm.      (* t[τ] *)

Inductive has_type : context -> tm -> ty -> Prop :=
  | T_TAbs : forall Γ X t T,
      has_type (add_tyvar X Γ) t T ->
      has_type Γ (TAbs X t) (Forall X T)
  | T_TApp : forall Γ t X T1 T2,
      has_type Γ t (Forall X T1) ->
      has_type Γ (TApp t T2) (subst_ty X T2 T1).
```

**Agda实现**:
```agda
data Type (n : ℕ) : Set where
  tyvar : Fin n → Type n
  _⇒_ : Type n → Type n → Type n
  ∀' : Type (suc n) → Type n  -- ∀α.τ

data Term (n : ℕ) (m : ℕ) : Set where  -- n类型变量, m项变量
  var : Fin m → Term n m
  ƛ_∶_⇒_ : Fin m → Type n → Term n (suc m) → Term n m
  _·_ : Term n m → Term n m → Term n m
  Λ_⇒_ : Term (suc n) m → Term n m       -- 类型抽象
  _[_] : Term n m → Type n → Term n m    -- 类型应用
```

## 六、习题

### 6.1 理解题 (L1-L2)

1. **多态类型推导** [难度⭐]
   
   给 `Λα.λx:α.x` 写出完整类型推导。
   
   <details>
   <summary>解答</summary>
   
   ```
   α:type, x:α ⊢ x : α
   -------------------- T-Abs
   α:type ⊢ λx:α.x : α→α
   ------------------------ T-TAbs
   ⊢ Λα.λx:α.x : ∀α.α→α
   ```
   
   </details>

2. **类型应用** [难度⭐⭐]
   
   计算 `(Λα.λx:α.x)[Int→Bool]` 的类型。
   
   <details>
   <summary>解答</summary>
   
   ```
   Λα.λx:α.x : ∀α.α→α
   
   (∀α.α→α)[Int→Bool] = (Int→Bool)→(Int→Bool)
   ```
   
   </details>

### 6.2 应用题 (L3)

3. **多态复合函数** [难度⭐⭐]
   
   定义多态版本compose : ∀α.∀β.∀γ.(β→γ)→(α→β)→α→γ
   
   <details>
   <summary>解答</summary>
   
   ```
   compose = Λα.Λβ.Λγ.λf:β→γ.λg:α→β.λx:α.f (g x)
   ```
   
   类型检查：
   - g x : β
   - f (g x) : γ
   - 结果类型: α→β→γ
   
   </details>

### 6.3 证明题 (L4-L5)

4. **参数性应用** [难度⭐⭐⭐]
   
   证明：任何 `t : ∀α.α→α` 必须等价于恒等函数。
   
   <details>
   <summary>解答</summary>
   
   由抽象定理，取关系R为任意A和B之间的关系。
   
   设$t[A] : A→A$和$t[B] : B→B$。
   
   对任意$(a,b) \in R$，有$(t[A]\ a, t[B]\ b) \in R$。
   
   取$R = \{(a,a)\}$（对角关系），则$t[A]\ a = a$。
   
   因此t是恒等函数。
   
   </details>

5. **类型替换保持性** [难度⭐⭐⭐]
   
   证明若 $\Gamma, \alpha \vdash t : \tau$ 则 $\Gamma[\sigma/\alpha] \vdash t : \tau[\sigma/\alpha]$。
   
   <details>
   <summary>解答</summary>
   
   对推导进行结构归纳：
   
   **变量**: 直接由上下文替换
   
   **类型抽象** `Λβ.t`:
   - 若$\beta = \alpha$，类型变量被遮蔽，无需替换
   - 否则，归纳假设应用
   
   **类型应用** `t[τ']`:
   - 归纳假设于t
   - 类型替换组合性
   
   </details>

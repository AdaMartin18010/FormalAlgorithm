## 一、核心概念

### 1.1 定义

**正式定义**:
**快速傅里叶变换（FFT）**: 计算多项式在 $n$ 次单位根处的值，复杂度 $O(n \log n)$。

**数论变换（NTT）**: 在有限域上的FFT，用于整数多项式。

**多项式乘法**: 给定 $A(x), B(x)$，计算 $C(x) = A(x) \cdot B(x)$。

**形式幂级数**: $\sum_{n=0}^\infty a_n x^n$，不关注收敛性。

**多项式求逆**: 给定 $A(x)$，求 $B(x)$ 使得 $A(x)B(x) \equiv 1 \pmod{x^n}$。

**直观解释**:
多项式算法就像"高效处理多项式的工具箱"——FFT是"神奇的计算器"，能把多项式乘法从 $O(n^2)$ 降到 $O(n \log n)$，原理是在特殊点（单位根）求值后相乘再插值；NTT则是FFT在整数世界的"同胞兄弟"，用模运算避免浮点误差；形式幂级数则是"无限长的多项式"，用于组合数学和生成函数。

**关键要点**:
- **FFT**: 分治 + 单位根性质
- **NTT**: 模素数原根替代复数单位根
- **卷积定理**: 时域卷积 = 频域乘积
- **牛顿迭代**: 多项式求逆、开方

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| FFT复杂度 | $O(n \log n)$ | $n$ 为2的幂 |
| 精度问题 | 浮点FFT有舍入误差 | NTT精确 |
| NTT限制 | 需模数支持原根 | 常用998244353 |
| 级数截断 | 形式幂级数需截断计算 | $O(n \log n)$ |

### 1.3 变体

**Cooley-Tukey**: 经典FFT算法
**Bluestein算法**: 任意长度FFT
**FFT素数**: 形如 $c \cdot 2^k + 1$ 的素数
**任意模数卷积**: Garner算法/CRT

## 三、形式化证明

### 3.1 核心定理

**定理 1 (卷积定理)**: 设 $D$ 为DFT算子，则 $D(f * g) = D(f) \cdot D(g)$。

**证明**:
$$(f * g)_n = \sum_{k=0}^{N-1} f_k g_{n-k \mod N}$$

$$D(f * g)_m = \sum_{n=0}^{N-1} (f * g)_n \omega_N^{mn} = \sum_{n,k} f_k g_{n-k} \omega_N^{mn}$$

令 $j = n - k$:
$$= \sum_k f_k \omega_N^{mk} \sum_j g_j \omega_N^{mj} = D(f)_m \cdot D(g)_m$$

∎

**定理 2 (FFT正确性)**: Cooley-Tukey算法正确计算DFT。

*证明*: 分治分解利用单位根性质，奇偶下标分离。∎

## 四、算法/方法详解

### 4.1 Cooley-Tukey FFT

```
输入: 多项式系数 a[0..n-1]，n为2的幂
输出: DFT值 y[0..n-1]

function FFT(a, ω):  // ω为n次本原单位根
    n = len(a)
    if n == 1:
        return a
    
    // 分奇偶
    a_even = [a[0], a[2], ...]
    a_odd = [a[1], a[3], ...]
    
    y_even = FFT(a_even, ω^2)
    y_odd = FFT(a_odd, ω^2)
    
    y = [0] * n
    w = 1
    for k = 0 to n/2 - 1:
        y[k] = y_even[k] + w * y_odd[k]
        y[k + n/2] = y_even[k] - w * y_odd[k]
        w = w * ω
    
    return y
```

### 4.2 NTT

```
// 与FFT结构相同，但：
// 1. 所有运算在模MOD下进行
// 2. ω = g^((MOD-1)/n)，g为原根
// 3. 逆变换需要乘以n^{-1} mod MOD
```

## 五、示例与实例

### 5.1 FFT多项式乘法

**多项式**: $A(x) = 1 + 2x$, $B(x) = 3 + 4x$

**扩展为4次**: $A = [1, 2, 0, 0]$, $B = [3, 4, 0, 0]$

**FFT点值**: 在 $\omega^0, \omega^1, \omega^2, \omega^3$ 处求值

**点值相乘后逆FFT**: 得 $[3, 10, 8, 0]$

**结果**: $3 + 10x + 8x^2$

验证: $(1+2x)(3+4x) = 3 + 4x + 6x + 8x^2 = 3 + 10x + 8x^2$ ✓

### 5.2 代码实现

```python
def fft(a, invert):
    """迭代版FFT"""
    n = len(a)
    j = 0
    for i in range(1, n):
        bit = n >> 1
        while j & bit:
            j ^= bit
            bit >>= 1
        j ^= bit
        if i < j:
            a[i], a[j] = a[j], a[i]
    
    length = 2
    while length <= n:
        ang = 2 * 3.14159265358979323846 / length * (-1 if invert else 1)
        wlen = complex(cos(ang), sin(ang))
        for i in range(0, n, length):
            w = 1
            for j in range(i, i + length // 2):
                u = a[j]
                v = a[j + length // 2] * w
                a[j] = u + v
                a[j + length // 2] = u - v
                w *= wlen
        length <<= 1
    
    if invert:
        for i in range(n):
            a[i] /= n
    
    return a

def multiply(a, b):
    """多项式乘法（FFT）"""
    n = 1
    while n < len(a) + len(b):
        n <<= 1
    
    fa = list(map(complex, a)) + [0] * (n - len(a))
    fb = list(map(complex, b)) + [0] * (n - len(b))
    
    fft(fa, False)
    fft(fb, False)
    
    for i in range(n):
        fa[i] *= fb[i]
    
    fft(fa, True)
    
    result = [int(round(fa[i].real)) for i in range(len(a) + len(b) - 1)]
    return result

# NTT实现（模998244353）
MOD = 998244353
G = 3  # 原根

def power(base, exp, mod):
    result = 1
    base %= mod
    while exp > 0:
        if exp & 1:
            result = result * base % mod
        base = base * base % mod
        exp >>= 1
    return result

def ntt(a, invert):
    n = len(a)
    j = 0
    for i in range(1, n):
        bit = n >> 1
        while j & bit:
            j ^= bit
            bit >>= 1
        j ^= bit
        if i < j:
            a[i], a[j] = a[j], a[i]
    
    length = 2
    while length <= n:
        wlen = power(G, (MOD - 1) // length, MOD)
        if invert:
            wlen = power(wlen, MOD - 2, MOD)
        for i in range(0, n, length):
            w = 1
            for j in range(i, i + length // 2):
                u = a[j]
                v = a[j + length // 2] * w % MOD
                a[j] = (u + v) % MOD
                a[j + length // 2] = (u - v + MOD) % MOD
                w = w * wlen % MOD
        length <<= 1
    
    if invert:
        n_inv = power(n, MOD - 2, MOD)
        for i in range(n):
            a[i] = a[i] * n_inv % MOD
    
    return a

# 使用示例
a = [1, 2]  # 1 + 2x
b = [3, 4]  # 3 + 4x
result = multiply(a, b)
print(f"FFT乘法结果: {result}")  # [3, 10, 8]
```

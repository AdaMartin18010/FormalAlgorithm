## 一、核心概念

### 1.1 定义

**正式定义**:
**模运算**: 整数在模 $n$ 下的算术，结果取余数。

**扩展欧几里得算法**: 求 $ax + by = \gcd(a, b)$ 的解。

**中国剩余定理（CRT）**: 求解同余方程组：
$$x \equiv a_i \pmod{n_i}, \quad \gcd(n_i, n_j) = 1$$

**素性测试**: 判断大数是否为素数。

- **费马测试**: 基于费马小定理
- **Miller-Rabin**: 强伪素数测试

**直观解释**:
数论算法就像"整数的密码学"——研究整数的基本性质和运算规则。模运算是在"时钟数学"中计算（12点之后是1点）；中国剩余定理是"韩信点兵"问题的数学解答；素性测试是判断一个数是否有因子的快速方法。

**关键要点**:

- **模逆元**: 存在当且仅当 $\gcd(a, n) = 1$
- **欧拉定理**: $a^{\phi(n)} \equiv 1 \pmod{n}$
- **快速幂**: $O(\log n)$ 计算大数幂模
- **概率素性测试**: 高效但可能有假阳性

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 模运算封闭性 | 结果仍在模范围内 | 有限域/环 |
| 逆元存在性 | 需与模数互素 | 欧拉定理保证 |
| 素数密度 | $\pi(n) \approx n/\ln n$ | 素数定理 |
| 计算复杂度 | 快速幂 $O(\log n)$ | 多项式时间 |

### 1.3 变体

**Pollard's Rho**: 整数分解算法
**Lucas-Lehmer**: 梅森素数测试
**AKS**: 确定性多项式时间素性测试
**Tonelli-Shanks**: 模平方根算法

## 三、形式化证明

### 3.1 核心定理

**定理 1 (中国剩余定理)**: 若 $n_1, \ldots, n_k$ 两两互素，则同余方程组
$$x \equiv a_i \pmod{n_i}$$
有唯一解模 $N = n_1 \cdots n_k$。

**证明**:
构造性证明：令 $N_i = N / n_i$，求 $y_i$ 使得 $N_i y_i \equiv 1 \pmod{n_i}$（由互素保证存在）。

解为 $x = \sum a_i N_i y_i \mod N$。

验证：$x \equiv a_i N_i y_i \equiv a_i \pmod{n_i}$。

∎

**定理 2 (Miller-Rabin正确性)**: 若 $n$ 是合数，Miller-Rabin测试以概率 $\geq 3/4$ 检测出。

*证明*: 基于强伪素数的性质，至少3/4的基能通过测试当且仅当 $n$ 为素数。∎

## 四、算法/方法详解

### 4.1 扩展欧几里得算法

```
输入: a, b
输出: (g, x, y) 使得 ax + by = g = gcd(a, b)

function extended_gcd(a, b):
    if b == 0:
        return (a, 1, 0)
    (g, x1, y1) = extended_gcd(b, a mod b)
    x = y1
    y = x1 - (a // b) * y1
    return (g, x, y)
```

### 4.2 Miller-Rabin素性测试

```
输入: n > 2, 迭代次数 k
输出: "素数" 或 "合数"

将 n-1 写成 2^r * d，d 为奇数

重复 k 次:
    随机选择 a ∈ [2, n-2]
    x = a^d mod n
    if x == 1 or x == n-1:
        continue (可能是素数)
    repeat r-1 次:
        x = x^2 mod n
        if x == n-1:
            break (可能是素数)
    if x != n-1:
        return "合数"

return "素数" (概率性)
```

## 五、示例与实例

### 5.1 中国剩余定理

**问题**: $x \equiv 2 \pmod{3}$, $x \equiv 3 \pmod{5}$, $x \equiv 2 \pmod{7}$

**求解**:

- $N = 3 \cdot 5 \cdot 7 = 105$
- $N_1 = 35$, $y_1 = 35^{-1} \equiv 2 \pmod{3}$
- $N_2 = 21$, $y_2 = 21^{-1} \equiv 1 \pmod{5}$
- $N_3 = 15$, $y_3 = 15^{-1} \equiv 1 \pmod{7}$
- $x = 2 \cdot 35 \cdot 2 + 3 \cdot 21 \cdot 1 + 2 \cdot 15 \cdot 1 = 233 \equiv 23 \pmod{105}$

### 5.2 代码实现

```python
def extended_gcd(a, b):
    if b == 0:
        return (a, 1, 0)
    g, x1, y1 = extended_gcd(b, a % b)
    x = y1
    y = x1 - (a // b) * y1
    return (g, x, y)

def mod_inverse(a, m):
    """模逆元"""
    g, x, _ = extended_gcd(a, m)
    if g != 1:
        return None  # 逆元不存在
    return (x % m + m) % m

def crt(a_list, n_list):
    """中国剩余定理"""
    N = 1
    for n in n_list:
        N *= n

    result = 0
    for a, n in zip(a_list, n_list):
        Ni = N // n
        yi = mod_inverse(Ni, n)
        result += a * Ni * yi

    return result % N

def fast_pow(base, exp, mod):
    """快速幂"""
    result = 1
    base %= mod
    while exp > 0:
        if exp & 1:
            result = result * base % mod
        base = base * base % mod
        exp >>= 1
    return result

def is_prime(n, k=5):
    """Miller-Rabin素性测试"""
    if n < 2:
        return False
    if n in (2, 3):
        return True
    if n % 2 == 0:
        return False

    # 写成 n-1 = 2^r * d
    r, d = 0, n - 1
    while d % 2 == 0:
        r += 1
        d //= 2

    import random
    for _ in range(k):
        a = random.randrange(2, n - 1)
        x = fast_pow(a, d, n)
        if x == 1 or x == n - 1:
            continue
        for _ in range(r - 1):
            x = (x * x) % n
            if x == n - 1:
                break
        else:
            return False
    return True

# 使用示例
print(f"17是素数: {is_prime(17)}")
print(f"CRT结果: {crt([2, 3, 2], [3, 5, 7])}")  # 23
```

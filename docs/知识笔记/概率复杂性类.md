## 一、核心概念

### 1.1 定义

**正式定义**:

**概率图灵机 (PTM)**: 有两条转移函数的图灵机，每步以1/2概率选择其一。

**BPP** (Bounded-error Probabilistic Polynomial time):
语言L ∈ BPP如果存在PTM M使得：
- 若x ∈ L，则Pr[M接受x] ≥ 2/3
- 若x ∉ L，则Pr[M接受x] ≤ 1/3

**RP** (Randomized Polynomial time):
- 若x ∈ L，则Pr[M接受x] ≥ 1/2（单侧错误）
- 若x ∉ L，则Pr[M接受x] = 0（永不错拒）

**co-RP**: RP的补类

**ZPP** (Zero-error Probabilistic Polynomial time):
ZPP = RP ∩ co-RP，期望多项式时间，永不错误

**PP** (Probabilistic Polynomial time):
- 若x ∈ L，则Pr[M接受x] > 1/2
- 若x ∉ L，则Pr[M接受x] ≤ 1/2

### 1.2 复杂性类层次

```
ZPP = RP ∩ co-RP
RP ⊆ BPP ⊆ PP ⊆ PSPACE
co-RP ⊆ BPP
P ⊆ ZPP ⊆ RP ⊆ NP
```

**关键要点**:
- 错误概率可放大：通过重复运行可将错误降至指数小
- P = BPP？ 是复杂性理论的主要猜想
- BPP ⊆ P/poly (Adleman定理)
- BPP ⊆ Σ₂ ∩ Π₂ (Sipser-Gács-Lautemann定理)

### 1.3 变体

**BPP的错误降低**:
运行k次，取多数投票，错误概率 ≤ (3/4)^(k/2)

## 三、形式化证明

### 3.1 错误放大引理

**引理**: 若L ∈ BPP，则对任意多项式p(n)，存在PTM M'使得错误概率 < 2^(-p(n))。

**证明**:

设M是错误概率≤1/3的BPP机器。构造M'：
- 运行M共k = O(p(n))次
- 接受当且仅当多数运行接受

设X_i为第i次运行的指示变量，X = ΣX_i。
E[X] ≥ 2k/3（若x ∈ L）或 ≤ k/3（若x ∉ L）。

由Chernoff界：
Pr[X ≤ k/2] ≤ Pr[|X - E[X]| ≥ k/6] ≤ 2e^(-2(k/6)²/k) = 2e^(-k/18)

取k = O(p(n))可使此 < 2^(-p(n))。

### 3.2 Adleman定理

**定理**: BPP ⊆ P/poly

**证明概要**:

P/poly是多项式规模电路可解决的问题。

对L ∈ BPP，由错误放大，存在M使得错误概率 < 2^(-n)。

固定输入长度n，考虑M的所有随机串。
对特定x，坏随机串比例 < 2^(-n)。

对所有2^n个输入，存在随机串r使得对所有|x|=n，M(x,r)正确。

将此r硬编码到电路中。

## 四、算法/方法详解

### 4.1 BPP算法的标准结构

```
算法: 概率算法模板
输入: x
输出: 接受/拒绝

1. 生成随机串 r（多项式长度）
2. 确定性计算 M(x, r)
3. return M的结果

重复k次取多数，降低错误
```

### 4.2 经典BPP算法

**多项式恒等测试**:
- 问题: 给定两个代数电路，是否计算同一多项式？
- Schwartz-Zippel引理: 随机点赋值，若不同则大概率不同

**素数测试**:
- Miller-Rabin测试: 多项式时间，单侧错误
- AKS算法 (2002): 确定性多项式时间

## 五、示例与实例

### 5.1 多项式恒等测试

**问题**: 给定多变量多项式p(x₁,...,xₙ)，p是否恒等于0？

**随机算法**:
- 从有限域F中选择随机点a₁,...,aₙ
- 计算p(a₁,...,aₙ)
- 若非零，则p不恒为零
- 若为零，大概率p恒为零

**Schwartz-Zippel引理**: 若p非零，次数d，则
Pr[p(a₁,...,aₙ) = 0] ≤ d/|F|

### 5.2 Miller-Rabin素数测试

**算法**:
- 输入: 奇数n
- 写n-1 = 2^s·d
- 随机选a ∈ [2, n-2]
- 计算a^d mod n
- 若 ≡ 1 (mod n)，可能为素数
- 否则检查a^(2^r·d) ≡ -1 (mod n) 对某个r

**性质**:
- 若n为素数，总接受
- 若n为合数，错误概率 ≤ 1/4

## 六、习题

### 6.1 理解题 (L1-L2)

1. **错误放大** [难度⭐⭐]
   
   设M是错误概率1/3的BPP机器。若重复运行9次取多数，错误概率上界是多少？
   
   <details>
   <summary>解答</summary>
   
   设X为正确运行的次数，E[X] ≥ 6（若x ∈ L）。
   错误当X ≤ 4。
   
   由Chernoff界（简化）：
   Pr[X ≤ 4.5] ≤ Pr[|X-6| ≥ 1.5] ≤ 2exp(-2·1.5²/9) = 2exp(-0.5) ≈ 1.2
   
   更精确分析：需至少5次正确，用二项分布。
   实际上Pr[错误] < (3/4)^(9/2) ≈ 0.015。
   
   </details>

2. **ZPP刻画** [难度⭐⭐⭐]
   
   证明ZPP等于"期望多项式时间且永不错误的PTM"可判定的语言类。
   
   <details>
   <summary>解答</summary>
   
   (⇒) 设L ∈ ZPP = RP ∩ co-RP。
   交替运行RP和co-RP算法：
   - RP算法接受则接受
   - co-RP算法拒绝则拒绝
   期望运行次数有限（几何分布），永不错误。
   
   (⇐) 设M期望多项式时间且永不错误。
   在p(n)步截断：
   - 若停机，给出正确答案
   - 若超时，可接受或拒绝
   因期望时间多项式，超时概率小，得RP和co-RP算法。
   
   </details>

### 6.2 证明题 (L3-L4)

3. **BPP ⊆ Σ₂** [难度⭐⭐⭐⭐]
   
   证明BPP ⊆ Σ₂（Sipser-Gács-Lautemann定理）。
   
   <details>
   <summary>解答概要</summary>
   
   设L ∈ BPP，错误概率< 2^(-n)。
   对固定x，好随机串占绝大多数。
   
   存在多项式p，使得：
   x ∈ L ⟺ ∃r₁,...,r_p ∀r: M(x,r)与多数M(x,r_i)一致
   
   这给出了Σ₂的定义。
   
   </details>

## 八、多维对比

### 8.1 概率类对比

| 类 | 错误类型 | 错误概率 | 关系 |
|----|----------|----------|------|
| ZPP | 零错误 | 0 | = RP ∩ co-RP |
| RP | 单侧 | ≤ 1/2 (仅可能错拒) | ⊆ NP |
| co-RP | 单侧 | ≤ 1/2 (仅可能错接) | ⊆ co-NP |
| BPP | 双侧有界 | ≤ 1/3 | 猜想 = P |
| PP | 无界 | < 1/2 | ⊇ NP |

# C++ 算法实现

## 概述

C++ 实现库位于 `examples/algorithms-cpp/`，采用 CMake 构建系统与 C++17 标准，强调模板泛型与零开销抽象。

## 项目结构

```
algorithms-cpp/
├── CMakeLists.txt
├── sorting.cpp          # 快速、归并、堆、计数、希尔排序
├── search.cpp           # 二分、线性搜索
├── graph.cpp            # BFS、DFS、Dijkstra、Floyd-Warshall
├── dynamic.cpp          # 背包、LCS、LIS
├── union_find.cpp       # 并查集
├── segment_tree.cpp     # 线段树
├── string.cpp           # KMP、Manacher、Z函数、滚动哈希
├── number_theory.cpp    # GCD、扩展欧几里得、快速幂、Miller-Rabin、筛法
├── tree.cpp             # LCA(倍增)、笛卡尔树
├── geometry.cpp         # 凸包、最近点对、线段相交
└── advanced.cpp         # 模拟退火
```

## 构建

```bash
cd examples/algorithms-cpp
mkdir build && cd build
cmake ..
make
```

## 模板与命名空间

- 所有算法置于 `namespace` 中（如 `sorting::`, `geometry::`）
- 排序与搜索使用 **函数模板** (`template<typename T>`)
- 数据结构使用 **类模板**，支持自定义类型

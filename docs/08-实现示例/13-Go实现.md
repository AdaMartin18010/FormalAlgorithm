# Go 算法实现

## 概述

Go 实现库位于 `examples/algorithms-go/`，采用标准 Go Modules 管理，强调简洁、并发友好与工程实践。

## 项目结构

```
algorithms-go/
├── go.mod
├── sorting.go / sorting_test.go
├── search.go / search_test.go
├── graph.go / graph_test.go
├── dynamic.go / dynamic_test.go
├── unionfind.go / unionfind_test.go
├── string.go / string_test.go         # KMP、Manacher、Z函数
├── number_theory.go / number_theory_test.go  # GCD、快速幂、素性测试
├── tree.go / tree_test.go             # LCA、笛卡尔树
├── geometry.go                        # 凸包、最近点对、线段相交
└── advanced.go                        # 模拟退火
```

## 构建与测试

```bash
cd examples/algorithms-go
go test ./...
go test -bench=.
```

## 设计特点

- **包级函数**：Go 无类，算法以包级函数组织（`package algorithms`）
- **切片（Slice）**：统一使用 `[]int` / `[]Point` 作为序列容器
- **结构体 + 方法**：面向对象场景使用结构体与方法（如 `LCA.Query()`）
- **零值初始化**：利用 Go 的零值特性简化构造函数

## 与 Rust 主实现的对齐

| Rust | Go | 状态 |
|------|-----|------|
| `sorting.rs` | `sorting.go` | ✅ |
| `graph.rs` | `graph.go` | ✅ |
| `union_find.rs` | `unionfind.go` | ✅ |
| `kmp.rs` | `KMPSearch()` | ✅ 新增 |
| `tree.rs` | `NewLCA()` / `NewCartesianTree()` | ✅ 新增 |
| `number_theory.rs` | `FastPow()` / `IsPrime()` | ✅ 新增 |

## 并发扩展方向

Go 的 goroutine 与 channel 天然适合并行算法教学：

- 并行归并排序（`goroutine` + `sync.WaitGroup`）
- 并行 BFS（`sync.Map` 作为访问标记）
- 流水线模式下的流算法

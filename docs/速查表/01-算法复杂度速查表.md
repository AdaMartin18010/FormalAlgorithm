## 🔍 复杂度符号速查

| 符号 | 名称 | 含义 | 示例 |
|-----|------|------|------|
| O(f(n)) | 大O | 渐进上界 | O(n²) 最坏情况不超过 n² |
| Ω(f(n)) | 大Ω | 渐进下界 | Ω(n) 最好情况至少 n |
| Θ(f(n)) | 大Θ | 紧致界 | Θ(n log n) 上下界相同 |
| o(f(n)) | 小o | 严格小于 | o(n²) 比 n² 增长慢 |
| ω(f(n)) | 小ω | 严格大于 | ω(n) 比 n 增长快 |

## 🎯 复杂度分析技巧

### 主定理（Master Theorem）

对于 T(n) = aT(n/b) + f(n)：

| 条件 | 结果 | 示例 |
|-----|------|------|
| f(n) = O(n^(log_b(a)-ε)) | T(n) = Θ(n^(log_b(a))) | 归并排序 |
| f(n) = Θ(n^(log_b(a))) | T(n) = Θ(n^(log_b(a)) log n) | 平衡分割 |
| f(n) = Ω(n^(log_b(a)+ε)) | T(n) = Θ(f(n)) | 快速排序最坏 |

### 平摊分析

| 方法 | 用途 | 示例 |
|-----|------|------|
| 聚合分析 | 总时间/操作数 | 栈的MULTIPOP |
| 核算法 | 为操作分配代价 | 动态表扩展 |
| 势能法 | 定义势函数 | 斐波那契堆 |

## 一、规范定义深化

### 1.1 计算问题的形式化定义

**定义 1.1.1**（判定问题 / Decision Problem）

一个判定问题是一个语言 $L \subseteq \Sigma^*$，其中 $\Sigma$ 为有限字母表。对于输入 $x \in \Sigma^*$：
- 若 $x \in L$，回答 **Yes**（接受）
- 若 $x \notin L$，回答 **No**（拒绝）

**定义 1.1.2**（搜索问题 / Search Problem）

搜索问题由一个关系 $R \subseteq \Sigma^* \times \Sigma^*$ 定义。给定 $x$，寻找 $y$ 使得 $(x,y) \in R$，或报告不存在。

**定义 1.1.3**（优化问题 / Optimization Problem）

优化问题由四元组 $(I, S, v, \text{goal})$ 定义：
- $I$: 实例集合
- $S(x)$: 实例 $x$ 的可行解集合
- $v(x,y)$: 解 $y$ 的代价函数
- $\text{goal} \in \{\min, \max\}$: 优化目标

### 1.2 算法与计算模型的公理化

**定义 1.2.1**（算法 / Algorithm）

算法是一个有限指令序列 $\mathcal{A} = (I_1, I_2, \ldots, I_k)$，在抽象计算模型 $M$ 上执行，将输入 $x \in \Sigma^*$ 映射到输出 $y \in \Sigma^*$ 或进入停机状态。

**定义 1.2.2**（时间复杂度函数）

给定算法 $\mathcal{A}$ 和计算模型 $M$，时间复杂度函数定义为：

$$
T_{\mathcal{A}}(n) = \max_{|x| = n} \{\text{算法 } \mathcal{A} \text{ 在输入 } x \text{ 上的运行步数}\}
$$

**定义 1.2.3**（空间复杂度函数）

$$
S_{\mathcal{A}}(n) = \max_{|x| = n} \{\text{算法 } \mathcal{A} \text{ 在输入 } x \text{ 上使用的工作空间单元数}\}
$$

## 二、模型设计深化

### 2.1 图灵机模型

**定义 2.1.1**（确定性图灵机 / DTM）

一个 $k$-带确定性图灵机是一个七元组 $M = (Q, \Sigma, \Gamma, \delta, q_0, q_{\text{accept}}, q_{\text{reject}})$：

- $Q$: 有限状态集
- $\Sigma$: 输入字母表（$\sqcup \notin \Sigma$）
- $\Gamma$: 带字母表（$\Sigma \subseteq \Gamma$，$\sqcup \in \Gamma$）
- $\delta: Q \times \Gamma^k \rightarrow Q \times \Gamma^k \times \{L, R, S\}^k$: 转移函数
- $q_0 \in Q$: 初始状态
- $q_{\text{accept}}, q_{\text{reject}} \in Q$: 接受/拒绝状态

**时间复杂度**：$T_M(n) = \max_{|x|=n} \{\text{计算路径的步数}\}$

### 2.2 RAM 模型

**定义 2.2.1**（随机存取机 / RAM）

RAM 是一个抽象计算模型：
- **内存**：无限个寄存器 $R_0, R_1, R_2, \ldots$，每个存储一个整数
- **程序**：有限指令序列
- **指令集**：加载、存储、算术、跳转、间接寻址

**均匀代价准则**：每条指令耗时 1 单位。
**对数代价准则**：操作耗时与操作数位数成正比，即 $\Theta(\log |x|)$。

### 2.3 模型等价性

**定理 2.3.1**：RAM（均匀代价）与图灵机在多项式时间类等价。

$$
T_{\text{TM}}(n) = O\bigl(T_{\text{RAM}}(n)^2 \cdot \log T_{\text{RAM}}(n)\bigr)
$$

**定理 2.3.2**（丘奇-图灵论题的时间版本）：所有"合理的"计算模型在多项式时间内等价。

## 三、数学符号与推导

### 3.1 复杂度类的基本运算

**定义 3.1.1**（复杂度类的并、交、补）

$$
\begin{aligned}
\mathcal{C}_1 \cup \mathcal{C}_2 &= \{L : L \in \mathcal{C}_1 \lor L \in \mathcal{C}_2\} \\
\mathcal{C}_1 \cap \mathcal{C}_2 &= \{L : L \in \mathcal{C}_1 \land L \in \mathcal{C}_2\} \\
\text{co-}\mathcal{C} &= \{\overline{L} : L \in \mathcal{C}\}
\end{aligned}
$$

**定义 3.1.2**（时间复杂度类）

$$
\text{DTIME}(f(n)) = \{L : \exists \text{ DTM } M \text{ 在时间 } O(f(n)) \text{ 内判定 } L\}
$$

$$
\text{NTIME}(f(n)) = \{L : \exists \text{ NTM } M \text{ 在时间 } O(f(n)) \text{ 内判定 } L\}
$$

### 3.2 核心复杂度类定义

| 类 | 定义 | 典型问题 |
|----|------|----------|
| **P** | $\bigcup_k \text{DTIME}(n^k)$ | 最短路径、匹配、线性规划 |
| **NP** | $\bigcup_k \text{NTIME}(n^k)$ | SAT、团、背包、旅行商（判定） |
| **PSPACE** | $\bigcup_k \text{DSPACE}(n^k)$ | QSAT、博弈论问题 |
| **EXP** | $\bigcup_k \text{DTIME}(2^{n^k})$ | 广义棋类游戏 |
| **L** | $\text{DSPACE}(\log n)$ | 连通性（无向图） |
| **NL** | $\text{NSPACE}(\log n)$ | 连通性（有向图） |
| **BPP** | 有界错误概率多项式时间 | 多项式恒等测试 |
| **BQP** | 有界错误量子多项式时间 | 因子分解、模拟量子系统 |

### 3.3 层次定理

**定理 3.3.1**（确定性时间层次定理）

若 $f(n)$ 和 $g(n)$ 为时间可构造函数，且 $f(n) \log f(n) = o(g(n))$，则：

$$
\text{DTIME}(f(n)) \subsetneq \text{DTIME}(g(n))
$$

**定理 3.3.2**（空间层次定理）

若 $f(n) = o(g(n))$，则：

$$
\text{DSPACE}(f(n)) \subsetneq \text{DSPACE}(g(n))
$$

use std::collections::{HashMap, HashSet};

#[derive(Debug, Clone, PartialEq, Eq, Hash)]
pub enum ComplexityClass {
    P, NP, CoNP, PSpace, EXP, NEXP,
    BPP, RP, CoRP, ZPP, BQP,
    L, NL, CoNL, NPComplete, NPHard,
}

impl ComplexityClass {
    pub fn name(&self) -> &'static str {
        match self {
            ComplexityClass::P => "P",
            ComplexityClass::NP => "NP",
            ComplexityClass::CoNP => "co-NP",
            ComplexityClass::PSpace => "PSPACE",
            ComplexityClass::EXP => "EXP",
            ComplexityClass::NEXP => "NEXP",
            ComplexityClass::BPP => "BPP",
            ComplexityClass::RP => "RP",
            ComplexityClass::CoRP => "co-RP",
            ComplexityClass::ZPP => "ZPP",
            ComplexityClass::BQP => "BQP",
            ComplexityClass::L => "L",
            ComplexityClass::NL => "NL",
            ComplexityClass::CoNL => "co-NL",
            ComplexityClass::NPComplete => "NP-complete",
            ComplexityClass::NPHard => "NP-hard",
        }
    }

    pub fn description(&self) -> &'static str {
        match self {
            ComplexityClass::P => "多项式时间确定性可解",
            ComplexityClass::NP => "多项式时间非确定性可解 / 解可验证",
            ComplexityClass::CoNP => "补集属于NP",
            ComplexityClass::PSpace => "多项式空间可解",
            ComplexityClass::EXP => "指数时间确定性可解",
            ComplexityClass::NEXP => "指数时间非确定性可解",
            ComplexityClass::BPP => "有界错误概率多项式时间",
            ComplexityClass::RP => "单侧错误概率多项式时间",
            ComplexityClass::CoRP => "RP的补集类",
            ComplexityClass::ZPP => "零错误概率多项式时间",
            ComplexityClass::BQP => "有界错误量子多项式时间",
            ComplexityClass::L => "对数空间确定性可解",
            ComplexityClass::NL => "对数空间非确定性可解",
            ComplexityClass::CoNL => "NL的补集类 (= NL)",
            ComplexityClass::NPComplete => "NP中最难的问题",
            ComplexityClass::NPHard => "至少与NP中最难问题一样难",
        }
    }
}

pub struct ComplexityGraph {
    edges: HashMap<ComplexityClass, Vec<(ComplexityClass, &'static str)>>,
}

impl ComplexityGraph {
    pub fn standard() -> Self {
        let mut edges = HashMap::new();
        let inclusions = vec![
            (ComplexityClass::L, ComplexityClass::NL, "⊆"),
            (ComplexityClass::NL, ComplexityClass::P, "⊆ (开放问题)"),
            (ComplexityClass::P, ComplexityClass::NP, "⊆ (P vs NP)"),
            (ComplexityClass::P, ComplexityClass::BPP, "⊆ (开放问题)"),
            (ComplexityClass::NP, ComplexityClass::PSpace, "⊆ (Savitch)"),
            (ComplexityClass::CoNP, ComplexityClass::PSpace, "⊆"),
            (ComplexityClass::BPP, ComplexityClass::PSpace, "⊆"),
            (ComplexityClass::PSpace, ComplexityClass::EXP, "⊆"),
            (ComplexityClass::EXP, ComplexityClass::NEXP, "⊆"),
            (ComplexityClass::RP, ComplexityClass::NP, "⊆"),
            (ComplexityClass::CoRP, ComplexityClass::CoNP, "⊆"),
            (ComplexityClass::ZPP, ComplexityClass::RP, "⊆"),
            (ComplexityClass::BQP, ComplexityClass::PSpace, "⊆"),
            (ComplexityClass::NPComplete, ComplexityClass::NP, "⊆"),
        ];
        for (from, to, label) in inclusions {
            edges.entry(from.clone()).or_default().push((to, label));
        }
        Self { edges }
    }

    pub fn generate_dot(&self) -> String {
        let mut dot = String::from("digraph ComplexityClasses {\n");
        dot.push_str("  rankdir=BT;\n");
        dot.push_str("  node [shape=box, style=rounded];\n\n");

        let mut nodes = HashSet::new();
        for (from, tos) in &self.edges {
            nodes.insert(from.clone());
            for (to, _) in tos { nodes.insert(to.clone()); }
        }

        for node in &nodes {
            dot.push_str(&format!(
                "  \"{}\" [label=\"{}\\n{}\"];\n",
                node.name(), node.name(), node.description()
            ));
        }
        dot.push_str("\n");
        for (from, tos) in &self.edges {
            for (to, label) in tos {
                dot.push_str(&format!(
                    "  \"{}\" -> \"{}\" [label=\"{}\"];\n",
                    from.name(), to.name(), label
                ));
            }
        }
        dot.push_str("}\n");
        dot
    }
}

fn main() {
    let graph = ComplexityGraph::standard();
    println!("{}", graph.generate_dot());
}

## 五、形式化证明

### 5.1 时间层次定理证明

**定理 5.1.1**：若 $f(n)$ 时间可构造，则 $\text{DTIME}(f(n)) \subsetneq \text{DTIME}(f(n)^3)$。

**证明概要**：

1. **对角化构造**：定义语言 $L = \{x : M_x \text{ 在 } f(|x|) \text{ 步内不接受 } x\}$，其中 $M_x$ 是由 $x$ 编码的图灵机。

2. **$L \in \text{DTIME}(f(n)^3)$**：模拟 $M_x$ 在 $x$ 上运行 $f(|x|)$ 步，需要 $O(f(n)^2)$ 时间进行单带模拟，总计 $O(f(n)^3)$。

3. **$L \notin \text{DTIME}(f(n))$**：假设存在 $M$ 在时间 $f(n)$ 内判定 $L$。取 $x$ 为 $M$ 的编码，则：
   - 若 $x \in L$，则 $M$ 接受 $x$，由 $L$ 定义 $M$ 不应接受 $x$，矛盾；
   - 若 $x \notin L$，则 $M$ 拒绝 $x$，由 $L$ 定义 $M$ 应接受 $x$，矛盾。

### 5.2 Savitch 定理

**定理 5.2.1**（Savitch, 1970）：

$$
\text{NSPACE}(S(n)) \subseteq \text{DSPACE}(S(n)^2)
$$

对于 $S(n) \ge \log n$。

**推论**：$\text{PSPACE} = \text{NPSPACE}$。

### 5.3 Immerman-Szelepcsényi 定理

**定理 5.3.1**：对于 $S(n) \ge \log n$：

$$
\text{NSPACE}(S(n)) = \text{co-NSPACE}(S(n))
$$

**推论**：$\text{NL} = \text{co-NL}$。

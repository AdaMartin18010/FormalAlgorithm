## 一、规范定义深化

### 1.1 主定理的完整形式化定义

**定义 1.1.1**（主定理 / Master Theorem）

设 $a \ge 1$ 和 $b > 1$ 为常数，$f(n)$ 为一个函数，$T(n)$ 由以下递推式定义：

$$
T(n) = a \cdot T\left(\frac{n}{b}\right) + f(n)
$$

其中 $n/b$ 可解释为 $\lfloor n/b \rfloor$ 或 $\lceil n/b \rceil$。令：

$$
\alpha = \log_b a = \frac{\ln a}{\ln b}
$$

则 $T(n)$ 有如下渐近界：

| Case | 条件 | 结论 |
|------|------|------|
| **1** | $f(n) = O(n^{\alpha - \epsilon})$，$\exists \epsilon > 0$ | $T(n) = \Theta(n^{\alpha})$ |
| **2** | $f(n) = \Theta(n^{\alpha})$ | $T(n) = \Theta(n^{\alpha} \log n)$ |
| **2'** | $f(n) = \Theta(n^{\alpha} \log^k n)$，$k \ge 0$ | $T(n) = \Theta(n^{\alpha} \log^{k+1} n)$ |
| **3** | $f(n) = \Omega(n^{\alpha + \epsilon})$，$\exists \epsilon > 0$，且满足**正则条件** | $T(n) = \Theta(f(n))$ |

**正则条件**（Regularity Condition）：

$$
\exists c < 1,\; \exists n_0 > 0:\quad \forall n \ge n_0,\; a \cdot f\left(\frac{n}{b}\right) \le c \cdot f(n)
$$

### 1.2 定义域与边界条件

- **定义域**：$n \in \mathbb{N}^+$，且 $n$ 为 $b$ 的幂次时分析最简洁
- **边界条件**：$T(1) = \Theta(1)$ 或 $T(c) = \Theta(1)$（$c$ 为常数）
- **单调性假设**：$f(n)$ 需满足多项式有界或平滑条件，确保渐近分析有效

## 二、模型设计深化

### 2.1 主定理适用的计算模型

| 模型 | 适用性 | 说明 |
|------|--------|------|
| **RAM（均匀代价）** | ✅ 直接适用 | 基本操作代价为 $O(1)$ |
| **RAM（对数代价）** | ⚠️ 需调整 | 大整数操作引入额外 $\log$ 因子 |
| **图灵机** | ⚠️ 需模拟开销 | 时间关系为多项式等价 |
| **并行 PRAM** | ❌ 不适用 | 需使用并行主定理变体 |

### 2.2 主定理失效的情形

以下情况主定理**不直接适用**：

1. **$f(n)$ 与 $n^{\alpha}$ 之间无多项式差距**：
   $$T(n) = 2T(n/2) + n \log n$$
   此时 $f(n) = n \log n$ 不是多项式大于或小于 $n$，属于 Case 2 与 Case 3 之间的"间隙"。

2. **非均匀划分**：$T(n) = T(\lfloor n/2 \rfloor) + T(\lceil 3n/4 \rceil) + f(n)$

3. **非常规子问题数量**：$T(n) = \sqrt{n} \cdot T(\sqrt{n}) + n$

4. **$a$ 或 $b$ 不是常数**：如 $T(n) = n \cdot T(n/2) + n^2$

对于这些情况，应使用 **Akra-Bazzi 方法**或**递归树法**。

## 三、数学符号与推导

### 3.1 ISO 80000 对齐的符号规范

| 符号 | 含义 | ISO 对齐 |
|------|------|----------|
| $T(n)$ | 时间复杂度函数 | 函数符号 $T: \mathbb{N} \to \mathbb{R}^+$ |
| $a$ | 子问题数 | 无量纲正实数 |
| $b$ | 规模缩小因子 | 无量纲实数，$b > 1$ |
| $\alpha = \log_b a$ | 关键指数 | 对数记号 $\log_b$ 符合 ISO 80000-2 |
| $f(n)$ | 合并代价 | 函数符号 |
| $\epsilon$ | 任意小正数 | 希腊字母小写，$\epsilon > 0$ |
| $\Theta, O, \Omega$ | 渐近紧界、上界、下界 | 符合 Knuth 记号规范 |

### 3.2 Case 1 的完整推导

**条件**：$f(n) = O(n^{\alpha - \epsilon})$，$\epsilon > 0$

递归树总代价：

$$
\begin{aligned}
T(n) &= \sum_{i=0}^{\log_b n} a^i \cdot f\left(\frac{n}{b^i}\right) \\
&\le \sum_{i=0}^{\log_b n} a^i \cdot C \left(\frac{n}{b^i}\right)^{\alpha - \epsilon} \\
&= C n^{\alpha - \epsilon} \sum_{i=0}^{\log_b n} a^i \cdot b^{-i(\alpha - \epsilon)} \\
&= C n^{\alpha - \epsilon} \sum_{i=0}^{\log_b n} \left(a \cdot b^{-\alpha} \cdot b^{\epsilon}\right)^i \\
&= C n^{\alpha - \epsilon} \sum_{i=0}^{\log_b n} \left(b^{\epsilon}\right)^i \quad (\because a \cdot b^{-\alpha} = 1) \\
&= C n^{\alpha - \epsilon} \cdot \frac{b^{\epsilon(\log_b n + 1)} - 1}{b^{\epsilon} - 1} \\
&= C n^{\alpha - \epsilon} \cdot \Theta(n^{\epsilon}) \\
&= \Theta(n^{\alpha})
\end{aligned}
$$

### 3.3 Case 3 的正则条件分析

正则条件 $a \cdot f(n/b) \le c \cdot f(n)$（$c < 1$）确保 $f(n)$ 的增长速度足以"吸收"递归开销。

**多项式情形的验证**：设 $f(n) = \Omega(n^{\alpha + \epsilon})$，即 $f(n) \ge d \cdot n^{\alpha + \epsilon}$。

$$
\begin{aligned}
a \cdot f\left(\frac{n}{b}\right) &\le a \cdot D \left(\frac{n}{b}\right)^{\alpha + \epsilon} \\
&= D \cdot a \cdot b^{-\alpha} \cdot b^{-\epsilon} \cdot n^{\alpha + \epsilon} \\
&= D \cdot b^{-\epsilon} \cdot n^{\alpha + \epsilon} \\
&\le \frac{b^{-\epsilon} \cdot D}{d} \cdot f(n)
\end{aligned}
$$

取 $c = b^{-\epsilon} < 1$（当 $D/d$ 有界时），正则条件成立。

## 四、示例性代码

### 4.1 Rust：主定理自动判断与求解器

```rust
/// Master Theorem 自动判断器与求解器
#[derive(Debug, Clone, PartialEq)]
pub enum MasterCase {
    Case1 { log_b_a: f64 },
    Case2 { log_b_a: f64, k: f64 },
    Case3 { log_b_a: f64 },
    ExtendedCase2Prime { log_b_a: f64, k: f64 },
    NonStandard,
}

impl MasterCase {
    pub fn describe(&self) -> String {
        match self {
            MasterCase::Case1 { log_b_a } => {
                format!("Case 1: f(n)=O(n^{:.3}-ε), T(n)=Θ(n^{:.3})",
                        log_b_a, log_b_a)
            }
            MasterCase::Case2 { log_b_a, .. } => {
                format!("Case 2: f(n)=Θ(n^{:.3}), T(n)=Θ(n^{:.3}·log n)",
                        log_b_a, log_b_a)
            }
            MasterCase::Case3 { log_b_a } => {
                format!("Case 3: f(n)=Ω(n^{:.3}+ε), T(n)=Θ(f(n))",
                        log_b_a)
            }
            MasterCase::ExtendedCase2Prime { log_b_a, k } => {
                format!(
                    "Case 2': f(n)=Θ(n^{:.3}·log^{:.0}n), \
                     T(n)=Θ(n^{:.3}·log^{:.0}n)",
                    log_b_a, k + 1.0, log_b_a, k + 1.0
                )
            }
            MasterCase::NonStandard => {
                "非标准形式，主定理不适用".to_string()
            }
        }
    }
}

pub fn classify(a: f64, b: f64, d: f64, k: f64)
    -> Result<MasterCase, String>
{
    if a < 1.0 {
        return Err("参数 a 必须 ≥ 1".to_string());
    }
    if b <= 1.0 {
        return Err("参数 b 必须 > 1".to_string());
    }

    let log_b_a = a.log(b);
    let eps = 1e-9;

    if (d - log_b_a).abs() < eps {
        if k.abs() < eps {
            Ok(MasterCase::Case2 { log_b_a, k })
        } else {
            Ok(MasterCase::ExtendedCase2Prime { log_b_a, k })
        }
    } else if d < log_b_a - eps {
        Ok(MasterCase::Case1 { log_b_a })
    } else if d > log_b_a + eps {
        // 多项式 f(n)=n^d·log^k n 在 d>log_b a 时正则条件自动满足
        Ok(MasterCase::Case3 { log_b_a })
    } else {
        Ok(MasterCase::NonStandard)
    }
}

fn main() {
    let examples = vec![
        ("归并排序", 2.0, 2.0, 1.0, 0.0),
        ("Strassen矩阵乘法", 7.0, 2.0, 0.0, 0.0),
        ("Karatsuba乘法", 3.0, 2.0, 1.0, 0.0),
        ("T(n)=4T(n/2)+n^2", 4.0, 2.0, 2.0, 0.0),
        ("T(n)=2T(n/2)+n·log n", 2.0, 2.0, 1.0, 1.0),
    ];

    for (name, a, b, d, k) in examples {
        let case = classify(a, b, d, k).unwrap();
        println!("{} → {}", name, case.describe());
    }
}
```

### 4.2 Rust：递归树可视化辅助

```rust
/// 递归树分析辅助：计算各层代价
pub fn recurrence_tree_levels(a: f64, b: f64, d: f64,
                               n: f64) -> Vec<(usize, f64, f64)>
{
    let mut levels = vec![];
    let mut level = 0usize;
    let mut size = n;
    let mut nodes = 1.0;

    while size >= 1.0 {
        let cost_per_node = size.powf(d);
        let total_cost = nodes * cost_per_node;
        levels.push((level, nodes, total_cost));

        nodes *= a;
        size /= b;
        level += 1;
    }
    levels
}
```

## 五、形式化证明

### 5.1 主定理完整证明框架

**引理 5.1.1**（递归树求和）：对于 $T(n) = aT(n/b) + f(n)$，

$$
T(n) = \Theta(n^{\alpha}) + \sum_{j=0}^{\log_b n - 1} a^j \cdot f\left(\frac{n}{b^j}\right)
$$

**证明**：展开递推式：

$$
\begin{aligned}
T(n) &= f(n) + aT\left(\frac{n}{b}\right) \\
&= f(n) + a f\left(\frac{n}{b}\right) + a^2 T\left(\frac{n}{b^2}\right) \\
&= \sum_{j=0}^{\log_b n - 1} a^j f\left(\frac{n}{b^j}\right) + a^{\log_b n} T(1) \\
&= \sum_{j=0}^{\log_b n - 1} a^j f\left(\frac{n}{b^j}\right) + \Theta(n^{\alpha})
\end{aligned}
$$

**Case 1 证明**：

由 $f(n) = O(n^{\alpha - \epsilon})$：

$$
\begin{aligned}
g(n) &= \sum_{j=0}^{\log_b n - 1} a^j f\left(\frac{n}{b^j}\right) \\
&\le C \sum_{j=0}^{\log_b n - 1} a^j \left(\frac{n}{b^j}\right)^{\alpha - \epsilon} \\
&= C n^{\alpha - \epsilon} \sum_{j=0}^{\log_b n - 1} \left(b^{\epsilon}\right)^j \\
&= C n^{\alpha - \epsilon} \cdot \Theta(n^{\epsilon}) = \Theta(n^{\alpha})
\end{aligned}
$$

故 $T(n) = \Theta(n^{\alpha}) + \Theta(n^{\alpha}) = \Theta(n^{\alpha})$。

**Case 2 证明**：

由 $f(n) = \Theta(n^{\alpha})$，第 $j$ 层代价：

$$
a^j f\left(\frac{n}{b^j}\right) = a^j \cdot \Theta\left(\left(\frac{n}{b^j}\right)^{\alpha}\right)
= \Theta(n^{\alpha})
$$

共 $\log_b n$ 层，故 $g(n) = \Theta(n^{\alpha} \log n)$。

**Case 3 证明**：

由正则条件，$a^j f(n/b^j) \le c^j f(n)$：

$$
g(n) \le f(n) \sum_{j=0}^{\infty} c^j = \frac{f(n)}{1-c} = O(f(n))
$$

又 $g(n) \ge f(n)$，故 $g(n) = \Theta(f(n))$，$T(n) = \Theta(f(n))$。

### 5.2 主定理的逆否命题

**定理 5.2.1**：若 $f(n)$ 不满足 Case 1、2、3 中的任一条件（即 $f(n)$ 与 $n^{\alpha}$ 之间无多项式差距），则主定理的结论可能不成立。

**示例**：$T(n) = 2T(n/2) + n \log n$

- $\alpha = 1$，$f(n) = n \log n$
- 既非 $O(n^{1-\epsilon})$，也非 $\Theta(n^1)$，亦非 $\Omega(n^{1+\epsilon})$
- 实际解：$T(n) = \Theta(n \log^2 n)$（需用扩展 Case 2' 或递归树）

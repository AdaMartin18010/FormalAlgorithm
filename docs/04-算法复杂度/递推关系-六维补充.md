## 一、规范定义深化

### 1.1 递推关系的严格数学定义

**定义 1.1.1**（递推关系 / Recurrence Relation）

设 $k \in \mathbb{N}^+$，函数 $T: \mathbb{N} \rightarrow \mathbb{R}^+$ 的**$k$ 阶递推关系**是一个等式：

$$
T(n) = F\bigl(n, T(n-1), T(n-2), \ldots, T(n-k)\bigr), \quad n \ge n_0
$$

其中 $F$ 为已知函数，$T(1), \ldots, T(k)$ 为初始条件（边界条件）。

**定义 1.1.2**（分治型递推 / Divide-and-Conquer Recurrence）

典型形式为：

$$
T(n) = a \cdot T\left(\frac{n}{b}\right) + f(n), \quad a \ge 1,\; b > 1
$$

其中：
- $a$: 子问题数量（$a \in \mathbb{R}^+$）
- $b$: 子问题规模缩小因子（$b \in \mathbb{R}^+$）
- $f(n)$: 分解与合并的代价

**定义 1.1.3**（线性常系数递推 / Linear Recurrence with Constant Coefficients）

$$
T(n) = c_1 T(n-1) + c_2 T(n-2) + \cdots + c_k T(n-k) + g(n)
$$

其**特征方程**为：

$$
r^k - c_1 r^{k-1} - c_2 r^{k-2} - \cdots - c_k = 0
$$

### 1.2 渐进解的精确性分类

| 类别 | 定义 | 示例 |
|------|------|------|
| **精确解** | $T(n)$ 的闭式表达式 | $T(n) = 2^n - 1$（汉诺塔） |
| **渐近上界** | $T(n) = O(g(n))$ | $T(n) = O(n \log n)$ |
| **渐近紧界** | $T(n) = \Theta(g(n))$ | $T(n) = \Theta(n^2)$ |
| **主控项** | 决定增长率的最高阶项 | $n^{\log_b a}$ 或 $f(n)$ |

## 二、模型设计深化

### 2.1 计算模型对递推的影响

不同计算模型下，同一算法的递推式可能不同：

**RAM 模型（均匀代价）**：
- 每次基本操作耗时 $1$
- $T(n) = aT(n/b) + O(n^d)$

**RAM 模型（对数代价）**：
- 操作耗时与操作数位数成正比
- 乘法等大整数操作引入 $\log$ 因子

**图灵机模型**：
- 模拟 RAM 需对数因子开销
- $T_{\text{TM}}(n) = O\bigl(T_{\text{RAM}}(n) \cdot \log T_{\text{RAM}}(n)\bigr)$

### 2.2 递推树模型

递推树是分析分治递推的直观工具：

- **深度**：$h = \log_b n$
- **第 $i$ 层节点数**：$a^i$
- **第 $i$ 层每个子问题规模**：$n / b^i$
- **第 $i$ 层总代价**：$a^i \cdot f(n / b^i)$

总代价：

$$
T(n) = \sum_{i=0}^{\log_b n} a^i \cdot f\left(\frac{n}{b^i}\right)
$$

## 三、数学符号与推导

### 3.1 几何级数求和公式

对于 $|r| \neq 1$：

$$
\sum_{i=0}^{k} r^i = \frac{r^{k+1} - 1}{r - 1}
$$

当 $|r| < 1$ 且 $k \to \infty$：

$$
\sum_{i=0}^{\infty} r^i = \frac{1}{1 - r}
$$

### 3.2 主定理的完整推导

**定理 3.2.1**（主定理 / Master Theorem）

对于 $T(n) = aT(n/b) + f(n)$，其中 $a \ge 1, b > 1$，令 $\alpha = \log_b a$：

**Case 1**：若 $f(n) = O(n^{\alpha - \epsilon})$（$\epsilon > 0$），则 $T(n) = \Theta(n^{\alpha})$。

*证明*：

$$
\begin{aligned}
T(n) &= \sum_{i=0}^{\log_b n} a^i \cdot f\left(\frac{n}{b^i}\right) \\
&\le \sum_{i=0}^{\log_b n} a^i \cdot C \left(\frac{n}{b^i}\right)^{\alpha - \epsilon} \\
&= C n^{\alpha - \epsilon} \sum_{i=0}^{\log_b n} a^i \cdot b^{-i(\alpha - \epsilon)} \\
&= C n^{\alpha - \epsilon} \sum_{i=0}^{\log_b n} (b^{\epsilon})^i \\
&= C n^{\alpha - \epsilon} \cdot \Theta(n^{\epsilon}) = \Theta(n^{\alpha})
\end{aligned}
$$

**Case 2**：若 $f(n) = \Theta(n^{\alpha})$，则 $T(n) = \Theta(n^{\alpha} \log n)$。

*证明*：

第 $i$ 层总代价：

$$
a^i \cdot f\left(\frac{n}{b^i}\right) = a^i \cdot \Theta\left(\left(\frac{n}{b^i}\right)^{\alpha}\right) = \Theta(n^{\alpha})
$$

共 $\log_b n + 1$ 层，每层 $\Theta(n^{\alpha})$：

$$
T(n) = (\log_b n + 1) \cdot \Theta(n^{\alpha}) = \Theta(n^{\alpha} \log n)
$$

**Case 3**：若 $f(n) = \Omega(n^{\alpha + \epsilon})$（$\epsilon > 0$）且满足**正则条件** $a \cdot f(n/b) \le c \cdot f(n)$（$c < 1$），则 $T(n) = \Theta(f(n))$。

*证明*：

由正则条件，$a^i \cdot f(n/b^i) \le c^i \cdot f(n)$，故：

$$
T(n) \le f(n) \sum_{i=0}^{\infty} c^i = \frac{f(n)}{1-c} = O(f(n))
$$

又 $T(n) \ge f(n)$（首项），故 $T(n) = \Theta(f(n))$。

### 3.3 Akra-Bazzi 方法

对于更一般的非均匀划分递推：

$$
T(x) = g(x) + \sum_{i=1}^{k} a_i T(b_i x + h_i(x))
$$

其中 $a_i > 0$，$0 < b_i < 1$，$|h_i(x)| = O(x / \log^2 x)$。

设 $p$ 满足 $\sum_{i=1}^{k} a_i b_i^p = 1$，则：

$$
T(x) = \Theta\left(x^p \left(1 + \int_{1}^{x} \frac{g(u)}{u^{p+1}} \, du\right)\right)
$$

## 四、示例性代码

### 4.1 Rust：递推关系求解器（主定理自动判断）

```rust
/// Master Theorem 自动判断器
/// 对于递推式 T(n) = a·T(n/b) + Θ(n^d · log^k n)
#[derive(Debug, Clone, PartialEq)]
pub enum MasterCase {
    Case1 { log_b_a: f64 },
    Case2 { log_b_a: f64, k: f64 },
    Case3 { log_b_a: f64 },
    ExtendedCase2Prime { log_b_a: f64, k: f64 },
    NonStandard,
}

impl MasterCase {
    pub fn describe(&self) -> String {
        match self {
            MasterCase::Case1 { log_b_a } => {
                format!("Case 1: f(n) = O(n^{:.3} - ε), T(n) = Θ(n^{:.3})",
                        log_b_a, log_b_a)
            }
            MasterCase::Case2 { log_b_a, .. } => {
                format!("Case 2: f(n) = Θ(n^{:.3}), T(n) = Θ(n^{:.3} · log n)",
                        log_b_a, log_b_a)
            }
            MasterCase::Case3 { log_b_a } => {
                format!("Case 3: f(n) = Ω(n^{:.3} + ε), T(n) = Θ(f(n))",
                        log_b_a)
            }
            MasterCase::ExtendedCase2Prime { log_b_a, k } => {
                format!(
                    "Case 2': f(n) = Θ(n^{:.3} · log^{:.0} n), \
                     T(n) = Θ(n^{:.3} · log^{:.0} n)",
                    log_b_a, k + 1.0, log_b_a, k + 1.0
                )
            }
            MasterCase::NonStandard => "非标准形式，主定理不适用".to_string(),
        }
    }
}

/// 判断主定理情形
pub fn classify(a: f64, b: f64, d: f64, k: f64) -> Result<MasterCase, String> {
    if a < 1.0 { return Err("参数 a 必须 ≥ 1".to_string()); }
    if b <= 1.0 { return Err("参数 b 必须 > 1".to_string()); }

    let log_b_a = a.log(b);
    let eps = 1e-9;

    if (d - log_b_a).abs() < eps {
        if k.abs() < eps {
            Ok(MasterCase::Case2 { log_b_a, k })
        } else {
            Ok(MasterCase::ExtendedCase2Prime { log_b_a, k })
        }
    } else if d < log_b_a - eps {
        Ok(MasterCase::Case1 { log_b_a })
    } else if d > log_b_a + eps {
        Ok(MasterCase::Case3 { log_b_a })
    } else {
        Ok(MasterCase::NonStandard)
    }
}

fn main() {
    let examples = vec![
        ("归并排序", 2.0, 2.0, 1.0, 0.0),
        ("Strassen", 7.0, 2.0, 0.0, 0.0),
        ("整数乘法", 3.0, 2.0, 1.0, 0.0),
        ("T(n)=4T(n/2)+n^2", 4.0, 2.0, 2.0, 0.0),
        ("T(n)=2T(n/2)+n·log n", 2.0, 2.0, 1.0, 1.0),
    ];

    for (name, a, b, d, k) in examples {
        let case = classify(a, b, d, k).unwrap();
        println!("{} → {}", name, case.describe());
    }
}
```

/// 求解线性常系数齐次递推
/// T(n) = c1·T(n-1) + ... + ck·T(n-k)
pub fn linear_recurrence_solution(coeffs: &[f64]) -> Vec<f64> {
    // 返回特征方程的近似根（简化演示）
    // 实际实现需使用数值方法或符号计算
    let k = coeffs.len();
    println!("特征方程: r^{} - ", k);
    for (i, &c) in coeffs.iter().enumerate() {
        print!("{:.2}·r^{} ", c, k - i - 1);
        if i < k - 1 { print!("- "); }
    }
    println!("= 0");
    vec![]
}

## 五、形式化证明

### 5.1 递推解的唯一性定理

**定理 5.1.1**：对于 $k$ 阶线性递推关系，给定 $k$ 个初始条件，其解存在且唯一。

**证明**（数学归纳法）：

- **基础**：$n = 1, 2, \ldots, k$ 时，$T(n)$ 由初始条件唯一确定。
- **归纳假设**：假设 $T(1), \ldots, T(n-1)$ 已唯一确定。
- **归纳步骤**：由递推式 $T(n) = F(n, T(n-1), \ldots, T(n-k))$，右侧所有值均已知，故 $T(n)$ 唯一确定。

### 5.2 分治递推的递归树严格分析

**定理 5.2.1**：对于 $T(n) = aT(n/b) + cn^d$，设 $\alpha = \log_b a$，则：

$$
T(n) = \begin{cases}
\Theta(n^{\alpha}) & \alpha > d \\[6pt]
\Theta(n^{\alpha} \log n) & \alpha = d \\[6pt]
\Theta(n^d) & \alpha < d
\end{cases}
$$

**证明概要**：

递归树总代价：

$$
T(n) = \sum_{i=0}^{\log_b n} a^i \cdot c \left(\frac{n}{b^i}\right)^d
     = c n^d \sum_{i=0}^{\log_b n} \left(\frac{a}{b^d}\right)^i
$$

令 $r = a / b^d = b^{\alpha - d}$：

- 若 $\alpha > d$（$r > 1$）：最后一项主导，$T(n) = \Theta(n^d \cdot r^{\log_b n}) = \Theta(n^{\alpha})$
- 若 $\alpha = d$（$r = 1$）：共 $\log_b n + 1$ 项，每项 $cn^d$，$T(n) = \Theta(n^d \log n) = \Theta(n^{\alpha} \log n)$
- 若 $\alpha < d$（$r < 1$）：几何级数收敛，$T(n) = \Theta(n^d)$

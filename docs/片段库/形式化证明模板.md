### 4. 形式化编码 (可选)

#### 4.1 Coq 证明

```coq
Theorem theorem_name : forall x, P x -> Q x.
Proof.
  (* 证明脚本 *)
Qed.
```

#### 4.2 Lean 证明

```lean
theorem theorem_name : ∀ x, P x → Q x :=
begin
  -- 证明脚本
end
```

#### 4.3 Agda 证明

```agda
theorem-name : ∀ x → P x → Q x
theorem-name x px = -- 证明
```

## 常见证明模式

### 数学归纳法

**结构**:

1. **基础情形**: 证明 $P(0)$ 成立
2. **归纳假设**: 假设 $P(k)$ 成立
3. **归纳步骤**: 证明 $P(k) \Rightarrow P(k+1)$

### 反证法

**结构**:

1. 假设结论不成立
2. 推导出矛盾
3. 因此原命题成立

### 构造性证明

**结构**:

1. 给出具体构造
2. 验证构造满足条件

# 递推关系求解方法对比

## 概述

本文档详细对比分析求解递推关系的三种主要方法：主定理(Master Theorem)、代入法(Substitution Method)和递归树法(Recursion Tree Method)，帮助理解各自的特点和适用场景。

## 主定理 (Master Theorem)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         主定理详解                                           │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  标准形式:                                                                  │
│  ════════════════════════════════════════════════════════════════════════   │
│                                                                             │
│       T(n) = aT(n/b) + f(n)                                                 │
│                                                                             │
│  其中: a ≥ 1, b > 1, f(n) 是渐近正函数                                       │
│                                                                             │
│  关键比较: f(n) vs n^(log_b a)                                              │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  三种情况:                                                                  │
│  ════════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐    │
│  │ Case 1: 叶子主导                                                     │    │
│  │ ────────────────────────────────────────────────────────────────── │    │
│  │ 条件: f(n) = O(n^(log_b a - ε))  对某个 ε > 0                       │    │
│  │                                                                    │    │
│  │ 结果: T(n) = Θ(n^(log_b a))                                        │    │
│  │                                                                    │    │
│  │ 含义: 叶子层的工作量主导总工作量                                    │    │
│  │ 例: T(n) = 4T(n/2) + n  →  T(n) = Θ(n²)                            │    │
│  │     (log_2 4 = 2, n = O(n^(2-ε)))                                  │    │
│  └─────────────────────────────────────────────────────────────────────┘    │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐    │
│  │ Case 2: 均匀分布                                                     │    │
│  │ ────────────────────────────────────────────────────────────────── │    │
│  │ 条件: f(n) = Θ(n^(log_b a))                                        │    │
│  │                                                                    │    │
│  │ 结果: T(n) = Θ(n^(log_b a) log n)                                  │    │
│  │                                                                    │    │
│  │ 含义: 每层工作量大致相等                                            │    │
│  │ 例: T(n) = 2T(n/2) + n  →  T(n) = Θ(n log n)                       │    │
│  │     (归并排序、堆排序的递归形式)                                    │    │
│  └─────────────────────────────────────────────────────────────────────┘    │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐    │
│  │ Case 3: 根节点主导                                                   │    │
│  │ ────────────────────────────────────────────────────────────────── │    │
│  │ 条件: f(n) = Ω(n^(log_b a + ε))  对某个 ε > 0                       │    │
│  │       且 af(n/b) ≤ cf(n) 对某个 c < 1 (正则条件)                    │    │
│  │                                                                    │    │
│  │ 结果: T(n) = Θ(f(n))                                               │    │
│  │                                                                    │    │
│  │ 含义: 根节点的工作量主导总工作量                                    │    │
│  │ 例: T(n) = 3T(n/4) + n  →  T(n) = Θ(n)                             │    │
│  │     (n 大于 n^(log_4 3) ≈ n^0.79)                                  │    │
│  └─────────────────────────────────────────────────────────────────────┘    │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  间隙情况 (Gap Cases):                                                      │
│  ════════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  主定理不适用于以下情况:                                                      │
│  • f(n) 介于 Case 1 和 Case 2 之间，但差距不是多项式级别                    │
│    例: T(n) = 2T(n/2) + n/log n                                             │
│                                                                             │
│  • f(n) 与 n^(log_b a) 振荡                                                 │
│    例: T(n) = 2T(n/2) + n(2 - cos n)                                        │
│                                                                             │
│  • a 不是整数，或子问题大小不是 n/b 的形式                                    │
│                                                                             │
│  这些情况需要用代入法或递归树法。                                             │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 代入法 (Substitution Method)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         代入法详解                                           │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  基本步骤:                                                                  │
│  ════════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  Step 1: 猜测解的形式                                                        │
│  ────────────────────────────────────────────────────────────────────────   │
│  • 基于经验或递归树法猜测 T(n) = O(g(n))                                    │
│  • 猜测形式如: O(n), O(n log n), O(n²), O(2ⁿ) 等                            │
│                                                                             │
│  Step 2: 数学归纳法证明                                                      │
│  ────────────────────────────────────────────────────────────────────────   │
│  • 基础情况: 证明 T(1) = O(g(1))                                            │
│  • 归纳假设: 假设对所有 k < n, T(k) ≤ cg(k)                                  │
│  • 归纳步: 证明 T(n) ≤ cg(n)                                                │
│                                                                             │
│  Step 3: 处理常数                                                            │
│  ────────────────────────────────────────────────────────────────────────   │
│  • 有时需要调整常数 c 或添加低阶项                                          │
│  • 可能需要减去低阶项的技巧 (见下例)                                         │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  经典示例: T(n) = 2T(n/2) + n                                               │
│  ════════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  猜测: T(n) = O(n log n)                                                    │
│                                                                             │
│  证明:                                                                      │
│  ────────────────────────────────────────────────────────────────────────   │
│                                                                             │
│  归纳假设: T(k) ≤ ck log k 对所有 k < n 成立                                │
│                                                                             │
│  归纳步:                                                                    │
│  T(n) = 2T(n/2) + n                                                         │
│      ≤ 2[c(n/2)log(n/2)] + n        (代入归纳假设)                          │
│      = cn(log n - log 2) + n                                              │
│      = cn log n - cn + n                                                  │
│      = cn log n - (c-1)n                                                  │
│                                                                             │
│  要使 T(n) ≤ cn log n，需要 (c-1)n ≥ 0，即 c ≥ 1                           │
│                                                                             │
│  因此，T(n) = O(n log n) 成立 (取 c ≥ 1, n ≥ 2)                            │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  减去低阶项技巧 (Subtraction Trick):                                        │
│  ════════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  当直接猜测导致失败时使用。                                                   │
│                                                                             │
│  例: T(n) = T(n/2) + T(n/4) + n                                             │
│                                                                             │
│  猜测 T(n) = O(n) 会失败，因为无法消去常数。                                  │
│                                                                             │
│  改为猜测 T(n) ≤ cn - d log n:                                              │
│  T(n) ≤ [c(n/2) - d log(n/2)] + [c(n/4) - d log(n/4)] + n                 │
│      = (3c/4 + 1)n - d(log(n/2) + log(n/4))                               │
│      = (3c/4 + 1)n - d(2log n - 3)                                        │
│                                                                             │
│  取 c = 4，则 3c/4 + 1 = 4 = c，满足！                                       │
│                                                                             │
│  因此 T(n) ≤ 4n - d log n = O(n)                                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 递归树法 (Recursion Tree Method)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         递归树法详解                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  基本思想:                                                                  │
│  ════════════════════════════════════════════════════════════════════════   │
│  • 将递归过程展开成树形结构                                                  │
│  • 计算每层的工作量                                                          │
│  • 对所有层的工作量求和                                                      │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  示例: T(n) = 3T(n/4) + n²                                                  │
│  ════════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  ASCII 表示的递归树:                                                         │
│  ────────────────────────────────────────────────────────────────────────   │
│                                                                             │
│                                    n²                                       │
│                                   / | \                                     │
│                                  /  |  \                                    │
│                               (n/4)² (n/4)² (n/4)²                         │
│                               /|\   /|\   /|\                               │
│                              ...  ...  ...                                  │
│                                                                             │
│  详细展开:                                                                  │
│  ────────────────────────────────────────────────────────────────────────   │
│                                                                             │
│  深度    节点数      每节点成本     层总成本                                 │
│  ────────────────────────────────────────────────────────────────────────   │
│   0        1           n²           n²                                       │
│   1        3          (n/4)²        3(n/4)² = (3/16)n²                      │
│   2        9         (n/16)²        9(n/16)² = (3/16)²n²                    │
│   ...     ...          ...          ...                                     │
│   i       3^i      (n/4^i)²        3^i(n/4^i)² = (3/16)^i n²               │
│   ...     ...          ...          ...                                     │
│  log₄n   3^log₄n      Θ(1)          Θ(n^log₄3)                             │
│                                                                             │
│  总成本 = Σ(i=0 to log₄n) (3/16)^i n² + Θ(n^log₄3)                         │
│                                                                             │
│  由于 3/16 < 1，几何级数收敛:                                               │
│  Σ(3/16)^i = 1/(1-3/16) = 16/13 = O(1)                                     │
│                                                                             │
│  因此: T(n) = O(n²)                                                        │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  对比: T(n) = 2T(n/2) + n (归并排序)                                        │
│  ════════════════════════════════════════════════════════════════════════   │
│                                                                             │
│                                    n                                        │
│                                   / \                                       │
│                                n/2  n/2                                     │
│                               / \   / \                                     │
│                            n/4 n/4 n/4 n/4                                  │
│                                                                             │
│  深度    节点数      每节点成本     层总成本                                 │
│  ────────────────────────────────────────────────────────────────────────   │
│   0        1           n            n                                        │
│   1        2          n/2           n                                        │
│   2        4          n/4           n                                        │
│   ...     ...          ...          ...                                     │
│   i       2^i        n/2^i          n                                       │
│  log n    n          Θ(1)           n                                       │
│                                                                             │
│  层数 = log n，每层成本 = n                                                │
│  总成本 = n × log n = O(n log n)                                           │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 基础排序算法对比

| 算法 | 时间(最坏) | 时间(平均) | 时间(最好) | 空间 | 稳定 | 原地 | 适用场景 |
|------|-----------|-----------|-----------|------|------|------|---------|
| **冒泡排序** | O(n²) | O(n²) | O(n) | O(1) | ✓ | ✓ | 教学演示、小数据、接近有序 |
| **选择排序** | O(n²) | O(n²) | O(n²) | O(1) | ✗ | ✓ | 数据小、交换代价高 |
| **插入排序** | O(n²) | O(n²) | O(n) | O(1) | ✓ | ✓ | 小数据、基本有序、在线排序 |
| **希尔排序** | O(n²)~O(n^1.5) | O(n^1.3) | O(n) | O(1) | ✗ | ✓ | 中等数据、内存受限 |

## 高级排序算法对比

| 算法 | 时间(最坏) | 时间(平均) | 时间(最好) | 空间 | 稳定 | 原地 | 适用场景 |
|------|-----------|-----------|-----------|------|------|------|---------|
| **归并排序** | O(n log n) | O(n log n) | O(n log n) | O(n) | ✓ | ✗ | 链表、外部排序、需要稳定 |
| **快速排序** | O(n²) | O(n log n) | O(n log n) | O(log n)~O(n) | ✗ | ✓ | 通用场景、数组、平均最快 |
| **堆排序** | O(n log n) | O(n log n) | O(n log n) | O(1) | ✗ | ✓ | 内存受限、需要保证最坏 |
| **二叉树排序** | O(n²) | O(n log n) | O(n log n) | O(n) | ✓ | ✗ | 需要有序遍历 |

## 特殊排序算法对比

| 算法 | 时间(最坏) | 时间(平均) | 时间(最好) | 空间 | 稳定 | 原地 | 适用场景 |
|------|-----------|-----------|-----------|------|------|------|---------|
| **计数排序** | O(n+k) | O(n+k) | O(n+k) | O(k) | ✓ | ✗ | 范围小整数、数据分布集中 |
| **基数排序** | O(d(n+k)) | O(d(n+k)) | O(d(n+k)) | O(n+k) | ✓ | ✗ | 固定位数整数、字符串 |
| **桶排序** | O(n²) | O(n) | O(n) | O(n) | ✓ | ✗ | 均匀分布数据、浮点数 |
| **TimSort** | O(n log n) | O(n log n) | O(n) | O(n) | ✓ | ✗ | 真实数据、Python/Java默认 |

## 算法选择决策参考

```
数据规模?
    │
    ├── 小 (n < 50)
    │       └── 插入排序
    │
    ├── 中 (50 ≤ n < 1000)
    │       └── 希尔排序 或 快速排序
    │
    └── 大 (n ≥ 1000)
            │
            ├── 需要稳定性?
            │       │
            │       ├── 是 → 归并排序
            │       │
            │       └── 否 → 快速排序
            │
            ├── 内存受限?
            │       └── 堆排序
            │
            └── 数据有特殊性质?
                    │
                    ├── 整数范围小 → 计数/基数排序
                    │
                    ├── 均匀分布 → 桶排序
                    │
                    └── 基本有序 → 插入/TimSort
```

## 实际应用推荐

| 场景 | 推荐算法 | 次选算法 | 避免使用 |
|------|---------|---------|---------|
| 通用数组排序 | 快速排序 | 归并排序 | 冒泡、选择 |
| 链表排序 | 归并排序 | 插入排序 | 快速排序 |
| 内存极度受限 | 堆排序 | 希尔排序 | 归并排序 |
| 需要稳定性 | 归并排序 | TimSort | 快速、堆 |
| 小数据 (<32) | 插入排序 | 选择排序 | 归并、快排 |
| 大量重复键 | 三向快排 | 计数排序 | 普通快排 |
| 浮点数均匀分布 | 桶排序 | 快速排序 | 计数排序 |
| 固定位数整数 | 基数排序 | 计数排序 | 比较排序 |
| 在线/流式数据 | 插入排序 | 堆排序 | 需要全数据的排序 |
| 教学入门 | 冒泡排序 | 选择排序 | - |

*注: k=数据范围, d=数字位数, n=数据量。实际选择需综合考虑数据特征、环境约束和代码维护性。*

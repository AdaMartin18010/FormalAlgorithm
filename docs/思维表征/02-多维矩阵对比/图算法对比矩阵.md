## 最短路径算法对比

| 算法 | 时间复杂度 | 空间复杂度 | 适用图类型 | 边权要求 | 处理负权 | 特点 |
|------|-----------|-----------|-----------|---------|---------|------|
| **BFS** | O(V+E) | O(V) | 无权图 | 无权 | ✗ | 单源、无权最短路径 |
| **Dijkstra(朴素)** | O(V²) | O(V) | 稠密图 | 非负 | ✗ | 实现简单 |
| **Dijkstra(堆优化)** | O((V+E)log V) | O(V) | 稀疏图 | 非负 | ✗ | 实际最常用 |
| **Dijkstra(Fib堆)** | O(V log V + E) | O(V) | 稀疏图 | 非负 | ✗ | 理论最优 |
| **Bellman-Ford** | O(VE) | O(V) | 通用 | 任意 | ✓ | 可检测负环 |
| **SPFA** | O(kE)~O(VE) | O(V) | 稀疏图 | 任意 | ✓ | Bellman-Ford优化 |
| **Floyd-Warshall** | O(V³) | O(V²) | 全源 | 任意 | ✓ | 多源、代码短 |
| **Johnson** | O(V² log V + VE) | O(V²) | 全源稀疏 | 任意 | ✓ | 无负权时高效 |

## 最小生成树算法对比

| 算法 | 时间复杂度 | 空间复杂度 | 适用图类型 | 策略 | 特点 |
|------|-----------|-----------|-----------|------|------|
| **Prim(朴素)** | O(V²) | O(V) | 稠密图 | 加点 | 稠密图高效 |
| **Prim(堆优化)** | O((V+E)log V) | O(V) | 稀疏图 | 加点 | 通用 |
| **Prim(Fib堆)** | O(E + V log V) | O(V) | 稀疏图 | 加点 | 理论最优 |
| **Kruskal** | O(E log E) | O(V) | 稀疏图 | 加边 | 实现简单 |
| **Kruskal+并查集** | O(E α(V)) | O(V) | 通用 | 加边 | 实际最优 |
| **Boruvka** | O(E log V) | O(V) | 分布式 | 多路 | 并行友好 |

## 连通性算法对比

| 算法/结构 | 预处理 | 查询连通性 | 添加边 | 删除边 | 空间 | 特点 |
|-----------|--------|-----------|--------|--------|------|------|
| **DFS/BFS** | O(V+E) | O(1)* | N/A | N/A | O(V) | 一次性查询 |
| **并查集** | O(V) | O(α(V)) | O(α(V)) | 困难 | O(V) | 动态加边 |
| **Tarjan** | O(V+E) | O(1)* | N/A | N/A | O(V) | 强连通分量 |
| **Kosaraju** | O(V+E) | O(1)* | N/A | N/A | O(V) | 强连通分量 |
| **割点/桥** | O(V+E) | O(1)* | N/A | N/A | O(V) | 找关键边/点 |

## 拓扑排序与环检测

| 算法 | 时间复杂度 | 空间复杂度 | 检测环 | 输出方案 | 适用场景 |
|------|-----------|-----------|--------|---------|---------|
| **Kahn(入度法)** | O(V+E) | O(V) | ✓ | 字典序可选 | DAG判定、多解 |
| **DFS后序** | O(V+E) | O(V) | ✓ | 逆后序 | 简单实现 |
| **Tarjan SCC** | O(V+E) | O(V) | ✓ | SCC分解 | 强连通分量 |
| **Floyd判圈** | O(μ+λ) | O(1) | ✓ | 环起点 | 链表/函数图 |

μ=环前节点数, λ=环长

## 图算法选择决策树

```
问题类型?
    │
    ├── 最短路径?
    │       │
    │       ├── 无权图? → BFS
    │       │
    │       ├── 单源?
    │       │       │
    │       │       ├── 非负权? → Dijkstra
    │       │       │
    │       │       └── 可能有负权? → Bellman-Ford / SPFA
    │       │
    │       └── 全源? → Floyd-Warshall (稠密) / Johnson (稀疏)
    │
    ├── 最小生成树?
    │       │
    │       ├── 稠密图? → Prim
    │       │
    │       └── 稀疏图? → Kruskal
    │
    ├── 连通性?
    │       │
    │       ├── 动态加边? → 并查集
    │       │
    │       ├── 强连通分量? → Tarjan / Kosaraju
    │       │
    │       └── 割点/桥? → Tarjan
    │
    ├── 网络流?
    │       │
    │       ├── 二分图匹配? → Dinic / Hopcroft-Karp
    │       │
    │       ├── 带费用? → 最小费用最大流
    │       │
    │       └── 一般最大流? → Dinic / Push-Relabel
    │
    └── 其他?
            │
            ├── 拓扑排序? → Kahn / DFS
            │
            ├── 欧拉回路? → Hierholzer
            │
            └── 匹配? → 看二分图/一般图选择相应算法
```

## 算法组合策略

| 问题场景 | 推荐组合 | 说明 |
|----------|---------|------|
| **带约束的最短路径** | Dijkstra + 状态压缩 | 处理额外约束条件 |
| **时间依赖网络** | 改进Dijkstra | 考虑时间窗口 |
| **多点最短路径** | 多源BFS / Johnson | 避免多次单源计算 |
| **分层图最短路** | 拆点 + Dijkstra | 处理有代价的状态转移 |
| **次短路/K短路** | A* / 可持久化堆 | 寻找非最优解 |
| **Steiner树** | 状态压缩DP / 近似算法 | NP难问题 |

---

*注: V=顶点数, E=边数, α=阿克曼函数反函数(极小常数)。图算法选择应结合图的稀疏性、特殊性质和实际需求。*

# 竞赛算法学习路径


> **版本**: 1.0
> **创建日期**: 2026-04-19
> **最后更新**: 2026-04-19

## 概述

本文档为算法竞赛爱好者提供一条系统的训练路径，从入门到参加高水平竞赛（ICPC、IOI、Codeforces等）。

│  • 1:30-3:30: 攻克中等难度题目                                               │
│  • 3:30-4:30: 尝试难题或检查已解题                                          │
│  • 4:30-5:00: 最后检查 + 提交                                               │

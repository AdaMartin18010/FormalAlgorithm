# 主定理 (Master Theorem) 推导


> **版本**: 1.0
> **创建日期**: 2026-04-19
> **最后更新**: 2026-04-19

## 定理陈述

对于递推关系：

$$T(n) = aT\left(\frac{n}{b}\right) + f(n)$$

其中 $a \geq 1$, $b > 1$, $f(n)$ 是渐近正函数。

**三种情况：**

| 情况 | 条件 | 结果 |
|-----|------|------|
| 1 | $f(n) = O(n^{\log_b a - \epsilon})$, $\epsilon > 0$ | $T(n) = \Theta(n^{\log_b a})$ |
| 2 | $f(n) = \Theta(n^{\log_b a})$ | $T(n) = \Theta(n^{\log_b a} \log n)$ |
| 3 | $f(n) = \Omega(n^{\log_b a + \epsilon})$, $\epsilon > 0$ 且正则条件 | $T(n) = \Theta(f(n))$ |

---

## 推导过程

### 第一步：展开递推式

设 $n = b^k$（即 $k = \log_b n$）

```
T(n) = aT(n/b) + f(n)

     = a[aT(n/b²) + f(n/b)] + f(n)
     = a²T(n/b²) + af(n/b) + f(n)

     = a³T(n/b³) + a²f(n/b²) + af(n/b) + f(n)

     ...

     = aᵏT(n/bᵏ) + Σ(i=0 to k-1) aⁱf(n/bⁱ)

     = aᵏT(1) + Σ(i=0 to k-1) aⁱf(n/bⁱ)
```

由于 $k = \log_b n$，所以 $a^k = a^{\log_b n} = n^{\log_b a}$

因此：

$$T(n) = n^{\log_b a} \cdot T(1) + \sum_{i=0}^{k-1} a^i f\left(\frac{n}{b^i}\right)$$

---

### 第二步：分析求和部分

定义：

$$g(n) = \sum_{i=0}^{k-1} a^i f\left(\frac{n}{b^i}\right)$$

我们需要根据 $f(n)$ 与 $n^{\log_b a}$ 的关系来分析 $g(n)$。

---

### 情况1：$f(n) = O(n^{\log_b a - \epsilon})$

假设 $f(n) \leq c \cdot n^{\log_b a - \epsilon}$

```
g(n) = Σ(i=0 to k-1) aⁱf(n/bⁱ)

     ≤ Σ(i=0 to k-1) aⁱ · c · (n/bⁱ)^(log_b a - ε)

     = c·n^(log_b a - ε) · Σ(i=0 to k-1) aⁱ/(bⁱ)^(log_b a - ε)

     = c·n^(log_b a - ε) · Σ(i=0 to k-1) aⁱ/(aⁱ · b^(-iε))    [因为 b^(log_b a) = a]

     = c·n^(log_b a - ε) · Σ(i=0 to k-1) (b^ε)ⁱ

     = c·n^(log_b a - ε) · [(b^ε)^k - 1]/(b^ε - 1)

     = c·n^(log_b a - ε) · [n^ε - 1]/(b^ε - 1)                [因为 b^(kε) = n^ε]

     = O(n^(log_b a - ε) · n^ε)

     = O(n^(log_b a))
```

因此：

$$T(n) = n^{\log_b a} \cdot T(1) + O(n^{\log_b a}) = \Theta(n^{\log_b a})$$

---

### 情况2：$f(n) = \Theta(n^{\log_b a})$

```
g(n) = Σ(i=0 to k-1) aⁱf(n/bⁱ)

     = Θ(Σ(i=0 to k-1) aⁱ · (n/bⁱ)^(log_b a))

     = Θ(n^(log_b a) · Σ(i=0 to k-1) aⁱ/aⁱ)

     = Θ(n^(log_b a) · Σ(i=0 to k-1) 1)

     = Θ(n^(log_b a) · k)

     = Θ(n^(log_b a) · log n)      [因为 k = log_b n = Θ(log n)]
```

因此：

$$T(n) = n^{\log_b a} \cdot T(1) + \Theta(n^{\log_b a} \log n) = \Theta(n^{\log_b a} \log n)$$

---

### 情况3：$f(n) = \Omega(n^{\log_b a + \epsilon})$ 且满足正则条件

**正则条件**：存在 $c < 1$ 使得对所有充分大的 $n$，有：

$$af\left(\frac{n}{b}\right) \leq cf(n)$$

这个条件保证了 $f(n)$ 在递归下降过程中增长足够快。

**推导：**

首先，由正则条件，通过归纳可得：

$$a^i f\left(\frac{n}{b^i}\right) \leq c^i f(n)$$

因此：

```
g(n) = Σ(i=0 to k-1) aⁱf(n/bⁱ)

     ≤ Σ(i=0 to k-1) cⁱf(n)

     = f(n) · Σ(i=0 to k-1) cⁱ

     = f(n) · (1 - c^k)/(1 - c)

     ≤ f(n) · 1/(1 - c)          [因为 c < 1, 所以 1-c^k < 1]

     = O(f(n))
```

同时，显然 $g(n) \geq f(n)$（第一项），所以 $g(n) = \Omega(f(n))$

因此：

$$g(n) = \Theta(f(n))$$

最终：

$$T(n) = n^{\log_b a} \cdot T(1) + \Theta(f(n))$$

由于 $f(n) = \Omega(n^{\log_b a + \epsilon})$，所以 $f(n)$ 主导 $n^{\log_b a}$：

$$T(n) = \Theta(f(n))$$

---

## 直观理解

```
递归树视角:
─────────────────────────────────────────

                    f(n)          ← 第0层: f(n)
                   /    \
                  /      \
            f(n/b)       f(n/b)    ← 第1层: a·f(n/b)
            / \            / \
           /   \          /   \
       f(n/b²) ...    f(n/b²)   ← 第2层: a²·f(n/b²)

       ...                        ← ...

       T(1) T(1) ... T(1)        ← 叶子层: a^k·T(1)

三种情况对应:
─────────────────────────────────────────
情况1: 叶子层主导 → 工作量集中在底层
       总工作量: Θ(a^k) = Θ(n^(log_b a))

情况2: 每层工作量相同 → 均匀分布
       总工作量: Θ(f(n)·k) = Θ(n^(log_b a)·log n)

情况3: 根层主导 → 工作量集中在顶层
       总工作量: Θ(f(n))
```

## 应用示例

| 递推式 | a | b | log_b(a) | f(n) | 情况 | 结果 |
|--------|---|---|----------|------|-----|------|
| T(n)=2T(n/2)+O(n) | 2 | 2 | 1 | O(n¹) | 2 | Θ(n log n) |
| T(n)=2T(n/2)+O(1) | 2 | 2 | 1 | O(n⁰) | 1 | Θ(n) |
| T(n)=2T(n/2)+O(n²) | 2 | 2 | 1 | Ω(n¹⁺¹) | 3 | Θ(n²) |
| T(n)=3T(n/2)+O(n) | 3 | 2 | log₂3≈1.58 | O(n¹) | 1 | Θ(n^log₂3) |
| T(n)=4T(n/2)+O(n²) | 4 | 2 | 2 | Θ(n²) | 2 | Θ(n² log n) |

## 主定理的局限性

主定理**不适用**的情况：

1. **间隙情况**：$f(n)$ 与 $n^{\log_b a}$ 之间没有多项式差距
   - 例：$T(n) = 2T(n/2) + \frac{n}{\log n}$

2. **非多项式差距**：正则条件不满足
   - 例：$f(n)$ 震荡太大

3. **a 不是常数**：如 $T(n) = nT(n/2) + n$

4. **b 不是常数**：如 $T(n) = T(n/2) + T(n/3) + n$

5. **非等分**：$T(n) = T(n/3) + T(2n/3) + n$

这些情况需要使用**递归树法**或**代入法**等其他方法。

*主定理是分析分治算法复杂度的强大工具，但需注意其适用条件*

## 主决策树

```
┌─────────────────────────────────────────────────────────────────────┐
│                    算法选择决策树                                    │
└─────────────────────────────────────────────────────────────────────┘

问题类型是什么?
    │
    ├──► 搜索问题?
    │       │
    │       ├──► 数据已排序?
    │       │       ├──► 是 → 二分查找 O(log n)
    │       │       └──► 否 → 继续判断...
    │       │
    │       ├──► 需要字符串匹配?
    │       │       ├──► 单模式 → KMP / Boyer-Moore
    │       │       └──► 多模式 → AC自动机
    │       │
    │       └──► 通用搜索 → 哈希表 O(1) 或 线性搜索 O(n)
    │
    ├──► 排序问题? (参见排序算法对比矩阵)
    │       │
    │       ├──► 数据规模 < 50? → 插入排序
    │       ├──► 需要稳定性? → 归并排序
    │       ├──► 内存受限? → 堆排序
    │       └──► 通用情况 → 快速排序
    │
    ├──► 图问题? (参见图算法对比矩阵)
    │       │
    │       ├──► 最短路径?
    │       │       ├──► 无权图? → BFS
    │       │       ├──► 非负权? → Dijkstra
    │       │       └──► 有负权? → Bellman-Ford
    │       │
    │       ├──► 最小生成树?
    │       │       ├──► 稠密图? → Prim
    │       │       └──► 稀疏图? → Kruskal
    │       │
    │       └──► 连通性? → 并查集 / Tarjan
    │
    ├──► 优化问题?
    │       │
    │       ├──► 最优子结构 + 重叠子问题?
    │       │       └──► 是 → 动态规划
    │       │
    │       ├──► 贪心选择性质?
    │       │       └──► 是 → 贪心算法
    │       │
    │       └──► 需要遍历解空间?
    │               ├──► 求任意解 → 回溯法
    │               └──► 求最优解 → 分支限界
    │
    └──► 其他问题?
            │
            ├──► 需要随机化? → 蒙特卡罗 / 拉斯维加斯
            ├──► 并行处理? → 分治 / MapReduce
            └──► 在线算法? → 贪心 / 竞争分析
```

## 数据规模决策

```
数据规模 n 是多少?
    │
    ├──► n < 50 (极小)
    │       ├──► 插入排序 - 简单高效
    │       ├──► 线性搜索 - 无需预处理
    │       └──► 暴力枚举 - 解空间小
    │
    ├──► 50 ≤ n < 1000 (小)
    │       ├──► 希尔排序
    │       ├──► 简单DP
    │       └──► Floyd-Warshall (图)
    │
    ├──► 1000 ≤ n < 10^5 (中)
    │       ├──► 快速排序 / 归并排序
    │       ├──► 线段树 / 树状数组
    │       ├──► Dijkstra + 堆
    │       └──► 常规DP
    │
    ├──► 10^5 ≤ n < 10^7 (大)
    │       ├──► 计数/基数排序 (整数)
    │       ├──► 并查集
    │       ├──► 高效图算法
    │       └──► 状态压缩优化
    │
    └──► n ≥ 10^7 (极大) 或 内存限制严格
            ├──► 外部排序
            ├──► 流式算法
            ├──► 近似算法
            └──► 并行/分布式算法
```

```
    ├──► 随机访问 O(1)?

    │
    ├──► 频繁插入删除?
    │       │
    │       ├──► 两端操作?
    │       │       ├──► 同端 → 栈
    │       │       └──► 异端 → 队列 / 双端队列
    │       │
    │       └──► 中间任意位置?
    │               ├──► 需要搜索? → 平衡树 / 跳表
    │               └──► 不需要搜索 → 链表
    │
    ├──► 需要快速查找?
    │       │
    │       ├──► 精确查找 O(1)? → 哈希表
    │       │
    │       ├──► 有序遍历 / 范围查询? → 平衡树 (RB/AVL)
    │       │
    │       └──► 前缀匹配? → Trie树
    │
    ├──► 需要优先处理?
    │       └──► 堆 / 优先队列
    │
    └──► 需要表示关系?
            │
            ├──► 一对一 → 映射 / 字典
            ├──► 一对多 → 树
            └──► 多对多 → 图
```

## 算法范式选择

```
问题特征?
    │
    ├──► 可以分解为相似子问题?
    │       │
    │       ├──► 子问题重叠?
    │       │       ├──► 是 → 动态规划 (记忆化搜索/填表)
    │       │       └──► 否 → 分治法 (归并排序、二分查找)
    │       │
    │       └──► 需要合并子问题解?
    │               ├──► 简单合并 → 分治
    │               └──► 复杂合并 → 分治+优化
    │
    ├──► 局部最优选择能导致全局最优?
    │       │
    │       ├──► 是 (贪心选择性质) → 贪心算法
    │       │       ├──► 活动选择
    │       │       ├──► 霍夫曼编码
    │       │       └──► 最小生成树 (Prim/Kruskal)
    │       │
    │       └──► 否 → 可能需要DP或穷举
    │
    ├──► 解空间可以系统遍历?
    │       │
    │       ├──► 需要所有解? → 回溯法 + 剪枝
    │       │
    │       ├──► 需要最优解? → 分支限界
    │       │
    │       └──► 满足即可? → 启发式搜索
    │               ├──► A*算法
    │               └──► 遗传算法 / 模拟退火
    │
    └──► 需要处理最坏情况?
            │
            ├──► 随机化 → 随机快速排序、随机选择
            │
            └──► 确定性 → 平衡数据结构
```

## 复杂度约束决策

```
约束条件?
    │
    ├──► 时间限制严格?
    │       │
    │       ├──► O(1) 需求? → 哈希、数组索引
    │       ├──► O(log n) 需求? → 二分、平衡树
    │       ├──► O(n) 需求? → 线性扫描、双指针
    │       └──► O(n log n) 可接受? → 排序、分治
    │
    ├──► 空间限制严格?
    │       │
    │       ├──► O(1) 额外空间? → 原地算法
    │       │       ├──► 快速排序
    │       │       ├──► 堆排序
    │       │       └──► Floyd判圈
    │       │
    │       └──► 允许 O(log n)? → 递归栈空间
    │
    └──► 预处理时间可用?
            │
            ├──► 是 → 预处理 + 快速查询
            │       ├──► 排序 + 二分
            │       ├──► 建图 + 预处理
            │       └──► 构建索引
            │
            └──► 否 → 在线算法、流式处理
```

## 图算法选择详细决策

```
图的特征?
    │
    ├──► 图类型?
    │       ├──► 无向图 → 相应算法简化
    │       └──► 有向图 → 考虑方向性
    │
    ├──► 边权?
    │       ├──► 无权 → BFS最短路
    │       ├──► 非负权 → Dijkstra
    │       └──► 有负权 → Bellman-Ford / SPFA
    │
    ├──► 图密度?
    │       ├──► 稀疏 (E ≈ V) → 邻接表 + 堆优化
    │       └──► 稠密 (E ≈ V²) → 邻接矩阵
    │
    └──► 问题类型?
            │
            ├──► 单源最短路 → Dijkstra / Bellman-Ford
            ├──► 全源最短路 → Floyd / Johnson
            ├──► 最小生成树 → Prim / Kruskal
            ├──► 连通性 → DFS / 并查集 / Tarjan
            ├──► 拓扑排序 → Kahn / DFS
            ├──► 网络流 → Dinic / ISAP
            └──► 匹配 → 匈牙利 / Hopcroft-Karp
```

## 代码模板速查

### 二分查找模板

```
条件: 有序数组，查找满足条件的边界

左闭右闭 [left, right]:
    while left <= right:
        mid = left + (right - left) / 2
        if check(mid): left = mid + 1
        else: right = mid - 1

左闭右开 [left, right):
    while left < right:
        mid = left + (right - left) / 2
        if check(mid): left = mid + 1
        else: right = mid
```

### 双指针模板

```
条件: 有序数组、滑动窗口

快慢指针:
    slow = fast = 0
    while fast < n:
        if condition: update slow
        fast++

对撞指针:
    left = 0, right = n-1
    while left < right:
        if condition: left++
        else: right--
```

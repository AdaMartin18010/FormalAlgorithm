# 证明方法选择决策树


> **版本**: 1.0
> **创建日期**: 2026-04-19
> **最后更新**: 2026-04-19

## 主决策树

```
要证明的命题类型?
    │
    ├──► 全称命题 (对所有x成立)?
    │       │
    │       ├──► x 是自然数 / 整数?
    │       │       │
    │       │       ├──► 可以递推? → 数学归纳法
    │       │       │       ├──► 基础步直接? → 简单归纳
    │       │       │       └──► 需要依赖多个? → 强归纳
    │       │       │
    │       │       └──► 涉及递归结构? → 结构归纳法
    │       │               ├──► 树结构 → 树归纳
    │       │               ├──► 列表 → 列表归纳
    │       │               └──► 一般归纳类型 → 结构归纳
    │       │
    │       └──► x 是其他集合? → 良基归纳
    │
    ├──► 存在性命题 (存在x使P成立)?
    │       │
    │       ├──► 能构造出具体的x? → 构造性证明
    │       │       ├──► 给出算法 → 算法构造
    │       │       └──► 给出公式 → 显式构造
    │       │
    │       └──► 难以直接构造? → 非构造性证明
    │               ├──► 反证法
    │               └──► 鸽巢原理 / 概率方法
    │
    ├──► 蕴含命题 (若P则Q)?
    │       │
    │       ├──► P的信息充分且易于推进? → 直接证明
    │       │       └── 假设P成立 → 推导Q成立
    │       │
    │       ├──► 非Q比Q更容易处理? → 逆否证明
    │       │       └── 证明: 若非Q则非P
    │       │
    │       └──► 需要利用矛盾的威力? → 反证法
    │               └── 假设P且非Q → 导出矛盾
    │
    ├──► 否定命题 (非P)?
    │       │
    │       ├──► P是存在性? → 证明对所有x非P(x)
    │       │
    │       └──► P是普遍性? → 反例证明
    │               └── 找一个具体例子使P不成立
    │
    └──► 等价命题 (P当且仅当Q)?
            ├──► 分别证明 P→Q 和 Q→P
            └──► 或者证明 P→Q 和 非P→非Q
```

## 数学归纳法细分

```
需要对什么进行归纳?
    │
    ├──► 自然数上的性质?
    │       │
    │       ├──► 只需要前一个? → 简单归纳
    │       │       模板:
    │       │       1. 证明P(0)或P(1)
    │       │       2. 假设P(k)，证明P(k+1)
    │       │
    │       ├──► 可能需要任意前面多个? → 强归纳/完全归纳
    │       │       模板:
    │       │       1. 证明基础情况
    │       │       2. 假设对所有m<k, P(m)成立
    │       │       3. 证明P(k)
    │       │
    │       └──► 多个参数? → 双重归纳 / 字典序归纳
    │
    ├──► 数据结构上的性质?
    │       │
    │       ├──► 树结构 → 树归纳
    │       │       ├──► 基础: 叶子节点性质
    │       │       └──► 归纳: 子树→整树
    │       │
    │       ├──► 列表/序列 → 列表归纳
    │       │       ├──► 基础: 空列表
    │       │       └──► 归纳: cons(x, xs)
    │       │
    │       └──► 归纳定义类型 → 结构归纳
    │               按类型的构造子进行归纳
    │
    └──► 偏序集上的性质? → 良基归纳
            利用偏序的良基性，无无限下降链
```

## 反证法使用场景

```
考虑使用反证法当:
    │
    ├──► 命题以"不存在"、"没有"、"不可能"形式出现
    │       例: 证明√2不是有理数
    │
    ├──► 结论信息较少，假设结论不成立信息更多
    │       例: 证明素数无穷多
    │
    ├──► 要证明某种对象不存在
    │       例: 证明停机问题不可解
    │
    ├──► 需要证明唯一性
    │       例: 证明方程解唯一
    │
    └──► 其他方法尝试失败

反证法执行步骤:
    1. 假设待证命题不成立
    2. 从这个假设出发进行逻辑推导
    3. 得到一个已知为假或自相矛盾的结论
    4. 因此原假设错误，原命题成立
```

## 构造性vs非构造性证明

```
证明存在性命题时:
    │
    ├──► 构造性证明 (优先选择)
    │       ├──► 优点: 提供具体算法/对象
    │       ├──► 适用: 算法分析、计算机科学
    │       └──► 方法:
    │               - 显式公式
    │               - 递归构造
    │               - 迭代算法
    │
    └──► 非构造性证明
            ├──► 优点: 有时更简单
            ├──► 适用: 纯存在性证明
            └──► 方法:
                    - 反证法
                    - 鸽巢原理
                    - 概率方法
                    - 紧致性原理
```

## 组合证明技术

```
组合数学问题?
    │
    ├──► 计数问题?
    │       ├──► 需要精确计数? → 双射 / 容斥原理
    │       ├──► 估计上下界? → 组合估计 / 概率方法
    │       └──► 证明恒等式? → 组合解释 / 双重计数
    │
    ├──► 存在性问题?
    │       ├──► 构造一个? → 显式构造
    │       └──► 证明必然存在? → 鸽巢原理 / 概率方法
    │
    └──► 极值问题?
            ├──► 找最大值/最小值? → 极值组合方法
            └──► 证明某性质必然出现? → Ramsey理论
```

## 算法正确性证明

```
证明算法正确性:
    │
    ├──► 终止性 (Termination)
    │       ├──► 找到递减的良基度量
    │       │       - 递归: 参数递减
    │       │       - 循环: 循环变量变化
    │       ├──► 势函数方法
    │       └──► 反证: 假设不终止导出矛盾
    │
    ├──► 部分正确性 (Partial Correctness)
    │       │
    │       ├──► 循环使用不变式
    │       │       1. 初始化: 循环前不变式成立
    │       │       2. 保持: 每次迭代保持不变式
    │       │       3. 终止: 循环结束时，不变式+终止条件⇒结论
    │       │
    │       └──► 递归使用归纳
    │               - 对递归深度归纳
    │               - 对数据结构归纳
    │
    └──► 完全正确性 = 终止性 + 部分正确性
```

## 复杂度下界证明

```
证明问题下界:
    │
    ├──► 信息论下界
    │       └── 决策树模型
    │           例: 比较排序Ω(n log n)
    │
    ├──► 归约方法
    │       └── 从已知下界问题归约
    │           例: 3-SAT归约证明NP困难
    │
    ├──► 对手论证
    │       └── 构造使任何算法都慢的输入
    │           例: 搜索无序数组Ω(n)
    │
    └──► 代数决策树
            利用问题的代数结构
```

## 类型安全证明

```
证明类型系统安全:
    │
    ├──► 进展性 (Progress)
    │       证明: 良类型项要么是值，要么可以求值
    │       方法: 对类型推导进行结构归纳
    │
    ├──► 保持性 (Preservation / Subject Reduction)
    │       证明: 求值保持类型
    │       方法: 对求值规则归纳
    │
    └──► 安全性 = 进展性 + 保持性
            保证: 良类型程序不会卡住(除非已是值)
```

## 证明策略速查表

| 问题特征 | 推荐方法 | 替代方法 |
|----------|---------|---------|
| 自然数性质 | 数学归纳法 | 良基归纳 |
| 递归数据结构 | 结构归纳法 | 双重归纳 |
| 存在性+可构造 | 构造性证明 | - |
| 存在性+难构造 | 反证法 / 概率方法 | 紧致性 |
| 否定/不可能 | 反证法 | - |
| 蕴含+前提强 | 直接证明 | 条件证明 |
| 蕴含+结论弱 | 逆否证明 | - |
| 等价 | 双向证明 | 循环蕴含 |
| 唯一性 | 反证法+假设两个 | 构造+证明相等 |
| 计数恒等式 | 双重计数 | 生成函数 |
| 算法终止 | 势函数 | 良基关系 |
| 算法正确 | 循环不变式 | 归纳法 |

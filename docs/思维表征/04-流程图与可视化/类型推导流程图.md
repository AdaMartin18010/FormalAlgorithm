## Hindley-Milner类型推导

```
┌─────────────────┐
│    开始推导     │
│   表达式 e      │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 第1步: 初始化   │
│ - 为e分配类型变量│
│ - 创建空约束集 C│
│ - 创建空代换 S  │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 第2步: 遍历AST  │
│ 对e的每个子表达式│
│ 递归生成约束    │
└────────┬────────┘
         │
         ▼
     ┌───────┐◄─────────────────────────┐
     │还有更多 │                           │
     │子表达式?│                          │
     └───┬───┘                           │
         │否                             │
         ▼                                │
┌─────────────────┐                       │
│ 转到第3步: 统一化│                       │
└─────────────────┘                       │
         是                               │
         ▼                                │
┌─────────────────┐                       │
│ 根据语法节点类型│                       │
│ 生成约束:       │                       │
│                 │                       │
│ - 变量: 查环境  │                       │
│ - 应用: t₁=t₂→t₃│                       │
│ - 抽象: 新类型  │                       │
│ - let: 泛化类型 │                       │
└────────┬────────┘                       │
         │                                │
         └────────────────────────────────►┘
         │
         ▼
┌─────────────────┐
│ 第3步: 统一化   │
│ 求解约束集 C    │
│ 找最一般合一    │
│ (MGU)           │
└────────┬────────┘
         │
         ▼
     ┌───────┐
     │约束可解?│
     └───┬───┘
         │否
         ▼
┌─────────────────┐
│ 类型错误!       │
│ - 冲突类型      │
│ - 无限类型      │
│ (occurs check)  │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  报告错误信息   │
└─────────────────┘
         是
         ▼
┌─────────────────┐
│ 第4步: 应用代换 │
│ 将MGU代换应用到 │
│ 表达式e的类型变量│
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 第5步: 泛化     │
│ (let绑定处)     │
│ 将自由类型变量  │
│ 泛化为∀量词     │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│    推导完成     │
│   得到类型 τ    │
└─────────────────┘
```

## 约束生成规则

```
变量:
┌─────────────────────────────────┐
│ Γ(x) = σ                        │
│ σ = ∀α₁...αₙ.τ                  │
│ β₁...βₙ 是新变量                 │
│─────────────────────────────────│
│ Γ ⊢ x : [βᵢ/αᵢ]τ  ▹  {}         │
└─────────────────────────────────┘

λ抽象:
┌─────────────────────────────────┐
│ β 是新类型变量                    │
│ Γ, x:β ⊢ e : τ  ▹  C            │
│─────────────────────────────────│
│ Γ ⊢ λx.e : β→τ  ▹  C            │
└─────────────────────────────────┘

应用:
┌─────────────────────────────────┐
│ β 是新类型变量                    │
│ Γ ⊢ e₁ : τ₁  ▹  C₁              │
│ Γ ⊢ e₂ : τ₂  ▹  C₂              │
│─────────────────────────────────│
│ Γ ⊢ e₁ e₂ : β  ▹  C₁∪C₂∪{τ₁=τ₂→β}│
└─────────────────────────────────┘

let绑定:
┌─────────────────────────────────┐
│ Γ ⊢ e₁ : τ₁  ▹  C₁              │
│ σ = Gen(Γ, solve(C₁, τ₁))       │
│ Γ, x:σ ⊢ e₂ : τ₂  ▹  C₂         │
│─────────────────────────────────│
│ Γ ⊢ let x=e₁ in e₂ : τ₂  ▹  C₂  │
└─────────────────────────────────┘
```

## 统一化算法流程

```
┌─────────────────┐
│ unify(C)        │
│ 统一化约束集    │
└────────┬────────┘
         │
         ▼
     ┌───────┐
     │ C = ∅? │
     └───┬───┘
         │是
         ▼
    ┌─────────┐
    │ 返回空  │
    │ 代换 [] │
    └─────────┘
         否
         ▼
┌─────────────────┐
│ 从C中取一个约束 │
│ (s = t)         │
│ C' = C \ {s=t}  │
└────────┬────────┘
         │
         ▼
     ┌───────┐
     │ s = t? │
     └───┬───┘
         │是
         │
         └────────► 继续处理C'
         否
         ▼
     ┌───────┐
     │ s是变量?│
     └───┬───┘
         │是
         ▼
┌─────────────────┐
│ occurs(s, t)?   │
│ (出现检查)      │
└────────┬────────┘
         │是
         ▼
┌─────────────────┐
│ 错误: 无限类型  │
│ (类型递归)      │
└─────────────────┘
         否
         ▼
┌─────────────────┐
│ S = [t/s]       │
│ 返回 S ∪ unify(S(C'))│
└─────────────────┘
         否(s不是变量)
         ▼
     ┌───────┐
     │ t是变量?│
     └───┬───┘
         │是
         ▼
┌─────────────────┐
│ S = [s/t]       │
│ 返回 S ∪ unify(S(C'))│
└─────────────────┘
         否(t不是变量)
         ▼
     ┌───────┐
     │s=t₁→t₂?│
     └───┬───┘
         │是
         ▼
┌─────────────────┐
│ s = s₁→s₂       │
│ C'' = C' ∪ {s₁=t₁, s₂=t₂}│
│ 返回 unify(C'') │
└─────────────────┘
         否
         ▼
┌─────────────────┐
│ 错误: 类型冲突  │
│ (unification失败)│
└─────────────────┘
```

## 类型检查流程 (双向类型检查)

```
┌─────────────────────────────────────────────────┐
│            双向类型检查流程                      │
└─────────────────────────────────────────────────┘

                    ┌─────────────┐
                    │ 表达式 e    │
                    └──────┬──────┘
                           │
            ┌──────────────┼──────────────┐
            │              │              │
            ▼              ▼              ▼
    ┌──────────────┐ ┌──────────┐ ┌──────────────┐
    │ 合成模式     │ │ 检查模式 │ │ 应用模式     │
    │ (synthesis)  │ │ (checking)│ │ (application)│
    │ Γ ⊢ e ⇒ τ    │ │ Γ ⊢ e ⇐ τ │ │ Γ ⊢ e • e ⇓ τ│
    └──────────────┘ └──────────┘ └──────────────┘

合成模式 (从表达式推导类型):
─────────────────────────────────────────────
┌─────────────────────────────────────────┐
│ 变量:                                   │
│ Γ(x) = τ                               │
│──────────────                          │
│ Γ ⊢ x ⇒ τ                               │
└─────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────┐
│ 应用:                                   │
│ Γ ⊢ e₁ ⇒ τ₁    Γ ⊢ τ₁ • e₂ ⇓ τ         │
│────────────────────────────────         │
│ Γ ⊢ e₁ e₂ ⇒ τ                           │
└─────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────┐
│ 类型注释:                               │
│ Γ ⊢ e ⇐ τ                               │
│──────────────                          │
│ Γ ⊢ (e : τ) ⇒ τ                         │
└─────────────────────────────────────────┘

检查模式 (验证表达式符合类型):
─────────────────────────────────────────────
┌─────────────────────────────────────────┐
│ λ抽象:                                  │
│ Γ, x:τ₁ ⊢ e ⇐ τ₂                       │
│────────────────                          │
│ Γ ⊢ λx.e ⇐ τ₁→τ₂                        │
└─────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────┐
│ 切换到合成:                             │
│ Γ ⊢ e ⇒ τ'    τ' = τ                   │
│────────────────────                      │
│ Γ ⊢ e ⇐ τ                               │
└─────────────────────────────────────────┘

应用模式 (函数应用特殊处理):
─────────────────────────────────────────────
┌─────────────────────────────────────────┐
│ 函数类型:                               │
│ Γ ⊢ e ⇐ τ₁    τ = τ₂                   │
│────────────────────                      │
│ Γ ⊢ τ₁→τ₂ • e ⇓ τ₂                      │
└─────────────────────────────────────────┘
```

## 多态实例化流程

```
┌─────────────────┐
│ 实例化多态类型  │
│ instantiate(σ)  │
│ σ = ∀α₁...αₙ.τ  │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 创建新类型变量  │
│ β₁, ..., βₙ    │
│ (fresh)         │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 应用代换        │
│ [β₁/α₁, ...,    │
│  βₙ/αₙ]τ        │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 返回实例化后的  │
│ 单态类型 τ'     │
└─────────────────┘
```

## 类型推导错误处理

```
错误类型:
─────────────────────────────────────────────────

1. 无限类型 (Occurs Check)
─────────────────────────
错误信息: "Occurs check failed"
含义: 试图构造无限类型，如 α = α → β
示例: λx. x x
修复: 检查递归定义，添加类型注释

2. 类型不匹配 (Unification Failed)
─────────────────────────────────
错误信息: "Couldn't match type X with Y"
含义: 两个不兼容的类型被要求统一
示例: if True then 1 else 'a'
修复: 确保分支类型一致

3. 未定义变量 (Unbound Variable)
───────────────────────────────
错误信息: "Not in scope: x"
含义: 使用了未定义的变量
修复: 检查变量名拼写，确保已定义

4. 刚性类型变量 (Rigid Type Variable)
────────────────────────────────────
错误信息: "Cannot match rigid type variable"
含义: 多态类型被不正确地实例化
修复: 使用显式类型注释
```

## 类型推导优化策略

```
性能优化:
─────────────────────────────────────────────────

1. 约束延迟求解
   - 延迟统一化直到必要
   - 批量处理约束

2. 增量式推导
   - 缓存子表达式的推导结果
   - 只重新推导变化部分

3. 层级类型系统
   - 使用等级系统避免过度多态
   - 限制let绑定深度

4. 启发式排序
   - 优先处理能立即确定的约束
   - 延迟涉及多态的约束
```

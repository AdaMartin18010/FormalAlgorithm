### ASCII流程图

```
┌─────────────────┐
│     开始       │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 初始化:         │
│ - dist[s] = 0   │
│ - dist[v] = ∞   │
│ - 优先队列Q     │
└────────┬────────┘
         │
         ▼
     ┌───────┐
     │ Q非空? │◄───────────────────────┐
     └───┬───┘                        │
         │否                          │
         ▼                            │
┌─────────────────┐                   │
│    返回dist[]   │                   │
└────────┬────────┘                   │
         │                            │
         ▼                            │
    ┌─────────┐                       │
    │   结束   │                       │
    └─────────┘                       │
         是                           │
         ▼                            │
┌─────────────────┐                   │
│ u = Q.extractMin│                   │
└────────┬────────┘                   │
         │                            │
         ▼                            │
┌─────────────────┐                   │
│ 对u的每个邻居v: │                   │
│ alt = dist[u]   │                   │
│     + w(u,v)    │                   │
└────────┬────────┘                   │
         │                            │
         ▼                            │
     ┌───────┐                        │
     │alt<dist[v]?│                   │
     └───┬───┘                        │
         │否                          │
         │                            │
         └────────────────────────────┘
         是
         ▼
┌─────────────────┐
│ dist[v] = alt   │
│ Q.decreaseKey(v)│
└────────┬────────┘
         │
         └────────────────────────────►┘
```

### Mermaid流程图

```mermaid
flowchart TD
    Start([开始]) --> Init[初始化:<br/>dist[s] = 0<br/>dist[v] = ∞<br/>优先队列Q]
    Init --> Loop{Q非空?}
    Loop -->|否| Return[返回dist数组]
    Return --> End([结束])
    Loop -->|是| Extract[u = Q.extractMin]
    Extract --> Iterate[对u的每个邻居v:<br/>alt = dist[u] + w]
    Iterate --> Check{alt < dist[v]?}
    Check -->|否| Loop
    Check -->|是| Update[dist[v] = alt<br/>Q.decreaseKey]
    Update --> Loop
```

## 快速排序

```
┌─────────────────┐
│   quicksort(A,l,r)│
└────────┬────────┘
         │
         ▼
     ┌───────┐
     │ l < r? │
     └───┬───┘
         │否
         ▼
    ┌─────────┐
    │  返回    │
    └─────────┘
         是
         ▼
┌─────────────────┐
│ p = partition   │
│ (A, l, r)       │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ quicksort(A,l,p-1)│
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ quicksort(A,p+1,r)│
└────────┬────────┘
         │
         ▼
    ┌─────────┐
    │  返回    │
    └─────────┘

partition过程:
┌─────────────────┐
│ pivot = A[r]    │
│ i = l - 1       │
└────────┬────────┘
         │
         ▼
    ┌───────┐◄────────────────────────┐
    │j从l到r-1│                        │
    └───┬───┘                          │
        │结束                          │
        ▼                              │
   ┌─────────┐                         │
   │交换A[i+1]│                         │
   │和A[r]   │                         │
   └───┬─────┘                         │
       │                               │
       ▼                               │
  ┌──────────┐                         │
  │返回i+1   │                         │
  └──────────┘                         │
       │A[j]≤pivot?                    │
       ▼是                              │
  ┌──────────┐                         │
  │ i = i + 1 │                         │
  │ 交换A[i]  │                         │
  │ 和A[j]   │                         │
  └──────────┘────────────────────────►┘
```

## 二分查找

```
┌─────────────────┐
│  binarySearch   │
│  (A, target)    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ left = 0        │
│ right = n-1     │
└────────┬────────┘
         │
         ▼
     ┌───────┐◄─────────────────────────┐
     │left <=│                           │
     │ right?│                           │
     └───┬───┘                           │
         │否                             │
         ▼                                │
┌─────────────────┐                       │
│   返回-1       │                       │
│ (未找到)       │                       │
└────────┬────────┘                       │
         │                               │
         ▼                               │
    ┌─────────┐                          │
    │   结束   │                          │
    └─────────┘                          │
         是                              │
         ▼                               │
┌─────────────────┐                      │
│ mid = left +    │                      │
│ (right-left)/2  │                      │
└────────┬────────┘                      │
         │                              │
         ▼                              │
     ┌───────┐                          │
     │A[mid] ==│                          │
     │ target?│                          │
     └───┬───┘                          │
    是    │否                           │
    ▼     │                             │
┌───────┐ │                             │
│返回mid│ │                             │
└───────┘ │                             │
          ▼                             │
     ┌───────┐                          │
     │A[mid] <│                          │
     │ target?│                          │
     └───┬───┘                          │
         │否                             │
         ▼                                │
┌─────────────────┐                       │
│ right = mid - 1 │                       │
└────────┬────────┘──────────────────────►┘
         是
         ▼
┌─────────────────┐
│ left = mid + 1  │
└────────┬────────┘
         │
         └───────────────────────────────►┘
```

## KMP字符串匹配

```
预处理阶段 (计算LPS数组):
┌─────────────────┐
│ i = 1, len = 0 │
│ lps[0] = 0     │
└────────┬────────┘
         │
         ▼
     ┌───────┐◄─────────────────────────┐
     │ i < m? │                           │
     └───┬───┘                           │
         │否                             │
         ▼                                │
    ┌─────────┐                           │
    │ LPS数组  │                           │
    │ 计算完成 │                           │
    └─────────┘                           │
         是                               │
         ▼                                │
     ┌───────┐                            │
     │pattern[i]│                          │
     │==pattern[len]?│                     │
     └───┬───┘                            │
         │否                               │
         ▼                                │
    ┌─────────┐                           │
    │len>0?  │                            │
    └───┬─────┘                           │
        │否                               │
        ▼                                 │
┌─────────────────┐                       │
│ lps[i] = 0      │                       │
│ i = i + 1       │                       │
└────────┬────────┘──────────────────────►┘
    是    │
    ▼     │
┌─────────────────┐
│ len = len + 1   │
│ lps[i] = len    │
│ i = i + 1       │
└─────────────────┘
         │
         └───────────────────────────────►┘

匹配阶段:
┌─────────────────┐
│ i = 0 (text)   │
│ j = 0 (pattern)│
└────────┬────────┘
         │
         ▼
     ┌───────┐◄─────────────────────────┐
     │ i < n? │                           │
     └───┬───┘                           │
         │否                             │
         ▼                                │
    ┌─────────┐                           │
    │ 未找到   │                           │
    └─────────┘                           │
         是                               │
         ▼                                │
     ┌───────┐                            │
     │text[i]==│                           │
     │pattern[j]?│                         │
     └───┬───┘                            │
         │否                               │
         ▼                                │
    ┌─────────┐                           │
    │j>0?    │                            │
    └───┬─────┘                           │
        │否                               │
        ▼                                 │
┌─────────────────┐                       │
│ i = i + 1       │                       │
└────────┬────────┘──────────────────────►┘
    是    │
    ▼     │
┌─────────────────┐
│ j = lps[j-1]    │
└─────────────────┘
         │
         └───────────────────────────────►┘
         是(相等)
         ▼
┌─────────────────┐
│ i = i + 1       │
│ j = j + 1       │
└────────┬────────┘
         │
         ▼
     ┌───────┐
     │ j == m? │
     └───┬───┘
         │否
         │
         └───────────────────────────────►┘
         是
         ▼
┌─────────────────┐
│ 找到匹配!       │
│ 位置: i - j    │
└─────────────────┘
```

## 并查集 (Union-Find)

```
查找操作 (带路径压缩):
┌─────────────────┐
│ find(x)         │
└────────┬────────┘
         │
         ▼
     ┌───────┐
     │parent[x]│
     │== x?   │
     └───┬───┘
         │否
         ▼
┌─────────────────┐
│ parent[x] =     │
│ find(parent[x]) │
└────────┬────────┘
         │
         ▼
    ┌─────────┐
    │ 返回     │
    │ parent[x]│
    └─────────┘
         是
         ▼
    ┌─────────┐
    │ 返回x   │
    └─────────┘

合并操作 (按秩合并):
┌─────────────────┐
│ union(x, y)     │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ rx = find(x)    │
│ ry = find(y)    │
└────────┬────────┘
         │
         ▼
     ┌───────┐
     │ rx == ry? │
     └───┬───┘
         │是
         ▼
    ┌─────────┐
    │ 已在同  │
    │ 一集合  │
    └─────────┘
         否
         ▼
     ┌───────┐
     │rank[rx] <│
     │ rank[ry]?│
     └───┬───┘
         │否
         ▼
┌─────────────────┐
│ parent[ry] = rx │
│ rank[rx]++      │
│ (如果相等)      │
└─────────────────┘
         是
         ▼
┌─────────────────┐
│ parent[rx] = ry │
└─────────────────┘
```

## BFS广度优先搜索

```
┌─────────────────┐
│ BFS(G, start)   │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ visited[start] =│
│ true            │
│ queue.push(start)│
└────────┬────────┘
         │
         ▼
     ┌───────┐◄─────────────────────────┐
     │!queue. │                           │
     │ empty()?│                          │
     └───┬───┘                           │
         │是                             │
         ▼                                │
    ┌─────────┐                           │
    │  BFS    │                           │
    │ 完成    │                           │
    └─────────┘                           │
         否                               │
         ▼                                │
┌─────────────────┐                       │
│ u = queue.pop() │                       │
└────────┬────────┘                       │
         │                               │
         ▼                               │
┌─────────────────┐                       │
│ 对u的每个邻居v: │                       │
└────────┬────────┘                       │
         │                               │
         ▼                               │
     ┌───────┐                            │
     │!visited[v]?│                        │
     └───┬───┘                            │
         │否                               │
         │                                 │
         └────────────────────────────────►┘
         是
         ▼
┌─────────────────┐
│ visited[v] = true│
│ queue.push(v)   │
└────────┬────────┘
         │
         └───────────────────────────────►┘
```

## P vs NP

```
╔═════════════════════════════════════════════════════════════════════════════╗
║                   P vs NP 核心问题                                           ║
╠═════════════════════════════════════════════════════════════════════════════╣
║                                                                             ║
║  P 类问题:                                                                  ║
║  • 确定性图灵机在多项式时间内可解                                            ║
║  • 已知有多项式时间算法                                                      ║
║  • 例如: 排序、最短路径、最小生成树                                          ║
║                                                                             ║
║  NP 类问题:                                                                 ║
║  • 非确定性图灵机在多项式时间内可解                                          ║
║  • 等价: 解可在多项式时间内被验证                                            ║
║  • 例如: 3-SAT、团问题、哈密顿回路、图着色                                   ║
║                                                                             ║
║  NP-完全问题 (NPC):                                                         ║
║  • 属于 NP 且所有 NP 问题可多项式归约到它                                    ║
║  • 如果任一 NPC 问题属于 P，则 P = NP                                        ║
║  • 例如: 3-SAT、团问题、顶点覆盖、子集和、TSP                                ║
║                                                                             ║
║  NP-困难问题:                                                               ║
║  • 所有 NP 问题可归约到它 (不一定在 NP 中)                                   ║
║  • 例如: 停机问题、广义围棋                                                  ║
║                                                                             ║
╚═════════════════════════════════════════════════════════════════════════════╝

╔═════════════════════════════════════════════════════════════════════════════╗
║                   P vs NP 关系图                                             ║
╠═════════════════════════════════════════════════════════════════════════════╣
║                                                                             ║
║                       ┌─────────────────┐                                   ║
║                       │    EXPTIME      │                                   ║
║                       └────────┬────────┘                                   ║
║                                │                                            ║
║                       ┌────────▼────────┐                                   ║
║                       │    PSPACE       │                                   ║
║                       └────────┬────────┘                                   ║
║                                │                                            ║
║              ┌─────────────────┼─────────────────┐                          ║
║              │                 │                 │                          ║
║              ▼                 ▼                 ▼                          ║
║      ┌───────────┐     ┌───────────┐     ┌───────────┐                     ║
║      │    NP     │     │  co-NP    │     │PSPACE-完全│                     ║
║      └─────┬─────┘     └─────┬─────┘     └───────────┘                     ║
║            │                 │                                              ║
║            └────────┬────────┘                                              ║
║                     │                                                       ║
║              ┌──────▼──────┐                                                ║
║              │  NP ∩ co-NP │                                                ║
║              │   (如整数   │                                                ║
║              │   分解)     │                                                ║
║              └──────┬──────┘                                                ║
║                     │                                                       ║
║              ┌──────▼──────┐                                                ║
║              │      P      │                                                ║
║              │  (多项式    │                                                ║
║              │   时间)     │                                                ║
║              └─────────────┘                                                ║
║                                                                             ║
║  开放问题:                                                                  ║
║  • P = NP? (千禧年难题)                                                     ║
║  • NP = co-NP?                                                              ║
║  • P = PSPACE?                                                              ║
║                                                                             ║
╚═════════════════════════════════════════════════════════════════════════════╝
```

## 经典 NP-完全问题

```
╔═════════════════════════════════════════════════════════════════════════════╗
║                   SAT 问题家族                                               ║
╠═════════════════════════════════════════════════════════════════════════════╣
║                                                                             ║
║  2-SAT ∈ P (多项式时间可解)                                                  ║
║  • 每个子句最多2个文字                                                       ║
║  • 可用蕴含图 + 强连通分量求解                                               ║
║                                                                             ║
║  3-SAT ∈ NP-完全                                                             ║
║  • 每个子句恰好3个文字                                                       ║
║  • 第一个被证明的NP-完全问题                                                 ║
║  • 所有其他NP-完全问题可归约到3-SAT                                          ║
║                                                                             ║
║  QSAT ∈ PSPACE-完全                                                          ║
║  • 带量词的布尔可满足性                                                      ║
```

## 复杂度分析检查清单

```
╔═════════════════════════════════════════════════════════════════════════════╗
║                   问题分类检查清单                                           ║
╠═════════════════════════════════════════════════════════════════════════════╣
║                                                                             ║
║  判断一个问题属于哪个复杂度类的步骤:                                          ║
║                                                                             ║
║  1. 它属于 P 吗?                                                            ║
║     □ 是否已知多项式时间算法?                                                ║
║     □ 是否可转化为已知的P问题?                                               ║
║     → 如果都是，属于 P                                                       ║
║                                                                             ║
║  2. 它属于 NP 吗?                                                           ║
║     □ 给定一个解，是否可在多项式时间内验证?                                  ║
║     → 如果是，属于 NP                                                        ║
║                                                                             ║
║  3. 它是 NP-完全的吗?                                                       ║
║     □ 它属于 NP 吗?                                                         ║
║     □ 是否存在已知的NP-完全问题可归约到它?                                   ║
║     → 如果都是，是 NP-完全                                                   ║
║                                                                             ║
║  4. 它是 NP-困难的吗?                                                       ║
║     □ 所有NP问题可归约到它吗?                                                ║
║     □ 它不一定在 NP 中吗?                                                    ║
║     → 如果都是，是 NP-困难                                                   ║
║                                                                             ║
╚═════════════════════════════════════════════════════════════════════════════╝
```

## 近似算法速查

```
╔═════════════════════════════════════════════════════════════════════════════╗
║                   NP-困难问题的近似算法                                      ║
╠═════════════════════════════════════════════════════════════════════════════╣
║                                                                             ║
║  ┌─────────────────┬────────────────┬─────────────────────────────────────┐ ║
║  │     问题        │   近似比       │           算法                      │ ║
║  ├─────────────────┼────────────────┼─────────────────────────────────────┤ ║
║  │                 │                │                                     │ ║
║  │   顶点覆盖      │     2          │  取最大匹配的所有端点               │ ║
║  │                 │                │                                     │ ║
║  ├─────────────────┼────────────────┼─────────────────────────────────────┤ ║
║  │                 │                │                                     │ ║
║  │   集合覆盖      │   ln(n)        │  贪心: 每次选覆盖最多未覆盖元素的集合│ ║
║  │                 │                │                                     │ ║
║  ├─────────────────┼────────────────┼─────────────────────────────────────┤ ║
║  │                 │                │                                     │ ║
║  │   TSP 度量      │    1.5         │  Christofides算法                   │ ║
║  │                 │                │  (最小生成树 + 最小权完美匹配)       │ ║
║  ├─────────────────┼────────────────┼─────────────────────────────────────┤ ║
║  │                 │                │                                     │ ║
║  │   背包问题      │    任意接近1   │  动态规划 或 FPTAS                  │ ║
║  │                 │                │                                     │ ║
║  ├─────────────────┼────────────────┼─────────────────────────────────────┤ ║
║  │                 │                │                                     │ ║
║  │   最大割        │    0.878       │  Goemans-Williamson (SDP)           │ ║
║  │                 │                │                                     │ ║
║  └─────────────────┴────────────────┴─────────────────────────────────────┘ ║
║                                                                             ║
║  近似难度:                                                                  ║
║  • APX-完全: 有常数近似比，但不存在PTAS (如顶点覆盖、TSP)                    ║
║  • PTAS: 有任意接近1的近似比 (如背包问题)                                    ║
║  • 无法近似: 如一般TSP (无度量限制)                                          ║
║                                                                             ║
╚═════════════════════════════════════════════════════════════════════════════╝
```

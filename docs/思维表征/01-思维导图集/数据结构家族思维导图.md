## 核心操作速查

| 数据结构 | 核心操作 | 典型应用 |
|---------|---------|---------|
| 数组 | 随机访问 | 缓存、矩阵运算 |
| 链表 | 动态插入删除 | LRU缓存、多项式 |
| 栈 | LIFO | 表达式求值、回溯 |
| 队列 | FIFO | BFS、任务调度 |
| 堆 | 极值获取 | 优先队列、TopK |
| BST | 有序维护 | 索引、范围查询 |
| 哈希表 | 快速查找 | 缓存、去重 |
| 图 | 连通性 | 网络、路径规划 |
| 并查集 | 连通分量 | Kruskal算法 |
| Trie | 前缀匹配 | 自动补全、拼写检查 |

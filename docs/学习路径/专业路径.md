# 专业学习路径


> **版本**: 1.0
> **创建日期**: 2026-04-19
> **最后更新**: 2026-04-19

> 目标：类型理论到形式化验证
> 预计时长：250小时
> 前置要求：完成[进阶路径](进阶路径.md)

## 路径概述

本路径面向希望成为领域专家的学习者，深入探索：

- 高级类型理论（同伦类型论、范畴论）
- 量子计算理论
- 并行与分布式算法
- 前沿研究主题

完成后，您将能够：

- 理解和贡献类型理论前沿研究
- 掌握量子算法设计与分析
- 设计大规模分布式算法
- 进行独立的形式化验证研究

### 1.2 同伦类型论 (30小时)

**文档**: `docs/05-类型理论/03-同伦类型论.md`

**学习目标**:

**关键概念**:

**延伸阅读**:

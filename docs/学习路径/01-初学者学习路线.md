# 🌱 初学者学习路线

> 目标：从零基础到能独立解决基础算法问题，预计耗时 3-6 个月

---

## 📊 学习阶段总览

```
第一阶段 ──────────────→ 第二阶段 ──────────────→ 第三阶段
(基础语法+数据结构)        (基础算法)              (实战练习)
   4-6周                   6-8周                  持续进行
```

---

## 🎯 第一阶段：编程基础与数据结构（4-6周）

### 学习目标
- 熟练掌握一门编程语言（推荐 C++/Python/Java）
- 理解基础数据结构的概念和操作

### 推荐语言特性（以 C++ 为例）

| 特性 | 掌握程度 | 学习资源 |
|------|---------|---------|
| 基础语法（变量、循环、条件） | ⭐⭐⭐ 必须熟练 | 任意入门教程 |
| 数组与字符串操作 | ⭐⭐⭐ 必须熟练 | 本项目 examples/basic |
| 函数与递归 | ⭐⭐⭐ 必须熟练 | 《算法笔记》|
| STL 容器（vector, map, set） | ⭐⭐⭐ 必须熟练 | cppreference.com |
| 指针与引用 | ⭐⭐ 理解即可 | 《C++ Primer》|

### 必学数据结构

```
顺序存储
├── 数组 Array ────────→ 随机访问 O(1)
└── 链表 Linked List ───→ 插入删除 O(1)

线性结构
├── 栈 Stack ──────────→ 后进先出 LIFO
├── 队列 Queue ────────→ 先进先出 FIFO
└── 双端队列 Deque ────→ 两端操作 O(1)

树形结构
└── 二叉树 Binary Tree ─→ 遍历、递归基础
```

## 💡 学习建议

1. **每天坚持**：至少 1-2 小时，保持连续性
2. **多写多练**：看懂 ≠ 会做，一定要动手写
3. **定期复习**：每周回顾错题，巩固薄弱点
4. **参与讨论**：加入学习小组，交流解题思路
5. **记录总结**：写博客或笔记，形成知识体系

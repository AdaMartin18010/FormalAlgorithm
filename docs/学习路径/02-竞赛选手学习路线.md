## 🎯 第一阶段：语言与基础强化（1-2个月）

### C++ 核心特性

```cpp
// 必须熟练掌握的 C++ 特性
#include <bits/stdc++.h>
using namespace std;

// 1. 快速 IO
typedef long long ll;
inline void fastIO() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
}

// 2. STL 容器
vector<int> v;          // 动态数组
set<int> s;             // 有序集合
map<string, int> mp;    // 映射
priority_queue<int> pq; // 优先队列

// 3. 常用算法
sort(v.begin(), v.end());
binary_search(v.begin(), v.end(), x);
lower_bound(v.begin(), v.end(), x);
```

### 必备模板库

| 模板类别 | 模板数量 | 熟练程度 |
|---------|---------|---------|
| 数学（GCD、快速幂、素数筛） | 5个 | 盲写 |
| 字符串（KMP、哈希、Manacher） | 4个 | 盲写 |
| 图论（最短路、最小生成树、LCA） | 6个 | 盲写 |
| 数据结构（并查集、树状数组、线段树） | 5个 | 盲写 |
| 计算几何（基础模板） | 3个 | 参考修改 |

## 🎯 第四阶段：实战与训练（持续）

### 训练计划

| 时间 | 活动 | 内容 |
|-----|------|------|
| 每日 | 个人训练 | 3-5 题，涵盖不同专题 |
| 每周 | 组队训练 | 模拟赛 1-2 场 |
| 每月 | 区域赛真题 | 5 小时模拟赛 |
| 赛前 | 冲刺训练 | 补题、模板复习 |

### 推荐 OJ 与比赛

| 平台 | 难度 | 特点 |
|-----|------|------|
| Codeforces | Div2-Div1 | 思维+算法，定期比赛 |
| AtCoder | ABC-ARC-AGC | 日本风格，质量高 |
| 洛谷 | 普及-省选 | 中文，适合专题训练 |
| 牛客 | 多校联合 | 国内竞赛风格 |
| HDU/POJ | 经典 | 老牌 OJ，题目经典 |
| Virtual Judge | 综合 | 汇集各平台题目 |

### 比赛策略

```
比赛中时间分配建议（5小时）
├── 前 30 分钟
│   ├── 通读所有题目
│   └── 评估难度，制定策略
├── 30分钟-2小时
│   ├── 主攻签到题（A、B）
│   └── 争取快速通过
├── 2-4小时
│   ├── 主攻中等题（C、D）
│   ├── 一人一道，并行推进
│   └── 及时沟通思路
└── 最后1小时
    ├── 攻难题或优化已有题
    ├── 检查边界情况
    └── 提交时注意文件名和格式
```

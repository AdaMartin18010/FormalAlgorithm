## 案例概述

**算法**: 最近点对 (Closest Pair of Points)
**应用领域**: 计算几何、碰撞检测、聚类分析、分子模拟
**案例来源**: 游戏物理引擎 / 地理信息系统 / 计算机图形学

## 应用场景描述

### 背景

最近点对问题是计算几何的经典问题：

- **游戏开发**: 物体碰撞检测
- **GIS**: 最近设施查询
- **分子模拟**: 原子距离计算
- **机器学习**: 密度聚类

### 问题定义

**场景 - 游戏碰撞检测系统**:

**输入**:

- 场景中数千个物体的位置
- 需要实时检测即将发生的碰撞

**输出**:

- 距离最近的物体对
- 碰撞预警

### 为什么选择分治算法

| 特性 | 分治优势 | 对比 |
|------|---------|------|
| 时间复杂度 | ✅ $O(n \log n)$ | 暴力 $O(n^2)$ |
| 可扩展 | ✅ 可处理百万级点 | - |
| 精确 | ✅ 精确解 | 空间哈希有漏检 |

## 算法实现

```rust
use std::cmp::Ordering;

#[derive(Clone, Copy, Debug)]
pub struct Point {
    pub x: f64,
    pub y: f64,
    pub id: usize,
}

impl Point {
    fn distance_sq(&self, other: &Point) -> f64 {
        let dx = self.x - other.x;
        let dy = self.y - other.y;
        dx * dx + dy * dy
    }
}

/// 暴力法
pub fn closest_pair_brute_force(points: &[Point]) -> Option<(f64, Point, Point)> {
    let n = points.len();
    if n < 2 {
        return None;
    }

    let mut min_dist = f64::INFINITY;
    let mut closest = None;

    for i in 0..n {
        for j in (i + 1)..n {
            let dist = points[i].distance_sq(&points[j]);
            if dist < min_dist {
                min_dist = dist;
                closest = Some((dist.sqrt(), points[i], points[j]));
            }
        }
    }

    closest
}

/// 分治算法
pub fn closest_pair(points: &[Point]) -> Option<(f64, Point, Point)> {
    let n = points.len();
    if n <= 3 {
        return closest_pair_brute_force(points);
    }

    // 按x排序
    let mut px: Vec<_> = points.to_vec();
    px.sort_by(|a, b| a.x.partial_cmp(&b.x).unwrap());

    // 按y排序
    let mut py: Vec<_> = points.to_vec();
    py.sort_by(|a, b| a.y.partial_cmp(&b.y).unwrap());

    closest_pair_recursive(&px, &py)
}

fn closest_pair_recursive(px: &[Point], py: &[Point]) -> Option<(f64, Point, Point)> {
    let n = px.len();

    if n <= 3 {
        return closest_pair_brute_force(px);
    }

    let mid = n / 2;
    let mid_point = px[mid];

    // 分割
    let (qx, rx) = px.split_at(mid);

    let mut qy = Vec::new();
    let mut ry = Vec::new();
    for p in py {
        if p.x <= mid_point.x {
            qy.push(*p);
        } else {
            ry.push(*p);
        }
    }

    // 递归求解
    let left = closest_pair_recursive(qx, &qy);
    let right = closest_pair_recursive(rx, &ry);

    let mut delta = f64::INFINITY;
    let mut best_pair = None;

    match (&left, &right) {
        (Some((d1, ..)), Some((d2, ..))) => {
            if d1 < d2 {
                delta = *d1;
                best_pair = left;
            } else {
                delta = *d2;
                best_pair = right;
            }
        }
        (Some((d, ..)), None) => {
            delta = *d;
            best_pair = left;
        }
        (None, Some((d, ..))) => {
            delta = *d;
            best_pair = right;
        }
        _ => {}
    }

    // 检查跨分割线的情况
    let strip: Vec<_> = py.iter()
        .filter(|p| (p.x - mid_point.x).abs() < delta)
        .cloned()
        .collect();

    for i in 0..strip.len() {
        for j in (i + 1)..strip.len().min(i + 7) {
            let dist = strip[i].distance_sq(&strip[j]).sqrt();
            if dist < delta {
                delta = dist;
                best_pair = Some((dist, strip[i], strip[j]));
            }
        }
    }

    best_pair
}
```

### 复杂度分析

| 算法 | 时间复杂度 | 空间复杂度 |
|-----|-----------|-----------|
| 暴力 | $O(n^2)$ | $O(1)$ |
| 分治 | $O(n \log n)$ | $O(n)$ |
| 随机化 | $O(n)$ 期望 | $O(n)$ |

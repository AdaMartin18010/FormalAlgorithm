## 案例概述

**算法**: 随机算法 (Randomized Algorithms)
**应用领域**: 蒙特卡洛模拟、随机采样、快速排序随机化、哈希
**案例来源**: 金融风险管理 / 机器学习采样 / 分布式系统

## 应用场景描述

### 背景

随机算法利用随机性获得效率或简单性：

- **金融**: 风险价值(VaR)计算、期权定价
- **机器学习**: 随机梯度下降、Dropout
- **系统**: 负载均衡、一致性哈希
- **图算法**: 最小割、连通性测试

### 问题定义

**场景 - 金融风险管理**:

**输入**:

- 投资组合配置
- 历史价格数据
- 置信水平（如95%）

**输出**:

- 风险价值(VaR)：给定置信水平下的最大预期损失

### 为什么选择随机算法

| 特性 | 随机算法优势 | 对比 |
|------|------------|------|
| 处理复杂性 | ✅ 高维积分、复杂分布 | 确定性方法难以处理 |
| 可扩展 | ✅ 易于并行化 | - |
| 简单性 | ✅ 实现简单 | 确定性方法可能很复杂 |

## 算法实现

```rust
use rand::Rng;
use rand_distr::{Normal, Distribution};

/// 蒙特卡洛积分
pub fn monte_carlo_integral<F>(f: F, a: f64, b: f64, n: usize) -> f64
where
    F: Fn(f64) -> f64,
{
    let mut rng = rand::thread_rng();
    let mut sum = 0.0;

    for _ in 0..n {
        let x = rng.gen_range(a..b);
        sum += f(x);
    }

    (b - a) * sum / n as f64
}

/// 风险价值(VaR)蒙特卡洛计算
pub struct Portfolio {
    returns: Vec<f64>,     // 历史收益率
    weights: Vec<f64>,     // 资产权重
    covariance: Vec<Vec<f64>>, // 协方差矩阵
}

impl Portfolio {
    /// 蒙特卡洛模拟计算VaR
    pub fn var_monte_carlo(&self, confidence: f64, days: usize, simulations: usize) -> f64 {
        let mut rng = rand::thread_rng();
        let mut losses = Vec::with_capacity(simulations);

        for _ in 0..simulations {
            // 模拟收益率路径
            let simulated_return = self.simulate_return(&mut rng, days);
            losses.push(-simulated_return);  // 损失为负收益
        }

        losses.sort_by(|a, b| a.partial_cmp(b).unwrap());

        // 取对应分位数的损失
        let index = ((1.0 - confidence) * simulations as f64) as usize;
        losses[index]
    }

    fn simulate_return<R: Rng>(&self, rng: &mut R, days: usize) -> f64 {
        // 简化的几何布朗运动模拟
        let normal = Normal::new(0.0, 1.0).unwrap();
        let dt = 1.0 / 252.0;  // 交易日

        let mut total_return = 0.0;

        for _ in 0..days {
            let z = normal.sample(rng);
            let daily_return = 0.05 * dt + 0.2 * z * dt.sqrt();  // 简化模型
            total_return += daily_return;
        }

        total_return
    }
}

/// 随机采样（Reservoir Sampling）
pub fn reservoir_sampling<T: Clone>(stream: &[T], k: usize) -> Vec<T> {
    let mut result = stream[..k.min(stream.len())].to_vec();

    if stream.len() <= k {
        return result;
    }

    let mut rng = rand::thread_rng();

    for i in k..stream.len() {
        let j = rng.gen_range(0..=i);
        if j < k {
            result[j] = stream[i].clone();
        }
    }

    result
}

/// 随机快速排序
pub fn randomized_quicksort<T: Ord + Clone>(arr: &mut [T]) {
    if arr.len() <= 1 {
        return;
    }

    let mut rng = rand::thread_rng();
    let pivot_idx = rng.gen_range(0..arr.len());
    arr.swap(pivot_idx, arr.len() - 1);

    let pivot = partition(arr);
    randomized_quicksort(&mut arr[..pivot]);
    randomized_quicksort(&mut arr[pivot + 1..]);
}

fn partition<T: Ord>(arr: &mut [T]) -> usize {
    let len = arr.len();
    let mut i = 0;

    for j in 0..len - 1 {
        if arr[j] <= arr[len - 1] {
            arr.swap(i, j);
            i += 1;
        }
    }

    arr.swap(i, len - 1);
    i
}

/// 近似计数（Count-Min Sketch简化）
pub struct CountMinSketch {
    width: usize,
    depth: usize,
    table: Vec<Vec<u64>>,
    hash_seeds: Vec<u64>,
}

impl CountMinSketch {
    pub fn new(width: usize, depth: usize) -> Self {
        let table = vec![vec![0u64; width]; depth];
        let hash_seeds: Vec<u64> = (0..depth).map(|i| rand::random()).collect();

        Self { width, depth, table, hash_seeds }
    }

    pub fn add(&mut self, item: &str) {
        for d in 0..self.depth {
            let idx = self.hash(item, d) % self.width;
            self.table[d][idx] += 1;
        }
    }

    pub fn estimate(&self, item: &str) -> u64 {
        (0..self.depth)
            .map(|d| self.table[d][self.hash(item, d) % self.width])
            .min()
            .unwrap_or(0)
    }

    fn hash(&self, item: &str, seed: usize) -> usize {
        use std::collections::hash_map::DefaultHasher;
        use std::hash::{Hash, Hasher};

        let mut hasher = DefaultHasher::new();
        item.hash(&mut hasher);
        self.hash_seeds[seed].hash(&mut hasher);
        hasher.finish() as usize
    }
}
```

### 复杂度分析

| 算法 | 期望时间 | 空间 | 误差 |
|-----|---------|------|------|
| 蒙特卡洛积分 | $O(n)$ | $O(1)$ | $O(1/\sqrt{n})$ |
| Reservoir Sampling | $O(n)$ | $O(k)$ | 无 |
| 随机快排 | $O(n \log n)$ | $O(\log n)$ | 无 |
| Count-Min Sketch | $O(1)$ | $O(w \cdot d)$ | 有界 |

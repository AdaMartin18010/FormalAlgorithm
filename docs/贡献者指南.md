# 贡献者指南

欢迎来到【算法规范与模型设计知识体系】项目！

## 关于项目

本项目是一个开源的算法理论文档项目，目标是建立系统、严谨、国际化的算法知识体系，为中文学习者提供高质量的算法教育资源。

**项目特点**:
- 📚 涵盖12个核心模块，200+文档
- 🎯 对标国际顶尖大学课程
- 📐 遵循严格的学术规范
- 🌍 中英双语，国际化视野
- ✅ 已完成多轮质量改进

## 如何贡献

### 贡献类型

#### A. 内容贡献
- **补充缺失主题** - 补充P0/P1/P2优先级主题
- **修正错误** - 修正概念、证明、代码中的错误
- **更新最新研究** - 添加2024-2025年最新研究进展
- **添加应用案例** - 为算法添加实际应用案例
- **改进形式化证明** - 完善定理证明

#### B. 质量改进
- **校对文档** - 检查拼写、语法、格式
- **统一术语** - 确保术语使用一致
- **改进引用** - 补充缺失引用，规范引用格式
- **代码审查** - 检查代码实现

#### C. 社区建设
- **回答问题** - 在Discussions中帮助其他学习者
- **撰写教程** - 为复杂概念编写入门教程
- **推广项目** - 向学术界和开发者社区推广
- **组织活动** - 组织线上/线下学习活动

### 贡献流程

#### 第一步：选择任务

查看项目看板了解当前任务：
- 🔴 P0优先级：立即补充的主题
- 🟡 P1优先级：1个月内补充
- 🟢 P2优先级：3个月内补充
- 🔵 社区任务：其他改进任务

#### 第二步：认领任务

1. 在相关Issue下评论："我想认领这个任务，预计X天完成"
2. 等待维护者确认
3. Fork仓库并开始工作

#### 第三步：开展工作

```bash
# Fork后克隆你的仓库
git clone https://github.com/YOUR_USERNAME/FormalAlgorithm.git
cd FormalAlgorithm

# 创建分支
git checkout -b feature/your-feature-name

# 进行修改
# ...

# 提交更改
git add .
git commit -m "feat: 添加XX内容"
git push origin feature/your-feature-name
```

## 第四步：提交PR
### 第四步：提交PR
#### 第四步：提交PR

1. 在GitHub上创建Pull Request
2. 使用PR模板填写信息
3. 关联相关Issue
4. 等待审查

#### 第五步：审查与合并

1. 维护者会进行审查
2. 根据反馈修改
3. 通过自动化检查
4. 合并到主分支
5. 更新贡献者名单

## 文档贡献规范

### 文档结构

标准文档必须包含以下章节：

```markdown
# 文档标题

> **版本**: 1.0
> **创建日期**: YYYY-MM-DD
> **最后更新**: YYYY-MM-DD
> **难度**: 初级/中级/高级

## 学习目标

完成本章节后，读者将能够：
1. ...
2. ...
3. ...

## 1. 引言

## 2. 形式化定义

## 3. 核心内容

## 4. 复杂度分析

## 5. 实现示例

## 6. 应用案例

## 7. 练习

## 8. 总结

## 参考文献

## 知识导航
- **前置知识**: [链接]
- **后续学习**: [链接]
- **相关主题**: [链接]
```

### 写作风格

- **中英文双语**：关键术语提供中英文对照
- **数学符号**：遵循ISO 80000-2标准
- **引用格式**：使用ACM引用格式
- **代码示例**：优先使用Rust，其次是Python

### 质量要求

- [ ] 所有定义有来源引用
- [ ] 所有定理有完整证明
- [ ] 复杂度分析准确
- [ ] 交叉引用正确
- [ ] 拼写和语法正确

## 代码贡献规范

- 使用Rust语言
- 遵循标准代码风格（cargo fmt）
- 包含完整文档注释
- 单元测试覆盖率>80%
- 通过clippy检查

## 贡献者等级

| 等级 | 贡献标准 | 特权 |
|------|----------|------|
| 👁️ 观察者 | 关注项目 | 接收更新通知 |
| 🐛 报告者 | 提交Issue | 参与讨论 |
| 📝 贡献者 | 合并1个PR | 列入贡献者名单 |
| ⭐ 活跃贡献者 | 合并5个PR | PR审查权限 |
| 🏆 核心贡献者 | 合并20个PR | 维护权限 |
| 👑 维护者 | 长期贡献 | 管理权限 |

- 保持专业态度

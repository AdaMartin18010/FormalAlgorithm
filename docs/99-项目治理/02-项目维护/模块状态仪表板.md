## 四、自动统计脚本

以下命令可自动更新"文档数"列：

```bash
# 在项目根目录执行 PowerShell
Get-ChildItem -Path "docs" -Directory | ForEach-Object {
    $count = (Get-ChildItem -Path $_.FullName -Recurse -Filter "*.md" | Measure-Object).Count
    [PSCustomObject]@{模块=$_.Name; 文档数=$count}
} | Sort-Object 模块
```

或使用以下 Python 片段：

```python
from pathlib import Path

docs_path = Path("docs")
for module_dir in sorted(docs_path.iterdir()):
    if module_dir.is_dir():
        count = len(list(module_dir.rglob("*.md")))
        print(f"| {module_dir.name} | {count} | ... |")
```

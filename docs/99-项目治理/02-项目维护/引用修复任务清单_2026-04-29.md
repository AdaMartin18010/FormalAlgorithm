## 使用说明

1. 运行本脚本自动生成最新清单：
   ```bash
   python scripts/generate_citation_tasks.py
   ```
2. 自动修复优先级高的文件时，请先备份原文件到 `.backup/` 目录。
3. 每修复完一个文件，请在对应行的"状态"列标记为 `✅ 已完成`。
4. 对于无法确定具体引用位置的文件，保留在清单中并标注 `❓ 需人工复核`。

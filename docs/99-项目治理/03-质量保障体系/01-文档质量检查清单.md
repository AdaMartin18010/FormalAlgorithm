# 文档质量检查清单

## 六维内容检查

- [ ] 概念定义完整
- [ ] 属性分析准确
- [ ] 关系图表清晰
- [ ] 解释通俗易懂
- [ ] 论证逻辑严密
- [ ] 形式证明严格

## 格式规范

- [ ] ISO 80000数学符号
- [ ] ACM引用规范
- [ ] Mermaid图表
- [ ] 中英文术语对照

## 元数据规范 (Metadata)

- [ ] 内容所有权 (Content Ownership)
  - [ ] `owner` 字段存在且非空
  - [ ] owner 值与[内容所有权分配表](../项目维护/内容所有权分配表_2026-04-29.md)一致
- [ ] Diátaxis 分类 (Diátaxis Classification)
  - [ ] `diataxis` 字段存在（Tutorial / How-To Guide / Reference / Explanation）
  - [ ] 取值符合项目规范：`tutorial`、`how-to`、`reference`、`explanation`

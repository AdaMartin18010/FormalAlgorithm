# 第1章 网络流

## 1.1 最大流与最小割

### 习题 02-01-001 ⭐

**类型**: 理解型

**题目**：
解释最大流最小割定理：对于任何网络，最大流的值等于最小割的容量。

**答案**：
<details>
<summary>点击展开答案</summary>

**定理陈述**：
$$
\max |f| = \min_{(S,T)} c(S,T)
$$

**证明概要**：

1. **弱对偶性**：对任意流 $f$ 和割 $(S,T)$，有 $|f| \leq c(S,T)$
   - 跨越割的流量不超过割容量

2. **强对偶性**：当流 $f$ 达到最大时，残差网络中不存在增广路
   - 令 $S$ 为源点可达顶点集，$T = V \setminus S$
   - 则 $(S,T)$ 形成割，且 $|f| = c(S,T)$

**意义**：
建立了组合优化中的原始-对偶关系，是线性规划对偶性的特例。
</details>

### 习题 02-01-002 ⭐⭐

**类型**: 实现型

**题目**：
实现Edmonds-Karp算法（基于BFS的Ford-Fulkerson），分析其时间复杂度 $O(VE^2)$。

**答案**：
<details>
<summary>点击展开答案</summary>

```python
from collections import deque

def edmonds_karp(graph, source, sink):
    """
    graph: 邻接表，graph[u] = [(v, capacity), ...]
    返回最大流值
    """
    n = len(graph)
    # 残差容量
    residual = [[0] * n for _ in range(n)]
    for u in range(n):
        for v, cap in graph[u]:
            residual[u][v] = cap
    
    max_flow = 0
    
    while True:
        # BFS找增广路
        parent = [-1] * n
        parent[source] = source
        queue = deque([source])
        
        while queue and parent[sink] == -1:
            u = queue.popleft()
            for v in range(n):
                if parent[v] == -1 and residual[u][v] > 0:
                    parent[v] = u
                    queue.append(v)
        
        if parent[sink] == -1:
            break  # 无增广路
        
        # 找到路径上的最小残差容量
        path_flow = float('inf')
        s = sink
        while s != source:
            path_flow = min(path_flow, residual[parent[s]][s])
            s = parent[s]
        
        # 更新残差网络
        v = sink
        while v != source:
            u = parent[v]
            residual[u][v] -= path_flow
            residual[v][u] += path_flow
            v = u
        
        max_flow += path_flow
    
    return max_flow
```

**复杂度分析**：
- 每次BFS增广：$O(E)$
- 增广次数上限：$O(VE)$（因为每次至少一条边饱和）
- 总计：$O(VE^2)$
</details>

### 习题 02-01-003 ⭐⭐⭐

**类型**: 证明型

**题目**：
证明Dinic算法在单位容量网络中的时间复杂度为 $O(\min(V^{2/3}, E^{1/2}) \cdot E)$。

**答案**：
<details>
<summary>点击展开答案</summary>

**Dinic算法**：
1. 构建层次图（BFS）
2. 在层次图中找阻塞流（DFS）
3. 重复直到不存在增广路

**关键引理**：
- 每轮BFS后，源点到汇点的最短距离严格增加
- 因此最多 $O(V)$ 轮

**单位容量分析**：

**情况1**：当 $d < V^{1/3}$ 时
- 每轮增流至少 $d$（因为层次图有 $d$ 层）
- 总轮数 $O(V/d)$

**情况2**：当阻塞流饱和了 $E' \geq E^{1/2}$ 条边时
- 最多 $O(E^{1/2})$ 轮

结合两种情况，得到 $O(\min(V^{2/3}, E^{1/2}) \cdot E)$
</details>

**答案**：

## 1.2 最小费用流

### 习题 02-02-001 ⭐⭐⭐

**类型**: 设计型

**题目**：
设计最小费用最大流算法：每条边有容量和费用，求流量为最大时的最小费用。

**答案**：
<details>
<summary>点击展开答案</summary>

**连续最短增广路算法**：
1. 在残差网络中找从 $s$ 到 $t$ 的最短费用路径（Bellman-Ford或SPFA）
2. 沿路径增广最大可能流量
3. 更新残差网络和费用
4. 重复直到不存在增广路

**为什么正确**：
基于线性规划对偶性，每次选择费用最小的增广路保证全局最优。

**复杂度**：$O(F \cdot VE)$，$F$ 为最大流值
</details>

**答案**：
<details>
<summary>点击展开答案</summary>

**标准形式**：
$$
\begin{aligned}
\max \quad & c^T x \\
\text{s.t.} \quad & Ax = b \\
& x \geq 0
\end{aligned}
$$

**转化方法**：

| 原问题 | 转化方法 |
|:-------|:---------|
| 最小化 | $\min c^Tx \Rightarrow \max (-c)^Tx$ |
| $\leq$ 约束 | 加松弛变量 |
| $\geq$ 约束 | 减剩余变量 |
| $=$ 约束 | 直接保留 |
| 无约束变量 | $x = x^+ - x^+$，$x^+, x^- \geq 0$ |
</details>

## 2.2 对偶理论

### 习题 02-04-001 ⭐⭐

**类型**: 理解型

**题目**：
写出以下原始LP的对偶问题：
$$
\begin{aligned}
\max \quad & c^T x \\
\text{s.t.} \quad & Ax \leq b \\
& x \geq 0
\end{aligned}
$$

**答案**：
<details>
<summary>点击展开答案</summary>

**对偶问题**：
$$
\begin{aligned}
\min \quad & b^T y \\
\text{s.t.} \quad & A^T y \geq c \\
& y \geq 0
\end{aligned}
$$

**对偶规则**：
- 原问题max ↔ 对偶问题min
- 约束系数矩阵转置
- 右端项与目标函数互换
- 约束方向反转
</details>

---

### 习题 02-04-002 ⭐⭐⭐

**类型**: 证明型

**题目**：
证明弱对偶定理：若 $x$ 是原问题可行解，$y$ 是对偶问题可行解，则 $c^T x \leq b^T y$。

**答案**：
<details>
<summary>点击展开答案</summary>

**证明**：
$$c^T x \leq (A^T y)^T x = y^T A x \leq y^T b = b^T y$$

第一个不等式：$A^T y \geq c$ 且 $x \geq 0$
第二个不等式：$Ax \leq b$ 且 $y \geq 0$

**推论**：若存在可行解使 $c^T x = b^T y$，则两者都是最优解。
</details>

- 扫描：$O(n)$
- 总计：$O(n \log n)$
</details>

### 习题 02-05-002 ⭐⭐⭐

**类型**: 实现型

**题目**：
实现凸包的Andrew单调链算法（按x坐标排序后分别求上下凸壳）。

**答案**：
<details>
<summary>点击展开答案</summary>

```python
def convex_hull(points):
    """Andrew单调链算法，返回凸包顶点（逆时针）"""
    points = sorted(set(points))  # 去重并排序
    
    if len(points) <= 1:
        return points
    
    def cross(o, a, b):
        """叉积OA × OB"""
        return (a[0]-o[0])*(b[1]-o[1]) - (a[1]-o[1])*(b[0]-o[0])
    
    # 构建下凸壳
    lower = []
    for p in points:
        while len(lower) >= 2 and cross(lower[-2], lower[-1], p) <= 0:
            lower.pop()
        lower.append(p)
    
    # 构建上凸壳
    upper = []
    for p in reversed(points):
        while len(upper) >= 2 and cross(upper[-2], upper[-1], p) <= 0:
            upper.pop()
        upper.append(p)
    
    # 去掉重复端点
    return lower[:-1] + upper[:-1]
```

**复杂度**：$O(n \log n)$（主要开销在排序）
</details>

# 第4章 快速傅里叶变换

## 4.1 FFT算法

### 习题 02-06-001 ⭐

**类型**: 理解型

**题目**：
解释离散傅里叶变换（DFT）的定义，说明为什么FFT能将复杂度从 $O(n^2)$ 降到 $O(n \log n)$。

**答案**：
<details>
<summary>点击展开答案</summary>

**DFT定义**：
$$y_k = \sum_{j=0}^{n-1} x_j \omega_n^{jk}, \quad k = 0, 1, \ldots, n-1$$

其中 $\omega_n = e^{2\pi i/n}$ 是 $n$ 次单位根

**FFT加速原理**：
分治策略：
$$y_k = \underbrace{\sum_{j=0}^{n/2-1} x_{2j} \omega_{n/2}^{jk}}_{\text{偶数项}} + \omega_n^k \underbrace{\sum_{j=0}^{n/2-1} x_{2j+1} \omega_{n/2}^{jk}}_{\text{奇数项}}$$

问题规模减半，递归求解，合并结果。

**复杂度**：$T(n) = 2T(n/2) + O(n) = O(n \log n)$
</details>

---

### 习题 02-06-002 ⭐⭐

**类型**: 应用型

**题目**：
使用FFT计算两个多项式 $A(x) = 1 + 2x + 3x^2$ 和 $B(x) = 4 + 5x$ 的乘积。

**答案**：
<details>
<summary>点击展开答案</summary>

**步骤**：
1. 补零使长度 $\geq \deg(A) + \deg(B) + 1 = 4$，取 $n=4$
2. DFT：计算 $A$ 和 $B$ 在4次单位根处的值
3. 点值相乘：$C(\omega^k) = A(\omega^k) \cdot B(\omega^k)$
4. IDFT：从点值恢复系数

**结果**：$C(x) = 4 + 13x + 22x^2 + 15x^3$
</details>

### 习题 02-07-002 ⭐⭐

**类型**: 实现型

**题目**：
实现支持区间修改、区间查询的带懒标记线段树。

**答案**：
<details>
<summary>点击展开答案</summary>

```python
class LazySegmentTree:
    def __init__(self, arr):
        self.n = len(arr)
        self.tree = [0] * (4 * self.n)
        self.lazy = [0] * (4 * self.n)
        self.build(arr, 0, 0, self.n - 1)
    
    def build(self, arr, node, start, end):
        if start == end:
            self.tree[node] = arr[start]
        else:
            mid = (start + end) // 2
            self.build(arr, 2*node+1, start, mid)
            self.build(arr, 2*node+2, mid+1, end)
            self.tree[node] = self.tree[2*node+1] + self.tree[2*node+2]
    
    def push_down(self, node, start, end):
        """下传懒标记"""
        if self.lazy[node] != 0:
            mid = (start + end) // 2
            left_len = mid - start + 1
            right_len = end - mid
            
            self.tree[2*node+1] += self.lazy[node] * left_len
            self.lazy[2*node+1] += self.lazy[node]
            
            self.tree[2*node+2] += self.lazy[node] * right_len
            self.lazy[2*node+2] += self.lazy[node]
            
            self.lazy[node] = 0
    
    def range_update(self, node, start, end, l, r, val):
        """区间加val"""
        if l <= start and end <= r:
            self.tree[node] += val * (end - start + 1)
            self.lazy[node] += val
            return
        
        self.push_down(node, start, end)
        mid = (start + end) // 2
        
        if l <= mid:
            self.range_update(2*node+1, start, mid, l, r, val)
        if r > mid:
            self.range_update(2*node+2, mid+1, end, l, r, val)
        
        self.tree[node] = self.tree[2*node+1] + self.tree[2*node+2]
    
    def range_query(self, node, start, end, l, r):
        """区间求和"""
        if l <= start and end <= r:
            return self.tree[node]
        
        self.push_down(node, start, end)
        mid = (start + end) // 2
        res = 0
        
        if l <= mid:
            res += self.range_query(2*node+1, start, mid, l, r)
        if r > mid:
            res += self.range_query(2*node+2, mid+1, end, l, r)
        
        return res
```
</details>

### 习题 02-08-002 ⭐⭐⭐

**类型**: 设计型

**题目**：
设计Count-Min Sketch数据结构，用于估计数据流中元素的频率。

**答案**：
<details>
<summary>点击展开答案</summary>

**Count-Min Sketch**：
- 二维数组 $CM[w][d]$，初始全0
- $d$ 个哈希函数 $h_1, \ldots, h_d$

**更新**：元素 $x$ 到达
```
for i in 1..d:
    CM[h_i(x)][i] += 1
```

**查询**：
$$\hat{f}(x) = \min_{i} CM[h_i(x)][i]$$

**性质**：
- 空间：$O((1/\epsilon) \log(1/\delta))$
- 估计值 $\hat{f}(x) \geq f(x)$（只高估不高估）
- $P[\hat{f}(x) \leq f(x) + \epsilon||f||_1] \geq 1-\delta$
</details>

## 7.1 子模函数

### 习题 02-09-001 ⭐

**类型**: 理解型

**题目**：
什么是子模函数？给出定义和两个常见例子。

**答案**：
<details>
<summary>点击展开答案</summary>

**定义**：函数 $f: 2^V \to \mathbb{R}$ 是子模的，若对任意 $A \subseteq B \subseteq V$ 和 $x \notin B$：
$$f(A \cup \{x\}) - f(A) \geq f(B \cup \{x\}) - f(B)$$

即：边际收益递减

**例子**：
1. **覆盖函数**：$f(S) = |\bigcup_{i \in S} A_i|$，集合覆盖的大小
2. **熵函数**：$f(S) = H(X_S)$，随机变量子集的联合熵
</details>

### 习题 02-09-002 ⭐⭐⭐

**类型**: 证明型

**题目**：
证明：对于单调子模函数，贪心算法在基数约束下达到 $(1-1/e)$ 近似比。

**答案**：
<details>
<summary>点击展开答案</summary>

**贪心算法**：
每次选择边际增益最大的元素，直到选 $k$ 个

**证明概要**：
设 $S_i$ 为贪心算法第 $i$ 步的解，$OPT$ 为最优解

关键引理：
$$f(S_i) - f(S_{i-1}) \geq \frac{1}{k}(f(OPT) - f(S_{i-1}))$$

递推得：
$$f(S_k) \geq (1 - (1 - 1/k)^k)f(OPT) \geq (1 - 1/e)f(OPT)$$

$(1-1/e) \approx 0.632$ 是紧的（集合覆盖问题）。
</details>

# 第8章 多臂老虎机

## 8.1 探索与利用

### 习题 02-10-001 ⭐

**类型**: 理解型

**题目**：
什么是探索-利用权衡（Exploration-Exploitation Tradeoff）？在多臂老虎机问题中如何形式化？

**答案**：
<details>
<summary>点击展开答案</summary>

**问题设定**：
- $K$ 个臂，每个臂有未知奖励分布
- 每轮选择一个臂，获得奖励
- 目标：最大化累积奖励（或最小化后悔）

**后悔（Regret）定义**：
$$R_T = T \cdot \mu^* - \sum_{t=1}^{T} \mu_{a_t}$$

其中 $\mu^*$ 是最优臂期望，$\mu_{a_t}$ 是第 $t$ 轮选择臂的期望

**探索vs利用**：
- **探索**：尝试不同臂以估计其奖励
- **利用**：选择当前估计最好的臂
- 过早利用可能陷入次优臂，过度探索浪费机会
</details>

### 习题 02-10-002 ⭐⭐

**类型**: 设计型

**题目**：
实现UCB1算法，分析其后悔界 $O(\sqrt{KT \log T})$。

**答案**：
<details>
<summary>点击展开答案</summary>

```python
import math

class UCB1:
    def __init__(self, K):
        self.K = K
        self.counts = [0] * K  # 各臂选择次数
        self.values = [0.0] * K  # 各臂平均奖励
        self.t = 0
    
    def select_arm(self):
        """选择UCB值最高的臂"""
        for arm in range(self.K):
            if self.counts[arm] == 0:
                return arm  # 先尝试未探索的臂
        
        ucb_values = []
        for arm in range(self.K):
            bonus = math.sqrt(2 * math.log(self.t) / self.counts[arm])
            ucb_values.append(self.values[arm] + bonus)
        
        return ucb_values.index(max(ucb_values))
    
    def update(self, arm, reward):
        """更新统计"""
        self.t += 1
        self.counts[arm] += 1
        n = self.counts[arm]
        value = self.values[arm]
        self.values[arm] = ((n - 1) / n) * value + (1 / n) * reward
```

**UCB公式**：
$$UCB_i = \bar{\mu}_i + \sqrt{\frac{2 \ln t}{n_i}}$$

第一项：利用（当前平均奖励）
第二项：探索（与选择次数负相关）
</details>

# 第9章 在线学习

## 9.1 专家问题

### 习题 02-11-001 ⭐⭐

**类型**: 理解型

**题目**：
解释Weighted Majority Algorithm，证明其后悔界为 $O(\sqrt{T \ln N})$。

**答案**：
<details>
<summary>点击展开答案</summary>

**算法**：
- $N$ 个专家，初始权重 $w_i = 1$
- 每轮按权重比例随机选择专家意见
- 专家犯错后，权重乘以 $\beta \in [0,1)$

**分析**：
设 $W_t$ 为第 $t$ 轮总权重，$m$ 为算法犯错次数
$$W_T \geq (1-\beta)^{m^*} \geq (1-\beta)^{m}$$

同时 $W_T \leq N \cdot (\frac{1+\beta}{2})^m$

取对数整理得：$m - m^* \leq O(\sqrt{T \ln N})$
</details>

### 习题 02-14-002 ⭐⭐⭐

**类型**: 理解型

**题目**：
简述Shor算法的核心步骤，说明为什么它能实现整数分解的多项式时间算法。

**答案**：
<details>
<summary>点击展开答案</summary>

**核心思想**：
将因子分解转化为周期寻找问题，用量子傅里叶变换加速

**步骤**：
1. 随机选 $a < N$，计算 $\gcd(a, N)$
2. 用量子电路找 $f(x) = a^x \mod N$ 的周期 $r$

**复杂度**：$O((\log N)^3)$，多项式时间

**意义**：
威胁RSA等基于因子分解困难的密码系统

## 13.1 公钥密码

### 习题 02-15-001 ⭐

**类型**: 理解型

**题目**：
解释RSA加密算法的数学基础（大数分解困难性），说明密钥生成和加密解密过程。

**答案**：
<details>
<summary>点击展开答案</summary>

**密钥生成**：
1. 选两个大素数 $p, q$
2. 计算 $n = pq$，$\phi(n) = (p-1)(q-1)$
3. 选 $e$ 使得 $\gcd(e, \phi(n)) = 1$
4. 计算 $d = e^{-1} \mod \phi(n)$

**公钥**：$(e, n)$
**私钥**：$(d, n)$

**加密**：$c = m^e \mod n$
**解密**：$m = c^d \mod n$

**正确性**：由欧拉定理，$m^{ed} \equiv m \pmod{n}$

**安全性**：基于大整数分解的困难性
</details>

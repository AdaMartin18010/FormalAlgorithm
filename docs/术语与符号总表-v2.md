## I. 规范约定

### 术语使用规范

| 术语 | 使用场景 | 说明 |
|------|----------|------|
| 可计算 (Computable) | 函数可计算 | 可由图灵机计算的函数 |
| 可判定 (Decidable) | 语言/性质判定 | 存在算法可判定的语言 |
| 递归 (Recursive) | 集合/函数 | 集合意义上等价于"可判定" |
| 原始递归 (Primitive recursive) | 函数 | 不使用μ算子的递归函数 |
| 可靠性 (Soundness) | 证明系统 | $\Gamma \vdash \phi \Rightarrow \Gamma \models \phi$ |
| 完备性 (Completeness) | 证明系统 | $\Gamma \models \phi \Rightarrow \Gamma \vdash \phi$ |

### 符号书写规范

1. **数学符号**：使用标准LaTeX表示
2. **引用格式**：使用 `[AuthorYear]` 格式
3. **交叉引用**：使用 `[术语](文档路径)` 格式
4. **定义标记**：首次定义使用粗体，如 **定义 (Definition)**

#### 题目 9.1.1：线段树基础
**难度**：⭐  
**标签**：线段树、基础

**题目描述**：  
实现一个基本的线段树，支持区间求和操作。

**要求**：
1. 支持单点修改：update(pos, val)
2. 支持区间查询：query(l, r) 返回[l,r]区间和
3. 使用数组实现
4. 分析时间复杂度和空间复杂度

**参考答案概要**：
```cpp
struct SegTree {
    vector<long long> tree;
    int n;
    
    SegTree(vector<int>& a) {
        n = a.size();
        tree.resize(4 * n);
        build(1, 0, n-1, a);
    }
    
    void build(int node, int l, int r, vector<int>& a) {
        if(l == r) tree[node] = a[l];
        else {
            int mid = (l + r) / 2;
            build(2*node, l, mid, a);
            build(2*node+1, mid+1, r, a);
            tree[node] = tree[2*node] + tree[2*node+1];
        }
    }
    
    void update(int node, int l, int r, int pos, int val);
    long long query(int node, int l, int r, int ql, int qr);
};
```

#### 题目 9.1.2：线段树区间修改（懒惰标记）
**难度**：⭐⭐  
**标签**：线段树、懒惰标记、区间修改

**题目描述**：  
实现支持区间修改的线段树：

1. 区间加：add(l, r, val) — 将[l,r]区间内每个元素加上val
2. 区间求和：query(l, r)

**要求**：
- 使用懒惰标记（Lazy Propagation）优化
- 分析时间复杂度

**提示**：
```cpp
struct SegTree {
    vector<long long> tree, lazy;
    
    void push_down(int node, int l, int r) {
        if(lazy[node] != 0) {
            int mid = (l + r) / 2;
            apply(2*node, l, mid, lazy[node]);
            apply(2*node+1, mid+1, r, lazy[node]);
            lazy[node] = 0;
        }
    }
    
    void apply(int node, int l, int r, long long val) {
        tree[node] += (r - l + 1) * val;
        lazy[node] += val;
    }
};
```

#### 题目 9.1.4：线段树维护最大子段和
**难度**：⭐⭐⭐  
**标签**：线段树、最大子段和、信息合并

**题目描述**：  
实现维护最大子段和的线段树。

每个节点需要维护：
- sum：区间和
- maxSum：最大子段和
- maxPrefix：最大前缀和
- maxSuffix：最大后缀和

**合并方式**：
```
maxSum = max(left.maxSum, right.maxSum, 
             left.maxSuffix + right.maxPrefix)
maxPrefix = max(left.maxPrefix, left.sum + right.maxPrefix)
maxSuffix = max(right.maxSuffix, right.sum + left.maxSuffix)
```

#### 题目 9.2.1：树状数组基础
**难度**：⭐  
**标签**：树状数组、前缀和

**题目描述**：  
实现树状数组，支持：
1. 单点修改：add(pos, delta)
2. 前缀和查询：sum(pos)

**核心代码**：
```cpp
struct BIT {
    vector<long long> tree;
    int n;
    
    void add(int i, long long delta) {
        for(; i <= n; i += i & -i) tree[i] += delta;
    }
    
    long long sum(int i) {
        long long res = 0;
        for(; i > 0; i -= i & -i) res += tree[i];
        return res;
    }
    
    long long rangeSum(int l, int r) {
        return sum(r) - sum(l - 1);
    }
};
```

#### 题目 9.2.3：树状数组区间修改区间查询
**难度**：⭐⭐⭐  
**标签**：树状数组、前缀和技巧

**题目描述**：

**公式推导**：
设原数组为a，差分数组为d，则：
```
a[i] = Σd[j] (j=1 to i)
sum[i] = Σa[j] (j=1 to i) = Σ(i-j+1)*d[j]
       = (i+1)*Σd[j] - Σj*d[j]
```

需要维护两个树状数组：一个存d[j]，一个存j*d[j]。

#### 题目 9.4.1：并查集基础实现
**难度**：⭐  
**标签**：并查集、路径压缩、按秩合并

**题目描述**：  
实现带路径压缩和按秩合并的并查集。

**代码**：
```cpp
struct DSU {
    vector<int> parent, rank;
    
    DSU(int n) {
        parent.resize(n);
        rank.resize(n);
        for(int i = 0; i < n; i++) parent[i] = i;
    }
    
    int find(int x) {
        if(parent[x] != x) parent[x] = find(parent[x]);
        return parent[x];
    }
    
    void unite(int x, int y) {
        x = find(x), y = find(y);
        if(x == y) return;
        if(rank[x] < rank[y]) swap(x, y);
        parent[y] = x;
        if(rank[x] == rank[y]) rank[x]++;
    }
};
```

## 附录：高级数据结构速查表

### 时间复杂度对比

| 数据结构 | 单点修改 | 区间查询 | 区间修改 | 空间 |
|---------|---------|---------|---------|------|
| 线段树 | O(log n) | O(log n) | O(log n) | O(n) |
| 树状数组 | O(log n) | O(log n) | O(log n)* | O(n) |
| 分块 | O(√n) | O(√n) | O(√n) | O(n) |
| Treap | O(log n) | - | O(log n)** | O(n) |

*需要技巧  **需要懒惰标记

### 适用场景

| 问题类型 | 推荐数据结构 |
|---------|------------|
| 区间和/最值查询 | 线段树、树状数组 |
| 区间修改+查询 | 线段树（带lazy） |
| 逆序对统计 | 树状数组、线段树 |
| 第k小/排名查询 | 权值线段树、平衡树 |
| 历史版本查询 | 主席树、可持久化Trie |
| 动态连通性 | 并查集、LCT |
| 字符串前缀匹配 | Trie、AC自动机 |
| 区间序列操作 | 无旋Treap、Splay |

### 经典问题对应

| 问题 | 解决方案 |
|------|---------|
| 区间第k小 | 主席树（可持久化权值线段树）|
| 动态区间和 | 树状数组/线段树 |
| 区间最大子段和 | 线段树维护4个值 |
| 区间众数 | 线段树+摩尔投票（不支持修改）|
| 区间gcd | 线段树/树状数组 |
| 最大异或对 | 01-Trie |
| 多模式匹配 | AC自动机 |
| 最小生成树 | Kruskal+并查集 |
| 动态MST | LCT |

## 📚 习题系统概述

本文档为《FormalAlgorithm》项目配套习题系统的总纲，旨在通过**1900+道精心设计的习题**，帮助学习者从理论理解到实践应用，全面掌握算法与复杂性理论的核心知识。

### 设计目标

- **全面覆盖**：涵盖基础算法、高级算法、复杂性理论、类型与形式化方法四大领域
- **难度递进**：从基础概念到研究前沿，五级难度体系循序渐进
- **类型多样**：六种习题类型培养不同层次的思维能力
- **理论与实践结合**：既重视形式化证明，也关注算法实现

## 📝 习题类型体系

### 1. 理解型 (Comprehension)

- **目标**：检验对基本概念、定义、定理的理解
- **形式**：概念辨析、判断正误、简答说明
- **示例**："解释大O符号的数学定义"

### 2. 应用型 (Application)

- **目标**：将理论应用于具体问题
- **形式**：手工演算、实例分析、算法执行追踪
- **示例**："用Dijkstra算法求下图的最短路径"

### 3. 证明型 (Proof)

- **目标**：培养形式化推理能力
- **形式**：数学证明、正确性证明、复杂度分析
- **示例**："证明归并排序的时间复杂度为O(n log n)"

### 4. 设计型 (Design)

- **目标**：创造性地设计算法或数据结构
- **形式**：算法设计、数据结构构造、问题转化
- **示例**："设计一个支持O(1)查询第k小元素的数据结构"

### 5. 实现型 (Implementation)

- **目标**：将算法转化为可执行代码
- **形式**：编程实现、代码补全、性能优化
- **示例**："实现红黑树的插入操作，并处理所有情况"

### 6. 研究型 (Research)

- **目标**：探索前沿问题，培养科研能力
- **形式**：文献调研、开放问题、实验研究
- **示例**："调研当前最优的矩阵乘法算法，分析其适用场景"

## 📖 使用指南

### 学习路径建议

```
阶段一：基础夯实 (L1-L2)
├── 渐近分析 → 排序算法 → 基础数据结构
├── 树结构 → 哈希表 → 并查集
└── 图论基础 → 贪心算法 → 分治策略

阶段二：核心能力 (L2-L3)
├── 动态规划 → 摊还分析 → 字符串算法
├── 网络流 → 线性规划 → 计算几何
└── FFT → 高级数据结构 → 自动机理论

阶段三：理论深入 (L3-L4)
├── 可计算性 → P与NP → 复杂性层级
├── 类型论 → λ演算 → 依赖类型
└── 证明助手 → 程序验证 → 形式化语义

阶段四：前沿探索 (L4-L5)
├── 在线算法 → 近似算法 → 随机算法
├── 流算法 → 子模优化 → 多臂老虎机
└── 量子算法 → 同伦类型论 → 定理证明
```

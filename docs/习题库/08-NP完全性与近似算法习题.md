**题目描述**：  
给定顶点覆盖的2-近似算法：

```
Approx-VERTEX-COVER(G):
    C = ∅
    E' = G.E
    while E' ≠ ∅:
        任取(u,v) ∈ E'
        C = C ∪ {u, v}
        从E'中删除所有与u或v关联的边
    return C
```

1. 证明该算法是2-近似的
2. 分析时间复杂度
3. 给出紧例子（即近似比接近2的例子）

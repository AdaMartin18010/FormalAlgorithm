## 一、时间复杂度分析 (30题)

### 入门 (10题)

**习题 1.1** [分析]

```
算法A: T(n) = 2T(n/2) + n
算法B: T(n) = T(n-1) + n

问:
1. 分别用主定理和展开法求解A的时间复杂度
2. 用迭代法求解B的时间复杂度
3. 当n很大时，哪个算法更快？
```

**习题 1.2** [分类]
将以下复杂度按增长速率排序：

- $O(n!)$, $O(2^n)$, $O(n^3)$, $O(n \log n)$, $O(\log n)$, $O(n^{\log n})$

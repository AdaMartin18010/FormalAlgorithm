#### 题目 4.1.1：爬楼梯

**难度**：⭐
**标签**：线性DP、递推

**题目描述**：
有 n 级台阶，每次可跨 1 级或 2 级，求到达第 n 级的方案数。

**约束**：
1 ≤ n ≤ 45

**状态定义**：
dp[i] = 到达第 i 级的方案数

**状态转移**：
dp[i] = dp[i-1] + dp[i-2]

**参考答案概要**：

```cpp
dp[0] = 1, dp[1] = 1;
for(int i = 2; i <= n; i++)
    dp[i] = dp[i-1] + dp[i-2];
// 空间优化：滚动数组
int a = 1, b = 1, c;
for(int i = 2; i <= n; i++) {
    c = a + b; a = b; b = c;
}
```

#### 题目 4.1.3：最长递增子序列（LIS）

**难度**：⭐⭐
**标签**：线性DP、二分优化

**题目描述**：
求数组中最长严格递增子序列的长度。

**约束**：
n ≤ 2500（O(n²)）或 n ≤ 10⁵（O(n log n)）

**参考答案概要**：

```cpp
// O(n²)版本
for(int i = 0; i < n; i++) {
    dp[i] = 1;
    for(int j = 0; j < i; j++)
        if(a[j] < a[i]) dp[i] = max(dp[i], dp[j]+1);
}
// O(n log n)：维护单调数组
tail[1] = a[0]; len = 1;
for(int i = 1; i < n; i++) {
    if(a[i] > tail[len]) tail[++len] = a[i];
    else *lower_bound(tail+1, tail+len+1, a[i]) = a[i];
}
```

#### 题目 4.2.1：矩阵链乘法

**难度**：⭐⭐
**标签**：区间DP

**题目描述**：
给定 n 个矩阵的链，求最小化标量乘法次数的括号化方案。

**状态定义**：
dp[i][j] = 计算矩阵链 i 到 j 的最小乘法次数

**状态转移**：

```cpp
dp[i][j] = min(dp[i][k] + dp[k+1][j] + p[i-1]*p[k]*p[j])
           for k in [i, j)
```

**参考答案概要**：

```cpp
for(int len = 2; len <= n; len++) {
    for(int i = 1; i + len - 1 <= n; i++) {
        int j = i + len - 1;
        dp[i][j] = INF;
        for(int k = i; k < j; k++) {
            dp[i][j] = min(dp[i][j],
                dp[i][k] + dp[k+1][j] + p[i-1]*p[k]*p[j]);
        }
    }
}
```

### 入门题（4题）

#### 题目 4.4.1：旅行商问题（TSP基础）

**难度**：⭐⭐
**标签**：状压DP

**题目描述**：
n 个城市，从城市 0 出发，经过所有城市恰好一次回到起点，求最短路程。

**约束**：
n ≤ 20

**状态定义**：
dp[mask][i] = 已访问城市集合 mask，当前在城市 i 的最短距离

**状态转移**：

```cpp
dp[mask][i] = min(dp[mask^(1<<i)][j] + dist[j][i])
              for all j in mask, j != i
```

**参考答案概要**：

```cpp
int n; vector<vector<int>> dist;
vector<vector<int>> dp(1<<n, vector<int>(n, INF));
dp[1][0] = 0;  // 从0出发
for(int mask = 1; mask < (1<<n); mask++) {
    for(int i = 0; i < n; i++) if(mask & (1<<i)) {
        for(int j = 0; j < n; j++) if(mask & (1<<j) && i != j) {
            dp[mask][i] = min(dp[mask][i],
                dp[mask^(1<<i)][j] + dist[j][i]);
        }
    }
}
// 答案：min(dp[(1<<n)-1][i] + dist[i][0])
```

## 附录：DP技巧与模板

### 线性DP模板

```cpp
// 一维
vector<int> dp(n);
dp[0] = init;
for(int i = 1; i < n; i++) {
    dp[i] = /* 基于 dp[i-1], dp[i-2] 等的状态转移 */;
}

// 二维
vector<vector<int>> dp(n, vector<int>(m));
for(int i = 0; i < n; i++) {
    for(int j = 0; j < m; j++) {
        dp[i][j] = /* 状态转移 */;
    }
}
```

### 区间DP模板

```cpp
// 枚举长度
for(int len = 2; len <= n; len++) {
    for(int i = 1; i + len - 1 <= n; i++) {
        int j = i + len - 1;
        dp[i][j] = INF;
        for(int k = i; k < j; k++) {
            dp[i][j] = min(dp[i][j], dp[i][k] + dp[k+1][j] + cost);
        }
    }
}
```

### 树形DP模板

```cpp
void dfs(int u, int fa) {
    dp[u][0] = 0; dp[u][1] = val[u];
    for(int v : adj[u]) if(v != fa) {
        dfs(v, u);
        dp[u][0] += max(dp[v][0], dp[v][1]);
        dp[u][1] += dp[v][0];
    }
}
```

### 状压DP模板

```cpp
int dp[1<<MAXN][MAXN];
memset(dp, 0x3f, sizeof(dp));
dp[1][0] = 0;
for(int mask = 1; mask < (1<<n); mask++) {
    for(int i = 0; i < n; i++) if(mask >> i & 1) {
        for(int j = 0; j < n; j++) if(!(mask >> j & 1)) {
            dp[mask|(1<<j)][j] = min(dp[mask|(1<<j)][j],
                                     dp[mask][i] + dist[i][j]);
        }
    }
}
```

### 常见优化技巧

1. **滚动数组**：当只依赖前一行/前几行时，用 %2 或 &1 优化空间
2. **单调队列优化**：用于形如 dp[i] = max(dp[j] + f(i,j)) 且 f 有单调性的转移
3. **斜率优化**：用于优化形如 dp[i] = min(dp[j] + (i-j)²) 的转移
4. **四边形不等式优化**：用于区间DP的决策单调性优化
5. **预处理**：预先计算常用的值，如前缀和、最值等

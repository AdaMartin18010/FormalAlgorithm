#### 题目 3.1.1：迷宫探索
**难度**：⭐  
**标签**：DFS、BFS

**题目描述**：  
给定一个 n×m 的迷宫，`.` 表示可通行，`#` 表示障碍物。从起点 (1,1) 出发，判断能否到达终点 (n,m)。

**输入格式**：  
第一行两个整数 n, m (1 ≤ n,m ≤ 100)  
接下来 n 行，每行 m 个字符

**输出格式**：  
能到达输出 "Yes"，否则输出 "No"

**提示**：  
- 使用 DFS 递归或 BFS 队列实现
- 注意标记访问过的位置避免重复
- 四个方向：上、下、左、右

**参考答案概要**：  
```cpp
// DFS版本
dfs(x, y):
    if 到达终点: return true
    标记vis[x][y] = true
    for 四个方向:
        if 新位置合法且未访问:
            if dfs(nx, ny): return true
    return false
```

**标签**：DFS、连通块

**题目描述**：  
给定一个由 '1'（陆地）和 '0'（水）组成的二维网格，计算岛屿的数量。岛屿被水包围，且水平或垂直相邻的陆地连接形成岛屿。

**输入格式**：

**输出格式**：

**提示**：

- 经典连通块计数问题

**参考答案概要**：  
```cpp

    for j in [0,m):
        if grid[i][j]=='1' and !vis[i][j]:
            ans++, dfs(i,j)  // 标记整个连通块
```

#### 题目 3.1.3：迷宫最短路（基础）
**难度**：⭐  
**标签**：BFS、最短路

**题目描述**：  
在 n×m 迷宫中，求从起点到终点的最短步数。

**输入/输出**：  
标准迷宫格式，输出最短步数（无法到达输出 -1）

**提示**：  
- BFS天然适合求无权图最短路
- 使用队列，dist数组记录距离

**参考答案概要**：  
```cpp
queue<pair<int,int>> q;
dist[sx][sy] = 0, q.push({sx, sy});
while(!q.empty()):
    u = q.front(); q.pop();
    for 四个邻居v:
        if v合法且未访问:
            dist[v] = dist[u] + 1
            q.push(v)
```

## 二、最短路径算法 - 25题

#### 题目 3.2.1：单源最短路（基础版）
**难度**：⭐  
**标签**：Dijkstra、模板

**题目描述**：  
给定 n 个点 m 条边的有向图，边权非负，求从起点 s 到所有点的最短距离。

**约束**：  
n ≤ 1000, m ≤ 10000

**提示**：  
- 标准Dijkstra模板题
- 可用朴素 O(n²) 或堆优化 O(m log n)

**参考答案概要**：  
```cpp
// Dijkstra堆优化
priority_queue<pii, vector<pii>, greater<pii>> pq;
dist[s] = 0, pq.push({0, s});
while(!pq.empty()):
    [d, u] = pq.top(); pq.pop();
    if(d > dist[u]) continue;
    for [v, w] in adj[u]:
        if(dist[u] + w < dist[v]):
            dist[v] = dist[u] + w
            pq.push({dist[v], v})
```

**难度**：⭐  
**标签**：Kruskal、Prim

**题目描述**：

```cpp
// Kruskal
sort(edges)  // 按权值排序
for each edge (u,v,w):
    if(find(u) != find(v)):
        union(u, v)
        ans += w
        cnt++
if(cnt < n-1): 不连通
```

## 附录：算法模板速查

### Dijkstra（堆优化）
```cpp
vector<ll> dijkstra(int s) {
    vector<ll> dist(n, INF);
    priority_queue<pii, vector<pii>, greater<pii>> pq;
    dist[s] = 0; pq.push({0, s});
    while(!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if(d > dist[u]) continue;
        for(auto [v, w] : adj[u]) {
            if(dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                pq.push({dist[v], v});
            }
        }
    }
    return dist;
}
```

### SPFA
```cpp
bool spfa(int s) {
    vector<ll> dist(n, INF);
    vector<int> cnt(n, 0);
    vector<bool> inq(n, false);
    queue<int> q;
    dist[s] = 0; q.push(s); inq[s] = true;
    while(!q.empty()) {
        int u = q.front(); q.pop(); inq[u] = false;
        for(auto [v, w] : adj[u]) {
            if(dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                cnt[v] = cnt[u] + 1;
                if(cnt[v] >= n) return false; // 负环
                if(!inq[v]) { q.push(v); inq[v] = true; }
            }
        }
    }
    return true;
}
```

### Kruskal
```cpp
struct Edge { int u, v, w; };
vector<Edge> edges;
int find(int x) { return fa[x] == x ? x : fa[x] = find(fa[x]); }
ll kruskal() {
    sort(edges.begin(), edges.end(), [](a,b){return a.w<b.w;});
    ll ans = 0; int cnt = 0;
    for(auto& e : edges) {
        int fu = find(e.u), fv = find(e.v);
        if(fu != fv) {
            fa[fu] = fv;
            ans += e.w;
            if(++cnt == n-1) break;
        }
    }
    return cnt == n-1 ? ans : -1;
}
```

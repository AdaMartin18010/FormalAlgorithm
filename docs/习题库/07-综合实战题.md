#### 题目 7.1.1：多算法综合应用（基础）
**难度**：⭐⭐  
**标签**：综合、基础算法

**题目描述**：  
给定一个综合性问题，需要组合排序、二分查找、贪心等基础算法解决。

**示例场景**：  
一个在线考试系统，需要：
1. 按分数排序（快排/归并）
2. 查询某分数的排名（二分）
3. 安排考试时间使冲突最少（贪心）

**参考答案概要**：  
```cpp
// 综合应用示例
struct Exam {
    int id, score, duration;
};

void solve(vector<Exam> &exams) {
    // 1. 按分数降序排序
    sort(exams.begin(), exams.end(), [](a,b){return a.score>b.score;});
    
    // 2. 查询某分数的排名
    auto getRank = [&](int score) {
        auto it = lower_bound(exams.begin(), exams.end(), score,
            [](const Exam &e, int s){return e.score > s;});
        return it - exams.begin() + 1;
    };
    
    // 3. 贪心安排考试时间（区间调度）
    sort(exams.begin(), exams.end(), [](a,b){return a.duration<b.duration;});
    int cnt = 0, lastEnd = 0;
    for(auto &e : exams) {
        if(e.duration >= lastEnd) {
            cnt++;
            lastEnd = e.duration;
        }
    }
}
```

#### 题目 7.2.1：设计LRU缓存
**难度**：⭐⭐  
**标签**：设计、哈希表、链表

**题目描述**：  
设计一个最近最少使用（LRU）缓存机制，支持 get 和 put 操作。

**要求**：  
- get(key)：获取key对应的值，不存在返回-1
- put(key, value)：插入或更新，满时淘汰最久未使用的
- 两个操作时间复杂度均为 O(1)

**设计方案**：  
```cpp
class LRUCache {
    int capacity;
    list<pair<int, int>> cache;  // (key, value)
    unordered_map<int, list<pair<int,int>>::iterator> map;
    
public:
    LRUCache(int cap) : capacity(cap) {}
    
    int get(int key) {
        if(!map.count(key)) return -1;
        auto it = map[key];
        int val = it->second;
        cache.erase(it);
        cache.push_front({key, val});
        map[key] = cache.begin();
        return val;
    }
    
    void put(int key, int value) {
        if(map.count(key)) {
            cache.erase(map[key]);
        } else if(cache.size() == capacity) {
            auto last = cache.back();
            map.erase(last.first);
            cache.pop_back();
        }
        cache.push_front({key, value});
        map[key] = cache.begin();
    }
};
```

## 附录：综合题解题策略

### 竞赛解题流程
```
1. 审题与理解
   └─ 明确输入输出格式
   └─ 识别约束条件
   └─ 提取关键信息

2. 算法选择
   └─ 根据复杂度反推算法
   └─ 考虑多种可能性
   └─ 评估实现难度

3. 设计与实现
   └─ 先写伪代码
   └─ 模块化设计
   └─ 注意边界条件

4. 测试与调试
   └─ 样例测试
   └─ 边界测试
   └─ 随机测试
   └─ 压力测试
```

### 系统设计原则

#### 1. 基本原则
- **单一职责**：每个组件只负责一件事
- **开闭原则**：对扩展开放，对修改关闭
- **依赖倒置**：依赖抽象而非具体实现

#### 2. 可扩展性设计
- **水平扩展**：支持多实例部署
- **数据分片**：将数据分布到多个节点
- **缓存策略**：多级缓存减少压力

#### 3. 高可用设计
- **冗余备份**：关键组件多副本
- **故障转移**：自动切换到备份
- **降级策略**：服务不可用时提供简化功能

#### 4. 性能优化
- **读写分离**：读多写少场景
- **异步处理**：非关键路径异步化
- **批处理**：减少IO次数

int binarySearch(int left, int right) {
    int ans = -1;
    while(left <= right) {
        int mid = left + (right - left) / 2;
        if(check(mid)) {
            ans = mid;
            right = mid - 1;  // 找左边界
        } else {
            left = mid + 1;
        }
    }

#### 整体二分模板
```cpp
void solve(int l, int r, vector<Query> &qs) {
    if(qs.empty()) return;
    if(l == r) { /* 处理答案为l的查询 */ return; }
    int mid = (l + r) / 2;
    // 处理左半部分的修改
    for(auto &q : qs) {
        if(/* q在左半部分 */) leftQs.push_back(q);
        else rightQs.push_back(q);
    }
    solve(l, mid, leftQs);
    // 撤销修改
    solve(mid+1, r, rightQs);
}
```

void cdq(int l, int r) {
    if(l == r) return;
    int mid = (l + r) / 2;
    cdq(l, mid); cdq(mid+1, r);
    // 处理左半部分对右半部分的影响
    sort(a+l, a+mid+1, cmpy);
    sort(a+mid+1, a+r+1, cmpy);
    int j = l;
    for(int i = mid+1; i <= r; i++) {
        while(j <= mid && a[j].y <= a[i].y) {
            bit.add(a[j].z, a[j].w);
            j++;
        }
        a[i].ans += bit.query(a[i].z);
    }
    // 清空树状数组
    for(int k = l; k < j; k++) bit.clear(a[k].z);
}

```markdown

如类型理论教材 [Pierce2002] 所述...
参见同伦类型论 [[HoTTBook2013]]...
```

### 查找引用

1. **按主题查找**: 查看 `docs/引用数据库-扩展-2025.md` 对应章节
2. **按ID查找**: 使用reference_id在文件中搜索
3. **按关键词查找**: 使用tags字段过滤

### 生成参考文献列表

```yaml

- quality: "classic"
  
# 提取特定主题引用

```

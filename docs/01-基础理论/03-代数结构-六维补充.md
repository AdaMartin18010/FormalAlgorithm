## 一、概念定义 (Concept Definition)

### 1.1 群 (Group)

**定义 1.1.1** (二元运算 / Binary Operation)

集合 $G$ 上的二元运算是一个函数：

$$\cdot: G \times G \rightarrow G$$

记作 $a \cdot b$ 或简写为 $ab$。

**定义 1.1.2** (群 / Group)

群是一个二元组 $(G, \cdot)$，其中 $G$ 是集合，$\cdot$ 是二元运算，满足：

1. **结合律**：$\forall a, b, c \in G: (a \cdot b) \cdot c = a \cdot (b \cdot c)$
2. **单位元**：$\exists e \in G, \forall a \in G: e \cdot a = a \cdot e = a$
3. **逆元**：$\forall a \in G, \exists a^{-1} \in G: a \cdot a^{-1} = a^{-1} \cdot a = e$

**定义 1.1.3** (阿贝尔群 / Abelian Group)

若群还满足交换律：

$$\forall a, b \in G: a \cdot b = b \cdot a$$

则称为阿贝尔群（或交换群）。

**定义 1.1.4** (子群 / Subgroup)

设 $(G, \cdot)$ 是群，$H \subseteq G$。若 $(H, \cdot)$ 也是群，则称 $H$ 是 $G$ 的子群，记作 $H \leq G$。

**子群判定定理**：$H \leq G$ 当且仅当：

1. $H \neq \emptyset$
2. $\forall a, b \in H: a \cdot b^{-1} \in H$

**定义 1.1.5** (正规子群 / Normal Subgroup)

子群 $N \leq G$ 是正规的，当且仅当：

$$\forall g \in G, \forall n \in N: gng^{-1} \in N$$

或等价地：$\forall g \in G: gN = Ng$

记作 $N \trianglelefteq G$。

**定义 1.1.6** (商群 / Quotient Group)

设 $N \trianglelefteq G$，定义商群：

$$G/N = \{gN \mid g \in G\}$$

运算：$(aN) \cdot (bN) = (ab)N$

**定义 1.1.7** (群同态 / Group Homomorphism)

映射 $\phi: G \rightarrow H$ 是群同态，当且仅当：

$$\forall a, b \in G: \phi(a \cdot b) = \phi(a) \cdot \phi(b)$$

**定义 1.1.8** (同态核与像)

- **核**：$\ker(\phi) = \{g \in G \mid \phi(g) = e_H\}$
- **像**：$\text{im}(\phi) = \{\phi(g) \mid g \in G\}$

### 1.2 环 (Ring)

**定义 1.2.1** (环 / Ring)

环是一个三元组 $(R, +, \cdot)$，其中：

1. $(R, +)$ 是阿贝尔群
2. $(R, \cdot)$ 是半群（结合律）
3. 乘法对加法满足分配律：
   - $a \cdot (b + c) = a \cdot b + a \cdot c$
   - $(a + b) \cdot c = a \cdot c + b \cdot c$

**定义 1.2.2** (含幺环 / Ring with Unity)

若存在乘法单位元 $1 \in R$：

$$\forall a \in R: 1 \cdot a = a \cdot 1 = a$$

**定义 1.2.3** (交换环 / Commutative Ring)

若乘法满足交换律：

$$\forall a, b \in R: a \cdot b = b \cdot a$$

**定义 1.2.4** (整环 / Integral Domain)

交换环 $R$ 是整环，若满足：

1. $1 \neq 0$
2. 无零因子：$a \cdot b = 0 \Rightarrow a = 0$ 或 $b = 0$

**定义 1.2.5** (理想 / Ideal)

子集 $I \subseteq R$ 是理想，当且仅当：

1. $(I, +)$ 是 $(R, +)$ 的子群
2. 吸收性：$\forall r \in R, \forall i \in I: r \cdot i \in I$ 且 $i \cdot r \in I$

**定义 1.2.6** (主理想 / Principal Ideal)

由单个元素生成的理想：

$$(a) = \{r \cdot a \mid r \in R\}$$

若环中所有理想都是主理想，则称为主理想环。

**定义 1.2.7** (主理想整环 / PID)

整环且是主理想环，记作 PID（Principal Ideal Domain）。

### 1.3 域 (Field)

**定义 1.3.1** (域 / Field)

域是三元组 $(F, +, \cdot)$，满足：

1. $(F, +)$ 是阿贝尔群
2. $(F \setminus \{0\}, \cdot)$ 是阿贝尔群
3. 乘法对加法满足分配律

**等价定义**：域是交换环，且所有非零元都有乘法逆元。

**定义 1.3.2** (域的特征 / Characteristic)

域 $F$ 的特征是最小的正整数 $n$ 使得：

$$\underbrace{1 + 1 + ... + 1}_{n \text{个}} = 0$$

若不存在这样的 $n$，则特征为 0，记作 $\text{char}(F)$。

**定义 1.3.3** (域扩张 / Field Extension)

若 $F \subseteq E$ 都是域，则称 $E$ 是 $F$ 的扩域，记作 $E/F$。

$E$ 可视为 $F$ 上的向量空间，扩张次数：

$$[E:F] = \dim_F(E)$$

**定义 1.3.4** (代数元与超越元)

设 $\alpha \in E$：

- **代数元**：存在非零多项式 $p(x) \in F[x]$ 使 $p(\alpha) = 0$
- **超越元**：不存在这样的多项式

**定义 1.3.5** (代数闭包 / Algebraic Closure)

域 $F$ 的代数闭包 $\overline{F}$ 是包含 $F$ 的最小代数闭域：

$$\forall p(x) \in \overline{F}[x], \deg(p) > 0: \exists \alpha \in \overline{F}: p(\alpha) = 0$$

**定义 1.3.6** (有限域 / Finite Field)

元素个数有限的域。若 $|F| = q$，则 $q = p^n$（$p$ 为素数）。

记作 $\mathbb{F}_q$ 或 $\text{GF}(q)$（Galois Field）。

### 1.4 格 (Lattice)

**定义 1.4.1** (偏序集 / Poset)

偏序集 $(P, \leq)$ 满足：

1. 自反性：$a \leq a$
2. 反对称性：$a \leq b \land b \leq a \Rightarrow a = b$
3. 传递性：$a \leq b \land b \leq c \Rightarrow a \leq c$

**定义 1.4.2** (格 / Lattice)

偏序集 $(L, \leq)$ 是格，若任意两元素都有：

- **最小上界**（并）：$a \lor b = \sup\{a, b\}$
- **最大下界**（交）：$a \land b = \inf\{a, b\}$

**代数定义**：$(L, \lor, \land)$ 满足：

1. 交换律、结合律、吸收律
2. 幂等律可由吸收律导出

**定义 1.4.3** (完备格 / Complete Lattice)

任意子集都有最小上界和最大下界。

**定义 1.4.4** (分配格 / Distributive Lattice)

满足分配律：

$$a \land (b \lor c) = (a \land b) \lor (a \land c)$$

**定义 1.4.5** (布尔代数 / Boolean Algebra)

有补分配格，即：

- 有界：存在 $0, 1$
- 有补：$\forall a, \exists a': a \land a' = 0, a \lor a' = 1$
- 分配律成立

**定义 1.4.6** (海廷代数 / Heyting Algebra)

存在相对伪补的格，即对任意 $a, b$，存在最大的 $x$ 使得：

$$a \land x \leq b$$

记作 $a \rightarrow b$。

## 二、属性 (Properties)

### 2.1 群的基本性质

| 性质 | 公式 | 说明 |
|------|------|------|
| **单位元唯一性** | $e_1 = e_1 \cdot e_2 = e_2$ | 群有唯一单位元 |
| **逆元唯一性** | $a^{-1}$ 唯一 | 每个元素有唯一逆元 |
| **逆元的逆** | $(a^{-1})^{-1} = a$ | 双重逆元还原 |
| **穿袜脱袜** | $(ab)^{-1} = b^{-1}a^{-1}$ | 乘积的逆元 |
| **消去律** | $ab = ac \Rightarrow b = c$ | 左右可消去 |

### 2.2 拉格朗日定理相关

| 性质 | 公式 | 条件 |
|------|------|------|
| **拉格朗日定理** | $H \leq G \Rightarrow |H| \mid |G|$ | $G$ 有限群 |
| **指数定义** | $[G:H] = |G|/|H|$ | $H \leq G$ |
| **元素阶整除** | $\text{ord}(g) \mid |G|$ | $G$ 有限群 |

### 2.3 环的性质

| 性质 | 说明 |
|------|------|
| **零乘性质** | $a \cdot 0 = 0 \cdot a = 0$ |
| **负元乘法** | $(-a)b = a(-b) = -(ab)$ |
| **双重负元** | $(-a)(-b) = ab$ |
| **无零因子** | 整环中：$ab = 0 \Rightarrow a = 0 \lor b = 0$ |

## 四、解释 (Explanation)

### 4.1 群的直观理解

**对称性的代数化**

群是"对称性"的代数抽象：

- **几何对称**：正方形的旋转和反射形成二面体群 $D_4$
- **代数对称**：方程根的对称性 → 伽罗瓦群
- **物理对称**：守恒律与诺特定理

**群的四个公理含义**：

1. **封闭性**：操作结果仍在集合内
2. **结合律**：操作顺序不影响结果
3. **单位元**：存在"不操作"
4. **逆元**：每个操作都可撤销

### 4.2 环的直观理解

**两种运算的协调**

环同时具有"加法"和"乘法"两种结构：

- 加法群：可逆、交换
- 乘法半群：不一定可逆、不一定交换

**整数环 $\mathbb{Z}$ 是原型**：

- 可以相加、相乘
- 不能随意相除（非域）
- 有零因子？不，是整环

**理想的直观**：

- 类似于群的正规子群
- "吸收"环中任意元素乘法
- 商环类似于模去某种等价关系

### 4.3 域的直观理解

**可进行四则运算的集合**

域是可以自由进行加减乘除（除零外）的代数结构：

- 有理数域 $\mathbb{Q}$
- 实数域 $\mathbb{R}$
- 复数域 $\mathbb{C}$
- 有限域 $\mathbb{F}_p$

**域扩张的直观**：

- $\mathbb{R}/\mathbb{Q}$：添加所有无理数
- $\mathbb{C}/\mathbb{R}$：添加 $\sqrt{-1}$
- $\mathbb{Q}(\sqrt{2})/\mathbb{Q}$：添加 $\sqrt{2}$ 生成的扩域

### 4.4 格的直观理解

**偏序关系的代数化**

格将偏序关系转化为代数运算：

- $a \land b$：$a$ 和 $b$ 的"最大公共部分"
- $a \lor b$：$a$ 和 $b$ 的"最小公共包含"

**布尔代数 = 集合代数**：

- $\land$ 对应集合交
- $\lor$ 对应集合并
- $'$ 对应集合补

## 五、论证 (Argumentation)

### 5.1 群论基本定理的重要性

**同态基本定理**

$$G/\ker(\phi) \cong \text{im}(\phi)$$

这一定理的重要性：

1. 将任意同态分解为满射、同构、单射的复合
2. 正规子群完全刻画了群的同态像
3. 是分类群的重要工具

### 5.2 有限域的存在性与唯一性

**Wedderburn小定理**

有限除环必为域（乘法交换）。

**有限域结构定理**

对每个素数幂 $q = p^n$：

1. 存在唯一（同构意义下）的 $q$ 元域 $\mathbb{F}_q$
2. $\mathbb{F}_q$ 是多项式 $x^q - x$ 在 $\mathbb{F}_p$ 上的分裂域
3. $\mathbb{F}_q^\times$ 是 $q-1$ 阶循环群

**应用**：

- 编码理论（Reed-Solomon码）
- 密码学（AES使用 $\mathbb{F}_{2^8}$）

### 5.3 代数闭包的存在性

**Steinitz定理**

任意域 $F$ 都有代数闭包 $\overline{F}$，且在 $F$-同构意义下唯一。

**构造思路**：

1. 枚举 $F[x]$ 中所有多项式
2. 逐步添加根构造扩域塔
3. 取并集得到代数闭包

## 六、形式证明 (Formal Proofs)

### 6.1 拉格朗日定理证明

**定理**：设 $G$ 是有限群，$H \leq G$，则 $|H| \mid |G|$。

**证明**：

**Step 1**：定义陪集

对 $a \in G$，左陪集 $aH = \{ah \mid h \in H\}$

**Step 2**：陪集的性质

1. 任意两个陪集要么相等，要么不相交
   - 若 $aH \cap bH \neq \emptyset$，则存在 $h_1, h_2 \in H$ 使 $ah_1 = bh_2$
   - 故 $a = bh_2h_1^{-1} \in bH$
   - 因此 $aH \subseteq bH$，同理 $bH \subseteq aH$
   - 所以 $aH = bH$

2. 所有陪集的并等于 $G$

**Step 3**：陪集的大小

映射 $f: H \rightarrow aH$，$f(h) = ah$ 是双射：

- 单射：$ah_1 = ah_2 \Rightarrow h_1 = h_2$（消去律）
- 满射：由定义

故 $|aH| = |H|$。

**Step 4**：结论

设 $[G:H] = k$ 为不同左陪集的个数，则：

$$|G| = k \cdot |H|$$

因此 $|H| \mid |G|$。

∎

### 6.2 第一同构定理证明

**定理**：设 $\phi: G \rightarrow H$ 是群同态，则：

$$G/\ker(\phi) \cong \text{im}(\phi)$$

**证明**：

设 $K = \ker(\phi)$，定义：

$$\psi: G/K \rightarrow \text{im}(\phi), \quad \psi(gK) = \phi(g)$$

**Step 1**：良定性

若 $gK = hK$，则 $h^{-1}g \in K$，故：

$$\phi(h^{-1}g) = e \Rightarrow \phi(h)^{-1}\phi(g) = e \Rightarrow \phi(g) = \phi(h)$$

所以 $\psi(gK) = \psi(hK)$。

**Step 2**：同态性

$$\psi(gK \cdot hK) = \psi(ghK) = \phi(gh) = \phi(g)\phi(h) = \psi(gK)\psi(hK)$$

**Step 3**：单射性

$$\psi(gK) = e \Rightarrow \phi(g) = e \Rightarrow g \in K \Rightarrow gK = K$$

核为平凡，故单射。

**Step 4**：满射性

对任意 $y \in \text{im}(\phi)$，存在 $g \in G$ 使 $\phi(g) = y$，故 $\psi(gK) = y$。

∎

### 6.3 有限域乘法群循环性证明

**定理**：$\mathbb{F}_q^\times$ 是循环群。

**证明**：

设 $n = q - 1 = |\mathbb{F}_q^\times|$。

**Step 1**：元素阶的性质

对任意 $a \in \mathbb{F}_q^\times$，由拉格朗日定理，$\text{ord}(a) \mid n$。

**Step 2**：多项式根的限制**

对 $d \mid n$，方程 $x^d - 1 = 0$ 在 $\mathbb{F}_q$ 中至多有 $d$ 个根。

**Step 3**：阶为 $d$ 的元素个数**

设 $\psi(d)$ 为阶恰为 $d$ 的元素个数。

则阶整除 $d$ 的元素满足 $x^d = 1$，故：

$$\sum_{e \mid d} \psi(e) = d$$

**Step 4**：莫比乌斯反演**

由莫比乌斯反演公式：

$$\psi(d) = \sum_{e \mid d} \mu(d/e) \cdot e = \phi(d)$$

其中 $\phi$ 是欧拉函数。

**Step 5**：结论**

特别地，$\psi(n) = \phi(n) > 0$（对 $n \geq 1$）。

因此存在阶为 $n$ 的元素，即生成元。

∎

```rust
        let mut result = [0u32; N];
```

### 7.2 置换群实现

```rust
/// 置换群的元素（对称群 S_n）
#[derive(Clone, Debug)]
pub struct Permutation {
    /// 映射：i -> mapping[i]
    mapping: Vec<usize>,
}

impl Permutation {
    /// 恒等置换
    pub fn identity(n: usize) -> Self {
        Permutation {
            mapping: (0..n).collect(),
        }
    }

    /// 复合置换：self ∘ other
    pub fn compose(&self, other: &Self) -> Self {
        assert_eq!(self.mapping.len(), other.mapping.len());
        Permutation {
            mapping: other.mapping.iter()
                .map(|&i| self.mapping[i])
                .collect(),
        }
    }

    /// 逆置换
    pub fn inverse(&self) -> Self {
        let n = self.mapping.len();
        let mut inv = vec![0; n];
        for (i, &j) in self.mapping.iter().enumerate() {
            inv[j] = i;
        }
        Permutation { mapping: inv }
    }

    /// 轮换表示
    pub fn cycle_decomposition(&self) -> Vec<Vec<usize>> {
        let n = self.mapping.len();
        let mut visited = vec![false; n];
        let mut cycles = Vec::new();

        for i in 0..n {
            if !visited[i] {
                let mut cycle = Vec::new();
                let mut j = i;
                while !visited[j] {
                    visited[j] = true;
                    cycle.push(j);
                    j = self.mapping[j];
                }
                if cycle.len() > 1 {
                    cycles.push(cycle);
                }
            }
        }

        cycles
    }
}
```

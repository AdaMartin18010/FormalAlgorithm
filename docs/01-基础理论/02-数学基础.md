## 1. 集合论基础 / Set Theory Foundations

### 1.1 基本概念 / Basic Concepts

**定义 1.1.1** 集合是不同对象的无序聚集。
**Definition 1.1.1** A set is an unordered collection of distinct objects.

**公理 1.1.1** (外延公理 / Axiom of Extensionality) 两个集合相等当且仅当它们包含相同的元素：
**Axiom 1.1.1** (Axiom of Extensionality) Two sets are equal if and only if they contain the same elements:
$$\forall x \forall y [\forall z(z \in x \leftrightarrow z \in y) \rightarrow x = y]$$

**公理 1.1.2** (空集公理 / Axiom of Empty Set) 存在一个不包含任何元素的集合：
**Axiom 1.1.2** (Axiom of Empty Set) There exists a set that contains no elements:
$$\exists x \forall y (y \notin x)$$

**定义 1.1.2** 空集记作 $\emptyset$。
**Definition 1.1.2** The empty set is denoted by $\emptyset$.

### 1.2 集合运算 / Set Operations

**定义 1.2.1** 并集 / Union:
**Definition 1.2.1** Union:
$$A \cup B = \{x : x \in A \lor x \in B\}$$

**定义 1.2.2** 交集 / Intersection:
**Definition 1.2.2** Intersection:
$$A \cap B = \{x : x \in A \land x \in B\}$$

**定义 1.2.3** 差集 / Set Difference:
**Definition 1.2.3** Set Difference:
$$A \setminus B = \{x : x \in A \land x \notin B\}$$

**定义 1.2.4** 幂集 / Power Set:
**Definition 1.2.4** Power Set:
$$\mathcal{P}(A) = \{X : X \subseteq A\}$$

### 1.3 关系 / Relations

**定义 1.3.1** 二元关系是笛卡尔积的子集：
**Definition 1.3.1** A binary relation is a subset of the Cartesian product:
$$R \subseteq A \times B$$

**定义 1.3.2** 等价关系是满足以下性质的关系：
**Definition 1.3.2** An equivalence relation is a relation that satisfies the following properties:

1. **自反性 / Reflexivity**: $\forall x (xRx)$
2. **对称性 / Symmetry**: $\forall x \forall y (xRy \rightarrow yRx)$
3. **传递性 / Transitivity**: $\forall x \forall y \forall z (xRy \land yRz \rightarrow xRz)$

## 2. 数论基础 / Number Theory Foundations

### 2.1 自然数 / Natural Numbers

**定义 2.1.1** 自然数集 $\mathbb{N}$ 满足皮亚诺公理：
**Definition 2.1.1** The set of natural numbers $\mathbb{N}$ satisfies Peano's axioms:

1. **零公理 / Zero Axiom**: $0 \in \mathbb{N}$
2. **后继公理 / Successor Axiom**: $\forall n \in \mathbb{N} (S(n) \in \mathbb{N})$
3. **零不是后继 / Zero is not a Successor**: $\forall n \in \mathbb{N} (S(n) \neq 0)$
4. **后继唯一性 / Successor Uniqueness**: $\forall m, n \in \mathbb{N} (S(m) = S(n) \rightarrow m = n)$
5. **数学归纳 / Mathematical Induction**: $\forall P \subseteq \mathbb{N} (0 \in P \land \forall n (n \in P \rightarrow S(n) \in P) \rightarrow P = \mathbb{N})$

### 2.2 整数 / Integers

**定义 2.2.1** 整数集 $\mathbb{Z}$ 是自然数集的扩展：
**Definition 2.2.1** The set of integers $\mathbb{Z}$ is an extension of the set of natural numbers:
$$\mathbb{Z} = \mathbb{N} \cup \{-n : n \in \mathbb{N} \setminus \{0\}\}$$

### 2.3 有理数 / Rational Numbers

**定义 2.3.1** 有理数集 $\mathbb{Q}$ 是整数对的等价类：
**Definition 2.3.1** The set of rational numbers $\mathbb{Q}$ is the set of equivalence classes of integer pairs:
$$\mathbb{Q} = \{(a, b) : a, b \in \mathbb{Z}, b \neq 0\}/\sim$$

其中 $(a, b) \sim (c, d)$ 当且仅当 $ad = bc$。
where $(a, b) \sim (c, d)$ if and only if $ad = bc$.

### 2.4 实数 / Real Numbers

**定义 2.4.1** 实数集 $\mathbb{R}$ 是有理数集的完备化，满足戴德金分割性质。
**Definition 2.4.1** The set of real numbers $\mathbb{R}$ is the completion of the set of rational numbers, satisfying the Dedekind cut property.

## 3. 代数结构 / Algebraic Structures

### 3.1 群 / Groups

**定义 3.1.1** 群是一个集合 $G$ 和一个二元运算 $\cdot$，满足以下公理：
**Definition 3.1.1** A group is a set $G$ with a binary operation $\cdot$ that satisfies the following axioms:

1. **结合律 / Associativity**: $\forall a, b, c \in G ((a \cdot b) \cdot c = a \cdot (b \cdot c))$
2. **单位元 / Identity**: $\exists e \in G \forall a \in G (e \cdot a = a \cdot e = a)$
3. **逆元 / Inverse**: $\forall a \in G \exists a^{-1} \in G (a \cdot a^{-1} = a^{-1} \cdot a = e)$

**定理 3.1.1** 群的单位元是唯一的。
**Theorem 3.1.1** The identity element of a group is unique.

### 3.2 环 / Rings

**定义 3.2.1** 环是一个集合 $R$ 和两个二元运算 $+$ 和 $\cdot$，满足：
**Definition 3.2.1** A ring is a set $R$ with two binary operations $+$ and $\cdot$ that satisfy:

1. $(R, +)$ 是阿贝尔群 / $(R, +)$ is an abelian group
2. **乘法结合律 / Multiplicative Associativity**: $\forall a, b, c \in R ((a \cdot b) \cdot c = a \cdot (b \cdot c))$
3. **分配律 / Distributivity**: $\forall a, b, c \in R (a \cdot (b + c) = a \cdot b + a \cdot c)$

### 3.3 域 / Fields

**定义 3.3.1** 域是一个环 $(F, +, \cdot)$，其中非零元素在乘法下形成阿贝尔群。
**Definition 3.3.1** A field is a ring $(F, +, \cdot)$ where the nonzero elements form an abelian group under multiplication.

**例子 / Examples:**
- $\mathbb{Q}$ 是有理数域 / $\mathbb{Q}$ is the field of rational numbers
- $\mathbb{R}$ 是实数域 / $\mathbb{R}$ is the field of real numbers
- $\mathbb{C}$ 是复数域 / $\mathbb{C}$ is the field of complex numbers

## 4. 序论 / Order Theory

### 4.1 偏序 / Partial Orders

**定义 4.1.1** 偏序集是一个集合 $P$ 和一个二元关系 $\leq$，满足：
**Definition 4.1.1** A partially ordered set is a set $P$ with a binary relation $\leq$ that satisfies:

1. **自反性 / Reflexivity**: $\forall x \in P (x \leq x)$
2. **反对称性 / Antisymmetry**: $\forall x, y \in P (x \leq y \land y \leq x \rightarrow x = y)$
3. **传递性 / Transitivity**: $\forall x, y, z \in P (x \leq y \land y \leq z \rightarrow x \leq z)$

### 4.2 全序 / Total Orders

**定义 4.2.1** 全序是满足完全性的偏序：
**Definition 4.2.1** A total order is a partial order that satisfies totality:
$$\forall x, y \in P (x \leq y \lor y \leq x)$$

### 4.3 良序 / Well-Orders

**定义 4.3.1** 良序是全序，其中每个非空子集都有最小元素。
**Definition 4.3.1** A well-order is a total order where every non-empty subset has a least element.

**定理 4.3.1** (良序定理) 任何集合都可以良序化。
**Theorem 4.3.1** (Well-Ordering Theorem) Any set can be well-ordered.

## 5. 范畴论基础 / Category Theory Foundations

### 5.1 范畴 / Categories

**定义 5.1.1** 范畴 $\mathcal{C}$ 包含：
**Definition 5.1.1** A category $\mathcal{C}$ consists of:

1. **对象集合 / Collection of objects**: $\text{Ob}(\mathcal{C})$
2. **态射集合 / Collection of morphisms**: $\text{Mor}(\mathcal{C})$
3. **复合运算 / Composition operation**: $\circ$
4. **单位态射 / Identity morphisms**: $1_A$ for each object $A$

**公理 / Axioms:**
- **结合律 / Associativity**: $(f \circ g) \circ h = f \circ (g \circ h)$
- **单位律 / Identity**: $f \circ 1_A = f = 1_B \circ f$

### 5.2 函子 / Functors

**定义 5.2.1** 函子 $F: \mathcal{C} \rightarrow \mathcal{D}$ 是范畴间的映射：
**Definition 5.2.1** A functor $F: \mathcal{C} \rightarrow \mathcal{D}$ is a mapping between categories:

1. **对象映射 / Object mapping**: $F: \text{Ob}(\mathcal{C}) \rightarrow \text{Ob}(\mathcal{D})$
2. **态射映射 / Morphism mapping**: $F: \text{Mor}(\mathcal{C}) \rightarrow \text{Mor}(\mathcal{D})$
3. **保持复合 / Preserves composition**: $F(f \circ g) = F(f) \circ F(g)$
4. **保持单位 / Preserves identity**: $F(1_A) = 1_{F(A)}$

### 5.3 自然变换 / Natural Transformations

**定义 5.3.1** 自然变换 $\eta: F \Rightarrow G$ 是两个函子间的态射族：
**Definition 5.3.1** A natural transformation $\eta: F \Rightarrow G$ is a family of morphisms between two functors:

$$\eta_A: F(A) \rightarrow G(A)$$

满足自然性条件 / satisfying the naturality condition:
$$G(f) \circ \eta_A = \eta_B \circ F(f)$$

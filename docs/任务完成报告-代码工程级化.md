## 七、使用指南

### 7.1 添加到项目

```toml
[dependencies]
formal-algorithms = { path = "examples/algorithms" }
```

### 7.2 基本使用示例

```rust
use formal_algorithms::*;

// 排序
let mut data = vec![3, 1, 4, 1, 5, 9];
merge_sort(&mut data);
assert_eq!(data, vec![1, 1, 3, 4, 5, 9]);

// 搜索
let arr = vec![1, 3, 5, 7, 9];
assert_eq!(binary_search(&arr, 5), Ok(2));

// 图算法
use std::collections::HashMap;
let mut graph: HashMap<char, Vec<char>> = HashMap::new();
graph.insert('A', vec!['B', 'C']);
let result = bfs(&graph, 'A');

// LCS
let seq1 = "ABCBDAB";
let seq2 = "BDCABA";
let result = lcs(seq1.as_bytes(), seq2.as_bytes());
assert_eq!(result.length, 4);
```

### 7.3 运行测试

```bash
cd examples/algorithms
cargo test              # 运行所有测试
cargo test --doc        # 运行文档测试
cargo bench             # 运行性能基准
cargo doc --open        # 生成并打开文档
cargo clippy            # 代码检查
```

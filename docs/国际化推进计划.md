### 5.3 代码表征

#### 5.3.1 多语言实现

**Rust实现**:

```rust
// 快速排序算法
pub fn quicksort<T: Ord + Clone>(arr: &[T]) -> Vec<T> {
    if arr.len() <= 1 {
        return arr.to_vec();
    }

    let pivot = &arr[0];
    let (less, equal, greater): (Vec<_>, Vec<_>, Vec<_>) =
        arr.iter().partition(|x| x < pivot);

    let mut result = quicksort(&less);
    result.extend(equal);
    result.extend(quicksort(&greater));
    result
}
```

**Haskell实现**:

```haskell
-- 快速排序算法
quicksort :: Ord a => [a] -> [a]
quicksort [] = []
quicksort (x:xs) =
    quicksort [y | y <- xs, y <= x] ++
    [x] ++
    quicksort [y | y <- xs, y > x]
```

### 2.1 CSE 521: Design and Analysis of Algorithms I

#### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Design and Analysis of Algorithms I (算法设计与分析 I) |
| **课程代码** | CSE 521 |
| **学分** | 4学分 |
| **先修课程** | CSE 326 或同等课程 |
| **课程性质** | 研究生核心课程 |
| **授课教师** | Shayan Oveis Gharan, Thomas Rothvoss 等 |

#### 课程描述与目标

CSE 521 是一门从现代视角研究算法设计与分析的研究生课程，特别强调在计算机科学多个子领域中都有广泛应用的算法技术。课程的"现代视角"意味着大量使用随机化、线性代数和优化方法。

**课程目标**：
- 掌握现代算法设计技术，特别是随机化方法
- 理解降维、谱方法、线性规划等高级主题
- 能够分析和设计用于大数据和机器学习的算法

#### 课程大纲

**第一部分: 随机化算法基础**

1. **随机算法导论**
   - Karger收缩算法 (Contraction Algorithm)
   - 最小割问题的随机算法

2. **集中不等式 (Concentration Bounds)**
   - Markov不等式、Chebyshev不等式
   - Chernoff界及其应用
   - 鞅 (Martingales) 方法

3. **哈希技术**
   - 成对独立哈希 (Pairwise Independent Hashing)
   - 通用哈希族
   - 局部敏感哈希 (Locality Sensitive Hashing, LSH)

4. **数据流算法 (Streaming Algorithms)**
   - AMS Sketch
   - Count-Min Sketch
   - 重 hitter 问题

**第二部分: 降维与代数方法**

5. **降维技术**
   - Johnson-Lindenstrauss引理
   - 快速JL变换
   - L1范数降维的不可能性

6. **代数算法**
   - Schwartz-Zippel引理
   - 多项式恒等式测试
   - 完美匹配问题的代数算法
   - 准NC算法

**第三部分: 线性代数与谱方法**

7. **线性代数基础**
   - SVD (奇异值分解)
   - 行列式与迹
   - 低秩逼近 (Low Rank Approximation)

8. **谱图理论 (Spectral Graph Theory)**
   - 图拉普拉斯矩阵
   - 谱聚类
   - Cheeger不等式

9. **最大割问题 (Max Cut)**
   - Goemans-Williamson近似算法
   - 半定规划松弛

**第四部分: 优化与高级主题**

10. **线性规划与对偶性**
    - 单纯形法回顾
    - 对偶理论
    - LP舍入技术

11. **马尔可夫决策过程 (MDPs)**
    - 线性建模
    - 价值迭代与策略迭代

12. **次模函数 (Submodular Functions)**
    - 次模性定义与性质
    - 贪心近似算法

### 2.3 CSE 505: Programming Languages

#### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Programming Languages (编程语言) |
| **课程代码** | CSE P505 (研究生), CSE 341 (本科基础) |
| **学分** | 4学分 |
| **先修课程** | CSE 341 (编程语言基础) |
| **课程性质** | 研究生核心课程 |

#### 课程描述与目标

CSE 505 是一门深入研究编程语言范式和形式语义的高级课程。课程涵盖函数式、面向对象、逻辑和约束编程等非命令式编程范式，以及编程语言语义学和类型理论。

**课程目标**：
- 精确理解和定义核心编程语言概念
- 掌握形式语义学方法（操作语义、指称语义、公理语义）
- 深入理解类型系统和类型理论
- 能够形式化推理程序行为

#### 课程大纲

**第一部分: 编程语言基础**

1. **归纳与递归**
   - 数学归纳法与强归纳法
   - 结构归纳法 (Structural Induction)
   - 代数数据类型 (Algebraic Data Types, ADT)
   - 原始递归函数

2. **λ演算**
   - λ演算语法与求值
   - Church编码
   - 归约策略
   - 不动点组合子

**第二部分: 类型理论**

3. **简单类型系统**
   - 简单类型λ演算
   - 类型安全性 (Type Safety)
   - 类型推导 (Type Inference)
   - Hindley-Milner类型系统

4. **多态性**
   - 参数多态 (Parametric Polymorphism)
   - 子类型多态 (Subtype Polymorphism)
   - 特设多态 (Ad-hoc Polymorphism)

5. **高级类型系统**
   - 依赖类型 (Dependent Types)
   - 归纳类型 (Inductive Types)
   - 归纳族 (Inductive Families)

**第三部分: 编程范式**

6. **函数式编程**
   - 高阶函数
   - 惰性求值与严格求值
   - 延续 (Continuations)
   - 单子 (Monads)

7. **逻辑编程**
   - Prolog基础
   - 统一算法 (Unification)
   - 约束逻辑编程 (CLP)
   - 回答集编程 (ASP)

8. **面向对象编程语义**
   - 类与对象的形式化
   - 继承与子类型
   - 动态分派 (Dynamic Dispatch)

**第四部分: 形式语义**

9. **操作语义**
   - 大步语义 (Big-step Semantics)
   - 小步语义 (Small-step Semantics)
   - 上下文语义

10. **指称语义**
    - 域论基础
    - 单调函数与不动点

11. **公理语义与霍尔逻辑**
    - 霍尔三元组
    -  weakest precondition
    - 程序验证

### 4.4 建议补充内容

#### 次模函数 (Submodular Functions)

次模函数是组合优化中的重要概念，具有"边际收益递减"特性。建议在优化理论模块补充：

- 次模性定义与性质
- 次模函数最大化/最小化
- 贪心算法的近似保证
- 在机器学习中的应用

参考：Nemhauser, Wolsey, Fisher (1978) "An analysis of approximations for maximizing submodular set functions"

#### 约束编程 (Constraint Programming)

约束编程是一种声明式编程范式，建议补充：

- 约束满足问题 (CSP) 定义
- 约束传播与一致性算法
- 回溯搜索策略
- CLP(R)与CLP(FD)实例

参考：Marriott & Stuckey *Programming with Constraints*

#### 谱图理论扩展

建议补充CSE 521中涵盖但未完全覆盖的谱图内容：

- 图拉普拉斯矩阵谱性质
- Cheeger不等式完整证明
- 谱聚类算法
- 有效电阻与谱稀疏化

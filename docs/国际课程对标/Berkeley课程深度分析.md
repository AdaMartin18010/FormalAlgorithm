## 1. Berkeley CS 170: Efficient Algorithms and Intractable Problems

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Efficient Algorithms and Intractable Problems |
| **课程代码** | CS 170 |
| **授课教师** | Prasad Raghavendra, John Wright, Sanjam Garg |
| **学期/年份** | Spring 2024, 2025 |
| **课程网址** | <https://cs170.org/> |
| **先修课程** | CS 61B, CS 70 |
| **学分** | 4 units |
| **教材** | Dasgupta, Papadimitriou, Vazirani《Algorithms》|

### 课程大纲

#### 第1部分: 算法基础

1. **渐进分析与分治**
   - Big-O记号
   - 主定理 (Master Theorem)
   - 分治策略
   - 快速傅里叶变换 (FFT)

2. **图论基础**
   - 图的表示
   - 深度优先搜索 (DFS)
   - 强连通分量
   - 拓扑排序

3. **路径与生成树**
   - 最短路径: Dijkstra, Bellman-Ford
   - 最小生成树: Kruskal, Prim
   - Union-Find数据结构

#### 第2部分: 算法设计范式

1. **贪心算法**
   - 贪心选择性质
   - 活动选择
   - 交换论证

2. **动态规划**
   - DP方法论
   - 编辑距离
   - 背包问题
   - 最优子结构

3. **线性规划**
   - LP建模
   - 对偶性
   - 网络流应用

#### 第3部分: 复杂度与不可解性

1. **网络流**
   - 最大流问题
   - Ford-Fulkerson
   - 最小割
   - 二分匹配

2. **NP完全性**
   - P vs NP
   - 多项式归约
   - Cook-Levin定理
   - 经典NP完全问题

3. **处理NP难性**
   - 近似算法
   - 启发式方法
   - 固定参数可追踪性

#### 第4部分: 高级主题 (可选)

1. **流算法**
    - 数据流模型
    - 采样技术

2. **密码学基础**
    - 单向函数
    - 零知识证明简介

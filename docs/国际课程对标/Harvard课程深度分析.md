## 1. Harvard CS 125: Algorithms & Complexity

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Algorithms & Complexity |
| **课程代码** | CS 125 |
| **授课教师** | Salil Vadhan, Jelani Nelson, Michael Mitzenmacher, Madhu Sudan |
| **学期/年份** | Fall 2015-2016 |
| **课程网址** | <https://salil.seas.harvard.edu/~cs125> |
| **先修课程** | 无 (替代CS 121+124) |
| **学分** | 4 units |
| **教材** | CLRS, Kleinberg-Tardos, Sipser |

### 课程大纲

#### 第1部分: 算法基础

1. **排序与选择**
   - 比较排序下界
   - 字操作排序
   - 线性时间选择

2. **贪心算法**
   - 最小生成树
   - 贪心选择性质
   - 交换论证

3. **分治法**
   - 矩阵乘法优化
   - Strassen算法
   - 最近点对

4. **动态规划**
   - 最短路径 (Bellman-Ford, Floyd-Warshall)
   - 序列对齐
   - 最优子结构

#### 第2部分: 计算模型

1. **随机存取机 (RAM)**
   - RAM模型定义
   - 字操作

2. **图灵机**
   - 图灵机定义
   - 多带图灵机
   - Church-Turing论题

#### 第3部分: 复杂度理论

1. **P vs NP**
   - P类与NP类定义
   - NP完全性
   - 多项式归约
   - Cook-Levin定理

2. **不可解性**
   - 停机问题
   - 递归可枚举集
   - 与哥德尔不完备定理的关系

3. **随机化算法与复杂度**
   - 哈希
   - 马尔可夫链
   - RP, BPP复杂度类

#### 第4部分: 高级主题 (可选)

1. **近似算法与hardness**
    - 近似比
    - hardness of approximation

2. **数据结构下界**
    - cell probe模型
    - 下界证明技术

## 2. Harvard CS 1200: Intro to Algorithms and their Limitations

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Introduction to Algorithms and their Limitations |
| **课程代码** | CS 1200 (原CS 120) |
| **授课教师** | 见Harvard CS |
| **学期/年份** | Fall 2025 |
| **课程网址** | <https://harvard-cs-1200.github.io/cs1200/> |
| **先修课程** | 无 |
| **学分** | 4 units |
| **教材** | 课程讲义 (Ch 1-29) |

### 课程大纲

#### 第1部分: 计算与算法基础

1. **计算问题建模**
   - 问题的形式化定义
   - 实例与解

2. **基本算法技术**
   - 暴力搜索
   - 剪枝
   - 回溯

#### 第2部分: 算法设计

1. **贪心算法**
   - 区间调度
   - Huffman编码

2. **分治法**
   - 归并排序
   - 逆序对计数

3. **动态规划**
   - 加权区间调度
   - 最长递增子序列

#### 第3部分: 数据结构

1. **基本数据结构**
   - 数组、链表
   - 栈、队列

2. **高级数据结构**
   - 二叉搜索树
   - 堆
   - 哈希表

#### 第4部分: 图算法

1. **图遍历**
   - BFS, DFS
   - 连通分量

2. **最短路径**
   - Dijkstra算法
   - 负权边处理

#### 第5部分: 计算的局限性

1. **可计算性**
    - 停机问题
    - 不可判定问题

2. **复杂度**
    - P vs NP简介
    - NP完全问题

3. **近似与启发式**
    - 当精确解不可行时
    - 启发式搜索

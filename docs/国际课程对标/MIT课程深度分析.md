## 1. MIT 6.006: Introduction to Algorithms

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Introduction to Algorithms |
| **课程代码** | 6.006 (新编号: 6.1210) |
| **授课教师** | Erik Demaine, Jason Ku, Justin Solomon |
| **学期/年份** | Spring 2020, Fall 2024+ |
| **课程网址** | <https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-spring-2020/> |
| **先修课程** | 6.0001 (Python编程), 6.042J (离散数学) |
| **学分** | 12 units (4-0-8) |
| **教材** | CLRS《Introduction to Algorithms》3rd Edition |

### 课程大纲

#### 第1部分: 基础与数据结构

1. **算法与计算模型**
   - 计算问题的数学建模
   - 算法性能度量与分析技术

2. **基本数据结构**
   - 动态数组 (Dynamic Arrays)
   - 堆 (Heaps)
   - 平衡二叉搜索树 (Balanced BST)
   - 哈希表 (Hash Tables)

#### 第2部分: 算法设计范式

1. **排序算法**
   - 比较排序: 归并排序、快速排序、堆排序
   - 线性时间排序: 计数排序、基数排序

2. **图算法**
   - 图的表示与遍历 (BFS/DFS)
   - 最短路径: Dijkstra、Bellman-Ford
   - 最小生成树: Prim、Kruskal

3. **高级算法技术**
   - 动态规划 (Dynamic Programming)
   - 贪心算法 (Greedy Algorithms)
   - 摊还分析 (Amortized Analysis)

## 2. MIT 18.404/6.5400: Theory of Computation

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Theory of Computation |
| **课程代码** | 18.404 / 6.5400 / 18.4041J |

| **学期/年份** | Fall 2025 |
| **课程网址** | <https://ocw.mit.edu/courses/18-404j-theory-of-computation-fall-2020/> |
| **先修课程** | 6.1200 或 18.200 |
| **学分** | 4-0-8 units |
| **教材** | Sipser《Introduction to the Theory of Computation》|

### 课程大纲

#### 第1部分: 自动机与形式语言

1. **正则语言**
   - 有限自动机 (DFA/NFA)
   - 正则表达式
   - 泵引理与正则语言性质
   - 非正则语言识别

2. **上下文无关语言**
   - 上下文无关文法 (CFG)
   - 下推自动机 (PDA)
   - 非上下文无关语言

#### 第2部分: 可计算性理论

1. **图灵机**
   - 图灵机定义与变体
   - Church-Turing 论题
   - 可判定性

2. **不可判定性**
   - 停机问题 (Halting Problem)
   - 归约技术
   - Rice定理
   - 递归函数理论

#### 第3部分: 复杂度理论

1. **时间复杂度**
   - P类与NP类
   - NP完全性
   - 多项式时间归约
   - Cook-Levin定理

2. **空间复杂度**
   - PSPACE与NPSPACE
   - Savitch定理
   - 复杂性层次定理

3. **高级主题**
   - 概率计算 (BPP)
   - 交互式证明系统
   - 近似算法与 hardness

## 3. MIT 6.046: Design and Analysis of Algorithms

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Design and Analysis of Algorithms |
| **课程代码** | 6.046 (新编号: 6.1220J/18.410J) |
| **授课教师** | Erik Demaine, Srini Devadas, Nancy Lynch |
| **学期/年份** | Spring 2015 |
| **课程网址** | <https://ocw.mit.edu/courses/6-046j-design-and-analysis-of-algorithms-spring-2015/> |
| **先修课程** | 6.006, 6.042J |
| **学分** | 12 units |
| **教材** | CLRS《Introduction to Algorithms》3rd Edition |

### 课程大纲

#### 第1部分: 高级算法设计技术

1. **分治法 (Divide and Conquer)**
   - 中位数查找与选择
   - 快速傅里叶变换 (FFT)
   - 矩阵乘法优化 (Strassen)

2. **动态规划高级应用**
   - 序列对齐 (Sequence Alignment)
   - 最优二叉搜索树
   - 树上的动态规划

3. **贪心算法高级主题**
   - 拟阵 (Matroids)
   - 贪心算法的理论基础
   - 交换论证

#### 第2部分: 高级数据结构与算法

1. **网络流**
   - 最大流问题
   - Edmonds-Karp算法
   - 最小费用流
   - 匹配问题

2. **线性规划**
   - 单纯形法
   - 对偶性理论
   - 应用: 零和博弈、网络流

3. **随机化算法**
   - 随机快速排序分析
   - 哈希与指纹
   - 概率分析技术

#### 第3部分: 近似与在线算法

1. **近似算法**
   - 近似比分析
   - PTAS与FPTAS
   - 经典问题的近似算法

2. **在线算法**
   - 竞争分析
   - 经典在线问题
   - 多臂老虎机

3. **计算几何**
   - 凸包算法
   - 线段交点
   - 最近点对

## 二、核心课程深度分析 / Core Course Analysis

### 2.1 高等算法分析（Advanced Algorithm Analysis）

#### 课程信息 / Course Information

| 项目 | 内容 |
|------|------|
| **课程代码** | 30240532（姚班）/ 30240243 |
| **中文名** | 高等算法分析 |
| **英文名** | Advanced Algorithm Analysis |
| **学分** | 3学分 |
| **先修课程** | 数据结构、离散数学、概率论 |
| **授课语言** | 中文 / 英文（姚班英文授课）|
| **推荐教材** | 《算法导论》(CLRS)、Kleinberg-Tardos《Algorithm Design》|

#### 课程目标 / Course Objectives

1. 掌握高级算法设计技术：分治、动态规划、贪心、网络流等
2. 理解算法分析的高级方法：摊还分析、势能方法、概率分析
3. 培养算法思维的严谨性与创新性
4. 建立算法设计与复杂性分析的完整知识框架

#### 详细教学大纲 / Detailed Syllabus

**第一部分：算法分析基础回顾 / Fundamentals Review**

1. **渐进分析进阶**
   - Master定理的完整证明与扩展
   - Akra-Bazzi方法
   - 递归树方法与代换法

2. **概率分析工具**
   - 指示器随机变量
   - 期望的线性性
   - 高阶矩与集中不等式（Chernoff界、Hoeffding不等式）

**第二部分：高级数据结构 / Advanced Data Structures**

1. **平衡搜索树**
   - AVL树：平衡因子与旋转操作
   - 红黑树：五个性质与插入删除
   - B树与B+树：磁盘存储优化
   - 跳表（Skip Lists）：随机化结构

2. **高级堆结构**
   - 二项堆：合并操作分析
   - 斐波那契堆：摊还分析
   - 配对堆：实践性能

3. **并查集（Disjoint Set Union）**
   - 路径压缩
   - 按秩合并
   - 摊还分析（Ackermann函数逆）

**第三部分：高级算法设计技术 / Advanced Design Techniques**

1. **网络流与匹配**
   - 最大流：Ford-Fulkerson、Edmonds-Karp、Dinic算法
   - 最小费用最大流
   - 二分图匹配：匈牙利算法
   - 一般图匹配：开花算法（Blossom Algorithm）

2. **线性规划与对偶性**
   - 单纯形法几何直观
   - 弱对偶与强对偶定理
   - 原始-对偶方法
   - 应用：最小费用流、近似算法

3. **字符串算法**
   - KMP算法：前缀函数
   - Z算法与扩展KMP
   - 后缀数组与后缀树
   - 字符串匹配的应用

**第四部分：近似与随机算法 / Approximation and Randomized Algorithms**

1. **近似算法**
   - 近似比分析
   - 顶点覆盖、集合覆盖
   - TSP近似
   - PTAS与FPTAS

2. **随机化算法**
   - Las Vegas与Monte Carlo
   - 随机快速排序分析
   - 全域哈希（Universal Hashing）
   - 布隆过滤器

**第五部分：高级主题 / Advanced Topics**

1. **在线算法与竞争分析**
   - 竞争比定义
   - 分页问题
   - k-server问题

2. **计算几何初步**
   - 凸包算法
   - 最近点对
   - 线段交点

### 2.2 计算复杂性理论（Computational Complexity Theory）

#### 课程信息 / Course Information

| 项目 | 内容 |
|------|------|
| **课程代码** | 80240412（研究生）/ 姚班高年级选修 |
| **中文名** | 计算复杂性理论 |
| **英文名** | Computational Complexity Theory |
| **学分** | 2-3学分 |
| **先修课程** | 算法分析与设计、离散数学、自动机理论 |
| **授课语言** | 英文（姚班）/ 中文 |
| **推荐教材** | Arora-Barak《Computational Complexity: A Modern Approach》、Sipser《Introduction to the Theory of Computation》|

#### 课程目标 / Course Objectives

1. 理解计算复杂性的基本框架与核心概念
2. 掌握P、NP、PSPACE等复杂度类的定义与关系
3. 学习归约技术与完备性理论
4. 了解复杂性理论的前沿研究方向（电路复杂性、证明复杂性等）
5. 培养理论计算机科学的严谨思维方式

#### 详细教学大纲 / Detailed Syllabus

**第一部分：计算模型与基础 / Computational Models**

1. **图灵机进阶**
   - 多带图灵机与等价性
   - 非确定性图灵机
   - 空间有界图灵机
   - 随机访问机器（RAM）与图灵机关系

2. **Church-Turing论题**
   - 计算的直观概念
   - 有效可计算性
   - 扩展与讨论

**第二部分：时间复杂度 / Time Complexity**

1. **确定性时间类**
   - P类定义与性质
   - 时间层次定理（Time Hierarchy Theorem）
   - P完全性

2. **非确定性时间类**
   - NP类定义与等价形式
   - NP完全性理论
   - Cook-Levin定理完整证明
   - Karp的21个NP完全问题

3. **P vs NP问题**
   - 问题陈述与重要性
   - 当前研究进展
   - 相对化障碍（Baker-Gill-Solovay定理）

**第三部分：空间复杂度 / Space Complexity**

1. **空间类定义**
   - L（对数空间）与NL
   - PSPACE与NPSPACE
   - Savitch定理

2. **空间层次定理**
   - 证明技术
   - 与时间层次定理的对比

3. **PSPACE完全性**
   - TQBF（真量化布尔公式）
   - 广义博弈游戏

**第四部分：高级复杂性理论 / Advanced Topics**

1. **多项式层次（Polynomial Hierarchy）**
   - ΣP、ΠP、ΔP类
   - 层次关系
   - 与NP oracle的关系

2. **电路复杂性**
   - 电路定义与规模
   - P/poly类
   - 电路下界结果（PARITY不在AC⁰中）

3. **概率复杂度类**
   - BPP、RP、coRP、ZPP
   - 去随机化概述
   - P vs BPP问题

4. **交互式证明系统**
   - IP类定义
   - IP = PSPACE定理概述
   - PCP定理简介

5. **量子复杂性**
   - 量子图灵机模型
   - BQP类
   - 量子算法与经典复杂性关系

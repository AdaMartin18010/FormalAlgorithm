## 1. Oxford L81: Proof Assistants

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Proof Assistants |
| **课程代码** | L81 |
| **授课教师** | Dr Thomas Bauereiss, Dr Meven Lennon-Bertrand, Prof Peter Sewell |
| **学期/年份** | Michaelmas Term 2024-25 |
| **课程网址** | <https://www.cl.cam.ac.uk/teaching/2425/L81/> |
| **先修课程** | 基础逻辑、编程经验 |
| **学位** | MPhil ACS, Part III |
| **班级限制** | 最多16名学生 |

### 课程目标

本模块向学生介绍使用Isabelle和Coq进行交互式定理证明，包括为软件和硬件系统指定形式化模型以及推导这些模型属性的技术。

### 课程大纲

#### 第1部分: 证明助手基础

1. **证明助手与逻辑介绍**
   - 交互式定理证明概述
   - 证明助手的应用领域
   - 逻辑基础回顾

2. **谓词逻辑与类型集合论推理**
   - 一阶逻辑形式化
   - 集合论基础
   - 高阶逻辑基础

#### 第2部分: 依赖类型理论

1. **依赖类型论推理**
   - 依赖函数类型 (Π-types)
   - 依赖对类型 (Σ-types)
   - 归纳类型
   - 类型族

2. **Curry-Howard对应**
   - 命题即类型
   - 证明即程序
   - 证明归一化

#### 第3部分: 归纳与递归

1. **归纳定义**
   - 归纳数据类型
   - 归纳谓词
   - 结构归纳法

2. **递归函数**
   - 原始递归
   - 结构递归
   - 良基递归
   - 递归函数的推理

#### 第4部分: 形式化建模

1. **操作语义形式化**
   - 小步语义
   - 大步语义
   - 操作语义的性质证明

2. **类型系统形式化**
   - 简单类型λ演算
   - 类型安全定理
   - 进展与保持

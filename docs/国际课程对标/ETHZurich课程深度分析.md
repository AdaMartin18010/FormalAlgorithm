## 2. 核心课程分析

### 2.1 Algorithmen und Datenstrukturen (算法与数据结构)

#### 课程信息

| 项目 | 内容 |
|------|------|
| **课程代码** | 252-0026-00L / 252-0027-00L |
| **英文名称** | Data Structures and Algorithms |
| **学分** | 7 ECTS (3V + 2U + 1A) |
| **授课学期** | 第一学期 (秋季) |
| **授课语言** | 德语 |
| **授课教师** | Johannes Lengler, Markus Püschel, David Steurer 等 |
| **先修课程** | Einführung in die Programmierung (程序设计导论) |

#### 课程目标

本课程是计算机科学本科的基础核心课程，旨在：
- 掌握基本数据结构的设计、实现与分析
- 理解经典算法的设计范式
- 培养算法效率分析能力
- 建立计算问题的数学建模思维

#### 课程大纲

**第一部分: 基础数据结构**

1. **数组与链表**
   - 动态数组的实现与摊还分析
   - 单向链表、双向链表
   - 循环链表与哨兵节点

2. **栈与队列**
   - 抽象数据类型 (ADT)
   - 基于数组与链表的实现
   - 双端队列 (Deque)

3. **树结构**
   - 二叉树遍历 (前序、中序、后序、层序)
   - 二叉搜索树 (BST)
   - 平衡二叉树 (AVL树)
   - 堆 (Heap) 与优先队列

4. **哈希表**
   - 哈希函数设计
   - 冲突解决方法 (链地址法、开放寻址法)
   - 动态哈希表

**第二部分: 图算法**

1. **图的基本概念**
   - 图的表示 (邻接矩阵、邻接表)
   - 图的遍历: BFS 与 DFS
   - 连通分量与强连通分量

2. **最短路径算法**
   - Dijkstra 算法
   - Bellman-Ford 算法
   - Floyd-Warshall 算法

3. **最小生成树**
   - Prim 算法
   - Kruskal 算法
   - 并查集 (Union-Find) 数据结构

**第三部分: 算法设计范式**

1. **分治算法**
   - 归并排序与快速排序
   - 最近点对问题
   - Master 定理

2. **贪心算法**
   - 活动选择问题
   - 哈夫曼编码
   - 交换论证技术

3. **动态规划**
   - 最长公共子序列 (LCS)
   - 背包问题
   - 最优二叉搜索树

### 2.2 Algorithmen und Wahrscheinlichkeit (算法与概率)

#### 课程信息

| 项目 | 内容 |
|------|------|
| **课程代码** | 252-0029-00L |
| **英文名称** | Algorithms and Probability |
| **学分** | 7 ECTS (3V + 2U + 1A) |
| **授课学期** | 第二学期 (春季) |
| **授课语言** | 德语 |
| **授课教师** | Angelika Steger, Emo Welzl |
| **先修课程** | Algorithmen und Datenstrukturen |

#### 课程目标

本课程将概率论与算法设计紧密结合，强调：
- 随机化算法的设计与分析
- 概率方法在组合优化中的应用
- 随机数据结构 (如随机搜索树)
- 计算几何中的随机算法

#### 课程大纲

**第一部分: 概率基础**

1. **概率论回顾**
   - 概率空间与随机变量
   - 期望与方差
   - 条件概率与独立性
   - 重要不等式 (Markov, Chebyshev, Chernoff)

2. **随机算法分析技术**
   - 期望运行时间分析
   - 高概率分析 (High Probability)
   - 概率放大技术

**第二部分: 随机化数据结构**

1. **随机搜索树 (Treaps)**
   - 随机二叉搜索树
   - 期望高度分析
   - 随机插入与删除

2. **跳表 (Skip Lists)**
   - 随机化层级结构
   - 期望时间复杂度
   - 与平衡树的比较

3. **哈希的高级主题**
   - 通用哈希族
   - 完美哈希
   - 布隆过滤器 (Bloom Filters)

**第三部分: 随机算法**

1. **随机排序与选择**
   - 随机快速排序分析
   - 随机化选择算法
   - 中位数查找

2. **随机图算法**
   - 随机最小割 (Karger算法)
   - 随机游走与混合时间

3. **计算几何中的随机算法**
   - 随机增量构造
   - 点定位问题
   - 凸包算法

**第四部分: 概率方法**

1. **存在性证明**
   - 基本概率方法
   - Lovász局部引理

2. **随机图论**
   - Erdős-Rényi 模型
   - 相变现象
   - 随机正则图

   - 多带图灵机
   - 非确定性图灵机
   - Church-Turing论题

2. **可判定性**
   - 可判定语言
   - 半可判定语言
   - 不可判定问题

3. **归约与完备性**
   - 归约技术

**第三部分: 复杂性理论**

1. **时间复杂性**

   - Cook-Levin定理
   - 多项式时间归约
   - 经典NP完全问题

2. **空间复杂性**
   - 空间复杂性类
   - PSPACE与NPSPACE

   - 空间层次定理

#### 课程目标

本课程是ETH Zurich形式化方法方向的核心课程：
- 函数式编程范式与抽象思维
- 形式化规约与验证技术
- 模型检测基础
- 提升程序正确性论证的抽象层次

#### 课程大纲

**第一部分: 函数式编程 (David Basin)**

1. **函数式编程基础**
   - Haskell语言入门
   - 纯函数与引用透明
   - 高阶函数
   - 递归与尾递归优化

2. **类型系统**
   - 多态类型
   - 类型推导
   - 代数数据类型
   - 类型类

3. **λ演算**
   - λ项与β归约
   - Church编码
   - 不动点组合子
   - 可计算性基础

**第二部分: 形式化方法 (Peter Müller)**

1. **操作语义**
   - 大步语义与小步语义
   - 程序验证基础
   - Hoare逻辑
   - 最弱前置条件

2. **演绎验证**
   - 循环不变量
   - 过程调用验证
   - 抽象数据类型验证

3. **模型检测**
   - 时序逻辑 (LTL/CTL)
   - Büchi自动机
   - 显式状态模型检测
   - 符号模型检测 (BDD)

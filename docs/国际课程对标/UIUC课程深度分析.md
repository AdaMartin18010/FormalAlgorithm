### 1.3 研究强项与特色

UIUC 计算机科学的核心优势体现在以下领域：

1. **并行与分布式计算 (Parallel & Distributed Computing)**
   - 拥有世界著名的 **Blue Waters 超级计算机** (2013-2021年运行)
   - 并行编程理论与实践的深厚积淀
   - MPI、OpenMP、CUDA 等并行编程模型的研究发源地之一

2. **计算机系统与架构 (Systems & Architecture)**
   - 超标量处理器设计、乱序执行、分支预测
   - 缓存一致性协议与内存一致性模型
   - GPU 与加速器架构研究

3. **高性能计算 (High-Performance Computing)**
   - 大规模科学计算应用优化
   - 异构计算 (CPU+GPU)
   - 并行算法可扩展性分析

4. **形式化方法与验证 (Formal Methods & Verification)**
   - 程序验证理论
   - 时序逻辑与模型检测
   - 形式化软件工程方法

## 二、核心课程深度分析

### 2.1 CS 473: Algorithms (算法)

#### 课程基本信息

| 项目 | 内容 |
|------|------|
| **课程代码** | CS 473 / Math 473 / CSE 414 |
| **课程名称** | Algorithms (算法) |
| **学分** | 3-4 学分 |
| **课程性质** | 本科/研究生核心课程 |
| **先修课程** | CS 374 (Intro to Algorithms & Models of Computation) |
| **授课教师** | Jeff Erickson, Timothy Chan 等 |
| **课程网址** | <https://courses.grainger.illinois.edu/cs473/> |

#### 课程描述与目标

CS 473 是 UIUC **最著名的高阶算法课程**，被广泛认为是北美计算机科学本科阶段**最具挑战性的算法课程之一**。本课程在 CS 374 基础上大幅深化，涵盖大量高级算法设计技术与分析理论：

- 掌握高级分治策略与递归分析
- 理解网络流与线性规划的深层联系
- 学习随机算法与概率分析技术
- 探索近似算法与计算难解性
- 培养解决复杂算法问题的数学能力

#### 详细教学大纲

| 模块 | 主题内容 | 深度要求 |
|------|----------|----------|
| **1. 分治进阶** | 线性时间选择 (Median of Medians)、快速傅里叶变换 (FFT)、Strassen 矩阵乘法、最近点对 | 高 |
| **2. 图算法深化** | 强连通分量、拓扑排序、最小生成树高级应用、匹配算法 | 高 |
| **3. 动态规划高级应用** | 状态压缩 DP、树上的 DP、字符串 DP、最优二叉搜索树 | 高 |
| **4. 网络流** | 最大流/最小割 (Ford-Fulkerson, Edmonds-Karp)、最小费用流、二分图匹配 | 高 |
| **5. 线性规划** | LP 建模、单纯形法、对偶性理论、网络流的 LP 解释 | 高 |
| **6. 随机算法** | 哈希、指纹技术、概率分析、Karger 最小割算法、Tail Inequalities | 中高 |
| **7. 近似算法** | 近似比分析、集合覆盖、顶点覆盖、调度问题近似 | 中高 |
| **8. 计算复杂性** | NP 完全性深化、P vs NP、多项式归约技巧 | 中 |
| **9. 高级数据结构** | 并查集 (Union-Find)、摊还分析高级应用 | 中 |
| **10. 计算几何** | 凸包、线段交点、范围搜索 (视时间而定) | 选讲 |

#### 推荐教材

| 教材 | 作者 | 出版社 | 用途 |
|------|------|--------|------|
| **《Algorithms》** | Jeff Erickson (UIUC) | 在线免费 | **主教材** |
| **《Introduction to Algorithms》** | CLRS | MIT Press | 标准参考 |
| **《Algorithm Design》** | Kleinberg, Tardos | Addison Wesley | 补充参考 |
| **《Randomized Algorithms》** | Motwani, Raghavan | Cambridge | 随机算法参考 |

> **特色**: Jeff Erickson 教授的教材可免费在线获取 (<https://jeffe.cs.illinois.edu/teaching/algorithms/>)，以极高的数学严谨性和清晰讲解著称。

### 2.3 CS 433: Computer System Organization (计算机系统组成)

#### 课程基本信息

| 项目 | 内容 |
|------|------|
| **课程代码** | CS 433 / CSE 422 |
| **课程名称** | Computer System Organization (计算机系统组成) |
| **学分** | 3 (本科) / 4 (研究生) 学分 |
| **课程性质** | 本科核心/研究生基础课程 |
| **先修课程** | CS 233 (Computer Architecture) |
| **课程网址** | <https://courses.grainger.illinois.edu/cs433/> |

#### 课程描述与目标

CS 433 是 UIUC **系统方向的核心课程**，深入讲解现代计算机处理器与内存系统的设计原理，特别强调**并行处理硬件基础**：

- 理解超标量处理器与乱序执行机制
- 掌握内存层次结构的高级优化技术
- 学习多处理器设计原理与缓存一致性
- 了解 GPU 与加速器架构

#### 详细教学大纲

| 模块 | 主题内容 | 课时 |
|------|----------|------|
| **1. 性能基础** | 性能评估指标、CPI、Amdahl 定律、功耗与可靠性 | 2-3 讲 |
| **2. 流水线进阶** | 多周期功能单元、静态分支预测、精确中断处理 | 3-4 讲 |
| **3. 指令级并行 (ILP)** | 动态调度、超标量架构、Tomasulo 算法、动态分支预测、推测执行 | 5-6 讲 |
| **4. 软件级 ILP** | 循环展开、轨迹调度、预测执行、内存访问重排序 | 2-3 讲 |
| **5. 缓存高级设计** | 预取技术、无锁缓存 (lockup-free)、多级缓存优化 | 3-4 讲 |
| **6. 主存与虚拟内存** | DRAM 组织、页表结构、TLB 优化、虚拟内存管理 | 3-4 讲 |
| **7. 多处理器基础** | 并行模型 (SIMD, MIMD)、共享内存架构、对称多处理 | 3-4 讲 |
| **8. 缓存一致性** | 监听协议 (Snoopy)、目录协议、MESI 及其变种 | 4-5 讲 |
| **9. 同步与一致性** | 同步原语实现、内存一致性模型 (Sequential, Weak, Release) | 3-4 讲 |
| **10. 高级主题** | GPU 架构、加速器设计、数据中心架构、安全架构 | 2-3 讲 |

### 2.4 CS 484: Parallel Programming (并行编程)

#### 课程基本信息

| 项目 | 内容 |
|------|------|
| **课程代码** | CS 484 |
| **课程名称** | Parallel Programming (并行编程) |
| **学分** | 3 (本科) / 4 (研究生) 学分 |
| **课程性质** | 本科/研究生专业选修课 |
| **先修课程** | CS 241 (System Programming) |
| **课程网址** | <https://courses.grainger.illinois.edu/cs484/> |

#### 课程描述与目标

CS 484 是 UIUC **并行计算方向的旗舰课程**，专注于教授如何编写高效的并行程序以充分利用现代并行计算机的强大性能。课程涵盖：

- 共享内存与分布式内存并行编程
- SIMD 处理器与加速器编程
- 并行性能分析与可扩展性理论
- OpenMP、MPI、Charm++ 等主流并行编程框架

#### 详细教学大纲

| 周次 | 主题内容 | 相关概念与技术 |
|------|----------|----------------|
| **1** | 性能与复杂性 | Moore 定律、并行性、延迟与带宽、流水线、分支预测 |
| **2** | 缓存与内存优化 | 缓存性能分析、预取、虚拟内存、数据布局 |
| **3** | 共享内存编程基础 | 向量化、SIMD、共享内存机器架构、OpenMP 入门 |
| **4** | OpenMP 基础 | 并行循环构造、变量共享、依赖分析与循环重构、调度策略 |
| **5** | OpenMP 高级 | 同步机制、临界区、顺序一致性、Flush 构造、案例分析 |
| **6** | OpenMP 性能问题 | 缓存相关性能问题、伪共享、嵌套并行、显式依赖、任务并行 |
| **7** | 期中考试 | - |
| **8** | 分布式内存编程 | Pthreads、C++11 原子操作、并行队列、分布式内存机器、MPI 基础 |
| **9** | MPI 集合通信 | Send/Recv 变体、集合操作、子通信器、并行前缀 |
| **10** | 春假 | - |
| **11** | 其他分布式模型 | 成本模型、单边通信、混合编程 (MPI + OpenMP) |
| **12** | Charm++ | 理论模型介绍 |
| **13** | 并行算法 | 并行排序、广播/归约算法、可扩展性与等效率分析 |
| **14** | 分布式并行应用 | 矩阵乘法、互连拓扑、容错计算 |
| **15** | 并行语言概述 | 并行离散事件模拟、组合搜索、GPGPU 编程 |
| **16** | 期末考试 | 其他并行语言与未来展望 |

#### 关键学习目标

完成本课程后，学生应能：

1. **分析性能**: 使用 α-β 成本模型分析并行程序完成时间
2. **评估可扩展性**: 推导并行算法的等效率函数 (Iso-efficiency)
3. **OpenMP 编程**: 识别并行循环、重构循环依赖、优化调度策略
4. **MPI 编程**: 使用 MPI 原语编写分布式内存程序，理解集合通信
5. **调试并行程序**: 识别并解决并行程序中的常见错误

## 五、总结与建议

### 5.1 UIUC 课程体系优势总结

1. **并行计算领军**: 从 Blue Waters 到 Charm++，理论与实践双强
2. **系统与算法结合**: CS 473 与 CS 433/484 形成完整的算法-系统链条
3. **形式化方法完善**: CS 475/476/477 覆盖计算理论到软件验证
4. **教学资源开放**: Jeff Erickson 算法教材等优质资源免费获取

### 5.2 对 FormalAlgorithm 项目的建议

#### 短期建议 (高优先级)

1. **新增计算几何专题**
   - 凸包、Voronoi 图、范围搜索
   - 计算几何在算法设计中的应用

2. **补充 FFT 完整理论**
   - 数学推导、多项式乘法应用
   - 与卷积定理的联系

3. **强化并行算法分析**
   - 补充等效率分析 (Iso-efficiency)
   - 扩展可扩展性理论

4. **完善随机算法内容**
   - Tail Inequalities (Chernoff, Hoeffding)
   - 指纹技术与哈希应用

#### 中期建议

5. **建立系统-算法交叉模块**
   - 缓存一致性协议形式化
   - 内存一致性模型
   - 硬件并行对算法设计的影响

6. **补充 GPU/加速器编程**
   - CUDA/OpenCL 编程模型
   - 异构并行算法设计

#### 长期建议

7. **参考 UIUC 教材体系**
   - 借鉴 Jeff Erickson 教材的数学严谨性
   - 引入更多开放教育资源

8. **建立实践项目对标**
   - 设计并行编程实验项目
   - 与 OpenMP/MPI 标准对齐

## 1. CMU 15-451/651: Algorithm Design and Analysis

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Algorithm Design and Analysis |
| **课程代码** | 15-451 (本科) / 15-651 (研究生) |
| **授课教师** | Daniel Anderson, David Woodruff (Spring 2025) |
| **学期/年份** | Spring 2025 |
| **课程网址** | <https://www.cs.cmu.edu/~15451-s25/> |
| **先修课程** | 15-210 或同等课程 |
| **学分** | 12 units |
| **教材** | 课程讲义 (Anderson, Blum, Gupta, Sleator) |

### 课程大纲

#### 第1部分: 算法分析模型

1. **线性时间选择**
   - 随机选择算法
   - 确定性中位数算法 (Median of Medians)
   - 递归树分析

2. **具体模型与下界**
   - 比较模型
   - 交换模型
   - 信息论下界

3. **整数排序**
   - 计数排序
   - 基数排序
   - 桶排序

#### 第2部分: 高级数据结构

1. **摊还分析**
   - 聚合分析
   - 会计方法
   - 势能方法

2. **伸展树 (Splay Trees)**
   - 伸展操作
   - 摊还分析
   - 应用

3. **哈希技术**
   - 通用哈希 (Universal Hashing)
   - 完美哈希
   - 流算法中的哈希
   - 指纹技术

#### 第3部分: 图与网络算法

1. **动态规划高级应用**
   - 图DP: Bellman-Ford, Floyd-Warshall
   - 状态压缩DP

2. **网络流**
   - 最大流: Ford-Fulkerson, Edmonds-Karp
   - 最小费用流
   - 匹配问题

3. **零和博弈**
   - Minimax定理
   - 博弈论基础

#### 第4部分: 优化与近似

1. **线性规划**
    - 单纯形法
    - 对偶性
    - Seidel算法

2. **机制设计**
    - 拍卖理论
    - 激励相容

3. **近似算法**
    - 近似比分析
    - 经典问题近似
    - PTAS/FPTAS

4. **在线算法**
    - 竞争分析
    - 多臂老虎机

5. **乘性权重算法**
    - 在线学习
    - 权重更新方法

#### 第5部分: 计算几何

1. **几何算法**
    - 几何原语
    - 凸包算法
    - 几何数据结构

## 2. CMU 15-414: Automated Program Verification

### 课程信息

| 项目 | 内容 |
|------|------|
| **课程名称** | Bug Catching: Automated Program Verification |
| **课程代码** | 15-414 |
| **授课教师** | Ruben Martins (Spring 2024) |
| **学期/年份** | Spring 2024 |
| **课程网址** | <https://www.cs.cmu.edu/~15414/s24/> |
| **先修课程** | 15-122, 15-251 |
| **学分** | 9 units |
| **教材** | 课程笔记 |

### 课程大纲

#### 第1部分: 验证基础

1. **程序正确性规约**
   - 前置/后置条件
   - 循环不变式
   - 霍尔逻辑 (Hoare Logic)

2. **形式化语义**
   - 操作语义
   - 指称语义
   - 公理语义

#### 第2部分: 自动验证工具

1. **演绎验证**
   - Why3平台使用
   - 证明义务生成
   - SMT求解器

2. **模型检测**
   - 时序逻辑 (LTL/CTL)
   - 显式状态模型检测
   - 符号模型检测
   - BDD (二元决策图)

3. **抽象解释**
   - 抽象域
   - widen/narrow
   - 静态分析应用

#### 第3部分: 高级主题

1. **程序合成**
   - 从规约合成程序
   - 语法引导合成 (SyGuS)

2. **并发验证**
   - 并发程序的验证
   - 死锁检测

3. **实际应用**
   - 验证编译器
   - 操作系统验证

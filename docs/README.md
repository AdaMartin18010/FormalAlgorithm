## 项目概述

本项目构建一个全面、深度、规范的算法知识体系，对标MIT/Stanford/CMU/Berkeley研究生级课程。

## 知识体系结构

```
00-算法规范设计框架        # 体系架构与标准
01-基础理论                # 数学基础
02-递归理论                # 递归函数论
03-形式化证明              # 证明系统
04-算法复杂度              # 复杂度分析
05-类型理论                # 类型系统
06-逻辑系统                # 数理逻辑
07-计算模型                # 可计算性
08-实现示例                # 代码实现
09-算法理论                # 核心算法
10-高级主题                # 前沿算法
11-国际化                  # 国际课程对标
12-应用领域                # 应用专题
13-LeetCode算法面试专题    # LeetCode算法面试专题
```

## 六维内容标准

每个核心知识点包含六个维度：

1. **概念定义** - 形式化定义与自然语言解释
2. **属性** - 性质、定理、复杂度分析
3. **关系** - 概念依赖与层次关系
4. **解释** - 动机、直观、示例
5. **论证** - 非形式化证明思路
6. **形式证明** - 严格数学证明

## 一、规范定义深化

### 1.1 抽象解释的格论基础

**定义 1.1**（完备格 / Complete Lattice）

集合 $L$ 配备偏序 $\sqsubseteq$ 形成**完备格**，当且仅当 $L$ 的任意子集 $S \subseteq L$ 都有上确界 $\bigsqcup S$ 和下确界 $\sqcap S$。完备格有最小元 $\bot = \sqcap L$ 和最大元 $\top = \sqcup L$。

**定义 1.2**（Galois 连接）

设 $(C, \sqsubseteq_C)$ 和 $(A, \sqsubseteq_A)$ 为偏序集。一对单调函数 $\alpha: C \to A$（抽象函数）和 $\gamma: A \to C$（具体化函数）形成**Galois 连接**，记作 $(\alpha, \gamma): C \rightleftharpoons A$，当且仅当：

$$
\forall c \in C, a \in A. \; \alpha(c) \sqsubseteq_A a \; \Leftrightarrow \; c \sqsubseteq_C \gamma(a)
$$

**等价刻画**：

$$
\begin{aligned}
&c \sqsubseteq_C \gamma(\alpha(c)) \quad \text{（具体化扩展）} \\
&\alpha(\gamma(a)) \sqsubseteq_A a \quad \text{（抽象收缩）}
\end{aligned}
$$

### 1.2 具体语义与抽象语义

**定义 1.3**（具体语义域）

设程序状态集合为 $\Sigma$。**具体语义**是状态转移的幂集上的完备格：

$$
\mathcal{D}_C = (\mathcal{P}(\Sigma), \subseteq, \cup, \cap, \varnothing, \Sigma)
$$

程序的语义函数 $F_C: \mathcal{P}(\Sigma) \to \mathcal{P}(\Sigma)$ 定义为：

$$
F_C(X) = \{ \sigma' \mid \exists \sigma \in X. \, \langle C, \sigma \rangle \Downarrow \sigma' \}
$$

程序的所有可达状态集合是 $F_C$ 的**最小不动点**：$\mathsf{lfp}(F_C) = \bigsqcup_{n \geq 0} F_C^n(\varnothing)$。

**定义 1.4**（抽象语义域）

**抽象语义域** $(\mathcal{D}_A, \sqsubseteq_A)$ 是对 $\mathcal{D}_C$ 的有限表示，通过 Galois 连接 $(\alpha, \gamma): \mathcal{D}_C \rightleftharpoons \mathcal{D}_A$ 关联。

### 1.3 抽象转移函数的可靠性

**定义 1.5**（抽象转移函数）

函数 $F_A^\#: \mathcal{D}_A \to \mathcal{D}_A$ 是**可靠的**（sound），当且仅当：

$$
\alpha \circ F_C \sqsubseteq_A F_A^\# \circ \alpha
$$

等价地：

$$
F_C \circ \gamma \sqsubseteq_C \gamma \circ F_A^\#
$$

**定理 1.1**（可靠性保证）

若 $F_A^\#$ 可靠，则：

$$
\mathsf{lfp}(F_C) \subseteq \gamma(\mathsf{lfp}(F_A^\#))
$$

即抽象分析的结果**覆盖**所有具体可达状态。

## 二、模型设计深化

### 2.1 区间抽象域

**定义 2.1**（区间抽象域）

区间抽象域 $A_{\mathrm{int}}$ 的元素是：

$$
A_{\mathrm{int}} = \{ [l, u] \mid l \in \mathbb{Z} \cup \{-\infty\}, u \in \mathbb{Z} \cup \{+\infty\}, l \leq u \} \cup \{\bot\}
$$

其中 $[l, u]$ 表示所有满足 $l \leq x \leq u$ 的整数 $x$。

**Galois 连接**：

$$
\begin{aligned}
\alpha_{\mathrm{int}}(S) &= [\min S, \max S] \quad (S \neq \varnothing) \\
\gamma_{\mathrm{int}}([l, u]) &= \{ x \in \mathbb{Z} \mid l \leq x \leq u \}
\end{aligned}
$$

**定义 2.2**（区间运算）

抽象算术运算定义为：

$$
\begin{aligned}
[l_1, u_1] +^\# [l_2, u_2] &= [l_1 + l_2, u_1 + u_2] \\
[l_1, u_1] -^\# [l_2, u_2] &= [l_1 - u_2, u_1 - l_2] \\
[l_1, u_1] \times^\# [l_2, u_2] &= [\min P, \max P]
\end{aligned}
$$

其中 $P = \{l_1 l_2, l_1 u_2, u_1 l_2, u_1 u_2\}$。

**定理 2.1**（区间运算的可靠性）

对任意 $S_1, S_2 \subseteq \mathbb{Z}$：

$$
\alpha_{\mathrm{int}}(S_1 + S_2) \sqsubseteq \alpha_{\mathrm{int}}(S_1) +^\# \alpha_{\mathrm{int}}(S_2)
$$

### 2.2 符号抽象域

**定义 2.3**（符号抽象域）

符号抽象域 $A_{\mathrm{sign}} = \{\bot, -, 0, +, \top\}$，偏序为：

$$
\bot \sqsubseteq - \sqsubseteq \top, \quad \bot \sqsubseteq 0 \sqsubseteq \top, \quad \bot \sqsubseteq + \sqsubseteq \top
$$

**抽象函数**：

$$
\alpha_{\mathrm{sign}}(S) = \begin{cases}
\bot & S = \varnothing \\
- & \forall x \in S. \, x < 0 \\
0 & S = \{0\} \\
+ & \forall x \in S. \, x > 0 \\
\top & \text{otherwise}
\end{cases}
$$

### 2.3 Widening 与 Narrowing 算子

**定义 2.4**（Widening 算子）

算子 $\nabla: A \times A \to A$ 是**widening**，当且仅当：

1. $a_1 \sqsubseteq a_1 \nabla a_2$；
2. $a_2 \sqsubseteq a_1 \nabla a_2$；
3. 对任意升链 $x_0, x_1, \ldots$，序列 $y_0 = x_0$，$y_{n+1} = y_n \nabla x_{n+1}$ 稳定（即存在 $N$ 使得 $y_{N+1} = y_N$）。

**区间 widening**：

$$
[l_1, u_1] \nabla [l_2, u_2] =
\begin{cases}
[\min(l_1, l_2), \max(u_1, u_2)] & \text{if } l_2 \geq l_1 \text{ and } u_2 \leq u_1 \\
[-\infty, \max(u_1, u_2)] & \text{if } l_2 < l_1 \text{ and } u_2 \leq u_1 \\
[\min(l_1, l_2), +\infty] & \text{if } l_2 \geq l_1 \text{ and } u_2 > u_1 \\
[-\infty, +\infty] & \text{otherwise}
\end{cases}
$$

**定义 2.5**（Narrowing 算子）

算子 $\Delta: A \times A \to A$ 是**narrowing**，当且仅当：

1. $a_1 \nabla a_2 \sqsubseteq a_1 \Delta a_2 \sqsubseteq a_1$；
2. 对任意降链，迭代应用 narrowing 稳定。

## 三、数学符号与推导

### 3.1 ISO 80000 对齐符号

| 符号 | 含义 | 说明 |
|------|------|------|
| $\sqsubseteq$ | 抽象序 | 信息序（精确到近似） |
| $\bigsqcup$ | 上确界 | 合并/连接 |
| $\sqcap$ | 下确界 | 交汇/相遇 |
| $\bot$ | 最小元 | 空信息/不可达 |
| $\top$ | 最大元 | 全信息/未知 |
| $\alpha$ | 抽象函数 | abstraction |
| $\gamma$ | 具体化函数 | concretization |
| $\nabla$ | Widening 算子 | 强制收敛 |
| $\Delta$ | Narrowing 算子 | 逐步精化 |
| $\mathsf{lfp}(F)$ | 最小不动点 | least fixed point |

### 3.2 抽象解释可靠性的推导

**定理 3.1**（抽象解释的可靠性）

设 $(\alpha, \gamma): \mathcal{D}_C \rightleftharpoons \mathcal{D}_A$ 为 Galois 连接，$F_A^\#$ 为可靠的抽象转移函数。则：

$$
\mathsf{lfp}(F_C) \subseteq \gamma(\mathsf{lfp}(F_A^\#))
$$

**证明**：

由 Tarski 不动点定理，$\mathsf{lfp}(F_C) = \bigsqcup_{n \geq 0} F_C^n(\bot_C)$。归纳证明 $F_C^n(\bot_C) \subseteq \gamma((F_A^\#)^n(\bot_A))$：

**基例** $n = 0$：$F_C^0(\bot_C) = \bot_C = \varnothing \subseteq \gamma(\bot_A)$。

**归纳步**：假设 $F_C^n(\bot_C) \subseteq \gamma((F_A^\#)^n(\bot_A))$。则：

$$
\begin{aligned}
F_C^{n+1}(\bot_C) &= F_C(F_C^n(\bot_C)) \\
&\subseteq F_C(\gamma((F_A^\#)^n(\bot_A))) \quad \text{[归纳假设，} F_C \text{ 单调]} \\
&\subseteq \gamma(F_A^\#((F_A^\#)^n(\bot_A))) \quad \text{[可靠性]} \\
&= \gamma((F_A^\#)^{n+1}(\bot_A))
\end{aligned}
$$

由连续性，$\mathsf{lfp}(F_C) = \bigsqcup_n F_C^n(\bot_C) \subseteq \bigsqcup_n \gamma((F_A^\#)^n(\bot_A)) \subseteq \gamma(\mathsf{lfp}(F_A^\#))$。$\square$

### 3.3 区间运算可靠性的推导

**定理 3.2**（区间加法可靠性）

对 $S_1, S_2 \subseteq \mathbb{Z}$：

$$
\{x + y \mid x \in S_1, y \in S_2\} \subseteq \gamma_{\mathrm{int}}(\alpha_{\mathrm{int}}(S_1) +^\# \alpha_{\mathrm{int}}(S_2))
$$

**证明**：设 $\alpha_{\mathrm{int}}(S_1) = [l_1, u_1]$，$\alpha_{\mathrm{int}}(S_2) = [l_2, u_2]$。则 $S_1 \subseteq \{x \mid l_1 \leq x \leq u_1\}$，$S_2 \subseteq \{y \mid l_2 \leq y \leq u_2\}$。

对任意 $x \in S_1, y \in S_2$：

$$
l_1 + l_2 \leq x + y \leq u_1 + u_2
$$

故 $x + y \in [l_1 + l_2, u_1 + u_2] = \alpha_{\mathrm{int}}(S_1) +^\# \alpha_{\mathrm{int}}(S_2)$。$\square$

## 四、示例性代码

### 4.1 Rust：区间抽象域实现

```rust
use std::cmp::{max, min};
use std::fmt;

/// 区间抽象值
#[derive(Clone, Copy, Debug, PartialEq, Eq)]
pub enum Interval {
    Bottom,           // ⊥：空集
    Range(i64, i64),  // [l, u]：闭区间
    Top,              // ⊤：无界
}

impl Interval {
    /// 从具体值构造
    pub fn from_const(v: i64) -> Self {
        Interval::Range(v, v)
    }

    /// 抽象加法
    pub fn add(self, other: Interval) -> Interval {
        match (self, other) {
            (Interval::Bottom, _) | (_, Interval::Bottom) => Interval::Bottom,
            (Interval::Top, _) | (_, Interval::Top) => Interval::Top,
            (Interval::Range(l1, u1), Interval::Range(l2, u2)) => {
                // 检查溢出
                let l = l1.saturating_add(l2);
                let u = u1.saturating_add(u2);
                if l == i64::MIN && (l1 < 0 || l2 < 0) {
                    Interval::Top
                } else if u == i64::MAX && (u1 > 0 || u2 > 0) {
                    Interval::Top
                } else {
                    Interval::Range(l, u)
                }
            }
        }
    }

    /// 抽象减法
    pub fn sub(self, other: Interval) -> Interval {
        match other {
            Interval::Bottom => Interval::Bottom,
            Interval::Top => Interval::Top,
            Interval::Range(l, u) => {
                // [l1, u1] - [l2, u2] = [l1 - u2, u1 - l2]
                self.add(Interval::Range(-u, -l))
            }
        }
    }

    /// 抽象乘法
    pub fn mul(self, other: Interval) -> Interval {
        match (self, other) {
            (Interval::Bottom, _) | (_, Interval::Bottom) => Interval::Bottom,
            (Interval::Top, _) | (_, Interval::Top) => Interval::Top,
            (Interval::Range(l1, u1), Interval::Range(l2, u2)) => {
                let products = [
                    l1.saturating_mul(l2),
                    l1.saturating_mul(u2),
                    u1.saturating_mul(l2),
                    u1.saturating_mul(u2),
                ];
                let l = *products.iter().min().unwrap();
                let u = *products.iter().max().unwrap();
                if l == i64::MIN || u == i64::MAX {
                    Interval::Top
                } else {
                    Interval::Range(l, u)
                }
            }
        }
    }

    /// Widening 算子
    pub fn widen(self, other: Interval) -> Interval {
        match (self, other) {
            (Interval::Bottom, _) => other,
            (_, Interval::Bottom) => self,
            (Interval::Top, _) | (_, Interval::Top) => Interval::Top,
            (Interval::Range(l1, u1), Interval::Range(l2, u2)) => {
                let l = if l2 < l1 { i64::MIN } else { l1 };
                let u = if u2 > u1 { i64::MAX } else { u1 };
                if l == i64::MIN && u == i64::MAX {
                    Interval::Top
                } else {
                    Interval::Range(l, u)
                }
            }
        }
    }

    /// 偏序：self ⊑ other（self 更精确）
    pub fn leq(self, other: Interval) -> bool {
        match (self, other) {
            (Interval::Bottom, _) => true,
            (_, Interval::Top) => true,
            (Interval::Range(l1, u1), Interval::Range(l2, u2)) => l2 <= l1 && u1 <= u2,
            _ => false,
        }
    }

    /// 交汇（meet）
    pub fn meet(self, other: Interval) -> Interval {
        match (self, other) {
            (Interval::Bottom, _) | (_, Interval::Bottom) => Interval::Bottom,
            (Interval::Top, x) | (x, Interval::Top) => x,
            (Interval::Range(l1, u1), Interval::Range(l2, u2)) => {
                let l = max(l1, l2);
                let u = min(u1, u2);
                if l > u { Interval::Bottom } else { Interval::Range(l, u) }
            }
        }
    }
}

impl fmt::Display for Interval {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        match self {
            Interval::Bottom => write!(f, "⊥"),
            Interval::Top => write!(f, "⊤"),
            Interval::Range(l, u) if l == u => write!(f, "[{}]", l),
            Interval::Range(l, u) => write!(f, "[{}, {}]", l, u),
        }
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_interval_add() {
        let a = Interval::Range(1, 3);
        let b = Interval::Range(2, 4);
        assert_eq!(a.add(b), Interval::Range(3, 7));
    }

    #[test]
    fn test_interval_widen() {
        let a = Interval::Range(1, 5);
        let b = Interval::Range(0, 6);
        assert_eq!(a.widen(b), Interval::Range(i64::MIN, i64::MAX));
    }
}
```

## 五、形式化证明

### 5.1 Galois 连接与最佳抽象

**定理 5.1**（最佳抽象的存在性）

设 $\alpha: C \to A$ 和 $\gamma: A \to C$ 形成 Galois 连接。则 $\alpha$ 是**最佳抽象**：对任何其他抽象 $\alpha': C \to A$，若 $\forall c. \, c \sqsubseteq_C \gamma(\alpha'(c))$，则 $\forall c. \, \alpha(c) \sqsubseteq_A \alpha'(c)$。

**证明**：设 $\alpha'$ 满足条件。对任意 $c \in C$，由假设 $c \sqsubseteq_C \gamma(\alpha'(c))$。由 Galois 连接定义，$\alpha(c) \sqsubseteq_A \alpha'(c)$。$\square$

### 5.2 Widening 保证终止性

**定理 5.2**（Widening 迭代终止）

设 $\nabla$ 是抽象域 $A$ 上的 widening 算子。对任意序列 $(a_n)_{n \geq 0}$，定义 $w_0 = a_0$，$w_{n+1} = w_n \nabla a_{n+1}$。则序列 $(w_n)$ 必然稳定（即存在 $N$ 使得 $w_{N+1} = w_N$）。

**证明**：由 widening 定义，$w_n \sqsubseteq w_{n+1}$（升链）。由 widening 的有限升链条件，该升链必有限。$\square$

### 5.3 抽象解释的完备性极限

**定理 5.3**（Rice 定理的抽象解释推论）

不存在完美的（既可靠又精确的）静态分析算法能判定任意程序的任意非平凡语义属性。

**证明**：由 Rice 定理，程序的非平凡语义属性不可判定。抽象解释通过近似使分析可计算，但近似必然引入信息损失。任何试图精确判定非平凡属性的分析要么不可计算，要么不可靠。$\square$

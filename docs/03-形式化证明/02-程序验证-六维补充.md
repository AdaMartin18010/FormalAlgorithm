## 一、规范定义深化

### 1.1 Hoare 三元组的严格语义定义

**定义 1.1**（程序状态 / Program State）

设 $\mathrm{Var}$ 为程序变量集合，$\mathrm{Val}$ 为值域。一个**程序状态**（state）是映射：

$$
\sigma : \mathrm{Var} \to \mathrm{Val}
$$

所有状态的集合记为 $\Sigma = \mathrm{Val}^{\mathrm{Var}}$。对表达式 $e$，其在状态 $\sigma$ 下的求值记为 $\mathcal{A}\llbracket e \rrbracket \sigma$；对布尔表达式 $B$，其真值记为 $\mathcal{B}\llbracket B \rrbracket \sigma \in \{\mathbf{tt}, \mathbf{ff}\}$。

**定义 1.2**（Hoare 三元组 / Hoare Triple）

设 $P, Q$ 为状态断言（一阶公式），$C$ 为命令。Hoare 三元组 $\{P\} \, C \, \{Q\}$ 的语义定义为：

$$
\models \{P\} \, C \, \{Q\} \quad \Leftrightarrow \quad \forall \sigma, \sigma'. \; \sigma \models P \land \langle C, \sigma \rangle \Downarrow \sigma' \Rightarrow \sigma' \models Q
$$

其中 $\langle C, \sigma \rangle \Downarrow \sigma'$ 表示命令 $C$ 在状态 $\sigma$ 下执行终止于 $\sigma'$ 的操作语义关系。

**定义 1.3**（部分正确性与完全正确性）

| 正确性类型 | 记号 | 定义 |
|-----------|------|------|
| 部分正确性 | $\models_{\mathrm{par}} \{P\} C \{Q\}$ | $\forall \sigma. \; \sigma \models P \Rightarrow (\exists \sigma'. \langle C, \sigma \rangle \Downarrow \sigma' \Rightarrow \sigma' \models Q)$ |
| 完全正确性 | $\models_{\mathrm{tot}} [P] C [Q]$ | $\forall \sigma. \; \sigma \models P \Rightarrow (\exists \sigma'. \langle C, \sigma \rangle \Downarrow \sigma' \land \sigma' \models Q)$ |

关键区别在于：部分正确性**不保证终止**，完全正确性要求**必然终止且满足后置条件**。

### 1.2 最弱前置条件的格论定义

**定义 1.4**（最弱前置条件 / Weakest Precondition, wp）

对命令 $C$ 和后置断言 $Q$，最弱前置条件 $wp(C, Q)$ 定义为：

$$
wp(C, Q) = \{ \sigma \in \Sigma \mid \forall \sigma'. \langle C, \sigma \rangle \Downarrow \sigma' \Rightarrow \sigma' \models Q \}
$$

等价地，$wp(C, Q)$ 是使 $\{P\} C \{Q\}$ 成立的所有前置条件 $P$ 的集合中**最弱**（即最一般）的断言。

**定义 1.5**（最强后置条件 / Strongest Postcondition, sp）

$$
sp(P, C) = \{ \sigma' \in \Sigma \mid \exists \sigma. \; \sigma \models P \land \langle C, \sigma \rangle \Downarrow \sigma' \}
$$

**定理 1.1**（wp 与 sp 的对偶性）

$$
\models \{P\} C \{Q\} \quad \Leftrightarrow \quad P \Rightarrow wp(C, Q) \quad \Leftrightarrow \quad sp(P, C) \Rightarrow Q
$$

## 二、模型设计深化

### 2.1 指称语义模型

**定义 2.1**（命令的指称语义）

命令 $C$ 的指称是一个偏函数（partial function）：

$$
\mathcal{M}\llbracket C \rrbracket : \Sigma \rightharpoonup \Sigma
$$

核心命令的指称定义如下：

$$
\begin{aligned}
\mathcal{M}\llbracket \mathbf{skip} \rrbracket(\sigma) &= \sigma \\
\mathcal{M}\llbracket x := e \rrbracket(\sigma) &= \sigma[x \mapsto \mathcal{A}\llbracket e \rrbracket \sigma] \\
\mathcal{M}\llbracket C_1; C_2 \rrbracket &= \mathcal{M}\llbracket C_2 \rrbracket \circ \mathcal{M}\llbracket C_1 \rrbracket \\
\mathcal{M}\llbracket \mathbf{if} \, B \, \mathbf{then} \, C_1 \, \mathbf{else} \, C_2 \rrbracket(\sigma) &=
  \begin{cases}
    \mathcal{M}\llbracket C_1 \rrbracket(\sigma) & \text{if } \mathcal{B}\llbracket B \rrbracket \sigma = \mathbf{tt} \\
    \mathcal{M}\llbracket C_2 \rrbracket(\sigma) & \text{if } \mathcal{B}\llbracket B \rrbracket \sigma = \mathbf{ff}
  \end{cases}
\end{aligned}
$$

while 循环的指称通过**最小不动点**定义：

$$
\mathcal{M}\llbracket \mathbf{while} \, B \, \mathbf{do} \, C \rrbracket = \mathsf{lfp}(F)
$$

其中 $F: (\Sigma \rightharpoonup \Sigma) \to (\Sigma \rightharpoonup \Sigma)$ 定义为：

$$
F(f)(\sigma) =
  \begin{cases}
    f(\mathcal{M}\llbracket C \rrbracket(\sigma)) & \text{if } \mathcal{B}\llbracket B \rrbracket \sigma = \mathbf{tt} \\
    \sigma & \text{if } \mathcal{B}\llbracket B \rrbracket \sigma = \mathbf{ff} \\
    \text{undefined} & \text{if } \mathcal{M}\llbracket C \rrbracket(\sigma) \text{ undefined}
  \end{cases}
$$

### 2.2 操作语义模型（大步语义）

**定义 2.2**（大步操作语义 / Big-Step Operational Semantics）

关系 $\langle C, \sigma \rangle \Downarrow \sigma'$ 由以下推理规则归纳定义：

$$
\begin{aligned}
&\overline{\langle \mathbf{skip}, \sigma \rangle \Downarrow \sigma} \quad \text{(Skip)} \\[4pt]
&\overline{\langle x := e, \sigma \rangle \Downarrow \sigma[x \mapsto \mathcal{A}\llbracket e \rrbracket \sigma]} \quad \text{(Assign)} \\[4pt]
&\frac{\langle C_1, \sigma \rangle \Downarrow \sigma_1 \quad \langle C_2, \sigma_1 \rangle \Downarrow \sigma'}{\langle C_1; C_2, \sigma \rangle \Downarrow \sigma'} \quad \text{(Seq)} \\[4pt]
&\frac{\mathcal{B}\llbracket B \rrbracket \sigma = \mathbf{tt} \quad \langle C_1, \sigma \rangle \Downarrow \sigma'}{\langle \mathbf{if} \, B \, \mathbf{then} \, C_1 \, \mathbf{else} \, C_2, \sigma \rangle \Downarrow \sigma'} \quad \text{(IfTrue)} \\[4pt]
&\frac{\mathcal{B}\llbracket B \rrbracket \sigma = \mathbf{ff} \quad \langle C_2, \sigma \rangle \Downarrow \sigma'}{\langle \mathbf{if} \, B \, \mathbf{then} \, C_1 \, \mathbf{else} \, C_2, \sigma \rangle \Downarrow \sigma'} \quad \text{(IfFalse)} \\[4pt]
&\frac{\mathcal{B}\llbracket B \rrbracket \sigma = \mathbf{ff}}{\langle \mathbf{while} \, B \, \mathbf{do} \, C, \sigma \rangle \Downarrow \sigma} \quad \text{(WhileFalse)} \\[4pt]
&\frac{\mathcal{B}\llbracket B \rrbracket \sigma = \mathbf{tt} \quad \langle C, \sigma \rangle \Downarrow \sigma_1 \quad \langle \mathbf{while} \, B \, \mathbf{do} \, C, \sigma_1 \rangle \Downarrow \sigma'}{\langle \mathbf{while} \, B \, \mathbf{do} \, C, \sigma \rangle \Downarrow \sigma'} \quad \text{(WhileTrue)}
\end{aligned}
$$

### 2.3 Galois 连接视角

**定义 2.3**（断言格与 wp 的 Galois 连接）

设 $\mathrm{Assn}$ 为断言集合，按逻辑蕴涵 $\Rightarrow$ 形成偏序。对固定命令 $C$，映射 $wp(C, -): \mathrm{Assn} \to \mathrm{Assn}$ 与 $sp(-, C): \mathrm{Assn} \to \mathrm{Assn}$ 形成**Galois 连接**：

$$
\forall P, Q. \; sp(P, C) \Rightarrow Q \quad \Leftrightarrow \quad P \Rightarrow wp(C, Q)
$$

这一视角将程序验证与**抽象解释**（abstract interpretation）理论统一在同一格论语义框架下。

## 三、数学符号与推导

### 3.1 ISO 80000 对齐符号

| 符号 | 含义 | 说明 |
|------|------|------|
| $\{P\} C \{Q\}$ | Hoare 三元组 | 部分正确性 |
| $[P] C [Q]$ | Hoare 三元组（方括号） | 完全正确性 |
| $wp(C, Q)$ | 最弱前置条件 | Dijkstra 记号 |
| $\langle C, \sigma \rangle \Downarrow \sigma'$ | 大步语义 | 操作语义关系 |
| $\mathcal{M}\llbracket C \rrbracket$ | 命令指称 | 指称语义 |
| $\sigma[x \mapsto v]$ | 状态更新 | 有限映射更新 |
| $\models_{\mathrm{par}}$ | 部分正确性语义满足 | 元语言关系 |
| $\models_{\mathrm{tot}}$ | 完全正确性语义满足 | 元语言关系 |

### 3.2 赋值公理的可靠性推导

**定理 3.1**（赋值公理的可靠性）

对任何变量 $x$、表达式 $e$ 和后置条件 $Q$：

$$
\models \{Q[e/x]\} \; x := e \; \{Q\}
$$

**证明**：设 $\sigma \models Q[e/x]$。执行 $x := e$ 后，新状态为 $\sigma' = \sigma[x \mapsto \mathcal{A}\llbracket e \rrbracket \sigma]$。

由**替换引理**（Substitution Lemma）：

$$
\sigma \models Q[e/x] \quad \Leftrightarrow \quad \sigma[x \mapsto \mathcal{A}\llbracket e \rrbracket \sigma] \models Q
$$

因此 $\sigma' \models Q$。$\square$

### 3.3 循环规则可靠性的归纳证明

**定理 3.2**（循环不变式规则的可靠性）

若 $\models \{I \land B\} \, C \, \{I\}$，则：

$$
\models \{I\} \; \mathbf{while} \, B \, \mathbf{do} \, C \; \{I \land \neg B\}
$$

**证明**：对 while 循环的迭代次数 $n$ 进行归纳。

**基例** $n = 0$：若 $\sigma \models I$ 且 $\mathcal{B}\llbracket B \rrbracket \sigma = \mathbf{ff}$，则循环立即终止于 $\sigma$，显然 $\sigma \models I \land \neg B$。

**归纳步**：假设对 $n$ 次迭代可靠性成立。考虑 $n+1$ 次迭代：

1. $\sigma \models I$ 且 $\mathcal{B}\llbracket B \rrbracket \sigma = \mathbf{tt}$；
2. 由前提 $\models \{I \land B\} C \{I\}$，执行 $C$ 后得到 $\sigma_1 \models I$；
3. 从 $\sigma_1$ 开始的剩余 $n$ 次迭代由归纳假设保证终止于 $\sigma' \models I \land \neg B$。

故对 $n+1$ 次迭代可靠性亦成立。由数学归纳法，定理得证。$\square$

### 3.4 最弱前置条件的递归方程

**定理 3.3**（wp 的结构性递归定义）

$$
\begin{aligned}
wp(\mathbf{skip}, Q) &= Q \\
wp(x := e, Q) &= Q[e/x] \\
wp(C_1; C_2, Q) &= wp(C_1, wp(C_2, Q)) \\
wp(\mathbf{if} \, B \, \mathbf{then} \, C_1 \, \mathbf{else} \, C_2, Q) &= (B \Rightarrow wp(C_1, Q)) \land (\neg B \Rightarrow wp(C_2, Q)) \\
wp(\mathbf{while} \, B \, \mathbf{do} \, C, Q) &= \mu X. \, (\neg B \land Q) \lor (B \land wp(C, X))
\end{aligned}
$$

其中 $\mu X. \, \phi(X)$ 表示在断言格上的**最小不动点**。

inductive AExp

  | le  (a1 a2 : AExp)
  | not (b : BExp)
  | and (b1 b2 : BExp)
  deriving Repr

inductive Com
  | skip
  | assign (x : String) (a : AExp)
  | seq    (c1 c2 : Com)
  | ifThenElse (b : BExp) (c1 c2 : Com)
  | whileDo  (b : BExp) (c : Com)
  deriving Repr

def State := String → Nat

def State.update (σ : State) (x : String) (v : Nat) : State :=

def Assertion := State → Prop

  pre  : Assertion
  com  : Com
  post : Assertion

/-- 循环不变式标注（用于验证） -/
structure WhileSpec where
  com   : Com
  inv   : Assertion
  varnt : AExp  -- 变式函数（todo: 良基性）

### 4.2 Rust：最弱前置条件计算器

```rust
use std::collections::HashMap;

#[derive(Clone, Debug, PartialEq)]
pub enum Expr {
    Var(String),
    Num(i64),
    Add(Box<Expr>, Box<Expr>),
}

#[derive(Clone, Debug, PartialEq)]
pub enum BExpr {
    True,
    False,
    Eq(Box<Expr>, Box<Expr>),
    Le(Box<Expr>, Box<Expr>),
    Not(Box<BExpr>),
}

#[derive(Clone, Debug, PartialEq)]
pub enum Command {
    Skip,
    Assign(String, Expr),
    Seq(Box<Command>, Box<Command>),
    If(BExpr, Box<Command>, Box<Command>),
    While(BExpr, Box<Command>),
}

#[derive(Clone, Debug, PartialEq)]
pub enum Assertion {
    True,
    Eq(Box<Expr>, Box<Expr>),
    Le(Box<Expr>, Box<Expr>),
    And(Box<Assertion>, Box<Assertion>),
    Implies(Box<Assertion>, Box<Assertion>),
    Not(Box<Assertion>),
}

pub fn subst_expr(e: &Expr, x: &str, repl: &Expr) -> Expr {
    match e {
        Expr::Var(y) if y == x => repl.clone(),
        Expr::Var(_) => e.clone(),
        Expr::Num(_) => e.clone(),
        Expr::Add(e1, e2) => Expr::Add(
            Box::new(subst_expr(e1, x, repl)),
            Box::new(subst_expr(e2, x, repl)),
        ),
    }
}

pub fn subst_assert(a: &Assertion, x: &str, repl: &Expr) -> Assertion {
    match a {
        Assertion::True => Assertion::True,
        Assertion::Eq(e1, e2) => Assertion::Eq(
            Box::new(subst_expr(e1, x, repl)),
            Box::new(subst_expr(e2, x, repl)),
        ),
        Assertion::Le(e1, e2) => Assertion::Le(
            Box::new(subst_expr(e1, x, repl)),
            Box::new(subst_expr(e2, x, repl)),
        ),
        Assertion::And(a1, a2) => Assertion::And(
            Box::new(subst_assert(a1, x, repl)),
            Box::new(subst_assert(a2, x, repl)),
        ),
        Assertion::Implies(a1, a2) => Assertion::Implies(
            Box::new(subst_assert(a1, x, repl)),
            Box::new(subst_assert(a2, x, repl)),
        ),
        Assertion::Not(a1) => Assertion::Not(Box::new(subst_assert(a1, x, repl))),
    }
}

pub fn wp(cmd: &Command, post: &Assertion) -> Assertion {
    match cmd {
        Command::Skip => post.clone(),
        Command::Assign(x, e) => subst_assert(post, x, e),
        Command::Seq(c1, c2) => {
            let wp2 = wp(c2, post);
            wp(c1, &wp2)
        }
        Command::If(b, c1, c2) => {
            let wp1 = wp(c1, post);
            let wp2 = wp(c2, post);
            let b_assert = bexpr_to_assert(b);
            Assertion::And(
                Box::new(Assertion::Implies(Box::new(b_assert.clone()), Box::new(wp1))),
                Box::new(Assertion::Implies(
                    Box::new(Assertion::Not(Box::new(b_assert))),
                    Box::new(wp2),
                )),
            )
        }
        Command::While(_, _) => {
            // 需要用户提供不变式，这里简化返回 True
            Assertion::True
        }
    }
}

fn bexpr_to_assert(b: &BExpr) -> Assertion {
    match b {
        BExpr::True => Assertion::True,
        BExpr::False => Assertion::Not(Box::new(Assertion::True)),
        BExpr::Eq(e1, e2) => Assertion::Eq(e1.clone(), e2.clone()),
        BExpr::Le(e1, e2) => Assertion::Le(e1.clone(), e2.clone()),
        BExpr::Not(b1) => Assertion::Not(Box::new(bexpr_to_assert(b1))),
        BExpr::And(_, _) => todo!(),
    }
}
```

## 五、形式化证明

### 5.1 Cook 相对完备性定理

**定理 5.1**（Cook 相对完备性 [Cook1978]）

设 $\mathcal{L}$ 为一阶逻辑语言，$T$ 为关于程序状态的**充分表达理论**（adequately expressive theory）。对 IMP 程序 $C$ 和断言 $P, Q \in \mathcal{L}$：

$$
\models_{\mathrm{par}} \{P\} C \{Q\} \quad \Rightarrow \quad T \vdash \{P\} C \{Q\}
$$

**证明概要**：

**步骤 1**：证明对每个命令 $C$ 和后置条件 $Q$，$wp(C, Q)$ 可以在理论 $T$ 中表达为一阶公式。对基本命令（skip、赋值、顺序、条件），由 wp 的递归定义直接可表达。对 while 循环，利用哥德尔编码或算术化技术将迭代展开表达为不动点。

**步骤 2**：证明对任何 $C$ 和 $Q$，$\vdash \{wp(C, Q)\} C \{Q\}$。对基本命令直接由公理得到；对 while 循环，构造归纳论证。

**步骤 3**：已知 $\models_{\mathrm{par}} \{P\} C \{Q\}$，则 $P \Rightarrow wp(C, Q)$ 在 $T$ 中成立。由步骤 2 和推论规则：

$$
\frac{P \Rightarrow wp(C, Q) \quad \{wp(C, Q)\} C \{Q\} \quad Q \Rightarrow Q}{\{P\} C \{Q\}}
$$

故 $T \vdash \{P\} C \{Q\}$。$\square$

### 5.2 阶乘程序的完全正确性证明

**程序**：

```
{ n ≥ 0 }
i := 0;
f := 1;
while i < n do
  { inv: f = i! ∧ i ≤ n }
  i := i + 1;
  f := f * i
end
{ f = n! }
```

**形式化验证**：

1. **初始化**：$wp(i:=0; f:=1, f = i! \land i \leq n) = (1 = 0! \land 0 \leq n) = (n \geq 0)$。前置条件足够。

2. **保持**：假设 $f = i! \land i \leq n \land i < n$。执行循环体后：
   - $i' = i + 1$
   - $f' = f \cdot i' = i! \cdot (i+1) = (i+1)!$
   - $i' = i+1 \leq n$（由 $i < n$）
   - 故 $f' = i'! \land i' \leq n$

3. **终止性**：取变式函数 $V = n - i$。
   - $V$ 取值于 $\mathbb{N}$（良基集）；
   - 每次迭代 $i$ 增加 1，故 $V$ 严格递减；
   - 由 $i \leq n$（不变式）且 $i < n$（循环条件），$V > 0$；
   - 因此循环必然终止。

4. **后条件**：循环终止时 $i \leq n \land \neg(i < n)$，即 $i = n$。由不变式 $f = i!$，得 $f = n!$。$\square$

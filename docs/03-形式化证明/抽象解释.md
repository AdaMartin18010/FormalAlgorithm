## 1. 引言

抽象解释（Abstract Interpretation）是由 Patrick Cousot 和 Radhia Cousot 于 1970 年代提出的程序分析理论框架 [Cousot1977][Cousot1996]。它通过抽象具体语义来构建可计算的近似分析。

## 2. 基本框架

### 2.1 具体语义与抽象语义

**具体域** $(C, \leq_C)$：程序实际执行的语义
**抽象域** $(A, \leq_A)$：可计算的近似语义

**抽象函数** $\alpha: C \to A$：将具体值映射到抽象值
**具体化函数** $\gamma: A \to C$：将抽象值映射回具体值的集合

### 2.2 Galois连接

**定义 2.1 (Galois连接)** [Cousot1977][DaveyPriestley2002]：$(\alpha, \gamma)$ 形成Galois连接，如果：
$$\forall c \in C, a \in A: \alpha(c) \leq_A a \Leftrightarrow c \leq_C \gamma(a)$$

**性质**：

- $\alpha$ 单调
- $\gamma$ 单调
- $\alpha \circ \gamma \sqsubseteq id_A$（抽象是近似）
- $id_C \sqsubseteq \gamma \circ \alpha$（具体化包含原始值）

**示例**：区间抽象

- $\alpha(\{1,2,3,5\}) = [1,5]$
- $\gamma([1,5]) = \{1,2,3,4,5\}$

## 3. 抽象转移函数

**定义 3.1 (抽象转移函数)** [Cousot1977]：对于具体转移函数 $f_C: C \to C$，抽象转移函数 $f_A: A \to A$ 是可靠的，如果：
$$\alpha \circ f_C \leq_A f_A \circ \alpha$$

**直观**：在抽象域中的计算不会遗漏具体执行的可能性。

## 4. 常见抽象域

### 4.1 区间抽象

**抽象值**：$[l, u]$，表示值在 $l$ 和 $u$ 之间

**抽象运算**：

- $[l_1, u_1] +^\# [l_2, u_2] = [l_1+l_2, u_1+u_2]$
- $[l_1, u_1] \times^\# [l_2, u_2] = [\min, \max]$（考虑所有组合）

**定理 4.1** [Cousot1977]：区间运算是可靠的抽象。

### 4.2 符号抽象

**抽象值**：$\{+, 0, -, \top, \bot\}$

**应用**：检测除零错误。

### 4.3 关系抽象（八边形）

**抽象值**：$\pm x \pm y \leq c$

**优势**：捕获变量间简单关系。

## 5. 终止性保证

### 5.1 Widening

**问题**：抽象域可能无限升链，迭代不终止。

**Widening算子** $\nabla: A \times A \to A$：

- $a_1 \leq a_1 \nabla a_2$
- $a_2 \leq a_1 \nabla a_2$
- 有限升链条件：任何升链 $a_0, a_0 \nabla a_1, \cdots$ 都有限

**示例（区间）**：
$$[l_1, u_1] \nabla [l_2, u_2] = [\text{if } l_1 < l_2 \text{ then } -\infty \text{ else } l_1, \text{if } u_1 > u_2 \text{ then } +\infty \text{ else } u_1]$$

### 5.2 Narrowing

**问题**：Widening可能过度近似。

**Narrowing算子** $\Delta$：逐步精细化近似。

## 6. 应用

### 6.1 ASTREE静态分析器

**应用**：验证航空电子代码 absence of runtime errors。
**技术**：区间抽象、关系抽象、内存抽象的组合。
**成果**：验证空客A340飞行控制软件。

### 6.2 编译器优化

**应用**：常量传播、死代码消除。
**方法**：稀疏条件常量传播（SCCP）使用抽象解释框架。

### 6.3 程序验证

**应用**：验证数组越界、空指针解引用。
**工具**：Facebook Infer使用分离逻辑抽象。

## 7. 理论性质

**定理 7.1 (可靠性)** [Cousot1977]：抽象解释的分析结果包含所有可能的具体执行。

**定理 7.2 (完备性极限)** [Cousot1977][Rice1953]：完美精确的分析是不可判定的。

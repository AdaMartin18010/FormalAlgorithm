## 一、规范定义深化

### 1.1 IMP 语言的抽象语法

**定义 1.1**（IMP 语言语法）

设 $\mathrm{Loc}$ 为位置（变量名）集合，$\mathrm{Aexp}$ 为算术表达式，$\mathrm{Bexp}$ 为布尔表达式。IMP 命令的抽象语法为：

$$
C ::= \mathbf{skip} \mid X := a \mid C_1; C_2 \mid \mathbf{if} \, b \, \mathbf{then} \, C_1 \, \mathbf{else} \, C_2 \mid \mathbf{while} \, b \, \mathbf{do} \, C
$$

其中 $X \in \mathrm{Loc}$，$a \in \mathrm{Aexp}$，$b \in \mathrm{Bexp}$。

**定义 1.2**（断言语言）

断言 $P, Q, \ldots$ 是标准一阶逻辑公式，允许：

- 算术项：变量、常数、$+, -, \cdot$；
- 关系：$=, <, \leq, >, \geq$；
- 逻辑连接词：$\land, \lor, \neg, \rightarrow, \forall, \exists$。

对状态 $\sigma$，$\sigma \models P$ 表示 $P$ 在 $\sigma$ 下为真。

### 1.2 霍尔逻辑的完整公理系统

**定义 1.3**（霍尔逻辑的推导关系）

霍尔逻辑的推导关系 $\vdash \{P\} C \{Q\}$ 由以下公理和规则定义：

**公理**：

$$
\overline{\{P\} \, \mathbf{skip} \, \{P\}} \quad \text{(Skip)} \qquad
\overline{\{Q[a/X]\} \, X := a \, \{Q\}} \quad \text{(Assign)}
$$

**结构规则**：

$$
\frac{\{P\} C_1 \{R\} \quad \{R\} C_2 \{Q\}}{\{P\} C_1; C_2 \{Q\}} \quad \text{(Seq)}
$$

**控制流规则**：

$$
\frac{\{P \land b\} C_1 \{Q\} \quad \{P \land \neg b\} C_2 \{Q\}}{\{P\} \mathbf{if} \, b \, \mathbf{then} \, C_1 \, \mathbf{else} \, C_2 \{Q\}} \quad \text{(If)}
$$

$$
\frac{\{I \land b\} C \{I\}}{\{I\} \mathbf{while} \, b \, \mathbf{do} \, C \{I \land \neg b\}} \quad \text{(While)}
$$

**推论规则**：

$$
\frac{P \Rightarrow P' \quad \{P'\} C \{Q'\} \quad Q' \Rightarrow Q}{\{P\} C \{Q\}} \quad \text{(Consequence)}
$$

### 1.3 完全正确性的变式函数

**定义 1.4**（变式函数 / Variant Function）

变式函数 $V$ 是从程序状态到良基集 $(W, <)$（通常为 $(\mathbb{N}, <)$）的映射。在 while 循环的完全正确性证明中，要求：

1. $I \land b \Rightarrow V \geq 0$（有界性）；
2. $\{I \land b \land V = v_0\} C \{I \land V < v_0\}$（严格递减）。

**定义 1.5**（完全正确性 while 规则）

$$
\frac{P \Rightarrow I \quad \{I \land b \land V = v_0\} C \{I \land V < v_0\} \quad I \land b \Rightarrow V \geq 0}{[P] \mathbf{while} \, b \, \mathbf{do} \, C [I \land \neg b]}
$$

其中方括号 $[P] C [Q]$ 表示完全正确性。

## 二、模型设计深化

### 2.1 指称语义：偏函数模型

**定义 2.1**（命令的指称）

命令 $C$ 的指称是状态上的偏函数：

$$
\mathcal{C}\llbracket C \rrbracket : \Sigma \rightharpoonup \Sigma
$$

核心定义：

$$
\begin{aligned}
\mathcal{C}\llbracket \mathbf{skip} \rrbracket &= \mathrm{id}_{\Sigma} \\
\mathcal{C}\llbracket X := a \rrbracket(\sigma) &= \sigma[X \mapsto \mathcal{A}\llbracket a \rrbracket\sigma] \\
\mathcal{C}\llbracket C_1; C_2 \rrbracket &= \mathcal{C}\llbracket C_2 \rrbracket \circ \mathcal{C}\llbracket C_1 \rrbracket \\
\mathcal{C}\llbracket \mathbf{if} \, b \, \mathbf{then} \, C_1 \, \mathbf{else} \, C_2 \rrbracket(\sigma) &=
  \begin{cases}
    \mathcal{C}\llbracket C_1 \rrbracket(\sigma) & \text{if } \mathcal{B}\llbracket b \rrbracket\sigma = \mathbf{tt} \\
    \mathcal{C}\llbracket C_2 \rrbracket(\sigma) & \text{if } \mathcal{B}\llbracket b \rrbracket\sigma = \mathbf{ff}
  \end{cases}
\end{aligned}
$$

while 循环的指称通过**最小不动点**在域 $(\Sigma \rightharpoonup \Sigma, \sqsubseteq)$ 上定义：

$$
\mathcal{C}\llbracket \mathbf{while} \, b \, \mathbf{do} \, C \rrbracket = \mathsf{lfp}(\Phi)
$$

其中：

$$
\Phi(f)(\sigma) =
  \begin{cases}
    f(\mathcal{C}\llbracket C \rrbracket(\sigma)) & \text{if } \mathcal{B}\llbracket b \rrbracket\sigma = \mathbf{tt} \\
    \sigma & \text{if } \mathcal{B}\llbracket b \rrbracket\sigma = \mathbf{ff} \\
    \text{undefined} & \text{otherwise}
  \end{cases}
$$

### 2.2 操作语义：大步与小步

**定义 2.2**（大步操作语义）

关系 $\langle C, \sigma \rangle \Downarrow \sigma'$ 的定义见主文档。其关键性质：

**定理 2.1**（大步语义的确定性）

若 $\langle C, \sigma \rangle \Downarrow \sigma_1$ 且 $\langle C, \sigma \rangle \Downarrow \sigma_2$，则 $\sigma_1 = \sigma_2$。

**定义 2.3**（小步操作语义）

小步语义 $\langle C, \sigma \rangle \rightarrow \langle C', \sigma' \rangle$ 描述单步执行：

$$
\begin{aligned}
&\overline{\langle X := a, \sigma \rangle \rightarrow \langle \mathbf{skip}, \sigma[X \mapsto \mathcal{A}\llbracket a \rrbracket\sigma] \rangle} \\[4pt]
&\frac{\langle C_1, \sigma \rangle \rightarrow \langle C_1', \sigma' \rangle}{\langle C_1; C_2, \sigma \rangle \rightarrow \langle C_1'; C_2, \sigma' \rangle} \\[4pt]
&\overline{\langle \mathbf{skip}; C_2, \sigma \rangle \rightarrow \langle C_2, \sigma \rangle}
\end{aligned}
$$

**定理 2.2**（大步与小步的等价性）

$\langle C, \sigma \rangle \Downarrow \sigma' \Leftrightarrow \langle C, \sigma \rangle \rightarrow^* \langle \mathbf{skip}, \sigma' \rangle$。

### 2.3 公理化语义与 Galois 连接

**定义 2.4**（断言格）

断言集合 $\mathrm{Assn}$ 形成完备格 $(\mathrm{Assn}, \Rightarrow, \land, \lor, \top, \bot)$。最弱前置条件 $wp(C, -)$ 和最强后置条件 $sp(-, C)$ 形成 Galois 连接：

$$
\forall P, Q. \; sp(P, C) \Rightarrow Q \; \Leftrightarrow \; P \Rightarrow wp(C, Q)
$$

此连接建立了指称语义与公理化语义之间的深层对应。

### 3.1 ISO 80000 对齐符号

| 符号 | 含义 | 说明 |
|------|------|------|
| $\{P\} C \{Q\}$ | Hoare 三元组 | 部分正确性 |
| $[P] C [Q]$ | Hoare 三元组 | 完全正确性 |
| $wp(C, Q)$ | 最弱前置条件 | Dijkstra 记号 |
| $sp(P, C)$ | 最强后置条件 | Floyd 记号 |

| $\rightarrow^*$ | 多步归约 | Kleene 星 |
| $\mathcal{C}\llbracket C \rrbracket$ | 命令指称 | 指称语义 |

### 3.2 可靠性定理的完整证明

**定理 3.1**（霍尔逻辑的可靠性）

若 $\vdash \{P\} C \{Q\}$，则 $\models_{\mathrm{par}} \{P\} C \{Q\}$。

**证明**（对推导树高度归纳）：

**基例**：公理。

- Skip：$\mathcal{C}\llbracket \mathbf{skip} \rrbracket(\sigma) = \sigma$。若 $\sigma \models P$，则终止状态满足 $P$。
- Assign：由替换引理，$\sigma \models Q[a/X] \Leftrightarrow \sigma[X \mapsto \mathcal{A}\llbracket a \rrbracket\sigma] \models Q$。

**归纳步**：

- Seq：设 $\vdash \{P\} C_1 \{R\}$ 且 $\vdash \{R\} C_2 \{Q\}$。由归纳假设，两者均语义有效。对任意 $\sigma \models P$，若 $\mathcal{C}\llbracket C_1; C_2 \rrbracket(\sigma)$ 有定义，则存在 $\sigma_1$ 使得 $\mathcal{C}\llbracket C_1 \rrbracket(\sigma) = \sigma_1$ 且 $\mathcal{C}\llbracket C_2 \rrbracket(\sigma_1) = \sigma'$。由第一个前提，$\sigma_1 \models R$；由第二个前提，$\sigma' \models Q$。

- While：设 $\vdash \{I \land b\} C \{I\}$ 且语义有效。对任意 $\sigma \models I$，若 while 循环终止，则存在有限迭代序列。对迭代次数 $n$ 归纳：
  - $n = 0$：$\sigma \models \neg b$，直接满足 $I \land \neg b$。
  - $n = k+1$：$\sigma \models b$，执行 $C$ 得 $\sigma_1 \models I$（由归纳假设），剩余 $k$ 次迭代由归纳假设保证终止于满足 $I \land \neg b$ 的状态。

- Consequence：由 $\models$ 的单调性直接可得。$\square$

### 3.3 wp 的单调性与分配律

**定理 3.2**（wp 的基本性质）

对任意命令 $C$ 和断言 $Q_1, Q_2$：

1. **单调性**：$Q_1 \Rightarrow Q_2 \Rightarrow wp(C, Q_1) \Rightarrow wp(C, Q_2)$；
2. **合取分配**：$wp(C, Q_1 \land Q_2) \Leftrightarrow wp(C, Q_1) \land wp(C, Q_2)$；
3. **析取弱分配**：$wp(C, Q_1) \lor wp(C, Q_2) \Rightarrow wp(C, Q_1 \lor Q_2)$。

**证明**：

1. 设 $Q_1 \Rightarrow Q_2$。若 $\sigma \models wp(C, Q_1)$，则对所有 $\sigma'$，$\langle C, \sigma \rangle \Downarrow \sigma' \Rightarrow \sigma' \models Q_1$。由 $Q_1 \Rightarrow Q_2$，$\sigma' \models Q_2$。故 $\sigma \models wp(C, Q_2)$。

2. $\sigma \models wp(C, Q_1 \land Q_2)$ 当且仅当所有终止状态满足 $Q_1 \land Q_2$，当且仅当所有终止状态同时满足 $Q_1$ 和 $Q_2$。

3. 若 $\sigma \models wp(C, Q_1)$，则所有终止状态满足 $Q_1$，从而满足 $Q_1 \lor Q_2$。同理对 $Q_2$。$\square$

## 四、示例性代码

### 4.1 Lean 4：霍尔逻辑规则的形式化陈述

```lean
namespace HoareLogicFormal

/-- 状态：变量到整数的映射 -/
def State := String → Int

def State.update (σ : State) (x : String) (v : Int) : State :=
  fun y => if x = y then v else σ y

/-- 断言：状态的谓词 -/
def Assertion := State → Prop

/-- IMP 命令 -/
inductive Com
  | skip
  | assign (x : String) (a : State → Int)
  | seq    (c1 c2 : Com)
  | ifThenElse (b : State → Bool) (c1 c2 : Com)
  | whileDo  (b : State → Bool) (c : Com)

/-- 霍尔三元组的部分正确性语义 -/
def SemValid (P : Assertion) (c : Com) (Q : Assertion) : Prop :=
  ∀ σ σ', (c.bigstep σ σ') → P σ → Q σ'

/-- 大步语义（简化，非完整定义） -/
inductive Com.bigstep : Com → State → State → Prop
  | skip (σ) : bigstep Com.skip σ σ
  | assign (σ x a) : bigstep (Com.assign x a) σ (σ.update x (a σ))
  | seq (σ σ' σ'' c1 c2)
      (h1 : bigstep c1 σ σ') (h2 : bigstep c2 σ' σ'') :
      bigstep (Com.seq c1 c2) σ σ''

/-- 可靠性：公理系统中可推导则语义有效 -/
axiom soundness :
  ∀ P c Q, (Derivable P c Q) → SemValid P c Q

/-- 可推导关系（公理化定义，此处用公理简化） -/
axiom Derivable (P : Assertion) (c : Com) (Q : Assertion) : Prop

end HoareLogicFormal
```

### 4.2 Rust：霍尔逻辑验证器（阶乘）

```rust
/// 阶乘程序的霍尔逻辑验证器
pub struct FactorialVerifier;

impl FactorialVerifier {
    /// 验证不变式：f = i! ∧ i ≤ n
    pub fn check_invariant(n: u64, i: u64, f: u64) -> bool {
        if i > n {
            return false;
        }
        let expected = Self::fact(i);
        f == expected
    }

    /// 计算阶乘（参考实现）
    fn fact(k: u64) -> u64 {
        (1..=k).product()
    }

    /// 验证变式函数 V = n - i 严格递减
    pub fn check_variant_decreases(n: u64, i_before: u64, i_after: u64) -> bool {
        let v_before = n - i_before;
        let v_after = n - i_after;
        i_after == i_before + 1 && v_after < v_before
    }

    /// 验证循环保持性
    pub fn verify_loop_preservation(n: u64, i: u64, f: u64) -> bool {
        if !Self::check_invariant(n, i, f) {
            return false;
        }
        if i >= n {
            return true; // 循环终止
        }
        // 模拟循环体：i := i + 1; f := f * i
        let new_i = i + 1;
        let new_f = f * new_i;
        Self::check_invariant(n, new_i, new_f)
    }

    /// 验证完整阶乘程序
    pub fn verify_factorial(n: u64, result: u64) -> bool {
        // 初始化后：i = 0, f = 1
        let mut i = 0u64;
        let mut f = 1u64;
        if !Self::check_invariant(n, i, f) {
            return false;
        }
        // 模拟循环
        while i < n {
            if !Self::verify_loop_preservation(n, i, f) {
                return false;
            }
            i += 1;
            f *= i;
        }
        // 终止时：i = n，故 f = n!
        i == n && f == result && result == Self::fact(n)
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_factorial_5() {
        assert!(FactorialVerifier::verify_factorial(5, 120));
    }

    #[test]
    fn test_factorial_0() {
        assert!(FactorialVerifier::verify_factorial(0, 1));
    }
}
```

## 五、形式化证明

### 5.1 Cook 相对完备性的证明概要

**定理 5.1**（Cook 相对完备性 [Cook1978]）

设 $T$ 为关于程序状态的充分表达理论。则：

$$
\models_{\mathrm{par}} \{P\} C \{Q\} \Rightarrow T \vdash \{P\} C \{Q\}
$$

**证明**：核心在于证明 $wp(C, Q)$ 可在 $T$ 中表达。

**步骤 1**（基本命令）：

- $wp(\mathbf{skip}, Q) = Q$；
- $wp(X := a, Q) = Q[a/X]$。

**步骤 2**（顺序组合）：

$$
wp(C_1; C_2, Q) = wp(C_1, wp(C_2, Q))
$$

若 $wp(C_2, Q)$ 可表达，且 $wp(C_1, -)$ 保持可表达性，则结论成立。

**步骤 3**（条件）：

$$
wp(\mathbf{if} \, b \, \mathbf{then} \, C_1 \, \mathbf{else} \, C_2, Q) = (b \Rightarrow wp(C_1, Q)) \land (\neg b \Rightarrow wp(C_2, Q))
$$

**步骤 4**（while 循环）：

$$
wp(\mathbf{while} \, b \, \mathbf{do} \, C, Q) = \exists k. \, I_k
$$

其中 $I_0 = \neg b \land Q$，$I_{k+1} = b \land wp(C, I_k)$。在充分表达理论中，该不动点可表达。

**步骤 5**（推导构造）：已知 $\models_{\mathrm{par}} \{P\} C \{Q\}$，则 $P \Rightarrow wp(C, Q)$ 在 $T$ 中成立。由公理和规则可构造 $\{wp(C, Q)\} C \{Q\}$ 的证明，再用推论规则得到 $\{P\} C \{Q\}$。$\square$

### 5.2 数组赋值的扩展规则

**定理 5.2**（数组赋值公理）

对数组赋值 $A[e_1] := e_2$，令 $A\{e_1 \leftarrow e_2\}$ 表示数组更新：

$$
(A\{e_1 \leftarrow e_2\})[i] = \begin{cases} e_2 & \text{if } i = e_1 \\ A[i] & \text{if } i \neq e_1 \end{cases}
$$

则：

$$
\overline{\{Q[A\{e_1 \leftarrow e_2\}/A]\} \, A[e_1] := e_2 \, \{Q\}}
$$

**证明**：执行 $A[e_1] := e_2$ 后，数组 $A$ 的新状态正是 $A\{e_1 \leftarrow e_2\}$。由替换引理，前置条件 $Q[A\{e_1 \leftarrow e_2\}/A]$ 保证了后置条件 $Q$ 在更新后的状态下成立。$\square$

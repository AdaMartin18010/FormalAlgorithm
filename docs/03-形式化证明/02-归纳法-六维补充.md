## 一、规范定义深化

### 1.1 数学归纳原理的 Peano 公理化

**定义 1.1**（Peano 算术的归纳公理）

设 $\mathcal{L}_{\mathrm{PA}} = \{0, S, +, \cdot, <\}$ 为皮亚诺算术的语言。归纳公理模式表述为：

$$
\forall \varphi(x) \in \mathcal{L}_{\mathrm{PA}}. \; \big( \varphi(0) \land \forall k. \, (\varphi(k) \rightarrow \varphi(S(k))) \big) \rightarrow \forall n. \, \varphi(n)
$$

其中 $\varphi(x)$ 是含自由变元 $x$ 的任意一阶公式。该公理模式是二阶 Peano 公理在一阶逻辑中的近似，保证了对所有可定义性质的归纳可证性。

**定义 1.2**（数学归纳法的推导规则形式）

在自然演绎系统中，数学归纳可表述为规则：

$$
\frac{\Gamma \vdash P(0) \quad \Gamma, P(k) \vdash P(k+1)}{\Gamma \vdash \forall n. \, P(n)} \quad (\mathrm{MI})
$$

其中 $k$ 是新鲜变元（eigenvariable），不在 $\Gamma$ 或 $P(0)$ 中自由出现。

### 1.2 良基关系的严格定义

**定义 1.3**（良基关系 / Well-Founded Relation）

设 $R \subseteq A \times A$ 为集合 $A$ 上的二元关系。$R$ 是**良基的**（well-founded），当且仅当 $A$ 的每个非空子集都有 $R$-极小元：

$$
\forall S \subseteq A. \; S \neq \varnothing \rightarrow \exists m \in S. \; \forall x \in S. \; \neg(x \, R \, m)
$$

等价地，$R$ 良基当且仅当不存在无穷下降链：

$$
\neg \exists (a_n)_{n \in \mathbb{N}}. \; \forall n. \; a_{n+1} \, R \, a_n
$$

**定义 1.4**（良基归纳原理）

设 $R$ 是集合 $A$ 上的良基关系，$P(x)$ 是关于 $A$ 中元素的性质。良基归纳原理断言：

$$
\big( \forall x \in A. \; (\forall y. \, y \, R \, x \rightarrow P(y)) \rightarrow P(x) \big) \rightarrow \forall x \in A. \, P(x)
$$

**定理 1.1**（数学归纳是良基归纳的特例）

取 $A = \mathbb{N}$，$R = \{(n, n+1) \mid n \in \mathbb{N}\}$（后继关系），则 $R$ 是良基的，且良基归纳退化为数学归纳。

**证明**：$R$ 的良基性等价于自然数的良序性（well-ordering）。对性质 $P$，良基归纳的前提为：

$$
\forall x. \, (\forall y. \, y = x - 1 \rightarrow P(y)) \rightarrow P(x)
$$

当 $x = 0$ 时，前提 $y \, R \, 0$ 恒假，故 $P(0)$ 必须成立（对应归纳基例）。对 $x = k+1$，前提要求 $P(k)$ 成立（对应归纳假设），从而推出 $P(k+1)$。$\square$

### 1.3 结构归纳的代数定义

**定义 1.5**（代数数据类型与结构归纳）

设 $\mathcal{T}$ 是由构造子签名 $S = \{c_1, \ldots, c_n\}$ 生成的代数数据类型，其中构造子 $c_i$ 的元数为 $k_i$（$k_i = 0$ 对应基础构造子）。$\mathcal{T}$ 的**结构归纳原理**表述为：

$$
\big( \bigwedge_{i=1}^{n} \forall t_1, \ldots, t_{k_i}. \; (\bigwedge_{j=1}^{k_i} P(t_j)) \rightarrow P(c_i(t_1, \ldots, t_{k_i})) \big) \rightarrow \forall t \in \mathcal{T}. \, P(t)
$$

**示例**：对二叉树类型 $\mathrm{Tree}(A)$，构造子为 $\mathrm{Leaf}$（元数 0）和 $\mathrm{Node}$（元数 2），结构归纳原理为：

$$
P(\mathrm{Leaf}) \land \forall l, r. \, (P(l) \land P(r) \rightarrow P(\mathrm{Node}(a, l, r))) \rightarrow \forall t. \, P(t)
$$

## 二、模型设计深化

### 2.1 自然数的集合论模型

**定义 2.1**（von Neumann 自然数构造）

在 ZFC 集合论中，自然数按如下方式构造：

$$
\begin{aligned}
0 &:= \varnothing \\
n + 1 &:= n \cup \{n\} = \{0, 1, \ldots, n\}
\end{aligned}
$$

此构造满足：
- $0 = \varnothing$
- $1 = \{\varnothing\}$
- $2 = \{\varnothing, \{\varnothing\}\}$
- $\ldots$

集合 $\omega = \{0, 1, 2, \ldots\}$ 是所有有限 von Neumann 序数的集合，构成 Peano 算术的标准模型。

**定理 2.1**（$\omega$ 的归纳性）

$\omega$ 是满足 $0 \in S \land \forall n \in S. \, n+1 \in S$ 的最小集合。即，若 $X \subseteq \omega$ 且 $0 \in X$ 且 $\forall n \in X. \, n+1 \in X$，则 $X = \omega$。

### 2.2 递归函数的计算模型

**定义 2.2**（原始递归函数）

原始递归函数类 $\mathcal{PR}$ 是包含零函数 $Z(x) = 0$、后继函数 $S(x) = x+1$、投影函数 $P_i^n(x_1, \ldots, x_n) = x_i$，且在**复合**与**原始递归**下封闭的最小函数类：

$$
\begin{aligned}
f(0, \vec{x}) &= g(\vec{x}) \\
f(n+1, \vec{x}) &= h(n, \vec{x}, f(n, \vec{x}))
\end{aligned}
$$

**定理 2.2**（原始递归与归纳定义的等价性）

所有原始递归函数的定义直接对应于自然数上的数学归纳证明。

### 2.3 类型论中的归纳类型模型

**定义 2.3**（W-类型 / Well-Founded Tree Types）

在 Martin-Löf 类型论中，良基树类型 $W_{x:A} B(x)$ 是归纳类型的通用形式，其中：

- $A$ 是节点标签类型；
- $B(x)$ 是标签为 $x$ 的节点的子节点索引类型。

$W$-类型的引入规则为：

$$
\frac{a : A \quad f : B(a) \to W_{x:A} B(x)}{\sup(a, f) : W_{x:A} B(x)}
$$

其消去规则（归纳原理）要求：对所有节点 $a$ 和子树族 $f$，若对 $B(a)$ 中每个 $b$ 都有 $P(f(b))$，则 $P(\sup(a, f))$。这直接编码了良基归纳。

### 3.2 求和公式的严格归纳证明

**定理 3.1**（算术级数求和）

$$
\forall n \in \mathbb{N}. \; \sum_{i=0}^{n} i = \frac{n(n+1)}{2}
$$

**证明**：设 $P(n) := \sum_{i=0}^{n} i = \frac{n(n+1)}{2}$。

**基例** $n = 0$：

$$
\sum_{i=0}^{0} i = 0 = \frac{0 \cdot 1}{2}
$$

故 $P(0)$ 成立。

**归纳步**：假设 $P(k)$ 成立，即 $\sum_{i=0}^{k} i = \frac{k(k+1)}{2}$。则：

$$
\begin{aligned}
\sum_{i=0}^{k+1} i &= \sum_{i=0}^{k} i + (k+1) \\
&= \frac{k(k+1)}{2} + (k+1) \quad \text{[归纳假设]} \\
&= \frac{k(k+1) + 2(k+1)}{2} \\
&= \frac{(k+1)(k+2)}{2}
\end{aligned}
$$

故 $P(k+1)$ 成立。由数学归纳法，$\forall n. \, P(n)$。$\square$

### 3.3 斐波那契数列上界的强归纳证明

**定理 3.2**（斐波那契上界）

设 $F_0 = 0, F_1 = 1, F_{n+2} = F_{n+1} + F_n$，则：

$$
\forall n \in \mathbb{N}. \; F_n < 2^n
$$

**证明**（强归纳法）：设 $P(n) := F_n < 2^n$。

**基例**：
- $n = 0$：$F_0 = 0 < 1 = 2^0$；
- $n = 1$：$F_1 = 1 < 2 = 2^1$。

**归纳步**：假设对所有 $i \leq k+1$，$P(i)$ 成立。则：

$$
\begin{aligned}
F_{k+2} &= F_{k+1} + F_k \\
&< 2^{k+1} + 2^k \quad \text{[强归纳假设]} \\
&= 2 \cdot 2^k + 2^k = 3 \cdot 2^k \\
&< 4 \cdot 2^k = 2^{k+2}
\end{aligned}
$$

故 $P(k+2)$ 成立。由强归纳法，$\forall n. \, P(n)$。$\square$

### 3.4 欧几里得算法终止性的良基归纳证明

**定理 3.3**（欧几里得算法终止）

对任意 $a, b \in \mathbb{N}$ 且 $b > 0$，$\gcd(a, b)$ 的计算在有限步内终止。

**证明**：定义关系 $R \subseteq \mathbb{N}^2 \times \mathbb{N}^2$ 为：

$$
(a', b') \, R \, (a, b) \quad \Leftrightarrow \quad a' + b' < a + b
$$

$R$ 是良基的，因为 $(\mathbb{N}, <)$ 是良序集。

欧几里得算法的递归步骤为 $\gcd(a, b) = \gcd(b, a \bmod b)$。由于 $0 \leq a \bmod b < b$，有：

$$
b + (a \bmod b) < a + b
$$

故 $(b, a \bmod b) \, R \, (a, b)$。由良基归纳，算法必然终止。$\square$

## 四、示例性代码

### 4.1 Lean 4：数学归纳法的形式化实现

```lean
namespace MathematicalInduction

/-- Peano 自然数 -/
inductive PeanoNat
  | zero : PeanoNat
  | succ : PeanoNat → PeanoNat
  deriving Repr

open PeanoNat

/-- 加法 -/
def add : PeanoNat → PeanoNat → PeanoNat
  | zero,   m => m
  | succ n, m => succ (add n m)

local infixl:65 " ⊕ " => add

/-- 右单位元：n ⊕ zero = n -/
theorem add_zero_right (n : PeanoNat) : n ⊕ zero = n := by
  induction n with
  | zero => rfl
  | succ n ih =>
    simp [add]
    rw [ih]

/-- 后继分配律：succ n ⊕ m = succ (n ⊕ m) -/
theorem add_succ_left (n m : PeanoNat) : (succ n) ⊕ m = succ (n ⊕ m) := by
  rfl  -- 由定义直接得到

/-- 加法交换律 -/
theorem add_comm (n m : PeanoNat) : n ⊕ m = m ⊕ n := by
  induction n with
  | zero =>
    rw [add_zero_right]
    -- 还需证明 zero ⊕ m = m，即 add 的定义
    rfl
  | succ n ih =>
    calc
      (succ n) ⊕ m = succ (n ⊕ m) := by rfl
      _ = succ (m ⊕ n)           := by rw [ih]
      _ = m ⊕ (succ n)           := by
        induction m with
        | zero => rfl
        | succ m ih2 =>
          simp [add]
          rw [ih2]

/-- 加法结合律 -/
theorem add_assoc (a b c : PeanoNat) : (a ⊕ b) ⊕ c = a ⊕ (b ⊕ c) := by
  induction a with
  | zero => rfl
  | succ a ih =>
    simp [add]
    rw [ih]

end MathematicalInduction
```

### 4.2 Lean 4：结构归纳法（二叉树）

```lean
namespace StructuralInduction

/-- 二叉树 -/
inductive Tree (A : Type)
  | leaf : Tree A
  | node : A → Tree A → Tree A → Tree A
  deriving Repr

open Tree

/-- 节点计数 -/
def countNodes {A : Type} : Tree A → Nat
  | leaf => 0
  | node _ l r => 1 + countNodes l + countNodes r

/-- 叶子计数 -/
def countLeaves {A : Type} : Tree A → Nat
  | leaf => 1
  | node _ l r => countLeaves l + countLeaves r

/-- 关键定理：叶子数 = 内部节点数 + 1 -/
theorem leaves_eq_internal_plus_one {A : Type} (t : Tree A) :
    countLeaves t = countNodes t + 1 := by
  induction t with
  | leaf =>
    -- 基例：叶子树有 1 个叶子，0 个内部节点
    rfl
  | node a l r ihl ihr =>
    -- 归纳步骤
    calc
      countLeaves (node a l r)
        = countLeaves l + countLeaves r := by rfl
      _ = (countNodes l + 1) + (countNodes r + 1)
          := by rw [ihl, ihr]
      _ = (1 + countNodes l + countNodes r) + 1
          := by simp [Nat.add_assoc, Nat.add_comm]
      _ = countNodes (node a l r) + 1
          := by rfl

end StructuralInduction
```

### 4.3 Rust：良基归纳验证器

```rust
/// 良基关系验证器
pub trait WellFoundedRelation<T> {
    /// 判断 a R b（a 在 b 之前）
    fn relation(&self, a: &T, b: &T) -> bool;

    /// 验证关系中无无穷下降链（对有限集可用）
    fn check_no_descending_chain(&self, elements: &[T]) -> bool {
        // Floyd 判圈思想：对每个起点检测下降链长度
        for start in elements {
            let mut current = start;
            let mut steps = 0;
            let max_steps = elements.len() + 1;

            loop {
                let next = elements.iter().find(|e| self.relation(e, current));
                match next {
                    Some(n) => {
                        current = n;
                        steps += 1;
                        if steps > max_steps {
                            return false; // 可能存在无穷链
                        }
                    }
                    None => break,
                }
            }
        }
        true
    }
}

/// 欧几里得算法的良基关系验证
pub struct EuclideanRelation;

impl WellFoundedRelation<(u64, u64)> for EuclideanRelation {
    fn relation(&self, a: &(u64, u64), b: &(u64, u64)) -> bool {
        let (a1, a2) = a;
        let (b1, b2) = b;
        a1 + a2 < b1 + b2
    }
}

/// 验证欧几里得算法的终止性
pub fn verify_euclidean_termination(a: u64, b: u64) -> bool {
    let rel = EuclideanRelation;
    if b == 0 {
        return true; // 直接终止
    }
    let next = (b, a % b);
    let current = (a, b);
    rel.relation(&next, &current)
}
```

## 五、形式化证明

### 5.1 归纳原理与良基性的等价性

**定理 5.1**（归纳原理等价于良基性）

设 $R$ 是集合 $A$ 上的二元关系。则以下命题等价：

1. $R$ 是良基的；
2. 良基归纳原理对 $R$ 成立：
   $$
   \forall P. \, (\forall x. \, (\forall y. \, y \, R \, x \rightarrow P(y)) \rightarrow P(x)) \rightarrow \forall x. \, P(x)
   $$

**证明** $(1 \Rightarrow 2)$：设 $R$ 良基。假设归纳前提成立：

$$
\forall x. \, (\forall y. \, y \, R \, x \rightarrow P(y)) \rightarrow P(x)
$$

反设 $\exists x. \, \neg P(x)$。令 $S = \{x \in A \mid \neg P(x)\}$，则 $S \neq \varnothing$。由 $R$ 的良基性，$S$ 有 $R$-极小元 $m$。

对任意 $y \, R \, m$，由极小性，$y \notin S$，即 $P(y)$ 成立。由归纳前提应用于 $m$，得 $P(m)$，与 $m \in S$ 矛盾。故 $\forall x. \, P(x)$。

$(2 \Rightarrow 1)$：假设良基归纳成立。反设 $R$ 不良基，则存在非空子集 $S \subseteq A$ 无 $R$-极小元。定义性质：

$$
P(x) := x \notin S
$$

对任意 $x$，若 $\forall y. \, y \, R \, x \rightarrow P(y)$，则所有 $R$-前驱都不在 $S$ 中。若 $x \in S$，则 $x$ 将是 $S$ 的 $R$-极小元（因无 $R$-前驱在 $S$ 中），矛盾。故 $x \notin S$，即 $P(x)$。

由良基归纳，$\forall x. \, P(x)$，即 $S = \varnothing$，矛盾。$\square$

### 5.2 结构归纳的合法性定理

**定理 5.2**（结构归纳的合法性）

设 $\mathcal{T}$ 是由有限构造子集生成的代数数据类型。则结构归纳原理是有效的证明方法。

**证明**：构造子签名 $S$ 在 $\mathcal{T}$ 上诱导出**直接子项**关系 $\prec$：$t \prec c(t_1, \ldots, t_k)$ 当 $t = t_i$ 对某个 $i$。由于每个构造子只有有限个子项，且构造深度有限，$\prec$ 是良基的。

结构归纳的前提正是良基归纳在 $(\mathcal{T}, \prec)$ 上的应用。由定理 5.1，结构归纳有效。$\square$

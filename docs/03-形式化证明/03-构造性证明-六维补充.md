## 一、规范定义深化

### 1.1 构造性证明的 BHK 解释

**定义 1.1**（Brouwer-Heyting-Kolmogorov 解释）

BHK 解释为直觉主义逻辑中的每个连接词和量词提供了**证明是什么**的构造性语义。设 $p, q$ 分别为命题 $A, B$ 的构造性证明：

| 公式 | 其构造性证明是…… |
|------|-------------------|
| $A \land B$ | 有序对 $(p, q)$，其中 $p$ 证 $A$，$q$ 证 $B$ |
| $A \lor B$ | 标记证明 $\mathrm{inl}(p)$ 或 $\mathrm{inr}(q)$，附分支信息 |
| $A \rightarrow B$ | 构造性函数 $f$，将 $A$ 的任意证明映射为 $B$ 的证明 |
| $\exists x \in X.\, P(x)$ | 对 $(a, p)$，其中 $a \in X$ 是见证，$p$ 证 $P(a)$ |
| $\forall x \in X.\, P(x)$ | 构造性函数 $g$，将每个 $a \in X$ 映射为 $P(a)$ 的证明 |
| $\neg A$ | 构造性函数 $f : A \rightarrow \bot$，将 $A$ 的证明映射为矛盾 |

**定义 1.2**（构造性存在）

对存在性命题 $\exists x.\, P(x)$ 的**构造性证明**是一个对 $\langle a, p_a \rangle$，其中：

- $a$ 是具体构造的**见证**（witness）；
- $p_a$ 是 $P(a)$ 的构造性证明。

这与经典证明形成鲜明对比：经典逻辑允许从 $\neg \forall x.\, \neg P(x)$ 推出 $\exists x.\, P(x)$，而无需提供任何见证。

### 1.2 直觉主义命题逻辑的形式系统 NJ

**定义 1.3**（直觉主义自然演绎 NJ）

NJ 系统与经典自然演绎 NK 的区别在于：**不允许直接使用排中律** $A \lor \neg A$ 作为公理，且**否定消去**（从 $\neg \neg A$ 推出 $A$）不成立。

NJ 的核心规则：

$$
\begin{aligned}
&\frac{\Gamma \vdash A \quad \Gamma \vdash B}{\Gamma \vdash A \land B}(\land I) \\[4pt]
&\frac{\Gamma \vdash A \land B}{\Gamma \vdash A}(\land E_1) \quad \frac{\Gamma \vdash A \land B}{\Gamma \vdash B}(\land E_2) \\[4pt]
&\frac{\Gamma, A \vdash B}{\Gamma \vdash A \rightarrow B}(\rightarrow I) \quad \frac{\Gamma \vdash A \rightarrow B \quad \Gamma \vdash A}{\Gamma \vdash B}(\rightarrow E) \\[4pt]
&\frac{\Gamma \vdash A}{\Gamma \vdash A \lor B}(\lor I_1) \quad \frac{\Gamma \vdash B}{\Gamma \vdash A \lor B}(\lor I_2) \\[4pt]
&\frac{\Gamma \vdash A \lor B \quad \Gamma, A \vdash C \quad \Gamma, B \vdash C}{\Gamma \vdash C}(\lor E) \\[4pt]
&\frac{\Gamma, A \vdash \bot}{\Gamma \vdash \neg A}(\neg I) \quad \frac{\Gamma \vdash \neg A \quad \Gamma \vdash A}{\Gamma \vdash \bot}(\neg E)
\end{aligned}
$$

注意 NJ 中没有规则 $\frac{\Gamma \vdash \neg \neg A}{\Gamma \vdash A}$。

### 1.3 可实现性理论

**定义 1.4**（Kleene 可实现性 / Kleene Realizability）

设 $e \in \mathbb{N}$ 为部分递归函数的哥德尔编码，$A$ 为算术命题。关系 $e \realizes A$（$e$ 实现 $A$）归纳定义：

$$
\begin{aligned}
e &\realizes t_1 = t_2 &&\Leftrightarrow t_1 = t_2 \text{ 为真} \\
e &\realizes A \land B &&\Leftrightarrow (e)_0 \realizes A \text{ 且 } (e)_1 \realizes B \\
e &\realizes A \lor B &&\Leftrightarrow ((e)_0 = 0 \text{ 且 } (e)_1 \realizes A) \text{ 或 } ((e)_0 = 1 \text{ 且 } (e)_1 \realizes B) \\
e &\realizes A \rightarrow B &&\Leftrightarrow \forall n. \, (n \realizes A \Rightarrow \{e\}(n) \text{ 有定义且 } \{e\}(n) \realizes B) \\
e &\realizes \exists x.\, A(x) &&\Leftrightarrow (e)_1 \realizes A((e)_0) \\
e &\realizes \forall x.\, A(x) &&\Leftrightarrow \forall n. \, (\{e\}(n) \text{ 有定义且 } \{e\}(n) \realizes A(n))
\end{aligned}
$$

其中 $(e)_i$ 为配对投影函数，$\{e\}(n)$ 为第 $e$ 个部分递归函数在输入 $n$ 下的计算。

## 二、模型设计深化

### 2.1 Heyting 代数语义

**定义 2.1**（Heyting 代数）

Heyting 代数是有界格 $(H, \land, \lor, \top, \bot)$ 附加二元运算 $\rightarrow$（蕴涵），满足：

$$
a \land b \leq c \quad \Leftrightarrow \quad a \leq (b \rightarrow c)
$$

**定理 2.1**（Heyting 代数与直觉主义逻辑）

直觉主义命题逻辑的 Lindenbaum 代数构成 Heyting 代数。经典逻辑的 Lindenbaum 代数构成布尔代数（Heyting 代数的特例，其中 $\neg \neg a = a$）。

在 Heyting 代数中，否定定义为 $\neg a = a \rightarrow \bot$。关键性质：

$$
a \leq \neg \neg a \quad \text{（成立）} \quad \text{但} \quad \neg \neg a \leq a \quad \text{（一般不成立）}
$$

### 2.2 Kripke 语义模型

**定义 2.2**（直觉主义 Kripke 模型）

直觉主义 Kripke 模型是三元组 $\mathcal{M} = (W, \leq, V)$，其中：

- $(W, \leq)$ 是偏序集（世界集合与可达关系）；
- $V: \mathrm{Prop} \to \mathcal{P}(W)$ 满足**遗传性**：$w \in V(p) \land w \leq w' \Rightarrow w' \in V(p)$。

强制关系 $\Vdash$ 定义为：

$$
\begin{aligned}
\mathcal{M}, w &\Vdash p &&\Leftrightarrow w \in V(p) \\
\mathcal{M}, w &\Vdash A \land B &&\Leftrightarrow \mathcal{M}, w \Vdash A \text{ 且 } \mathcal{M}, w \Vdash B \\
\mathcal{M}, w &\Vdash A \lor B &&\Leftrightarrow \mathcal{M}, w \Vdash A \text{ 或 } \mathcal{M}, w \Vdash B \\
\mathcal{M}, w &\Vdash A \rightarrow B &&\Leftrightarrow \forall w' \geq w. \, (\mathcal{M}, w' \Vdash A \Rightarrow \mathcal{M}, w' \Vdash B) \\
\mathcal{M}, w &\Vdash \bot &&\Leftrightarrow \text{永假}
\end{aligned}
$$

**定理 2.2**（排中律在 Kripke 模型中不有效）

存在 Kripke 模型 $\mathcal{M}$ 和世界 $w$，使得 $\mathcal{M}, w \not\Vdash A \lor \neg A$。

**证明**：构造模型 $W = \{w_0, w_1, w_2\}$，其中 $w_0 < w_1$，$w_0 < w_2$，且 $w_1, w_2$ 不可比较。设 $V(p) = \{w_1\}$。则：

- $w_0 \not\Vdash p$（因 $w_2 \geq w_0$ 且 $w_2 \not\Vdash p$）；
- $w_0 \not\Vdash \neg p$（因 $w_1 \geq w_0$ 且 $w_1 \Vdash p$）。

故 $w_0 \not\Vdash p \lor \neg p$。$\square$

### 2.3 拓扑语义

**定义 2.3**（拓扑语义 / Topological Semantics）

设 $(X, \mathcal{O})$ 为拓扑空间。将命题解释为开集 $[\![A]\!] \in \mathcal{O}$：

$$
\begin{aligned}
[\![A \land B]\!] &= [\![A]\!] \cap [\![B]\!] \\
[\![A \lor B]\!] &= [\![A]\!] \cup [\![B]\!] \\
[\![A \rightarrow B]\!] &= \mathrm{int}((X \setminus [\![A]\!]) \cup [\![B]\!]) \\
[\![\bot]\!] &= \varnothing
\end{aligned}
$$

其中 $\mathrm{int}(S)$ 是集合 $S$ 的内部。否定解释为 $[\![\neg A]\!] = \mathrm{int}(X \setminus [\![A]\!])$。

**定理 2.3**（拓扑完备性）

直觉主义命题逻辑在拓扑语义下是可靠且完备的：$\vdash_{\mathrm{NJ}} A \Leftrightarrow$ 对所有拓扑空间 $(X, \mathcal{O})$，$[\![A]\!] = X$。

### 3.2 排中律不可证的严格推导

**定理 3.1**（排中律在 NJ 中不可证）

不存在 NJ 中的证明使得 $\vdash_{\mathrm{NJ}} A \lor \neg A$ 对所有命题 $A$ 成立。

**证明**：假设存在这样的证明。由 BHK 解释，$A \lor \neg A$ 的证明必须是 $\mathrm{inl}(p)$ 或 $\mathrm{inr}(q)$，其中 $p$ 是 $A$ 的证明或 $q$ 是 $\neg A$ 的证明。但对任意命题 $A$，我们无法保证总能构造其中之一。Kripke 反模型（定理 2.2）提供了具体的语义反例。$\square$

**定理 3.2**（双重否定不可消去）

$\vdash_{\mathrm{NJ}} \neg \neg A \rightarrow A$ 不普遍成立。

**证明**：由 Kripke 模型，取 $W = \{w_0 < w_1\}$，$V(p) = \{w_1\}$。则 $w_0 \Vdash \neg \neg p$（因不存在 $w \geq w_0$ 使得 $w \Vdash \neg p$），但 $w_0 \not\Vdash p$。故 $\neg \neg p \rightarrow p$ 不有效。$\square$

### 3.3 可实现性 Soundness 推导

**定理 3.3**（可实现性 Soundness）

若 $A$ 在 Heyting 算术 HA 中可证，则存在自然数 $e$ 使得 $e \realizes A$。

**证明概要**：对 HA 中 $A$ 的证明进行结构归纳，为每个证明规则构造对应的可实现数：

- 等式公理：任意数实现；
- 合取引入：配对函数实现；
- 蕴涵引入：哥德尔编码的递归函数实现；
- 归纳规则：原始递归函数实现。

由于 HA 的所有证明规则保持可实现性，结论成立。$\square$

## 四、示例性代码

### 4.1 Lean 4：构造性存在证明

```lean
namespace ConstructiveProof

/-- 构造性存在：直接给出见证 -/
example : ∃ n : Nat, n > 10 := by
  -- 直接构造见证 11
  exact ⟨11, by norm_num⟩

/-- 构造性析取：选择左分支并给出证明 -/
example : 2 + 2 = 4 ∨ 2 + 2 = 5 := by
  -- 选择左分支
  apply Or.inl
  rfl

/-- 构造性蕴涵：给出从前提到结论的函数 -/
def impl_example : (n : Nat) → n = 0 → n + n = 0 :=
  fun n hn => by rw [hn]; rfl

/-- 不可判定命题的决策：在直觉主义中需谨慎 -/
inductive Dec (P : Prop)
  | isFalse : ¬P → Dec P
  | isTrue  :  P → Dec P

/-- 可判定相等的自然数 -/
def decEqNat (m n : Nat) : Dec (m = n) :=
  match m, n with
  | 0,   0   => Dec.isTrue rfl
  | 0,   _+1 => Dec.isFalse (by intro h; cases h)
  | _+1, 0   => Dec.isFalse (by intro h; cases h)
  | m+1, n+1 =>
    match decEqNat m n with
    | Dec.isTrue h  => Dec.isTrue (by rw [h])
    | Dec.isFalse h => Dec.isFalse (by intro h2; apply h; injection h2)

end ConstructiveProof
```

### 4.2 Lean 4：Curry-Howard 对应示例

```lean
namespace CurryHoward

/-- 命题即类型：A ∧ B 对应 A × B -/
def and_intro {A B : Prop} (a : A) (b : B) : A ∧ B :=
  And.intro a b

/-- 证明即程序：从 A ∧ B 提取 A 的证明 -/
def and_elim_left {A B : Prop} (p : A ∧ B) : A :=
  p.left

/-- 蕴涵即函数类型：A → B 对应 A → B -/
def modus_ponens {A B : Prop} (f : A → B) (a : A) : B :=
  f a

/-- 全称量词即依赖函数：∀ x, P x 对应 (x : T) → P x -/
def universal {T : Type} {P : T → Prop} (f : (x : T) → P x) (t : T) : P t :=
  f t

/-- 存在量词即依赖对：∃ x, P x 对应 Σ x, P x -/
def existential {T : Type} {P : T → Prop} (t : T) (p : P t) : ∃ x, P x :=
  ⟨t, p⟩

end CurryHoward
```

### 4.3 Rust：构造性证明验证器

```rust
/// 构造性存在证明：必须提供见证
pub struct ConstructiveExists<T, P> {
    pub witness: T,
    pub proof: Box<dyn Fn(&T) -> bool>,
    pub property: std::marker::PhantomData<P>,
}

impl<T, P> ConstructiveExists<T, P> {
    pub fn new(witness: T, proof: Box<dyn Fn(&T) -> bool>) -> Option<Self> {
        if proof(&witness) {
            Some(Self { witness, proof, property: std::marker::PhantomData })
        } else {
            None
        }
    }

    pub fn verify(&self) -> bool {
        (self.proof)(&self.witness)
    }
}

/// 构造性析取：必须选择分支
pub enum ConstructiveOr<A, B> {
    Left(A),
    Right(B),
}

/// 示例：构造性证明存在大于 10 的素数
pub fn constructive_prime_exists() -> ConstructiveExists<u32, ()> {
    ConstructiveExists::new(
        11,
        Box::new(|&n| {
            if n < 2 { return false; }
            for i in 2..=(n as f64).sqrt() as u32 {
                if n % i == 0 { return false; }
            }
            true
        })
    ).unwrap()
}
```

## 五、形式化证明

### 5.1 构造性实数的存在性证明

**定理 5.1**（构造性平方根存在）

存在构造性实数 $r$ 使得 $r^2 = 2$。

**构造性证明**：定义 $r$ 为如下柯西序列（Babylonian 方法）：

$$
r_0 = 1, \quad r_{n+1} = \frac{1}{2}\left(r_n + \frac{2}{r_n}\right)
$$

**验证**：对每个 $n$，$r_n > 0$ 且 $r_n^2$ 收敛到 2。具体地：

$$
|r_{n+1}^2 - 2| = \frac{(r_n^2 - 2)^2}{4r_n^2} < \frac{|r_n^2 - 2|^2}{4}
$$

当 $|r_n^2 - 2| < 1$ 时，误差呈二次收敛。此构造给出了 $\sqrt{2}$ 的有效逼近算法，而非仅断言其存在。$\square$

### 5.2 构造性与经典存在性证明的对比

**经典证明**（非构造性）：

> 考虑 $\sqrt{2}^{\sqrt{2}}$。若它是有理数，则 $a = b = \sqrt{2}$ 即为所求。若它是无理数，则 $a = \sqrt{2}^{\sqrt{2}}$，$b = \sqrt{2}$，有 $a^b = (\sqrt{2}^{\sqrt{2}})^{\sqrt{2}} = \sqrt{2}^2 = 2$，是有理数。

此证明使用了排中律（$\sqrt{2}^{\sqrt{2}}$ 要么有理要么无理），但未确定究竟是哪种情况。

**构造性证明**：

取 $a = \sqrt{2}$，$b = 2\log_2 3$。则 $a^b = (\sqrt{2})^{2\log_2 3} = 2^{\log_2 3} = 3$，是有理数。直接构造了满足条件的 $a, b$。$\square$

### 5.3 可实现性蕴含一致性

**定理 5.2**（Heyting 算术的一致性）

Heyting 算术 HA 是一致的：不存在公式 $A$ 使得 HA $\vdash A$ 且 HA $\vdash \neg A$。

**证明**：假设 HA $\vdash A$ 且 HA $\vdash \neg A$。由可实现性 soundness，存在 $e_1, e_2$ 使得 $e_1 \realizes A$ 且 $e_2 \realizes \neg A$。由 $\neg A$ 的可实现性定义，$e_2$ 是将 $A$ 的证明映射为矛盾的函数。故 $\{e_2\}(e_1)$ 应计算出矛盾的可实现数，但 $\bot$ 不可被任何数实现（因其永假）。矛盾。$\square$

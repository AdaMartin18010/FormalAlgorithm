# 案例：Z算法在基因组重复序列检测中的应用

## 背景

基因组中存在大量重复序列（如转座子、微卫星 DNA），这些重复序列的检测对于理解基因组结构、进化历史和疾病关联具有重要意义。Z算法可以高效地计算字符串每个后缀与原串的最长公共前缀（LCP），从而快速定位重复模式。

## 问题定义

给定 DNA 序列 $S$（长度 $n$ 可达 $10^9$），找出所有长度大于阈值 $L$ 的重复子串。

## Z算法的应用

Z函数 $Z[i]$ 定义为：$S$ 与 $S[i..n-1]$ 的最长公共前缀长度。

若 $Z[i] \geq L$，则子串 $S[0..L-1]$ 在位置 $i$ 处重复出现。

### 算法流程

1. 计算整个序列的 Z 函数（$O(n)$）
2. 遍历 $Z$ 数组，收集所有 $Z[i] \geq L$ 的位置
3. 对重复模式进行聚类和过滤

## 复杂度优势

| 方法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 后缀数组 + LCP | $O(n \log n)$ 或 $O(n)$ | $O(n)$ |
| 后缀自动机 | $O(n)$ | $O(n)$ |
| **Z算法扫描** | **$O(n)$** | **$O(n)$** |

对于检测**特定模式**的重复（以模式为前缀），Z算法是最简洁高效的选择。

## 实际扩展

- **多模式检测**：为每个候选模式单独计算 Z 函数
- **近似匹配**：允许 $k$ 个错配的扩展 Z 算法
- **串联重复检测**：在 Z 数组上寻找周期性峰值

# 案例：离散对数在椭圆曲线密码学（ECC）与 Diffie-Hellman 密钥交换中的应用

## 背景

离散对数问题（Discrete Logarithm Problem, DLP）是现代公钥密码学的基石之一。
其困难性保证了 Diffie-Hellman 密钥交换、ElGamal 加密和椭圆曲线密码（ECC）的安全性。
理解离散对数算法对于评估密码系统参数选择和安全性至关重要。

## 问题定义

给定乘法群 $G$（如 $\mathbb{Z}_p^*$ 或椭圆曲线点群），生成元 $g$，以及元素 $h = g^x$。

**求 $x$**：$x = \log_g h$

## 核心算法

### Baby-step Giant-step（BSGS）

时间-空间权衡算法：$O(\sqrt{p})$ 时间，$O(\sqrt{p})$ 空间。

```
m = ceil(sqrt(p))
baby steps: 存储 g^j for j = 0..m-1
for i = 0..m-1:
    giant = h * (g^(-m))^i
    if giant in baby table:
        return i*m + j
```

适用于中等规模群（$p \leq 10^{20}$）。

### Pollard's Rho

$O(\sqrt{p})$ 时间，$O(1)$ 空间，利用 Floyd 判圈算法。

### Index Calculus

对 $\mathbb{Z}_p^*$ 亚指数算法：$L_p[1/3, c]$，是实际中最快的经典算法。

## 密码学应用

### Diffie-Hellman 密钥交换

- Alice: 选择私钥 $a$，发送 $A = g^a \pmod{p}$
- Bob: 选择私钥 $b$，发送 $B = g^b \pmod{p}$
- 共享密钥：$K = B^a = A^b = g^{ab} \pmod{p}$

安全性：从 $g, g^a, g^b$ 求 $g^{ab}$ 等价于求解 DLP（ Computational Diffie-Hellman 假设）。

### 椭圆曲线密码（ECC）

将 DLP 从乘法群推广到椭圆曲线点群：

- 密钥更短：256-bit ECC ≈ 3072-bit RSA 安全性
- 计算更快：模乘 vs 大整数分解
- 广泛应用于 TLS、比特币、iMessage

## 参数安全要求

| 群类型 | 安全位 | 最小群大小 |
|--------|--------|-----------|
| $\mathbb{Z}_p^*$ | 128 bit | $p \geq 3072$ bit |
| 椭圆曲线 | 128 bit | $q \geq 256$ bit |

## 量子威胁

Shor 算法可在多项式时间内求解 DLP，意味着大规模量子计算机将破解所有基于 DLP 的密码系统。后量子密码（如 Lattice-based）正在标准化中。

# 案例：欧几里得算法在密码学与分数约简中的应用

## 背景

欧几里得算法（辗转相除法）是数学史上最古老的算法之一，用于计算两个整数的最大公约数（GCD）。
它是现代密码学、分数运算和数论算法的基石。

## 问题定义

- **输入**：两个非负整数 $a, b$
- **输出**：$\gcd(a, b)$

## 算法

### 经典欧几里得

```
gcd(a, b):
    while b != 0:
        (a, b) = (b, a mod b)
    return a
```

**复杂度**：$O(\log \min(a, b))$

### 扩展欧几里得

同时求出 $x, y$ 使得：
$$ax + by = \gcd(a, b)$$

用于求模逆元：若 $\gcd(a, m) = 1$，则 $a^{-1} \equiv x \pmod{m}$。

## 实际应用

### RSA 密钥生成

- 选择公钥指数 $e$，需满足 $\gcd(e, \phi(n)) = 1$
- 扩展欧几里得求私钥 $d \equiv e^{-1} \pmod{\phi(n)}$
- 每次 RSA 操作都依赖 GCD 和模逆元

### 分数运算

- 分数约简：$\frac{a}{b} = \frac{a/\gcd(a,b)}{b/\gcd(a,b)}$
- 有理数运算库的核心操作
- 精确计算避免浮点误差

### 密码学协议

- Diffie-Hellman：生成原根需判断 $\gcd$ 条件
- ECC：椭圆曲线点的阶计算
- 中国剩余定理（CRT）：模数需互素

### 数论函数

- Euler 函数：$\phi(n) = n \prod_{p|n} (1 - 1/p)$
- 需要质因数分解，分解依赖试除到 $\sqrt{n}$

### 音频处理

- 采样率转换：求两个采样率的 GCD
- 确定最小公共周期

## 与更快速算法的对比

| 算法 | 复杂度 | 特点 |
|------|--------|------|
| 欧几里得 | $O(\log \min(a,b))$ | 通用，简单 |
| 二进制 GCD | $O(\log^2 N)$ | 无除法，适合硬件 |
| Lehmer | $O(\log^5 N)$ | 大整数更快 |

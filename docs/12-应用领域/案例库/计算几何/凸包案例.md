# 案例：凸包算法在自动驾驶障碍物检测中的应用

## 背景

自动驾驶系统依赖激光雷达（LiDAR）和摄像头采集周围环境的三维点云数据。为了高效识别障碍物的轮廓并进行碰撞检测，需要将离散的点云数据压缩为简洁的多边形表示。**凸包（Convex Hull）** 是这一过程中的关键算法。

## 问题定义

给定激光雷达扫描得到的二维点集 $P$（鸟瞰图投影），求包含所有点的最小凸多边形。

## 算法选择：Monotone Chain

在自动驾驶场景中，Monotone Chain（Andrew 算法）是首选：

- **时间复杂度**：$O(n \log n)$，主要由排序决定
- **空间复杂度**：$O(n)$
- **数值稳定性**：仅使用叉积，避免角度计算
- **输出简洁**：直接得到凸包顶点序列

## 实际流程

1. **点云过滤**：去除地面点、噪声点
2. **聚类**：使用 DBSCAN 将点云分簇（每簇代表一个障碍物）
3. **凸包计算**：对每个簇计算二维凸包
4. **碰撞检测**：将凸包作为障碍物的包围体，与车辆路径进行相交测试

## 效果评估

| 指标 | 效果 |
|------|------|
| 计算速度 | 对 1000 个点的簇，凸包计算 < 1ms |
| 近似精度 | 凸包略大于实际障碍物，偏保守（安全） |
| 内存占用 | 仅需存储顶点序列，压缩率 > 90% |

## 扩展：非凸障碍物

对于 L 形等非凸障碍物，凸包会产生过多冗余空间。此时可采用：

- **凹包（Concave Hull）**：Alpha Shape 算法
- **旋转卡壳**：在凸包基础上求最小面积矩形包围盒

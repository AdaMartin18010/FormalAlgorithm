# 案例：背包问题在资源分配与投资组合优化中的应用

## 背景

背包问题（Knapsack）是组合优化中最经典的问题之一，广泛应用于资源分配、货物装载、投资组合选择和云计算调度等场景。

## 问题定义

### 0/1 背包

- **物品**：$n$ 个，每个有重量 $w_i$ 和价值 $v_i$
- **容量**：背包最大承重 $W$
- **目标**：选择物品子集，总重 $\leq W$，总价值最大
- **约束**：每个物品要么选要么不选（0/1）

### 完全背包

- 每种物品可选取无限件

### 多重背包

- 每种物品有数量限制 $c_i$

## 动态规划解法

### 0/1 背包 DP

```
dp[i][w] = 前 i 个物品、容量 w 时的最大价值
dp[i][w] = max(dp[i-1][w], dp[i-1][w-wi] + vi)  // 不选 / 选
```

**优化**：一维数组，逆序更新：

```
for w = W downto wi:
    dp[w] = max(dp[w], dp[w-wi] + vi)
```

**复杂度**：$O(n \cdot W)$（伪多项式）

## 实际应用

### 云计算资源调度

- 虚拟机请求：CPU、内存、磁盘需求
- 物理服务器：资源容量限制
- 目标：最大化服务器利用率或最小化使用服务器数
- **变种**：多维背包（CPU + 内存 + 网络）

### 投资组合优化

- 资产：股票、债券、基金
- 约束：总资金、风险预算、行业分散度
- 目标：最大化预期收益
- **变种**：带风险约束的分数背包 + 整数约束

### 货物装载

- 物流：卡车/集装箱容量有限
- 货物：重量、体积、价值（运费）
- 目标：最大化单趟运输价值
- **变种**：二维/三维装箱（更复杂的几何约束）

### 广告位分配

- 广告位：时间长度、展示位置
- 广告商：预算、目标受众
- 目标：平台收益最大化
- **变种**：在线背包（广告实时竞价）

## 与贪心算法的对比

| 方法 | 适用场景 | 结果 |
|------|---------|------|
| 贪心（按价值密度）| 分数背包 | 最优 |
| 贪心（按价值密度）| 0/1 背包 | ❌ 非最优 |
| 动态规划 | 0/1 背包 | 最优（伪多项式）|
| 分支限界 | 0/1 背包 | 最优（精确）|
| 近似方案（FPTAS）| 大规模 0/1 背包 | $(1-\epsilon)$ 近似 |

## 效果评估

- DP 可处理 $n=10^4, W=10^6$ 规模
- 对更大规模，使用 FPTAS 或分支限界
- 实际云调度中，启发式 + LP 松弛更常用（毫秒级决策）

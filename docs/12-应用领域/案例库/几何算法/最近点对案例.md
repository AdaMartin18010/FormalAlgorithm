# 案例：最近点对在聚类分析与分子动力学模拟中的应用

## 背景

在平面上给定大量点，找到距离最近的两个点是计算几何的基础问题。分治法以 $O(n \log n)$ 解决，优于暴力的 $O(n^2)$，在大规模数据（如分子模拟、天文数据）中至关重要。

## 问题定义

- **输入**：平面上 $n$ 个点 $P = \{p_1, ..., p_n\}$
- **输出**：$\min_{i \neq j} \text{dist}(p_i, p_j)$

## 分治算法

1. **分**：按 x 坐标中位数将点分为左右两半
2. **治**：递归求解左右子问题的最近点对
3. **合**：设 $\delta = \min(\delta_L, \delta_R)$，检查中线附近宽度 $2\delta$ 的带状区域内的点对
4. **关键**：带状区域内点按 y 排序，每个点只需与后面最多 6 个点比较

**复杂度**：$O(n \log n)$

## 实际应用

### 分子动力学模拟

- 原子/分子坐标在三维空间
- 最近邻距离决定相互作用力（Lennard-Jones 势）
- 空间划分（网格/八叉树）加速邻居搜索

### 天文数据分析

- 星表中天体坐标
- 寻找双星系统、星系对
- 大规模巡天数据（数十亿天体）

### 聚类分析（DBSCAN）

- 判断点是否在核心点的 $\epsilon$ 邻域内
- 最近邻距离指导 $\epsilon$ 参数选择
- k-distance 图：每个点到第 k 近邻的距离排序

### 地理信息系统

- 设施选址：最近两个设施的距离（竞争分析）
- 异常检测：距离过近的点对可能为重复记录

## 与高维扩展的对比

| 维度 | 算法 | 复杂度 |
|------|------|--------|
| 2D | 分治法 | $O(n \log n)$ |
| 3D+ | 分治法 | $O(n \log n)$（常数更大）|
| 高维 | KD-Tree / LSH | 近似 $O(n \log n)$ |
| 极高维 | 暴力或 LSH | $O(n^2)$ 或近似 |

## 效果评估

- 百万点 2D 最近点对：$< 100$ms
- 分子模拟中，空间划分将邻居搜索从 $O(n^2)$ 降至 $O(n)$
- 天文数据（10^9 点）使用近似算法（LSH）

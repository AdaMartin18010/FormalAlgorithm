# 案例：树链剖分在网络路由与文件系统权限查询中的应用

## 背景

在树结构上频繁进行路径查询（如求路径上的最大值、和、更新）时，朴素的 DFS 每次查询 $O(n)$ 不可接受。树链剖分（Heavy-Light Decomposition, HLD）将树分解为 $O(\log n)$ 条链，每条链用线段树维护，将路径查询降至 $O(\log^2 n)$。

## 问题定义

- **输入**：带权有根树，$n$ 个节点
- **操作**：
  - 查询路径 $u \to v$ 上的聚合值（max/sum）
  - 更新路径/子树上的值
- **目标**：每次操作 $O(\log^2 n)$

## 树链剖分结构

### 重链与轻边

- **重儿子**：子树大小最大的儿子
- **重链**：由重儿子连续连接形成的链
- **轻边**：连接不同重链的边
- **性质**：任意根到叶子的路径上，重链数 $\leq O(\log n)$

### 剖分后处理

- 按 DFS 序将节点映射到数组
- 同一条重链上的节点在数组中连续
- 线段树维护数组

### 路径查询

```
query(u, v):
    while head[u] != head[v]:
        if depth[head[u]] < depth[head[v]]: swap(u, v)
        query segment tree on [pos[head[u]], pos[u]]
        u = parent[head[u]]
    // now on same chain
    query segment tree on [pos[v], pos[u]] (v is LCA)
```

**复杂度**：$O(\log^2 n)$

## 实际应用

### 网络路由优化

- 路由器拓扑树
- 查询两点间链路的最大带宽/最小延迟
- 动态更新链路状态

### 文件系统权限

- 目录树，文件继承父目录权限
- 查询文件到根路径上的有效权限
- 批量修改某子树的权限

### 在线游戏

- 技能树/天赋树查询
- 公会组织架构权限
- 道具继承链

### 进化树分析

- 物种分类树的路径分析
- 基因特征的传递路径

## 与欧拉序+线段树的对比

| 查询类型 | HLD | 欧拉序 |
|---------|-----|--------|
| 路径查询 | ✅ $O(\log^2 n)$ | ❌ 不支持 |
| 子树查询 | ✅ $O(\log n)$ | ✅ $O(\log n)$ |
| 实现复杂度 | 较复杂 | 简单 |

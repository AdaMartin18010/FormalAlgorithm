# 案例：模拟退火在电路布局与旅行商问题中的应用

## 背景

许多实际优化问题（电路布局、调度、网络设计）是 NP-hard 的，精确算法在大规模实例上不可行。模拟退火（Simulated Annealing, SA）是一种通用启发式算法，通过模拟物理退火过程，在解空间中随机搜索，以高概率收敛到全局最优或近似最优解。

## 问题定义

- **解空间**：所有可行解的集合 $S$
- **能量函数**：$E(s)$，目标是最小化
- **邻居结构**：$N(s)$，从 $s$ 可达的邻近解
- **目标**：找到 $s^* = \arg\min_{s \in S} E(s)$

## 算法流程

```
1. 随机初始化解 s，设定初始温度 T
2. while T > T_min:
3.   for i = 1 to N_iter:
4.     生成邻居 s' ∈ N(s)
5.     计算 ΔE = E(s') - E(s)
6.     if ΔE < 0: 接受 s'（更优）
7.     else: 以概率 exp(-ΔE / T) 接受 s'（允许劣化解）
8.   T = T * α  // 降温，α ∈ [0.95, 0.999]
```

### 关键参数

| 参数 | 作用 | 典型值 |
|------|------|--------|
| 初始温度 $T_0$ | 初始接受劣解的概率 | $T_0 \approx -\Delta E_{max} / \ln(p_0)$ |
| 降温系数 $\alpha$ | 温度衰减速度 | 0.95 - 0.995 |
| 终止温度 | 停止条件 | $10^{-6} \cdot T_0$ |
| 每温度迭代次数 | 平衡搜索 | $100 \sim 10000$ |

## 实际应用

### 电路布局（VLSI Placement）

- 将逻辑门放置在芯片二维区域
- 目标：最小化连线总长度 + 避免重叠
- 解：每个门的位置坐标
- 邻居：交换两个门的位置或微移一个门
- SA 是商用 EDA 工具（Cadence、Synopsys）的核心组件

### 旅行商问题（TSP）

- 解：城市的访问顺序
- 邻居：2-opt（翻转一段路径）、3-opt
- SA 在数千城市实例上找到与最优解差距 $< 1\%$ 的方案
- Concorde（精确求解器）处理万级城市，SA 用于更大规模

### 调度优化

- 车间作业调度（Job Shop Scheduling）
- 航班/机组排班
- 解：任务顺序或资源分配
- SA 在工业界广泛使用（航空、制造）

### 蛋白质折叠

- 简化模型：氨基酸序列在网格上的折叠
- 能量：相邻非键合疏水氨基酸的接触数
- SA 搜索低能量构象

## 与遗传算法的对比

| 特性 | 模拟退火 | 遗传算法 |
|------|---------|---------|
| 搜索方式 | 单解随机游动 | 种群进化 |
| 参数调优 | 温度 schedule | 交叉、变异概率 |
| 收敛速度 | 较慢但稳定 | 可能更快 |
| 实现复杂度 | 简单 | 较复杂 |
| 适用问题 | 连续/离散均可 | 离散编码更自然 |

## 效果评估

- VLSI 布局：SA 在百万级单元上找到合理方案（数小时）
- TSP：2-opt + SA 在 1000 城市上差距 $< 0.5\%$
- 蛋白质折叠：SA 找到近天然态构象

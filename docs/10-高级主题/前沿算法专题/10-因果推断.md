## 1. 引言

### 1.1 关联 vs. 因果

统计学经典格言："相关性不等于因果性"。但传统机器学习主要关注预测（关联），而许多科学问题需要回答**干预**（intervention）的效果：

- 吸烟是否导致肺癌？
- 某药物是否能降低死亡率？
- 推荐系统改变用户行为还是仅仅反映了既有偏好？

**因果推断**提供了一套形式化框架，从观测数据中识别和估计因果效应。

### 1.2 核心挑战：混淆变量

**定义 1.1 (Confounder / 混淆变量)**：
变量 $Z$ 是处理 $X$ 和结果 $Y$ 的混淆变量，如果：

1. $Z$ 影响 $X$
2. $Z$ 影响 $Y$
3. $Z$ 不是 $X$ 和 $Y$ 之间的中介变量

混淆变量导致 $P(Y \mid X)$ 不能反映真实的因果效应 $P(Y \mid do(X))$。

## 2. 潜在结果框架（Rubin Causal Model）

### 2.1 基本定义

**定义 2.1 (潜在结果)**：
对于二元处理 $X \in \{0, 1\}$，每个个体 $i$ 有两个潜在结果：

- $Y_i(1)$：接受处理时的结果
- $Y_i(0)$：未接受处理时的结果

**观测结果**：$Y_i = X_i \cdot Y_i(1) + (1 - X_i) \cdot Y_i(0)$

**核心问题**：个体因果效应 $Y_i(1) - Y_i(0)$ 永远无法同时观测（**因果推断的基本问题**）。

### 2.2 平均处理效应

**定义 2.2 (ATE, Average Treatment Effect)**：
$$\text{ATE} = \mathbb{E}[Y(1) - Y(0)]$$

**定义 2.3 (ATT, Average Treatment Effect on the Treated)**：
$$\text{ATT} = \mathbb{E}[Y(1) - Y(0) \mid X = 1]$$

### 2.3 识别条件

**假设 2.1 (无混淆性 / Ignorability)**：
$$(Y(0), Y(1)) \perp\!\!\!\perp X \mid Z$$

即在给定协变量 $Z$ 后，处理分配与潜在结果独立。

**假设 2.2 (正值性 / Positivity)**：
$$0 < P(X = 1 \mid Z = z) < 1, \quad \forall z$$

**定理 2.1 (ATE 的可识别性)**：
在满足 Ignorability 和 Positivity 的条件下：
$$\text{ATE} = \mathbb{E}_Z\left[\mathbb{E}[Y \mid X = 1, Z] - \mathbb{E}[Y \mid X = 0, Z]\right]$$

### 2.4 估计方法

| 方法 | 核心思想 | 适用场景 |
|------|---------|---------|
| **匹配** | 为每个处理组个体找相似对照组个体 | 协变量维度低 |
| **逆概率加权 (IPW)** | 用倾向得分 $e(Z) = P(X=1 \mid Z)$ 加权 | 倾向得分可准确估计 |
| **双重稳健估计 (AIPW)** | 结合结果回归和 IPW，只要一个正确即可 | 高维协变量 |
| **双重机器学习 (DML)** | 用 ML 估计 nuisance functions，交叉拟合去偏 | 高维复杂数据 |

## 3. 结构因果模型（Pearl's SCM）

### 3.1 因果图

**定义 3.1 (DAG / 有向无环图)**：
节点表示变量，有向边 $X \rightarrow Y$ 表示 $X$ 是 $Y$ 的直接原因。

**基本结构**：

- **链** $X \rightarrow Z \rightarrow Y$：$X$ 通过 $Z$ 间接影响 $Y$
- **叉** $X \leftarrow Z \rightarrow Y$：$Z$ 是 $X$ 和 $Y$ 的共同原因（混淆）
- **对撞** $X \rightarrow Z \leftarrow Y$：$Z$ 是共同结果

**d-分离规则**：
在 DAG 中，路径 $p$ 被节点集 $Z$ d-分离，如果：

1. $p$ 包含链或叉结构且中间节点在 $Z$ 中；或
2. $p$ 包含对撞结构且中间节点及其后代都不在 $Z$ 中。

### 3.2 do-演算

**定义 3.2 (do-算子)**：
$P(Y \mid do(X = x))$ 表示对 $X$ 施加干预（强制设为 $x$）后 $Y$ 的分布。

**与条件概率的区别**：
$$P(Y \mid do(X = x)) \neq P(Y \mid X = x)$$

**do-演算三条规则**：

**规则 1（插入/删除观测）**：
若 $Y \perp\!\!\!\perp Z \mid X, W$ 在 $G_{\overline{X}}$ 中，则：
$$P(Y \mid do(X), Z, W) = P(Y \mid do(X), W)$$

**规则 2（交换行动与观测）**：
若 $Y \perp\!\!\!\perp Z \mid X, W$ 在 $G_{\overline{X}\underline{Z}}$ 中，则：
$$P(Y \mid do(X), do(Z), W) = P(Y \mid do(X), Z, W)$$

**规则 3（删除行动）**：
若 $Y \perp\!\!\!\perp Z \mid X, W$ 在 $G_{\overline{X}\overline{Z(W)}}$ 中，则：
$$P(Y \mid do(X), do(Z), W) = P(Y \mid do(X), W)$$

**定理 3.1 (Pearl, 1995)**：
若 $P(Y \mid do(X))$ 可通过 do-演算化简为仅含观测概率的表达式，则因果效应可识别。

## 4. 工具变量法

### 4.1 基本设定

当存在未观测混淆变量时，潜在结果框架的条件无法满足。工具变量提供了一种替代识别策略。

**定义 4.1 (工具变量)**：
变量 $Z$ 是处理 $X$ 对结果 $Y$ 的工具变量，如果：

1. **相关性**：$Z$ 与 $X$ 相关
2. **排他性**：$Z$ 只通过 $X$ 影响 $Y$（无直接效应）
3. **单调性**（可选）：$X_i(z)$ 对 $z$ 单调

### 4.2 局部平均处理效应

**定理 4.1 (LATE / Wald Estimator)**：
在工具变量假设下：
$$\text{LATE} = \frac{\mathbb{E}[Y \mid Z = 1] - \mathbb{E}[Y \mid Z = 0]}{\mathbb{E}[X \mid Z = 1] - \mathbb{E}[X \mid Z = 0]}$$

LATE 是**依从者**（compliers，即 $Z$ 改变其 $X$ 取值的子群体）的平均处理效应。

## 5. 因果发现

### 5.1 基于约束的方法

**PC 算法**（Spirtes et al., 2000）：

1. **骨架学习**：从完全图开始，用条件独立性检验删除边
2. **方向确定**：利用 v-结构（对撞结构）确定边的方向
3. **方向传播**：应用定向规则确定剩余边的方向

**假设**：因果充分性（无潜伏混淆变量）、 faithful（条件独立对应 d-分离）

### 5.2 基于分数的方法

**GES (Greedy Equivalence Search)**：
在等价类空间中进行贪心搜索，用 BIC 或 BDeu 分数评估模型。

## 6. 前沿方向

### 6.1 因果机器学习

- **因果表示学习**：学习解耦的因果变量表示
- **因果强化学习**：在存在因果结构的环境中优化策略
- **因果迁移学习**：利用因果不变性实现跨域泛化

### 6.2 因果与深度学习

- **因果效应估计网络 (CEVAE)**：用 VAE 处理未观测混淆变量
- **DragonNet**：利用深度学习实现高效的双重稳健估计

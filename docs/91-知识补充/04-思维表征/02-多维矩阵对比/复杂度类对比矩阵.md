## 时间复杂度类

| 复杂度类 | 定义 | 典型问题 | 与P关系 | 完备问题 | 包含关系 |
|----------|------|---------|--------|---------|---------|
| **P** | 确定图灵机多项式时间可解 | 排序、最短路径、最小生成树 | = | 电路求值 | L⊆NL⊆P |
| **NP** | 非确定图灵机多项式时间可解/多项式时间可验证 | 3-SAT、哈密顿回路、图着色 | P⊆NP | 3-SAT、团问题 | P⊆NP⊆PSPACE |
| **co-NP** | 补问题属于NP | 素数判定(Primes∈P)、UNSAT | P⊆co-NP | 合数判定 | NP∩co-NP |
| **RP** | 随机多项式时间，单侧错误 | 多项式恒等测试 | P⊆RP⊆NP | - | RP⊆NP |
| **co-RP** | RP的补 | 素性测试(Miller-Rabin) | P⊆co-RP | - | co-RP⊆co-NP |
| **BPP** | 随机多项式时间，双侧有界错误 | - | P⊆BPP | - | RP⊆BPP⊆Σ₂ |
| **ZPP** | 零错误概率多项式期望时间 | - | =RP∩co-RP | - | P⊆ZPP⊆NP∩co-NP |
| **NP∩co-NP** | 同时属于NP和co-NP | 素数判定(现在知∈P)、整数分解 | P⊆NP∩co-NP | - | - |

## 空间复杂度类

| 复杂度类 | 定义 | 典型问题 | 与P关系 | 完备问题 | 资源限制 |
|----------|------|---------|--------|---------|---------|
| **L** | 对数空间可解 | 图连通性(USTCON)、有向图可达? | ⊆P | USTCON | O(log n)空间 |
| **NL** | 非确定对数空间可解 | 有向图可达性(STCON) | ⊆P | STCON | O(log n)空间 |
| **P** | 多项式时间 | 大多数实际问题 | = | CVP | 时间受限 |
| **PSPACE** | 多项式空间可解 | QSAT、博弈问题 | ⊇NP | QSAT、广义地理 | O(n^k)空间 |
| **NPSPACE** | =PSPACE(Savitch定理) | - | =PSPACE | - | 空间等价 |
| **EXPSPACE** | 指数空间可解 | - | ⊇PSPACE | 带指数界限的停机问题 | 2^O(n^k)空间 |

## 指数时间复杂度类

| 复杂度类 | 定义 | 典型问题 | 与其他类关系 | 完备问题 |
|----------|------|---------|-------------|---------|
| **EXPTIME** | 指数时间可解 | 广义棋类游戏 | ⊇PSPACE | 带时间界限的停机问题 |
| **NEXPTIME** | 非确定指数时间 | - | ⊇EXPTIME | - |
| **EXPTIME-complete** | EXPTIME中最难问题 | 国际象棋、围棋(有界) | - | - |
| **2-EXPTIME** | 双重指数时间 | - | ⊇EXPTIME | Presburger算术 |
| **ELEMENTARY** | 有限层指数塔 | - | ⊇2-EXPTIME | - |

## 计数与概率类

| 复杂度类 | 定义 | 典型问题 | 与NP关系 | 特点 |
|----------|------|---------|---------|------|
| **#P** | 计算NP问题的解的数量 | 完美匹配计数、SAT解计数 | ⊇NP | 计数版本 |
| **PP** | 多数判定类 | MAJSAT | ⊇NP | 概率>1/2即可 |
| **BPP** | 有界错误概率多项式 | - | 与P关系未知 | 错误<1/3 |
| **AM** | Arthur-Merlin协议 | - | ⊇NP | 公共随机串 |
| **MA** | Merlin-Arthur协议 | - | ⊇NP | 私有随机串 |
| **IP** | 交互式证明 | - | =PSPACE | 多项式轮交互 |
| **PCP** | 概率可检查证明 | - | =NP | 对数随机、常数查询 |

## 多项式层级 (Polynomial Hierarchy)

| 层级 | 定义 | 等价形式 | 典型问题 | 包含关系 |
|------|------|---------|---------|---------|
| **Σ₀=Π₀=P** | 多项式时间 | - | P完全问题 | PH基座 |
| **Σ₁=NP** | ∃多项式时间验证 | 存在量词+多项式验证 | 3-SAT、团 | ⊇P |
| **Π₁=co-NP** | ∀多项式时间验证 | 全称量词+多项式验证 | UNSAT、团覆盖 | ⊆PH |
| **Σ₂** | ∃∀交替 | NP^NP | - | ⊇NP |
| **Π₂** | ∀∃交替 | co-NP^NP | - | ⊆PH |
| **Σₖ, Πₖ** | k层量词交替 | - | - | PH层级 |
| **PH** | ∪Σₖ = ∪Πₖ | 多项式层级并集 | - | ⊆PSPACE |

## 重要包含关系

```
L ⊆ NL ⊆ P ⊆ NP ⊆ PH ⊆ PSPACE ⊆ EXPTIME ⊆ NEXPTIME ⊆ EXPSPACE
    ⊆              ⊆         ⊆
    RP ⊆ BPP    Σ₂,Π₂      NPSPACE=PSPACE

    ZPP = RP ∩ co-RP

    NP ∩ co-NP ⊆ PH

    #P ⊆ PSPACE
```

## 完备问题对比

| 复杂度类 | 完备问题 | 问题描述 | 归约类型 |
|----------|---------|---------|---------|
| **L** | USTCON | 无向图s-t连通性 | 对数空间归约 |
| **NL** | STCON | 有向图s-t连通性 | 对数空间归约 |
| **P** | CVP | 电路值问题 | 对数空间归约 |
| **NP** | 3-SAT | 3-满足性问题 | 多项式时间归约 |
| **co-NP** | UNSAT | 公式不可满足 | 多项式时间归约 |
| **PSPACE** | QSAT | 量词化布尔公式 | 多项式时间归约 |
| **EXPTIME** | 有界停机问题 | 带时间限的图灵机停机 | 多项式时间归约 |
| **NEXPTIME** | SUCCINCT-SAT | 压缩编码的SAT | 多项式时间归约 |
| **#P** | #SAT | SAT解的计数 | 图灵归约 |
| **PP** | MAJSAT | 多数满足的赋值 | 多项式时间归约 |

## 复杂度类性质对比

| 性质 | P | NP | PSPACE | EXPTIME |
|------|---|----|--------|---------|
| **对补封闭** | ✓ | 未知 | ✓ | ✓ |
| **对交封闭** | ✓ | ✓ | ✓ | ✓ |
| **对并封闭** | ✓ | ✓ | ✓ | ✓ |
| **对连接封闭** | ✓ | ✓ | ✓ | ✓ |
| **存在完备问题** | ✓ | ✓ | ✓ | ✓ |
| **严格包含于上级** | 未知 | 已知⊂PSPACE | 已知⊂EXPTIME | - |

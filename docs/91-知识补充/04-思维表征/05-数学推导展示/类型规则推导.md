## 简单类型λ演算 (Simply Typed Lambda Calculus)

### 语法

```
类型 τ ::= Bool | Nat | τ → τ
表达式 e ::= x | λx:τ.e | e e | true | false | if e then e else e
           | 0 | succ e | pred e | iszero e
```

### 类型环境

$$\Gamma ::= \emptyset \mid \Gamma, x:\tau$$

表示变量到类型的映射。

## 类型规则推导

### 变量规则 (T-Var)

```
若 x:τ ∈ Γ
────────────────
Γ ⊢ x : τ
```

**含义：** 如果变量x在环境Γ中有类型τ，那么x的类型就是τ。

**示例：**

```
Γ = {x:Bool, y:Nat→Bool}
─────────────────────────
Γ ⊢ x : Bool
```

---

### 抽象规则 (T-Abs)

```
Γ, x:τ₁ ⊢ e : τ₂
───────────────────────
Γ ⊢ λx:τ₁.e : τ₁ → τ₂
```

**推导过程：**

假设在扩展环境Γ, x:τ₁下，表达式e的类型是τ₂，那么λ抽象λx:τ₁.e的类型是函数类型τ₁→τ₂。

**示例：**

```
Γ, x:Bool ⊢ if x then 0 else 1 : Nat
─────────────────────────────────────
Γ ⊢ λx:Bool. if x then 0 else 1 : Bool → Nat
```

---

### 应用规则 (T-App)

```
Γ ⊢ e₁ : τ₁ → τ₂    Γ ⊢ e₂ : τ₁
────────────────────────────────
Γ ⊢ e₁ e₂ : τ₂
```

**推导过程：**

```
已知:
1. e₁ 的类型是 τ₁ → τ₂ (函数，接受τ₁，返回τ₂)
2. e₂ 的类型是 τ₁ (符合函数参数类型)

推导:
应用 e₁ 到 e₂ 的结果类型是 τ₂
```

**示例：**

```
Γ ⊢ λx:Bool.x : Bool → Bool    Γ ⊢ true : Bool
──────────────────────────────────────────────
Γ ⊢ (λx:Bool.x) true : Bool
```

---

### 布尔值规则

```
────────────
Γ ⊢ true : Bool

─────────────
Γ ⊢ false : Bool

Γ ⊢ e₁ : Bool    Γ ⊢ e₂ : τ    Γ ⊢ e₃ : τ
─────────────────────────────────────────
Γ ⊢ if e₁ then e₂ else e₃ : τ
```

**条件规则要求：** 两个分支必须有相同类型！

**正确示例：**

```
Γ ⊢ true : Bool    Γ ⊢ 0 : Nat    Γ ⊢ 1 : Nat
─────────────────────────────────────────────
Γ ⊢ if true then 0 else 1 : Nat
```

**错误示例：**

```
Γ ⊢ true : Bool    Γ ⊢ 0 : Nat    Γ ⊢ false : Bool
    ←────────────────────────────────────────────→
    类型不匹配! e₂:Nat ≠ e₃:Bool
```

## 类型推导算法示例

### 示例1：推导 (λx:Bool.x) true 的类型

```
目标: 推导 ∅ ⊢ (λx:Bool.x) true : ?

步骤1: 分析应用
需要:
  - e₁ = λx:Bool.x 有类型 τ₁ → τ₂
  - e₂ = true 有类型 τ₁
结果类型将是 τ₂

步骤2: 推导 e₁ 的类型
  使用 T-Abs:
  前提: x:Bool ⊢ x : Bool (T-Var)
  ─────────────────────────
  ∅ ⊢ λx:Bool.x : Bool → Bool

步骤3: 推导 e₂ 的类型
  ∅ ⊢ true : Bool (T-True)

步骤4: 应用 T-App
  ∅ ⊢ λx:Bool.x : Bool → Bool    ∅ ⊢ true : Bool
  ───────────────────────────────────────────────
  ∅ ⊢ (λx:Bool.x) true : Bool

结果: Bool
```

### 示例2：推导 λf:Bool→Bool. λx:Bool. f (f x)

```
目标: 推导 ∅ ⊢ λf:Bool→Bool. λx:Bool. f (f x) : ?

步骤1: 外层抽象 (T-Abs)
  假设 f:Bool→Bool
  推导 f:Bool→Bool ⊢ λx:Bool. f (f x) : ?

步骤2: 内层抽象 (T-Abs)
  假设 f:Bool→Bool, x:Bool
  推导 f (f x) 的类型

步骤3: 分析 f (f x)
  子步骤3a: 推导 f x
    - f : Bool → Bool
    - x : Bool
    - f x : Bool (T-App)

  子步骤3b: 推导 f (f x)
    - f : Bool → Bool
    - f x : Bool
    - f (f x) : Bool (T-App)

步骤4: 回代
  f:Bool→Bool, x:Bool ⊢ f (f x) : Bool
  ────────────────────────────────────
  f:Bool→Bool ⊢ λx:Bool. f (f x) : Bool → Bool (T-Abs)

  ────────────────────────────────────────────────────
  ∅ ⊢ λf:Bool→Bool. λx:Bool. f (f x) : (Bool→Bool) → Bool → Bool (T-Abs)

结果: (Bool → Bool) → Bool → Bool
```

## 多态类型规则 (System F)

### 语法扩展

```
类型 τ ::= ... | α | ∀α.τ
表达式 e ::= ... | Λα.e | e[τ]
```

### 类型抽象规则

```
Γ ⊢ e : τ    α ∉ FV(Γ)
───────────────────────
Γ ⊢ Λα.e : ∀α.τ
```

### 类型应用规则

```
Γ ⊢ e : ∀α.τ
─────────────────
Γ ⊢ e[τ'] : [τ'/α]τ
```

### 示例：多态恒等函数

```
推导: ∅ ⊢ Λα.λx:α.x : ∀α.α→α

步骤1: 类型抽象
  假设 α 是类型变量
  推导 α 在环境中 ⊢ λx:α.x : α→α

步骤2: 普通抽象
  α 在环境中, x:α ⊢ x : α (T-Var)
  ─────────────────────────────
  α 在环境中 ⊢ λx:α.x : α→α (T-Abs)

步骤3: 类型抽象 (T-TAbs)
  ─────────────────────────────
  ∅ ⊢ Λα.λx:α.x : ∀α.α→α

使用:
(Λα.λx:α.x)[Bool] : Bool→Bool
(Λα.λx:α.x)[Nat] : Nat→Nat
```

## 类型安全定理

### 定理陈述

**定理 (类型安全)：** 如果 ∅ ⊢ e : τ，那么：

1. **进展性 (Progress)：** e 或者是值，或者存在 e' 使得 e → e'
2. **保持性 (Preservation)：** 如果 e → e'，那么 ∅ ⊢ e' : τ

### 保持性证明 (Subject Reduction)

```
定理: 若 Γ ⊢ e : τ 且 e → e', 则 Γ ⊢ e' : τ

证明: 对求值规则进行归纳

情况1: (λx:τ₁.e) v → [v/x]e   (β归约)
────────────────────────────────────
前提:
  Γ ⊢ λx:τ₁.e : τ₁→τ    Γ ⊢ v : τ₁

由 T-Abs 反推:
  Γ, x:τ₁ ⊢ e : τ

由替换引理 (Substitution Lemma):
  若 Γ, x:τ₁ ⊢ e : τ 且 Γ ⊢ v : τ₁, 则 Γ ⊢ [v/x]e : τ

因此: Γ ⊢ e' : τ  ✓

情况2: if true then e₂ else e₃ → e₂
────────────────────────────────────
前提:
  Γ ⊢ true : Bool    Γ ⊢ e₂ : τ    Γ ⊢ e₃ : τ

显然: Γ ⊢ e₂ : τ  ✓

其他情况类似...
```

### 替换引理 (Substitution Lemma)

```
引理: 若 Γ, x:τ₁ ⊢ e : τ 且 Γ ⊢ v : τ₁, 则 Γ ⊢ [v/x]e : τ

证明: 对 e 的结构进行归纳

基本情况: e = x
  [v/x]x = v
  由前提 Γ ⊢ v : τ₁ = τ  ✓

基本情况: e = y ≠ x
  [v/x]y = y
  由前提 Γ, x:τ₁ ⊢ y : τ
  由于 y ≠ x, 有 Γ ⊢ y : τ  ✓

归纳步骤: e = e₁ e₂
  由归纳假设:
    Γ ⊢ [v/x]e₁ : τ₁→τ
    Γ ⊢ [v/x]e₂ : τ₁
  由 T-App:
    Γ ⊢ [v/x](e₁ e₂) : τ  ✓

其他情况类似...
```

## 类型推导算法 (Hindley-Milner)

### 算法W概要

```
W(Γ, e) = (S, τ)

输入: 环境Γ, 表达式e
输出: 代换S, 类型τ
使得: S(Γ) ⊢ e : τ

情况1: e = x (变量)
  若 x:∀α₁...αₙ.τ' ∈ Γ
  返回 ([], [βᵢ/αᵢ]τ')  其中βᵢ是新变量

情况2: e = λx.e₁
  β = 新类型变量
  (S₁, τ₁) = W(Γ, x:β, e₁)
  返回 (S₁, S₁(β) → τ₁)

情况3: e = e₁ e₂
  (S₁, τ₁) = W(Γ, e₁)
  (S₂, τ₂) = W(S₁(Γ), e₂)
  β = 新类型变量
  V = unify(S₂(τ₁), τ₂ → β)
  返回 (V ∘ S₂ ∘ S₁, V(β))

情况4: e = let x = e₁ in e₂
  (S₁, τ₁) = W(Γ, e₁)
  Γ' = S₁(Γ), x:Gen(Γ', τ₁)
  (S₂, τ₂) = W(Γ', e₂)
  返回 (S₂ ∘ S₁, τ₂)
```

## 类型规则总结表

| 规则名 | 前提 | 结论 | 说明 |
|-------|------|------|------|
| T-Var | x:τ ∈ Γ | Γ ⊢ x : τ | 变量查找 |
| T-Abs | Γ,x:τ₁ ⊢ e:τ₂ | Γ ⊢ λx:τ₁.e : τ₁→τ₂ | 函数引入 |
| T-App | Γ ⊢ e₁:τ₁→τ₂, Γ ⊢ e₂:τ₁ | Γ ⊢ e₁ e₂ : τ₂ | 函数消除 |
| T-True | - | Γ ⊢ true : Bool | 布尔真 |
| T-False | - | Γ ⊢ false : Bool | 布尔假 |
| T-If | Γ ⊢ e₁:Bool, Γ ⊢ e₂:τ, Γ ⊢ e₃:τ | Γ ⊢ if e₁ then e₂ else e₃ : τ | 条件 |

---

*类型规则是形式化类型系统的基础，理解这些规则有助于深入理解编程语言的类型机制*

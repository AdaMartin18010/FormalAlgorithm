## 一、核心概念

### 1.1 定义

**正式定义**:
程序验证是使用形式化方法证明程序满足其规范的过程。

**主要方法**:

1. **公理语义** (Hoare逻辑)
2. **操作语义** (迹验证)
3. **指称语义** (不动点)
4. **自动定理证明** (SMT、模型检验)

**Hoare三元组**:
$$\{P\}\ C\ \{Q\}$$

含义: 若前置条件$P$成立，执行命令$C$后，后置条件$Q$成立。

**最弱前置条件 (WP)**:
$$wp(C, Q) = \text{使} \{P\}C\{Q\} \text{成立的最弱} P$$

**直观解释**:
程序验证回答"程序是否正确"的问题，通过数学证明确保程序在所有可能的执行路径上都满足规范。

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 完备性 | Hoare逻辑对偏正确性完备 | Gödel限制 |
| 可判定性 | 一般问题不可判定 | 近似方法 |
| 组合性 | 可模块化验证 | 自底向上 |
| 自动化 | 部分可自动化 | 循环需不变式 |

### 1.3 变体

**部分正确性**:

- 若程序终止，则规范成立
- 不完备（停机问题）

**完全正确性**:

- 程序必定终止，且规范成立
- 需要秩函数/变体

**并发验证**:

- Owicki-Gries方法
- 分离逻辑

## 三、形式化证明

### 3.1 核心定理

**定理 1** (Hoare逻辑合理性):
若 $\vdash \{P\}C\{Q\}$，则 $\models \{P\}C\{Q\}$（语义满足）。

**定理 2** (相对完备性 - Cook):
若 $\models \{P\}C\{Q\}$ 且表达力足够，则 $\vdash \{P\}C\{Q\}$。

### 3.2 Hoare推理规则

$$
\text{赋值: } \{Q[e/x]\}\ x := e\ \{Q\}
$$

$$
\text{顺序: } \frac{\{P\}C_1\{R\} \quad \{R\}C_2\{Q\}}{\{P\}C_1;C_2\{Q\}}
$$

$$
\text{条件: } \frac{\{P \wedge b\}C_1\{Q\} \quad \{P \wedge \neg b\}C_2\{Q\}}{\{P\}\text{if } b \text{ then } C_1 \text{ else } C_2\{Q\}}
$$

$$
\text{循环: } \frac{\{I \wedge b\}C\{I\}}{\{I\}\text{while } b \text{ do } C\{I \wedge \neg b\}}
$$

## 四、算法/方法详解

### 4.1 最弱前置条件计算

```
算法: WP计算
输入: 命令C，后置条件Q
输出: 最弱前置条件

wp(skip, Q) = Q
wp(x:=e, Q) = Q[e/x]
wp(C1;C2, Q) = wp(C1, wp(C2, Q))
wp(if b then C1 else C2, Q) =
  (b → wp(C1, Q)) ∧ (¬b → wp(C2, Q))
wp(while b do C, Q) =
  需要循环不变式I
  满足: I ∧ ¬b → Q
        I ∧ b → wp(C, I)
```

### 4.2 验证条件生成

```
算法: VC生成
1. 将程序注释为Hoare逻辑证明轮廓
2. 对每个循环，提供不变式
3. 生成验证条件（纯逻辑公式）
4. 使用SMT求解器或定理证明器验证
```

## 五、示例与实例

### 5.1 标准示例

**示例 1**: 阶乘计算验证

```
{ n >= 0 }
i := 0;
f := 1;
while i < n do
  i := i + 1;
  f := f * i
{ f = n! }

不变式: 0 <= i <= n ∧ f = i!
```

**证明**:

- 初始化: i=0, f=1 → 0<=0<=n ∧ 1=0! ✓
- 保持: 若i<n, f=i!，则i:=i+1, f:=f*(i+1) → f=(i+1)!
- 终止: i>=n ∧ f=i! ∧ i<=n → i=n ∧ f=n!

### 5.2 代码实现

**Dafny示例**:

```dafny
method Factorial(n: int) returns (f: int)
  requires n >= 0
  ensures f == FactorialFunction(n)
{
  f := 1;
  var i := 0;
  while i < n
    invariant 0 <= i <= n
    invariant f == FactorialFunction(i)
  {
    i := i + 1;
    f := f * i;
  }
}

function FactorialFunction(n: int): int
  requires n >= 0
{
  if n == 0 then 1 else n * FactorialFunction(n-1)
}
```

**Coq/CFML示例**:

```coq
Require Import CFML.CFLib.

(* 阶乘的函数式实现 *)
Fixpoint fact (n: nat) : nat :=
  match n with
  | 0 => 1
  | S n' => n * fact n'
  end.

(* 证明其正确性 *)
Lemma fact_correct : forall n, fact n = n!.
Proof.
  induction n; simpl; auto.
Qed.

(* 命令式版本验证 *)
(* 使用Separation Logic *)
```

**Why3示例**:

```why3
theory Factorial
  use import int.Int
  use import int.Fact

  let factorial (n: int) : int
    requires { n >= 0 }
    ensures  { result = fact n }
  =
    let ref f = 1 in
    let ref i = 0 in
    while i < n do
      invariant { 0 <= i <= n /\ f = fact i }
      variant   { n - i }
      i := i + 1;
      f := f * i
    done;
    f
end
```

## 六、习题

### 6.1 理解题 (L1-L2)

1. **Hoare三元组** [难度⭐]

   计算 `wp(x := x + 1, x > 5)`

   <details>
   <summary>解答</summary>

   ```
   wp(x := x + 1, x > 5)
   = (x > 5)[x+1/x]
   = x + 1 > 5
   = x > 4
   ```

   </details>

2. **循环不变式** [难度⭐⭐]

   为以下程序找到循环不变式：

   ```
   s := 0; i := 0;
   while i < n do
     s := s + a[i];
     i := i + 1
   ```

   <details>
   <summary>解答</summary>

   不变式: `0 <= i <= n ∧ s = sum(a[0..i-1])`

   即s是数组前i个元素的和。

   </details>

### 6.2 应用题 (L3)

1. **二分查找验证** [难度⭐⭐]

   写出二分查找的规范并找出关键不变式。

   <details>
   <summary>解答</summary>

   规范:

   ```
   前置: 数组a已排序
   后置: 若返回值r >= 0，则a[r] = x
         若返回值r < 0，则x不在a中
   ```

   不变式: `0 <= lo <= hi <= n` 且若x在a中，则x在a[lo..hi-1]中

   </details>

### 6.3 证明题 (L4-L5)

1. **欧几里得算法** [难度⭐⭐⭐]

   验证欧几里得算法计算GCD的正确性。

   <details>
   <summary>解答</summary>

   ```
   程序:
   while b != 0 do
     t := b;
     b := a mod b;
     a := t
   done

   不变式: gcd(a, b) = gcd(A, B) （A,B为初始值）

   终止性: b严格递减且>=0

   后置条件: a = gcd(A, B)
   ```

   </details>

2. **分离逻辑** [难度⭐⭐⭐⭐]

   用分离逻辑验证链表反转。

   <details>
   <summary>解答概要</summary>

   分离逻辑断言: `l ↦ [v, n] * list(n, xs)` 表示l指向值为v、下一节点为n的单元，n指向列表xs。

   ```
   { list(x, xs) }
   y := null;
   while x != null do
     t := x.next;
     x.next := y;
     y := x;
     x := t
   { list(y, reverse(xs)) }
   ```

   不变式: `∃ xs1 xs2. list(x, xs1) * list(y, xs2) ∧ xs = xs1 ++ reverse(xs2)`

   </details>

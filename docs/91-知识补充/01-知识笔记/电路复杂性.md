## 一、核心概念

### 1.1 定义

**正式定义**:

**布尔电路**: 有向无环图，其中：

- 输入顶点：标记为变量x₁,...,xₙ或常量0,1
- 内部顶点（门）：标记为布尔运算（AND, OR, NOT等）
- 输出顶点：一个或多个

**电路规模 (Size)**: 电路中门的数量
**电路深度 (Depth)**: 最长输入到输出路径的长度

**电路族**: {Cₙ}ₙ≥₁，其中Cₙ处理n位输入

**P/poly**: 被多项式规模电路族可判定的语言类
$$\text{P/poly} = \{L : \exists \text{ 电路族 } \{C_n\}, |C_n| = n^{O(1)}, L(C_n) = L \cap \{0,1\}^n\}$$

**AC⁰**: 常数深度、多项式规模、无界扇入AND/OR/NOT门
**NC¹**: O(log n)深度、多项式规模、有界扇入门
**NC**: Nick's Class，可用多对数深度并行计算的类

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 非一致性 | 不同输入长度可有完全不同的电路 | 比图灵机更强大 |
| P ⊆ P/poly | 所有P问题有多项式电路 | 电路模拟图灵机 |
| 下界困难 | 证明超多项式下界极难 | 与P vs NP相关 |
| AC⁰限制 | 无法计算奇偶校验函数 | Furst-Saxe-Sipser |

### 1.3 复杂性类层次

```
AC⁰ ⊂ NC¹ ⊆ NC² ⊆ ... ⊆ NC = ⋃ NCⁱ ⊆ P ⊆ P/poly
```

## 三、形式化证明

### 3.1 P ⊆ P/poly

**定理**: P ⊆ P/poly

**证明**:

设L ∈ P，被图灵机M在nᵏ时间内判定。

对输入长度n，M使用nᵏ时间，最多访问nᵏ个带单元。

将M的计算表格化：

- 行：时间步0到nᵏ
- 列：带单元1到nᵏ
- 内容：(状态, 符号, 头位置)

每个单元仅依赖于上一行的3个单元（当前及左右）。

构造电路模拟：

- 每层电路模拟一个时间步
- 大小O(nᵏ)，深度O(nᵏ)
- 总门数n^O(k) = n^O(1)

故L ∈ P/poly。

### 3.2 AC⁰下界 (奇偶校验)

**定理** (Furst-Saxe-Sipser, Ajtai): PARITY ∉ AC⁰

**证明概要** (Håstad switching lemma):

对深度d的AC⁰电路，通过随机限制（随机设置部分变量为0/1）：

- 以高概率，低层OR门可替换为小规模的DNF
- 重复应用，最终电路退化为常数函数或依赖很少变量
- 奇偶校验依赖所有变量，矛盾

**推论**: AC⁰ ⊂ NC¹

## 四、算法/方法详解

### 4.1 电路综合

**从真值表构造**:

- DNF/CNF构造：规模指数
- 最小电路：NP困难

**启发式方法**:

- 二元决策图 (BDD)
- 布尔可满足性 (SAT) 求解器

### 4.2 电路下界技术

**限制法 (Restriction Method)**:

- 随机设置部分输入
- 分析电路简化

**多项式法**:

- 用多项式近似布尔函数
- 分析多项式次数

**自然证明障碍**:

- 大多数下界技术都是"自然的"
- 若P ≠ NP，则不存在能分离P和NP的自然证明
- 暗示需要全新技术

## 五、示例与实例

### 5.1 AC⁰中的函数

**多数函数 (MAJORITY)**: AC⁰中可计算吗？

- 答案：不在AC⁰中 (Razborov-Smolensky)

**加法电路**:

- 进位前瞻加法器：深度O(log n)，规模O(n)
- 属于NC¹

| 函数 | AC⁰ | NC¹ | 备注 |
|------|-----|-----|------|
| AND/OR | ✓ | ✓ | 平凡 |
| PARITY | ✗ | ✓ | AC⁰下界经典结果 |
| MAJORITY | ✗ | ✓ | Razborov-Smolensky |
| 乘法 | ✗ | ✓ | 深度O(log n) |

## 六、习题

### 6.1 理解题 (L1-L2)

1. **P/poly vs P** [难度⭐⭐]

   解释为什么P/poly包含不可判定语言。

   <details>
   <summary>解答</summary>

   P/poly允许电路族"非一致"：不同输入长度可有完全不同的电路。

   可构造电路族识别停机问题的特征串：
   - Cₙ硬编码前n个实例的答案
   - 每个Cₙ规模O(n)，但构造Cₙ需要无限计算能力

   因此P/poly甚至包含不可判定语言。

   </details>

2. **电路模拟TM** [难度⭐⭐⭐]

   估计模拟n步图灵机所需的电路规模。

   <details>
   <summary>解答</summary>

   设TM使用s(n)空间，t(n)时间。

   配置大小：O(s(n))
   配置数：t(n) × s(n)

   模拟转移需要O(s(n))门。
   总门数：O(t(n) · s(n))

   对s(n)=t(n)=nᵏ，规模O(n²ᵏ)。

   </details>

### 6.2 证明题 (L3-L4)

1. **BPP ⊆ P/poly** [难度⭐⭐⭐⭐]

   用Adleman定理证明BPP ⊆ P/poly。

   <details>
   <summary>解答</summary>

   设L ∈ BPP，由错误放大，存在M错误概率 < 2^(-n)。

   对固定n，随机选择r：
   - 坏r（使某x出错）比例 < 2ⁿ · 2^(-n) = 1

   实际上用联合界：坏r比例 < 1，故存在好r。

   将此r硬编码到电路Cₙ中。
   电路模拟M(x,r)，规模多项式。

   </details>

### 7.1 经典应用

| 应用场景 | 说明 |

**NC类**: 可高效并行计算的问题

- 深度≈并行时间
- 规模≈总工作量
- NC = P？是开放问题（P完全问题可能不在NC中）

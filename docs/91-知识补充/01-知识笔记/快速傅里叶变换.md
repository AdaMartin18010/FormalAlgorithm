## 一、核心概念

### 1.1 定义

**正式定义**:
**离散傅里叶变换(DFT)**将时域上的$n$个复数样本$x_0, x_1, \ldots, x_{n-1}$转换为频域表示$X_0, X_1, \ldots, X_{n-1}$：

$$X_k = \sum_{j=0}^{n-1} x_j \cdot \omega_n^{jk}, \quad k = 0, 1, \ldots, n-1$$

其中$\omega_n = e^{2\pi i / n}$是**n次单位根**，满足$\omega_n^n = 1$。

**快速傅里叶变换(FFT)**是计算DFT的高效算法，时间复杂度$O(n \log n)$。

**逆DFT(IDFT)**:
$$x_j = \frac{1}{n} \sum_{k=0}^{n-1} X_k \cdot \omega_n^{-jk}$$

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 时间复杂度 | $O(n \log n)$ | Cooley-Tukey算法 |
| 空间复杂度 | $O(n)$ | 可原地计算 |
| 数值稳定性 | 良好 | 需注意蝶形运算舍入误差 |

### 1.3 变体

- **Cooley-Tukey**: 经典算法，要求n为2的幂
- **Bluestein**: 适用于任意长度
- **NTT**: 模运算版本，无浮点误差

## 三、形式化证明

### 3.1 核心定理

**定理 1**: Cooley-Tukey FFT算法正确计算DFT，时间复杂度$O(n \log n)$。

**证明**:

```
设n为偶数，将DFT分为奇偶两部分：

X_k = Σ_{j=0}^{n-1} x_j ω_n^{jk}
    = Σ_{m=0}^{n/2-1} x_{2m} ω_n^{2mk} + Σ_{m=0}^{n/2-1} x_{2m+1} ω_n^{(2m+1)k}
    = E_k + ω_n^k O_k

其中E_k是偶数下标的DFT，O_k是奇数下标的DFT。

复杂度：T(n) = 2T(n/2) + O(n) = O(n log n) ∎
```

## 五、示例与实例

### 5.2 代码实现

**Python实现**:

```python
import cmath
import math

def fft(a, invert=False):
    n = len(a)
    if n == 1:
        return a[:]

    # 位逆序置换
    j = 0
    for i in range(1, n):
        bit = n >> 1
        while j & bit:
            j ^= bit
            bit >>= 1
        j ^= bit
        if i < j:
            a[i], a[j] = a[j], a[i]

    # 迭代FFT
    length = 2
    while length <= n:
        ang = 2 * math.pi / length * (-1 if invert else 1)
        wlen = complex(math.cos(ang), math.sin(ang))

        for i in range(0, n, length):
            w = 1
            for j in range(i, i + length // 2):
                u = a[j]
                v = a[j + length // 2] * w
                a[j] = u + v
                a[j + length // 2] = u - v
                w *= wlen
        length <<= 1

    if invert:
        for i in range(n):
            a[i] /= n
    return a
```

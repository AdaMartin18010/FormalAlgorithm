## 一、核心概念

### 1.1 定义

**正式定义**:
**后缀数组（Suffix Array）**: 字符串所有后缀按字典序排序后的起始位置数组。

**后缀树（Suffix Tree）**: 包含字符串所有后缀的压缩字典树。

**后缀自动机（SAM）**: 识别字符串所有后缀的最小DFA。

**最长公共前缀（LCP）数组**: 后缀数组中相邻后缀的最长公共前缀长度。

**直观解释**:
后缀数组就像"字典的索引"——把字符串的所有后缀（从每个位置开始到结尾的子串）按字典序排列，记录每个后缀的起始位置。后缀树是这些后缀的"压缩电话簿"，共享前缀的后缀合并存储。后缀自动机则是"最精简的后缀识别器"，用最少状态识别所有后缀。

**关键要点**:

- **后缀数组**: 空间高效 $O(n)$，查询 $O(\log n)$ 或 $O(m + \log n)$
- **后缀树**: 构建复杂，但支持更多操作
- **后缀自动机**: 线性构造，适合在线处理
- **LCP数组**: 与后缀数组配合，支持RMQ查询

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 空间复杂度 | 后缀数组 $O(n)$，后缀树 $O(n)$ | SAM $O(n)$ |
| 构造时间 | $O(n \log n)$ 或 $O(n)$ | 取决于算法 |
| 子串查询 | 转为后缀前缀问题 | 二分或遍历 |
| 在线构造 | SAM支持 | 逐个添加字符 |

### 1.3 变体

**后缀数组**: 配合LCP数组和RMQ数据结构
**后缀树**: 广义后缀树处理多字符串
**后缀自动机**: 可持久化SAM处理子串问题
**回文树**: 处理回文子串

## 三、形式化证明

### 3.1 核心定理

**定理 1 (后缀数组与后缀树的等价性)**: 后缀数组 + LCP数组可在线性时间构造后缀树。

**证明**: 利用Cartesian树构造，LCP数组定义了后缀树的内部节点高度。

∎

**定理 2 (SAM的最小性)**: 后缀自动机是识别字符串所有后缀的最小DFA。

*证明*: SAM的每个状态对应endpos等价类，这是最粗的右不变等价关系。∎

## 四、算法/方法详解

### 4.1 后缀数组构造（倍增法）

```
输入: 字符串 s，长度 n

// 初始: 按单个字符排序
sa = sorted(range(n), key=lambda i: s[i])
rank[i] = s[i] 的排序位置

k = 1
while k < n:
    // 按 (rank[i], rank[i+k]) 排序
    sa.sort(key=lambda i: (rank[i], rank[i+k] if i+k < n else -1))

    // 更新rank
    new_rank = [0] * n
    for i in range(1, n):
        new_rank[sa[i]] = new_rank[sa[i-1]] + 1 if (rank[sa[i]], rank[sa[i]+k]) != (rank[sa[i-1]], rank[sa[i-1]+k]) else new_rank[sa[i-1]]
    rank = new_rank
    k *= 2

return sa
```

### 4.2 后缀自动机构造

```
初始化:
    last = root = 新状态
    root.len = 0, root.link = -1

对于每个字符 c:
    cur = 新状态，cur.len = last.len + 1
    p = last
    while p != -1 且 p没有c转移:
        添加转移 p --c--> cur
        p = p.link

    if p == -1:
        cur.link = root
    else:
        q = p通过c转移到的状态
        if p.len + 1 == q.len:
            cur.link = q
        else:
            clone = q的副本
            clone.len = p.len + 1
            while p != -1 且 p通过c转移到q:
                修改p的c转移指向clone
                p = p.link
            q.link = cur.link = clone

    last = cur
```

## 五、示例与实例

### 5.1 后缀数组示例

**字符串**: "banana"

**后缀**:

- 0: banana
- 1: anana
- 2: nana
- 3: ana
- 4: na
- 5: a

**排序后**: a(5), ana(3), anana(1), banana(0), na(4), nana(2)

**后缀数组**: [5, 3, 1, 0, 4, 2]

**LCP数组**: [0, 1, 3, 0, 0, 2]

### 5.2 代码实现

```python
def build_suffix_array(s):
    """倍增法构造后缀数组 O(n log n)"""
    n = len(s)
    k = 1
    sa = list(range(n))
    rank = [ord(c) for c in s]
    tmp = [0] * n

    while True:
        sa.sort(key=lambda x: (rank[x], rank[x+k] if x+k < n else -1))
        tmp[sa[0]] = 0
        for i in range(1, n):
            prev, cur = sa[i-1], sa[i]
            tmp[cur] = tmp[prev] + ((rank[prev], rank[prev+k] if prev+k < n else -1) <
                                     (rank[cur], rank[cur+k] if cur+k < n else -1))
        rank, tmp = tmp, rank
        if rank[sa[-1]] == n-1:
            break
        k *= 2

    return sa

def build_lcp(s, sa):
    """Kasai算法构造LCP数组 O(n)"""
    n = len(s)
    rank = [0] * n
    for i, pos in enumerate(sa):
        rank[pos] = i

    lcp = [0] * n
    k = 0
    for i in range(n):
        if rank[i] == n - 1:
            k = 0
            continue
        j = sa[rank[i] + 1]
        while i + k < n and j + k < n and s[i+k] == s[j+k]:
            k += 1
        lcp[rank[i]] = k
        if k:
            k -= 1

    return lcp

# 使用
s = "banana"
sa = build_suffix_array(s)
lcp = build_lcp(s, sa)
print(f"后缀数组: {sa}")
print(f"LCP数组: {lcp}")
```

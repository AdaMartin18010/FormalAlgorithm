## 一、核心概念

### 1.1 定义

**正式定义**:
简单类型λ演算（Simply Typed Lambda Calculus, λ→）是在无类型λ演算基础上添加类型约束的形式系统。

**类型语法**:
$$\tau, \sigma ::= \iota \mid \tau \to \sigma$$

其中：

- $\iota$：基本类型（如 Bool, Int）
- $\tau \to \sigma$：函数类型

**项语法**（Curry风格）:
$$t ::= x \mid \lambda x.t \mid t\ s$$

**类型判断**:
$$\Gamma \vdash t : \tau$$
表示在上下文$\Gamma$中，项$t$具有类型$\tau$。

**直观解释**:
简单类型系统将"类型"作为项的静态标签，确保类型正确的程序不会陷入特定运行时错误。

**关键要点**:

- 类型是静态性质，编译时检查
- 保证强规范化（所有良类型项都有范式）
- 类型检查可判定

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 强规范化 | 良类型项必定归约到范式 | Gentzen证明 |
| 合流性 | 归约结果唯一 | 继承自无类型λ演算 |
| 类型安全 | 进步性 + 保持性 | Subject Reduction |
| 可判定性 | 类型推断可判定 | Hindley-Milner算法 |

**性质总结**:

1. **进步性 (Progress)**: 良类型的闭项要么是值，要么可以归约
2. **保持性 (Preservation)**: 归约保持类型

### 1.3 变体

**Curry风格 (类型推导)**:

- 项不含类型注解，类型由推导获得
- 更简洁，支持多态类型推断
- 适用于ML/Haskell风格语言

**Church风格 (类型标注)**:

- λ抽象带有参数类型：`λx:τ.t`
- 显式类型标注，类型唯一
- 适用于系统F、依赖类型系统

## 三、形式化证明

### 3.1 核心定理

**定理 1** (类型保持性 / Subject Reduction):
若 $\Gamma \vdash t : \tau$ 且 $t \to_\beta t'$，则 $\Gamma \vdash t' : \tau$。

**证明**:
对推导 $t \to_\beta t'$ 进行归纳。

**关键情形** (β归约):
假设 $t = (\lambda x:\tau_1.t_1)\ v$ 且 $t' = t_1[x:=v]$，其中 $v$ 是值。

由类型假设：

- $\Gamma \vdash \lambda x:\tau_1.t_1 : \tau_1 \to \tau_2$ （应用规则左部）
- 因此 $\Gamma, x:\tau_1 \vdash t_1 : \tau_2$ （抽象规则）
- $\Gamma \vdash v : \tau_1$ （应用规则右部）

由替换引理：

- $\Gamma \vdash t_1[x:=v] : \tau_2$ ∎

**证明要点分析**:

1. **替换引理**: 类型正确项的替换保持类型
2. **归纳结构**: 按归约规则结构归纳
3. **变量条件**: 确保自由变量在上下文中

**定理 2** (进步性):
若 $\vdash t : \tau$（$t$为闭项），则要么$t$是值，要么存在$t'$使得$t \to t'$。

**定理 3** (强规范化):
所有良类型的项都有有限的归约序列终止于范式。

### 3.2 辅助引理

**引理 1** (弱化):
若 $\Gamma \vdash t : \tau$ 且 $x \not\in dom(\Gamma)$，则 $\Gamma, x:\sigma \vdash t : \tau$。

**引理 2** (替换):
若 $\Gamma, x:\sigma \vdash t : \tau$ 且 $\Gamma \vdash s : \sigma$，则 $\Gamma \vdash t[x:=s] : \tau$。

**引理 3** (反演):

- 若 $\Gamma \vdash x : \tau$，则 $(x:\tau) \in \Gamma$
- 若 $\Gamma \vdash \lambda x.t : \tau_1 \to \tau_2$，则 $\Gamma, x:\tau_1 \vdash t : \tau_2$
- 若 $\Gamma \vdash t\ s : \tau$，则存在$\sigma$使得$\Gamma \vdash t : \sigma \to \tau$且$\Gamma \vdash s : \sigma$

## 四、算法/方法详解

### 4.1 类型推导算法

```
算法: 类型推导
输入: 上下文Γ，项t
输出: 类型τ或失败

1. case t of
2.   x → 查找(x:τ) ∈ Γ，返回τ
3.   λx.t' →
4.     生成新类型变量α
5.     τ' ← 推导(Γ, x:α, t')
6.     返回 α → τ'
7.   t1 t2 →
8.     τ1 ← 推导(Γ, t1)
9.     τ2 ← 推导(Γ, t2)
10.    生成新类型变量β
11.    统一(τ1, τ2 → β)
12.    返回 β
```

**统一算法 (Martelli-Montanari)**:

```
规则:
1. τ = τ → 删除
2. τ1→τ2 = σ1→σ2 → 替换为 τ1=σ1, τ2=σ2
3. α = τ (α∉FV(τ)) → 代入
4. 结构不同 → 失败
```

### 4.2 正确性分析

**统一算法正确性**:

- **终止性**: 每次规则应用减少度量
- **可靠性**: 若返回σ，则σ是合一元
- **完备性**: 若存在合一元，算法找到最一般的

**类型推断正确性**:

- 若算法返回τ，则$\Gamma \vdash t : \tau$
- 若$\Gamma \vdash t : \tau$，则算法成功并返回更一般的类型

### 4.3 复杂度分析

| 操作 | 复杂度 |
|------|--------|
| 无let类型推断 | O(n) |
| HM算法（有let） | O(n)平均，最坏O(n³) |
| 统一 | 几乎线性O(α(n)) |

## 五、示例与实例

### 5.1 标准示例

**示例 1**: 推导 `λx.λy.x y` 的类型

**推导过程**:

```
1. 假设 x : α, y : β
2. 需要推导 x y : γ
3. 因此 x : β → γ
4. 统一 α = β → γ
5. 整体类型: α → β → γ = (β → γ) → β → γ
```

**结果**: `(β → γ) → β → γ`（应用组合子类型）

---

**示例 2**: 类型推导树

推导 `λf.λx.f (f x)` 的类型：

$$
\frac{
  \frac{
    \frac{(f:\alpha \to \beta) \in \Gamma_1 \quad (x:\alpha) \in \Gamma_1}
         {\Gamma_1 \vdash f : \alpha \to \beta} \quad
    \frac{(f:\alpha \to \alpha) \in \Gamma_1 \quad (x:\alpha) \in \Gamma_1}
         {\Gamma_1 \vdash f\ x : \alpha}}
       {\Gamma_1 = f:\alpha \to \alpha, x:\alpha \vdash f\ (f\ x) : \alpha}}
  {f:\alpha \to \alpha \vdash \lambda x.f\ (f\ x) : \alpha \to \alpha}}
{\vdash \lambda f.\lambda x.f\ (f\ x) : (\alpha \to \alpha) \to \alpha \to \alpha}
$$

**类型**: $(\alpha \to \alpha) \to \alpha \to \alpha$（重复应用函数）

### 5.2 代码实现

**Coq实现**:

```coq
(* 类型定义 *)
Inductive ty : Type :=
  | Base : nat -> ty
  | Arrow : ty -> ty -> ty.

Notation "t1 → t2" := (Arrow t1 t2) (at level 50).

(* 上下文: 变量到类型的部分映射 *)
Definition context := list (nat * ty).

(* 查找 *)
Fixpoint lookup (n : nat) (Γ : context) : option ty :=
  match Γ with
  | [] => None
  | (m, t) :: rest => if n =? m then Some t else lookup n rest
  end.

(* 类型推导关系 *)
Inductive has_type : context -> term -> ty -> Prop :=
  | T_Var : forall Γ x τ,
      lookup x Γ = Some τ ->
      has_type Γ (var x) τ
  | T_Abs : forall Γ x t τ1 τ2,
      has_type ((x, τ1) :: Γ) t τ2 ->
      has_type Γ (abs x t) (Arrow τ1 τ2)
  | T_App : forall Γ t1 t2 τ1 τ2,
      has_type Γ t1 (Arrow τ1 τ2) ->
      has_type Γ t2 τ1 ->
      has_type Γ (app t1 t2) τ2.

(* 进度性定理 *)
Theorem progress : forall t τ,
  has_type [] t τ ->
  value t \/ exists t', step t t'.

(* 保持性定理 *)
Theorem preservation : forall Γ t t' τ,
  has_type Γ t τ ->
  step t t' ->
  has_type Γ t' τ.
```

**Agda实现**:

```agda
-- 简单类型定义
data Type : Set where
  base : ℕ → Type
  _⇒_ : Type → Type → Type

-- 上下文
Context : Set
Context = List (ℕ × Type)

-- 类型判断关系
data _⊢_∶_ : Context → Term → Type → Set where
  T-var : ∀ {Γ x τ} → (x , τ) ∈ Γ → Γ ⊢ var x ∶ τ

  T-abs : ∀ {Γ x t τ₁ τ₂} →
    ((x , τ₁) ∷ Γ) ⊢ t ∶ τ₂ →
    Γ ⊢ (ƛ x ⇒ t) ∶ (τ₁ ⇒ τ₂)

  T-app : ∀ {Γ t₁ t₂ τ₁ τ₂} →
    Γ ⊢ t₁ ∶ (τ₁ ⇒ τ₂) →
    Γ ⊢ t₂ ∶ τ₁ →
    Γ ⊢ (t₁ · t₂) ∶ τ₂
```

**Lean实现**:

```lean
-- 类型定义
inductive ty
| base : ℕ → ty
| arrow : ty → ty → ty

notation t1 ` →' ` t2 := ty.arrow t1 t2

-- 上下文
def context := list (ℕ × ty)

-- 查找
def lookup (n : ℕ) : context → option ty
| [] := none
| ((m, t) :: rest) := if n = m then some t else lookup rest

-- 类型判断
inductive has_type : context → term → ty → Prop
| T_var {Γ x τ} : lookup x Γ = some τ → has_type Γ (term.var x) τ
| T_abs {Γ x t τ₁ τ₂} :
    has_type ((x, τ₁) :: Γ) t τ₂ →
    has_type Γ (term.abs x t) (ty.arrow τ₁ τ₂)
| T_app {Γ t₁ t₂ τ₁ τ₂} :
    has_type Γ t₁ (ty.arrow τ₁ τ₂) →
    has_type Γ t₂ τ₁ →
    has_type Γ (term.app t₁ t₂) τ₂
```

### 5.3 反例

**常见错误1**: 类型不匹配的应用

```
错误: (λx:Bool.x) (λy:Int.y)
错误原因: 期望Bool，实际得到Int → Int
正确做法: 类型检查失败
```

**常见错误2**: 自应用

```
错误: λx.x x
错误原因: x需要同时是τ和τ→σ
正确理解: 简单类型中无法类型化
```

## 六、习题

### 6.1 理解题 (L1-L2)

1. **类型推导** [难度⭐]

   推导以下项的类型：
   - `λx.λy.x`
   - `λf.λg.λx.f (g x)`

   <details>
   <summary>解答</summary>

   1. `λx.λy.x : α → β → α`（K组合子）

   2. `λf.λg.λx.f (g x) : (β → γ) → (α → β) → α → γ`（函数复合）

   </details>

2. **类型规则应用** [难度⭐]

   写出 `λf.λx.f x` 的完整类型推导树。

   <details>
   <summary>解答</summary>

   ```
   f:α→β, x:α ⊢ f : α→β    f:α→β, x:α ⊢ x : α
   ------------------------------------------- T-App
   f:α→β, x:α ⊢ f x : β
   ------------------------ T-Abs
   f:α→β ⊢ λx.f x : α → β
   ----------------------------- T-Abs
   ⊢ λf.λx.f x : (α→β) → α → β
   ```

   </details>

### 6.2 应用题 (L3)

1. **统一算法** [难度⭐⭐]

   使用Martelli-Montanari算法求解以下统一问题：
   $$\alpha \to \beta = \gamma \to (\delta \to \gamma)$$

   <details>
   <summary>解答</summary>

   ```
   初始: {α→β = γ→(δ→γ)}

   步骤1: 分解函数类型
   {α = γ, β = δ→γ}

   步骤2: 第一个方程
   代入[α:=γ]
   {β = δ→γ}

   结果: [α:=γ, β:=δ→γ]

   最一般合一元(mgu)。
   ```

   </details>

2. **类型推断** [难度⭐⭐]

   为 `let id = λx.x in id id` 推导类型。

   <details>
   <summary>解答</summary>

   ```
   id = λx.x : α → α

   在 id id 中:
   - 第一个id实例化: (β → β) → (β → β)
   - 第二个id实例化: β → β

   结果: β → β
   ```

   这是多态let的典型案例。

   </details>

### 6.3 证明题 (L4-L5)

1. **弱化引理** [难度⭐⭐⭐]

   证明：若 $\Gamma \vdash t : \tau$ 且 $x \not\in dom(\Gamma)$，则 $\Gamma, x:\sigma \vdash t : \tau$。

   <details>
   <summary>解答</summary>

   对推导$\Gamma \vdash t : \tau$进行归纳：

   **情况 T-Var**: $t = y$，$(y:\tau) \in \Gamma$
   - 由于$x \not\in dom(\Gamma)$，有$y \neq x$
   - 因此$(y:\tau) \in \Gamma, x:\sigma$
   - 由T-Var，$\Gamma, x:\sigma \vdash y : \tau$ ✓

   **情况 T-Abs**: $t = \lambda y.t_1$，$\tau = \tau_1 \to \tau_2$
   - 归纳假设: $\Gamma, y:\tau_1 \vdash t_1 : \tau_2$ 蕴含
     $\Gamma, y:\tau_1, x:\sigma \vdash t_1 : \tau_2$
   - 由于$x \not\in dom(\Gamma)$，可交换$x$和$y$
   - 由T-Abs得证 ✓

   **情况 T-App**: 直接应用归纳假设于两个前提。

   </details>

2. **类型唯一性** [难度⭐⭐⭐⭐]

   证明在Church风格简单类型系统中，若$\Gamma \vdash t : \tau$和$\Gamma \vdash t : \tau'$，则$\tau = \tau'$。

   <details>
   <summary>解答</summary>

   对$t$的结构进行归纳：

   **变量**: 由反演引理，$(x:\tau) \in \Gamma$和$(x:\tau') \in \Gamma$
   - 上下文中变量类型唯一，故$\tau = \tau'$

   **抽象** $t = \lambda x:\tau_1.t_2$:
   - 由反演，$\tau = \tau_1 \to \tau_2$且$\tau' = \tau_1 \to \tau_2'$
   - 其中$\Gamma, x:\tau_1 \vdash t_2 : \tau_2$和$\Gamma, x:\tau_1 \vdash t_2 : \tau_2'$
   - 归纳假设: $\tau_2 = \tau_2'$
   - 因此$\tau = \tau'$ ✓

   **应用** $t = t_1\ t_2$:
   - 由反演，存在$\sigma$使得$\Gamma \vdash t_1 : \sigma \to \tau$
   - 且$\Gamma \vdash t_1 : \sigma' \to \tau'$
   - 归纳假设于$t_1$: $\sigma \to \tau = \sigma' \to \tau'$
   - 因此$\sigma = \sigma'$且$\tau = \tau'$ ✓

   </details>

Haskell基于Hindley-Milner类型推断，是简单类型λ演算的扩展。

**应用方式**:

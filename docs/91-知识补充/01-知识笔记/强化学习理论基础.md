## 一、核心概念

### 1.1 定义

**正式定义**:
**马尔可夫决策过程（MDP）**: 元组 $(\mathcal{S}, \mathcal{A}, \mathcal{P}, \mathcal{R}, \gamma)$

- $\mathcal{S}$: 状态空间
- $\mathcal{A}$: 动作空间
- $\mathcal{P}(s'|s,a)$: 转移概率
- $\mathcal{R}(s,a)$: 奖励函数
- $\gamma \in [0,1)$: 折扣因子

**策略**: $\pi(a|s)$，状态到动作分布的映射

**价值函数**:

- 状态价值: $V^\pi(s) = \mathbb{E}[\sum_{t=0}^\infty \gamma^t R_t | S_0 = s, \pi]$
- 动作价值: $Q^\pi(s,a) = \mathbb{E}[\sum_{t=0}^\infty \gamma^t R_t | S_0=s, A_0=a, \pi]$

**Bellman方程**:
$$V^\pi(s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a)[R(s,a) + \gamma V^\pi(s')]$$

**最优策略**: $\pi^* = \arg\max_\pi V^\pi(s)$ 对所有 $s$

**Q-Learning**: 离策略时序差分控制算法
$$Q(s,a) \leftarrow Q(s,a) + \alpha [r + \gamma \max_{a'} Q(s',a') - Q(s,a)]$$

**策略梯度**: $\nabla J(\theta) = \mathbb{E}[\nabla_\theta \log \pi_\theta(a|s) \cdot Q^{\pi_\theta}(s,a)]$

**直观解释**:
强化学习就像"训练宠物"——智能体（宠物）在环境（世界）中采取行动，获得奖励或惩罚，目标是学习最大化长期奖励的策略。MDP是"世界规则说明书"，价值函数是"状态好坏评分"，Q-Learning是"试错学习法"。

**关键要点**:

- **探索vs利用**: 试新动作 vs 用已知好动作
- **信用分配**: 长期奖励归因
- **离/在策略**: 学习策略与执行策略是否相同
- **收敛保证**: 表格情形下Q-Learning收敛到最优

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 马尔可夫性 | 未来只依赖当前状态 | 历史无关 |
| 折扣因子 | 未来奖励的现值 | 避免无穷和 |
| 平稳性 | 策略和价值不随时间变 | 非平稳问题更复杂 |
| 可观测性 | 完全/部分可观测 | POMDP更复杂 |

### 1.3 变体

**蒙特卡洛方法**: 基于完整回合的回报估计
**时序差分（TD）**: 自举，用当前估计更新
**SARSA**: 在策略TD控制
**Actor-Critic**: 结合价值函数和策略梯度
**TRPO/PPO**: 带约束的策略优化

## 三、形式化证明

### 3.1 核心定理

**定理 1 (Bellman最优性)**: 最优价值函数 $V^*$ 是Bellman最优方程的唯一解：
$$V^*(s) = \max_a \sum_{s'} P(s'|s,a)[R(s,a) + \gamma V^*(s')]$$

**证明**:
**存在性**: 由压缩映射定理，Bellman算子 $T$ 是压缩的，有唯一不动点。

**最优性**: 设 $\pi^*$ 为贪心策略，则 $V^{\pi^*} = V^*$。

∎

**定理 2 (Q-Learning收敛性)**: 在表格情形下，若 $\sum \alpha_t(s,a) = \infty$ 且 $\sum \alpha_t^2(s,a) < \infty$，Q-Learning以概率1收敛到 $Q^*$。

*证明*: 随机逼近理论， Robbins-Monro条件。∎

### 3.2 辅助引理

**引理 1 (Bellman算子的压缩性)**: $\|TV - TU\|_\infty \leq \gamma \|V - U\|_\infty$

*证明*: 由折扣因子的性质直接得出。∎

## 四、算法/方法详解

### 4.1 算法描述

**Q-Learning**:

```
初始化 Q(s,a) 对所有 s,a
for episode = 1 to M do
    初始化状态 s
    for t = 1 to T do
        选择动作 a（ε-贪心）
        执行a，观察 r, s'
        Q(s,a) ← Q(s,a) + α[r + γ*max_a' Q(s',a') - Q(s,a)]
        s ← s'
    end for
end for
return Q
```

**REINFORCE（蒙特卡洛策略梯度）**:

```
初始化策略参数 θ
for episode = 1 to M do
    生成回合 τ = (s0,a0,r0, ..., sT)
    for t = 0 to T do
        G ← Σ γ^k * r_{t+k}  // 回报
        θ ← θ + α * γ^t * G * ∇_θ log π_θ(a_t|s_t)
    end for
end for
return θ
```

## 五、示例与实例

### 5.1 网格世界示例

**场景**: 4x4网格，(0,0)起点，(3,3)终点(+1)，(1,1)陷阱(-1)

**状态**: 16个格子
**动作**: 上、下、左、右
**奖励**: 每步-0.01，终点+1，陷阱-1

**值迭代后**: 靠近终点的格子价值高，陷阱附近价值低

### 5.2 代码实现

```python
import numpy as np

class QLearning:
    def __init__(self, n_states, n_actions, lr=0.1, gamma=0.99, epsilon=0.1):
        self.q_table = np.zeros((n_states, n_actions))
        self.lr = lr
        self.gamma = gamma
        self.epsilon = epsilon
        self.n_actions = n_actions

    def choose_action(self, state):
        if np.random.uniform() < self.epsilon:
            return np.random.randint(self.n_actions)
        return np.argmax(self.q_table[state])

    def learn(self, state, action, reward, next_state, done):
        q_predict = self.q_table[state, action]
        if done:
            q_target = reward
        else:
            q_target = reward + self.gamma * np.max(self.q_table[next_state])
        self.q_table[state, action] += self.lr * (q_target - q_predict)

# 简单环境
class GridWorld:
    def __init__(self, size=4):
        self.size = size
        self.n_states = size * size
        self.n_actions = 4  # 上、下、左、右
        self.goal = size * size - 1

    def reset(self):
        self.state = 0
        return self.state

    def step(self, action):
        row, col = divmod(self.state, self.size)
        if action == 0: row = max(0, row-1)  # 上
        elif action == 1: row = min(self.size-1, row+1)  # 下
        elif action == 2: col = max(0, col-1)  # 左
        elif action == 3: col = min(self.size-1, col+1)  # 右

        self.state = row * self.size + col
        reward = 1.0 if self.state == self.goal else -0.01
        done = self.state == self.goal
        return self.state, reward, done

# 训练
env = GridWorld()
agent = QLearning(env.n_states, env.n_actions)

for episode in range(1000):
    state = env.reset()
    done = False
    while not done:
        action = agent.choose_action(state)
        next_state, reward, done = env.step(action)
        agent.learn(state, action, reward, next_state, done)
        state = next_state

print("Q表:")
print(agent.q_table)
```

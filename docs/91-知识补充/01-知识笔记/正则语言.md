## 一、核心概念

### 1.1 定义

**形式定义**:
**正则语言**是可以被以下任一形式描述的语言类：

1. **有限自动机** (DFA/NFA)
2. **正则表达式**
3. **正则文法** (右线性文法)
4. **有限幺半群**

**DFA定义**: 五元组 M = (Q, Σ, δ, q₀, F)

- Q: 有限状态集
- Σ: 有限字母表
- δ: Q × Σ → Q 转移函数
- q₀ ∈ Q: 初始状态
- F ⊆ Q: 接受状态集

### 1.2 等价性

**定理**: 以下描述能力等价：

- DFA = NFA = ε-NFA = 正则表达式

**转换关系**:

```
DFA ←→ NFA ←→ ε-NFA ←→ 正则表达式
```

## 二、核心定理

### 2.1 Kleene定理

**定理** (Kleene): 一个语言是正则的当且仅当它可被正则表达式描述。

**证明概要**:

```
(⇒) 正则表达式 → ε-NFA:
    对正则表达式的结构归纳：
    - ∅: 两个状态，无转移
    - ε: 直接连接初始和接受
    - a: a标记的转移
    - R₁∪R₂: 并行连接
    - R₁R₂: 串行连接
    - R*: Kleene星

(⇐) DFA → 正则表达式:
    使用状态消除法或代数方法。
```

### 2.2 泵引理

**定理** (正则语言泵引理): 若L是正则语言，则存在 pumping length p，使得对任意s ∈ L且|s| ≥ p，s可分解为xyz满足：

1. |xy| ≤ p
2. |y| ≥ 1
3. 对所有i ≥ 0，xyⁱz ∈ L

**应用**: 证明语言非正则

## 三、自动机实现

### 3.1 DFA模拟

```python
class DFA:
    def __init__(self, states, alphabet, transition, start, accepts):
        self.states = states
        self.alphabet = alphabet
        self.transition = transition  # {(state, char): next_state}
        self.start = start
        self.accepts = accepts

    def accepts_string(self, string):
        """判断DFA是否接受字符串"""
        current = self.start
        for char in string:
            if char not in self.alphabet:
                return False
            current = self.transition.get((current, char))
            if current is None:
                return False
        return current in self.accepts


# 示例：接受以01结尾的字符串
dfa = DFA(
    states={'q0', 'q1', 'q2'},
    alphabet={'0', '1'},
    transition={
        ('q0', '0'): 'q1',
        ('q0', '1'): 'q0',
        ('q1', '0'): 'q1',
        ('q1', '1'): 'q2',
        ('q2', '0'): 'q1',
        ('q2', '1'): 'q0',
    },
    start='q0',
    accepts={'q2'}
)

print(dfa.accepts_string('101'))  # True
print(dfa.accepts_string('111'))  # False
```

## 四、正则表达式

### 4.1 基本操作

| 操作 | 含义 | 示例 |
|------|------|------|
| ab | 连接 | "ab"匹配ab |
| a\|b | 或 | 匹配a或b |
| a* | Kleene星 | 零或多个a |
| a+ | 正闭包 | 一或多个a (aa*) |
| a? | 可选 | 零或一个a (ε\|a) |

### 4.2 Python正则应用

```python
import re

# 匹配邮箱
pattern = r'^[\w.-]+@[\w.-]+\.\w+$'
print(re.match(pattern, 'test@example.com'))

# 匹配IP地址
ip_pattern = r'^(\d{1,3}\.){3}\d{1,3}$'
```

## 一、核心概念

### 1.1 定义

**正式定义**:
椭圆曲线 $E$ 是Weierstrass方程定义的平面曲线：
$$E: y^2 = x^3 + ax + b$$
其中判别式 $\Delta = -16(4a^3 + 27b^2) \neq 0$。

**群运算**: 椭圆曲线点集与无穷远点 $\mathcal{O}$ 构成Abel群。

**椭圆曲线离散对数问题（ECDLP）**: 给定 $P, Q = kP$，求 $k$ 是计算困难的。

**ECDH密钥交换**:

1. Alice选择 $a$，发送 $A = aG$
2. Bob选择 $b$，发送 $B = bG$
3. 共享密钥: $S = aB = bA = abG$

**直观解释**:
椭圆曲线密码学利用"在曲线上沿直线跳跃"的几何运算构建困难问题。就像在一个巨大迷宫（椭圆曲线上的点群）中，从一个起点（基点）出发，按固定规则跳跃若干次（私钥）到达终点（公钥）。知道跳跃次数到终点容易，但从终点反推跳跃次数困难。

**关键要点**:

- **短密钥高安全**: 256位ECC密钥 ≈ 3072位RSA安全性
- **计算效率**: 点乘运算比模幂更快
- **困难问题**: ECDLP被认为比整数分解更难

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 群结构 | 点集构成有限Abel群 | 群运算有高效算法 |
| Hasse界 | $|\#E(\mathbb{F}_q) - (q+1)| \leq 2\sqrt{q}$ | 点数上下界 |
| 嵌入度 | 曲线到乘法群的最小扩张次数 | 影响MOV攻击效率 |

### 1.3 变体

**素域曲线 $E(\mathbb{F}_p)$**: 定义在 $\mathbb{F}_p$ 上，实现简单
**二元扩域曲线 $E(\mathbb{F}_{2^m})$**: 硬件实现高效
**Edwards曲线**: 完整加法公式，计算更快

## 三、形式化证明

### 3.1 核心定理

**定理 1 (椭圆曲线群结构的结合律)**: 点加运算满足结合律。

**证明**: 使用Weil互反律或直接的代数验证，通过Cayley-Bacharach定理证明。∎

**定理 2 (Hasse界)**: $|q + 1 - \#E(\mathbb{F}_q)| \leq 2\sqrt{q}$

**证明**: 考虑Frobenius自同态的迹，证明判别式 $t^2 - 4q \leq 0$。∎

### 3.2 辅助引理

**引理 1**: 点加的显式公式可由直线与曲线的交点计算得出。

## 四、算法/方法详解

### 4.1 算法描述

**椭圆曲线点加**:

```
输入: P=(x1,y1), Q=(x2,y2)
输出: R = P + Q

if P = O then return Q
if Q = O then return P
if x1 = x2 then
    if y1 ≠ y2 or y1 = 0 then return O
    λ = (3*x1^2 + a) / (2*y1)  // 倍点
else
    λ = (y2 - y1) / (x2 - x1)  // 不同点
x3 = λ^2 - x1 - x2
y3 = λ*(x1 - x3) - y1
return (x3, y3)
```

**标量乘法（Double-and-Add）**:

```
输入: k, P
输出: Q = kP

Q = O
for bit in binary(k) from MSB to LSB:
    Q = 2Q
    if bit = 1 then Q = Q + P
return Q
```

### 4.2 复杂度分析

**点加**: 1次模逆 + 2-3次模乘
**标量乘法**: $O(\log n)$ 次点运算

## 五、示例与实例

### 5.1 标准示例

**secp256k1参数**:

- $p = 2^{256} - 2^{32} - 977$
- $a = 0, b = 7$
- 基点 $G$ 坐标已知

**ECDSA签名**:

1. 选择随机 $k \in [1, n-1]$
2. 计算 $(x, y) = kG$
3. $r = x \mod n$
4. $s = k^{-1}(H(m) + dr) \mod n$

### 5.2 代码实现

```python
# 简化示例（完整实现见加密库）
def point_add(P, Q, a, p):
    if P is None: return Q
    if Q is None: return P
    x1, y1 = P
    x2, y2 = Q
    if x1 == x2:
        if y1 != y2: return None
        lam = (3*x1*x1 + a) * pow(2*y1, -1, p) % p
    else:
        lam = (y2 - y1) * pow(x2 - x1, -1, p) % p
    x3 = (lam*lam - x1 - x2) % p
    y3 = (lam*(x1 - x3) - y1) % p
    return (x3, y3)

def scalar_mult(k, P, a, p):
    result = None
    addend = P
    while k:
        if k & 1:
            result = point_add(result, addend, a, p)
        addend = point_add(addend, addend, a, p)
        k >>= 1
    return result
```

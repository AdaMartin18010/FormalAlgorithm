## 一、核心概念

### 1.1 定义

**正式定义**:
**优化问题**: $\min_{x \in \mathcal{X}} f(x)$，其中 $f$ 为目标函数，$\mathcal{X}$ 为可行域。

**梯度下降**: $x_{t+1} = x_t - \eta \nabla f(x_t)$，其中 $\eta$ 为学习率。

**随机梯度下降（SGD）**: 使用随机样本的梯度估计：$x_{t+1} = x_t - \eta \nabla f_i(x_t)$

**动量法**: $v_{t+1} = \beta v_t + \nabla f(x_t)$，$x_{t+1} = x_t - \eta v_{t+1}$

**Adam**: 自适应矩估计，结合动量和学习率自适应。

**收敛率**: $O(1/T)$ 凸函数，$O(1/\sqrt{T})$ 一般非凸。

**直观解释**:
梯度下降就像"下山"——你站在山坡上，每次朝最陡的方向（梯度的反方向）走一步，最终会到达谷底（最小值）。动量法就像"滚雪球"，下山时积累速度，越过小坑洼。Adam则是"智能导游"，根据路况自动调整步长。

**关键要点**:
- **学习率选择**: 太大不收敛，太慢收敛慢
- **局部最小值**: 非凸优化可能陷入
- **随机性**: SGD引入噪声帮助逃离鞍点
- **自适应**: Adam等算法自动调整学习率

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 收敛性 | 凸函数保证收敛到最优 | 非凸可能收敛到局部最优 |
| 收敛速度 | 与学习率、条件数相关 | 病态问题收敛慢 |
| 内存需求 | SGD $O(1)$，二阶方法 $O(d^2)$ | $d$ 为参数维度 |
| 并行性 | 可数据并行、模型并行 | 分布式训练基础 |

### 1.3 变体

**SGD**: 基本随机梯度下降
**Momentum**: 累积动量加速
**AdaGrad**: 自适应学习率，适合稀疏梯度
**RMSprop**: 指数移动平均的AdaGrad
**Adam**: 结合Momentum和RMSprop

## 三、形式化证明

### 3.1 核心定理

**定理 1 (GD收敛性)**: 对于 $L$-光滑凸函数 $f$，步长 $\eta \leq 1/L$，梯度下降满足：
$$f(x_T) - f^* \leq \frac{\|x_0 - x^*\|^2}{2\eta T}$$

**证明**:
由 $L$-光滑性：
$$f(y) \leq f(x) + \nabla f(x)^T(y-x) + \frac{L}{2}\|y-x\|^2$$

代入 $y = x_{t+1} = x_t - \eta \nabla f(x_t)$：
$$f(x_{t+1}) \leq f(x_t) - \eta(1 - \frac{\eta L}{2})\|\nabla f(x_t)\|^2$$

由凸性，$\|\nabla f(x_t)\|^2 \geq \frac{1}{\|x_t - x^*\|^2}(f(x_t) - f^*)^2$，递推得证。

∎

**定理 2 (SGD收敛性)**: 对于凸函数，步长 $\eta_t = O(1/\sqrt{t})$，SGD满足：
$$\mathbb{E}[f(\bar{x}_T)] - f^* = O(1/\sqrt{T})$$

*证明*: 利用随机梯度是无偏估计，控制方差。∎

## 四、算法/方法详解

### 4.1 算法描述

**SGD with Momentum**:
```
输入: 初始参数 x, 学习率 η, 动量系数 β
v = 0
for t = 1 to T do
    采样小批量 B
    g = ∇f_B(x)  // 计算梯度
    v = β*v + g  // 更新动量
    x = x - η*v  // 参数更新
end for
return x
```

**Adam**:
```
输入: 初始参数 x, 学习率 η, β1=0.9, β2=0.999, ε=1e-8
m = 0, v = 0
for t = 1 to T do
    采样小批量 B
    g = ∇f_B(x)
    m = β1*m + (1-β1)*g  // 一阶矩估计
    v = β2*v + (1-β2)*g^2  // 二阶矩估计
    m_hat = m / (1-β1^t)  // 偏差修正
    v_hat = v / (1-β2^t)
    x = x - η*m_hat / (sqrt(v_hat) + ε)
end for
return x
```

## 五、示例与实例

### 5.1 收敛示例

**凸函数**: $f(x) = x^2$
- GD以 $O(1/T)$ 收敛到0
- 最优步长 $\eta = 1/2$

**非凸函数**: $f(x) = x \cdot \sin(x)$
- 可能收敛到局部最小值
- 需要随机重启或多起点

### 5.2 代码实现

```python
import numpy as np

def sgd(f, df, x0, lr=0.01, epochs=100):
    """基础SGD"""
    x = x0.copy()
    history = []
    for _ in range(epochs):
        grad = df(x)
        x = x - lr * grad
        history.append(f(x))
    return x, history

def adam(f, df, x0, lr=0.001, beta1=0.9, beta2=0.999, eps=1e-8, epochs=100):
    """Adam优化器"""
    x = x0.copy()
    m, v = np.zeros_like(x), np.zeros_like(x)
    history = []
    
    for t in range(1, epochs+1):
        grad = df(x)
        m = beta1 * m + (1 - beta1) * grad
        v = beta2 * v + (1 - beta2) * (grad ** 2)
        m_hat = m / (1 - beta1**t)
        v_hat = v / (1 - beta2**t)
        x = x - lr * m_hat / (np.sqrt(v_hat) + eps)
        history.append(f(x))
    
    return x, history

# 使用示例
f = lambda x: x**2  # 目标函数
df = lambda x: 2*x  # 梯度

x_opt, hist = adam(f, df, np.array([10.0]), lr=0.1)
print(f"最优解: {x_opt}, 最终值: {f(x_opt)}")
```

## 一、核心概念

### 1.1 定义

**正式定义**:

**两方通信模型**:

- Alice持有输入x ∈ {0,1}ⁿ
- Bob持有输入y ∈ {0,1}ⁿ
- 他们要通过通信计算函数f(x,y)

**确定性通信复杂性 D(f)**: 计算f所需的最小通信比特数（确定性协议）

**随机通信复杂性**:

- **R_ε(f)**: 错误概率≤ε的随机协议
- **R^pub_ε(f)**: 使用公共随机源的随机协议
- **R^priv_ε(f)**: 使用私有随机源的随机协议

**非确定性通信复杂性 N(f)**: 非确定性协议的最小通信

### 1.2 基本技术

**通信矩阵 M_f**:

- 行标记Alice的可能输入
- 列标记Bob的可能输入
- M_f[x,y] = f(x,y)

**矩形 (Rectangle)**: X' × Y'，其中X'⊆X, Y'⊆Y

**关键引理**: 通信c的确定性协议将矩阵划分为最多2^c个单色矩形

### 1.3 下界技术

**矩形 rank 下界**:
$$D(f) \geq \log_2(\text{rank}(M_f))$$

**discrepency 方法**: 衡量矩阵与均匀分布的偏离

**fooling set 方法**: 找到大量输入对，任何矩形不能包含两个

## 三、形式化证明

### 3.1 等式函数 EQ

**问题**: EQ(x,y) = 1 iff x = y

**定理**: D(EQ) = n

**证明**:

(上界) Alice发送整个x，Bob比较，通信n比特。

(下界) 用 fooling set：

- 对角线S = {(x,x) : x∈{0,1}ⁿ}
- |S| = 2ⁿ
- 对任意(x,x)≠(x',x')∈S，f(x,x')=0≠1=f(x,x)
- 因此任何单色矩形最多含S中一个元素

协议将矩阵划分为单色矩形，至少2ⁿ个矩形。
通信至少log(2ⁿ) = n比特。

### 3.2 内积函数 IP

**问题**: IP(x,y) = Σ x_i y_i mod 2

**定理**: D(IP) = Ω(n)

**证明 (Rank方法)**:

M_IP在实数域上的rank = 2ⁿ（满秩）。

对GF(2)：rank = 2ⁿ - 1。

由rank下界：D(IP) ≥ log(rank) = n-1 = Ω(n)。

**随机复杂性**: R^pub_ε(IP) = O(1) 对常数ε！

## 四、算法/方法详解

### 4.1 常见函数复杂性

| 函数 | 定义 | D(f) | R(f) |
|------|------|------|------|
| EQ | x=y? | n | O(log n) |
| GT | x>y? | n | O(log n) |
| DISJ | x∩y=∅? | n | O(√n) |
| IP | Σx_iy_i mod 2 | n | O(1) |

### 4.2 随机协议

**EQ的O(log n)随机协议**:

1. Alice和Bob共享随机素数p，范围[n², 2n²]
2. Alice发送(x mod p)
3. Bob检查(y mod p) = (x mod p)

若x=y，总接受。
若x≠y，Pr[x≡y (mod p)] ≤ O(1/n)（由中国剩余定理）。

通信：O(log n)比特（发送余数）。

## 五、示例与实例

### 5.1 集合不交问题 DISJ

**问题**: DISJ(x,y) = 1 iff x和y作为集合不交

**复杂性**:

- D(DISJ) = n
- R(DISJ) = Θ(√n)（上界用采样，下界用discrepancy）

**应用**: 数据库查询、数据流算法

### 5.2 大数据流算法

**问题**: 数据流中统计不同元素个数

**下界**: 需要Ω(n)空间（由通信复杂性归约）

**思路**:

- Alice和Bob将输入编码为流
- 单遍算法可通信解决DISJ
- 因此需要大空间

## 六、习题

### 6.1 理解题 (L1-L2)

1. **GT函数** [难度⭐⭐]

   证明GT(x,y) = 1 iff x>y（作为二进制数）的通信复杂性为n。

   <details>
   <summary>解答</summary>

   用fooling set S = {(x, x-1) : x∈[2ⁿ]}。

   对(x, x-1)和(x', x'-1)，若x>x'，则：
   - f(x, x-1) = 0（x≯x-1）
   - f(x, x'-1) = 1（x>x'-1）

   因此任何矩形不能同时包含两者。
   至少2ⁿ个矩形，通信≥n。

   </details>

2. **随机与确定性** [难度⭐⭐]

   解释为什么R(EQ) = O(log n) << D(EQ) = n。

   <details>
   <summary>解答</summary>

   随机协议允许小概率错误：
   - Alice发送(x mod p)对随机素数p
   - 若x=y，总正确
   - 若x≠y，错误概率很小

   因此可用对数比特达到高正确率。

   确定性协议必须处理最坏情况，任何短协议都存在无法区分的(x,y)对。

   </details>

### 6.2 证明题 (L3-L4)

1. **Rank下界** [难度⭐⭐⭐]

   证明：D(f) ≥ log₂(rank(M_f))。

   <details>
   <summary>解答</summary>

   设协议通信c比特，划分矩阵为2^c个单色矩形。

   每个单色矩形R的rank ≤ 1（所有元素相等）。

   M_f = Σ R_i（矩形指示矩阵之和）。

   rank(M_f) ≤ Σ rank(R_i) ≤ 2^c。

   因此 c ≥ log(rank)。

   </details>

2. **Discrepancy方法** [难度⭐⭐⭐⭐]

   定义discrepancy并用其证明DISJ下界。

   <details>
   <summary>解答概要</summary>

   Discrepancy: disc(A) = max_R |Pr[(x,y)∈R且f(x,y)=1] - Pr[(x,y)∈R且f(x,y)=0]|

   定理: R_ε(f) ≥ log((1-2ε)/disc(f))。

   对DISJ，disc(DISJ) ≤ O(1/√n)，因此R(DISJ) = Ω(log n)。

   更精细分析得R(DISJ) = Ω(√n)。

   </details>

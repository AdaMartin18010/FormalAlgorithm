## 一、核心概念

### 1.1 定义

**正式定义**:
**自监督学习（Self-Supervised Learning, SSL）**是从无标签数据中学习表示的方法，通过构造辅助任务（Pretext Task）生成监督信号。

**对比学习**: 学习使相似样本靠近、不相似样本远离的表示空间。
**InfoNCE损失**:
$$\mathcal{L} = -\mathbb{E}\left[\log \frac{\exp(f(x)^T f(x^+)/\tau)}{\sum_{x'} \exp(f(x)^T f(x')/\tau)}\right]$$

**SimCLR**: 简单对比学习框架，使用数据增强和对比损失。

**掩码语言模型（MLM）**: BERT等模型的预训练任务，预测被掩码的token。

**直观解释**:
自监督学习就像"自己出题自己做"——没有老师给答案（标签），自己设计练习题（辅助任务）。对比学习是"找不同"：看两张图片是否来自同一张原图的不同视角。掩码语言模型是"填空游戏"：遮住句子中的词，猜测被遮住的是什么。

**关键要点**:

- **数据增强**: 创造正样本对的关键
- **负样本**: 对比学习需要足够多的负样本
- **表示质量**: 下游任务性能评估
- **预训练-微调**: 通用范式

### 1.2 属性

| 属性 | 描述 | 备注 |
|------|------|------|
| 无需标签 | 利用数据结构 | 降低标注成本 |
| 可扩展性 | 可利用海量无标签数据 | 数据越多越好 |
| 迁移性 | 学到的表示可迁移 | 跨任务泛化 |
| 语义层次 | 不同任务学习不同层次特征 | 多层次表示 |

### 1.3 变体

**对比式**: SimCLR、MoCo、BYOL
**生成式**: BERT、GPT、MAE
**蒸馏式**: DINO、BOT
**聚类式**: SwAV、DeepCluster

## 三、形式化证明

### 3.1 核心定理

**定理 1 (InfoNCE与互信息的关系)**: 最小化InfoNCE损失等价于最大化表示与目标之间的互信息下界。

**证明**:
设 $f(x)^T f(x^+)/\tau$ 估计对数密度比 $\log p(x^+|x) / p(x^+)$。

InfoNCE可重写为：
$$\mathcal{L} = -\mathbb{E}_{p(x, x^+)}[\log \frac{p(x, x^+)}{p(x)p(x^+)}] + \text{const} = -I(x; x^+) + \text{const}$$

因此最小化InfoNCE最大化互信息。

∎

**定理 2 (对比学习的表示坍缩)**: 无负样本或适当正则化，对比学习可能坍缩到平凡解（所有样本映射到同一点）。

*证明*: 若 $f(x) = c$ 对所有 $x$，则对比损失为常数，无梯度信号阻止此解。∎

### 3.2 辅助引理

**引理 1 (数据增强的重要性)**: 适当的增强保持语义信息同时创造足够变化，对对比学习至关重要。

## 四、算法/方法详解

### 4.1 算法描述

**SimCLR**:

```
输入: 无标签数据集 D
参数: 温度 τ, 批次大小 N

for sampled minibatch {x_k} do
    for all k do
        // 生成两个视图
        x_{2k-1} = t(x_k), t ~ T  // 随机增强
        x_{2k} = t'(x_k), t' ~ T
    end for

    // 编码
    h_i = f(x_i)  // 编码器
    z_i = g(h_i)  // 投影头

    // 计算对比损失
    for all i in {1, ..., 2N} do
        // 正样本: 来自同一张原图的另一个视图
        // 负样本: 批次中其他样本
        loss_i = -log(exp(sim(z_i, z_{j(i)})/τ) / Σ_{k} exp(sim(z_i, z_k)/τ))
    end for

    更新参数最小化总损失
end for
```

**BERT预训练**:

```
输入: 无标签文本

MLM任务:
    随机选择15%的token
    其中80%替换为[MASK]
    10%替换为随机token
    10%保持不变
    预测原始token

NSP任务 (Next Sentence Prediction):
    50%样本为连续句子对
    50%为随机句子对
    预测是否连续
```

## 五、示例与实例

### 5.1 SimCLR图像表示

**数据集**: ImageNet（无标签）
**增强**: 裁剪、颜色抖动、高斯模糊
**投影头**: 2层MLP
**下游任务**: 线性分类器达到76.5% top-1

### 5.2 代码实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class SimCLR(nn.Module):
    def __init__(self, base_encoder, projection_dim=128, temperature=0.5):
        super().__init__()
        self.encoder = base_encoder
        self.temperature = temperature

        # 投影头
        self.projector = nn.Sequential(
            nn.Linear(512, 512),
            nn.ReLU(),
            nn.Linear(512, projection_dim)
        )

    def forward(self, x1, x2):
        # 编码两个视图
        h1 = self.encoder(x1)
        h2 = self.encoder(x2)

        # 投影
        z1 = F.normalize(self.projector(h1), dim=1)
        z2 = F.normalize(self.projector(h2), dim=1)

        return z1, z2

def nt_xent_loss(z1, z2, temperature=0.5):
    """归一化温度缩放交叉熵损失"""
    batch_size = z1.size(0)
    z = torch.cat([z1, z2], dim=0)  # (2N, D)

    # 相似度矩阵
    sim_matrix = torch.mm(z, z.t()) / temperature  # (2N, 2N)

    # 正样本对: (i, N+i) 和 (N+i, i)
    mask = torch.eye(2*batch_size, device=z.device).bool()
    sim_matrix.masked_fill_(mask, -9e15)

    # 正样本位置
    pos_sim = torch.cat([
        torch.diag(sim_matrix, batch_size),
        torch.diag(sim_matrix, -batch_size)
    ])  # (2N,)

    # 损失
    loss = -torch.log(pos_sim / sim_matrix.exp().sum(dim=1))
    return loss.mean()

# 训练循环
def train_step(model, images, optimizer):
    # 生成两个增强视图
    x1 = augment(images)
    x2 = augment(images)

    z1, z2 = model(x1, x2)
    loss = nt_xent_loss(z1, z2)

    optimizer.zero_grad()
    loss.backward()
    optimizer.step()

    return loss.item()
```

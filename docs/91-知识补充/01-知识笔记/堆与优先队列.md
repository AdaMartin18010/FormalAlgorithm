## 一、核心概念

### 1.1 定义

**堆**: 一种近似完全二叉树的数据结构，满足**堆性质**：

- **最大堆**: 父节点 ≥ 子节点，根为最大值
- **最小堆**: 父节点 ≤ 子节点，根为最小值

**优先队列**: 抽象数据类型，支持：

- Insert(S, x): 插入元素
- Maximum/Minimum(S): 返回最大/最小元素
- Extract-Max/Min(S): 删除并返回最大/最小元素
- Increase/Decrease-Key(S, x, k): 修改键值

### 1.2 堆的操作

| 操作 | 时间复杂度 |
|------|------------|
| Build-Heap | O(n) |
| Insert | O(log n) |
| Extract-Max | O(log n) |
| Heapify | O(log n) |
| Heap-Sort | O(n log n) |

## 二、算法详解

### 2.1 堆的实现

```python
class MaxHeap:
    def __init__(self):
        self.heap = []

    def parent(self, i):
        return (i - 1) // 2

    def left(self, i):
        return 2 * i + 1

    def right(self, i):
        return 2 * i + 2

    def max_heapify(self, i):
        """维护堆性质，O(log n)"""
        l = self.left(i)
        r = self.right(i)
        largest = i

        if l < len(self.heap) and self.heap[l] > self.heap[largest]:
            largest = l
        if r < len(self.heap) and self.heap[r] > self.heap[largest]:
            largest = r

        if largest != i:
            self.heap[i], self.heap[largest] = self.heap[largest], self.heap[i]
            self.max_heapify(largest)

    def build_heap(self, arr):
        """建堆，O(n)"""
        self.heap = arr[:]
        for i in range(len(arr) // 2 - 1, -1, -1):
            self.max_heapify(i)

    def insert(self, key):
        """插入，O(log n)"""
        self.heap.append(float('-inf'))
        self.increase_key(len(self.heap) - 1, key)

    def increase_key(self, i, key):
        """增加键值，O(log n)"""
        if key < self.heap[i]:
            raise ValueError("New key is smaller")
        self.heap[i] = key
        while i > 0 and self.heap[self.parent(i)] < self.heap[i]:
            self.heap[i], self.heap[self.parent(i)] = self.heap[self.parent(i)], self.heap[i]
            i = self.parent(i)

    def extract_max(self):
        """提取最大值，O(log n)"""
        if not self.heap:
            raise IndexError("Heap is empty")
        max_val = self.heap[0]
        self.heap[0] = self.heap[-1]
        self.heap.pop()
        if self.heap:
            self.max_heapify(0)
        return max_val
```

### 2.2 堆排序

```python
def heap_sort(arr):
    """堆排序，O(n log n)"""
    heap = MaxHeap()
    heap.build_heap(arr)

    sorted_arr = []
    for _ in range(len(arr)):
        sorted_arr.append(heap.extract_max())

    return sorted_arr[::-1]  # 最大堆，需要反转
```

```markdown
- 图论问题：3-着色、团、独立集

- 数论问题：子集和

### NP中间问题（NPI）
- 图同构
- 离散对数
- 因数分解

### NP与co-NP的关系
```

#### 2.1 Yao通信模型
```markdown
## Yao的两方通信模型

### 形式化定义
- 输入：Alice有x ∈ {0,1}^n，Bob有y ∈ {0,1}^n
- 目标：计算f(x,y)
- 代价：交换的比特数

### 确定性通信复杂度
D(f) = 最少比特数使得协议正确计算f

### 非确定性通信复杂度
N(f) = 最少比特数使得非确定性协议正确计算f
```

```markdown
## 通信复杂度下界证明方法

### 1. 矩形划分方法
- 单色矩形概念
- 矩形覆盖下界

### 2. 秩方法（Rank Method）
- 通信矩阵的秩
- log(rank(M_f)) 下界

### 3. 差异方法（Discrepancy Method）
- 差异定义
- 与随机化复杂度的关系

### 4. 信息复杂度
- 信息论方法
- 直接和（Direct Sum）结果
```

#### 2.3 经典问题分析
```markdown
## 经典问题通信复杂度

### 相等性问题（EQ）
- D(EQ) = n
- R(EQ) = O(log n)（随机化）

### 不交性问题（DISJ）
- D(DISJ) = n
-  Razborov下界：Ω(n)

### 内积问题（IP）
- D(IP) = n
- 量子通信复杂度：O(√n)
```

#### 3.1 决策树模型
```markdown
## 决策树模型

### 形式化定义
- 决策树：每个内部节点测试一个变量
- 深度：最坏情况查询次数
- 规模：节点总数

### 决策树复杂度度量
- D(f)：确定性决策树复杂度
- R(f)：随机化决策树复杂度
- Q(f)：量子决策树复杂度

### 下界技术
#### 1. 敏感性与块敏感性
- 敏感性s(f)：翻转一个输入位改变输出的最大位置数
- 块敏感性bs(f)：翻转一组位改变输出的最大组数
- 定理：bs(f) ≤ D(f) ≤ bs(f)^3

#### 2. 次数方法
- 多项式次数deg(f)
- 定理：D(f) ≥ deg(f)
- 近似次数与随机化复杂度
```

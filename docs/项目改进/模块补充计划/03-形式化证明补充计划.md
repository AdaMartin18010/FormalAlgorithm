```coq
(* 自然数加法结合律 *)
Theorem add_assoc : forall n m p : nat, n + (m + p) = (n + m) + p.
Proof.
  intros n m p. induction n as [| n' IHn'].
  - reflexivity.
  - simpl. rewrite IHn'. reflexivity.
Qed.
```

2. **Lean示例**:
```lean
-- 列表反转定理
theorem reverse_reverse {α : Type} (l : List α) :
    List.reverse (List.reverse l) = l := by
  induction l with
  | nil => rfl
  | cons x xs ih => simp [List.reverse_cons, ih]
```

3. **Agda示例**:
```agda
-- 向量长度保持
data Vec (A : Set) : ℕ → Set where
  []  : Vec A 0
  _∷_ : {n : ℕ} → A → Vec A n → Vec A (suc n)
```

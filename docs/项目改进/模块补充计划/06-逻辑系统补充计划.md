#### 3.1 LTL详细语义
```markdown
## 线性时序逻辑（LTL）详细语义

### 形式化语义
给定模型 M = (S, →, L)，路径 π = s₀ → s₁ → s₂ → ...

#### 满足关系定义
- M, π ⊨ p  iff  p ∈ L(s₀)
- M, π ⊨ Xφ  iff  M, π¹ ⊨ φ
- M, π ⊨ Fφ  iff  ∃i ≥ 0, M, πⁱ ⊨ φ
- M, π ⊨ Gφ  iff  ∀i ≥ 0, M, πⁱ ⊨ φ
- M, π ⊨ φ U ψ  iff  ∃i ≥ 0, M, πⁱ ⊨ ψ ∧ ∀j < i, M, πʲ ⊨ φ

### 表达力分析
- LTL能/不能表达的性质
- 与正则语言的关系
- 安全性与活性（Safety & Liveness）

### 模型检测算法
- 基于自动机的LTL模型检测
-  tableau构造
- 复杂度分析：PSPACE完全
```

#### 3.2 CTL详细语义
```markdown
## 计算树逻辑（CTL）详细语义

### 形式化语义
- M, s ⊨ EX φ  iff  ∃路径从s开始, M, π¹ ⊨ φ
- M, s ⊨ EG φ  iff  ∃路径从s开始, ∀i ≥ 0, M, πⁱ ⊨ φ
- M, s ⊨ E[φ U ψ]  iff  ∃路径从s开始, ∃i ≥ 0, M, πⁱ ⊨ ψ ∧ ∀j < i, M, πʲ ⊨ φ

### CTL模型检测
- 不动点计算
- 标记算法
- 复杂度：多项式时间

### CTL* 简介
- 表达LTL和CTL
- 模型检测复杂度
```
